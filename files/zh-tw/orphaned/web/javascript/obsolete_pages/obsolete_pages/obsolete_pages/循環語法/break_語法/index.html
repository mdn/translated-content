break 語法

使用 break 語法可以終止 loop、switch、label 語法。

  • 當你使用不加標籤的 break 時，他會立即終止最內層的 while、do-while、for、switch 的循環，並把控制權轉移給後面的語句。
  • 當你使用加上標籤的 break 時，他會終止被指定的標籤所標記的語句。

break 語法如下︰

  1. break;
  2. break label;

第一個語句形式會終止最內層的循環或 switch，第二個語句形式會終止指定標籤的語句。

範例
在以下的範例中，反覆操作陣列裡的元素，直到找到某個索引的元素的值等於 theValue︰

for (i = 0; i < a.length; i++) {
   if (a[i] == theValue)
      break;
}
