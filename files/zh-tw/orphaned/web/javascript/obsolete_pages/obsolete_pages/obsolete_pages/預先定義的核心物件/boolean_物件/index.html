Boolean 物件

Boolean 物件是原始的 Boolean 資料類型包裝而成。使用下面的語法來建立 Boolean 物件︰

booleanObjectName = new Boolean(value)

別把原始的布林值的 true 和 false 與 Boolean 物件的 true 和 false 值給混淆了。任何物件的值只要不是 undefined、null、0、NaN、空字串，即使是值為 false 的 Boolean 物件，當傳入到條件語句裡的時候，其求值結果仍為 true。參閱 if...else 語法 取得更進一步資訊。
