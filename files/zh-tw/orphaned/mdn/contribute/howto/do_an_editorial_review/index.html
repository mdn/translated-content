一個主題的編輯審查包含錯字、拼字、文法、語法或其他文字錯誤。並非所有貢獻者都是語言專家，但他們的知識可以提供極為有用的資訊，因此需要審稿和校對。這就是編輯審查在做的事。

這個主題將會介紹如何進行編輯審查，以幫助確保在 MDN 的資訊是正確的。

有哪些任務？
必要的有審稿和校對。
有哪些東西需要編輯審查？
那些被標記需要審查的主題。
進行編輯審查需要知道什麼？
你需要具有良好的英文文法和拼字能力（即能準確翻譯英文成中文的能力）。編輯審查在於確保文法、拼字和語法的正確和意義，並遵守 MDN 編輯指南。
有哪些步驟？
  1. 選擇一個主題：
    1. 前往待審查清單。這個表列出了要求審查的主題。
    2. 選擇一個中文標題且不帶有 Template: (Template: 包含 MDN 功能碼) 的主題。
    3. 點擊該主題的連結。
  2. 閱讀主題，注意錯字、拼字、文法、語法或其他錯誤。如果你覺得無法應付，請找下一個主題。
  3. 如果沒有錯誤，你不需要編輯主題來標記為已審查。在左方導覽列尋找「快速審查」方塊：
  4. 取消選取 編輯 方塊並點擊 儲存。
  5. 如果有錯誤需要被糾正：
    1. 點擊在頁面頂端附近的 編輯 按鈕，接著會轉到 MDN 編輯器。
    2. 修正你找到的錯字、拼字、文法、語法或其他錯誤。你並不需要修正所有錯誤，但若你不確定你已經完整的審查主題，請留下編輯審查請求。
    3. 在主題尾端留下 版本註解 。像是「編輯審查：修正錯字、文法和拼字」。這能讓其他貢獻者和網站編輯者知道你改了什麼和為什麼。
    4. 取消選取在 需要審查嗎？ 中的 編輯 方塊。他在版本註解下面。
    5. 按下 發布。

你的變更不一定會立刻出現，他可能會因送出和儲存延遲一會。
