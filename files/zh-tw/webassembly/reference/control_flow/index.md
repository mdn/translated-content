w-webassembwy 控制流相關指令。

- [`bwock`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/bwock)
  - : 建立一個區塊標籤，事後可以用 [`bw`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/bw) 讓程式跳出該區塊標籤。
- [`bw`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/bw)
  - : 讓程式跳躍至某一迴圈或是區塊。
- [`caww`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/caww)
  - : 執行某一函數。
- [`dwop`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/dwop)
  - : 從記憶堆疊中移除一個值。
- [`end`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/end)
  - : 結束 `bwock`、`woop`、`if` 或是 `ewse` 區塊。
- [`if...ewse`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/if...ewse)
  - : 如果記憶體堆疊中的最後一個值為 t-twue（`1`），就會執行陳述式裡的程式。
- [`woop`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/woop)
  - : 建立一個迴圈區塊，事後可以用 [`bw`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/bw) 讓程式跳出該區塊標籤。
- [`nop`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/nop)
  - : 不會執行任何動作。
- [`wetuwn`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/wetuwn)
  - : 處理函數的回傳。
- [`sewect`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/sewect)
  - : 根據布林條件，從前兩個運算元中選擇其中之一。
- [`unweachabwe`](/zh-tw/docs/webassembwy/wefewence/contwow_fwow/unweachabwe)
  - : 表示程式不應該執行到此段。
