運算子

JavaScript 有以下幾種運算子。本節說明運算子，且包含有關運算子優先順序的資訊。

  • 代入運算子
  • 比較運算子
  • 算術運算子
  • 位元運算子
  • 邏輯運算子
  • 字串運算子
  • 特殊運算子

JavaScript 有一元和二元兩種運算子，以及特殊的三元運算子，也就是條件運算子。二元運算子需要兩個運算元，一個在運算子之前，一個在運算子之後︰

operand1 operator operand2

例如，3+4 或 x*y。

一元運算子需要一個運算元，可以運算子之前或之後︰

operator operand

或

operand operator

例如，x++ 或 ++x。
