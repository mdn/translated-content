ECMAScript 是一種 scripting language 的規範，JavaScript 即是植基於他實做的。 ECMAScript 是由 ECMA-262 specification 中的 Ecma International standards organization 訂立標準的。以下是目前已經被發表的 ECMAScript 標準：

ECMA 名稱 相關連結 發表日期 基本資訊
ECMA-262 PDF June 1997 ECMAScript standard 的最初始版本
ECMA-262 Edition 2 PDF August 1998 ECMAScript standard 的第二版，同時也是 standard 16262
ECMA-262 Edition 3 PDF Word HTML December 1999 ECMAScript standard 的第二版。JavaScript 1.5 即是此版本的實做，更多資料請見 errata
ECMA-262 Edition 5 PDF HTML December 2009 ECMAScript 5。更多資料請見 ES5 errata 以及 ECMAScript 5 support in Mozilla
ECMA-262 Edition 5.1 PDF June 2011 ECMAScript 5.1。這是目前最新的版本。這個版本同時對上了 3 edition of the international standard ISO/IEC 16262:2011。他只多了 ES5 額外的修正，並沒增加新的 features。
ECMA-357 PDF June 2004 ECMAScript for XML (E4X)。更多資料請見 E4X errata。

   你也可以在 wikipedia ECMAScript entry 得知更多關於 ECMAScript 的歷史。此外，你也可以加入 ECMAScript 下一版名為 Harmony 的修訂，只要透過 ecmascript.org 的公開 wiki 或 es-discuss mailing list 。

相關實做

  • SpiderMonkey - Firefox 使用的 JavaScript engine。
  • Rhino - 以 Java 撰寫的 JavaScript engine 。
  • Tamarin - ActionScript virtual machine (Adobe® Flash® Player 使用)。
  • Other implementations (wikipedia).
