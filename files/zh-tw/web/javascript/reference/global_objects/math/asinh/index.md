## 語法

```js-nolint
Math.asinh(x)
```

### 參數

- `x`
  - : 一個數字。

### 回傳值

`x` 的反雙曲正弦值。

## 描述

由於 `asinh()` 是 `Math` 的靜態方法，你必須使用 `Math.asinh()` 來呼叫它，而非呼叫你建立的 `Math` 物件的方法（`Math` 並非建構子）。

## 範例

### 使用 Math.asinh()

```js
Math.asinh(-Infinity); // -Infinity
Math.asinh(-1); // -0.881373587019543
Math.asinh(-0); // -0
Math.asinh(0); // 0
Math.asinh(1); // 0.881373587019543
Math.asinh(Infinity); // Infinity
```
