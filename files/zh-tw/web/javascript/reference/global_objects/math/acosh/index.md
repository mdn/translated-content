## 語法

```js-nolint
Math.acosh(x)
```

### 參數

- `x`
  - : 一個大於或等於 1 的數字。

### 回傳值

`x` 的反雙曲餘弦值。如果 `x` 小於 1，則回傳 {{jsxref("NaN")}}。

## 描述

由於 `acosh()` 是 `Math` 的靜態方法，你必須使用 `Math.acosh()` 來呼叫它，而非呼叫你建立的 `Math` 物件的方法（`Math` 並非建構子）。

## 範例

### 使用 Math.acosh()

```js
Math.acosh(0); // NaN
Math.acosh(1); // 0
Math.acosh(2); // 1.3169578969248166
Math.acosh(Infinity); // Infinity
```
