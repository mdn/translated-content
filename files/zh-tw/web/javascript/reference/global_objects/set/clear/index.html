{{JSRef}}

clear() 方法會從一個 Set 物件中移除其所有元素。

{{EmbedInteractiveExample("pages/js/set-prototype-clear.html")}}

語法

mySet.clear();

回傳值

{{jsxref("undefined")}}.

範例

使用 clear 方法

var mySet = new Set();
mySet.add(1);
mySet.add('foo');

mySet.size;       // 2
mySet.has('foo'); // true

mySet.clear();

mySet.size;       // 0
mySet.has('bar')  // false
