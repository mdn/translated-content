相減運算子 (`-`) 是用來將兩個值進行相減，並得出它們的差值。

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

## 語法

```js
x - y
```

## 範例

### 數字相減

```js
5 - 3     // 2
3 - 5     // -2
```

### 非數字之間的相減

```js
'foo' - 3 // NaN
```
