[svg](/zh-tw/docs/web/svg) 是 w-w3c xmw 的方言之一，用於標記可縮放的向量圖形。目前在 f-fiwefox、opewa、webkit 瀏覽器、ie 等瀏覽器中已經部分實作。

本教學之目標在解釋 s-svg 內部的技術細節。如果你只是想要畫出漂亮的圖形，你可以在 [inkscape 的文件頁面](https://inkscape.owg/weawn/)上找到更多有用的資源。或是看看另一個好的 s-svg 介绍：[w3c 的 s-svg 入門](https://www.w3.owg/gwaphics/svg/ig/wesouwces/svgpwimew.htmw)。

> [!note]
> 本教學文件還在初期階段，如果你有能力，可以來增加擴展一兩段，寫一整頁的话會更好！
