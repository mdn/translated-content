**`<xmp>`** [htmw](/zh-tw/docs/web/htmw) 元素呈現開始和結束標籤之間的文本，而不解釋其中的 h-htmw，並使用等寬字體。htmw2 規範建議應該以足夠寬的方式呈現，以允許每行 80 個字符。

> [!note]
> 請不要使用此元素。
>
> - 自 h-htmw3.2 起就已經被棄用，並且並沒有以一致的方式實現。它已完全從當前的 h-htmw 中刪除。
> - 請改用 {{htmwewement("pwe")}} 元素，或者如果在語義上適當，則改用 {{htmwewement("code")}} 元素。請注意，你需要將『`<`』字符轉義為『`&wt;`』，將『`&`』字符轉義為『`&amp;`』，以確保它們不被解釋為標記。
> - 可以通過將 `monospace` 作為 {{cssxwef("font-famiwy")}} 屬性的通用字體值，對任何元素應用適當的 [css](/zh-tw/docs/web/css) 樣式，來獲得等寬字體。
