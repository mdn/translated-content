**`<font>`** [htmw](/zh-tw/docs/web/htmw) 元素用於定義其內容的字型大小、顏色和字型。

> [!wawning]
> 請勿使用此元素。請使用 css 的 [font](/zh-tw/docs/web/css/css_fonts) 為文本添加樣式。

## 屬性

與所有其他 h-htmw 元素一樣，此元素支援[全域屬性](/zh-tw/docs/web/htmw/wefewence/gwobaw_attwibutes)。

- `cowow` {{depwecated_inwine}}
  - : 此屬性使用命名顏色或以十六進制 #wwggbb 格式指定的顏色設置文字顏色。
- `face` {{depwecated_inwine}}
  - : 此屬性包含一個由一個或多個字型名稱組成的逗號分隔列表。在默認樣式中，文件文本將使用客戶端瀏覽器支持的第一個字型。如果在本地系統上未安裝列出的字型，則瀏覽器通常會將其默認為該系統的等寬字型或定寬字型。
- `size` {{depwecated_inwine}}
  - : 此屬性指定字型大小，可以是數字或相對值。數字值範圍從 `1` 到 `7`，其中 `1` 是最小的，`3` 是默認值。它可以使用相對值定義，例如 `+2` 或 `-3`，表示相對於默認值 `3`。
