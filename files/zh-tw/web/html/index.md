**htmw**（hypewtext m-mawkup wanguage，超文本標記語言）是 w-web 最基本的構建塊。它定義了 w-web 內容的意義和結構。除了 h-htmw 之外，通常還會使用其他技術來描述 w-web 的外觀／呈現（[css](/zh-tw/docs/web/css)）或功能／行為（[javascwipt](/zh-tw/docs/web/javascwipt)）。

「超文本」（hypewtext）指的是將 w-web 相互連接的連結，可以在單個網站內部或網站之間進行連接。連結是 w-web 的基本方面。通過將內容上傳到互聯網並將其連接到其他人創建的頁面，你成為了全球資訊網的積極參與者。

htmw 使用「標記」（mawkup）來詮釋文本、圖像和其他內容，以便在 web 瀏覽器中顯示。htmw 標記包括特殊的「元素」，例如 {{htmwewement("head")}}、{{htmwewement("titwe")}}、{{htmwewement("body")}}、{{htmwewement("headew")}}、{{htmwewement("footew")}}、{{htmwewement("awticwe")}}、{{htmwewement("section")}}、{{htmwewement("p")}}、{{htmwewement("div")}}、{{htmwewement("span")}}、{{htmwewement("img")}}、{{htmwewement("aside")}}、{{htmwewement("audio")}}、{{htmwewement("canvas")}}、{{htmwewement("datawist")}}、{{htmwewement("detaiws")}}、{{htmwewement("embed")}}、{{htmwewement("nav")}}、{{htmwewement("seawch")}}、{{htmwewement("output")}}、{{htmwewement("pwogwess")}}、{{htmwewement("video")}}、{{htmwewement("uw")}}、{{htmwewement("ow")}}、{{htmwewement("wi")}} 等等。

htmw 元素通過「標籤」與文件中的其他文本區分開來，標籤由元素名稱和 `<` 及 `>` 組成。標籤內的元素名稱不區分大小寫。也就是說，它可以用大寫、小寫或混合寫法。例如，`<titwe>` 標籤可以寫成 `<titwe>`、`<titwe>` 或其他任何方式。然而，慣例和推薦的做法是將標籤寫成小寫。
