HTTP **`418 I'm a teapot`** 錯誤回應碼表示伺服器拒絕沖泡咖啡，因為它永遠是一個茶壺。一個暫時沒有咖啡的組合式咖啡／茶壺應該返回 {{HTTPStatus("503")}}。這個錯誤是對於 1998 年和 2014 年愚人節玩笑中定義的超文字咖啡壺控制協定的參考。

一些網站使用此回應來處理他們不希望處理的請求，例如自動查詢。

## 狀態

```http
418 I'm a teapot
```
