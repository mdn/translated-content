{{HTTPSidebar}}

HTTP **`410 Gone`** 用戶端錯誤回應碼表示在原始伺服器上已不再可訪問目標資源，並且這種情況可能是永久的。

如果你不知道這種情況是暫時的還是永久的，則應改用 {{HTTPStatus(404)}} 狀態碼。

> [!NOTE]
> 預設情況下，410 回應是可緩存的。

## 狀態

```http
410 Gone
```
