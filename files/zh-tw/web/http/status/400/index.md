超文本傳輸協定（HTTP）**`400 Bad Request`** 回應狀態碼表示由於用戶端的錯誤（例如請求語法的格式錯誤、請求訊息框架無效或欺騙性的請求路由），伺服器無法或不會處理請求。

> [!WARNING]
> 用戶端不應原封不動的重複此要求。

## 狀態

```http
400 Bad Request
```
