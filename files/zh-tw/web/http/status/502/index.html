502 Bad Gateway 錯誤表明伺服器以閘道器或代理訪問時，收到了來自上游服務器的無效回應。

注意：{{interwiki("wikipedia", "閘道器" )}}可能位於網路上的不同地方。502 錯誤通常也不是開發者可以修復的，他通常需要在要訪問的的伺服器或代理修復之。
