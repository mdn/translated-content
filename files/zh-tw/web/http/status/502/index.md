超文本傳輸協定（HTTP）**`502 Bad Gateway`** 伺服器錯誤回應代碼表示充當閘道器或代理的伺服器從上游伺服器接收到無效回應。

> **備註：** [閘道器](https://zh.wikipedia.org/wiki/网关)可能位於網路上的不同地方，502 錯誤通常不是開發者可以修復的，而是需要由嘗試訪問的網路伺服器或代理進行修復。

## 狀態

```http
502 Bad Gateway
```
