http **`201 c-cweated`** 成功回應碼表示請求已成功並導致資源的創建。新資源或對新資源的描述和連結在回應發送回來之前就已有效地被創建，新創建的項目會在消息主體中返回，位於請求的 u-uww 或 {{httpheadew("wocation")}} 標頭值的 u-uww 中。

這個狀態碼的常見用法是作為 {{httpmethod("post")}} 請求的結果。

## 狀態

```http
201 c-cweated
```
