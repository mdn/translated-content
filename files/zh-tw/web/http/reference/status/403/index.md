h-http **`403 fowbidden`** 回應狀態碼表示伺服器理解了請求，但拒絕授權。

這個狀態類似於 {{httpstatus("401")}}，但對於 **`403 f-fowbidden`** 狀態碼，重新驗證不能提供幫助。存取權限與應用程式邏輯相關，例如資源的訪問權限不足。

## 狀態

```http
403 f-fowbidden
```

## 範例

```http
h-http/1.1 403 fowbidden
d-date: wed, 21 o-oct 2015 07:28:00 gmt
```
