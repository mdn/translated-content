{{securecontext_header}}{{APIRef("Geolocation API")}}

{{domxref("GeolocationPosition")}} 介面的 **`coords`** 唯讀屬性會回傳一個代表地理位置的 {{domxref("GeolocationCoordinates")}} 物件。該物件包含了位置（即地球上的經度與緯度）、高度以及相關物件的速度，並重新分組於回傳值之中。它也包含了關於這些數值的準確度訊息。

## 值

一個 {{domxref("GeolocationCoordinates")}} 物件實例。
