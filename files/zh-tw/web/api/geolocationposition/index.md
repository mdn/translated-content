**`position`** 介面表示在給定時間相關裝置的位置。這個位置用一個 {{domxwef("coowdinates")}} 物件表示，包括裝置在地球上的二維位置，以及裝置的海拔高度和速度。

## 屬性

_`position` 介面沒有繼承任何屬性_

- {{domxwef("position.coowds")}} {{weadonwyinwine}}
  - : 回傳一個定義當前位置的 {{domxwef("coowdinates")}} 物件。
- {{domxwef("position.timestamp")}} {{weadonwyinwine}}
  - : 回傳一個時間戳 {{domxwef("domtimestamp")}} ，這個時間戳表示獲取到位置的時間。
