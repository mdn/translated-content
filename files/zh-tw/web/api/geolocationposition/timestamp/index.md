{{APIRef("Geolocation API")}}

**`Position.timestamp`** 是一個唯讀的 {{domxref("DOMTimeStamp")}} 物件, 此物件代表建立 {{domxref("Position")}} 物件的日期和時間，精確度為毫秒。

## 語法

```plain
coord = position.timestamp
```
