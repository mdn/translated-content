## 摘要

打開列印視窗來列印當前的文件。

## 語法

```plain
window.print()
```

## 注釋

Starting with Chrome V46.0 this method is blocked inside an {{htmlelement("iframe")}} unless its sandbox attribute has the value `allow-modals`.
