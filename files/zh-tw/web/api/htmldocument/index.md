{{APIRef("HTML DOM")}}

由於歷史原因，{{domxref("Window")}} 物件具有 `window.HTMLDocument` 屬性，其值為 {{DOMxRef("Document")}} 介面。因此你可以將 `HTMLDocument` 視為 {{DOMxRef("Document")}} 的別名，且你可以在 {{DOMxRef("Document")}} 介面的文件中找到 `HTMLDocument` 成員的文件。
