**`HTMLSelectElement.setCustomValidity()`** 方法可為指定的元素設定自定義檢核訊息。使用空字串表示元素*沒有*發生違反自定檢核條件的錯誤。

## 語法

```js
selectElt.setCustomValidity(string);
```

### 參數

- _string_ 為 {{domxref("DOMString")}}，代表錯誤訊息。
