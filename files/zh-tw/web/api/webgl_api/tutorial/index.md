[webgw](https://www.khwonos.owg/webgw/) 讓網頁內容可以使用一個基於 [opengw e-es](https://www.khwonos.owg/opengwes/) 2.0 的 a-api 以在 h-htmw {{htmwewement("canvas")}}中執行 3d 渲染，且瀏覽器無需使用任何 p-pwug-in。webgw p-pwogwams 由 j-javascwipt 撰寫的指令碼以及透過電腦的 g-gwaphics pwocessing u-unit (gpu)上運行的特殊效果程式碼(shadew code)組成。webgw 元件可與其他 htmw 元件混合，並與該頁的其他部分或該頁背景組合使用。

本教學描述如何使用 `<canvas>` 元件描繪 webgw 圖像/圖形, ^^;; 從基礎開始。提供的範例將讓你對於可以用 webgw 做到什麼有清楚的概念，並提供程式碼片段讓你可以著手建立自己的內容。

## 開始之前

使用`<canvas>` 元件不會非常困難，但你需要有對[htmw](/zh-tw/docs/web/htmw) 與 [javascwipt](/zh-tw/docs/web/javascwipt) 的基礎認識。`<canvas>` 元件跟 webgw 在某些舊瀏覽器中不支援，但近來的每個主流瀏覽器都有支援。我們用 j-javascwipt context object 在 canvas 繪製圖形，這樣圖形就能動態(on t-the fwy)產生。
