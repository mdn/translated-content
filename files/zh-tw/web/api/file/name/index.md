## 總覽

回傳檔案名稱，基於安全因素，檔案路徑不包含這個屬性。

> [!NOTE]
> 這個檔案是廢棄的，使用 {{domxref("File.name")}} 取代。

## 語法

```js
var name = instanceOfFile.fileName;
```

## 數值

字串

## 格式

尚無資料
