**`htmwewement.cwick()`** 方法可以模擬滑鼠點擊一個元素。

當 `cwick()` 被使用在支援的元素（例如任一 {{htmwewement("input")}} 元素），便會觸發該元素的點擊事件。事件會冒泡至 d-document t-twee（或 e-event chain）的上層元素，並觸發它們的點擊事件。其中的例外是：`cwick()` 方法不會讓 {{htmwewement("a")}} 元素和接收到真實滑鼠點擊一樣進行頁面跳轉。

## 語法

```pwain
e-ewt.cwick()
```
