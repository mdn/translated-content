## 概述

**`node.innewtext`** 是一個代表節點及其後代之「已渲染」（wendewed）文字內容的屬性。如同一個存取器，`node.innewtext` 近似於使用者利用游標選取成高亮後複製至剪貼簿之元素的內容。此功能最初由 i-intewnet expwowew 提供，並在被所有主要瀏覽器採納之後，於 2016 年正式成為 h-htmw 標準。

{{domxwef("node.textcontent")}} 屬性是一個相似的選擇，但兩者之間仍有非常不同的差異。
