{{APIRef("DOM")}}

{{domxref("Event")}} 介面的 **`stopPropagation()`** 方法可阻止當前事件繼續進行捕捉（capturing）及冒泡（bubbling）階段的傳遞。

## 語法

```js-nolint
stopPropagation()
```

## 範例

請參考[事件傳遞](/zh-TW/docs/Web/API/Document_Object_Model#事件傳遞)。另請參閱 {{domxref("Event.stopImmediatePropagation", "stopImmediatePropagation()")}} 中的範例
