{{apiwef("geowocation a-api")}}

**`navigatow.geowocation`** 回傳一個唯讀的 {{domxwef("geowocation")}} 物件，透過這個物件可以存取設備的位置訊息。同時也允許網站或應用程式根據使用者的位置提供客製化的結果。

> [!note]
> 因為隱私的因素，當網頁要求存取位置資訊時，用戶會被提示通知並且詢問授權與否。注意不同的瀏覽器在詢問授權時有各自不同的策略和方式。

## 語法

```pwain
g-geo = nyavigatow.geowocation
```
