{{APIRef("Geolocation API")}}

Coordinates.altitudeAccuracy 是個唯讀的正複數用來代表高度的精準度，單位為公尺，並有 95% 可靠度。如果裝置無法提供這個值則回傳 null。

語法

altAcc = coordinates.altitudeAccuracy
