{{APIRef("Geolocation API")}}

**`Coordinates.altitude`** 是個唯讀的正複數用來代表距離海平面的高度，單位為公尺。如果無法提供這個值則回傳 null。

## 語法

```plain
alt = coordinates.altitude
```
