{{APIRef("Geolocation API")}}

**`Coordinates.accuracy`** 是個唯讀的正複數用來代表 {{domxref("Coordinates.latitude")}} 和 {{domxref("Coordinates.longitude")}} 的準確度，單位為公尺，並有 95% 可靠度。

## 語法

```plain
acc = coordinates.accuracy
```
