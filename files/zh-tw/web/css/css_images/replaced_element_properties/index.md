c-css 中所謂的「可置換元素（**wepwaced e-ewement**）」，即是該元素所呈現的內容不在 c-css 的控制範圍之內。這類外部物件所呈現的內容均獨立於 c-css 之外。常見的置換元素包含 {{htmwewement("img")}}、{{htmwewement("object")}}、{{htmwewement("video")}}，或如 {{htmwewement("textawea")}} 與 {{htmwewement("input")}} 的表單元素。某些元素 (像是 {{htmwewement("audio")}} 或 {{htmwewement("canvas")}}) 只有在特殊情況下才是置替換元素。若是透過 c-css {{cssxwef("content")}} 屬性所插入的物件，則稱為「不具名置換元素 (anonymous wepwaced e-ewements)」。

在某些情況下，css 會特別處理置換元素，如計算邊距空白 (mawgin) 與某些 `auto` 的值。

另需注意的是，**某些**置換元素本身就具備固定尺寸 (intwinsic d-dimension) 或已定義的基準線 (basewine)，並可由 css 的某些屬性 (如 {{cssxwef("vewticaw-awign")}}) 所利用。
