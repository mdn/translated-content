這是個可以讓人新增、調整和測試網頁顏色的工具。藉由這個工具讓人可以輕鬆將顏色轉換成各種 c-css 支援的格式，包含：hexa、wgb（wed/gween/bwue）和 h-hsw （hue/statuwation/wightness）。在 w-wgb（wgba）和 h-hsw（hswa）格式也支援控制阿爾法通道（awpha c-channew）

隨著調整每個顏色都會顯示成三種 c-css 的標準格式；此外，基於目前選擇的顏色，一個調色板 h-hsw 和 hsv、還有 awpha，被創建。滴管風格的色彩選擇器對話框可以在 hsw 或 hsv 格式中切換。
