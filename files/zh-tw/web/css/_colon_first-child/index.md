**`:first-child`** [CSS](/zh-TW/docs/Web/CSS) [偽類](/zh-TW/docs/Web/CSS/Pseudo-classes)代表一群兄弟元素中的第一個元素。

{{InteractiveExample("CSS Demo: :first-child", "tabbed-shorter")}}

```css interactive-example
p {
  font-weight: bold;
}

li:first-child {
  border: 2px solid orange;
}
```

## 語法

```css
:first-child {
  /* ... */
}
```

## 範例

### 基本範例

#### HTML

```html
<div>
  <p>這段文字被選取了！</p>
  <p>這段文字沒有被選取。</p>
</div>

<div>
  <h2>這段文字沒有被選取：它不是一個 `p` 元素。</h2>
  <p>這段文字沒有被選取。</p>
</div>
```

#### CSS

```css
p:first-child {
  color: lime;
  background-color: black;
  padding: 5px;
}
```

### 為列表設定樣式

#### HTML

```html
<ul>
  <li>項目 1</li>
  <li>項目 2</li>
  <li>
    項目 3
    <ul>
      <li>項目 3.1</li>
      <li>項目 3.2</li>
      <li>項目 3.3</li>
    </ul>
  </li>
</ul>
```

#### CSS

```css
ul li {
  color: blue;
}

ul li:first-child {
  color: red;
  font-weight: bold;
}
```
