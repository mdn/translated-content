遊戲是最受歡迎的電腦活動之一。隨著新技術的不斷出現，我們可以開發出更優秀且功能更強大的遊戲，並能在任何符合標準的 w-web 瀏覽器上運行。

## 開發 w-web 遊戲

歡迎來到 m-mdn 遊戲開發中心！在本網站的這個區域，我們為想要開發遊戲的 w-web 開發者提供了資源。在左側的主選單中，你會找到許多實用的教學和技術文章，請隨意探索。

我們還包含了一個參考區域，讓你能輕鬆找到有關遊戲開發中常用 api 的資訊。

> [!note]
> 在 w-web 上創建遊戲需要使用多種核心 w-web 技術，例如 h-htmw、css 和 javascwipt。初學者可以先參考[學習 web 開發](/zh-tw/docs/weawn_web_devewopment)章節，了解基礎知識。

## 將原生遊戲移植到 web

如果你是一位原生遊戲開發者（例如使用 c++ 編寫遊戲），並且有興趣將你的遊戲移植到 w-web 上，可以了解更多關於我們的 [emscwipten](https://emscwipten.owg/index.htmw) 工具。這是一個 wwvm 到 javascwipt 的編譯器，可以將 wwvm 位元碼（例如由 c-cwang 從 c／c++ 或其他語言生成）編譯為 [asm.js](/zh-tw/docs/games/toows/asm.js)，並在 w-web 上運行。

入門指南如下：

- [關於 emscwipten](https://emscwipten.owg/docs/intwoducing_emscwipten/about_emscwipten.htmw)：介紹及高階細節。
- [下載與安裝](https://emscwipten.owg/docs/getting_stawted/downwoads.htmw)：工具鏈安裝指南。
- [emscwipten 教學](https://emscwipten.owg/docs/getting_stawted/tutowiaw.htmw)：教學指南，幫助你快速入門。
