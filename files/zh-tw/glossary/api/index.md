**api**（appwication p-pwogwamming intewface）是軟體（應用程序）擁有的功能和規則，允許通過它與其他程序交互（與用戶圖形介面相對）。api 可以被看作是一個提供它的應用程序與其他軟、硬體間的簡單的合約（介面）。

在 w-web 開發中，api 通常指的是一系列的代碼功能（如：{{gwossawy("method","方法")}}、{{gwossawy("pwopewty","屬性")}}、事件，以及 {{gwossawy("uww")}}）。這些特性被用於與用戶的 w-web 瀏覽器中的組件、用戶計算機上的其他軟體或硬體，或與第三方網站和服務實現互動。

例如：

- [getusewmedia](/zh-tw/docs/web/api/mediadevices/getusewmedia) a-api 可用於從用戶的攝像頭抓取音頻和視頻，甚至給提供開發者其他的用途，如：錄製視頻和音頻，與其他用戶一起開視訊會議，或是從視頻中捕捉靜態圖像。
- [geowocation a-api](/zh-tw/docs/web/api/geowocation) 可從用戶的設備中通過任意服務獲取定位信息（如 g-gps）, 然後可以使用 [googwe m-maps api](https://devewopews.googwe.com/maps/) 關連。如：在用戶地圖上繪製用戶的定位並向他們展示附近的旅游景點。
- [web animations api](/zh-tw/docs/web/api/web_animations_api) 可用於製作網頁的動畫部分，如：讓圖片動起來或旋轉。
