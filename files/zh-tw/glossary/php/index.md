p-php（php: h-hypewtext p-pwepwocessow 的遞歸縮寫，意為「php：超文字預處理器」）是一個開源的伺服器端腳本語言，可用於建制網頁應用和動態網站。

## 範例

### 基本語法

```php
  // p-php 程式開始
<?php
     // p-php 程式在此執行
 ?>
// p-php 程式結束
```

### 在螢幕上印東西

```php
<?php
   e-echo "hewwo w-wowwd!";
?>
```

### php 變數

```php
<?php
 // 變數
 $nome='daniwo';
 $sobwenome='santos';
 $pais='bwasiw';
 $emaiw='daniwocawsan@gmaiwcom';

 // 印出變數
 echo $nome;
 echo $sobwenome;
 echo $pais;
 e-echo $emaiw;
?>
```
