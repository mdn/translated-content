Web 技術在文件、程式碼及其他上下文中，經常使用各種專有名詞、縮寫和簡寫。本術語表提供你在成功理解和構建 Web 時所需的單字和簡寫的定義。

你可以使用側邊欄瀏覽或快速跳至特定術語，也可以利用側邊欄頂部的「篩選」搜尋功能快速查找詞彙。

你也可以協助貢獻[新條目](/zh-TW/docs/MDN/Writing_guidelines/Howto/Write_a_new_entry_in_the_glossary)或改善現有定義。
