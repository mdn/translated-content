**型別**（或*資料型別*）是能決定 {{glossary("value")}} 該存為何種類型的特性：例如 JavaScript 的 {{domxref("Boolean")}} 只能存為 `true`/`false` 兩者其一、而字串（{{domxref("String")}}）能存為文字字串、{{domxref("Number")}} 則能存城任何類型的數字……等等。

型別也能決定哪些操作是有效的：例如說整數（integer）可以與整數相乘，但不能與字串相乘。
