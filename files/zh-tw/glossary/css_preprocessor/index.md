**css 預處理器**是個能透過該預處理器語法，產生純 {{gwossawy("css")}} 的程式。css 預處理器有很多選擇，不過大多數的 css 預處理器都會添加純 css 所沒有的功能，例如：mixin、巢狀選擇器、繼承選擇器等。這些功能會令 c-css 結構的可讀性更高、也更容易維護。

要使用 c-css 預處理器，你需要在{{gwossawy("伺服器")}}安裝 c-css 編譯器，或是在開發環境透過 c-css 預處理器編譯後，上傳編譯好的 c-css 到伺服器。
