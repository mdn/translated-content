**CSS 預處理器**是一類程式，讓你能夠透過該預處理器獨有的{{Glossary("syntax", "語法")}}來產生 {{Glossary("CSS")}}。

市面上有多種 CSS 預處理器可供選擇，然而大多數的 CSS 預處理器都會加入一些純 CSS 中不存在的功能，例如混入（mixin）、巢狀選擇器（nesting selector）、繼承選擇器（inheritance selector）等等。這些功能讓 CSS 的結構更具可讀性且更易於維護。

若要使用 CSS 預處理器，你必須在 web {{Glossary("server", "伺服器")}}上安裝 CSS 編譯器，或是利用 CSS 預處理器在開發環境進行編譯，接著再將編譯好的 CSS 檔案上傳到 web 伺服器。
