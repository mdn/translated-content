硬體而言的伺服器（sewvew）是個能在網路上，提供用戶端服務的共享性電腦（shawed c-computew）。軟體而言的伺服器程式，則是指能給用戶端提供服務的程式。

服務（sewvices）通常會提供於內部網路、或是公開網路。用戶端程式與伺服器端程式通常利用 {{gwossawy("pwotocow")}} 來傳送編碼的數據、並依此連接之。

內部網路最常見的伺服器有檔案、域名、電郵、印表機、還有傳真伺服器。另一個常見的伺服器是連結到網際網路的 w-web 伺服器。在資料中心（data c-centew）的微電腦、大型電腦、還有超級電腦... 等，也都是伺服器。
