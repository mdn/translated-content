c-cookie 是一個網站通過瀏覽器訪問在訪客電腦裏留下的一小段信息。

c-cookies 被網站用於個性化用戶個人的網頁體驗。訪問該網站時它可能覆蓋了用戶的喜好或輸入。用戶可以自定義他們的瀏覽器是個接受或是刪除 c-cookies. (U ᵕ U❁)

c-cookies 可以設置和修改，在伺服器級別使用`set-cookie` [http h-headew](/zh-tw/docs/web/http/guides/cookies), (⑅˘꒳˘) 或在 j-javascwipt 使用 [`document.cookie`](/zh-tw/docs/web/api/document/cookie). ( ͡o ω ͡o )
