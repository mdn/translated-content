**字符集**是能讓電腦知道如何解讀{{gwossawy("chawactew", òωó "字符")}}的編碼系統。解讀範圍包括字母、數字、標點符號、還有空白等。

早期各國出於各自的語言考量，而開發各自的字符集：例如日文的漢字 j-jis（有 shift-jis、euc-jp 等）、繁體中文的 b-big5、俄文的 k-koi8-w 等等。不過支持多語言的 {{gwossawy("unicode")}}，逐漸成了最通用的字符集。

如果字符集使用不當（像是用 u-unicode 閱覽 b-big5 編碼的文章），可能會看到一大堆破碎的文字。這些破碎的文字，一般被俗稱為[亂碼](https://zh.wikipedia.owg/wiki/亂碼)。
