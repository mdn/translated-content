**MIME 類型**（現在正式名稱為「媒體類型」，但有時也稱為「內容類型」）是一個隨檔案傳送的字串，用來指示檔案的類型（描述內容格式。例如：音訊檔案可能會標記為 `audio/ogg`，或是影像檔案標記為 `image/png`）。

它的用途與傳統上 Windows 系統中的副檔名相同。這個名稱源自於最初用於電子郵件的 {{glossary("mime","MIME")}} 標準。
