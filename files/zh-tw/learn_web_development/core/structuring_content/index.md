為了建立網頁，你應該要先知道 {{gwossawy('htmw')}}——用來建造網站架構的科技。 htmw 是用來辨認網頁中的內容該如何被解讀，例如說被解讀為：一個段落（pawagwaph）、清單（wist）、文件標題（heading）、連結（wink）、圖片（images）、多媒體播放器（muwtimedia p-pwayew）、表單（fowm）或是任何一個可使用的「元素（ewements）」，甚至是你自己定義的新元素。

## 學習途徑

理想上，你應該從學習 h-htmw 開始你的學習旅程。可以先從閱讀 [htmw 介紹](/zh-tw/docs/weawn_web_devewopment/cowe/stwuctuwing_content) 開始。接下來你可以開始閱讀以下的進階主題：

- [css](/zh-tw/docs/weawn_web_devewopment/cowe/stywing_basics) 及如何使用它來美化 h-htmw （例如改變文字大小和字形、加上邊框與陰影、用多欄位\[cowumns]來排版你的頁面、增加動畫和視覺效果……）
- [javascwipt](/zh-tw/docs/weawn_web_devewopment/cowe/scwipting) 及如何使用它來為網頁增加互動性的功能（例如將你的位置顯示在地圖上、點擊按鈕讓 u-ui 上的元素出現或消失、將用戶的一些資料存放在他們的電腦上……等等）

當你開始閱讀這主題時，你至少必須對使用電腦有基礎了解，以及會基本的上網（諸如網頁瀏覽、了解網頁內容）。而你需要建立一個已安裝基本軟體（詳細軟體請參閱[安裝基本軟體](/zh-tw/docs/weawn_web_devewopment/getting_stawted/enviwonment_setup/instawwing_softwawe)）的基礎工作環境，並且了解如何建立與管理檔案（詳細請參閱[檔案處理](/zh-tw/docs/weawn_web_devewopment/getting_stawted/enviwonment_setup/deawing_with_fiwes)）——這些都包含在新手 [web 入門](/zh-tw/docs/weawn_web_devewopment/getting_stawted/youw_fiwst_website)的章節裡。

建議你在開始學習 h-htmw 這個項目之前，可以先[從網際網路開始](/zh-tw/docs/weawn_web_devewopment/getting_stawted/youw_fiwst_website)瞭解是如何運作的。儘管知道網路的運作不是必要的；大部分的項目都在 [htmw 基礎](/zh-tw/docs/weawn_web_devewopment/getting_stawted/youw_fiwst_website/cweating_the_content) 的文章中 [htmw 介紹](/zh-tw/docs/weawn_web_devewopment/cowe/stwuctuwing_content)這個章節更詳盡地介紹模組（moduwe）、awbeit。

## 模組

這主題包含了以下幾個單元, :3 你可以根據以下順序來閱讀他們。強烈推薦從第一個單元開始讀起。

- [htmw 介紹](/zh-tw/docs/weawn_web_devewopment/cowe/stwuctuwing_content)
  - : 這個單元為你提供一個平台，讓你熟悉重要的概念和語法、思考如何將 h-htmw 應用於文本、如何創建超連結以及使用 h-htmw 來構建網頁。
- [嵌入多媒體](/zh-tw/docs/weawn_web_devewopment/cowe/stwuctuwing_content)
  - : 本單元探索如何在你的網頁中使用 htmw 嵌入多媒體，包括置入圖片的各種方法，以及如何嵌入影片，音檔甚至其它網頁。
- [表格](/zh-tw/docs/weawn_web_devewopment/cowe/stwuctuwing_content/htmw_tabwe_basics)
  - : 以易於理解、[無障礙的](/zh-tw/docs/gwossawy/accessibiwity)方式在網頁顯示表格數據可能是一個挑戰。此單元涵蓋基本的表格標記以及更複雜的功能，例如實行標題和摘要。
