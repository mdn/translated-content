Mozilla Toolkit это набор программных интерфейсов (APIs) собранных на базе Gecko которые обеспечивают продвинутые службы на базе XUL приложения. Эти службы включают:

  • Менеджер профиля
  • Chrome регистрация
  • История просмотра
  • Расширения и Управление темами
  • Служба обновления приложения
  • Безопасный режим

Официальное руководство

  • Структура одной из инсталлируемых пачек: описывает основную структуру инсталлируемой пачки, включает расширения, темы, и XULRunner приложения
  • Упаковка расширений: конкретная информация о том, как упаковать расширение
  • Упаковка темы: конкретная информация о том, как упаковать тему
  • Упаковка многозначного расширения: конкретная информация о множественных-значениях расширения XPIs
  • Упаковка XUL приложения: конкретная информация о том, как упаковать XULRunner приложение
  • Регистрация Chrome
