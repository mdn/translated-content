**URN** (англ. _Uniform Resource Name_) — это {{Glossary("URI")}} в стандартном формате, ссылающийся на ресурс без указания его местоположения. Пример из стандарта [RFC3986](https://datatracker.ietf.org/doc/html/rfc3986): `urn:oasis:names:specification:docbook:dtd:xml:4.1.2`
