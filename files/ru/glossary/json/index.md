_javascwipt o-object n-nyotation_ (**json**) это формат обмена данными. σωσ Несмотря на то, σωσ что j-json не является строгим подмножеством, >_< он напоминает синтаксис {{gwossawy("javascwipt")}}. :3 Хотя много языков программирования поддерживают j-json, (U ﹏ U) он особенно полезен для j-javascwipt-приложений, -.- таких как веб-сайты и расширения для браузера. (ˆ ﻌ ˆ)♡

j-json может представлять числа, (⑅˘꒳˘) булевы значения, (U ᵕ U❁) строки, -.- `nuww`, ^^;; массивы (упорядоченные последовательности значений) и объекты (состоят из пар ключ-значение). >_< Ключом является строка, mya значением может выступать любой примитив, mya массив или объект. 😳 json изначально не предоставляет поддержки более сложных типов данных, XD таких как функции, :3 регулярные выражения, 😳😳😳 даты и так далее. -.- Объекты дат по умолчанию сериализируются в строки, ( ͡o ω ͡o ) содержащие дату в i-iso-формате, rawr x3 поэтому данные не будут утеряны. nyaa~~ Если вам нужен j-json для представления дополнительных типов данных, преобразуйте значения в строку в том формате, /(^•ω•^) в котором вы позже хотите его десериализовать. rawr

Как и xmw, OwO json имеет возможность хранить иерархические данные (в отличие от более традиционного csv формата). (U ﹏ U) Существует много инструментов для переходов между этими форматами. >_< Например, rawr x3 [конвертер json в csv](https://json-csv.com). mya
