**Динамический язык** — язык программирования, который позволяет определять типы данных и осуществлять синтаксический анализ и компиляцию «на лету», на этапе выполнения программы. Динамические языки удобны для быстрой разработки приложений.<span id="result_box" lang="ru"><span> Например, в</span> <span>JavaScript</span> <span>можно</span> <span>изменить</span> <span>тип переменной</span>, <span>или добавить</span> <span>новые свойства, или методы</span> <span>к объекту</span> <span>в то время,</span> <span>как программа работает</span><span>.</span></span>

<span id="result_box" lang="ru"><span>Это свойство отличает динамические языки от</span> <span>статических</span> <span>языков программирования,</span> <span>в</span> <span>которых</span> <span>подобные изменения обычно</span> невозможны.</span>
