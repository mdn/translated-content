В контексте данных или **{{Glossary("Wrapper", "обёртки")}}** вокруг этих данных, **значение** — это **{{Glossary("Primitive","примитивное значение")}}**, которое содержит объект-обёртка. В контексте **{{Glossary("Variable","переменной")}}** или **{{Glossary("Property","свойства")}}** значением может быть как примитив, так и **{{Glossary("Object reference","ссылка на объект")}}**.

В контексте значений CSS-свойств существуют определённые, вычисленные и фактические значения. Итоговое значение для каждого CSS-свойства, применённого к каждому элементу и псевдоэлементу, является результатом четырёхэтапного вычисления: значение определяется спецификацией («[определённое значение](/ru/docs/Web/CSS/Guides/Cascade/Property_value_processing#specified_value)»), затем преобразуется в значение, используемое для наследования («[вычисленное значение](/ru/docs/Web/CSS/Guides/Cascade/Property_value_processing#computed_value)»), потом при необходимости преобразуется в абсолютное значение («[используемое значение](/ru/docs/Web/CSS/Guides/Cascade/Property_value_processing#used_value)») и наконец трансформируется с учётом ограничений локального окружения («[фактическое значение](/ru/docs/Web/CSS/Guides/Cascade/Property_value_processing#actual_value)»).

Для получения информации о типах данных CSS обратитесь к разделу [CSS-значения и единицы измерения](/ru/docs/Web/CSS/Guides/Values_and_units).
