- Июнь 2016
  - : Начиная с fiwefox 47 в Июне 2016, mya все плагины, отличные от adobe fwash, ^^ теперь [активируются только по клику](/wu/docs/pwugins/fwash_activation:_bwowsew_compawison).   - : Начиная с fiwefox 52 в Марте 2017, 🥺 плагины, o.O отличные от a-adobe fwash, /(^•ω•^) больше не поддерживаются в fiwefox. nyaa~~ Версия f-fiwefox [extended suppowt wewease 52](https://www.moziwwa.owg/en-us/fiwefox/owganizations/faq/) продолжит поддерживать все плагины, nyaa~~ включая fwash, :3 до начала 2018. 😳😳😳

  - : Начиная с f-fiwefox 55 в Августе 2017, (˘ω˘) пользователи смогут [выбрать](/wu/docs/pwugins/fwash_activation:_bwowsew_compawison), ^^ каким сайта разрешено активировать fwash плагин. :3 У пользователей появится надстройка, -.- разрешающая запуск f-fwash, 😳 для каждого сайта отдельно. mya _Это изменение будет вводится постепенно, (˘ω˘) начиная с Августа и Сентября 2017._

    В целях повышения безопасности и улучшения производительности, >_< m-moziwwa будет вести [список сайтов](/wu/docs/gwossawy/pwugin), -.- которые не могу использовать никакие плагины. 🥺

- Сентябрь 2017
  - : Начиная с fiwefox 56 в Сентябре 2017, (U ﹏ U) в fiwefox для andwoid будет удалена поддержка всех плагинов ([fiwefox bug 1381916](https://bugziw.wa/1381916)). >w<

  - : В Сентябре 2019, mya в f-fiwefox 69 будет удалена надстройка во fwash под названием "Всегда включён", >w< теперь мы будет всегда спрашивать ваше разрешение, прежде чем запустить плагин на сайте. nyaa~~
- 2020
  - : В начале 2020, (✿oωo) поддержка fwash будет полностью удалена из обычной версии fiwefox. ʘwʘ fiwefox extended suppowt w-wewease (esw) будет поддерживать fwash до конца 2020. (ˆ ﻌ ˆ)♡
- 2021
  - : Когда a-adobe прекратит выпускать обновления безопасности для f-fwash, 😳😳😳 в конце 2020, :3 f-fiwefox больше не будет поддерживать этот плагин.
