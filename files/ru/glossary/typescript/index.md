**TypeScript** — это язык программирования, который добавляет в JavaScript {{glossary("static_typing", "статическую проверку типов")}}.

TypeScript является надмножеством JavaScript. Это означает: всё, что доступно в JavaScript, доступно и в TypeScript, а любая программа на JavaScript синтаксически является корректной программой на TypeScript. Кроме того, поведение TypeScript и JavaScript во время выполнения идентично.

Однако TypeScript добавляет проверку типов на этапе компиляции, реализуя правила о том, как различные типы могут использоваться и комбинироваться. Это позволяет выявить широкий спектр ошибок программирования, которые в JavaScript обнаруживаются только во время выполнения.

Некоторые правила типизации выводятся автоматически из кода JavaScript. Например, в приведённом ниже фрагменте TypeScript выводит, что `myVariable` имеет тип `string`, и не позволяет присвоить этой переменной значение другого типа:

```js
let myVariable = "Hello World";
myVariable = 1;
// Ошибка:
// Тип 'number' не может быть присвоен типу 'string'.
```

TypeScript также позволяет программисту добавлять аннотации типов в код, чтобы явно указывать, например, типы параметров функции или свойства объекта:

```ts
function add(left: number, right: number): number {
  return left + right;
}

add("hello", "world");
// Ошибка:
// Аргумент типа 'string' не может быть присвоен параметру типа 'number'.
```

После компиляции аннотации типов удаляются, и результирующий код становится обычным JavaScript, что позволяет выполнять его в любой среде выполнения JavaScript.
