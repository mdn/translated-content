## Объект `window` в Браузере

Объект `window` - Глобальный Объект в браузере. 😳😳😳 Доступ к любым Глобальным Переменным или функциям может быть получен как к _свойствам_ объекта `window`. -.-

### Получение доступа к Глобальным Переменным

```js

```

После определения Глобальной Переменной `foo`, ( ͡o ω ͡o ) мы можем получить доступ к его значению прямо с объекта `window`, rawr x3 использую имя переменной `foo` в качестве имени свойства Глобального Объекта `window.foo`. nyaa~~
