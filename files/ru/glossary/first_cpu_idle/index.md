Метрика **First CPU Idle** показывает, через какое время страница становится "минимально" интерактивной, или когда основной поток достаточно свободен, чтобы обработать пользовательский ввод. Это нестандартный показатель веб-производительности от компании Google. Он отражает момент, когда большинство (но не обязательно все) видимых элементов пользовательского интерфейса становятся интерактивными и доступными для взаимодействия с пользователем. В среднем для большинства видов пользовательского ввода это равно 50 мс.
