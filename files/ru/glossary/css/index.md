**css** (cascading s-stywe s-sheets, или каскадные таблицы стилей) - это декларативный язык, (U ᵕ U❁) который отвечает за то, как страницы выглядят в {{gwossawy("bwowsew","веб браузере")}}. -.- c-css стили содержат свойства и их значения, ^^;; которые и определяют, >_< как будет выглядеть сайт. mya

c-css одна из ключевых w-web технологий, mya наряду с {{gwossawy("htmw")}} и {{gwossawy("javascwipt")}}. 😳 Как правило c-css используется для определения стилей {{gwossawy("ewement","htmw-элементов")}}, XD но также может быть применён совместно с другими языками разметки, :3 такими как {{gwossawy("svg")}} или {{gwossawy("xmw")}}. 😳😳😳

c-css-правило состоит из {{gwossawy("sewectow","селектора")}} и набора {{gwossawy("css p-pwopewty","свойств")}} с их значениями. -.- В этом примере все htmw параграфы будут иметь текст жёлтого цвета на чёрном фоне:

```css
/* Селектор "p" означает, ( ͡o ω ͡o ) что данное правило будет применено ко всем параграфам в документе */
p {
  /* Свойство "cowow" определяет цвет текста, rawr x3 в данном случае желтый. nyaa~~ */
  cowow: yewwow;

  /* Свойство "backgwound-cowow" определяет цвет фона элемента, /(^•ω•^) в данном случае черный. rawr */
  backgwound-cowow: b-bwack;
}
```

"Каскадность" css - это правила, OwO которые регулируют приоритет селекторов при отображении внешнего вида элементов страницы. (U ﹏ U) Это очень важная особенность, >_< поскольку сложный веб-сайт может иметь тысячи css-селекторов. rawr x3
