---
title: SDK (Software Development Kit)
slug: Glossary/SDK
---

**SDK** (**комплект средств разработки**, от англ. _Software Development Kit_) — это интегрированный набор инструментов, которые разработчик может использовать для создания программного обеспечения под конкретную платформу, операционную систему или фреймворк. SDK может включать:

- редактор;
- компилятор;
- отладчик;
- эмулятор или симулятор, если целевая платформа отличается от платформы, используемой для создания программы;
- инструменты для тестирования и упаковки программы для распространения.

SDK обычно предоставляются владельцем программной платформы для поддержки разработчиков, создающих приложения под эту платформу. Например, Google предоставляет [Android SDK](https://developer.android.com/studio) для разработчиков, пишущих приложения под Android.

Во многих отношениях {{Glossary("developer tools", "инструменты разработчика")}}, встроенные в современные веб-браузеры, выполняют аналогичную функцию для веб-разработчиков.
