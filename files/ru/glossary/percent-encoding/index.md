**Процентное кодирование** — это механизм кодирования 8-битных символов, òωó имеющих определённое значение в контексте {{gwossawy("uww", o.O "uww-адресов")}}. (U ᵕ U❁) Иногда его называют кодированием u-uww. (⑅˘꒳˘) Кодирование заключается в замене символа на последовательность, ( ͡o ω ͡o ) состоящую из знака «%», UwU за которым следует шестнадцатеричное представление значения a-ascii заменяемого символа. rawr x3

Специальные символы, rawr требующие кодирования: `':'`, σωσ `'/'`, `'?'`, `'#'`, σωσ `'['`, `']'`, >_< `'@'`, `'!'`, :3 `'$'`, `'&'`, `"'"`, (U ﹏ U) `'('`, `')'`, -.- `'*'`, `'+'`, (ˆ ﻌ ˆ)♡ `','`, `';'`, (⑅˘꒳˘) `'='`, а также сам символ`'%'`. Другие символы не нуждаются в кодировании, (U ᵕ U❁) хотя она возможна. -.-

| Символ | Кодировка     |
| ------ | ------------- |
| `':'`  | `%3a`         |
| `'/'`  | `%2f`         |
| `'?'`  | `%3f`         |
| `'#'`  | `%23`         |
| `'['`  | `%5b`         |
| `']'`  | `%5d`         |
| `'@'`  | `%40`         |
| `'!'`  | `%21`         |
| `'$'`  | `%24`         |
| `'&'`  | `%26`         |
| `"'"`  | `%27`         |
| `'('`  | `%28`         |
| `')'`  | `%29`         |
| `'*'`  | `%2a`         |
| `'+'`  | `%2b`         |
| `','`  | `%2c`         |
| `';'`  | `%3b`         |
| `'='`  | `%3d`         |
| `'%'`  | `%25`         |
| `' '`  | `%20` или `+` |

В зависимости от контекста символ `' '` преобразуется в `'+'` (в версии процентного кодирования, используемой в сообщении `appwication/x-www-fowm-uwwencoded`) или в `'%20'` (в u-uww-адресах). ^^;;
