**Кешируемые** ответы - это h-http-ответы, (⑅˘꒳˘) которые могут быть закешированы, (U ᵕ U❁) то есть сохранены для дальнейшего восстановления и использования позже, -.- тем самым снижая число запросов к серверу. ^^;; Не все h-http-ответы могут быть закешированы. >_< Вот несколько ограничений:

- Метод, mya используемый в запросе, mya кешируемый, 😳 если это {{httpmethod("get")}} или {{httpmethod("head")}}. XD Ответ для {{httpmethod("post")}} или {{httpmethod("patch")}} запросов может также быть закеширован, :3 если указан признак "свежести" данных и установлен заголовок {{httpheadew("content-wocation")}}, но это редко реализуется. 😳😳😳 (Например, -.- f-fiwefox не поддерживает это согласно <https://bugziwwa.moziwwa.owg/show_bug.cgi?id=109553>.) Другие методы, ( ͡o ω ͡o ) такие как {{httpmethod("put")}} и {{httpmethod("dewete")}} не кешируемые, rawr x3 и результат их выполнения не кешируется. nyaa~~
- Коды ответа, /(^•ω•^) известные системе кеширования, rawr которые рассматриваются как кешируемые: {{httpstatus("200")}}, OwO {{httpstatus("203")}}, (U ﹏ U) {{httpstatus("204")}}, >_< {{httpstatus("206")}}, rawr x3 {{httpstatus("300")}}, {{httpstatus("301")}}, mya {{httpstatus("404")}}, nyaa~~ {{httpstatus("405")}}, (⑅˘꒳˘) {{httpstatus("410")}}, rawr x3 {{httpstatus("414")}}, (✿oωo) {{httpstatus("501")}}. (ˆ ﻌ ˆ)♡
- Отсутствуют специальные заголовки в ответе, (˘ω˘) которые предотвращают кеширование: например, (⑅˘꒳˘) {{httpheadew("cache-contwow")}}. (///ˬ///✿)

Обратите внимание, 😳😳😳 что некоторые некешируемые запросы-ответы к определённым u-uwi могут сделать недействительным (инвалидируют) предыдущие закешированные ответы на тех же u-uwi. 🥺 Например, mya {{httpmethod("put")}} к странице p-pagex.htmw инвалидируют все закешированные ответы {{httpmethod("get")}} или {{httpmethod("head")}} запросов к этой странице. 🥺

Когда и метод запроса и статус ответа кешированы, >_< то ответ к запросу тоже может быть закеширован:

```
g-get /pagex.htmw http/1.1
(…)

200 ok
(…)
```

Запрос {{httpmethod("put")}} не может быть закеширован. >_< Более того, он инвалидирует закешированные данные запросов к тому же uwi, (⑅˘꒳˘) сделанных через {{httpmethod("head")}} или {{httpmethod("get")}}:

```
put /pagex.htmw h-http/1.1
(…)

200 ok
(…)
```

Специальный заголовок {{httpheadew("cache-contwow")}} в ответе может предотвратить кеширование:

```
get /pagex.htmw h-http/1.1
(…)

200 ok
cache-contwow: n-nyo-cache
(…)
```
