**Соотношение сторон** — это пропорциональное отношение между шириной и высотой элемента или {{glossary("viewport", "области просмотра")}}. Оно записывается как {{cssxref("ratio", "соотношение")}} двух чисел.

Наличие соотношения сторон — будь то собственное (внутреннее) соотношение, как у изображений и видео, или заданное извне — сохраняет задуманные пропорции элемента. Вы также можете запросить соотношение сторон элемента или области просмотра, что полезно при разработке гибких компонентов и макетов.

В CSS тип данных {{cssxref("ratio")}} записывается как `ширина / высота` (например, `1 / 1` для квадрата, `16 / 9` для широкоэкранного формата) либо одним числом — в этом случае число обозначает ширину, а высота принимается равной `1`.

```css
.wideBox {
  aspect-ratio: 5 / 2;
}
.tallBox {
  aspect-ratio: 0.25;
}
```

В SVG соотношение сторон определяется атрибутом [`viewBox`](/ru/docs/Web/SVG/Reference/Attribute/viewBox) с четырьмя значениями. Первые два значения задают минимальные координаты X и Y начала координат SVG, а вторые два — ширину и высоту, которые и устанавливают соотношение сторон SVG.

```svg
<svg viewBox="0 0 300 100" xmlns="http://www.w3.org/2000/svg"></svg>
```

В JavaScript API запрос соотношения сторон возвращает число с плавающей запятой двойной точности, представляющее ширину, делённую на высоту. Также с помощью JavaScript можно задать соотношение сторон элемента. Например, ограничение соотношения сторон для видео 1920×1080, заданное через свойство [`aspectRatio`](/ru/docs/Web/API/MediaTrackSettings/aspectRatio) словарей {{domxref("MediaStreamTrack")}} или {{domxref("MediaTrackSettings")}}, будет вычислено как 16/9, или 1920/1080, то есть `1.7777777778`:

```js
const constraints = {
  width: 1920,
  height: 1080,
  aspectRatio: 1.777777778,
};

myTrack.applyConstraints(constraints);
```
