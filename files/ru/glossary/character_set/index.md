**Набор символов** — это система кодирования, которая позволяет компьютеру распознавать {{Glossary("Character", "символы")}}, включая буквы, цифры, знаки препинания и пробелы.

В прошлом страны разрабатывали собственные наборы символов из-за различий в языках, например, кодировки JIS для иероглифов кандзи (такие как Shift-JIS, EUC-JP и др.) для японского языка, Big5 для традиционного китайского и KOI8-R для русского. Однако со временем {{Glossary("Unicode")}} стал наиболее распространённым набором символов благодаря поддержке всех языков.

Если набор символов используется неправильно (например, кодировка Unicode для статьи, сохранённой в Big5), вы можете увидеть вместо текста нечитаемые символы, что называется [Модзибакэ](https://ru.wikipedia.org/wiki/Модзибакэ).
