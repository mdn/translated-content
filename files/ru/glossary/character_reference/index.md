**Символьная ссылка** (character reference) в {{glossary("HTML")}} — это {{glossary("escape character", "escape-последовательность")}} из {{glossary("character", "символов")}}, используемая для представления другого символа на отображаемой веб-странице.

Символьные ссылки применяются как замена символам, зарезервированным в HTML, например, знакам «меньше» (`<`) и «больше» (`>`), которые используются парсером HTML для определения {{Glossary('tag','тегов')}} элементов, или кавычкам `"` и `'` внутри атрибутов, которые могут быть ограничены этими же символами.
Также они используются для невидимых символов, которые невозможно ввести напрямую: неразрывных пробелов, управляющих символов (например, меток направления письма слева направо или справа налево), а также для символов, которые трудно набрать на стандартной клавиатуре.

Существует три типа символьных ссылок:

- **Именованные символьные ссылки**
  - : Используют строку имени между амперсандом (`&`) и точкой с запятой (`;`) для указания на соответствующий символ. Например, `&lt;` для знака «меньше» (`<`), а `&copy;` для символа авторского права (`©`). Строка ссылки часто представляет собой сокращение или инициализацию имени символа в стиле {{glossary("Camel case","camelCase")}}.

- **Десятичные числовые символьные ссылки**
  - : Начинаются с `&#`, затем следуют одна или несколько ASCII-цифр, представляющих десятичное целое число, соответствующее {{glossary("code point", "кодовой позиции")}} символа в {{glossary("Unicode")}}, и завершаются `;`. Например, десятичная ссылка для `<` — `&#60;`, так как кодовая позиция символа в Юникоде — `U+0003C`, а `3C` в шестнадцатеричной системе равно 60 в десятичной.

- **Шестнадцатеричные числовые символьные ссылки**
  - : Начинаются с `&#x` или `&#X`, затем следуют одна или несколько шестнадцатеричных ASCII-цифр, представляющих шестнадцатеричное целое число, соответствующее кодовой позиции символа в Юникоде, и завершаются `;`. Например, шестнадцатеричная ссылка для `<` — `&#x3C;` или `&#X3C;`, так как кодовая позиция символа — `U+0003C`.

Ниже приведён небольшой набор полезных именованных символьных ссылок с соответствующими кодовыми позициями Юникода.

| Символ | Именованная ссылка | Кодовая позиция Юникода |
| ------ | ------------------ | ----------------------- |
| &      | `&amp;`            | U+00026                 |
| <      | `&lt;`             | U+0003C                 |
| >      | `&gt;`             | U+0003E                 |
| "      | `&quot;`           | U+00022                 |
| '      | `&apos;`           | U+00027                 |
|        | `&nbsp;`           | U+000A0                 |
| –      | `&ndash;`          | U+02013                 |
| —      | `&mdash;`          | U+02014                 |
| ©      | `&copy;`           | U+000A9                 |
| ®      | `&reg;`            | U+000AE                 |
| ™      | `&trade;`          | U+02122                 |
| ≈      | `&asymp;`          | U+02248                 |
| ≠      | `&ne;`             | U+02260                 |
| £      | `&pound;`          | U+000A3                 |
| €      | `&euro;`           | U+020AC                 |
| °      | `&deg;`            | U+000B0                 |

Полный список именованных символьных ссылок HTML можно найти в [спецификации HTML](https://html.spec.whatwg.org/multipage/named-characters.html#named-character-references).
