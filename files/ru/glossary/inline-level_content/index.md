В CSS содержимое встроенной раскладки называется **содержимым встроенного уровня**. Большинство текстовых последовательностей, замещаемых элементов и сгенерированного содержимого по умолчанию является встроенным.

Во встроенной раскладке смешанный поток из текста, [замещаемых элементов](/ru/docs/Web/CSS/CSS_images/Replaced_element_properties) и других встроенных контейнеров представляется в виде набора строчных контейнеров. Внутри каждого строчного контейнера встроенные контейнеры выравниваются относительно друг друга по вертикали или по горизонтали, в зависимости от применяемого режима письма (writing mode). Обычно они выравниваются по своей базовой линии текста. Это можно настроить с помощью CSS.

![inline layout](inline_layout.png)

> [!NOTE]
> HTML-элементы исторически делятся на "блочные" и "встроенные". Поскольку это характеристика представления, то теперь это определяется в CSS.

## Примеры

```html
<p>
  Этот span является
  <span class="highlight">элементом встроенного уровня</span>. Его фон
  подсвечен, чтобы показать начало и конец действия элемента. Элементы ввода,
  такие как <input type="radio" /> и <input type="checkbox" />, тоже являются
  содержимым встроенного уровня.
</p>
```

В этот примере элемент {{HTMLElement("p")}} содержит текст. В этом тексте есть элемент {{HTMLElement("span")}} и два элемента {{HTMLElement("input")}}, все они являются элементами встроенного уровня. Если `<span>` занимает две строки, то создаются два строчных контейнера. Поскольку эти элементы встроенные, абзац отображается корректно, без разделения на отдельные строки:

```css hidden
body {
  margin: 0;
  padding: 4px;
  border: 1px solid #333;
}

.highlight {
  background-color: #ee3;
}
```
