Спецификация [Server Timing](https://w3c.github.io/server-timing/) позволяет серверу передавать агенту пользователя метрики производительности, связанные с циклом «запрос-ответ», и использует интерфейс JavaScript, чтобы приложения могли собирать, обрабатывать и использовать эти метрики для оптимизации доставки приложения.
