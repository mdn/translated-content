На этой странице представлены руководства по MathML.

- [MathML для начинающих](/ru/docs/Web/MathML/Tutorials/For_beginners)
  - : Это руководство поможет вам освоить создание математических формул с использованием структурной разметки. Оно начинается со введения в добавление MathML в HTML-документ, после чего подробно рассматриваются ключевые компоненты: дроби и корни, элементы с верхними и нижними индексами, а также текстовые контейнеры. Затем руководство охватывает табличную компоновку для матриц и расширенное форматирование математических выражений. В конце вам будет предложено проверить свои знания, воссоздав три известные математические формулы с использованием изученных концепций.
