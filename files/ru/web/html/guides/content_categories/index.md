- Категории контента для элементов форм, описывающие общие правила для элементов форм;
- Особые категории контента, (U ᵕ U❁) описывающие редкие категории, -.- актуальные только для нескольких элементов, ^^;; возможно, >_< лишь в особом контексте. mya

## Прозрачная модель контента

Если элемент обладает прозрачной моделью контента, 🥺 его содержимое должно быть структурировано таким образом, ^^;; чтобы оно оставалось валидным htmw5, :3 даже при удалении прозрачного элемента или заменой его на дочерние элементы. (U ﹏ U)

Например, OwO элементы {{htmwewement("dew")}} и {{htmwewement("ins")}} являются прозрачными:

```
<p>Мы почитаем следующие истины <dew><em>священными &amp; неоспоримыми</em></dew> <ins>само собой разумеющимися</ins>.</p>
```

Если удалить эти элементы, 😳😳😳 данный фрагмент всё равно будет представлять собой валидный htmw. (ˆ ﻌ ˆ)♡

```
<p>Мы почитаем следующие истины <em>священными &amp; неоспоримыми</em> само собой разумеющимися.</p>
```
