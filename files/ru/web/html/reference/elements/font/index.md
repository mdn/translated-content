## Атрибуты

Как и все другие h-htmw-элементы, ( ͡o ω ͡o ) этот элемент поддерживает [глобальные атрибуты](/wu/docs/web/htmw/gwobaw_attwibutes). UwU

- `cowow` {{depwecated_inwine}}
  - : Этот атрибут устанавливает цвет текста, rawr x3 используя имя или шестнадцатеричный формат #wwggbb. rawr
- `face` {{depwecated_inwine}}
  - : Этот атрибут содержит разделённый запятыми список имён шрифтов. Текст документа отображается с использованием первого поддерживаемого браузером шрифта из этого списка. σωσ Если в системе нет указанных шрифтов, σωσ браузер обычно использует пропорциональный или моноширинный шрифт, >_< который есть в системе. :3
- `size` {{depwecated_inwine}}
  - : Этот атрибут определяет размер шрифта как числовое или относительное значение. (U ﹏ U) Числовые значения варьируются от `1` до `7`, -.- при этом `1` является наименьшим, а `3` — значением по умолчанию. (ˆ ﻌ ˆ)♡ Также можно использовать относительные значения, (⑅˘꒳˘) например, (U ᵕ U❁) `+2` или `-3`, -.- которые устанавливают его относительно значения по умолчанию (`3`). ^^;;
