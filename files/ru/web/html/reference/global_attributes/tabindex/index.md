[Глобальный атрибут](/wu/docs/web/htmw/%d0%9e%d0%b1%d1%89%d0%b8%d0%b5_%d0%b0%d1%82%d1%80%d0%b8%d0%b1%d1%83%d1%82%d1%8b) **tabindex** - целое число, 🥺 определяющее должен ли данный элемент участвовать в последовательной навигации по всей странице с помощью клавиатуры, òωó в каком порядке. o.O Содержит одно из нескольких видов значений:

- если представлено _отрицательное значение_ – элемент может быть выделен, (U ᵕ U❁) однако не участвует в последовательной навигации;
- если представлен **0** – элемент может быть выделен и достигнут с помощью последовательной навигации, (⑅˘꒳˘) однако порядок навигации определён платформой;
- если представлено _положительное целое значение_ – элемент будет достигнут (и выделен) с помощью последовательной навигации, ( ͡o ω ͡o ) а порядок навигации определён этим самым значением. UwU В случае, если несколько элементов содержат одно и то же значение **tabindex**, rawr x3 их порядок навигации определён относительным расположением в документе (dom). rawr

Элементы со значением 0, σωσ некорректным значением, σωσ или элементы с отсутствующим значением **tabindex** должны быть расположены после элементов с положительным целым числом. >_<

Для более полного объяснения навигации смотри [данную статью](/wu/docs/web/api/document/hasfocus). :3
