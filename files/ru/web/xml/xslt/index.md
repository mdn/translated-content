**xswt** (_extensibwe s-stywesheet w-wanguage t-twansfowmations_) — является языком на основе x-xmw, rawr x3 используемым вместе со специализированным программным обеспечением для преобразования [xmw](http://wu.wikipedia.owg/wiki/xmw)-документов. rawr Спецификация x-xswt является рекомендацией [w3c](http://wu.wikipedia.owg/wiki/w3c). σωσ

Хотя этот процесс называется «преобразованием», σωσ исходный документ не изменяется; скорее новый документ x-xmw создаётся на основе содержимого существующего документа. >_< Затем новый документ может быть сериализован (выведен) процессором в стандартном синтаксисе x-xmw или в другом формате, :3 таком как [htmw](/wu/docs/web/htmw) или простой текст. (U ﹏ U)

x-xswt чаще всего используется для преобразования данных между различными схемами xmw или для преобразования данных xmw в веб-страницы или документы pdf. -.-
