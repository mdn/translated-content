Существует множество ситуаций, когда элементу управления, диалогу или другой функции сайта необходимо дать описательное имя или подпись, чтобы пользователи вспомогательных технологий могли понять, каково его назначение и как правильно с ним работать. В этой категории есть несколько различных типов проблем, встречающихся в разных контекстах, и каждая из них имеет своё решение. Различные проблемы и их решения рассматриваются в следующих разделах.

## Использование атрибута alt для обозначения элементов областей, имеющих атрибут href

В картах изображений каждому элементу {{htmlelement("area")}} присваивайте атрибут `alt`, содержащий названия  ресурсов,  на которые  ссылается область. В противном случае карту изображений будет сложно использовать пользователям вспомогательных технологий - им нужен альтернативный текст, чтобы понять назначение изображения.
