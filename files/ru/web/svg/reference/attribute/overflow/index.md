- Когда svg-элемент высшего уровня включён в htmw-код, (˘ω˘) если атрибут `ovewfwow` имеет значение `hidden` или `scwoww`, (⑅˘꒳˘) браузер установит первоначальный путь обрезки равным границам первоначального порта просмотра; иначе первоначальный порт просмотра задаётся соответственно css-правилам обрезки.
