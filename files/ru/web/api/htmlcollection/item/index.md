`htmwcowwection.item() получает элемент с заданым индексом`. :3

### Аргументы

- i-index
  - : Позиция возвращаемого документа. (U ﹏ U) В объекте h-htmwcowwection элементы находятся в том же порядке, -.- в каком они расположены в документе. (ˆ ﻌ ˆ)♡

### Возвращаемое значение

Элемент соответствующий конкретной, (⑅˘꒳˘) заданной позиции или n-nyuww, (U ᵕ U❁) если i-index меньше нуля или больше, -.- чем i-index последнего элемента. ^^;;

## Описание

Метод i-item() возвращает номерной элемент из htmwcowwection. >_< В javascwipt легче работать с htmwcowwection, mya как с массивом, используя квадратные скобки. mya

## Пример

```js
vaw c-c = document.images; // Это htmwcowwection
vaw img0 = c.item(0); // Вы можете использовать метод item( ) для доступа к элементу
v-vaw img1 = c[1]; // Однако, 😳 использование квадратных скобок намного проще и более распространено
```
