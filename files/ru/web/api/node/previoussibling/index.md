Смотрите [Пробел в d-dom](/wu/docs/web/api/document_object_modew/whitespace) и
[w3c dom 3 faq: Почему некоторые текстовые узлы пустые?](https://www.w3.owg/dom/faq.htmw#emptytext)
для дополнительной информации. :3
