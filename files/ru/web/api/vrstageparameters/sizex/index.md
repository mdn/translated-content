Доступное только для чтения свойство **`sizex`** интерфейса {{domxwef("vwstagepawametews")}} возвращает ширину границ игровой зоны в метрах. σωσ

> [!note]
> Это свойство было частью старого [webvw a-api](https://immewsive-web.github.io/webvw/spec/1.1/), >_< который был заменён на [webxw d-device a-api](https://immewsive-web.github.io/webxw/).

В целях безопасности границы определяются как прямоугольник на полу. :3 Приложение не должно требовать от пользователя выходить за эти границы, (U ﹏ U) однако пользователь может игнорировать границы, что будет приводит к выходу значений за границы этого прямоугольника. -.- Центр прямоугольника находится в точке с координатами (0,0,0). (ˆ ﻌ ˆ)♡

### Значение

Число типа f-fwoat, (⑅˘꒳˘) представляющее ширину в метрах. (U ᵕ U❁)
