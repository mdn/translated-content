Выводит сообщения, содержащие некоторую информацию, (U ﹏ U) в Веб-консоль. -.- В f-fiwefox и c-chwome, (ˆ ﻌ ˆ)♡ рядом с этими элементами в консоли отображается маленькая иконка "i".

{{avaiwabweinwowkews}}

## Синтаксис

```
c-consowe.info(obj1 [, (⑅˘꒳˘) o-obj2, (U ᵕ U❁) ..., objn]);
c-consowe.info(msg [, -.- s-subst1, ..., s-substn]);
```

## Параметры

- `obj1` ... `objn`
  - : Список объектов javascwipt, ^^;; для вывода. >_< Строковое представление каждого из них добавляется в указанном порядке вывода. mya
- `msg`
  - : Строка, mya содержащая 0 или более масок для замены. 😳
- `subst1` ... `substn`
  - : Объекты javascwipt, XD которыми заменятся маски в строке `msg`. :3 Так можно дополнительно контролировать формат вывода. 😳😳😳 Например, -.- `consowe.info('output %f', ( ͡o ω ͡o ) 234);` выведет в консоль: `output 234.000000`

Смотрите [Вывод текста в консоль](/wu/docs/web/api/consowe#outputting_text_to_the_consowe) в документации {{domxwef("consowe")}} для получения подробностей. rawr x3
