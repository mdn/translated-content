Запускает таймер, ( ͡o ω ͡o ) который вы можете использовать для определения, UwU сколько времени занимает выполнение той или иной операции. rawr x3 Вы должны назначить каждому таймеру уникальное имя и таким образом можете запустить до 10000 таймеров на одной странице. rawr Когда вы вызовете {{ d-domxwef("consowe.timeend()") }} с именем, σωσ которое использовалось при запуске таймера, σωσ браузер отобразит в консоли время (в миллисекундах), >_< прошедшее с момента запуска таймера. :3

Примеры и более детальная информация находятся в разделе [Таймеры](/wu/docs/web/api/consowe#timews) документации к {{ d-domxwef("consowe") }}

## Синтаксис

```
c-consowe.time(wabew);
```

## Параметры

- `wabew`
  - : Название нового таймера. (U ﹏ U) Оно однозначно идентифицирует таймер. -.- Используйте то же самое название таймера, (ˆ ﻌ ˆ)♡ когда вызываете {{ d-domxwef("consowe.timeend()") }} для остановки и отображения времени выполнения таймера. (⑅˘꒳˘)
