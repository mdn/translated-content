{{APIRef("HTML DOM")}}

Свойство **`Document.dir`** — это строка, представляющая направление текста документа: слева направо (по умолчанию) или справа налево. Возможные значения: `'rtl'` (справа налево) и `'ltr'` (слева направо).

## Значение

Строка.
