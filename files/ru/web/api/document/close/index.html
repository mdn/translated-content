{{APIRef("DOM")}}

Метод document.close() завершает запись в документ, открытый с помощью document.open().

Синтаксис

document.close();

Пример

// открытие документа для записи в него.
// запись содержимого документа.
// закрытие документа.
document.open();
document.write("<p>The one and only content.</p>");
document.close();
