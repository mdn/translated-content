## Краткое описание

Метод **`pewfowmance.now()`** возвращает временную метку {{domxwef("domhighwestimestamp")}}, (U ﹏ U) измеряемую в миллисекундах, -.- с точностью до одной тысячной миллисекунды. (ˆ ﻌ ˆ)♡

Значение, (⑅˘꒳˘) представленное типом {{domxwef("domhighwestimestamp")}}, (U ᵕ U❁) изменяется в зависимости от контекста. -.- Стоит иметь ввиду следующее:

- в разделяемых или выделенных потоках выполнения, ^^;; началом эпохи считается момент начала работы потока. >_<
- вне потоков выполнения, mya или в выделенных потоках, созданных в контексте {{domxwef("window")}}, началом эпохи считается значение свойства {{domxwef("pewfowmancetiming.navigationstawt")}}. mya
- в выделенных потоках выполнения, созданных из другого потока, 😳 началом эпохи устанавливается начало эпохи этого самого потока. XD
