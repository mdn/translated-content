{{APIRef("HTML DOM")}}

Интерфейс `MediaError` представляет собой ошибку, которая возникает при обработке мультимедиа в HTML-элементе на основе {{domxref ("HTMLMediaElement")}}, например {{HTMLElement ("audio")}} или {{HTMLElement ("video")}}.

Объект `MediaError` описывает ошибку в общих чертах, используя числовой код `code`, классифицирующий тип ошибки, и сообщение `message`, которое предоставляет информацию о том, что пошло не так.

## Свойства экземпляра

_Этот интерфейс не наследует никаких свойств._

- {{domxref("MediaError.code")}}
  - : Число, которое представляет [тип произошедшей ошибки](/ru/docs/Web/API/MediaError/code#media_error_code_constants).
- {{domxref("MediaError.message")}}
  - : Понятная человеку строка, которая содержит _конкретную диагностическую информацию_, которая должна помочь причины произошедшей ошибки. Это не краткое изложение того, что означает код ошибки, а фактическая диагностическая информация, помогающая понять, что именно пошло не так. Текст и формат не определены в спецификациях и поэтому отличаются в разных {{Glossary("user agent", "пользовательских агентах")}}. Если диагностическая информация отсутствует, то это значение может быть пустой строкой (`""`).
