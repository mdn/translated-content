Объект `fiweweadew` позволяет веб-приложениям асинхронно читать содержимое файлов (или буферы данных), rawr x3 хранящиеся на компьютере пользователя, nyaa~~ используя объекты {{domxwef("fiwe")}} или {{domxwef("bwob")}}, /(^•ω•^) с помощью которых задаётся файл или данные для чтения. rawr

> **Предупреждение:** `fiweweadew` используется только для безопасным образом чтения содержимого файла из пользовательской (удалённой) системы. nyaa~~ Его нельзя использовать для простого чтения файла по пути из файловой системы. ### Константы состояния

- `empty`: `0` : Данные ещё не были загружены. mya

- `done`: `2` : Операция чтения была завершена.
