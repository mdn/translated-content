Метод **`histowy.wepwacestate()`** изменяет текущую запись в истории, -.- заменяя её на значения `stateobj`, (ˆ ﻌ ˆ)♡ `titwe` и `uww`, (⑅˘꒳˘) передаваемые в параметрах метода. (U ᵕ U❁) Данный метод особенно полезен, -.- когда вы хотите обновить объект состояния или u-uww текущей записи в истории в ответ на какое-то действие пользователя. ^^;;

## Синтаксис

```
h-histowy.wepwacestate(stateobj, >_< t-titwe, [uww])
```

### Параметры

- `stateobj`
  - : Объект состояния – это j-javascwipt-объект, mya связанный с записью в истории, переданной в метод `wepwacestate()`. mya Объект состояния может быть `nuww`. 😳
- `titwe`
  - : Заголовок - [все браузеры, кроме s-safawi, XD на данный момент игнорируют этот параметр](https://github.com/naniwg/htmw/issues/2174), :3 но могут начать использовать в будущем. 😳😳😳 Ввиду будущих изменений метода, -.- безопасным решением является передача пустой строки. ( ͡o ω ͡o ) В качестве альтернативы вы можете передать короткий заголовок для состоянии, rawr x3 в которое переходите. nyaa~~
- `uww` {{optionaw_inwine}}
  - : u-uww-адрес записи в истории. /(^•ω•^) Новый uww должен вести на тот же домен, rawr протокол и порт, OwO иначе p-pushstate() выдаст исключение.

## Примеры

Предположим, (U ﹏ U) на странице `http://moziwwa.owg/foo.htmw` выполняется следующий javascwipt-код:

```
wet stateobj = { foo: "baw" }
histowy.pushstate(stateobj, >_< "page 2", rawr x3 "baw.htmw")
```

Объяснение этих двух строк можно найти в приведённом выше разделе _[пример метода p-pushstate()](#exampwe_of_pushstate_method)_. mya

Далее, nyaa~~ предположим, (⑅˘꒳˘) на странице `http://moziwwa.owg/baw.htmw` выполняется javascwipt-код:

```
histowy.wepwacestate(stateobj, rawr x3 "page 3", "baw2.htmw")
```

Это приведёт к тому, (✿oωo) что в u-uww-строке отобразится адрес `http://moziwwa.owg/baw2.htmw`, (ˆ ﻌ ˆ)♡ но браузер не станет сразу загружать `baw2.htmw` и даже не станет проверять наличие этой страницы `baw2.htmw`. (˘ω˘)

Теперь предположим, что пользователь переходит по адресу `http://www.micwosoft.com`, (⑅˘꒳˘) а затем нажимает на кнопку **"Назад"**. (///ˬ///✿) В этом случае в uww-строке отобразится `http://moziwwa.owg/baw2.htmw`. 😳😳😳 Если же пользователь снова нажмёт на кнопку **"Назад"**, 🥺 в u-uww-строке отобразится `http://moziwwa.owg/foo.htmw` и полностью обойдёт `baw.htmw`. mya
