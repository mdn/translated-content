## Методы

- {{domxwef("cwients.get()")}}
  - : Получает клиента сервис воркера, :3 соответствующего данному `id,` и возвращает его в {{jsxwef("pwomise")}}. (U ﹏ U)

  - : Открывает {{domxwef("cwient")}} сервис воркера в новом окне браузера. >_<

  - : Позволяет активному Сервис Воркеру установить себя, mya как активного воркера для клиентской страницы, mya когда воркер и страница находятся в одной области. 😳
