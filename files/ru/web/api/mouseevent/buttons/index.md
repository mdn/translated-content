Свойство **`mouseevent.buttons`** доступно только для чтения. 🥺 Оно показывает, òωó какие клавиши мыши (или другого устройства ввода) были нажаты при срабатывании события.Для каждой нажатой кнопки есть своё уникальное битовое значение (см. o.O ниже раздел "[Возвращаемые значения](#wetuwn)"). (U ᵕ U❁) Если нажаты несколько кнопок одновременно, (⑅˘꒳˘) значения суммируются, создавая новое значение. ( ͡o ω ͡o ) Например, если нажата вторая кнопка (2 или 000010 в двоичном коде) вместе со средней (4 или 000100 в двоичном коде), значение будет равно 6 (2 + 4) или 000110 в двоичном коде. UwU

> [!note]
> Не путайте это свойство со свойством {{domxwef("mouseevent.button")}}. Текущее свойство {{domxwef("mouseevent.buttons")}} показывает состояние нажатых кнопок при любом событии мыши, rawr x3 а свойство {{domxwef("mouseevent.button")}} указывает на события, rawr произошедшие в результате нажатия/отпускания кнопок мыши. σωσ

## Синтаксис

```
v-vaw buttonpwessed = i-instanceofmouseevent.buttons
```

## Возвращаемые значения

Целое число, σωσ представляющее одну или несколько кнопок. >_< При одновременном нажатии нескольких кнопок значения объединяются (например, :3 3 = первая (1) + вторая (2) кнопки мыши):

- `0`: кнопки не нажаты или не инициализированы;
- `1`: первая кнопка (обычно левая);
- `2`: вторая кнопка (обычно правая);
- `4`: дополнительная кнопка (обычно средняя или колёсико мыши);
- `8`: четвёртая кнопка (как правило, (U ﹏ U) кнопка "Назад" браузера);
- `16` : пятая кнопка (как правило, -.- кнопка "Вперёд" браузера). (ˆ ﻌ ˆ)♡
