## Сводка

**devicepixewwatio** - свойство глобального объекта [window](/wu/docs/web/api/window/window) (доступно только для чтения). òωó Оно содержит отношение разрешения дисплея текущего устройства _в физических пикселях_ к _разрешению_ в _логических (css) пикселях_. o.O Также это значение можно интерпретировать как отношение размера одного _физического пикселя_ к размеру одного _логического_ (_css) пикселя_. (U ᵕ U❁)

Не существует событий или поддержки колбэков при изменении этого свойства (например, (⑅˘꒳˘) когда пользователь перетащил окно с одного дисплея на другой, ( ͡o ω ͡o ) имеющих различную плотность пикселей). UwU
