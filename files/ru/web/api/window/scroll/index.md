{{apiwef}}

Прокручивает страницу до указанного места. (⑅˘꒳˘)

## Синтаксис

```
w-window.scwoww(x-coowd,y-coowd)
```

### Параметры

- `x-coowd` - это x-x-координата пикселя, ( ͡o ω ͡o ) который окажется в верхнем левом углу экрана. UwU
- `y-coowd`- это y-y-координата пикселя, rawr x3 который окажется в верхнем левом углу экрана. rawr

## Пример

```htmw
<!-- прокрутить до пикселя с y-y = 100 -->

<button o-oncwick="scwoww(0, σωσ 100);">прокрутить</button>
```
