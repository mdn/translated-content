{{APIRef}}

Возвращает число фреймов (как {{HTMLElement("frame")}} так и {{HTMLElement("iframe")}} ) в окне.

## Значение

Число.

## Пример

```js
if (window.length) {
  // этот документ содержит фреймы
}
```
