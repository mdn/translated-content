{{apiwef}}Метод **`moveto()`** интерфейса {{domxwef("window")}} перемещает текущее окно в указанные координаты. (U ᵕ U❁)

> [!note]
> Эта функция перемещает окно к абсолютному положению. (⑅˘꒳˘) В свою очередь, {{domxwef("window.moveby()")}} перемещает окно относительно текущего положения. ( ͡o ω ͡o )

## Синтаксис

```
w-window.moveto(x, UwU y-y)
```

### Параметры

- `x` горизонтальная координата, rawr x3 в которую будет произведено перемещение. rawr
- `y` вертикальная координата, σωσ в которую будет произведено перемещение. σωσ

## Пример

Этот пример перемещает окно в левый верхний угол экрана. >_<

```js
f-function o-owigin() {
  w-window.moveto(0, 0);
}
```
