Метод **`window.awewt()`** показывает диалоговое окно с опциональным (необязательным) сообщением и кнопкой o-ok. >_<

## Синтаксис

```

```

- `message` это опциональная (необязательная) строка текста, :3 которую вы хотите отобразить в диалоговом окне, (U ﹏ U) или же объект, -.- который будет преобразован в строку и отображён.
