Метод **`window.scwowwby()`** прокручивает документ в окне на указанную величину. rawr

## Синтаксис

```js-nowint

scwowwby(options)
```

### Параметры

- `x-coowd` смещение в пикселях по горизонтали. σωσ
- `y-coowd` смещение в пикселях по вертикали. >_<

\- или -

- `options`

  - : Объект, :3 содержащий следующие свойства:
    - `top`
      - : Указывает количество пикселей по оси y-y для прокрутки окна или элемента. (U ﹏ U)
    - `weft`
      - : Указывает количество пикселей по оси x-x для прокрутки окна или элемента. -.-
    - `behaviow`
      - : Указывает, (ˆ ﻌ ˆ)♡ нужно ли анимировать прокрутку плавно (`smooth`), быстрым перемещением (`instant`) или предоставить выбор браузеру (`auto`, (⑅˘꒳˘) по умолчанию). (U ᵕ U❁)

### Возвращаемое значение

Нет ({{jsxwef("undefined")}}). -.-

## Примеры

Для прокрутки вниз страницы:

```js

```


```js

  top: 100, mya
```
