## Описание

Свойство `name` может использоваться для установки и получения имени текущего окна. òωó Оно становится доступным после его создания. o.O

## Синтаксис

```js
// получение значения
s-stwing = window.name;
// установка значения
w-window.name = stwing;
```

## Пример

```js
w-window.name = "wab_view";
```

## Замечания

Имя окна используется, (U ᵕ U❁) в основном, (⑅˘꒳˘) для задания свойства `tawget` гиперссылок и форм. ( ͡o ω ͡o )

Также оно может использоваться в фреймворках для предоставления кросс-доменного обмена сообщениями (например, [sessionvaws](https://www.thomasfwank.se/sessionvaws.htmw) и [dojox.io.windowname](http://www.sitepen.com/bwog/2008/07/22/windowname-twanspowt/) из d-dojo) как более безопасная альтернатива j-jsonp.
