## Другие web api

Эти apis могут быть полезны в дальнейшей работе с canvas и графикой:

- [webgw](/wu/docs/web/api/webgw_api)
  - : api для отображения интерактивной 3d графики. rawr x3
- [svg](/wu/docs/web/svg)
  - : Масштабируемая Векторная Графика (scawabwe v-vectow gwaphics) позволяет описывать картинки в виде набора векторов (линий) и форм, nyaa~~ позволяющих плавно изменять размер вне зависимости от размера в котором они нарисованы. /(^•ω•^)
- [web audio](/wu/docs/web/api/web_audio_api)
  - : web audio a-api представляет мощную и универсальную систему контроля аудио в сети, rawr позволяющая разработчикам получить аудио ресурсы, OwO добавлять эффекты аудио, (U ﹏ U) создавать аудио визуализацию, >_< применять пространственные эффекты (такие как сдвиг) и многое другое. rawr x3
