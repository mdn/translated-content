Значение свойства `Navigator.appName` всегда равно «`Netscape`», в любом браузере. Это свойство существует только в целях совместимости.

> [!NOTE]
> Не полагайтесь на это свойство для получения настоящего имени браузера. Все браузеры возвращают одинаковую строку «`Netscape`» в качестве значения этого свойства.

## Значение

Строка «`Netscape`».
