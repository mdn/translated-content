{{APIRef("HTML DOM")}} {{Deprecated_Header}}

Значение свойства `Navigator.product` всегда равно «`Gecko`», в любом браузере. Это свойство существует только в целях совместимости.

> [!NOTE]
> Не полагайтесь на это свойство для получения настоящего имени продукта. Все браузеры возвращают одинаковую строку «`Gecko`» в качестве значения этого свойства.

## Значение

Строка «`Gecko`».
