Значение свойства `Navigator.appCodeName` всегда равно «`Mozilla`», в любом браузере. Это свойство существует только в целях совместимости.

> [!NOTE]
> Не полагайтесь на это свойство для получения настоящего имени продукта. Все браузеры возвращают одинаковую строку «`Mozilla`» в качестве значения этого свойства.

## Значение

Строка «`Mozilla`».
