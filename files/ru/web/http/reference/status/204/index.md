## Примечания совместимости

- Хотя этот статус код предназначен для описания ответа без тела, 😳 серверы могут ошибочно включать данные, следующие за заголовками. Протокол позволяет идентификаторам браузера изменять способ обработки таких ответов ([обсуждение этого текста спецификации можно найти здесь](https://github.com/httpwg/http11bis/issues/26)).
