HTTP код **`103 Early Hints`** в первую очередь предназначен для использования с заголовком {{HTTPHeader("Link")}}, чтобы клиент мог начать предварительную загрузку пока сервер готовит ответ.

## Синтаксис

```
103 Early Hints
```
