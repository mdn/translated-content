{{HTTPSidebar}}

Код состояния ответа "HTTP **400 Bad Request**" указывает, что сервер не смог понять запрос из-за недействительного синтаксиса. Клиент не должен повторять этот запрос без изменений.

## Статус

```
400 Bad Request
```
