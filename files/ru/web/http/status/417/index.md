{{HTTPSidebar}}

Код ошибки HTTP **`417 Expectation Failed`** указывает на то, что ожидание, указанное в {{HTTPHeader("Expect")}}, не может быть выполнено.

Дополнительную информацию смотрите в {{HTTPHeader("Expect")}}.

## Статус

```
417 Expectation Failed
```
