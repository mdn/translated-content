{{HTTPSidebar}}

Статус ответа HTTP **`205 Reset Content`** сообщает клиенту об изменении вида документа, например, для очистки содержимого формы, сброса состояния холста или обновления пользовательского интерфейса.

## Статус

```
205 Reset Content
```
