Элемент {{HTMLElement("img")}} позволяет вставлять изображения в {{Glossary("HTML")}}-документ, а {{HTMLElement("picture")}} отвечает за [адаптивные изображения](/ru/docs/Web/HTML/Guides/Responsive_images).
В этих статьях содержатся рекомендации по добавлению и настройке изображений.

- [Типы и форматы файлов изображений](/ru/docs/Web/Media/Guides/Formats/Image_types)
  - : Руководство по различным типам файлов изображений, которые обычно поддерживаются веб-браузерами, с описанием вариантов использования, возможностей и совместимости. Кроме того, эта статья содержит рекомендации по выбору оптимального типа файла изображения для конкретной ситуации.
- [Использование атрибутов `width` и `height` помогает избежать задержек](/ru/docs/Learn_web_development/Extensions/Performance/Multimedia#rendering_strategy_preventing_jank_when_loading_images)
  - : Вычисление {{glossary("aspect ratio", "соотношения сторон")}} изображения на ранней стадии может использоваться для резервирования необходимого места на странице до её полной загрузки.

## Полезные ссылки

В этих статьях рассматриваются некоторые элементы HTML и свойства CSS, которые используются для управления отображением изображений.

### HTML

- {{HTMLElement("img")}}
  - : **HTML-элемент `<img>`** используется для добавления изображения на веб-страницу. Он поддерживает широкий набор атрибутов, управляющих поведением изображения, и позволяет добавлять важную информацию, например, тектовое описание с помощью атрибута [`alt`](/ru/docs/Web/HTML/Reference/Elements/img#alt), для тех, кто не видит изображение.
- {{HTMLElement("picture")}}
  - : **HTML-элемент `<picture>`** может содержать несколько элементов {{HTMLElement("source")}} и один элемент {{HTMLElement("img")}} и предоставляет версии изображения для различных вариантов отображения. Браузер учитывает каждый дочерний элемент `<source>` и выбирает наиболее подходящий из них.

### CSS

- {{cssxref("object-fit")}}
  - : Свойство `object-fit` определяет, как содержимое {{glossary("replaced elements", "замещаемых элементов")}} (например, `<img>` или `<video>`) масштабируется внутри своего контейнера.
- {{cssxref("object-position")}}
  - : Свойство `object-position` определяет выравнивание содержимого выбранного {{ glossary("replaced elements", "замещаемого элемента")}} внутри его области. Пустые участки заполняются фоном контейнера.
- {{cssxref("background-image")}}
  - : Свойство `background-image` определяет одно или несколько фоновых изображений для элемента.
