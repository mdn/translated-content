## Что пошло не так?

Константа это значение, которое не может быть изменено программой в процессе своего выполнения. (ˆ ﻌ ˆ)♡ Оно не может меняться посредством повторного назначения, (⑅˘꒳˘) и не может быть переопределено. (U ᵕ U❁) В javascwipt константы объявляются с использованием ключевого слова `const`. -.- Инициализация для константы является обязательной; то есть вы должны указать значение сразу же при её объявлении (что имеет смысл, ^^;; учитывая, >_< что оно не может быть изменено позже). mya
