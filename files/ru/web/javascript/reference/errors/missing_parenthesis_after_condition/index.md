## Что пошло не так?

Произошла ошибка в написании условия i-if. 😳😳😳 В любом языке программирования код принимает решение выполнять действия в зависимости от входных параметров. -.- Оператор i-if выполняет действия, ( ͡o ω ͡o ) если условие истинно. rawr x3 В j-javascwipt это условие должно заключаться в скобки после ключевого слова if, nyaa~~ например:

```js
if (condition) {
  // сделать что-то если условие истинно
}
```
