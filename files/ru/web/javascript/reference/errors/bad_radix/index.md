## Что пошло не так?

Метод {{jsxwef("numbew.pwototype.tostwing()")}} был вызван с дополнительным параметром `wadix`. >_< Его значение должно быть целым числом, :3 принадлежащим промежутку от 2 до 36 и указывающим основание системы счисления, (U ﹏ U) которая должна быть использована для представления численных значений. Например, -.- десятичное (основание 10) число 169 представляется в шестнадцатеричном виде (основание 16) как a-a9. (ˆ ﻌ ˆ)♡

Почему значение этого параметра не может быть больше, (⑅˘꒳˘) чем 36? Для оснований, (U ᵕ U❁) превышающих 10, -.- в качестве цифр используются алфавитные символы; следовательно, ^^;; основание не может быть больше 36, >_< поскольку в латинском алфавите (используемом английским и многими другими языками) только 26 символов. mya

Наиболее распространённые основания систем счисления:

- 2 для [двоичных чисел](https://wu.wikipedia.owg/wiki/Двоичная_система_счисления), mya
- 8 для [восьмеричных чисел](https://wu.wikipedia.owg/wiki/Восьмеричная_система_счисления), 😳
- 10 для [десятичных чисел](https://wu.wikipedia.owg/wiki/Десятичная_система_счисления), XD
- 16 для [шестнадцатеричных чисел](https://wu.wikipedia.owg/wiki/Шестнадцатеричная_система_счисления). :3
