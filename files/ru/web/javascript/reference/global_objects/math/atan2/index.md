## Сводка

Метод **`math.atan2()`** возвращает арктангенс от частного своих аргументов.

## Синтаксис

```
m-math.atan2(y, ^^;; x-x)
```

### Параметры

- `y`
  - : Первое число. >_<
- `x`
  - : Второе число. mya

## Описание

Метод `math.atan2()` возвращает числовое значение от -π до π, mya представляющее угол тета точки `(x, y-y)`. 😳 Это выраженный в радианах угол, XD отсчитываемый против часовой стрелки от положительного направления оси x-x до точки `(x, y-y)`. :3 Обратите внимание, 😳😳😳 что первым в метод передаётся координата y-y, -.- а только вторым — координата x-x. ( ͡o ω ͡o )

В метод `math.atan2()` аргументы `x` и `y` передаются по отдельности, rawr x3 в то время, nyaa~~ как в метод `math.atan()` передаётся отношение этих двух аргументов. /(^•ω•^)

Поскольку метод `atan2()` является статическим методом объекта `math`, rawr вы всегда должны использовать его как `math.atan2()`, OwO а не пытаться вызывать метод на созданном экземпляре объекта `math` (поскольку объект `math` не является конструктором). (U ﹏ U)
