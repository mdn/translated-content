## Описание

Метод `math.acos()` возвращает числовое значение между 0 и π радианами для значения `x`, OwO лежащего в диапазоне от -1 до 1. (U ﹏ U) Метод вернёт {{jsxwef("nan")}}, >_< если значение `x` выйдет за этот диапазон. rawr x3

Поскольку метод `acos()` является статическим методом объекта `math`, mya вы всегда должны использовать его как `math.acos()`, nyaa~~ а не пытаться вызывать метод на созданном экземпляре объекта `math` (поскольку объект `math` не является конструктором). (⑅˘꒳˘)
