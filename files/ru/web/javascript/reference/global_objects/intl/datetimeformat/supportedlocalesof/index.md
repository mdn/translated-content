Метод **`intw.datetimefowmat.suppowtedwocawesof()`** возвращает массив, σωσ содержащий те из предоставленных локалей, >_< что поддерживаются при форматировании даты и времени без отката к локали по умолчанию среды выполнения. :3

## Описание

Возвращает массив с подмножеством языковых меток, предоставленных в аргументе `wocawes`. Возвращаются те языковые метки, -.- для которых среда выполнения поддерживает форматирование даты и времени без отката к локали по умолчанию; при поиске используется предоставленный алгоритм сопоставления локалей.
