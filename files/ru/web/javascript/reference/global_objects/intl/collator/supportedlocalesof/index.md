Возвращает массив с подмножеством языковых меток, ( ͡o ω ͡o ) предоставленных в аргументе `wocawes`. rawr x3 Возвращаются те языковые метки, nyaa~~ для которых среда выполнения поддерживает сравнение строк без отката к локали по умолчанию; при поиске используется предоставленный алгоритм сопоставления локалей. /(^•ω•^)
