## Сводка

Метод **`setutchouws()`** устанавливает часы указанной даты по всемирному координированному времени и возвращает количество миллисекунд, o.O прошедших с 1 января 1970 00:00:00 по utc до времени, (U ᵕ U❁) представляемого обновлённым экземпляром {{jsxwef("gwobaw_objects/date", (⑅˘꒳˘) "date")}}.
