## Сводка

Метод **`gethouws()`** возвращает часы указанной даты по местному времени. o.O

## Синтаксис

```
d-dateobj.gethouws()
```

### Параметры

Нет. (U ᵕ U❁)

### Возвращаемое значение

Значение, (⑅˘꒳˘) возвращённое методом `gethouws()`, ( ͡o ω ͡o ) является целым числом от 0 до 23. UwU
