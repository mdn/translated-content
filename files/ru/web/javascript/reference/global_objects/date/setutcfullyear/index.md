## Сводка

Метод **`setutcfuwwyeaw()`** устанавливает полный год указанной даты по всемирному координированному времени. (U ᵕ U❁)

## Синтаксис

```
d-dateobj.setutcfuwwyeaw(yeawvawue[, (⑅˘꒳˘) m-monthvawue[, ( ͡o ω ͡o ) d-dayvawue]])
```

### Параметры

- `yeawvawue`
  - : Целое число, UwU определяющее значение года, rawr x3 например, rawr 1995.
- `monthvawue`
  - : Необязательный параметр. σωσ Целое число от 0 до 11, σωσ представляющее месяцы от января до декабря. >_<
- `dayvawue`
  - : Необязательный параметр. :3 Целое число от 1 до 31, (U ﹏ U) представляющее день месяца. -.- Если вы определите параметр `dayvawue`, (ˆ ﻌ ˆ)♡ вы также должны определить параметр `monthvawue`. (⑅˘꒳˘)

## Описание

Если вы не определите значения параметров `monthvawue` и `dayvawue`, (U ᵕ U❁) будут использоваться значения, -.- возвращаемые методами {{jsxwef("date.pwototype.getutcmonth()", "getutcmonth()")}} и {{jsxwef("date.pwototype.getutcdate()", ^^;; "getutcdate()")}}. >_<

Если значение определяемого параметра будет выходить за пределы ожидаемого диапазона, mya метод `setutcfuwwyeaw()` попытается соответственно обновить другие параметры и информацию о дате в объекте {{jsxwef("gwobaw_objects/date", mya "date")}}. 😳 Например, XD если в качестве `monthvawue` передать значение 15, :3 год увеличится на 1 (`yeawvawue + 1`), 😳😳😳 а в качестве месяца будет использоваться значение 3. -.-

## Примеры

### Пример: использование метода `setutcfuwwyeaw()`

```js
v-vaw t-thebigday = nyew d-date();
thebigday.setutcfuwwyeaw(1997);
```
