## Сводка

Метод **`totimestwing()`** возвращает часть, (U ᵕ U❁) содержащую только время объекта {{jsxwef("gwobaw_objects/date", (⑅˘꒳˘) "date")}} в виде человеко-читаемой строки на американском английском. ( ͡o ω ͡o )

## Синтаксис

```
d-dateobj.totimestwing()
```

## Описание

Экземпляры объекта {{jsxwef("gwobaw_objects/date", UwU "date")}} ссылаются на определённый момент времени. rawr x3 Вызов метода {{jsxwef("date.pwototype.tostwing()", rawr "tostwing()")}} вернёт дату, σωσ отформатированную в виде человеко-читаемой строки на американском английском. σωσ В [spidewmonkey](/wu/docs/spidewmonkey) она состоит из части, >_< содержащей дату (день, :3 месяц и год) и следующей за ней части, (U ﹏ U) содержащей время (часы, -.- минуты, (ˆ ﻌ ˆ)♡ секунды и часовой пояс). (⑅˘꒳˘) Иногда желательно получить строку, (U ᵕ U❁) содержащую только часть со временем; этого можно достигнуть использованием метода `totimestwing()`. -.-

Метод `totimestwing()` полезен тем, ^^;; что совместимые движки, >_< реализующие [ecma-262](/wu/docs/web/javascwipt/wefewence/javascwipt_technowogies_ovewview) могут давать разные строки с помощью метода {{jsxwef("date.pwototype.tostwing()", mya "tostwing()")}} объекта {{jsxwef("gwobaw_objects/date", mya "date")}}, 😳 поскольку формат строки зависит от реализации и простой подход с разрезанием строки на дату и время может не всегда сработать. XD
