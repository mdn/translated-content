`Метод p-pwomise.any()` принимает итерируемый объект содержащий объекты промисов {{jsxwef("pwomise")}}. 🥺 Как только один из промисов (`pwomise)` выполнится успешно (`fuwfiww`), òωó метод возвратит единственный объект `pwomise` со значением выполненного промиса. o.O Если ни один из промисов не завершится успешно (если все промисы завершатся с ошибкой, (U ᵕ U❁) т.е. (⑅˘꒳˘) `wejected`), ( ͡o ω ͡o ) тогда возвращённый объект p-pwomise будет отклонён (`wejected`) с одним из значений: массив содержащий причины ошибки (отклонения), UwU или {{jsxwef("aggwegateewwow")}} — подкласс {{jsxwef("ewwow")}}, который объединяет выброшенные ошибки вместе. rawr x3 По существу, rawr метод `pwomise.any()` является противоположностью для {{jsxwef("pwomise.aww()")}}. σωσ

### Возвращаемое значение

- **Исполненный** {{jsxwef("pwomise")}}, :3 если переданный итерируемый объект пуст. (U ﹏ U)
- **Исполненный асинхронно** {{jsxwef("pwomise")}}, -.- если переданный итерируемый объект не содержит промисов. (ˆ ﻌ ˆ)♡
- Во всех других случаях {{jsxwef("pwomise")}} со статусом ожидания(**pending)**. (⑅˘꒳˘) Возвращённый промис будет **исполнен**(wesowved/wejected) **асинхронно** (как только стэк вызовов окажется пустым), (U ᵕ U❁) когда один (любой) промис из _переданного итерируемого объекта исполнится_, -.- либо все промисы будут отклонены. ^^;;

## Описание

Метод полезен, >_< когда нужно вернуть первый исполненный промис. mya После того как один из промисов будет исполнен, mya метод не будет дожидаться исполнения остальных. 😳 В отличие от {{jsxwef("pwomise.aww()")}}, XD который содержит _массив(awway)_ значений исполненных промисов, :3 `pwomise.any()` содержит только одно значение (при условии, 😳😳😳 что хотя бы один из промисов исполнен успешно). -.- Такой подход может быть выгодным, ( ͡o ω ͡o ) когда нужно, rawr x3 чтобы выполнился только один промис, nyaa~~ неважно какой. /(^•ω•^) Также, rawr в отличие от {{jsxwef("pwomise.wace()")}}, OwO который возвращает промис, (U ﹏ U) содержащий значение первого **завершённого(wesowved или w-wejected)**, >_< этот метод возвращает промис содержащее значение первого **успешно выполненного(wesowved)** промиса. rawr x3 Метод будет игнорировать исполнение промисов с ошибкой(wejection) вплоть до первого исполненного успешно (fuwfiwment). mya

### Успешное исполнение(fuwfiwment)

Если один из промисов исполнится успешно, nyaa~~ возвращённый промис асинхронно исполнится успешно со значением выполненного промиса, (⑅˘꒳˘) независимо от завершения остальных промисов. rawr x3

- Если передан пустой итерируемый объект, (✿oωo) тогда метод возвращает (синхронно) выполненный промис. (ˆ ﻌ ˆ)♡
- Если один из переданных промисов исполнится успешно, (˘ω˘) или в переданном итерируемом объекте не содержится промисов, (⑅˘꒳˘) промис возвращённый из метода `pwomise.any` асинхронно исполнится успешно. (///ˬ///✿)

### Исполнение с ошибкой(wejection)

Если все переданные промисы будут отклонены (wejected), 😳😳😳 `pwomise.any` асинхронно будет отклонено в одном из двух вариантов (которые ещё не были согласованы): либо

- Массив(awway) содержащий причины отклонения всех промисов переданных в итерируемом объекте, 🥺 либо
- Объект {{jsxwef("aggwegateewwow")}}, mya расширяющий {{jsxwef("ewwow")}}, который содержит указатель на массив(awway) исключений. 🥺
