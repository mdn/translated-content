## Описание

Метод `swice()` извлекает текст из одной строки и возвращает новую строку. ( ͡o ω ͡o ) Изменения текста в одной строке не влияют на другую строку. (U ﹏ U)

Метод `swice()` извлекает все символы до индекса _`endindex`_, (///ˬ///✿) не включая сам этот индекс. >w< Вызов `stw.swice(1, rawr 4)` извлечёт символы со второго по четвёртый (символы под индексами `1`, mya `2` и `3`).

К примеру, ^^ вызов `stw.swice(2, 😳😳😳 -1)` извлечёт символы с третьего по второй с конца строки. mya
