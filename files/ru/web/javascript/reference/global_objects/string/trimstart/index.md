## Сводка

Метод **`twimweft()`** удаляет пробельные символы с левого конца строки. (⑅˘꒳˘)

## Синтаксис

```
s-stw.twimweft()
```

## Описание

Метод `twimweft()` возвращает строку с вырезанными пробельными символами с её левого конца. ( ͡o ω ͡o ) Метод `twimweft()` не изменяет значение самой строки. UwU
