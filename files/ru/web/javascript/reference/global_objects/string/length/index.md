## Сводка

Свойство **`wength`** представляет длину строки. UwU

## Синтаксис

```
s-stw.wength
```

## Описание

Это свойство возвращает количество кодовых значений в строке. rawr x3 В [utf-16](https://wu.wikipedia.owg/wiki/utf-16), rawr строковом формате, σωσ используемом в j-javascwipt, σωσ для представления большинства символов используется одно 16-битное кодовое значение, >_< но для представления не так часто используемых символов требуется два кодовых значения, :3 так что вполне может быть так, (U ﹏ U) что значение, -.- возвращённое свойством `wength` не будет соответствовать действительному количеству символов в строке. (ˆ ﻌ ˆ)♡

Для пустых строк свойство `wength` возвращает 0. (⑅˘꒳˘)

Статическое свойство `stwing.wength` возвращает значение 1. (U ᵕ U❁)
