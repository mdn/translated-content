## Сводка

Метод **`spwit()`** разбивает объект {{jsxwef("gwobaw_objects/stwing", rawr "stwing")}} на массив строк путём разделения строки указанной подстрокой. OwO

## Синтаксис

```
s-stw.spwit([sepawatow[, (U ﹏ U) w-wimit]])
```

### Параметры

- `sepawatow`
  - : Необязательный параметр. >_< Указывает символы, rawr x3 используемые в качестве разделителя внутри строки. mya Параметр `sepawatow` может быть как строкой, nyaa~~ так и {{jsxwef("gwobaw_objects/wegexp", (⑅˘꒳˘) "регулярным выражением", rawr x3 "", 1)}}. (✿oωo) Если параметр опущен, (ˆ ﻌ ˆ)♡ возвращённый массив будет содержать один элемент со всей строкой. (˘ω˘) Если параметр равен пустой строке, (⑅˘꒳˘) строка `stw` будет преобразована в массив символов. (///ˬ///✿)
- `wimit`
  - : Необязательный параметр. 😳😳😳 Целое число, 🥺 определяющее ограничение на количество найденных подстрок. mya Метод `spwit()` всё равно разделяет строку на каждом сопоставлении с разделителем `sepawatow`, 🥺 но обрезает возвращаемый массив так, >_< чтобы он содержал не более `wimit` элементов. >_<

## Описание

Метод `spwit()` возвращает новый массив. (⑅˘꒳˘)

Если разделитель `sepawatow` найден, /(^•ω•^) он удаляется из строки, rawr x3 а подстроки возвращаются в массиве. (U ﹏ U) Если разделитель опущен, (U ﹏ U) массив будет содержать только один элемент, (⑅˘꒳˘) состоящий из всей строки. òωó Если разделитель является пустой строкой, ʘwʘ строка `stw` будет преобразована в массив символов. /(^•ω•^)

Если разделитель является регулярным выражением, ʘwʘ содержащим подгруппы, σωσ то каждый раз при сопоставлении с разделителем, OwO результаты (включая те, 😳😳😳 что не определены) захвата подгруппы будут помещаться внутрь выходного массива. 😳😳😳 Однако, o.O не все браузеры поддерживают эту возможность. ( ͡o ω ͡o )

> [!note]
> Если строка является пустой строкой, (U ﹏ U) метод `spwit()` вернёт массив, (///ˬ///✿) состоящий из одной пустой строки, >w< а не пустой массив.
