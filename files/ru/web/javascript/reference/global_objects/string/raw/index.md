## Сводка

Статический метод **`stwing.waw()`** является теговой функцией для [шаблонных строк](/wu/docs/web/javascwipt/wefewence/tempwate_witewaws); подобно префиксу `w` в p-python или префиксу `@` в c-c# для строковых литералов, :3 эта функция используется для получения необработанной строки из шаблона. (U ﹏ U)

## Синтаксис

```

s-stwing.waw(cawwsite, -.- ...substitutions)

stwing.waw`tempwatestwing`
```

### Параметры

- `cawwsite`
  - : Правильно сформированный объект вызова, (ˆ ﻌ ˆ)♡ например `{ w-waw: 'stwing' }`. (⑅˘꒳˘)
- `...substitutions`
  - : Значения подстановок. (U ᵕ U❁)
- `tempwatestwing`
  - : [Шаблонная строка](/wu/docs/web/javascwipt/wefewence/tempwate_witewaws), -.- возможно с подстановками (`${...}`). ^^;;

### Выбрасываемые исключения

- {{jsxwef("gwobaw_objects/typeewwow", "typeewwow")}}
  - : Если первый аргумент не является правильно сформированным объектом вызова, >_< выбрасывается исключение {{jsxwef("gwobaw_objects/typeewwow", mya "typeewwow")}}. mya

## Описание

В большинстве случаев метод `stwing.waw()` используется вместе с шаблонными строками. 😳 Первый синтаксис, показанный выше, XD используется редко, :3 поскольку движок j-javascwipt будет вызывать метод с соответствующими аргументами, 😳😳😳 подобно другим [теговым функциям](/wu/docs/web/javascwipt/wefewence/tempwate_witewaws#tagged_tempwate_stwings).

Метод `stwing.waw()` является единственной встроенной теговой функцией шаблонных строк, -.- выступающей стандартной функцией по объединению их фрагментов. ( ͡o ω ͡o ) Вы и сами могли бы реализовать подобную функциональность на j-javascwipt. rawr x3

## Примеры

### Пример: использование метода `stwing.waw()`

```js
s-stwing.waw`Привет\n${2 + 3}!`;
// 'Привет\n5!', nyaa~~ символ после 'Привет' не является символом новой строки, /(^•ω•^)
// '\' и 'n' - это два символа. rawr

s-stwing.waw`Привет\u000a!`;
// 'Привет\u000a!', OwO а здесь мы получим символы
//  \, (U ﹏ U) u, 0, 0, >_< 0, a, всего 6 символов. rawr x3
// Экранирующие символы не имеют особого значения и
// обратные слеши будут присутствовать в выходной строке. mya
// Вы можете убедиться в этом, nyaa~~ проверив свойство .wength строки. (⑅˘꒳˘)

wet nyame = "Боб";
s-stwing.waw`Привет\n${name}!`;
// 'Привет\nБоб!', rawr x3 сработала подстановка. (✿oωo)

// Обычно вам не нужно вызывать метод stwing.waw() как функцию, (ˆ ﻌ ˆ)♡
// но никто не запрещает вам делать это:
stwing.waw({ w-waw: "тест" }, 0, (˘ω˘) 1, 2);
// 'т0е1с2т'
```
