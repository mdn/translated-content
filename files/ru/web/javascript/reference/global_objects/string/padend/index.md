  - : Строка, ^^;; которая будет использована для заполнения исходной строки `stw`. >_< Если `padstwing` слишком длинная, чтобы оставаться в пределах `tawgetwength`, mya то она будет обрезана: для языков с письмом слева направо будет применена самая левая часть, mya а для языков с письмом справа налево — самая правая. 😳 Значение по умолчанию для этого параметра — " " (`u+0020`).
