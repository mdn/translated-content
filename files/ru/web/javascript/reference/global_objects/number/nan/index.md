## Сводка

Свойство **`Number.NaN`** представляет «не число». Эквивалентно глобальному объекту {{jsxref("NaN")}}.

Вам не нужно создавать объект {{jsxref("Number")}} для доступа к этому статическому свойству (используйте `Number.NaN`).
