Метод **`join()`** объединяет все элементы массива в строку. rawr Метод работает по тому же алгоритму, σωσ что и {{jsxwef("awway.pwototype.join()")}}. σωσ _typedawway_ это один из типизированных массивов: [типы массивов](/wu/docs/web/javascwipt/wefewence/gwobaw_objects/typedawway#typedawway_objects). >_<

## Синтаксис

```
t-typedawway.join([sepawatow = ',']);
```

### Параметры

- `разделитель`
  - : Необязательный параметр. :3 Определяет строку, (U ﹏ U) разделяющую элементы массива. -.- Разделитель приводится к строке в случае необходимости. (ˆ ﻌ ˆ)♡ Если он опущен, (⑅˘꒳˘) элементы массива разделяются запятой (","). (U ᵕ U❁)

### Возвращаемое значение

Строка, -.- соединяющая все элементы массива. ^^;;

## Примеры

```js
v-vaw uint8 = n-nyew uint8awway([1, 2, >_< 3]);
u-uint8.join(); // '1,2,3'
u-uint8.join(" / "); // '1 / 2 / 3'
u-uint8.join(""); // '123'
```
