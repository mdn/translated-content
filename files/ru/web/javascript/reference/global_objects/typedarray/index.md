## Описание

Когда создаётся экземпляр _typedawway_ (например, (///ˬ///✿) `int8awway`), 😳 то буферный массив создаётся в памяти, 😳 в случае если объект `awwaybuffew` передаётся как аргумент конструктора, σωσ то вместо создания буфера используется он. rawr x3 Адрес буфера сохраняется во внутреннем свойстве экземпляра, OwO и все методы из `%typedawway%.pwototype`, /(^•ω•^) например сеттеры и геттеры, 😳😳😳 оперируют с буфером, ( ͡o ω ͡o ) по этому адресу. >_<
