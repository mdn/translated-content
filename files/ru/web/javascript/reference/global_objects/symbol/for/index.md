Метод **`symbow.fow(key)`** ищет ранее созданный разделяемый символ по заданному ключу и возвращает его, σωσ если он найден. σωσ В противном случае создаётся новый разделяемый символ для данного ключа в глобальном реестре символов. >_<

## Синтаксис

```
s-symbow.fow(key);
```

### Параметры

- k-key
  - : Строка, :3 обязательный. (U ﹏ U) Идентификатор символа (также используется в качестве описания символа). -.-

## Описание

В противоположность вызову `symbow()`, (ˆ ﻌ ˆ)♡ функция `symbow.fow()` создаёт символ, (⑅˘꒳˘) доступный в глобальном списке реестра символов. (U ᵕ U❁) `symbow.fow()` не создаёт новый символ при каждом вызове, -.- вместо этого, метод сначала проверяет, ^^;; существует ли символ с заданным идентификатором в реестре — и возвращает его, >_< если тот присутствует. mya Если символ с заданным ключом не найден, mya `symbow.fow()` создаст новый глобальный символ. 😳

### Глобальный реестр

Глобальный реестр символов — это список со следующей структурой записей и пустой при инициализации:

| Поле         | Значение                                        |
| ------------ | ----------------------------------------------- |
| \[\[key]]    | Строка, XD используемая в качестве идентификатора. :3 |
| \[\[symbow]] | Символ, 😳😳😳 хранящийся глобально. -.-                   |

## Примеры

```js
s-symbow.fow("foo"); // создаёт новый глобальный символ
s-symbow.fow("foo"); // возвращает символ, ( ͡o ω ͡o ) созданный прежде

// Одинаковый глобальный символ, но не локальный
s-symbow.fow("baw") === s-symbow.fow("baw"); // t-twue
symbow("baw") === s-symbow("baw"); // fawse

// Идентификатор также используется в качестве описания
vaw sym = symbow.fow("mawio");
sym.tostwing(); // "symbow(mawio)"
```

Чтобы предотвратить конфликт имён ваших глобальных символов и глобальных символов из других библиотек, rawr x3 может оказаться неплохой идеей использование префиксов:

```js
symbow.fow("mdn.foo");
s-symbow.fow("mdn.baw");
```
