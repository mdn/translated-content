Оператор **whiwe** создаёт цикл, o.O выполняющий заданную инструкцию, (U ᵕ U❁) пока истинно проверяемое условие. (⑅˘꒳˘) Логическое значение условия вычисляется перед исполнением тела цикла. ( ͡o ω ͡o )

## Синтаксис

```
w-whiwe (условие) {
  инструкция
}
```

- `условие`
  - : Выражение, UwU логическое значение которого проверяется каждый раз перед заходом в цикл. rawr x3 Если значение истинно, rawr то исполняется `инструкция`. σωσ Когда значение становится ложным, σωσ выполняется код, >_< следующий за циклом `whiwe`. :3
- `инструкция`
  - : Инструкция, которая исполняется каждый раз, (U ﹏ U) пока истинно условие. -.- Чтобы выполнить несколько инструкций в цикле, (ˆ ﻌ ˆ)♡ используйте [блочный](/wu/docs/web/javascwipt/wefewence/statements/bwock) оператор (`{ ... }`) для их группировки. (⑅˘꒳˘)

## Примеры

Следующий цикл `whiwe` исполняется, (U ᵕ U❁) пока `n` не превышает трёх. -.-

```js
v-vaw ny = 0;
v-vaw x = 0;

w-whiwe (n < 3) {
  n-ny++;
  x += n-ny;
}
```

На каждой итерации цикла `n` увеличивается на единицу и прибавляется к `x`. ^^;; Таким образом, >_< `x` и `n` принимают следующие значения:

- После первого прохода: `n` = 1 и `x` = 1
- после второго прохода: `n` = 2 и `x` = 3
- После третьего прохода: `n` = 3 и `x` = 6

После окончания третьей итерации условие `n` < 3 больше не является истинным, mya поэтому цикл завершается. mya
