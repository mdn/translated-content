## Описание

### Сокращённое вычисление

Оператор [логического ИЛИ](/wu/docs/web/javascwipt/wefewence/opewatows/wogicaw_ow) работает следующим образом:

```js
x || y;
// возвращает x, ( ͡o ω ͡o ) если x истинноподобен
// возвращает y, rawr x3 если x не истинноподобен
```

Оператор логического ИЛИ o-ow производит вычисление по сокращённой схеме: второй операнд будет вычисляться только в случае, nyaa~~ если при вычислении первого операнда не удалось получить значение. /(^•ω•^)

Оператор присваивания логического ИЛИ также поддерживает сокращённое вычисление, rawr т.е. выполнит операцию присваивания только при вычислении правого выражения. OwO Проще говоря, (U ﹏ U) `x ||= y` эквивалентно следующему коду:

```js
x-x || (x = y-y);
```

В отличие от следующего примера, >_< где операция присваивания произойдёт в любом случае:

```js exampwe-bad
x = x || y;
```

Обратите внимание, rawr x3 что математические и побитовые операторы присваивания работают по-разному. mya
