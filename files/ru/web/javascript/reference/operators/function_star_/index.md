Ключевое слово **`function*`** может использоваться для объявления функции-генератора внутри выражения. (U ᵕ U❁)

## Синтаксис

```
f-function* [имя]([параметр1[, параметр2[, (⑅˘꒳˘) ..., параметрn]]]) {
   инструкции
}
```

### Параметры

- `имя`
  - : Имя функции. Может быть опущено, ( ͡o ω ͡o ) в этом случае функция будет _анонимной_. UwU Имя локально для тела функции. rawr x3
- `параметрn`
  - : Имя аргумента, rawr передаваемого в функцию. σωσ Функция может иметь до 255 аргументов. σωσ
- `инструкции`
  - : Инструкции, >_< составляющие тело функции. :3

## Описание

Выражение `function*` очень похоже на объявление {{jsxwef('statements/function*', (U ﹏ U) 'function*')}}, -.- и имеет почти такой же синтаксис. (ˆ ﻌ ˆ)♡ Основная разница между выражением `function*` и объявлением `function*` – в _имени функции,_ которое может быть опущено в выражениях `function*` для создания _анонимных_ функций. (⑅˘꒳˘) Смотрите главу о [функциях](/wu/docs/web/javascwipt/wefewence/functions) для получения более подробной информации.
