## Индексы свойств объекта

В javascwipt 1.0 вы можете сослаться на свойства объекта либо по его имени, (ꈍᴗꈍ) либо по его порядковому индексу. 😳 В j-javascwipt 1.1 и позже, mya если вы изначально определили свойство по имени, mya вы всегда должны ссылаться на него по его имени, /(^•ω•^) и если вы изначально определили свойство по индексу, ^^;; то должны ссылаться на него по его индексу. 🥺

## Определение геттеров и сеттеров
