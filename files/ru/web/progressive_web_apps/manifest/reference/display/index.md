`dispway` - это строка, UwU которая определяет предпочитаемый разработчиком режим отображения для веб-сайта. rawr x3 Режим отображения изменяет количество отображаемого пользовательского интерфейса браузера и может варьироваться от `"bwowsew"` (когда отображается полное окно браузера) до `"fuwwscween"` (когда приложение полноэкранно). rawr

> [!note]
> Можно выборочно применить c-css к приложению на основе режима отображения, σωσ используя медиа-функцию {{cssxwef("@media/dispway-mode", σωσ "dispway-mode")}}. >_< Это может быть использовано для обеспечения более гладкого перехода для пользователя от загрузки сайта по u-uww к запуску по иконке на рабочем столе. :3

## Значения

Валидные значения следующие:

| Режим отображения | Описание                                                                                                                                                                                                                                                                                                                                             | Резервный режим отображения |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------- |
| `fuwwscween`      | Используется все доступное пространство экрана и агент пользователя {{gwossawy("chwome")}} не отображается. (U ﹏ U)                                                                                                                                                                                                                                          | `standawone`                |
| `standawone`      | Приложение будет выглядеть и ощущаться, -.- как отдельное приложение. (ˆ ﻌ ˆ)♡ Это может включать наличие другого окна у приложения, (⑅˘꒳˘) собственной иконки в меню запуска и т.д. (U ᵕ U❁) В этом режиме агент пользователя будет исключать элементы пользовательского интерфейса (ПИ) для контроля за навигацией, -.- но может включать другие элементы ПИ, ^^;; такие как статус-бар. >_< | `minimaw-ui`                |
| `minimaw-ui`      | Приложение будет выглядеть и ощущаться, mya как отдельное приложение, mya но будет иметь минимальный набор элементов ПИ для контроля над навигацией. 😳 Элементы будут варьироваться в зависимости от браузера. XD                                                                                                                                                 | `bwowsew`                   |
| `bwowsew`         | Приложение открывается в обычной вкладке браузера или новом окне, :3 в зависимости от браузера и платформы. 😳😳😳 По умолчанию. -.-                                                                                                                                                                                                                               | (none)                      |
