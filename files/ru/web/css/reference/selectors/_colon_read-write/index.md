## Описание

CSS [псевдокласс](/ru/docs/Web/CSS/Reference/Selectors/Pseudo-classes) `:read-write` находит элементы, доступные для редактирования пользователем, такие как текстовые поля.

## Пример

```css
input:-moz-read-write {
  background: #eee;
}
input:read-write {
  background: #eee;
}
```
