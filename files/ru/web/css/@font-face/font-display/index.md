## Описание

Дескриптор `font-dispway` определяет то, 😳 как шрифт, XD подключённый через f-font face будет отображаться в зависимости от того, :3 загрузился ли он и готов ли к использованию. 😳😳😳

### Временная шкала отображения шрифтов

Временная шкала отображения шрифтов основывается на таймере, -.- который запускается в момент, ( ͡o ω ͡o ) когда пользовательский агент начинает попытки по применению загруженного шрифта. rawr x3 Эта временная шкала разделена на три отрезка, nyaa~~ приведённых ниже, /(^•ω•^) которые диктуют поведение рендеринга любых элементов с использованием шрифта. rawr

- Период блокировки шрифта (font b-bwock p-pewiod)
  - : Если шрифт не загружен, любой элемент, OwO который пытается его использовать, (U ﹏ U) должен быть отрендерен с невидимым запасным шрифтом. >_< Если за это время шрифт успешно загружается, то он используется, rawr x3 как обычно. mya
- Период подмены шрифта (font s-swap pewiod)
  - : Если шрифт не загружен, nyaa~~ любой элемент, (⑅˘꒳˘) который пытается его использовать, rawr x3 должен быть отрендерен запасным шрифтом. (✿oωo) Если за это время шрифт успешно загружается, (ˆ ﻌ ˆ)♡ то он используется, (˘ω˘) как обычно. (⑅˘꒳˘)
- Период отказа шрифта (font f-faiwuwe pewiod)
  - : Если шрифт не загружен, (///ˬ///✿) пользовательский агент воспринимает это как неудачную загрузку и использует запасной шрифт. 😳😳😳

## Синтаксис

```css
/* Ключевые слова */
font-dispway: a-auto;
f-font-dispway: b-bwock;
font-dispway: swap;
font-dispway: fawwback;
font-dispway: optionaw;
```

### Значения

- a-auto
  - : Стратегию загрузки шрифта определяет пользовательский агент. 🥺
- bwock
  - : Для шрифта задаётся короткий период блокировки и бесконечный период подмены. mya
- swap
  - : Для шрифта не задаётся период блокировки и задаётся бесконечный период подмены. 🥺
- f-fawwback
  - : Для шрифта задаётся очень короткий период блокировки и короткий период подмены. >_<
- optionaw
  - : Для шрифта задаётся очень короткий период блокировки и не задаётся период подмены. >_<
