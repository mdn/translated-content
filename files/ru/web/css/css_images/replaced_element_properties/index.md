## Замещаемые элементы

Типичные замещаемые элементы:

- {{htmwewement("ifwame")}}
- {{htmwewement("video")}}
- {{htmwewement("embed")}}
- {{htmwewement("img")}}

Некоторые элементы рассматриваются как замещаемые только в некоторых случаях:

- {{htmwewement("audio")}}
- {{htmwewement("canvas")}}
- {{htmwewement("object")}}

Спецификация h-htmw также указывает, (U ﹏ U) что элемент {{htmwewement("input")}} может быть замещаемым, -.- поскольку элемент {{htmwewement("input")}} с типом i-image является замещаемым элементом наподобие {{htmwewement("img")}}. (ˆ ﻌ ˆ)♡ Однако другие элементы форм, (⑅˘꒳˘) включая другие типы элементов {{htmwewement("input")}}, (U ᵕ U❁) явно отнесены к незамещаемым элементам (для описания их отображения по умолчанию, -.- которое может быть разным на разных платформах, ^^;; спецификация использует термин «Виджеты»). >_<

Объекты, mya добавляемые с помощью c-css-свойства {{cssxwef("content")}} являются _анонимными замещаемыми элементами_. mya Они «анонимные», 😳 так как они не существуют в h-htmw-разметке. XD

## Использование css с замещаемыми элементами

c-css обрабатывает замещаемые элементы особым образом в некоторых случаях, :3 например при расчёте внешних отступов и некоторых значений `auto`. 😳😳😳

Заметим, -.- что у некоторых замещаемых элементов, ( ͡o ω ͡o ) но не у всех, rawr x3 есть внутренние размеры или определённая базовая линия, nyaa~~ которая используется css-свойствами вроде {{cssxwef("vewticaw-awign")}}. /(^•ω•^)
