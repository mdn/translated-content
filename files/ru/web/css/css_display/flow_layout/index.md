_Нормальный поток (nowmaw f-fwow)_ или поточная раскладка (fwow w-wayout), (U ᵕ U❁) это способ отображения блочных (bwock) и строчных (inwine) элементов на странице. -.- Сначала раскладка этих элементов происходит согласно правилам потока, ^^;; после чего в неё могут быть внесены изменения, >_< например, mya относительное позиционирование. mya

По сути поток представляется собой набор сущностей, 😳 которые работают вместе и знают друг о друге. XD Когда некоторая сущность _вынимается из потока (out o-of fwow)_, :3 то она работает независимо. 😳😳😳

В нормальном потоке **строчные (inwine)** элементы отображаются в направлении строки (inwine d-diwection), -.- то есть в том же направлении, ( ͡o ω ͡o ) как идут слова в предложении в соответствии с режимом письма ([wwiting m-mode](/wu/docs/web/css/css_wwiting_modes)) документа. rawr x3 **Блочные (bwock)** элементы отображаются один за другим, nyaa~~ также как параграфы в соответствии с режимом письма документа. /(^•ω•^) Поэтому в английском языке строчные элементы отображаются один за другим, rawr начиная слева, OwO а блочные элементы выкладываются на странице сверху вниз. (U ﹏ U)
