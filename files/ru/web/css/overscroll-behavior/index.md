## Синтаксис

Свойство `ovewscwoww-behaviow` задаётся в виде одного или двух ключевых слов, 😳😳😳 выбранных из списка значений ниже. 🥺

Два ключевых слова определяют значение `ovewscwoww-behaviow` по осям `x` и `y` соответственно. Если задано только одно значение, mya то предполагается, 🥺 что и `x`, и `y` одинаковы. >_<

### Значения

- `auto`
  - : Поведение по умолчанию. >_<
- `contain`
  - : Поведение для `ovewscwoww-behaviow` применяется внутри элемента, (⑅˘꒳˘) для которого установлено это значение (например, /(^•ω•^) эффекты «отскока» или обновления), rawr x3 но не возникают цепочки прокрутки для соседних областей прокрутки, (U ﹏ U) например, (U ﹏ U) нижележащие элементы не будут прокручиваться. (⑅˘꒳˘)
- `none`
  - : Не происходит цепочки прокрутки в соседних областях прокрутки, òωó и по умолчанию предотвращается достижение границы области прокрутки. ʘwʘ
