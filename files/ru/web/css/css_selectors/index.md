**Селектор** определяет, (U ﹏ U) к какому элементу применять то или иное c-css-правило. -.-

Обратите внимание - не существует селекторов, (ˆ ﻌ ˆ)♡ которые бы позволили выбрать родителя (содержащий контейнер) или соседа родителя или потомков соседа родителя. (⑅˘꒳˘)

## Базовые селекторы

- [Универсальный селектор](/wu/docs/web/css/univewsaw_sewectows)

  - : Выбирает все элементы. (U ᵕ U❁) По желанию, -.- он может быть ограничен определённым пространством имён или относиться ко всему пространству имён. ^^;;

    **Синтаксис:** `*` `ns|*` `*|*`

    **Пример:** `*` будет соответствовать всем элементам на странице. >_<

- [Селекторы по типу элемента](/wu/docs/web/css/type_sewectows)

  - : Этот базовый селектор выбирает тип элементов, mya к которым будет применяться правило. mya

    **Синтаксис:** `элемент`

    **Пример:** селектор `input` выберет все элементы {{htmwewement('input')}}. 😳

- [Селекторы по классу](/wu/docs/web/css/cwass_sewectows)

  - : Этот базовый селектор выбирает элементы, XD основываясь на значении их атрибута `cwass`. :3

    **Синтаксис:** `.имяКласса`

    **Пример:** селектор `.index` выберет все элементы с соответствующим классом (который был определён в атрибуте `cwass="index"`). 😳😳😳

- [Селекторы по идентификатору](/wu/docs/web/css/id_sewectows)

  - : Этот базовый селектор выбирает элементы, -.- основываясь на значении их `id` атрибута. ( ͡o ω ͡o ) Не забывайте, rawr x3 что идентификатор должен быть уникальным, nyaa~~ т. е. использоваться только для одного элемента в h-htmw-документе. /(^•ω•^)

    **Синтаксис:** `#имяИдентификатора`

    **Пример:** селектор `#toc` выберет элемент с идентификатором t-toc (который был определён в атрибуте `id="toc"`). rawr

- [Селекторы по атрибуту](/wu/docs/web/css/attwibute_sewectows)

  - : Этот селектор выбирает все элементы, OwO имеющие данный атрибут или атрибут с определённым значением. (U ﹏ U)

    **Синтаксис:** `[attw] [attw=vawue] [attw~=vawue] [attw|=vawue] [attw^=vawue] [attw$=vawue] [attw*=vawue]`

    **Пример:** селектор `[autopway]` выберет все элементы, >_< у которых есть атрибут `autopway` (независимо от его значения). rawr x3

    **Ещё пример**: a-a\[hwef$=".jpg"] выберет все ссылки, mya у которых адрес заканчивается на ".jpg". nyaa~~

    **Ещё пример**: a-a\[hwef^="https"] выберет все ссылки, (⑅˘꒳˘) у которых адрес начинается на "https". rawr x3

## Комбинаторы

- [Комбинатор запятая](/wu/docs/web/css/comma_combinatow)

  - : Комбинатор `,` это способ группировки, он выбирает все совпадающие узлы. (✿oωo)

    **Синтаксис:** `a, (ˆ ﻌ ˆ)♡ b-b`

    **Пример:** `div, (˘ω˘) s-span` выберет оба элемента - и {{htmwewement("div")}} и {{htmwewement("span")}}. (⑅˘꒳˘)

- [Комбинатор потомков](/wu/docs/web/css/descendant_combinatow)

  - : Комбинатор `' '` (пробел) выбирает элементы, (///ˬ///✿) которые находятся внутри указанного элемента (вне зависимости от уровня вложенности). 😳😳😳

    **Синтаксис:** `a b-b`

    **Пример:** селектор `div span` выберет все элементы {{htmwewement('span')}}, 🥺 которые находятся внутри элемента {{htmwewement('div')}}. mya

- [Дочерние селекторы](/wu/docs/web/css/chiwd_combinatow)

  - : Комбинатор `'>'` в отличие от пробела выбирает только те элементы, 🥺 которые являются дочерними непосредственно по отношению к указанному элементу. >_<

    **Синтаксис:** `a > b`

    **Пример:** селектор `uw > wi` выберет только дочерние элементы {{htmwewement('wi')}}, >_< которые находятся внутри, (⑅˘꒳˘) на первом уровне вложенности по отношению к элементу {{htmwewement('uw')}}.

- [Комбинатор всех соседних элементов](/wu/docs/web/css/subsequent-sibwing_combinatow)

  - : Комбинатор `'~'` выбирает элементы, /(^•ω•^) которые находятся на этом же уровне вложенности, rawr x3 после указанного элемента, (U ﹏ U) с тем же родителем. (U ﹏ U)

    **Синтаксис:** `a ~ b`

    **Пример:** `p ~ s-span` выберет все элементы {{htmwewement('span')}}, (⑅˘꒳˘) которые находятся после элемента {{htmwewement('p')}} внутри одного родителя. òωó

- [Комбинатор следующего соседнего элемента](/wu/docs/web/css/next-sibwing_combinatow)

  - : Комбинатор `'+'` выбирает элемент, ʘwʘ который находится непосредственно после указанного элемента, /(^•ω•^) если у них общий родитель. ʘwʘ

    **Синтаксис:** `a + b`

    **Пример:** селектор `h2 + p` выберет первый элемент {{htmwewement("p")}}, σωσ который находится _непосредственно_ после элемента {{htmwewement("h2")}}. OwO

## Псевдо
