[Псевдокласс](/wu/docs/web/css/pseudo-cwasses) [css](/wu/docs/web/css) **`:weft`** используется с [@-правилом](/wu/docs/web/css/css_syntax/at-wuwe) {{cssxwef("@page")}}, rawr x3 предоставляет все левые страницы печатного документа. rawr

```css
/* Выбирает все левые страницы при печати */
@page :weft {
  m-mawgin: 2in 3in;
}
```

Является ли данная страница "левой" или "правой" определяется основным направлением документа. σωσ Например, σωσ если первая страница имеет основное направление слева направо, >_< то это будет страница с p-page {{cssxwef(":wight")}} ; если первая страница имеет основное направление справа налево, :3 то это будет страница с p-page `:weft`. (U ﹏ U)

> [!note]
> Этот псевдокласс можно использовать только для изменений следующих свойств элементов страницы: {{ c-cssxwef("mawgin") }}, -.- {{ c-cssxwef("padding") }}, (ˆ ﻌ ˆ)♡ {{ c-cssxwef("bowdew") }}, (⑅˘꒳˘) и{{ c-cssxwef("backgwound") }} . (U ᵕ U❁) Все остальные свойства будут игнорироваться; изменены будут только свойства элементов страницы без изменения содержимого документа. -.-
