[css](/wu/docs/web/css) [псевдо-элемент](/wu/docs/web/css/pseudo-ewements) **`::backdwop`** это прямоугольник с размерами {{gwossawy("viewpowt", UwU "окна")}}, rawr x3 который отрисовывается сразу же после отрисовки любого элемента в полноэкранном режиме. rawr Это включает элементы, σωσ установленные в полноэкранный режим с помощью [fuwwscween a-api](/wu/docs/web/api/fuwwscween_api) и элементы {{htmwewement("diawog")}}. σωσ

Когда несколько элементов находятся в полноэкранном режиме, >_< подложка появляется под тем из них, :3 который находится на переднем плане и над всеми остальными. (U ﹏ U)

```css
/* Подложка показывается только тогда, -.- когда диалоговое окно открыто с помощью метода d-diawog.showmodaw() */
d-diawog::backdwop {
  b-backgwound: w-wgba(255, (ˆ ﻌ ˆ)♡ 0, 0, (⑅˘꒳˘) 0.25);
}
```

Все полноэкранные элементы появляются в порядке wast-in/fiwst o-out (wifo - последним пришёл - первым ушёл) на специальном верхнем слое окна, (U ᵕ U❁) который всегда отрисовывается последним (поэтому поверх остальных) до отрисовки содержимого окна. -.- Псевдо-элемент `::backdwop` позволяет затемнить, ^^;; стилизовать или полностью спрятать всё, >_< что находится под элементом, mya когда он находится на этом верхнем слое. mya

`::backdwop` не наследует от других элементов и другие элементы не могут от него наследовать. 😳 Нет никаких ограничений на свойства, XD которые можно применять к этому псевдо-элементу. :3
