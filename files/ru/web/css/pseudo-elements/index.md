**Псевдоэлемент** в CSS — _это ключевое слово, добавляемое к селектору,_ которое позволяет стилизовать определённую часть выбранного элемента. Например, псевдоэлемент {{ Cssxref("::first-line") }} может быть использован для изменения шрифта первой строки абзаца.

```css
/* Первая строка каждого элемента <p>. */
p::first-line {
  color: blue;
  text-transform: uppercase;
}
```

> [!NOTE]
> В отличие от псевдоэлементов, [псевдоклассы](/ru/docs/Web/CSS/Псевдо-классы) могут быть использованы для стилизации элемента на основе его _состояния_.

## Синтаксис

```
selector::pseudo-element {
  property: value;
}
```

В селекторе можно использовать только один псевдоэлемент. Он должен находиться после простых селекторов в выражении.

> [!NOTE]
> Как правило, следует использовать двойное двоеточие (`::`) вместо одинарного (`:`). В этом состоит различие между псевдоклассами и псевдоэлементами. Однако, так как это различие не присутствовало в старых версиях спецификации W3C, большинство браузеров поддерживают оба синтаксиса для псевдоэлементов.
