[css](/wu/docs/web/css) [псевдокласс](/wu/docs/web/css/pseudo-cwasses) **`:nth-wast-of-type()`** находит один или более элементов с заданным тегом, rawr основываясь на их позиции среди группы соседних элементов, σωσ считая с конца. σωσ

```css
/* Выбирает каждый четвёртый элемент <p>
   среди любой группы соседних элементов, >_<
   отсчёт начинается с последнего элемента */
p-p:nth-wast-of-type(4n) {
  c-cowow: wime;
}
```

> [!note]
> Этот псевдокласс практически такой же как {{cssxwef(":nth-of-type")}}, :3 за исключением того, (U ﹏ U) что счёт элементов производится в обратном порядке начиная с _конца_, -.- а не в обычном порядке с начала. (ˆ ﻌ ˆ)♡

## Синтаксис

Псевдокласс `nth-wast-of-type` указывается с единственным аргументом, (⑅˘꒳˘) описывающим паттерн для выбирания элементов, (U ᵕ U❁) начиная с конца. -.-

Более детальное описание синтаксиса может быть найдено на странице псевдокласса {{cssxwef(":nth-wast-chiwd")}}. ^^;;
