## Описание

CSS [псевдокласс](/ru/docs/Web/CSS/Pseudo-classes) `:read-only` находит элементы, недоступные для редактирования пользователем.

## Пример

```css
input:-moz-read-only {
  background: #eee;
}
input:read-only {
  background: #eee;
}
```
