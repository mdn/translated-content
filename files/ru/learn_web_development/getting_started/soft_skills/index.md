В этом модуле представлены рекомендации по развитию гибких навыков (англ. _soft skills_), которые будут полезны при изучении веб-разработки, и являются важными качествами для входа в индустрию. Эти навыки помогут сформировать правильный подход к обучению, исследованию и сотрудничеству, а также повысят шансы на трудоустройство.

## Необходимые требования

Этот модуль по большей части теоретический и не требует каких-либо специальных практических знаний о веб-технологиях. Тем не менее, мы рекомендуем предварительно изучить по порядку предыдущие модули, прежде чем приступать к этому.

- [Сотрудничество и командная работа](/ru/docs/Learn_web_development/Getting_started/Soft_skills/Collaboration_and_teamwork)
  - : Специалисту веб-разработки необходимо взаимодействовать с разными людьми на разных этапах реализации проекта — обсуждать идеи, вносить предложения, писать код и т.д. Эта статья о том, как работать вместе с другими людьми.
