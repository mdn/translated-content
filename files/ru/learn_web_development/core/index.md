Наши **основные модули** охватывают темы, которые, как мы считаем, должен хорошо понимать каждый веб-разработчик. Сюда входит вся необходимая информация для проектирования и создания базового, доступного веб-приложения, соответствующего современным лучшим практикам, а также для управления и развёртывания кода с использованием соответствующих инструментов.

## Предварительные требования

Хотя для начала изучения этого набора модулей не требуется никаких предварительных знаний, мы рекомендуем вам ознакомиться со своим компьютером, убедиться, что у вас установлен базовый набор необходимых приложений (таких как веб-браузеры и редактор кода), и, если вы ещё этого не сделали, прочитать некоторые вводные материалы о вебе и веб-технологиях. Все эти темы освещены в наших [модулях для начинающих](/ru/docs/Learn_web_development/Getting_started).

В частности, если вы никогда раньше не занимались программированием, мы рекомендуем модуль [Ваш первый веб-сайт](/ru/docs/Learn_web_development/Getting_started/Your_first_website), который очень мягко познакомит вас с веб-технологиями и проведёт через процесс создания простого веб-сайта.

## Модули

- [Структурирование содержимого с помощью HTML](/ru/docs/Learn_web_development/Core/Structuring_content)
  - : Чтобы создать высококачественный, удобный и доступный веб-сайт, необходимо понимать, как определить структуру вашего контента с помощью семантического HTML. Этот модуль охватывает основы языка HTML, а затем рассматривает такие ключевые области, как структура документа, ссылки, списки, изображения, формы и многое другое.
- [Основы стилизации с помощью CSS](/ru/docs/Learn_web_development/Core/Styling_basics)
  - : CSS (каскадные таблицы стилей) используется для стилизации и компоновки веб-страниц — например, для изменения шрифта, цвета, размера и интервалов вашего контента, разделения его на несколько столбцов, добавления анимации и других декоративных элементов. Этот модуль предоставляет все основы CSS, которые вам понадобятся на данный момент, включая синтаксис, возможности и методы.
- [Стилизация текста с помощью CSS](/ru/docs/Learn_web_development/Core/Text_styling)
  - : После того как основы языка CSS освоены, следующая тема CSS, на которой вам следует сосредоточиться, — это стилизация текста, одно из самых распространённых действий в CSS. Здесь мы рассматриваем основы, включая настройку шрифта, насыщенности, курсива, межстрочного и межбуквенного интервалов, теней и других свойств текста. В конце модуля мы рассмотрим применение пользовательских шрифтов на вашей странице, а также стилизацию списков и ссылок.
- [CSS макет](/ru/docs/Learn_web_development/Core/CSS_layout)
  - : В предыдущих модулях мы рассмотрели, как стилизовать блоки, в которых находится ваш контент, и управлять ими. Теперь пришло время узнать, как правильно располагать эти блоки относительно друг друга и области просмотра браузера. В этом модуле рассматриваются свойства float, позиционирование, другие современные инструменты макета и создание адаптивных дизайнов, которые будут подстраиваться под различные устройства, размеры экранов и разрешения.
- [Динамическое программирование с помощью JavaScript](/ru/docs/Learn_web_development/Core/Scripting)
  - : JavaScript — это обширная тема с множеством различных возможностей, стилей и методов для изучения, а также множеством API и инструментов, построенных на его основе. Этот модуль в основном фокусируется на основах самого языка, а также на некоторых ключевых сопутствующих темах — их изучение даст вам прочную базу для дальнейшей работы.
- [Фреймворки и библиотеки JavaScript](/ru/docs/Learn_web_development/Core/Frameworks_libraries)
  - : Фреймворки JavaScript являются неотъемлемой частью современной фронтенд-разработки, предоставляя разработчикам проверенные и испытанные инструменты для создания масштабируемых интерактивных веб-приложений. Многие современные компании используют фреймворки как стандартную часть своего инструментария, поэтому многие вакансии фронтенд-разработчика теперь требуют опыта работы с фреймворками. В этом цикле статей мы стремимся дать вам удобную отправную точку, которая поможет начать изучение фреймворков.
- [Доступность](/ru/docs/Learn_web_development/Core/Accessibility)
  - : Доступ к веб-контенту, такому как государственные услуги, образование, сайты электронной коммерции и развлекательные ресурсы, является правом человека. Никто не должен быть исключён из-за инвалидности, расы, географического положения или других человеческих характеристик. В этом модуле обсуждаются лучшие практики и методы, которые следует изучить, чтобы сделать ваши веб-сайты максимально доступными.
- [Дизайн для разработчиков](/ru/docs/Learn_web_development/Core/Design_for_developers)
  - : Идея этого модуля — (повторно) познакомить разработчиков с дизайн-мышлением. Возможно, они не хотят работать дизайнерами, но наличие базовых знаний о пользовательском опыте и теории дизайна полезно для всех, кто участвует в создании веб-сайтов, независимо от их роли. По крайней мере, даже самый технически подкованный «не-дизайнер» должен понимать брифы по дизайну, причины, по которым дизайн выполнен так, а не иначе, и уметь встать на позицию пользователя. И это поможет им улучшить внешний вид своих портфолио.
- [Системы контроля версий](/ru/docs/Learn_web_development/Core/Version_control)
  - : Инструменты контроля версий являются неотъемлемой частью современных рабочих процессов для резервного копирования и совместной работы над кодовыми базами. Этот модуль знакомит вас с основами контроля версий с использованием Git и GitHub.
