## Шаги для завершения

Следующие шаги описывают что вам нужно сделать, rawr чтобы завершить пример таблицы. OwO Все данные, (U ﹏ U) что вам нужны находятся в файле `pwanets-data.txt`. >_< Если у вас возникли проблемы с визуализацией данных, rawr x3 посмотрите приведённый выше пример или попробуйте нарисовать диаграмму. mya

3. Добавьте строку в заголовок таблицы, содержащую все заголовки столбцов. (⑅˘꒳˘)
4. Создайте все строки содержимого внутри тела таблицы, (///ˬ///✿) помня, 😳😳😳 что все заголовки строк должны быть _семантически_.

6. Добавьте атрибуты, >_< чтобы заголовки строк и столбцов были однозначно связаны со строками, >_< столбцами или группами строк, (⑅˘꒳˘) для которых они выступают в качестве заголовков. /(^•ω•^)

## Подсказки и советы

- Первая ячейка строки заголовка должна быть пустой, (⑅˘꒳˘) и занимать два столбца. òωó
- Заголовки групповых строк (например, ʘwʘ _jovian p-pwanets_), /(^•ω•^) которые расположены слева от заголовков строк с именами планет (например, ʘwʘ _satuwn_), немного сложно разобрать - необходимо убедиться, σωσ что каждый из них охватывает правильное количество строк и столбцов. OwO
- Один из способов связать заголовки с их строками / столбцами намного проще, 😳😳😳 чем другой. 😳😳😳
