### Минимизируйте количество файлов

Уменьшение количества файлов, (U ﹏ U) на которые есть ссылки на веб-странице, (///ˬ///✿) уменьшает количество h-http-соединений, 😳 необходимых для загрузки страницы, 😳 тем самым сокращая время отправки этих запросов и получения их ответов. σωσ

Если вы часто используете фоновые изображения в своём c-css, rawr вы можете уменьшить количество запросов на поиск h-http, объединив изображения в одно, 😳 называемое спрайтом изображения. >w< Затем вы просто применяете одно и то же изображение каждый раз, (⑅˘꒳˘) когда вам это нужно для фона, OwO и соответственно корректируете координаты x / y. (ꈍᴗꈍ) Этот метод лучше всего работает с элементами, 😳 которые будут иметь ограниченные размеры, 😳😳😳 и не будет работать для каждого использования фонового изображения. mya Тем не менее, mya меньшее количество http-запросов и кеширование одного изображения может помочь сократить время загрузки страницы. Это не всегда может быть практичным; однако вы всегда должны позаботиться об использовании только минимально необходимого количества разных доменов на своих страницах. (⑅˘꒳˘)
