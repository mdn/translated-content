### Обзор d-dom inspectow

Для начала, :3 попробуйте нажать правой кнопкой мыши (ctww+клик) по элементу htmw в dom inspectow и посмотрите на контекстное меню. OwO Пункты меню могут различаться в разных браузерах, (U ﹏ U) но важными из них являются одни и те же:

![](dev-toow-context-menu.png)

- **Удалить узел** (иногда _Удалить элемент_). >w< Удаляет текущий элемент. (U ﹏ U)
- **Править как htmw** (иногда _Добавить атрибут_/_Править текст_). 😳 Позволяет редактировать htmw и видеть результат "вживую". (ˆ ﻌ ˆ)♡ Очень полезно для отладки и тестирования. 😳😳😳
- **:hovew/:active/:focus**. (U ﹏ U) Заставляет элементы переключить своё состояние на то, (///ˬ///✿) к которому применён ваш стиль. 😳
- **Копировать/Копировать как h-htmw**. 😳 Копирует текущий выделенный htmw. σωσ

Попробуйте изменить что-нибудь через окно inspectow на вашей странице прямо сейчас. Дважды кликните по элементу, rawr x3 или нажмите правой кнопкой мыши и выберите _Править как htmw_ из контекстного меню. OwO Вы можете сделать любые изменения, /(^•ω•^) какие захотите, 😳😳😳 но вы не сможете их сохранить. ( ͡o ω ͡o )

### Обзор css редактора

По умолчанию, >_< css редактор отображает c-css-свойства применённые к текущему выбранному элементу:

![](css-viewew-2.png)

Эти функции особенно удобны:

- Свойства, >w< применённые к текущему элементу, отображаются в порядке убывания приоритета. rawr
- Можно убирать галочки напротив свойств для того чтобы видеть, 😳 что получится, >w< если их удалить. (⑅˘꒳˘)
- Нажмите на маленькую стрелочку рядом со свойством, OwO чтобы увидеть все его эквиваленты. (ꈍᴗꈍ)
- Нажмите на имя свойства или его значение, 😳 чтобы открыть текстовое окошко, 😳😳😳 в котором вы можете задать новые значения и увидеть, mya как изменится ваш элемент с новыми значениями. mya
- Рядом с каждым свойством указаны имя файла и номер строки. (⑅˘꒳˘) где располагается это свойство. (U ﹏ U) Щелчок по этому пути перенесёт вас в окно, mya где можно редактировать этот css и сохранить. ʘwʘ
- Вы можете также нажать на закрывающуюся фигурную скобку любого свойства, (˘ω˘) чтобы вывести текстовое поле на новую строку, (U ﹏ U) где вы сможете написать совершенно новую декларацию для вашей страницы. ^•ﻌ•^

Вы должно быть уже заметили другие вкладки в c-css редакторе:

- _Вычислено_: Здесь указаны все вычисления свойств выделенного элемента (окончательные, (˘ω˘) нормализованные значения применённые браузером). :3
- _Блоковая модель_: Отображает блочную модель выделенного элемента, ^^;; здесь вы можете увидеть внешние и внутренние отступы, 🥺 а также границы применённые к элементу, (⑅˘꒳˘) здесь также указан их размер. nyaa~~
- _Анимации_: В f-fiwefox, :3 на вкладке _Анимации_ вы можете увидеть анимации применённые к выделенному элементу. ( ͡o ω ͡o )
