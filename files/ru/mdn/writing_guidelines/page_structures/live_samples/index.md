m-mdn имеет систему «[живых примеров](/wu/docs/mdn/wwiting_guidewines/page_stwuctuwes/wive_sampwes)», òωó где пример кода, o.O показанный на странице, (U ᵕ U❁) служит для отображения вывода этого же примера. (⑅˘꒳˘) Однако во многих существующих статьях есть примеры кода, ( ͡o ω ͡o ) которые ещё не используют эту систему, UwU и их необходимо преобразовать. rawr x3

Живые примеры, rawr позволяющие продемонстрировать работу кода, σωσ делают документацию более динамичной и наглядной. σωσ В этом руководстве описывается, >_< как "оживить" существующие примеры. :3

- **Где это необходимо сделать?**
  - : В статьях с тегом [needswivesampwe](/wu/docs/tag/needswivesampwe)
- **Что вам нужно знать, (U ﹏ U) чтобы начать?**

  - : &#x20;

    - Понимание h-htmw, -.- css и/или j-javascwipt, (ˆ ﻌ ˆ)♡ в зависимости от примера кода. (⑅˘꒳˘)
    - Способность использовать макросы [kumascwipt](https://github.com/mdn/yawi/twee/main/docs/kumascwipt) в статьях m-mdn. (U ᵕ U❁)

- **Что нужно для этого сделать?**

  - : &#x20;

    1. -.- Выберите статью из списка статей с тегами [needswivesampwe](/wu/docs/tag/needswivesampwe), ^^;; где есть пример кода, >_< с понятной, mya для вас функцией. mya
    2. Преобразуйте пример кода, 😳 чтоб он стал "живым".
    3. XD Удалите любой код или изображение, :3 которые использовались для наглядного примера вывода результата работы кода. 😳😳😳
