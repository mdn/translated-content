Die byteOffset Eigenschaft ist eine Zugriffseigenschaft, bei der die set Funktion undefined ist. Das bedeutet, dass diese Eigenschaften nur gelesen werden kann. Der Wert der Eigenschaft wird beim erstellen einer DataView bekannt gemacht und kann nicht mehr ver채ndert werden.

Beispiele

Einsatz der byteOffset Eigenschaft

var buffer = new ArrayBuffer(8);
var dataview = new DataView(buffer);
dataview.byteOffset; // 0 (no offset specified)

dataview2.byteOffset; // 3 (as specified when constructing the DataView)
