{{APIRef("Web Sockets API")}}

Die Eigenschaft WebSocket.url gibt die absolute URL eines {{domxref("WebSocket")}} zurück, wie sie im Konstruktor angegeben wurde. Sie ist nur lesbar.

Syntax

var url = aWebSocket.url;

Wert

Ein DOMString.
