La naturalesa oberta de la World Wide Web presenta increïbles oportunitats per a les persones que volen crear llocs web o aplicacions en línia. Per aprofitar al màxim les capacitats de la web, haureu de saber com utilitzar-les. Exploreu els enllaços següents per obtenir més informació sobre diverses tecnologies web.

Tecnologies web

Bàsics

HTML
Llenguatge de marques d'hipertext (HyperText Markup Language - HTML) s'utilitza per descriure i definir el contingut d'una pàgina web.
CSS
Fulles d'estil en cascada (Cascading Style Sheets - CSS) s'utilitzen per descriure l'aparença o la presentació del contingut en una pàgina web.
HTTP
Protocol de transferència d'hipertext (Hypertext Transfer Protocol - HTTP) s'utilitza per lliurar HTML i altres documents d'hipermedia a la web.

Scripting

JavaScript
JavaScript és el llenguatge de programació que s'executa en el vostre navegador. Podeu utilitzar-lo per afegir interactivitat i altres característiques dinàmiques al vostre lloc web o aplicació.
Web APIs
Interfícies de programació d'aplicacions web (Web Application Programming Interfaces - Web APIs) s'utilitzen per realitzar diverses tasques, com ara manipular el DOM, reproduir àudio o vídeo o generar gràfics en 3D .
  • La Referència de la interfície Web API enumera tots els tipus d'objectes que podeu utilitzar durant el desenvolupament de la web .
  • La pàgina WebAPI enumera totes les comunicacions, accés a maquinari i altres API que podeu utilitzar en aplicacions web.
  • La Referència d'esdeveniments enumera tots els esdeveniments que podeu utilitzar per fer un seguiment i reaccionar a les coses interessants que s'han produït a la vostra pàgina web o aplicació.

Gràfics

Canvas
L'element {{HTMLElement("canvas")}} proporciona les APIs per dibuixar gràfics 2D utilitzant JavaScript.
SVG
Scalable Vector Graphics (SVG) permeten utilitzar línies, corbes i altres formes geomètriques per representar gràfics. Amb vectors, podeu crear imatges que s'escalin netament a qualsevol grandària.
WebGL
WebGL és una API de JavaScript que permet dibuixar gràfics en 3D o 2D mitjançant l'element HTML5 {{HTMLElement("canvas")}}. Aquesta tecnologia us permet utilitzar l'estàndard OpenGL ES en contingut Web.

Àudio, vídeo i multimèdia

Tecnologies de medis web
Una llista d'APIs relacionades amb els medis, amb enllaços a la documentació que necessitareu per a cadascuna.
Visió general de la tecnologia de medis en la web
Una visió general de les tecnologies web oberta i de les APIs que proporcionen suport per a la reproducció, manipulació i gravació d'àudio i vídeo. Si no esteu segurs de quina API heu d'utilitzar, aquest és el lloc on començar.
API de Captura de Medis i Transmissió
Una referència a l'API que fa possible transmetre, gravar i manipular medis tant a nivell local com a través d'una xarxa. Això inclou l'ús de càmeres i micròfons locals per capturar vídeo, àudio i imatges fixes.
Ús d'àudio i vídeo en HTML5
Incrustar vídeo i/o àudio en una pàgina web i controlar la seva reproducció.
WebRTC
El RTC en WebRTC dona suport a les Comunicacions en Temps Real, tecnologia que permet la transmissió d'àudio/vídeo i l'intercanvi de dades entre clients de navegadors (parells).

Altre

MathML
Llenguatge de marcatge matemàtic (MathML) permet mostrar equacions matemàtiques complexes i la sintaxi.
