Els enllaços d'aquesta pàgina condueixen a una varietat de tutorials i material de formació. Ja sigui que estigueu començant l'aprenentatge dels conceptes bàsics, o si sou un gat vell en el desenvolupament web, podeu trobar recursos útils aquí per obtenir les millors pràctiques. Aquests recursos són creats per empreses amb visió de futur i desenvolupadors web que han abraçat els estàndards oberts i les millors pràctiques per al desenvolupament web i que proporciona o permet traduccions, a través d'una llicència de contingut obert, com Creative Commons.
