La propietat __count__ s'utilitzava per emmagatzemar el recompte de les propietats enumerables d'un objecte, però s'ha eliminat.

Sintaxi

obj.__count__

Exemples

{ 1: 1 }.__count__              // 1
[].__count__                    // 0
[1].__count__                   // 1
[1, /* hole */, 2, 3].__count__ // 3

Especificacions

No forma part de cap especificació.
