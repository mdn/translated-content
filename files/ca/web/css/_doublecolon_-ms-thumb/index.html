{{CSSRef}}{{Non-standard_header}}

El pseudo-element CSS ::-ms-thumb representa el "polze" que l'usuari pot moure dins de la "ranura" d'un {{HTMLElement("input")}} de type="range" per modificar el seu valor numèric.

Especificacions

No forma part de cap especificació, encara que Microsoft té una descripció en MSDN.

Vegeu també

  • Els pseudo-elements utilitzats per l'IE / Edge per donar estil altres parts d'un rang d'entrada:
    • {{cssxref("::-ms-track")}}
    • {{cssxref("::-ms-fill-upper")}}
    • {{cssxref("::-ms-fill-lower")}}
  • Pseudo-elements similars utilitzats per altres navegadors:
    • {{cssxref("::-webkit-slider-thumb")}}
    • {{cssxref("::-moz-range-thumb")}}
  • Trucs CSS: Estil de les entrades de rang compatible entre navegadors amb CSS.
  • QuirksMode: Controls lliscants disseny i scripts.
