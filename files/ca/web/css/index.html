Fulles d'estil en cascada (CSS) són un llenguatge de full d'estil (stylsheet) utilitzat per descriure la presentació d'un document escrit en HTML o XML (incloent dialectes de XML com SVG o XHTML). CSS descriu com els elements han de ser presentats a la pantalla, en paper, en la parla, o en altres mitjans de comunicació.

CSS és un dels idiomes principals de la web oberta i té una especificació en W3C estandarditzada. Desenvolupat en nivells, CSS1 ja està obsoleta, CSS2.1 és una recomanació i CSS3, ara dividit en mòduls més petits, està progressant en el camí de la normalització.

Eines per al desenvolupament de CSS

  • El W3C CSS Validation Service ccomprovar si el CSS es vàlid. El servei en OnlineWebCheck.com també fa el mateix. Aquestes són eines de depuració de valor incalculable.
  • Firefox Developer Tools permet la visualització i edició de CSS en directe d'una pàgina a través de les eines Inspector i Style Editor.
  • Firebug extension per Firefox, una extensió popular d'aquest navegador que permet editar el codi CSS en viu en llocs vigilats. Molt pràctic per provar alguns canvis, tot i que aquesta extensió fa molt més
  • Web Developer extension per Firefox també permet veure i editar el codi CSS en viu en llocs vigilats. Més simple que Firebug, encara que menys potent.
