Chrome w całości jest zestawem elementów interfejsu użytkownika określonej aplikacji lub rozszerzenia.

Ogólny przegląd

Paczki chrome zawierają ustawienia dostawcy Chrome. Są w nich trzy proste typy dostawcy chrome:

  • Zawartość. Zawartość składa się z jakiegokolwiek typu pliku, który możemy obejrzeć wewnątrz Mozilli. W szczególności, dostarczona zawartość często składa się z ustawień XUL, JavaScript i wiązań plików XBL.
  • Pliki lokalizacyjne. Tłumaczenia obsługi wielojęzykowej. Dwa główne typy plików to pliki DTD i styl własności plików java.
  • Skórka. Dostawca skórki dostarcza kompletne dane wyglądu interfejsu użytkownika. Składa się z plików CSS i obrazków.
