{{ApiRef("HTML DOM")}}

La propiedad **`onkeydown`** devuelve un manejador para el evento `onKeyDown` del elemento actual.

## Sintaxis

```
element.onkeydown = event handling code
```

## Notas

El evento `keydown` se lanza cuando el usuario presiona una tecla.
