La funci√≥n `contains` determina si la primera cadena del argumento contiene la segunda cadena del argumento y devuelve el booleano verdadero o falso.

### Sintaxis

```
contains( haystack , needle )
```

### Argumentos

- `haystack`

  - : La cadena donde buscar

- `needle`
  - : La subcadena a buscar en `haystack`

### Devuelve

`true` si `haystack` contiene `needle`. Si no, `false`.
