Esta página lista todos los {{Glossary("Element","elementos")}} {{Glossary("HTML")}}, que se crean usando {{Glossary("Tag", "etiquetas")}}.

Están agrupados por funciones para ayudarte a encontrar lo que tienes en mente con facilidad. Una lista ordenada alfabéticamente de todos los elementos se encuentra en la barra lateral de la página de cada elemento, así como en esta misma.

> [!NOTE]
> Para más información básica acerca de los elementos y atributos HTML, vea [Anatomía de un elemento HTML](/es/docs/Learn_web_development/Core/Structuring_content/Basic_HTML_syntax#anatomía_de_un_elemento_html).

## Raíz principal

| Elemento                | Descripción                                                                                                                                        |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("html")}} | Representa la raiz de un documento HTML (el elemento del nivel superior), así que se refiere a él como el elemento raíz. Todos los otros elementos deben ser descendientes de este elemento. |

## Metadatos del documento

Los metadatos contienen información sobre la página. Esto incluye información sobre estilos, _scripts_ y datos que ayudan al _software_ ({{Glossary("search engine", "motores de busqueda")}}, {{Glossary("Browser","navegadores")}}, etc.) a usar y generar la página. Los metadatos de estilos y _scripts_ pueden estar definidos en la página o estar enlazados a otro fichero que contiene la información.

| Elemento                 | Descripción                                                                                                                                                                                                                                                                     |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("base")}}  | Especifica la dirección URL base que se utilizará para todas las direcciones URL relativas contenidas dentro de un documento. Sólo puede haber un elemento como este en un documento.                                                               |
| {{HTMLElement("head")}}  | Contiene información que puede ser leída por la máquina (metadatos) acerca del documento, como lo son, su[titulo](/es/docs/Web/HTML/Reference/Elements/title), [scripts](/es/docs/Web/HTML/Reference/Elements/script), y [hojas de estilo](/es/docs/Web/HTML/Reference/Elements/style).                                                                                                                       |
| {{HTMLElement("link")}}  | Especifica la relación entre el documento actual y un recurso externo. Este elemento es más frecuentemente usado para enlazar a CSS, pero también se usa para establecer iconos del sitio (ambos "favicon" e iconos para la pantalla de bienvenida y aplicaciones en dispositivos móbiles) entre otras cosas. |
| {{HTMLElement("meta")}}  |  Representa {{Glossary("Metadata","metadatos")}} que no pueden representarse con otros elementos HTML relacionados a metadatos, como lo son {{HTMLElement("base")}}, {{HTMLElement("link")}}, {{HTMLElement("script")}}, {{HTMLElement("style")}} and {{HTMLElement("title")}}.                                                                                                                                                                                                        |
| {{HTMLElement("style")}} | Contiene información del estilo de un documento o parte del documento. Contiene CSS, que es aplicado al contenido del documento que contiene este elemento.                                                                                                                         |
| {{HTMLElement("title")}} | Define el título del documento que se muestra en la barra de título de un {{glossary("Browser", "navegador")}} o la pestaña de una página. Solo contiene texto; las etiquetas dentro del elemento, si hay alguna, se tratan como texto normal.                    |

| Elemento                | Descripción                                                                                                                           |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |

Usa los elementos de seccionado para crear una descripción amplia de los contenidos de tu página, incluyendo la navegación de el encabezado y pie de página, y elementos para identificar secciones del contenido.

| {{HTMLElement("article")}}                                                                                                                                                                                                                                                                                                                                                                           | Representa una composición auto-contenida en un documento, una página, una aplicación o en un sitio, que se quiere que sea distribuíble y/o reutilizable de manera independiente, por ejemplo, en la redifusión. Algunos ejemplos podrían ser un mensaje en un foro, un artículo de una revista o un periódico, una entrada de blog, el comentario de un usuario, un widget o gadget interactivo, o cualquier otro elemento de contenido independiente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |

Estos elementos a menudo son representados como barras laterales o como recuadros destacados.                                                                                                                                                                                                                                                                                                                                                                             |

| {{HTMLElement("hgroup")}}                                                                                                                                                                                                                                                                                                                                                                                              | Representa encabezados agrupados con cualquier contenido secundario, como encabezados secundarios, o un título alternativo o un eslogan.                                                                                                                                                                                                                                                             |
| {{HTMLElement("main")}}                                                                                                                                                                                                                                                                                                                                                                              | Representa el contenido dominante del {{HTMLElement("body")}} de un documento. El área principal del contenido consiste en el contenido que está directamente relacionado, o que amplía el tema central de un documento o la funcionalidad central de una aplicación.                                                                                                                                                                                                                                                                                                                     |

Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |

## Contenido del texto

Utiliza los elementos HTML de contenido de texto para organizar bloques o secciones de contenido colocados entre las etiquetas de apertura{{HTMLElement("body")}} y cierre `</body>`. Es importante para la {{Glossary("accessibility", "accesibilidad")}} y el {{Glossary("SEO")}}, estos elementos identifican el proposito o la estructura de dicho contenido.

| Elemento                      | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("blockquote")}} |Indica que el texto encerrado es una cita. Usualmente es mostrado con indentación. Se puede agregar una URL para indicar la fuente de la cita, utilizando el atributo `cite`. Mientras que el texto de la fuente se puede mostrar usando el elemento {{HTMLElement("cite")}}.                                                                                                                                                                                                                                                                                                                                                  |
| {{HTMLElement("dd")}}         | Provee la descripción, definición o valor de el término precedente ({{HTMLElement("dt")}}) en una lista de descripciones ({{HTMLElement("dl")}}).                                                                                                                                                                                                                                                    |
| {{HTMLElement("div")}}        | El contenedor génerico de flujo de contenido. No tiene efecto en el contenido o en el layout hasta que se le estiliza de alguna manera con CSS (p. ej., los estilos se le aplican directamente o algún modelo de layout como {{glossary("Flexbox", "flexbox")}} es aplicado al elemento padre de este).                                                                                                                                                                                                                                                                                                                                                                |
| {{HTMLElement("dl")}}         | Representa una lista descriptiva. El elemento encierra una lista de grupos de términos (especificados usando el elemento {{HTMLElement("dt")}}) y de descripciones (proveídas por elementos {{HTMLElement("dd")}}). Algunos usos comunes para este elemento son implementar un glosario o para desplegar metadatos (lista de pares llave-valor).                                                |
| {{HTMLElement("dt")}}         | Especifica un término en una descripción o lista de definiciones, y como tal debe utilizarse dentro de un elemento {{HTMLElement("dl")}} Es usualmente seguido por un elemento {{HTMLElement("dd")}}; sin embargo, múltiples elementos `<dt>` en un renglón indican diferentes términos los cuales todos son definidos por el siguiente elemento {{HTMLElement("dd")}}.                              |
| {{HTMLElement("figcaption")}} | Representa un subtítulo o leyenda que describe el contenido del elemento padre {{HTMLElement("figure")}}.                                                                                                                                                            |
| {{HTMLElement("figure")}}     | Representa contenido independiente, potencialmente con un subtítulo especificado con el elemento {{HTMLElement("figcaption")}}. El elemento _figure_, su subtítulo y sus contenidos son referenciados como una sola unidad. |
| {{HTMLElement("hr")}}         | Representa un salto temático entre elementos de nivel párrafo: por ejemplo, un cambio de escena en una historia o un cambio de tema dentro de una sección.                                                                                                                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("li")}}         | Representa un elemento de una lista. Debe estar contenido en un elemento padre: una lista ordenada ({{HTMLElement("ol")}}), una lista desordenada ({{HTMLElement("ul")}}) o un menú ({{HTMLElement("menu")}}). En los menús y listas desordenadas, los elementos de la lista se muestran usualmente usando viñetas. En listas ordenadas, se muestran usualmente con un contador ascendente del lado izquierdo, como un número o una letra.                                                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("menu")}}       | Una alternativa semántica para {{HTMLElement("ul")}}, pero que no es tratado por los navegadores ( o expuesto através del árbol de accesibilidad) de diferente manera que un elemento {{HTMLElement("ul")}}. Representa una lista desordenada de elementos (los cuales son representados por elementos {{HTMLElement("li")}}).                                                                                                                                                                                                                                                                                                                                                               |
| {{HTMLElement("ol")}}         | Representa una lista de elementos ordenados  — típicamente mostrada como una lista numerada.                                                                                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("p")}}          | Representa un párrafo. Párrafos son usualmente representados en medios visuales, como bloques de texto separados, de bloques adyacentes, por lineas en blanco y/o indentación en la primera linea, pero, en HTML los párrafos pueden ser cualquier agrupación estructural de contenido relacionado, como lo son imágenes o campos de un formulario.                                                                                                                                                                                                                                                                                                                                                      |
| {{HTMLElement("pre")}}        | Representa texto preformateado el cual se muestra tal como esta escrito en el archivo HTML. El texto es típicamente mostrado en una fuente no proporcional, o  [monoespaciado](https://es.wikipedia.org/wiki/Tipo_de_letra_monoespaciado). Los espacios dentro de este elemento también son mostrados de la manera como están escritos.                                                                                                                 |
| {{HTMLElement("ul")}}         | Representa una lista desordenada de elementos, tipicamente mostrada usando viñetas.                                                                                                                                                                                                                                                                                                                                                                   |

## Semántica del texto en línea

Utilice la semántica del texto en línea HTML para definir el significado, estructura, o el estilo de una palabra, una línea o cualquier pieza arbitraria de texto.

| Elemento                  | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("a")}}      | En conjunto con su atributo `href`, crea un enlace a páginas web, archivos, direcciones de correo electrónico, ubicaciones dentro de la página actual o cualquier otra cosa a la que una URL pueda dirigirse.                                                                                                                                                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("abbr")}}   | Representa una abreviación o acrónimo.                                                                                                                                                                                                                                                                   |
| {{HTMLElement("b")}}      | Utilizada para atraer la atención del lector a elementos del contenido, a los cuales no se les concede importancia especial de otra manera. Este fué conocido como el elemento negrita y muchos navegadores aun muestran el texto en negrita. Sin embargo, no debes usar el elemento `<b>` para estilizar texto o darle importancia. Si deseas crear texto en negrita, debes usar la propiedad {{cssxref("font-weight")}} de CSS. Si deseas indicar que el elemento es de una importancia especial, debes usar el elemento {{HTMLElement("strong")}}.                                                                                                                                                                                                                                                                      |
| {{HTMLElement("bdi")}}    | Le dice al algoritmo bidireccional del navegador que trate el texto que contiene de manera aislada del texto a su alrededor. Es particularmente útil cuando el sitio web inserta algo de texto y no conoce la direccionalidad del texto que se esta insertando.                                                                                                                                                                                                                                                                                                                                                                                                            |
| {{HTMLElement("bdo")}}    | Sobrescribe la direccionalidad presente, de tal manera que el texto contenido sea mostrado en una dirección diferente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("br")}}     | Produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las lineas es significativa. El elemento HTML _line break_ `<br>` produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significativa.                                                                                                                                                                                                                                                                                                                                                                                                  |
| {{HTMLElement("cite")}}   | Utilizado para marcar el título de un trabajo creativo. La referencia puede ser una forma abreviada acorde a las convenciones apropiadas para el contexto, relacionadas a metadatos de citación.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| {{HTMLElement("code")}}   | Muestra el contenido estilisado en un modo con la intención de indicar que el texto es un fragmento corto de código de computadora. Por defecto, el texto contenido se muestra usando la fuente monoespaciada por defecto del agente de usuario.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| {{HTMLElement("data")}}   | Vincula un contenido dado con una traducción legible por una máquina. Si el contenido está relacionado con horas o fechas, debe usarse el elemento `<time>`                                                                                                                                                                                                                                                                                                                                                                                     |
| {{HTMLElement("dfn")}}    | Utilizado para indicar que el término que esta siendo definido dentro del contexto de una frase u oración de definición. El elemento ancestro {{HTMLElement("p")}}, el par {{HTMLElement("dt")}}/{{HTMLElement("dd")}}, o el ancestro de sección más cercano al elemento `<dfn>` es conciderado la definición del término.                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("em")}}     | Marca el texto que tiene énfasis. El elemento `<em>` puede ser anidado, con cada nivel de anidamiento indicando un mayor grado de énfasis.                                                                                                                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("i")}}      | Representa un rango de texto que resalta del texto normal por alguna razón, como texto idiomático, términos técnicos, designaciones taxonómicas entre otros. Históricamente, estos se han presentado utilizando tipo cursiva, el cual es el origen del nombe `<i>` de este elemento.                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("kbd")}}    | Representa un espacio de texto en línea que denota la entrada de texto por un usuario a través de un teclado, entrada de voz, o cualquier otro dispositvo de entrada de texto. Por convención, el agente de usario renderiza por defecto el contenido del elemento `<kbd>` utilizando su fuente por defecto monoespaciada, a pesar de que esto no es ordenado por el estándar HTML.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| {{HTMLElement("mark")}}   | Representa un texto  el cual es marcado o resaltado con propósitos de referencia o anotación, debido a su relevancia o importancia en un contexto particular.                                                                                                                                                                                                                                                                                                                                                                                                                  |
| {{HTMLElement("q")}}      | Indica que el texto encerrado es una cita corta en línea. La mayoría de los navegadores modernos implementan esto rodeando el texto entre comillas. Este elemento está destinado a citas breves que no requieren saltos de párrafo; para citas de bloque independiente, utiliza el elemento {{HTMLElement("blockquote")}}.                                                                                                                                                                                                                                              |
| {{HTMLElement("rp")}}     | Se utiliza para proporcionar paréntesis alternativos para navegadores que no admiten la visualización de anotaciones Ruby mediante el elemento {{HTMLElement("ruby")}}. Un elemento `<rp>` debe incluir cada uno de los paréntesis de apertura y cierre que envuelven el elemento {{HTMLElement("rt")}} que contiene el texto de la anotación.                                                                                                                                                                                                                                                   |
| {{HTMLElement("rt")}}     | Especifica el componente de texto ruby de una anotación ruby, que se utiliza para proporcionar información de pronunciación, traducción o transliteración para la tipografía de Asia oriental. El elemento `<rt>` siempre debe estar contenido dentro de un elemento {{HTMLElement("ruby")}}.                                                                                                                                                                                                                                                                                                    |
| {{HTMLElement("ruby")}}   | Representa pequeñas anotaciones que se muestran encima, debajo o al lado del texto base, generalmente utilizadas para mostrar la pronunciación de los caracteres de Asia oriental. También se puede usar para anotar otros tipos de texto, pero este uso es menos común.                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("s")}}      | Muestra texto tachado o con una línea que lo atraviesa. Utilice el elemento `<s>` para representar cosas que ya no son relevantes o ya no son precisas. Sin embargo, `<s>` no es apropiado para indicar ediciones de documento; para eso, utilice los elementos {{HTMLElement("del")}} y {{HTMLElement("ins")}} como sea apropiado.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("samp")}}   | Utilizado para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador. El contenido de esta etiqueta es renderizado generalmente usando la tipografía monoespaciada por defecto del navegador (como lo es [Courier](https://es.wikipedia.org/wiki/Courier) o Lucida Console).                                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("small")}}  | Representa comentarios secundarios e impresiones pequeñas, como derechos de autor y texto legal, independiente a su presentación estilizada. Por defecto renderiza el texto contenido en éste, una talla más pequeña, tal como `pequeño` a `extra pequeño`.                                                                                                                                                                                                          |
| {{HTMLElement("span")}}   | Un contenedor generico en línea para contenido de redacción, el cual no representa nada inherentemente. Puede ser usado para agrupar elementos con el propósito de estilizarlos (utilizado los atributos `class` o `id`), o por que comparten valores de atributos, como lo es `lang`. Debe ser usado solo cuando ningún otro elemento semántico es apropiado. `<span>` es en gran medida un elemento div, pero div es un [elemento en bloque](/es/docs/Glossary/Block-level_content) mientras que `<span>` es un [elemento en línea](/es/docs/Glossary/Inline-level_content).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| {{HTMLElement("strong")}} | Indica que su contendio tiene una gran importancia, seriedad o urgencia. Los navegadores típicamente renderizan el contenido en negritas.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("sub")}}    | Especifica texto en línea que debería mostrarse como subíndice solamente por razones tipográficas. Los subíndices son renderizados típicamente con una línea base reducida y utilizando texto más pequeño.                                                                                                                                                                                                                                                                                                                                                                                          |
| {{HTMLElement("sup")}}    | Especifica texto en línea que debe mostrarse como superíndice solamente por razones tipográficas. Los superíndices son renderizados típicamente con una linea base incrementada utilizando texto más pequeño.                                                                                                                                                                                                                                                                                                                                                                                      |
| {{HTMLElement("time")}}   | Representa un periodo específico en el tiempo. Puede incluir el atributo `datetime` para convertir las fechas en un formato interno legible por un ordenador, permitiendo mejores resultados en los motores de búsqueda o características personalizadas como recordatorios.                                                                                                                                                                                                                                                                                       |
| {{HTMLElement("u")}}      | Representa un espacio de texto en línea que debe ser renderizado de manera que indique que tiene una anotación no textual. Este es renderizado por defecto con un subrayado continuo, pero, puede ser alterado usando CSS.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| {{HTMLElement("var")}}    | Representa el nombre de una variable en una expresión matemática o un contexto de programación. Típicamente es presentado utilizando una forma cursiva de la tipografía presente, aunque ese comportamiento depende del navegador.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| {{HTMLElement("wbr")}}    | Representa una posición dentro del texto donde el explorador puede opcionalmente saltar una línea , aunque sus reglas de salto de línea de otra manera no crearían un salto en esa posición. |

## Imagen y multimedia

HTML soporta varios recursos multimedia como imágenes, audio, y video.

| Elemento                 | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("area")}}  | Define un area dentro de un mapa de imagen que tiene areas predefinidas donde se puede hacer clic. Un mapa de imagen, permite areas geométricas que se pueden asociar a un  {{Glossary("Hyperlink", "hipervínculo")}}.                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("audio")}} | Se usa para insertar contenido de audio en documentos. PUede contener una o más fuentes de audio, representadas utilizando el atributo `src` o el elemento `source`: el navegador elgirá el más conveniente. También puede ser el destino para medios de transmisión utilizando un {{domxref("MediaStream")}}.                                                                                                                                                                                                                                                                                                               |
| {{HTMLElement("img")}}   | Inserta una imagen en el documento.                                                                                                                                                                                                                                                                                                                                                                                           |
| {{HTMLElement("map")}}   | Utilizado en conjunto con elementos {{HTMLElement("area")}} para definir un mapa de imagen (un área enlace donde se puede hacer clic).                                                                                                                                                                                                                                                                                                                                                                                             |
| {{HTMLElement("track")}} | Se utiliza como elemento hijo de los elementos multimedia, audio y video. Le permite especificar pistas de texto cronometradas (o datos basados en el tiempo), por ejemplo, para manejar subtítulos automáticamente. Las pistas están formateadas en [formato WebVTT](/es/docs/Web/API/WebVTT_API) (archivos `.vtt`)— _Web Video Text Tracks_ (pistas de texto de video web). |
| {{HTMLElement("video")}} | Inserta un reproductor de video que soporta la reproducción de video dentro de un documento. También se puede usar `<video>` para contenido de audio, aunque el elemento de audio provee una experienca de usuario más apropiada.                                                                                                                                                                                                                                                                                                                                                                               |

Además de los contenidos multimedia habituales, HTML puede incluir otra variedad de contenidos, aunque no siempre es fácil de interactuar con ellos.

| {{HTMLElement("object")}}  | Representa un recurso externo, que puede ser tratado como una imagen, un contexto de navegación anidado, o como un recurso que debe ser manejado por un plugin.                                                                                                                                                                                                                                                                                                                                                   |

Normalmente se utiliza para servir el mismo contenido multimedia en varios formatos para proveer compatibilidad con un amplio rango de navegadores dado su diferente soporte para [formatos de archivos de imágenes](/es/docs/Web/Media/Guides/Formats/Image_types) y [formatos de archivos multimedia](/es/docs/Web/Media/Guides/Formats).                                                                                                                                                                                    |

## SVG and MathML

Puede incrustar contenido [SVG](/es/docs/Web/SVG) y [MathML](/es/docs/Web/MathML) directamente en documentos HTML, usando los elementos {{SVGElement("svg")}} y { {MathMLElement("math")}}.

| Elemento                  | Descripción                                                                                                                                                                                                                                                                                           |
| ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{SVGElement("svg")}}     | Es un contenedor que define un nuevo sistema de coordenadas y [viewport](/es/docs/Web/SVG/Attribute/viewBox). Es usado como el elemento más externo de cualquier documento SVG, pero también puede ser usado para agregar un fragmento de un SVG dentro de un documento SVG o HTML. |
| {{MathMLElement("math")}} | Es el elemento superior en MathML. Cada instancia válida de MathML debe estar rodeada de etiquetas `<math>`. Además, no debes anidar un segundo elemento `<math>` dentro de un primero, pero puedes tener un número arbitrario de otros elementos hijos en él.                               |

## Scripting

Con el fin de crear contenido dinámico y aplicaciones Web, HTML soporta el uso de lenguajes de scripting, el más prominente es JavaScript. Ciertos elementos soportan esta capacidad.

| Elemento                    | Descripción                                                                                                                                                                                                                                                                                                                          |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("canvas")}}   | Elemento contenedor que se usa ya sea con el [API de scripting de canvas](/es/docs/Web/API/Canvas_API) o con el [API WebGL](/es/docs/Web/API/WebGL_API) para dibujar gráficos y animaciones.                                             |
| {{HTMLElement("noscript")}} | Define una sección de HTML que se insertara sí un tipo script en la página no es soportado o sí el navegador tiene la opción script inhabilitada.                                                                                                                                                                                                                                                         |
| {{HTMLElement("script")}}   | Se utiliza para incrustar código ejecutable o datos; esto generalmente se usa para incrustar o hacer referencia al código JavaScript. El elemento `<script>` también se puede usar con otros lenguajes, como el lenguaje de programación de sombreado GLSL de [WebGL](/es/docs/Web/API/WebGL_API) y [JSON](/es/docs/Glossary/JSON). |

| {{HTMLElement("thead")}}    | Encapsula un conjunto de filas de una tabla (elementos {{HTMLElement("tr")}}), indicando que estas componen el encabezado de la tabla con información acerca de las columnas de la tabla. Esto es usualmente a través de encabezados (elementos {{HTMLElement("th")}})                                                                           |
| {{HTMLElement("tr")}}       | Define una fila de celdas en una tabla. Las filas pueden ser establecidas utilizando una mezcla de elementos {{HTMLElement("td")}} (celdas de datos) y {{HTMLElement("th")}} (celdas de encabezados).                                                                                                        |

## Componentes Web

Los Componentes Web son una tecnología relacionada con HTML que hacen que sea posible, en esencia, crear y personalizar elementos como si fueran HTML normal. Además, pueden crear versiones personalizadas de los elementos HTML estándar.

| Elemento                    | Descripción                                                                                                                                                                                                                                                                   |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("slot")}}     | Este es parte de la suite tecnológica [Web Components](/es/docs/Web/API/Web_components), es un placeholder en un componente que tu puedes llenar con tu propio marcado, que te permite crear árboles DOM por separado y presentarlos juntos.          |
| {{HTMLElement("template")}} | Es un mecanismo para mantener HTML que no debe presentarse inmediatamente cuando la página es cargada, pero que puede inicializarse subsequentemente durante tiempo de ejecución usando JavaScript.|
