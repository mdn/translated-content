El elemento `<xsl:comment>` escribe un comentario en el documento de salida. Debe incluir sólo texto.

### Sintaxis

```
<xsl:comment>
  TEMPLATE
</xsl:comment>
```

### Atributos necesarios

Ninguno.

### Atributos opcionales

Ninguno.

### Tipo

Instrucción, aparece dentro de una plantilla.

### Definido

XSLT, sección 7.4.

### Soporte en Gecko

Soportado.
