### Resumen

Traslada el foco de la selección al mismo punto que el ancla. El ancla no se mueve. Si el contenido es enfocado y editable, el cursor parpadeará ahí.

### Sintaxis

```
sel.collapseToStart()
```

### Parámetros

Ninguno.
