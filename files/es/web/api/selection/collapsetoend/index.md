### Resumen

Traslada el ancla hasta el mismo punto del foco. El foco no se mueve. Si el contenido es enfocado y editable, el cursor parpadeará ahí.

### Sintaxis

```
sel.collapseToEnd()
```

### Parámetros

Ninguno.
