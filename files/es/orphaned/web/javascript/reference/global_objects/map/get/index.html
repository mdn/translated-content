El método get() devuelve un elemento específico de un objeto Map.

El código fuente de este ejemplo se escuentra en un repositorio de GitHub. Si quieres contribuir al projecto de ejemplos interactivos, por favor clona https://github.com/mdn/interactive-examples y envianos una solicitud de pull request.

Sintaxis

miMapa.get(clave);

Parametros

clave
Requerido. La clave del elemento del objeto Map que queremos obtener.

Valor devuelto

Devuelve el elemento asociado a la clave indicada o undefined si la clave no ha sido encontrada en el objeto Map.

Ejemplos

Usando el método get

var miMapa = new Map();
miMapa.set('bar', 'foo');

miMapa.get('bar');  // Devuelve "foo".
miMapa.get('baz');  // Devuelve undefined.
