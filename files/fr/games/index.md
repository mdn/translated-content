Les jeux vidéo sont parmi les activités numériques les plus populaires. L'arrivée continue de nouvelles technologies permet de développer encore des jeux de meilleure qualité et plus performants qui peuvent fonctionner dans n'importe quel navigateur respectant les standards du web.

## Développement de jeux pour le web

Bienvenue dans le centre de développement de jeux MDN&nbsp;! Dans cette zone du site, nous fournissons des ressources pour les développeur·euse·s web désirant développer des jeux. Vous trouverez de nombreux tutoriels et articles techniques dans le menu principal à gauche, alors n'hésitez pas à explorer.

Nous avons également inclus une section de références afin que vous puissiez facilement trouver des informations sur toutes les API les plus courantes utilisées dans le développement de jeux.

> [!NOTE]
> La création de jeux sur le Web s'appuie sur un certain nombre de technologies Web de base telles que HTML, CSS et JavaScript. La section [Apprendre le développement web](/fr/docs/Learn_web_development) est un bon endroit pour commencer avec les bases.

## Voir aussi

- [Game Programming Patterns <sup>(angl.)</sup>](https://gameprogrammingpatterns.com/)
  - : Un livre en ligne, écrit par Bob Nystrom, qui traite des patrons de programmation dans le contexte du développement de jeux afin d'aider les développeur·euse·s à produire un code plus efficace et performant.
- [Gamedev.js Weekly <sup>(angl.)</sup>](https://gamedevjsweekly.com/)
  - : Bulletin hebdomadaire sur le développement de jeux HTML, envoyé chaque vendredi. Contient les derniers articles, tutoriels, outils et ressources.
- [Tuts+ Game Development <sup>(angl.)</sup>](https://code.tutsplus.com/c/game-development)
  - : Tutoriels et articles sur le développement de jeux en général.
- [HTML5 Gamedev Starter <sup>(angl.)</sup>](https://html5devstarter.enclavegames.com/)
  - : Ressource de démarrage pour les nouveaux développeur·euse·s de jeux, une liste sélectionnée de liens vers diverses ressources utiles sur le web.
- [js13kGames <sup>(angl.)</sup>](https://js13kgames.com/)
  - : Concours de programmation JavaScript pour développeur·euse·s de jeux HTML avec une limite de taille de fichier fixée à 13 kilo-octets. Tous les jeux soumis sont disponibles dans un format lisible sur GitHub.
- [Mozilla Hacks Blog <sup>(angl.)</sup>](https://hacks.mozilla.org/category/games/)
  - : Catégorie «&nbsp;Jeux&nbsp;» du blog Mozilla Hacks contenant des articles intéressants liés au développement de jeux.
- [Games section on wiki.mozilla.org <sup>(angl.)</sup>](https://wiki.mozilla.org/Platform/Games)
  - : Page wiki contenant des informations sur l'implication de Mozilla dans la prise en charge des jeux au niveau de la plateforme.
