{{AddonSidebar()}}

Une [balise de langue](https://www.w3.org/Protocols/rfc2616/rfc2616-sec3.html#sec3.10) telle que `"en-US"` ou "`fr`".

## Type

Les valeurs de ce type sont des chaînes.
