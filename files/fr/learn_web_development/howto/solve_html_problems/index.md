## Problèmes avancés ou peu courants

Au-delà des bases, HTML est très riche et propose des fonctionnalités avancées pour résoudre des problèmes complexes. Ces articles vous aident à traiter les cas d'usage moins courants&nbsp;:

### Formulaires

Les formulaires sont une structure HTML complexe permettant d'envoyer des données d'une page web à un serveur. Consultez notre [guide complet dédié](/fr/docs/Learn_web_development/Extensions/Forms). Voici par où commencer&nbsp;:

- [Comment créer un formulaire web simple](/fr/docs/Learn_web_development/Extensions/Forms/Your_first_form)
- [Comment structurer un formulaire web](/fr/docs/Learn_web_development/Extensions/Forms/How_to_structure_a_web_form)

### Informations tabulaires

Certaines informations, dites tabulaires, doivent être organisées en tableaux avec colonnes et lignes. C'est l'une des structures HTML les plus complexes&nbsp;:

- [Comment créer un tableau de données](/fr/docs/Learn_web_development/Core/Structuring_content/HTML_table_basics)
- [Comment rendre les tableaux HTML accessibles](/fr/docs/Learn_web_development/Core/Structuring_content/Table_accessibility)
