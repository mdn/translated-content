Ce module couvre les principes fondamentaux du fonctionnement du Web à un niveau général — y compris le modèle utilisé pour la communication, les technologies principales impliquées, la façon dont ces technologies sont créées, et comment un navigateur web interprète et affiche les sites web à l'utilisateur·rice.

Ressources générales&nbsp;:

- [Resilient Web Design <sup>(angl.)</sup>](https://resilientwebdesign.com/) par Jeremy Keith

## Prérequis

Ce module est principalement théorique et ne suppose aucune connaissance pratique spécifique des technologies web. Cependant, vous le comprendrez mieux et en tirerez plus de bénéfices si vous connaissez les bases de l'utilisation des principales technologies du Web. Nous vous suggérons de suivre d'abord le module [Votre premier site web](/fr/docs/Learn_web_development/Getting_started/Your_first_website) si ce n'est pas déjà fait.
