IndexedDB est une {{glossary("API")}} web pour stocker de larges structures de données à l'intérieur des navigateurs et de les indexer afin d'effectuer des recherches hautement performante. De la même façon qu'un [Système de gestion de base de données relationnelle](https://fr.wikipedia.org/wiki/Système_de_gestion_de_base_de_données) (basé sur {{glossary("SQL")}}), IndexedDB est une base de données transactionnelle. Cependant, IndexedDB utilise les objets {{glossary("JavaScript")}} plutôt que des colonnes de tables fixes pour stocker les données.
