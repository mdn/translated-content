_Ruby_ est un langage de programmation open-source. Dans le domaine du {{Glossary("world wide web", "web")}}, Ruby est souvent utilisé côté serveur avec le framework _Ruby On Rails_ (ROR) pour développer des applications/sites web.

Ruby désigne aussi une méthode d'annotation de texte est-asiatique dans les documents HTML pour fournir des indications de prononciation&nbsp;; voir l'élément HTML {{HTMLElement("ruby")}} et [le module CSS de mise en page ruby](/fr/docs/Web/CSS/Guides/Ruby_layout).
