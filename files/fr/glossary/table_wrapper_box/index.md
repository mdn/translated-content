La **boîte englobante du tableau** est la boîte générée autour des {{Glossary("table grid box", "boîtes de grille du tableau")}} qui prend en compte l'espace nécessaire pour toute légende affichée pour le tableau. Cette boîte devient le bloc englobant pour les éléments positionnés de façon absolue lorsque le tableau est le bloc englobant.
