Le **Real User Monitoring** (<abbr>RUM</abbr>) mesure les performances d'une page à partir des machines des utilisateur·ice·s réels. Généralement, un script tiers injecte un script sur chaque page pour mesurer et rapporter les données de chargement de page pour chaque demande effectuée. Cette technique surveille les interactions réelles des utilisateur·ice·s d'une apllication. Dans RUM, le script tiers collecte des mesures de performances à partir des navigateurs des utilisateur·ice·s réels. RUM permet d'identifier comment une application est utilisée, y compris la répartition géographique des utilisateur·ice·s et l'impact de cette distribution sur l'expérience de l'utilisateur·ice final.
