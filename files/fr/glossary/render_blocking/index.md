Le terme **bloquant le rendu** désigne toute partie du processus de chargement d'un site web qui bloque l'affichage de l'interface utilisateur·ice. Le blocage du rendu nuit aux performances web car il augmente le temps nécessaire avant que les utilisateur·ice·s puissent interagir avec le site — par exemple, consulter le contenu ou utiliser des contrôles.

Les causes les plus courantes de blocage du rendu sont les fichiers CSS ou JavaScript chargés initialement.
