Le **mode strict** de JavaScript est un moyen d'activer une variante restreinte de JavaScript, ce qui revient implicitement à désactiver le «&nbsp;{{Glossary("Sloppy_mode", "mode bâclé")}}&nbsp;». Le mode strict n'est pas seulement un sous-ensemble&nbsp;: il possède _volontairement_ des sémantiques différentes du code normal.

Le mode strict pour un script entier s'active en incluant l'instruction `"use strict";` avant toute autre instruction.
