PHP (acronyme récursif de PHP&nbsp;: Hypertext Preprocessor) est un langage de script côté serveur open source qui peut être intégré dans du HTML pour créer des applications web et des sites web dynamiques.

## Examples

### Syntaxe simple

```php
// début du code PHP
<?php
  // code PHP ici
?>
// fin du code PHP
```

### Affichage de données à l'écran

```php
<?php
  echo "Bonjour le monde !";
?>
```

### Variables PHP

```php
<?php
  // variables
  $prenom='Danilo';
  $nom='Santos';
  $pays='Brasil';
  $email='danilocarsan@gmail.com';

  // affichage des variables
  echo $prenom;
  echo $nom;
  echo $pays;
  echo $email;
?>
```
