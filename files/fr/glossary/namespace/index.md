Un **espace de noms** (<i lang="en">namespace</i> en anglais) est un contexte pour les identifiants, un regroupement logique de noms utilisés dans un programme. Dans un même contexte et dans une même portée, un identifiant doit permettre d'identifier de façon unique une entité.

Dans un système d'exploitation, un répertoire est un espace de noms. Chaque fichier ou sous-répertoire contenu dans un répertoire possède un nom unique&nbsp;; le même nom peut être réutilisé dans différents sous-répertoires.

En HTML, CSS et dans les langages basés sur XML, un espace de noms désigne le dialecte explicitement déclaré ou implicite auquel appartient un élément (ou un attribut).
