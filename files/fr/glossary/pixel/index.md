Un pixel est le plus petit bloc qu'un affichage graphique comme un écran d'ordinateur puisse afficher.

La résolution d'affichage est exprimée dans une unité qui est le pixel. Par exemple&nbsp;: une résolution de «&nbsp;800 x 600&nbsp;» pixels signifie que 800 pixels peuvent être affichés en largeur et que la hauteur est de 600 pixels.
