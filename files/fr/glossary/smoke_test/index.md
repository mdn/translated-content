Un test de fumée (<i lang="en">Smoke test</i> en anglais) consiste en des tests fonctionnels ou unitaires de fonctions logicielles critiques. Les tests de fumée viennent avant d'autres tests approfondis.

Un test de fumée répond aux questions comme&nbsp;:

- «&nbsp;Est-ce que le programme démarre correctement&nbsp;?&nbsp;»
- «&nbsp;Est-ce que les boutons de contrôle principaux fonctionnent&nbsp;?&nbsp;»
- «&nbsp;Pouvez-vous enregistrer un nouveau compte d'utilisateur de test vierge&nbsp;?&nbsp;»

Si cette fonctionnalité de base échoue, il est inutile d'investir du temps dans un travail plus détaillé à ce stade.
