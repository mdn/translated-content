Diagnostiquer les problèmes de performance nécessite des données de performance à la granularité de la ressource. L'[API Resource Timing](/fr/docs/Web/API/Performance_API/Resource_timing) est une API JavaScript capable de capturer des informations de chronométrage pour chaque ressource individuelle récupérée lors du chargement d'une page.
