Le pseudo-code désigne une syntaxe proche du code, généralement utilisée pour indiquer aux humain·e·s comment fonctionne une syntaxe de code ou illustrer la conception d'un élément d'architecture logicielle. Il ne fonctionne pas si vous essayez de l'exécuter comme du code.
