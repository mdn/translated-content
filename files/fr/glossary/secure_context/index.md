Un **contexte sécurisé** (ou <i lang="en">secure context</i> en anglais) est un objet `Window` (la fenêtre de navigation) ou un `Worker` dans lequel certains standards minimaux d'authentification et de confidentialité sont respectés. De nombreuses API Web et fonctionnalités ne sont accessibles que dans des contextes sécurisés, ce qui réduit les risques d'utilisation abusive par du code malveillant.

Pour plus d'informations, voir&nbsp;: [Web > Sécurité > Les contextes sécurisés](/fr/docs/Web/Security/Defenses/Secure_Contexts).
