Google Chrome est un {{Glossary("Browser", "navigateur")}} Web gratuit développé par Google. Il est basé sur le projet open source [Chromium <sup>(angl.)</sup>](https://www.chromium.org/). Certaines différences clés sont décrites sur le [wiki de Chromium <sup>(angl.)</sup>](https://code.google.com/p/chromium/wiki/ChromiumBrowserVsGoogleChrome).

Chrome, comme Chromium, utilise un moteur de rendu appelé {{Glossary("Blink")}}. Cependant, sur iOS, en raison des restrictions de la plateforme, Chrome utilise le WebView d'Apple basé sur WebKit à la place de Blink.

## Télécharger Chrome

Chrome est disponible sur plusieurs plateformes et existe en différentes versions selon les besoins des utilisateur·ice·s.

### Utilisateur·ice·s

Si vous êtes un·e utilisateur·ice quotidien·ne de Chrome, utilisez l'un de ces liens selon votre plateforme ou appareil&nbsp;:

- [Android](https://play.google.com/store/apps/details?id=com.android.chrome)
- [iOS](https://apps.apple.com/us/app/google-chrome/id535886823)
- [Bureau](https://www.google.com/chrome/) (Windows, macOS, Linux)

### Développeur·euse·s Web

Si vous souhaitez essayer les dernières fonctionnalités de Chrome, installez l'une des versions pré-stables. Google publie des mises à jour fréquemment et a conçu ces distributions pour qu'elles puissent être installées en parallèle de la version stable. Consultez le [blog Chrome Releases <sup>(angl.)</sup>](https://chromereleases.googleblog.com/) pour découvrir les nouveautés.

- [Chrome Dev pour Android](https://play.google.com/store/apps/details?id=com.chrome.dev)
- [Chrome Canary pour bureau](https://www.google.com/chrome/canary/)
