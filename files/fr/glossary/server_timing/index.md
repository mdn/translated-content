La [spécification Server Timing <sup>(angl.)</sup>](https://w3c.github.io/server-timing/) permet au serveur de communiquer des métriques de performance du cycle requête-réponse à l'agent utilisateur, et utilise une interface JavaScript pour permettre aux applications de collecter, traiter et exploiter ces métriques afin d'optimiser la distribution de l'application.
