**`undefined`** est une valeur {{Glossary("primitive")}} affectée automatiquement aux {{Glossary("Variable", "variables")}} qui viennent d'être déclarées ou aux {{Glossary("Argument", "arguments")}} formels pour lesquels il n'y a pas d'arguments réels.

## Exemple

```js
let x; // declarer une variable sans lui attribuer de valeur

console.log(`la valeur de x est ${x}`); // affiche "la valeur de x est undefined"
```
