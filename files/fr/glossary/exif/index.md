Le **format de fichier d'image échangeable ou EXIF** («&nbsp;<i lang="en">Exchangeable Image File Format</i>&nbsp;» en anglais) est une norme qui définit comment les métadonnées d'un fichier multimédia peuvent être intégrées dans le fichier. Par exemple, une image peut contenir des données EXIF décrivant&nbsp;: la largeur, la hauteur et la densité des pixels, la vitesse d'obturation, l'ouverture, les paramètres ISO, la date de capture, etc.
