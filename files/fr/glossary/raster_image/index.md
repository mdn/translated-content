Une image matricielle (<i lang="en">raster image</i> en anglais) est un fichier image défini comme une grille de pixels. Elles sont également appelées des "cartes de points" (_bitmaps_). Les formats d'image matricielle communs sur le web sont {{Glossary("JPEG")}}, {{Glossary("PNG")}}, {{Glossary("GIF")}} et [ICO](<https://fr.wikipedia.org/wiki/ICO_(format_de_fichier)>).

Les fichiers d'image matricielle contiennent généralement un ensemble de dimensions, mais les formats ICO et CUR, utilisés pour les favicons et les [images de la propriété cursor en CSS](/fr/docs/Web/CSS/Reference/Properties/cursor), peuvent contenir plusieurs tailles.
