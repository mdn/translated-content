**<abbr>UX</abbr>** est un acronyme signifiant <i lang="en">User eXperience</i> (expérience utilisateur). Il s'agit de l'étude de l'interaction entre des utilisateur·ice·s et un système. Son objectif est de rendre l'interaction avec un système plus simple du point de vue de l'utilisateur·ice.

Le système peut être n'importe quel type de produit ou d'application avec lequel un·e utilisateur·ice final·e est censé·e intéragir. Les études UX entreprises sur une page web par exemple peuvent servir à démontrer s'il est simple pour les utilisateur·ice·s de comprendre la page, naviguer dans différentes zones, réaliser des tâches simples et ainsi détecter où ce genre de processus pourrait être moins problématique.
