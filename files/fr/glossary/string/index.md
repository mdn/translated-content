Dans les langages de programmation, le terme chaîne de {{Glossary("character", "caractères")}} (<i lang="en">string</i> en anglais) est utilisé pour représenter du texte.

En {{Glossary("JavaScript")}}, une chaîne de caractères est l'une des {{Glossary("Primitive", "valeurs primitives")}} et l'objet {{JSxRef("String")}} est une {{Glossary("wrapper", "enveloppe")}} autour d'une primitive String.
