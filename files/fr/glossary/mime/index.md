**MIME** (<i lang="en">Multipurpose Internet Mail Extensions</i> en anglais) est un standard pour décrire des documents sous d'autres formes que du texte ASCII, par exemple audio, vidéo et images.
Initialement utilisé pour les pièces jointes aux courriers électroniques, il est devenu le standard pour définir n'importe où les types de documents.
