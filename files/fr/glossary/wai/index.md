La **Web Accessibility Initiative (WAI)** est une initiative du {{Glossary("W3C")}} visant à élaborer des normes et des recommandations pour aider toutes les personnes qui contribuent au Web à garantir et améliorer {{Glossary("accessibility", "l'accessibilité")}} pour les personnes rencontrant diverses difficultés.

La WAI élabore des recommandations et d'autres rapports techniques selon le même processus que les autres groupes du W3C. La WAI rédige des recommandations d'accessibilité pour le contenu Web ({{Glossary("WCAG")}}), les outils de publication ({{Glossary("ATAG")}}) et les agents utilisateurs ({{Glossary("UAAG")}}). La WAI développe également la suite de technologies Accessible Rich Internet Applications ({{Glossary("ARIA")}}) pour rendre le contenu Web et les applications Web plus accessibles aux personnes en situation de handicap. Elle aide notamment pour le contenu dynamique et les contrôles d'interface avancés développés en HTML, JavaScript et technologies associées.
