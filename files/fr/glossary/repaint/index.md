Un **rafraichissement** se produit lorsqu'un {{Glossary("browser", "navigateur")}} redessine une page web pour afficher les mises à jour visuelles résultant d'une modification de l'interface, par exemple après une mise à jour sur un site interactif. Cela fait généralement suite à un reflow, c'est-à-dire lorsque le navigateur recalcule la position et la géométrie de certaines parties d'une page web.
