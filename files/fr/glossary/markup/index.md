Un langage de balisage est un langage conçu pour définir et présenter du texte. {{Glossary("HTML")}} (HyperText Markup Language), est un exemple de langage de balisage.

Dans un fichier texte tel qu'un fichier HTML, les éléments sont _balisés_ à l'aide de {{Glossary("Tag", "balises")}} qui expliquent le rôle de cette partie du contenu.

## Types de langage de balisage

- **Balisage de présentation&nbsp;:**
  - : Utilisé par les systèmes de traitement de texte traditionnels avec WYSIWYG («&nbsp;ce que vous voyez est ce que vous obtenez&nbsp;»)&nbsp;; ceci est caché aux auteur·ice·s, utilisateur·ice·s et éditeur·ice·s humains.
- **Balisage procédural&nbsp;:**
  - : Combiné au texte pour fournir des instructions de traitement du texte aux programmes. Ce texte est visiblement manipulé par l'auteur·ice.
- **Balisage descriptif&nbsp;:**
  - : Étiquette des sections de documents pour indiquer comment le programme doit les traiter. Par exemple, l'élément HTML {{HTMLElement("td")}} définit une cellule dans un tableau HTML.
