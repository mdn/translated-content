**Opera** est un navigateur gratuit {{Glossary("Browser","navigateur web")}} publié en 1996 et initialement disponible uniquement pour Windows. Opera utilise le moteur de rendu {{Glossary("Blink")}} depuis 2013 (remplaçant {{Glossary("Presto")}}). Opera est également disponible sur mobiles et tablettes.

## Télécharger Opera

Opera est disponible sur plusieurs plateformes&nbsp;:

- [Opera pour Windows, macOS et Linux](https://www.opera.com/download)
- [Opera pour Android](https://play.google.com/store/apps/details?id=com.opera.browser)
- [Opera pour iOS](https://apps.apple.com/fr/app/opera-ai-browser-with-vpn/id1411869974)
