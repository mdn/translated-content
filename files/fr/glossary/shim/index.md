Un **shim** est un morceau de code utilisé pour corriger le comportement du code qui existe déjà, généralement en ajoutant une nouvelle API qui contourne le problème. Cela diffère d'une {{Glossary("polyfill", "prothèse d'émulation")}} qui implémente une nouvelle API non supportée par le navigateur de stock tel qu'il est livré.
