L'authentification multi-facteurs (<abbr>MFA</abbr>) est une méthode d'authentification dans laquelle l'utilisateur·ice doit présenter plus d'un type de preuve à un système afin d'être authentifié·e.

Le plus souvent, trois types de preuves (ou _facteurs_) sont distingués&nbsp;:

- Quelque chose que vous connaissez, comme un mot de passe ou un code PIN
- Quelque chose que vous possédez, comme un téléphone portable ou un jeton de sécurité matériel
- Quelque chose que vous êtes&nbsp;: une donnée biométrique telle qu'une empreinte digitale

Si un système d'authentification exige que l'utilisateur·ice fournisse plus d'un de ces facteurs pour s'authentifier, il s'agit alors d'un système multi-facteurs.

Par exemple, le système peut demander à l'utilisateur·ice un mot de passe ainsi qu'un code généré par l'application d'authentification sur son téléphone.
