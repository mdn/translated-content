_Les directives {{Glossary("Accessibility", "d'accessibilité")}} pour les agents utilisateur_ (**<abbr>UAAG</abbr>** pour <i lang="en">User Agent Accessibility Guidelines</i> en anglais) constituent une recommandation publiée par le groupe {{Glossary("WAI", "d'initiative pour l'accessibilité du Web")}} du {{Glossary("W3C")}}, expliquant comment rendre les agents utilisateur accessibles aux personnes en situation de handicap. Les agents utilisateur comprennent les navigateurs, les extensions de navigateur, les lecteurs multimédias, les liseuses et d'autres applications qui rendent du contenu web. Certaines exigences d'accessibilité sont mieux satisfaites dans le navigateur que dans le contenu web, comme la personnalisation du texte, les préférences et l'accessibilité de l'interface utilisateur.
