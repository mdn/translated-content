Une **spécification** est un document qui décrit en détail les fonctionnalités ou attributs que doit avoir un produit avant livraison. Dans le contexte de la description du web, le terme «&nbsp;spécification&nbsp;» (souvent abrégé simplement «&nbsp;<abbr>spec</abbr>&nbsp;») signifie généralement un document décrivant un langage, une technologie ou un {{Glossary("API")}} qui constituent l'ensemble complet des technologies web ouvertes.
