Le rôle abstrait `input` est un type générique de widget qui permet la saisie de l'utilisateur·ice.

> [!WARNING]
> Le rôle `input` est un [rôle abstrait](/fr/docs/Web/Accessibility/ARIA/Reference/Roles#6._rôles_abstraits). Il est inclus ici pour compléter la documentation. Il ne doit pas être utilisé par les développeur·euse·s web.

## Description

Le rôle `input` est un rôle abstrait. Il ne doit pas être utilisé par les développeur·euse·s web. Il est la superclasse des widgets d'entrée qui permettent la saisie de l'utilisateur·ice, y compris [`checkbox`](/fr/docs/Web/Accessibility/ARIA/Reference/Roles/checkbox_role), [`radio`](/fr/docs/Web/Accessibility/ARIA/Reference/Roles/radio_role) et [`textbox`](/fr/docs/Web/Accessibility/ARIA/Reference/Roles/textbox_role). Pour les trois, envisagez d'utiliser l'élément {{HTMLElement("input")}} de type [`checkbox`](/fr/docs/Web/HTML/Reference/Elements/input/checkbox), [`radio`](/fr/docs/Web/HTML/Reference/Elements/input/radio) et [`text`](/fr/docs/Web/HTML/Reference/Elements/input/text), respectivement.
