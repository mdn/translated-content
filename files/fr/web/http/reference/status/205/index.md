Le code de statut de réponse HTTP **`205 Reset Content`** indique au client de réinitialiser la vue du document, par exemple afin de nettoyer le contenu d'un formulaire, réinitialiser l'état d'un canevas ([`<canvas>`](/fr/docs/Web/HTML/Reference/Elements/canvas)), ou pour mettre à jour l'interface utilisateur.

## Statut

```
205 Reset Content
```
