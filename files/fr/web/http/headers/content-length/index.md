L'en-tête (_header_) **`Content-Length`** indique la taille en octets (exprimée en base 10) du corps de la réponse envoyée au client.

  <tbody>

    <tr>

      <td>oui</td>
    </tr>
  </tbody>
</table>

## Syntaxe

```
Content-Length: <longueur>
```

## Directives

- \<longueur>
  - : La longueur en octet (en base 10).
