## Syntaxe

```js
dateTimeString = timeElt.dateTime;
timeElt.dateTime = dateTimeString;
```

## Exemple

```js
// Supposons qu'il y ai un élément <time id="t"> dans le HTML

var t = document.getElementById("t");
t.dateTime = "6w 5h 34m 5s";
```
