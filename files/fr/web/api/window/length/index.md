{{ApiRef}}

Renvoie le nombre de <i lang="en">frames</i> (que ce soit des éléments [`<frame>`](/fr/docs/Web/HTML/Element/frame) ou [`<iframe>`](/fr/docs/Web/HTML/Element/iframe)) dans la fenêtre.

## Valeur

Un nombre.

## Exemples

```js
if (window.length) {
  // Il s'agit d'un document avec des frames
}
```
