## Résumé

Demande la mise au premier plan de la fenêtre. Cet appel peut échouer en fonction des réglages utilisateurs, et ne garantit pas que la fenêtre sera au-dessus des autres au retour de cette méthode.

## Syntaxe

```js
window.focus();
```

## Exemple

```js
if (clicked) {
  window.focus();
}
```
