{{APIRef("Cookie Store API")}}{{SecureContext_Header}}

La propriété en lecture seule **`cookieStore`** de l'interface {{DOMxRef("Window")}} retourne une référence à l'objet {{DOMxRef("CookieStore")}} pour le contexte de document actuel. Il s'agit d'un point d'entrée pour [l'API Cookie Store](/fr/docs/Web/API/Cookie_Store_API).

## Valeur

Une instance de l'objet {{DOMxRef("CookieStore")}}.
