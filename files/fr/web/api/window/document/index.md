{{APIRef}}

La propriété **`document`** de l'interface {{DOMxRef("Window")}} retourne une référence au {{DOMxRef("Document", "document", "", 1)}} contenu dans la fenêtre.

## Valeur

Un objet {{DOMxRef("Document")}}.

## Exemples

```js
console.log(window.document.title);
```
