{{APIRef}}

Éloigne la mise au point de la fenêtre.

## Syntaxe

```js
window.blur();
```

## Exemple

```js
window.blur();
```

## Notes

La méthode `window.blur()` est l'équivalent programmatique du déplacement du focus de l'utilisateur loin de la fenêtre courante.
