{{APIRef("HTML DOM")}}

La propriété en lecture seule **`form`** de l'interface {{DOMxRef("HTMLInputElement")}} retourne l'objet {{DOMxRef("HTMLFormElement")}} auquel appartient cet {{HTMLElement("input")}}, ou `null` si cet input n'appartient à aucun formulaire.

## Valeur

Un objet {{DOMxRef("HTMLFormElement")}} ou `null`.
