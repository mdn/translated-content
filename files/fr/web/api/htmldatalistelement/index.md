{{APIRef("HTML DOM")}}

L'interface **`HTMLDataListElement`** fournit des propriétés spécifiques (en plus de l'interface objet {{DOMxRef("HTMLElement")}} dont elle hérite) pour manipuler les éléments HTML {{HTMLElement("datalist")}} et leur contenu.

{{InheritanceDiagram}}

## Propriétés d'instance

_Hérite des propriétés de son parent, {{DOMxRef("HTMLElement")}}._

- {{DOMxRef("HTMLDataListElement.options")}} {{ReadOnlyInline}}
  - : Un objet {{DOMxRef("HTMLCollection")}} représentant la collection des éléments option contenus.

## Méthodes d'instance

_Aucune méthode spécifique&nbsp;; hérite des méthodes de son parent, {{DOMxRef("HTMLElement")}}._
