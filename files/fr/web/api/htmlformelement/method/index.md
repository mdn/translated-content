{{APIRef("HTML DOM")}}

La propriété **`HTMLFormElement.method`** représente la méthode HTTP utilisée pour soumettre un formulaire.

## Syntaxe

```js
string = form.method;
form.method = string;
```

## Exemple

```js
document.forms["myform"].method = "post";
```
