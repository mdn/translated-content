La propriété en lecture seule **`x`** de l'interface **`DOMPointReadOnly`** représente la coordonnée x d'un point.

En général, x positif signifie à droite et négatif à gauche (par rapport à l'origine).

## Syntaxe

```js
var xPos = DOMPoint.x;
```

### Valeur

Un double.
