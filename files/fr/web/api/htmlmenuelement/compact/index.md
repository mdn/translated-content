La propriété **`compact`** de l'interface {{DOMxRef("HTMLMenuElement")}} indique que l'espacement entre les éléments de la liste doit être réduit. La gestion exacte de l'attribut `compact` dépend du navigateur. Au lieu d'utiliser cette propriété, il est conseillé d'utiliser la propriété CSS {{CSSxRef("line-height")}}.

Elle reflète l'attribut [`compact`](/fr/docs/Web/HTML/Reference/Elements/menu#compact) de l'élément HTML {{HTMLElement("menu")}}.

## Valeur

Un booléen.
