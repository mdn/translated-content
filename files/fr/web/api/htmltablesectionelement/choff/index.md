La propriété **`chOff`** de l'interface {{DOMxRef("HTMLTableSectionElement")}} ne fait rien. Elle reflète l'attribut `charoff` de l'élément de section.

> [!NOTE]
> Cette propriété a été conçue pour permettre d'aligner le contenu des cellules de tableau sur un caractère spécifique (généralement le point décimal), mais elle n'a jamais été implémentée par les navigateurs.
>
> Pour obtenir un tel alignement, surveillez la prise en charge d'une valeur de chaîne de caractères avec la propriété CSS {{CSSxRef("text-align")}}.

## Valeur

Un entier.
