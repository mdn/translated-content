{{APIRef("HTML DOM")}}

La propriété en lecture seule **`form`** de l'interface {{DOMxRef("HTMLLegendElement")}} retourne un objet {{DOMxRef("HTMLFormElement")}} qui possède le {{DOMxRef("HTMLFieldSetElement")}} associé à un élément HTML {{HTMLElement("legend")}}, ou `null` si cette légende n'est pas associée à un {{HTMLElement("fieldset")}} possédé par un formulaire.

## Valeur

Un objet {{DOMxRef("HTMLFormElement")}} ou `null`.
