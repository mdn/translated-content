{{APIRef("DOM")}}

La propriété en lecture seule **`assignedSlot`** de l'interface {{DOMxRef("Text")}} retourne l'objet {{DOMxRef("HTMLSlotElement")}} associé à l'élément.

## Valeur

Un objet {{DOMxRef("HTMLSlotElement")}}, ou `null` si aucun élément HTML {{HTMLElement("slot")}} n'est associé au nœud de texte.
