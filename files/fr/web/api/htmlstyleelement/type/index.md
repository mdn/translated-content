La propriété **`type`** de l'interface {{DOMxRef("HTMLStyleElement")}} retourne le type du style courant.
La valeur reflète l'attribut [`type`](/fr/docs/Web/HTML/Reference/Elements/style#type) de l'élément HTML `<style>`.

Il est déconseillé aux auteur·ice·s d'utiliser cette propriété ou de se fier à sa valeur.

## Valeur

Les valeurs autorisées sont une chaîne vide ou une correspondance insensible à la casse pour `"text/css"`.
