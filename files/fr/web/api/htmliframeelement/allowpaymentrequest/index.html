La propriété allowPaymentRequest de l'interface {{domxref("HTMLIFrameElement")}} retourne une valeur booléenne indiquant si la Payment Request API doit être invoquée sur une iframe en cross-origin.

Syntaxe

var allow = htmlIFrameElement.allowPaymentRequest

Valeur

Une valeur booléenne.
