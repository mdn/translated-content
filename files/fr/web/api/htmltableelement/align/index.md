La propriété **`align`** de l'interface {{DOMxRef("HTMLTableElement")}} représente l'alignement du tableau.

## Valeur

L'une des chaînes de caractères suivantes&nbsp;:

- `left`
- `center`
- `right`

## Exemples

```js
// Définir l'alignement d'un tableau
const t = document.getElementById("TableA");
t.align = "center";
```
