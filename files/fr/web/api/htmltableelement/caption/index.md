### Résumé

**caption** renvoie la légende du tableau.

### Syntaxe

```js
string = table.caption;
```

### Exemple

```js
if (table.caption) {
  // On peut alors faire quelque chose avec la légende du tableau
}
```

### Notes

Cette propriété ne renvoie rien s'il n'y a aucune légende pour le tableau.
