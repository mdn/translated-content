La propriété **`width`** de l'interface {{DOMxRef("HTMLTableElement")}} représente la largeur souhaitée du tableau.

## Valeur

Une chaîne de caractères représentant la largeur en nombre de pixels ou en pourcentage.

## Exemples

```js
myTable.width = "75%";
```
