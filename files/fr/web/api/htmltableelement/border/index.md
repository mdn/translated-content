La propriété **`border`** de l'interface {{DOMxRef("HTMLTableElement")}} représente la largeur de la bordure de l'élément HTML {{HTMLElement("table")}}.

## Valeur

Une chaîne de caractères représentant la largeur de la bordure en pixels.

## Exemples

```js
// Définir la largeur de la bordure d'un tableau à 2 pixels
const t = document.getElementById("TableA");
t.border = "2";
```
