{{APIRef("HTML DOM")}}

La propriété **`tFoot`** de l'interface {{DOMxRef("HTMLTableElement")}} représente l'élément HTML {{HTMLElement("tfoot")}} d'un tableau ({{HTMLElement("table")}}). Sa valeur sera `null` s'il n'existe pas un tel élément.

## Valeur

Un élément {{HTMLElement("tfoot")}} ou `null`.

## Exemples

```js
if (table.tFoot === myFoot) {
  // …
}
```
