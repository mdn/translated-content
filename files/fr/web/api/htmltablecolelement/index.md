{{APIRef("HTML DOM")}}

L'interface **`HTMLTableColElement`** fournit des propriétés pour manipuler des éléments de colonne de tableau, seuls ou groupés.

{{InheritanceDiagram}}

## Propriétés d'instance

_Hérite des propriétés de son parent, {{DOMxRef("HTMLElement")}}._

- {{DOMxRef("HTMLTableColElement.align")}} {{Deprecated_Inline}}
  - : Une chaîne de caractères indiquant l'alignement horizontal des données de cellule dans la colonne.
- {{DOMxRef("HTMLTableColElement.ch")}} {{Deprecated_Inline}}
  - : Une chaîne de caractères représentant le caractère d'alignement pour les données de cellule.
- {{DOMxRef("HTMLTableColElement.chOff")}} {{Deprecated_Inline}}
  - : Une chaîne de caractères représentant le décalage pour le caractère d'alignement.
- {{DOMxRef("HTMLTableColElement.span")}}
  - : Un nombre positif qui reflète l'attribut HTML [`span`](/fr/docs/Web/HTML/Reference/Elements/col#span), indiquant le nombre de colonnes auxquelles appliquer les attributs de cet objet.
- {{DOMxRef("HTMLTableColElement.vAlign")}} {{Deprecated_Inline}}
  - : Une chaîne de caractères indiquant l'alignement vertical des données de cellule dans la colonne.
- `HTMLTableColElement.width` {{Deprecated_Inline}}
  - : Une chaîne de caractères représentant la largeur par défaut de la colonne.

## Méthodes d'instance

_Aucune méthode spécifique&nbsp;; hérite des méthodes de son parent, {{DOMxRef("HTMLElement")}}._
