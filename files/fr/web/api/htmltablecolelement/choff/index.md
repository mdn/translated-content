{{APIRef("HTML DOM")}}{{Deprecated_Header}}

La propriété **`chOff`** de l'interface {{DOMxRef("HTMLTableColElement")}} ne fait rien. Elle reflète l'attribut `charoff` de l'élément {{HTMLElement("col")}}.

> [!NOTE]
> Cette propriété était conçue pour permettre l'alignement du contenu des cellules d'un tableau sur un caractère spécifique (généralement le point décimal), mais n'a jamais été implémentée par les navigateurs.
>
> Pour obtenir un tel alignement, surveillez la prise en charge d'une valeur de type chaîne de caractères avec la propriété CSS {{CSSxRef("text-align")}}.

## Valeur

Un entier.
