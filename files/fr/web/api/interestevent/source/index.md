{{APIRef("Popover API")}}

La propriété en lecture seule **`source`** de l'interface {{DOMxRef("InterestEvent")}} est une instance d'objet {{DOMxRef("Element")}} qui représente l'élément invocateur d'intérêt sur lequel l'intérêt a été montré ou perdu, entraînant le déclenchement de l'événement.

## Valeur

Une instance d'objet {{DOMxRef("Element")}}.

## Exemples

Voir le guide [Utilisation des invocateurs d'intérêt](/fr/docs/Web/API/Popover_API/Using_interest_invokers) et la page de référence de l'événement {{DOMxRef("HTMLElement.interest_event", "interest")}} pour des exemples.
