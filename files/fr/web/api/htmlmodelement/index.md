{{APIRef("HTML DOM")}}

L'interface **`HTMLModElement`** fournit des propriétés spécifiques (en plus des méthodes et propriétés habituelles disponibles via l'interface {{DOMxRef("HTMLElement")}}, qu'elle hérite également) pour manipuler les éléments de modification, c'est-à-dire {{HTMLElement("del")}} et {{HTMLElement("ins")}}.

{{InheritanceDiagram}}

## Propriétés d'instance

_Hérite des propriétés de son parent, {{DOMxRef("HTMLElement")}}._

- {{DOMxRef("HTMLModElement.cite")}}
  - : Une chaîne de caractères reflétant l'attribut HTML [`cite`](/fr/docs/Web/HTML/Reference/Elements/del#cite), contenant une URI vers une ressource expliquant le changement.
- {{DOMxRef("HTMLModElement.dateTime")}}
  - : Une chaîne de caractères reflétant l'attribut HTML [`datetime`](/fr/docs/Web/HTML/Reference/Elements/del#datetime), contenant une chaîne de caractères date-heure représentant l'horodatage du changement.
