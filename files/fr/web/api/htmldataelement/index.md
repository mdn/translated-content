{{APIRef("HTML DOM")}}

L'interface **`HTMLDataElement`** fournit des propriétés spécifiques (en plus de l'interface {{DOMxRef("HTMLElement")}} dont elle hérite) pour manipuler les éléments HTML {{HTMLElement("data")}}.

{{InheritanceDiagram}}

## Propriétés d'instance

_Hérite des propriétés de son parent, {{DOMxRef("HTMLElement")}}._

- {{DOMxRef("HTMLDataElement.value")}}
  - : Une chaîne de caractères reflétant l'attribut HTML [`value`](/fr/docs/Web/HTML/Reference/Elements/data#value), contenant une forme interprétable par une machine de la valeur de l'élément.

## Méthodes d'instance

_Aucune méthode spécifique&nbsp;; hérite des méthodes de son parent, {{DOMxRef("HTMLElement")}}._
