{{APIRef("HTML DOM")}}

La propriété en lecture seule **`form`** de l'interface {{DOMxRef("HTMLButtonElement")}} retourne un objet {{DOMxRef("HTMLFormElement")}} qui possède ce {{HTMLElement("button")}}, ou `null` si ce bouton n'appartient à aucun formulaire.

## Valeur

Un objet {{DOMxRef("HTMLFormElement")}} ou `null`.
