{{APIRef("HTML DOM")}}

La propriété en lecture seule **`form`** de l'interface {{DOMxRef("HTMLTextAreaElement")}} retourne un objet {{DOMxRef("HTMLFormElement")}} qui possède un élément HTML {{HTMLElement("textarea")}}, ou `null` si cette zone de texte n'appartient à aucun formulaire.

## Valeur

Un objet {{DOMxRef("HTMLFormElement")}} ou `null`.
