{{APIRef("HTML DOM")}}

La propriété en lecture seule **`type`** de l'interface {{DOMxRef("HTMLTextAreaElement")}} retourne la chaîne de caractères `"textarea"`.

## Valeur

La chaîne de caractères `"textarea"`.

## Exemple

```js
const textArea = document.querySelector("textarea");
console.log(textArea.type); // "textarea"
```
