{{APIRef("DOM")}}

La propriété **`accessKey`** de l'interface {{DOMxRef("HTMLElement")}} définit la touche sur laquelle l'utilisateur·ice doit appuyer pour accéder à l'élément.

> [!NOTE]
> La propriété `HTMLElement.accessKey` est rarement utilisée en raison de ses nombreux conflits avec les raccourcis clavier déjà présents dans les navigateurs. Pour contourner ce problème, les navigateurs appliquent le comportement de `accesskey` uniquement si les touches sont pressées avec d'autres touches «&nbsp;qualifiantes&nbsp;» (comme <kbd>Alt</kbd> + touche d'accès).
