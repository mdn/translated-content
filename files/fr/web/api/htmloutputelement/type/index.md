{{APIRef("HTML DOM")}}

La propriété en lecture seule **`type`** de l'interface {{DOMxRef("HTMLOutputElement")}} retourne la chaîne de caractères `"output"`.

## Valeur

La chaîne de caractères `"output"`.

## Exemple

```js
const output = document.querySelector("output");
console.log(output.type); // "output"
```
