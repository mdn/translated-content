{{APIRef("HTML DOM")}}

La propriété en lecture seule **`form`** de l'interface {{DOMxRef("HTMLOutputElement")}} retourne un objet {{DOMxRef("HTMLFormElement")}} qui possède cet élément HTML {{HTMLElement("output")}}, ou `null` si l'élément n'appartient à aucun formulaire.

## Valeur

Un objet {{DOMxRef("HTMLFormElement")}} ou `null`.
