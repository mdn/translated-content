{{APIRef("HTML DOM API")}}{{Deprecated_Header}}

La propriété **`noWrap`** de l'interface {{DOMxRef("HTMLTableCellElement")}} retourne une valeur booléenne indiquant si le texte de la cellule peut être scindé en plusieurs lignes ou non.

> [!NOTE]
> Cette propriété est obsolète et il convient d'utiliser la propriété CSS {{CSSxRef("white-space")}} avec la valeur `nowrap` à la place.

## Valeur

Une valeur booléenne.

## Exemples

Utilisez plutôt CSS `white-space`. Un [exemple](/fr/docs/Web/CSS/Reference/Properties/white-space#controller_le_retour_a_la_ligne_dans_les_tableaux) est disponible sur la page {{CSSxRef("white-space")}}.
