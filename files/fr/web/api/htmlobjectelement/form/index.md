{{APIRef("HTML DOM")}}

La propriété en lecture seule **`form`** de l'interface {{DOMxRef("HTMLObjectElement")}} retourne un objet {{DOMxRef("HTMLFormElement")}} qui possède cet élément {{HTMLElement("object")}}, ou `null` si cet élément object n'appartient à aucun formulaire.

## Valeur

Un objet {{DOMxRef("HTMLFormElement")}} ou `null`.
