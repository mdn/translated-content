{{APIRef("HTML DOM")}}

La propriété **`name`** de l'interface {{DOMxRef("HTMLObjectElement")}} retourne une chaîne de caractères reflétant l'attribut HTML [`name`](/fr/docs/Web/HTML/Reference/Elements/object#name), définissant le nom du contexte de navigation.

## Valeur

Une chaîne de caractères.
