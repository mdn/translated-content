{{APIRef("HTML DOM")}}

La propriété **`type`** de l'interface {{DOMxRef("HTMLObjectElement")}} retourne une chaîne de caractères reflétant l'attribut HTML [`type`](/fr/docs/Web/HTML/Reference/Elements/object#type), définissant le type MIME de la ressource.

## Valeur

Une chaîne de caractères.
