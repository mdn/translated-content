{{APIRef("HTML DOM")}}

La propriété en lecture seule **`contentDocument`** de l'interface {{DOMxRef("HTMLObjectElement")}} retourne un {{DOMxRef("Document")}} représentant le document actif du contexte de navigation imbriqué de l'élément HTML {{HTMLElement("object")}}, s'il existe&nbsp;; sinon, retourne `null`.

## Valeur

Un objet {{DOMxRef('Document')}}.
