{{APIRef("HTML DOM")}}

La propriété **`height`** de l'interface {{DOMxRef("HTMLObjectElement")}} retourne une chaîne de caractères reflétant l'attribut HTML [`height`](/fr/docs/Web/HTML/Reference/Elements/object#height), définissant la hauteur affichée de la ressource en pixels CSS.

## Valeur

Une chaîne de caractères.
