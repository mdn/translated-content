{{APIRef("DOM")}}

Le constructeur **`Document`** crée un nouvel objet {{DOMxRef("Document")}}, qui est une page web chargée dans le navigateur et servant de point d'entrée vers le contenu de la page.

## Syntaxe

```js-nolint
new Document()
```

### Paramètres

Aucun.
