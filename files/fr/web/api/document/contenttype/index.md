{{APIRef("DOM")}}

La propriété en lecture seule **`contentType`** de l'interface {{DOMxRef("Document")}} retourne le type MIME dans lequel le document est rendu. Il peut provenir d'en-têtes HTTP ou d'autres sources d'informations MIME, et peut être affecté par les conversions automatiques de type effectuées par le navigateur ou les extensions.

> [!NOTE]
> Cette propriété n'est pas affectée par les éléments HTML {{HTMLElement("meta")}}.

## Valeur

`contentType` est une propriété en lecture seule.
