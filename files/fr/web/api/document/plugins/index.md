{{APIRef("DOM")}}

La propriété en lecture seule **`plugins`** de l'interface {{DOMxRef("Document")}} retourne un objet {{DOMxRef("HTMLCollection")}} contenant un ou plusieurs {{DOMxRef("HTMLEmbedElement")}} représentant les éléments HTML {{HTMLElement("embed")}} dans le document actuel.

> [!NOTE]
> Pour obtenir la liste des plugins installés, utilisez plutôt [Navigator.plugins](/fr/docs/Web/API/Navigator/plugins).

## Valeur

Un objet {{DOMxRef("HTMLCollection")}}.
