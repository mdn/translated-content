{{APIRef("HTML DOM")}}

L'interface **`HTMLSpanElement`** représente un élément HTML {{HTMLElement("span")}} et dérive de l'interface {{DOMxRef("HTMLElement")}}, sans implémenter de propriétés ou méthodes supplémentaires.

{{InheritanceDiagram}}

## Propriétés d'instance

_Cette interface ne possède pas de propriétés propres, mais hérite des propriétés de {{DOMxRef("HTMLElement")}}._

## Méthodes d'instance

_Cette interface ne possède pas de méthodes propres, mais hérite des méthodes de {{DOMxRef("HTMLElement")}}._
