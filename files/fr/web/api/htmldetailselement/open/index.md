{{APIRef("HTML DOM")}}

La propriété **`open`** de l'interface {{DOMxRef("HTMLDetailsElement")}} est une valeur booléenne reflétant l'attribut HTML [`open`](/fr/docs/Web/HTML/Reference/Elements/details#open), indiquant si le contenu de l'élément HTML {{HTMLElement("details")}} (à l'exception du {{HTMLElement("summary")}}) doit être affiché à l'utilisateur·ice.

## Valeur

Une valeur booléenne.

La valeur booléenne représente l'état de l'attribut HTML [`open`](/fr/docs/Web/HTML/Reference/Elements/details#open). La valeur `true` signifie qu'il est défini, avec le résumé et les informations supplémentaires affichés à l'utilisateur·ice. La valeur `false` signifie qu'il n'est pas défini, seul le résumé est affiché.
