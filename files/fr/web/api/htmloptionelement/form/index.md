{{APIRef("HTML DOM")}}

La propriété en lecture seule **`form`** de l'interface {{DOMxRef("HTMLOptionElement")}} retourne un objet {{DOMxRef("HTMLFormElement")}} qui possède l'élément {{DOMxRef("HTMLSelectElement")}} associé à cet {{HTMLElement("option")}}, ou `null` si cette option n'est pas associée à un élément HTML {{HTMLElement("select")}} appartenant à un formulaire.

## Valeur

Un objet {{DOMxRef("HTMLFormElement")}} ou `null`.
