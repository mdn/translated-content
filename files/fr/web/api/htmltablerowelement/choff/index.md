La propriété **`chOff`** de l'interface {{DOMxRef("HTMLTableRowElement")}} ne fait rien. Elle reflète l'attribut `charoff` de l'élément HTML {{HTMLElement("tr")}}.

> [!NOTE]
> Cette propriété était conçue pour permettre l'alignement du contenu des cellules de tableau sur un caractère spécifique (généralement le séparateur décimal), mais n'a jamais été implémentée par les navigateurs.
>
> Pour obtenir cet alignement, surveillez la prise en charge d'une valeur chaîne avec la propriété CSS {{CSSxRef("text-align")}}.

## Valeur

Un entier.
