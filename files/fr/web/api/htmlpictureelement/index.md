{{APIRef("HTML DOM")}}

L'interface **`HTMLPictureElement`** représente un élément HTML {{HTMLElement("picture")}}. Elle n'implémente aucune propriété ni méthode spécifique.

{{InheritanceDiagram}}

## Propriétés d'instance

_Aucune propriété spécifique, mais hérite des propriétés de son parent, {{DOMxRef("HTMLElement")}}._

## Méthodes d'instance

_Aucune méthode spécifique, mais hérite des méthodes de son parent, {{DOMxRef("HTMLElement")}}._
