## SVG a DOM interface

L'interface `SVGAElement` fournit un accès aux propriétés des éléments [`<a>`](/fr/docs/Web/SVG/Element/a) ainsi qu'aux méthodes pour les manipuler.

| Nom     | Type                               | Description                                                                              |
