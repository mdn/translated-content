{{APIRef("HTML DOM")}}

La méthode **`pause()`** de l'interface {{DOMxRef("HTMLMediaElement")}} mettra en pause la lecture du média. Si le média est déjà en pause, cette méthode n'aura aucun effet.

## Syntaxe

```js-nolint
pause()
```

### Paramètres

Aucun.

### Valeur de retour

Aucune ({{JSxRef("undefined")}}).

### Exceptions

Aucune.
