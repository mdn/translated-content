{{APIRef("HTML DOM")}}

La propriété en lecture seule **`paused`** de l'interface {{DOMxRef("HTMLMediaElement")}} indique si l'élément média est en pause.

## Valeur

Une valeur booléenne. `true` signifie que l'élément est en pause et `false` qu'il ne l'est pas.

## Exemples

```js
const obj = document.createElement("video");
console.log(obj.paused); // true
```
