{{APIRef("Encrypted Media Extensions")}}{{SecureContext_Header}}

La propriété en lecture seule **`mediaKeys`** de l'interface {{DOMxRef("HTMLMediaElement")}} retourne un objet {{DOMxRef("MediaKeys")}}, qui est un ensemble de clés que l'élément peut utiliser pour le déchiffrement des données multimédias pendant la lecture.

## Valeur

Un objet {{DOMxRef("MediaKeys")}}, ou `null` si aucune clé n'est disponible.
