{{APIRef("HTMLMediaElement")}}

L'évènement **`durationchange`** est déclenché lorsque l'attribut `duration` a été mis à jour.

## Syntaxe

Utilisez le nom de l'évènement dans des méthodes comme {{DOMxRef("EventTarget.addEventListener", "addEventListener()")}}, ou définissez une propriété de gestionnaire d'évènements.

```js-nolint
addEventListener("durationchange", (event) => { })

ondurationchange = (event) => { }
```

## Type d'évènement

Un objet {{DOMxRef("Event")}} générique.

## Exemples

Ces exemples ajoutent un écouteur d'évènements pour l'évènement `durationchange` de `HTMLMediaElement`, puis affichent un message lorsque ce gestionnaire d'évènements a réagi au déclenchement de l'évènement.

Utiliser `addEventListener()`&nbsp;:

```js
const video = document.querySelector("video");

video.addEventListener("durationchange", (event) => {
  console.log("Pas sûr de pourquoi, mais la durée de la vidéo a changé.");
});
```

Utiliser la propriété de gestionnaire d'évènements `ondurationchange`&nbsp;:

```js
const video = document.querySelector("video");

video.ondurationchange = (event) => {
  console.log("Pas sûr de pourquoi, mais la durée de la vidéo a changé.");
};
```
