{{APIRef("HTML DOM")}}

La propriété en lecture seule **`type`** de l'interface {{DOMxRef("HTMLFieldSetElement")}} retourne la chaîne de caractères `"fieldset"`.

## Valeur

La chaîne de caractères `"fieldset"`.

## Exemple

```js
const fs = document.querySelector("fieldset");
console.log(fs.type); // "fieldset"
```
