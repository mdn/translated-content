{{APIRef("HTML DOM")}}

La propriété en lecture seule **`form`** de l'interface {{DOMxRef("HTMLFieldSetElement")}} retourne un objet {{DOMxRef("HTMLFormElement")}} qui possède un élément HTML {{HTMLElement("fieldset")}}, ou `null` si ce jeu de champs n'appartient à aucun formulaire.

## Valeur

Un objet {{DOMxRef("HTMLFormElement")}} ou `null`.
