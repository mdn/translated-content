{{APIRef("HTML DOM")}}

La propriété en lecture seule **`willValidate`** de l'interface {{DOMxRef("HTMLFieldSetElement")}} retourne `false`, car les éléments HTMLO {{HTMLElement("fieldset")}} ne sont pas candidats à la validation par contrainte ({{DOMxRef("HTMLFieldSetElement.willValidate")}} est `false`).

## Valeur

La valeur booléenne `false`.
