{{ APIRef("Console API") }}

Affiche la stack trace dans la Web Console.

{{AvailableInWorkers}}

Voir Stack traces dans la documentation de {{ domxref("console") }} pour plus de détails et d'exemples.

Syntaxe

console.trace();

Exemple

function foo() {
  function bar() {
    console.trace();
  }
  bar();
}

foo();

Dans la console, la trace suivante sera affichée :

bar
foo
<anonymous>
