Le module **d'ajustement de couleur CSS** fournit un modèle et contrôle l'ajustement automatique des couleurs par l'agent utilisateur pour gérer les préférences de l'utilisateur·ice, telles que le «&nbsp;Mode sombre&nbsp;», l'ajustement du contraste et d'autres préférences de schéma de couleurs.

En combinaison avec les fonctionnalités {{CSSxRef("@media")}} {{CSSxRef("@media/prefers-color-scheme", "prefers-color-scheme")}}, {{CSSxRef("@media/prefers-contrast", "prefers-contrast")}} et {{CSSxRef("@media/forced-colors", "forced-colors")}}, ce module définit comment et quand les couleurs sont automatiquement ajustées par le navigateur.
