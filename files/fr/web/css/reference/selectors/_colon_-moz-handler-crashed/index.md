La [pseudo-classe](/fr/docs/Web/CSS/Reference/Selectors/Pseudo-classes) [CSS](/fr/docs/Web/CSS) **`:-moz-handler-crashed`** est une [extension de Mozilla](/fr/docs/Web/CSS/Reference/Mozilla_extensions) qui permet de cibler un élément qui ne peut pas être affiché car le plugin sensé le dessiner a planté.

> [!NOTE]
> Cette pseudo-classe est principalement destinée aux développeur·euse·s de thèmes.

## Syntaxe

```css
:-moz-handler-crashed {
  /* ... */
}
```

## Spécifications

Cette pseudo-classe ne fait partie d'aucun standard.
