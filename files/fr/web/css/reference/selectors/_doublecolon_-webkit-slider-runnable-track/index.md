Le [pseudo-élément](/fr/docs/Web/CSS/Reference/Selectors/Pseudo-elements) [CSS](/fr/docs/Web/CSS) **`::-webkit-slider-runnable-track`** est une [extension de WebKit](/fr/docs/Web/CSS/Reference/Webkit_extensions) qui représente la piste utilisée par un élément {{HTMLElement("input/range", '&lt;input type="range"&gt;')}}.

## Syntaxe

```css
::-webkit-slider-runnable-track {
  /* ... */
}
```

## Spécifications

Ce pseudo-élément ne fait partie d'aucun standard.
