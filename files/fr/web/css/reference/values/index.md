Les **valeurs CSS** sont les valeurs qui peuvent être attribuées aux propriétés CSS. Elles incluent des mots-clés, des [types de données](/fr/docs/Web/CSS/Reference/Values/Data_types) et des [fonctions](/fr/docs/Web/CSS/Reference/Values/Functions).

> [!NOTE]
> Cette page est un index de toutes les fonctionnalités de valeurs en CSS. La page [Valeurs et unités CSS](/fr/docs/Web/CSS/Guides/Values_and_units) présente le module qui définit certaines, mais pas toutes, de ces valeurs.
