Cette page répertorie tous les {{Glossary("Element", "éléments")}} {{Glossary("HTML")}}, qui sont créés à l'aide de {{Glossary("Tag", "balises")}}.

Ils sont regroupés par fonction pour vous aider à trouver facilement ce que vous avez en tête. Une liste alphabétique de tous les éléments est fournie dans la barre latérale de la page de chaque élément ainsi que de celle-ci.

> [!NOTE]
> Pour plus d'informations sur les bases des éléments et attributs HTML, voir [l'article Commencer avec le HTML](/fr/docs/Learn_web_development/Core/Structuring_content/Basic_HTML_syntax).

## Racine principale

| Élément                 | Description                                                                                                                                  |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("html")}} | Représente la racine (élément de plus haut niveau) d'un document HTML. Tous les autres éléments doivent être des descendants de cet élément. |

## Métadonnées du document

Les métadonnées contiennent des informations à propos de la page. Elles comprennent des informations sur les styles, les scripts et les données qui aident des logiciels ({{Glossary("search engine", "moteur de recherche")}}, {{Glossary("Browser","navigateurs")}}, etc.) à utiliser et à rendre la page. Les métadonnées liées aux styles et aux scripts peuvent être définies dans la page ou renvoyées vers un autre fichier contenant ces informations.

| Élément                  | Description                                                                                                                                                                                                                                                                                |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("base")}}  | Définit l'URL de base à utiliser pour recomposer toutes les URL relatives contenues dans un document. Il ne peut y avoir qu'un seul élément `<base>` au sein d'un document.                                                                                                                |
| {{HTMLElement("head")}}  | Contient des informations lisibles par machine (métadonnées) sur le document, comme son [titre](/fr/docs/Web/HTML/Reference/Elements/title), des [scripts](/fr/docs/Web/HTML/Reference/Elements/script) et des [feuilles de style](/fr/docs/Web/HTML/Reference/Elements/style).            |
| {{HTMLElement("link")}}  | Définit des relations entre le document courant et une ressource externe. Cet élément est principalement utilisé pour lier des feuilles CSS, mais sert aussi à définir des icônes de site (favicon) ou des icônes pour l'écran d'accueil et les applications mobiles, entre autres usages. |
| {{HTMLElement("meta")}}  | Représente des {{Glossary("Metadata", "métadonnées")}} qui ne peuvent pas être exprimées par d'autres éléments méta HTML, comme {{HTMLElement("base")}}, {{HTMLElement("link")}}, {{HTMLElement("script")}}, {{HTMLElement("style")}} et {{HTMLElement("title")}}.                         |
| {{HTMLElement("style")}} | Contient des informations de style pour un document ou une portion de document. Il contient du CSS appliqué au contenu du document dans lequel cet élément se trouve.                                                                                                                      |
| {{HTMLElement("title")}} | Définit le titre du document affiché dans la barre de titre du {{Glossary("Browser", "navigateurs")}} ou dans l'onglet de la page. Il ne contient que du texte&nbsp;; toute balise HTML présente à l'intérieur est traitée comme du texte brut.                                            |

## Racine de sectionnement

| Élément                 | Description                                                                                                      |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("body")}} | Représente le contenu principal d'un document HTML. Il ne peut y avoir qu'un seul élément `<body>` par document. |

## Sectionnement du contenu

Les éléments de sectionnement permettent d'organiser le contenu d'un document en parties logiques. Utilisez ces éléments pour construire le plan général de la page, incluant les en-têtes, les pieds de page et les éléments de titre qui identifient les différentes sections.

| Élément                                                                                                                                                                                                                                                                                                                                                                                              | Description                                                                                                                                                                                                                                                                                                                                      |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("address")}}                                                                                                                                                                                                                                                                                                                                                                           | Indique que le contenu HTML inclus fournit des informations de contact pour une personne, plusieurs personnes ou une organisation.                                                                                                                                                                                                               |
| {{HTMLElement("article")}}                                                                                                                                                                                                                                                                                                                                                                           | Représente une composition autonome dans un document, une page, une application ou un site, destinée à être distribuée ou réutilisée indépendamment (par exemple en syndication). Exemples&nbsp;: un message de forum, un article de magazine, une entrée de blog, une fiche produit, un commentaire utilisateur·ice, un widget interactif, etc. |
| {{HTMLElement("aside")}}                                                                                                                                                                                                                                                                                                                                                                             | Représente une partie d'un document dont le contenu n'a qu'un rapport indirect avec le contenu principal. Les apartés sont souvent affichés comme des encadrés ou des boîtes d'information.                                                                                                                                                      |
| {{HTMLElement("footer")}}                                                                                                                                                                                                                                                                                                                                                                            | Représente le pied de page de l'élément d'[sectionnement](/fr/docs/Web/HTML/Guides/Content_categories#sectioning_content) ou de la racine de section la plus proche. Un `<footer>` contient généralement des informations sur l'auteur·ice, des données de droit d'auteur ou des liens vers des documents liés.                                  |
| {{HTMLElement("header")}}                                                                                                                                                                                                                                                                                                                                                                            | Contient du contenu introductif, généralement un groupe d'éléments d'introduction ou d'aides à la navigation. Il peut inclure des éléments de titre, mais aussi un logo, un formulaire de recherche, le nom de l'auteur·ice, etc.                                                                                                                |
| [`<h1>`](/fr/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h2>`](/fr/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h3>`](/fr/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h4>`](/fr/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h5>`](/fr/docs/Web/HTML/Reference/Elements/Heading_Elements), [`<h6>`](/fr/docs/Web/HTML/Reference/Elements/Heading_Elements) | Représente six niveaux de titres de section. `<h1>` est le niveau le plus élevé et `<h6>` le plus bas.                                                                                                                                                                                                                                           |
| {{HTMLElement("hgroup")}}                                                                                                                                                                                                                                                                                                                                                                            | Représente un titre groupé avec du contenu secondaire, comme des sous-titres, un titre alternatif ou un slogan.                                                                                                                                                                                                                                  |
| {{HTMLElement("main")}}                                                                                                                                                                                                                                                                                                                                                                              | Représente le contenu principal du corps d'un document. La zone principale contient le contenu directement lié au sujet central du document ou à la fonctionnalité principale d'une application.                                                                                                                                                 |
| {{HTMLElement("nav")}}                                                                                                                                                                                                                                                                                                                                                                               | Représente une section d'une page destinée à fournir des liens de navigation, soit au sein du document courant, soit vers d'autres documents. Exemples courants&nbsp;: menus, tables des matières, index.                                                                                                                                        |
| {{HTMLElement("section")}}                                                                                                                                                                                                                                                                                                                                                                           | Représente une section autonome générique d'un document qui n'a pas d'élément sémantique plus spécifique pour la représenter. Les sections doivent généralement comporter un titre.                                                                                                                                                              |
| {{HTMLElement("search")}}                                                                                                                                                                                                                                                                                                                                                                            | Représente une partie contenant un ensemble de contrôles de formulaire ou d'autres contenus liés à la réalisation d'une recherche ou à une opération de filtrage.                                                                                                                                                                                |

## Contenu textuel

Utilisez les éléments de contenu textuel HTML pour organiser des blocs ou des sections de contenu placés entre la balise d'ouverture {{HTMLElement("body")}} et la balise de fermeture `</body>`. Importants pour {{Glossary("accessibility")}} et {{Glossary("SEO")}}, ces éléments identifient la finalité ou la structure de ce contenu.

| Élément                       | Description                                                                                                                                                                                                                                                                                                                                        |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("blockquote")}} | Indique que le texte inclus est une citation longue. Habituellement, cela se traduit visuellement par une indentation. Une URL de la source peut être fournie via l'attribut `cite`, et une représentation textuelle de la source peut être donnée avec l'élément {{HTMLElement("cite")}}.                                                         |
| {{HTMLElement("dd")}}         | Fournit la description, la définition ou la valeur du terme précédent ({{HTMLElement("dt")}}) dans une liste de descriptions ({{HTMLElement("dl")}}).                                                                                                                                                                                              |
| {{HTMLElement("div")}}        | Conteneur générique pour le contenu de flux. Il n'a aucun effet sur le contenu ou la mise en page tant qu'il n'est pas mis en forme via du CSS (par exemple, en appliquant un style directement ou en utilisant un modèle de mise en page tel que {{Glossary("Flexbox", "flexbox")}} sur son parent).                                              |
| {{HTMLElement("dl")}}         | Représente une liste de descriptions. Cet élément contient des groupes de termes (spécifiés par {{HTMLElement("dt")}}) et des descriptions (fournies par {{HTMLElement("dd")}}). On l'utilise souvent pour implémenter un glossaire ou pour afficher des métadonnées sous forme de paires clé/valeur.                                              |
| {{HTMLElement("dt")}}         | Spécifie un terme dans une liste de définition ou de descriptions et doit être utilisé à l'intérieur d'un élément {{HTMLElement("dl")}}. Il est généralement suivi d'un élément {{HTMLElement("dd")}}&nbsp;; toutefois, plusieurs `<dt>` consécutifs indiquent plusieurs termes définis par le {{HTMLElement("dd")}} suivant.                      |
| {{HTMLElement("figcaption")}} | Représente une légende décrivant le reste du contenu de son élément parent {{HTMLElement("figure")}}.                                                                                                                                                                                                                                              |
| {{HTMLElement("figure")}}     | Représente un contenu autonome, éventuellement accompagné d'une légende facultative spécifiée par {{HTMLElement("figcaption")}}. La figure, sa légende et son contenu forment une unité référencée ensemble.                                                                                                                                       |
| {{HTMLElement("hr")}}         | Représente une rupture thématique entre des éléments de niveau paragraphe&nbsp;: par exemple, un changement de scène dans un récit ou un changement de sujet au sein d'une section.                                                                                                                                                                |
| {{HTMLElement("li")}}         | Représente un élément d'une liste. Il doit être contenu dans un élément parent tel que {{HTMLElement("ol")}}, {{HTMLElement("ul")}} ou {{HTMLElement("menu")}}. Dans les listes non ordonnées, les éléments sont généralement affichés avec des puces&nbsp;; dans les listes ordonnées, ils sont affichés avec un compteur (nombre, lettre, etc.). |
| {{HTMLElement("menu")}}       | Alternative sémantique à {{HTMLElement("ul")}}&nbsp;; toutefois, les navigateurs et l'arbre d'accessibilité le traitent comme équivalent à {{HTMLElement("ul")}}. Il représente une liste non ordonnée d'éléments (chacun représenté par {{HTMLElement("li")}}).                                                                                   |
| {{HTMLElement("ol")}}         | Représente une liste ordonnée d'éléments — généralement rendue sous forme de liste numérotée.                                                                                                                                                                                                                                                      |
| {{HTMLElement("p")}}          | Représente un paragraphe. Visuellement, les paragraphes sont souvent séparés par une ligne blanche et/ou une indentation de la première ligne, mais en HTML un paragraphe peut contenir divers regroupements structurels (images, champs de formulaire, etc.).                                                                                     |
| {{HTMLElement("pre")}}        | Représente du texte préformaté qui doit être présenté tel qu'il est écrit dans le fichier HTML. Le texte est généralement rendu avec une police à chasse fixe ([monospace](https://fr.wikipedia.org/wiki/Monospaced_font)), et les espaces sont affichés tels qu'ils apparaissent dans le code.                                                    |
| {{HTMLElement("ul")}}         | Représente une liste non ordonnée d'éléments, typiquement rendue sous forme de liste à puces.                                                                                                                                                                                                                                                      |

## Sémantique du texte en ligne

Utilisez les éléments en ligne pour définir la signification, la structure ou la mise en forme d'un mot, d'une ligne ou d'un fragment de texte.

| Élément                   | Description                                                                                                                                                                                                                                                                                                                                                                           |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("a")}}      | Avec son attribut `href`, crée un lien hypertexte vers des pages web, des fichiers, des adresses e-mail, des emplacements au sein de la page courante, ou toute autre destination qu'une URL peut adresser.                                                                                                                                                                           |
| {{HTMLElement("abbr")}}   | Représente une abréviation ou un acronyme.                                                                                                                                                                                                                                                                                                                                            |
| {{HTMLElement("b")}}      | Sert à attirer l'attention sur le contenu sans lui conférer une importance particulière. Autrefois appelé élément Boldface, la plupart des navigateurs rendent encore le texte en gras. N'utilisez pas `<b>` pour la mise en forme&nbsp;: utilisez plutôt la propriété CSS {{CSSxRef("font-weight")}}. Pour indiquer une importance particulière, utilisez {{HTMLElement("strong")}}. |
| {{HTMLElement("bdi")}}    | Demande à l'algorithme bidirectionnel du navigateur de traiter le texte qu'il contient isolément du texte environnant. Utile lorsque l'on insère dynamiquement du texte sans connaître sa direction.                                                                                                                                                                                  |
| {{HTMLElement("bdo")}}    | Permet de remplacer la direction du texte, afin que le texte à l'intérieur soit rendu dans une direction différente.                                                                                                                                                                                                                                                                  |
| {{HTMLElement("br")}}     | Insère un saut de ligne dans le texte. Utile pour écrire un poème ou une adresse où la division en lignes est pertinente.                                                                                                                                                                                                                                                             |
| {{HTMLElement("cite")}}   | Sert à marquer le titre d'une œuvre créative. La référence peut être abrégée conformément aux conventions contextuelles de citation.                                                                                                                                                                                                                                                  |
| {{HTMLElement("code")}}   | Affiche son contenu en indiquant qu'il s'agit d'un court fragment de code. Par défaut, le texte est rendu en police monospace.                                                                                                                                                                                                                                                        |
| {{HTMLElement("data")}}   | Lie un contenu à une version interprétable par machine. Si le contenu concerne une date ou une heure, utilisez l'élément `<time>`.                                                                                                                                                                                                                                                    |
| {{HTMLElement("dfn")}}    | Utilisé pour indiquer le terme défini dans le contexte d'une phrase définition. L'élément {{HTMLElement("p")}} ancestor, la paire {{HTMLElement("dt")}}/{{HTMLElement("dd")}} ou la section la plus proche contenant `<dfn>` est considérée comme la définition du terme.                                                                                                             |
| {{HTMLElement("em")}}     | Marque un texte comme devant être prononcé avec emphase. L'élément `<em>` peut être imbriqué, chaque niveau d'imbrication augmentant le degré d'emphase.                                                                                                                                                                                                                              |
| {{HTMLElement("i")}}      | Représente un fragment de texte distinct du texte courant (termes techniques, expressions étrangères, pensées d'un personnage, etc.). Historiquement rendu en italique, d'où le nom `<i>`.                                                                                                                                                                                            |
| {{HTMLElement("kbd")}}    | Représente une saisie utilisateur·ice (clavier, saisie vocale ou autre dispositif). Par convention, l'agent utilisateur rend le contenu de `<kbd>` en police monospace.                                                                                                                                                                                                               |
| {{HTMLElement("mark")}}   | Représente un texte mis en évidence en raison de sa pertinence dans le contexte.                                                                                                                                                                                                                                                                                                      |
| {{HTMLElement("q")}}      | Indique une citation courte en incise. Les navigateurs entourent généralement le texte de guillemets. Pour des citations longues, utilisez {{HTMLElement("blockquote")}}.                                                                                                                                                                                                             |
| {{HTMLElement("rp")}}     | Fournit des parenthèses de secours pour les navigateurs qui ne prennent pas en charge les annotations ruby (`{{HTMLElement("ruby")}}`). Un `<rp>` doit entourer chaque parenthèse ouvrante et fermante qui entoure l'élément {{HTMLElement("rt")}}.                                                                                                                                   |
| {{HTMLElement("rt")}}     | Spécifie le composant de texte d'une annotation ruby, utilisé pour fournir la prononciation, la traduction ou la translittération dans la typographie d'Asie de l'Est. `<rt>` doit toujours être contenu dans un élément {{HTMLElement("ruby")}}.                                                                                                                                     |
| {{HTMLElement("ruby")}}   | Représente de petites annotations rendues au-dessus, en dessous ou à côté du texte de base, généralement utilisées pour indiquer la prononciation de caractères d'Asie de l'Est.                                                                                                                                                                                                      |
| {{HTMLElement("s")}}      | Rendu avec une ligne traversante (barré). Utilisez `<s>` pour indiquer du contenu qui n'est plus pertinent ou incorrect. Pour indiquer des modifications de document, préférez {{HTMLElement("del")}} et {{HTMLElement("ins")}}.                                                                                                                                                      |
| {{HTMLElement("samp")}}   | Contient un exemple de sortie d'un programme informatique. Son contenu est généralement rendu en police monospace.                                                                                                                                                                                                                                                                    |
| {{HTMLElement("small")}}  | Représente des commentaires ou du petit texte (mentions légales, droit d'auteur). Par défaut, il réduit la taille de police d'un niveau.                                                                                                                                                                                                                                              |
| {{HTMLElement("span")}}   | Conteneur en ligne générique pour du contenu phrasé. Il n'a pas de signification sémantique propre&nbsp;; on l'utilise pour grouper des éléments en vue d'un style ou parce qu'ils partagent des attributs (`class`, `id`, `lang`).                                                                                                                                                   |
| {{HTMLElement("strong")}} | Indique que le contenu a une importance particulière. Les navigateurs affichent généralement ce contenu en gras.                                                                                                                                                                                                                                                                      |
| {{HTMLElement("sub")}}    | Indique du texte en indice (sous la ligne de base), généralement rendu avec une taille de police plus petite.                                                                                                                                                                                                                                                                         |
| {{HTMLElement("sup")}}    | Indique du texte en exposant (au-dessus de la ligne de base), généralement rendu avec une taille de police plus petite.                                                                                                                                                                                                                                                               |
| {{HTMLElement("time")}}   | Représente une période temporelle spécifique. Il peut contenir l'attribut `datetime` pour fournir un format lisible par machine, utile pour le référencement ou des fonctionnalités comme des rappels.                                                                                                                                                                                |
| {{HTMLElement("u")}}      | Représente un fragment de texte annoté non textuellement&nbsp;; il est rendu par défaut avec un soulignement solide, modifiable via CSS.                                                                                                                                                                                                                                              |
| {{HTMLElement("var")}}    | Représente le nom d'une variable dans une expression mathématique ou un contexte de programmation, généralement rendu en italique.                                                                                                                                                                                                                                                    |
| {{HTMLElement("wbr")}}    | Représente un point où le navigateur peut éventuellement couper la ligne si nécessaire.                                                                                                                                                                                                                                                                                               |

## Images et médias

HTML prend en charge différents médias tels que les images, l'audio et la vidéo.

| Élément                  | Description                                                                                                                                                                                                                                                                                                                       |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("area")}}  | Définit une zone cliquable dans une image cartographique. Une _image map_ permet d'associer des zones géométriques d'une image à des {{Glossary("Hyperlink", "liens")}}.                                                                                                                                                          |
| {{HTMLElement("audio")}} | Permet d'intégrer du contenu sonore dans un document. Il peut contenir une ou plusieurs sources audio, définies via l'attribut `src` ou par l'élément `source`&nbsp;: le navigateur choisira la source la plus adaptée. Il peut également être la destination de médias en streaming, en utilisant un {{DOMxRef("MediaStream")}}. |
| {{HTMLElement("img")}}   | Permet d'insérer une image dans le document.                                                                                                                                                                                                                                                                                      |
| {{HTMLElement("map")}}   | Utilisé avec des éléments {{HTMLElement("area")}} pour définir une image cartographique (zone cliquable).                                                                                                                                                                                                                         |
| {{HTMLElement("track")}} | Élément enfant des médias (`audio`, `video`) permettant de spécifier des pistes de texte synchronisées (par exemple pour les sous-titres). Les pistes sont formatées en [WebVTT](/fr/docs/Web/API/WebVTT_API) (`.vtt`).                                                                                                           |
| {{HTMLElement("video")}} | Intègre dans le document un lecteur multimédia capable de lire des vidéos. Vous pouvez également utiliser `<video>` pour du contenu audio, mais l'élément `<audio>` offre souvent une expérience utilisateur plus adaptée.                                                                                                        |

## Contenu embarqué

En plus des médias courants, un document HTML peut embarquer d'autres types de contenu, même si les interactions y sont parfois limitées.

| Élément                        | Description                                                                                                                                                                                                                                                                                                      |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("embed")}}       | Intègre du contenu externe à l'endroit spécifié dans le document. Ce contenu provient d'une application externe ou d'une autre source de contenu interactif, comme un module d'extension du navigateur.                                                                                                          |
| {{HTMLElement("fencedframe")}} | Représente un contexte de navigation imbriqué, similaire à `<iframe>` mais offrant des fonctionnalités de confidentialité natives supplémentaires.                                                                                                                                                               |
| {{HTMLElement("iframe")}}      | Représente un contexte de navigation imbriqué, intégrant une autre page HTML dans la page courante.                                                                                                                                                                                                              |
| {{HTMLElement("object")}}      | Représente une ressource externe pouvant être traitée comme une image, un contexte de navigation imbriqué ou une ressource prise en charge par un module (plug‑in).                                                                                                                                              |
| {{HTMLElement("picture")}}     | Contient zéro ou plusieurs éléments {{HTMLElement("source")}} et un élément {{HTMLElement("img")}} afin de proposer des versions alternatives d'une image selon le dispositif ou le contexte d'affichage.                                                                                                        |
| {{HTMLElement("source")}}      | Spécifie plusieurs ressources médias pour l'élément `picture`, `audio` ou `video`. C'est un élément vide&nbsp;: il n'a pas de contenu et n'a pas de balise de fermeture. Il est couramment utilisé pour proposer le même contenu média dans plusieurs formats afin d'assurer la compatibilité entre navigateurs. |

## SVG et MathML

Vous pouvez intégrer du contenu [SVG](/fr/docs/Web/SVG) et [MathML](/fr/docs/Web/MathML) directement dans les documents HTML en utilisant les éléments {{SVGElement("svg")}} et {{MathMLElement("math")}}.

| Élément                   | Description                                                                                                                                                                                                                                                       |
| ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{SVGElement("svg")}}     | Conteneur définissant un nouveau système de coordonnées et un [viewport](/fr/docs/Web/SVG/Reference/Attribute/viewBox). Il sert d'élément racine pour les documents SVG, mais peut aussi être utilisé pour intégrer un fragment SVG dans un document SVG ou HTML. |
| {{MathMLElement("math")}} | Élément racine du MathML. Toute instance MathML valide doit être enveloppée par cet élément. De plus, il ne faut pas imbriquer un second élément `<math>` à l'intérieur d'un autre, mais on peut avoir un nombre arbitraire d'autres éléments fils.               |

## Scripts

Pour créer du contenu dynamique et des applications Web, HTML prend en charge l'utilisation de langages de script, principalement JavaScript. Certains éléments facilitent cette utilisation.

| Élément                     | Description                                                                                                                                                                                                                                                                    |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| {{HTMLElement("canvas")}}   | Conteneur utilisé avec l'API [canvas](/fr/docs/Web/API/Canvas_API) ou l'API [WebGL](/fr/docs/Web/API/WebGL_API) pour dessiner des graphiques et des animations.                                                                                                                |
| {{HTMLElement("noscript")}} | Définit une section HTML à insérer si un type de script n'est pas pris en charge ou si le script est désactivé dans le navigateur.                                                                                                                                             |
| {{HTMLElement("script")}}   | Permet d'insérer du code exécutable ou des données&nbsp;; il est généralement utilisé pour JavaScript. L'élément `<script>` peut également être employé pour d'autres langages (par exemple le GLSL de [WebGL](/fr/docs/Web/API/WebGL_API) ou [JSON](/fr/docs/Glossary/JSON)). |

## Indication des modifications

Ces éléments permettent d'indiquer que certaines parties du texte ont été modifiées.

| Élément                | Description                                                                                                                                                                                                                                                                                                                       |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("del")}} | Représente une plage de texte supprimée d'un document. Cela peut être utilisé, par exemple, pour afficher un suivi des modifications (<i lang="en">track changes</i>) ou des informations de diff de code source. L'élément `<ins>` peut être utilisé pour l'effet inverse&nbsp;: indiquer du texte qui a été ajouté au document. |
| {{HTMLElement("ins")}} | Représente une plage de texte ajoutée à un document. Vous pouvez utiliser l'élément `<del>` pour représenter de manière similaire une plage de texte qui a été supprimée du document.                                                                                                                                             |

## Contenu tabulaire

Les éléments suivants servent à créer et gérer des données tabulaires.

| Élément                     | Description                                                                                                                                                                                                                                                                                                                        |
| --------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("caption")}}  | Définit la légende (ou le titre) d'un tableau.                                                                                                                                                                                                                                                                                     |
| {{HTMLElement("col")}}      | Définit une ou plusieurs colonnes dans un groupe de colonnes représenté par son élément parent implicite ou explicite {{HTMLElement("colgroup")}}. L'élément `<col>` n'est valide que s'il est enfant d'un {{HTMLElement("colgroup")}} qui ne définit pas l'attribut [`span`](/fr/docs/Web/HTML/Reference/Elements/colgroup#span). |
| {{HTMLElement("colgroup")}} | Définit un groupe de colonnes au sein d'un tableau.                                                                                                                                                                                                                                                                                |
| {{HTMLElement("table")}}    | Représente des données tabulaires — c'est‑à‑dire des informations présentées sous la forme d'un tableau en deux dimensions composé de lignes et de colonnes de cellules contenant des données.                                                                                                                                     |
| {{HTMLElement("tbody")}}    | Contient un ensemble de lignes de tableau (éléments {{HTMLElement("tr")}}) correspondant au corps principal des données du tableau.                                                                                                                                                                                                |
| {{HTMLElement("td")}}       | Enfant de {{HTMLElement("tr")}}, il définit une cellule de tableau contenant des données.                                                                                                                                                                                                                                          |
| {{HTMLElement("tfoot")}}    | Contient un ensemble de lignes de tableau ({{HTMLElement("tr")}}) correspondant au pied du tableau, généralement un résumé des colonnes (par exemple la somme des valeurs).                                                                                                                                                        |
| {{HTMLElement("th")}}       | Enfant de {{HTMLElement("tr")}}, il définit une cellule d'en-tête pour un groupe de cellules. La nature de ce groupe peut être explicitée via les attributs [`scope`](/fr/docs/Web/HTML/Reference/Elements/th#scope) et [`headers`](/fr/docs/Web/HTML/Reference/Elements/th#headers).                                              |
| {{HTMLElement("thead")}}    | Contient un ensemble de lignes de tableau ({{HTMLElement("tr")}}) qui forment l'en-tête du tableau, généralement sous la forme d'en-têtes de colonnes ({{HTMLElement("th")}}).                                                                                                                                                     |
| {{HTMLElement("tr")}}       | Définit une ligne de cellules dans un tableau. Les cellules de la ligne sont établies avec une combinaison de {{HTMLElement("td")}} et {{HTMLElement("th")}}.                                                                                                                                                                      |

## Formulaires

HTML fournit plusieurs éléments pour créer des formulaires que l'utilisateur·ice peut remplir et soumettre. Pour plus d'informations, consultez le [guide des formulaires HTML](/fr/docs/Learn_web_development/Extensions/Forms).

| Élément                            | Description                                                                                                                                                                                                                                                                                                                                                           |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("button")}}          | Élément interactif activé par l'utilisateur·ice (souris, clavier, doigt, commande vocale ou autres aides techniques). Lors de son activation, il exécute une action, par exemple soumettre un [formulaire](/fr/docs/Learn_web_development/Extensions/Forms) ou ouvrir une boîte de dialogue.                                                                          |
| {{HTMLElement("datalist")}}        | Contient un ensemble d'éléments {{HTMLElement("option")}} représentant les valeurs possibles ou recommandées pour d'autres contrôles.                                                                                                                                                                                                                                 |
| {{HTMLElement("fieldset")}}        | Sert à regrouper plusieurs contrôles ainsi que leurs étiquettes ({{HTMLElement("label")}}) au sein d'un formulaire.                                                                                                                                                                                                                                                   |
| {{HTMLElement("form")}}            | Représente une section de document contenant des contrôles interactifs permettant la saisie et la soumission d'informations.                                                                                                                                                                                                                                          |
| {{HTMLElement("input")}}           | Permet de créer des contrôles interactifs pour les formulaires web afin de recevoir des données de l'utilisateur·ice&nbsp;; de nombreux types d'entrée et widgets existent selon l'appareil et l'agent utilisateur. L'élément `<input>` est l'un des plus puissants et complexes du HTML en raison du grand nombre de combinaisons possibles d'attributs et de types. |
| {{HTMLElement("label")}}           | Représente une légende pour un élément d'interface utilisateur.                                                                                                                                                                                                                                                                                                       |
| {{HTMLElement("legend")}}          | Représente une légende pour le contenu de son parent {{HTMLElement("fieldset")}}.                                                                                                                                                                                                                                                                                     |
| {{HTMLElement("meter")}}           | Représente une valeur scalaire dans un intervalle connu ou une valeur fractionnaire.                                                                                                                                                                                                                                                                                  |
| {{HTMLElement("optgroup")}}        | Crée un groupe d'options au sein d'un élément {{HTMLElement("select")}}.                                                                                                                                                                                                                                                                                              |
| {{HTMLElement("option")}}          | Définit un élément contenu dans un {{HTMLElement("select")}}, un {{HTMLElement("optgroup")}} ou un {{HTMLElement("datalist")}}. Ainsi, `<option>` peut représenter des éléments de menu dans des popups et d'autres listes.                                                                                                                                           |
| {{HTMLElement("output")}}          | Conteneur dans lequel un site ou une application peut injecter le résultat d'un calcul ou d'une action utilisateur·ice.                                                                                                                                                                                                                                               |
| {{HTMLElement("progress")}}        | Affiche une indication de progression d'une tâche, généralement sous la forme d'une barre de progression.                                                                                                                                                                                                                                                             |
| {{HTMLElement("select")}}          | Représente un contrôle fournissant un menu d'options.                                                                                                                                                                                                                                                                                                                 |
| {{HTMLElement("selectedcontent")}} | Affiche le contenu de l'`<option>` actuellement sélectionné à l'intérieur d'un `<select>` fermé.                                                                                                                                                                                                                                                                      |
| {{HTMLElement("textarea")}}        | Représente un contrôle d'édition de texte multi‑ligne, utile pour permettre à l'utilisateur de saisir une grande quantité de texte libre (commentaire, avis, etc.).                                                                                                                                                                                                   |

## Éléments interactifs

HTML propose un ensemble d'éléments pour créer des composants d'interface utilisateur interactifs.

| Élément                    | Description                                                                                                                                                                                                                |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("details")}} | Crée un widget de révélation dans lequel l'information n'est visible que lorsque le widget est basculé en état «&nbsp;ouvert&nbsp;». Un résumé ou une étiquette doit être fourni via l'élément {{HTMLElement("summary")}}. |
| {{HTMLElement("dialog")}}  | Représente une boîte de dialogue ou un autre composant interactif (alerte réductible, inspecteur, sous‑fenêtre, etc.).                                                                                                     |
| {{HTMLElement("summary")}} | Spécifie un résumé, une légende ou un titre pour la zone révélée par un élément `details`. Cliquer sur `<summary>` bascule l'état du parent {{HTMLElement("details")}} entre ouvert et fermé.                              |

## Composants Web

Les composants Web (Web Components) permettent de créer et d'utiliser des éléments personnalisés comme s'il s'agissait d'éléments HTML standards. Ils permettent également de créer des versions personnalisées d'éléments standards.

| Élément                     | Description                                                                                                                                                                                                                                                           |
| --------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("slot")}}     | Partie de la suite technologique des [Web Components](/fr/docs/Web/API/Web_components)&nbsp;; `slot` est un espace réservé à l'intérieur d'un composant dans lequel vous pouvez insérer votre propre balisage, permettant de présenter plusieurs arbres DOM ensemble. |
| {{HTMLElement("template")}} | Mécanisme pour contenir du HTML qui ne doit pas être rendu immédiatement lors du chargement de la page, mais qui peut être instancié ultérieurement au moment de l'exécution via JavaScript.                                                                          |

## Éléments obsolètes et dépréciés

> [!WARNING]
> Ces éléments HTML sont obsolètes et dépréciés&nbsp;: **n'utilisez jamais ces éléments dans de nouveaux projets, et remplacez‑les dans les anciens projets dès que possible**. Ils sont listés ici uniquement pour complétude.

| Élément                      | Description                                                                                                                                                                                                                                                                                     |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| {{HTMLElement("acronym")}}   | Ancien élément permettant d'indiquer explicitement une séquence de caractères composant un acronyme ou une abréviation.                                                                                                                                                                         |
| {{HTMLElement("big")}}       | Rend le texte contenu un niveau de taille de police supérieur à celui du texte environnant. La taille est plafonnée par la taille maximale autorisée par le navigateur.                                                                                                                         |
| {{HTMLElement("center")}}    | Centre horizontalement le contenu de bloc ou en ligne dans son conteneur.                                                                                                                                                                                                                       |
| `<content>`                  | Partie obsolète de la suite [Web Components](/fr/docs/Web/API/Web_components) — était utilisée dans le Shadow DOM comme point d'insertion&nbsp;; elle a été remplacée par {{HTMLElement("slot")}}. Utilisez `slot` à la place.                                                                  |
| {{HTMLElement("dir")}}       | Ancien conteneur pour un répertoire de fichiers et/ou dossiers. N'utilisez pas cet élément obsolète&nbsp;: utilisez plutôt {{HTMLElement("ul")}} pour représenter des listes, y compris des listes de fichiers.                                                                                 |
| {{HTMLElement("font")}}      | Définissait la taille, la couleur et la police du contenu.                                                                                                                                                                                                                                      |
| {{HTMLElement("frame")}}     | Définissait une zone particulière dans laquelle un autre document HTML pouvait être affiché. Les frames étaient utilisées au sein d'un {{HTMLElement("frameset")}}.                                                                                                                             |
| {{HTMLElement("frameset")}}  | Conteneur d'éléments {{HTMLElement("frame")}}.                                                                                                                                                                                                                                                  |
| `<image>`                    | Précurseur ancien et peu supporté de l'élément {{HTMLElement("img")}}&nbsp;; à proscrire.                                                                                                                                                                                                       |
| {{HTMLElement("marquee")}}   | Insérait une zone de texte défilant. Ses attributs contrôlaient le comportement lorsque le texte atteignait les bords de la zone.                                                                                                                                                               |
| `<menuitem>`                 | Représentait une commande exécutable par l'utilisateur·ice via un menu contextuel ou un menu associé à un bouton.                                                                                                                                                                               |
| {{HTMLElement("nobr")}}      | Empêchait le texte de se couper automatiquement sur plusieurs lignes, ce qui pouvait forcer l'utilisateur à faire défiler horizontalement pour voir le texte complet.                                                                                                                           |
| {{HTMLElement("noembed")}}   | Méthode obsolète et non standard pour fournir du contenu de secours aux navigateurs qui ne supportent pas `<embed>` ou le type de contenu intégré souhaité. Dépréciée au profit d'un contenu de repli placé entre les balises d'un élément {{HTMLElement("object")}}.                           |
| {{HTMLElement("noframes")}}  | Fournissait du contenu à afficher dans les navigateurs qui ne supportent pas ou ont désactivé les frames. Malgré une large prise en charge, certains navigateurs spécialisés ou textuels n'acceptaient pas les frames.                                                                          |
| {{HTMLElement("param")}}     | Définissait des paramètres pour un élément {{HTMLElement("object")}}.                                                                                                                                                                                                                           |
| {{HTMLElement("plaintext")}} | Rende tout le contenu suivant la balise de début comme du texte brut, sans interpréter le HTML suivant. Il n'y a pas de balise de fermeture.                                                                                                                                                    |
| {{HTMLElement("rb")}}        | Délimite le composant de texte de base d'une annotation ruby&nbsp;; un `<rb>` doit entourer chaque segment atomique du texte de base.                                                                                                                                                           |
| {{HTMLElement("rtc")}}       | Accueille des annotations sémantiques de caractères présentées dans une annotation ruby (éléments {{HTMLElement("rb")}}). Les éléments {{HTMLElement("rb")}} peuvent contenir des annotations de prononciation ({{HTMLElement("rt")}}) et des annotations sémantiques ({{HTMLElement("rtc")}}). |
| `<shadow>`                   | Partie obsolète des Web Components destinée à servir de point d'insertion du Shadow DOM. Elle a été remplacée par {{HTMLElement("slot")}}.                                                                                                                                                      |
| {{HTMLElement("strike")}}    | Applique une ligne horizontale (barré) sur le texte.                                                                                                                                                                                                                                            |
| {{HTMLElement("tt")}}        | Rend le texte en police monospace par défaut de l'agent utilisateur.                                                                                                                                                                                                                            |
| {{HTMLElement("xmp")}}       | Rends le texte entre les balises sans interpréter le HTML et en utilisant une police à chasse fixe.                                                                                                                                                                                             |
