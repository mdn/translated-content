{{jsSidebar("Operators")}}

L'opérateur d'addition avec assignement (+=) permet d'ajouter une valeur à une variable. Le type des deux valeurs permet de définir si l'utilisation de cet opérateur les concatènera ou les additionnera.

{{EmbedInteractiveExample("pages/js/expressions-addition-assignment.html")}}

La source de cet exemple interactif est stocké sur GitHub. Si vous voulez contribuer au projet de l'exemple interactif, merci de cloner ceci : https://github.com/mdn/interactive-examples et nous envoyer une "pull-request".

Syntaxe

Opérateur : x += y
Signifie :  x = x + y

Examples

Utilisation de l'opérateur

// On considère les variables suivantes :
var chaine = "Hello";
var nombre = 5;
var booleen = true;

// Nombre + Nombre
nombre += 2;
// 7

// Booléen + Nombre
booleen += 1;
// 2

// Booléen + Booléen
booleen += false;
// 1

// Nombre + Chaîne
nombre += "World";
// "5World"

// Chaîne + Booléen
chaine += false;
// "Hellofalse"

// Chaîne + Chaîne
chaine += "World"
// "HelloWorld"
