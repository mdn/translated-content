La propriété accesseur **`yearOfWeek`** des instances de {{JSxRef("Temporal.ZonedDateTime")}} retourne un entier représentant l'année à associer à la {{JSxRef("Temporal/ZonedDateTime/weekOfYear", "weekOfYear")}} de cette date, ou `undefined` si le calendrier n'a pas de système de semaines bien défini. Cela dépend du [calendrier](/fr/docs/Web/JavaScript/Reference/Global_Objects/Temporal#calendriers).

L'accesseur de définition de `yearOfWeek` est `undefined`. Vous ne pouvez pas modifier cette propriété directement.

Pour des informations générales et plus d'exemples, voir {{JSxRef("Temporal/PlainDate/yearOfWeek", "Temporal.PlainDate.prototype.yearOfWeek")}}.
