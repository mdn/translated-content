La fonction `true` retourne la valeur booléenne `true`.

### Syntaxe

```
true()
```

### Retour

Le booléen `true`.

### Définition

[XPath 1.0, section 4.3](https://www.w3.org/TR/xpath#function-true).

### Support Gecko

Supportée.
