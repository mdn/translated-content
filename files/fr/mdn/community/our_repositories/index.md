MDN Web Docs est un projet complexe avec de nombreux éléments en interaction.
Il est utile de se familiariser avec les différents dépôts de code.
Ce document décrit les dépôts («&nbsp;repositories&nbsp;» ou «&nbsp;repos&nbsp;» en anglais) que vous pouvez utiliser pour contribuer à MDN Web Docs.

## Dépôts principaux

- [content](https://github.com/mdn/content)
  - : C'est ici que tout le contenu en anglais du site est maintenu, et où vous effectuerez toutes les modifications sur le contenu des pages, la rédaction et les exemples de code intégrés.
- [rari](https://github.com/mdn/rari)
  - : Le **backend** de la plateforme MDN Web Docs, où vous pouvez modifier la structure des pages, les modèles et le moteur de rendu.
- [fred](https://github.com/mdn/fred)
  - : Le **frontend** de MDN Web Docs, où se trouvent les styles, le design et la mise en page.
- [browser-compat-data](https://github.com/mdn/browser-compat-data)
  - : Données utilisées pour générer les tableaux de compatibilité des navigateurs présents sur nos pages de référence.
    Si vous avez des informations sur la compatibilité des fonctionnalités Web — ou si vous souhaitez faire des recherches et/ou des tests — vous pouvez aider à mettre à jour les [données de compatibilité des navigateurs de MDN <sup>(angl.)</sup>](https://github.com/mdn/browser-compat-data/blob/main/docs/contributing.md).
- [translated-content](https://github.com/mdn/translated-content)
  - : C'est ici que se trouve le contenu localisé.
    Rendez-vous ici si vous souhaitez aider à traduire des pages dans l'une de nos [langues activement maintenues <sup>(angl.)</sup>](https://github.com/mdn/translated-content#locales).

### Infrastructure

- [rumba](https://github.com/mdn/rumba)
  - : Un système backend qui alimente MDN Plus.
- [workflows](https://github.com/mdn/workflows)
  - : Une collection d'actions GitHub réutilisables pour les dépôts MDN Web Docs.

### Planification et coordination

Ces dépôts servent à la planification, à la documentation du projet lui-même et aux projets communautaires.

- [mdn/mdn](https://github.com/mdn/mdn)
  - : Les propositions de projet et les sujets de planification sont gérés ici.
- [mdn-community](https://github.com/mdn/mdn-community)
  - : Ce dépôt héberge les discussions GitHub lorsque les rédacteur·ice·s et contributeur·ice·s doivent décider de la suite à donner à la rédaction ou aux choix techniques.

> Si vous mettez à jour le code d'une page, vérifiez s'il est référencé dans un dépôt d'exemples correspondant et, le cas échéant, assurez-vous de mettre à jour ce dépôt également.
