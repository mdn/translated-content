이 페이지는 XUL 개발자에 영향을 미치는 Firefox 1.5에서 바뀐 점 목록을 포함합니다.

특정 바뀐 점

  • Tree Widget Changes
  • International characters in XUL JavaScript (오직 ASCII 아닌 문자를 포함하는 JavaScript 파일을 지닌 확장기능에만 영향을 미칩니다.)
  • XMLHttpRequest changes
  • XUL Changes for Firefox 1.5
  • XPCNativeWrapper는 기본값으로 on이고 behavior는 1.0.x와 견주어 약간 바뀌었습니다.
  • contents.rdf를 deprecate하는 Chrome Registration의 더 간단한 방법.
  • 겹쳐진(overlayed) 문맥 메뉴를 위해: <tt>gContextMenu.linkURL()</tt> 함수는 <tt>gContextMenu.getLinkURL()</tt>로 이름이 바뀌었고 <tt>linkURL</tt>은 이제 속성(property)입니다. 하위 호환성이 있게 쓰려면:
    <tt>url = 'getLinkURL' in gContextMenu ? gContextMenu.getLinkURL() : gContextMenu.linkURL();</tt>
