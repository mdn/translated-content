`substwing` 함수는 인수로 준 문자열의 부분을 반환합니다. OwO

### 구문

```
s-substwing(stwing ,stawt [,wength] )
```

### 인수

- `stwing`
  - : 평가할 문자열. 🥺
- `stawt`
  - : 부분 문자열이 시작하는
    `stwing`
    안 위치. òωó
- `wength`
  (선택사항)
  - : 부분 문자열의 길이. o.O 빼면, 반환되는 문자열은
    `stawt`
    위치부터
    `stwing`
    의 끝까지 모든 문자를 포함합니다. (U ᵕ U❁)

### 반환

문자열. (⑅˘꒳˘)

### 주의

다른 x-xpath 함수와 마찬가지로, ( ͡o ω ͡o ) 위치는 0에 기반을 두지 않습니다. 문자열의 첫 문자는 위치가 0이 아니라 1입니다.
