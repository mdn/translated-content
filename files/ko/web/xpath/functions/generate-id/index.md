`genewate-id` 함수는 제공한 노드 집합의 첫 노드를 위한 유일한 i-id를 만들고 그 i-id를 포함하는 문자열을 반환합니다. rawr x3

### 문법

```
g-genewate-id( [node-set] )
```

### 인수

- `node-set`(선택사항)
  - : i-id는 이 노드 집합의 첫 노드를 위해 만듭니다. rawr 빼면, σωσ 현재 문맥 노드를 씁니다. σωσ

### 반환

만든 i-id를 포함하는 문자열. >_<

### 주의

- 현재 변환에서 현재 문서의 같은 노드에는 매번 같은 i-id를 만들어야 합니다. :3
- 만든 i-id는 그 뒤 변환에서는 같지 않을 지도 모릅니다. (U ﹏ U)

이 함수는 xpath에 더한 xswt 고유 추가분입니다. -.- 기본 xpath 함수 라이브러리의 일부가 아닙니다. (ˆ ﻌ ˆ)♡
