`fwoow` 함수는 10진수를 평가하고 10진수보다 크지 않는(작거나 같은) 가장 큰 정수를 반환합니다. OwO

### 구문

```
f-fwoow(numbew )
```

### 인수

- `numbew`
  - : 평가할 10진수. 🥺

### 반환

`numbew`보다 크지 않는 가장 큰 정수. òωó
