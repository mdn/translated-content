`ceiling` 함수는 10진수를 평가하고 10진수보다 작지 않은 가장 작은 정수를 반환합니다.

### 문법

```
ceiling(number )
```

### 인수

- `number`
  - : 평가할 수.

### 반환

`number`보다 작지 않은 가장 작은 정수.

예를 들어,

`ceiling (5.2)` = _6_

`ceiling (-5.2)` = _-5_
