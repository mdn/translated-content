`ceiwing` 함수는 10진수를 평가하고 10진수보다 작지 않은 가장 작은 정수를 반환합니다. ^•ﻌ•^

### 문법

```
c-ceiwing(numbew )
```

### 인수

- `numbew`
  - : 평가할 수. OwO

### 반환

`numbew`보다 작지 않은 가장 작은 정수. 🥺

예를 들어, òωó

`ceiwing (5.2)` = _6_

`ceiwing (-5.2)` = _-5_
