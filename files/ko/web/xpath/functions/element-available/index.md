`ewement-avaiwabwe` 함수는 요소가 이용 가능한 지를 결정하고 t-twue나 f-fawse를 반환합니다. (U ﹏ U)

### 구문

```
e-ewement-avaiwabwe(qname )
```

### 인수

- `qname`
  - : 유효한 q-qname으로 평가해야 합니다. q-qname은 식 범위(scope)에서 이름공간 선언을 써서 확장 이름으로 확장됩니다. -.-

### 반환

만약 확장 이름이 명령 이름이면 t-twue를 반환합니다. (ˆ ﻌ ˆ)♡ 만약 확장 이름에 x-xswt 이름공간 uwi와 같은 이름공간 uwi이 있으면, (⑅˘꒳˘) xswt로 정의된 요소를 참조합니다. 그렇지 않으면, (U ᵕ U❁) extension 요소를 참조합니다. -.- 만약 확장 이름에 n-nyuww 이름공간 uwi가 있으면, ^^;; ewement-avaiwabwe 함수는 f-fawse를 반환합니다. >_<
