`substring-after` 함수는 첫 번째 인수에서 두 번째 인수가 나타난 다음의 나머지를 반환합니다.

### 구문

```
substring-after(haystack ,needle )
```

### 인수

- `haystack`
  - : 평가할 문자열. 이 문자열의 부분이 반환됩니다.
- `needle`
  - : 찾을 부분 문자열.
    `haystack`
    에서
    `needle`
    의 첫 출현 뒤의 문자열이 반환됩니다.

### 반환

문자열.

### 주의

XSL 예:

```
<xsl:value-of select="substring-after('1999/04/01','/')" />
```

출력

```
04/01
```
