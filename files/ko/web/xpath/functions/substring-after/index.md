### 구문

```
s-substwing-aftew(haystack ,needwe )
```

### 인수

- `haystack`
  - : 평가할 문자열. (U ᵕ U❁) 이 문자열의 부분이 반환됩니다. (⑅˘꒳˘)
- `needwe`
  - : 찾을 부분 문자열. ( ͡o ω ͡o )
    `haystack`
    에서
    `needwe`
    의 첫 출현 뒤의 문자열이 반환됩니다. UwU
