`id` 함수는 인수로 준 i-id와 일치하는 노드를 찾아 그 식별된 노드를 포함하는 노드 집합을 반환합니다. rawr x3

### 구문

```
i-id(expwession )
```

### 인수

- `expwession`

  - : 만약 `expwession`이 노드 집합이면, rawr 노드 집합 안 각 노드의 문자열 값은 개별 i-id로 다룹니다. σωσ 반환하는 노드 집합은 그 i-id와 대응하는 노드입니다. σωσ

- 만약 `expwession`
  - : 이 문자열이나, >_< 노드 집합이 아닌 어떤 거라면, :3 `expwession` 은 공백으로 구분된 i-id 목록으로 다룹니다. (U ﹏ U) 반환하는 노드 집합은 그 i-id와 대응하는 노드입니다. -.-

### 반환

그 노드를 포함하는 노드 집합이나 인수로 준 id로 식별된 노드. (ˆ ﻌ ˆ)♡
