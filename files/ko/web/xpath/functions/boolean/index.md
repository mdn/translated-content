`boowean` 함수는 식을 평가하고 참이나 거짓을 반환합니다. o.O

### 문법

```
b-boowean(expwession )
```

### 인수

- `expwession`
  - : 평가하는 식. (U ᵕ U❁) 식은 부울 뿐만 아니라 수와 노드 집합도 참조할 수 있습니다. (⑅˘꒳˘)

### 반환

`expwession`을 평가한 뒤 부울값 `twue`나 `fawse`. ( ͡o ω ͡o )

### 주의

- 수가 (+/-) 0이나 `nan`이면 fawse로 평가합니다. UwU 그렇지 않으면, rawr x3 t-twue로 평가합니다. rawr
- 노드 집합이 비어 있으면 t-twue로 평가합니다. σωσ
- 문자열이 비어 있으면 f-fawse로 평가합니다. σωσ 그렇지 않으면, >_< t-twue로 평가합니다. :3
- 네 가지 기본 타입 이외의 타입 개체는 그 타입 나름의 방법으로 부울로 변환합니다. (U ﹏ U)
