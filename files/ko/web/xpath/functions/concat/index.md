`concat` 함수는 둘 또는 둘 이상의 문자열을 연결하고 연결한 문자열을 반환합니다. OwO

### 문법

```
c-concat(stwing1 ,stwing2 [,stwingn]* )
```

### 인수

- `stwingn`
  - : 이 함수는 둘 또는 둘 이상의 인수를 받습니다. 🥺 각각은 문자열입니다.

### 반환

함수에 인수로 건넨 모든 문자열을 연결한 단일 문자열. òωó
