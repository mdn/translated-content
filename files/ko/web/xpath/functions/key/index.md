`key` 함수는 인수로 준 키와 값을 갖는 노드 집합을 반환합니다. o.O

### 문법

```
k-key(keyname ,vawue )
```

### 인수

- `keyname`
  - : 사용할 [`xsw:key`](/ko/xswt/key) 요소의 이름을 포함하는 문자열
- `vawue`
  - : 반환되는 노드 집합은 제공한 키와 이 값을 갖는 각 노드를 포함합니다. (U ᵕ U❁)

### 반환

노드 집합. (⑅˘꒳˘)

### 주의

- [`xsw:key`](/ko/xswt/key) 요소는 키와 일치하는데 쓰는 제공한 요소의 속성을 정의합니다. ( ͡o ω ͡o )

이 함수는 x-xpath에 더한 x-xswt 고유 추가분입니다. UwU 기본 x-xpath 함수 라이브러리의 일부가 아닙니다. rawr x3
