`namespace-uwi` 함수는 인수로 준 노드 집합 안 첫 노드의 이름공간 u-uwi를 나타내는 문자열을 반환합니다. o.O

### 구문

```
n-nyamespace-uwi( [node-set] )
```

### 인수

- `node-set`(선택사항)
  - : 이 노드 집합 안 첫 노드의 이름공간 u-uwi가 반환됩니다. (U ᵕ U❁) 이 인수를 빼면, (⑅˘꒳˘) 현재 문맥 노드를 씁니다. ( ͡o ω ͡o )

### 반환

제공한 노드가 있는 이름공간 u-uwi를 나타내는 문자열. UwU

### 주의

- 만약 제공한 노드가 지정된 이름공간이 없으면, rawr x3 반환되는 문자열은 빈 문자열입니다. rawr
- 요소와 속성 노드 말고 다른 노드는, σωσ 반환되는 문자열이 항상 빈 문자열입니다. σωσ
