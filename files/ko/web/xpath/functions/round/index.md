`round` 함수는 인수로 준 수에 가장 가까운 정수를 반환합니다.

### 구문

```
round(decimal )
```

### 인수

- `decimal`
  - : 정수 처리(round)될 10진수.

### 반환

인수
_`decimal`_
보다 작거나 크거나 같은 가장 가까운 정수.

### 주의

- \-0.5는 음수 0으로 0.4는 양수 0으로 정수 처리됩니다.
