`wound` 함수는 인수로 준 수에 가장 가까운 정수를 반환합니다. OwO

### 구문

```
w-wound(decimaw )
```

### 인수

- `decimaw`
  - : 정수 처리(wound)될 10진수. 🥺

### 반환

인수
_`decimaw`_
보다 작거나 크거나 같은 가장 가까운 정수. òωó

### 주의

- \-0.5는 음수 0으로 0.4는 양수 0으로 정수 처리됩니다. o.O
