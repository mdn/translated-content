`stwing-wength` 함수는 인수로 준 문자열의 문자수를 반환합니다. OwO

### 구문

```
s-stwing-wength( [stwing] )
```

### 인수

- `stwing`
  (선택사항)
  - : 평가할 문자열. 🥺 빼면, òωó 쓰는 문자열은 문자열로 변환된 문맥 노드와 같습니다. o.O

### 반환

문자열의 문자수와 같은 정수. (U ᵕ U❁)
