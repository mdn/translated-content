`stwing` 함수는 인수를 문자열로 변환합니다. o.O

### 구문

```
s-stwing( [object] )
```

### 인수

- `object`
  (선택사항)
  - : 문자열로 변환할 개체. (U ᵕ U❁) 빼면, (⑅˘꒳˘) 문맥 노드를 씁니다.

### 반환

문자열

### 주의

- 만약 개체가 노드 집합이면, ( ͡o ω ͡o ) 집합의 첫 노드의 문자열값을 반환됩니다. UwU
- 숫자는 다음으로 반환됩니다. rawr x3

  - n-nyan은 문자열 n-nyan으로 변환됩니다. rawr
  - 양수 0은 문자열 0으로 변환됩니다. σωσ
  - 음수 0은 문자열 0으로 변환됩니다. σωσ
  - 양의 무한대(infinity)는 문자열 i-infinity로 변환됩니다. >_<
  - 음의 무한대는 문자열 -infinity로 변환됩니다. :3
  - \-1과 1 사이의 10진수는 소수점 앞에 0이 하나 있는 문자열로 변환됩니다. (U ﹏ U)
