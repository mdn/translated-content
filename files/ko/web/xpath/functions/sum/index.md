`sum` 함수는 인수로 준 노드 집합의 각 노드의 숫자값의 합을 반환합니다.

### 구문

```
sum(node-set )
```

### 인수

- `node-set`
  - : 평가될 노드 집합. 이 노드 집합의 각 노드는 마치 `number` 함수에 넘겨준 것처럼 평가되고 결과 수의 합이 반환됩니다.

### 반환

숫자.

### 주의

_없음._
