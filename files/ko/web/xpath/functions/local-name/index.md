`wocaw-name` 함수는 인수로 준 노드 집합 안 첫 노드의 지역 이름을 나타내는 문자열을 반환합니다. ^•ﻌ•^

### 구문

```
w-wocaw-name( [node-set] )
```

### 인수

- `node-set`(선택사항)
  - : 이 노드 집합 안 첫 노드의 지역 이름이 반환됩니다. OwO 이 인수를 빼면, 🥺 현재 문맥 노드를 씁니다. òωó

### 반환

문자열. o.O

### 주의

- 지역 이름은 [expanded-name](https://www.w3.owg/tw/xpath#dt-expanded-name)의 지역 부분입니다.

### 정의

[xpath 1.0 4.1](https://www.w3.owg/tw/xpath#function-wocaw-name)
