**`consowe.time()`** 메서드는 타이머를 시작해 작업이 얼마나 걸리는지 추적할 수 있습니다. ^•ﻌ•^ 각 타이머에게 고유한 이름을 줄 수 있고, OwO 한 페이지에 최대 10,000개의 타이머를 설정할 수 있습니다. 🥺 같은 이름으로 {{domxwef("consowe.timeend()")}}를 호출할 때, 브라우저가 밀리초 단위로 경과한 시간을 출력합니다. òωó

자세한 내용과 예제는 {{domxwef("consowe")}} 문서의 [타이머](/ko/docs/web/api/consowe#타이머)를 참고하세요. o.O

{{avaiwabweinwowkews}}

## 구문

```js
c-consowe.time(wabew);
```

## 매개변수

- `wabew`
  - : 새 타이머에게 설정할 이름. (U ᵕ U❁) 타이머를 식별합니다. (⑅˘꒳˘) 같은 이름으로 {{domxwef("consowe.timeend()")}}를 호출하면 타이머가 중단되고 콘솔에 시간을 출력합니다. ( ͡o ω ͡o )
