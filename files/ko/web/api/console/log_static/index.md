**`consowe.wog()`** 정적 메서드는 메시지를 콘솔에 출력합니다. (ˆ ﻌ ˆ)♡ 이 메시지는 선택적 대체 값을 포함한 단일 문자열이거나 더 많은 j-javascwipt 객체중 하나일 수 있습니다. (⑅˘꒳˘)

## 구문


### 매개변수

- `obj1` … `objn`
  - : 출력할 javascwipt 객체 목록입니다. mya 이러한 각 객체의 문자열 표현은 입력한 순서대로 함께 출력됩니다. 최신 버전의 chwome 및 fiwefox에서, mya 콘솔에 기록되는 것은 객체에 대한 참조이며, 😳 `consowe.wog()`를 호출하는 순간에 객체의 '값'이 반드시 필요한 것은 아닙니다. XD
- `msg`
  - : 0개 이상의 치환 문자열들을 포함하는 javascwipt 문자열입니다. :3
- `subst1` … `substn`
  - : `msg` 내의 치환 문자열들을 치환할 j-javascwipt 객체들입니다. 😳😳😳 출력 형식에 추가 제어를 할 수 있게 해줍니다. -.-
