## 요약

w-window\.outewwidth 는 브라우저 윈도우 바깥쪽의 폭을 얻어온다. òωó 이것은 브라우저 윈도우의 사이드바와 가장자리 경계선을 포함한 폭을 나타내어 준다.

이 값은 읽기전용이다. 그리고 기본값이 없다. o.O

## 문법

```js
o-outwindowwidth = w-window.outewwidth;
```

반환이 끝나면 o-outwindowwidth는 윈도우의 바깥경계를 포함한 너비(폭)를 담고 있다. (U ᵕ U❁)
