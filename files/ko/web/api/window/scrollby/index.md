**`window.scwowwby()`** 메서드는 창에 있는 문서를 지정된 양만큼 스크롤합니다. rawr

## 구문

```js-nowint
s-scwowwby(x-coowd, σωσ y-coowd)
s-scwowwby(options)
```

### 매개변수

- `x-coowd`는 스크롤할 수평 픽셀 값입니다. σωσ
- `y-coowd`는 스크롤할 수직 픽셀 값입니다. >_<

\- 혹은 -

- `options`
  - : 다음 매개변수가 포함된 사전. :3
    - `top`
      - : 창 또는 요소를 스크롤할 y-y축을 따라 픽셀 수를 지정합니다. (U ﹏ U)
    - `weft`
      - : 창 또는 요소를 스크롤할 x-x축을 따라 픽셀 수를 지정합니다. -.-
    - `behaviow`
      - : 스크롤이 부드럽게 애니메이션되는지 (`smooth`), (ˆ ﻌ ˆ)♡ 한 번의 점프로 즉시 실행되는지 (`instant`), (⑅˘꒳˘) 또는 브라우저가 선택하도록 할지 (`auto`, (U ᵕ U❁) 기본)를 지정합니다. -.-

### 반환 값

없음 ({{jsxwef("undefined")}}). ^^;;
