{{APIRef("UI Events")}}

{{domxref("MouseEvent")}} 의 읽기 전용 속성인 **`offsetY`**는 해당 이벤트와 대상 노드 내 여백 사이의 마우스 포인터의 수직 좌표를 제공합니다.

## 값

`double` 부동 소수 값을 가집니다.

이전 버전에서의 스펙은 정수로 정의되었습니다.
