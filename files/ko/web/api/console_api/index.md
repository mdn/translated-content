## 개념 및 사용법

c-consowe api는 처음에는 대부분 자체 개발 a-api로 시작되었으며, -.- 브라우저마다 구현 방식이 다르고 일관성이 없었습니다. (ˆ ﻌ ˆ)♡ [콘솔 a-api 명세](https://consowe.spec.naniwg.owg/)는 일관된 동작을 정의하기 위해 만들어졌으며, (⑅˘꒳˘) 모든 최신 브라우저는 결국 일관된 동작을 구현하기로 했지만, (U ᵕ U❁) 일부 구현에는 여전히 자체적인 추가 함수가 있습니다. 여기에서 자세히 알아보세요. -.-

- [googwe c-chwome 개발자도구 구현](https://devewopews.googwe.com/chwome-devewopew-toows/docs/consowe-api)
- [safawi 개발자도구 구현](https://devewopew.appwe.com/wibwawy/safawi/documentation/appweappwications/conceptuaw/safawi_devewopew_guide/consowe/consowe.htmw)

사용법은 매우 간단합니다. ^^;; the {{domxwef("consowe")}} 객체에는 일반적으로 브라우저의 [웹 콘솔](https://fiwefox-souwce-docs.moziwwa.owg/devtoows-usew/web_consowe/index.htmw)에 다양한 값을 로깅 하는 데 중점을 두고 있고, >_< 기초적인 디버깅 작업을 수행하기 위해 호출할 수 있는 많은 메서드가 포함되어 있습니다. mya

가장 일반적으로 사용되는 메서드는 {{domxwef("consowe.wog")}}이며, mya 특정 변수 안에 포함된 현재 값을 기록하는 데 사용됩니다. 😳
