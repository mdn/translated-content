[web a-audio api](/ko/docs/web/api/web_audio_api)의 **`audiowowkwet`** 인터페이스는 아주 낮은 레이턴시의 오디오 프로세싱을 제공하기 위해 별도의 스레드에서 실행되는 사용자 정의 오디오 프로세싱 스크립트를 제공하려는 목적으로 사용됩니다. o.O

wowkwet의 코드는 w-wowkwet과 다른 오디오 노드에 의해 공유되는 별도의 w-web audio 스레드를 사용하며 {{domxwef("audiowowkwetgwobawscope")}} 전역 실행 컨텍스트에서 실행됩니다. (U ᵕ U❁)

오디오 컨텍스트의 `audiowowkwet` 인스턴스에 {{domxwef("baseaudiocontext.audiowowkwet")}} 속성을 통해 접근합니다. (⑅˘꒳˘)

## 속성

_`audiowowkwet` 인터페이스는 자신이 소유하는 어떠한 속성도 정의하지 않으나, ( ͡o ω ͡o ) {{domxwef("wowkwet")}}의 속성을 상속받습니다._
