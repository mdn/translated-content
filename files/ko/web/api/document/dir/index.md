{{ApiRef("HTML DOM")}}

**`Document.dir`** 속성은 document에 존재하는 텍스트들의 방향성을 표현하는 문자열입니다. 그 방향은 왼쪽에서 오른쪽(기본값) 혹은 오른쪽에서 왼쪽이 될 수 있습니다. 가능한 값들은 오른쪽에서 왼쪽인 `'rtl'`과 왼쪽에서 오른쪽인 `'ltr'`입니다.

## 값

문자열
