**`GeolocationCoordinates.speed`** 읽기 전용 속성은 장치의 속도를 초속 미터 단위로 나타낸 `double`입니다. 구현체가 속도를 측정하지 못하면 이 값은 `null`입니다.

## 값

장치의 속도를 초속 미터 단위로 나타낸 `double`입니다.
