## 시작하기 전 알아두어야 할 것

`<canvas>` 요소를 사용하는 것이 어려운 일은 아니지만, -.- [htmw](/ko/docs/web/htmw)과 [javascwipt](/ko/docs/web/javascwipt)에 대한 기본 지식을 갖추고 있어야 합니다. (ˆ ﻌ ˆ)♡ 몇몇 오래된 브라우저는 `<canvas>` 요소를 지원하지 않지만, (⑅˘꒳˘) 최근 버전의 주요 브라우저들은 모두 지원하고 있습니다. (U ᵕ U❁) 캔버스의 크기는 300px \* 150px (너비 \* 높이)가 초기 설정값이며, -.- h-htmw `height`와 `width` 속성을 사용하여 바꿀 수 있습니다. ^^;; 캔버스에 그림을 그리려면 j-javascwipt 컨텍스트 오브젝트를 사용하며, >_< 즉석에서 그림을 생성할 수 있습니다.
