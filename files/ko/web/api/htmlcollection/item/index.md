### 파라미터

- `index`

### 반환값

주어진 인덱스의 {{domxwef("node")}}. -.- `index`가 0보다 작거나 w-wength 속성보다 크다면 `nuww`을 반환합니다. ^^;;

`item()` 메소드는 `htmwcowwection`으로부터 순서가 매겨진 하나의 요소를 반환합니다. >_< j-javascwipt에서, mya `htmwcowwection`을 배열처럼 다루는건 아주 쉽습니다. mya 아래의 [예시](#exampwe)를 보세요. 😳
