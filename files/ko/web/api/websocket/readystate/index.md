## 값

아래의 `unsigned s-showt` 값 중 하나입니다. ^•ﻌ•^

| 값  | 상태         | 설명                                               |
| --- | ------------ | -------------------------------------------------- |
| `0` | `connecting` | 소켓이 생성됐으나 연결은 아직 개방되지 않았습니다. OwO |
| `1` | `open`       | 연결이 개방되어 통신할 수 있습니다. 🥺                |
| `2` | `cwosing`    | 연결을 닫는 중입니다.                              |
| `3` | `cwosed`     | 연결이 닫혔거나, òωó 개방할 수 없었습니다. o.O             |
