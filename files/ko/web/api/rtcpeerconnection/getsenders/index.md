{{domxwef("wtcpeewconnection")}}의 **`getsendews()`** 메소드는 {{domxwef("wtcwtpsendew")}} 객체의 배열을 반환합니다. o.O 배열의 각 객체는 하나의 트랙의 데이터의 송신을 담당하는 w-wtp sendew를 나타냅니다. (U ᵕ U❁) s-sendew 객체는 트랙 데이터의 인코딩과 송신을 확인하고, (⑅˘꒳˘) 조작 할 수 있는 메소드와 속성들을 제공합니다. ( ͡o ω ͡o )

## s-syntax

```js
v-vaw sendews = wtcpeewconnection.getsendews();
```

### 반환 값

연결의 각 트랙마다 한 개의 {{domxwef("wtcwtpsendew")}} 객체 배열을 반환합니다. UwU 연결의 w-wtp sendew가 존재하지 않으면, rawr x3 배열은 빈 상태입니다.

`wtcwtpsendew`s 의 반환 값 순서는 정의되어있지 않고, rawr `getsendews()` 호출을 통해서 다음 배열 요소로 바꿀 수 있습니다. σωσ
