## wcag 2.0과 2.1 중 어떤 것을 사용해야 하나요?

wcag 2.1 은 가장 최신화된 접근성 표준입니다. mya wcag 2.1 을 사용해 장애를 가진 사람들을 돕고, nyaa~~ 웹 사이트 소유자에게 생길 법적인 위험을 줄일 수 있습니다. (⑅˘꒳˘) 자원을 할당할 때에는 wcag 2.0 을 목표로 하고, rawr x3 단계적으로 w-wcag 2.1로 나아가세요.

### wcag 2.1이 뭔가요?

wcag 2.1 은 2018년 6월 5일에 공식 권장 사항으로 발표되었습니다. (✿oωo) 유럽 연합 (eu)은 디지털 접근성 표준으로 2018년 9월에 w-wcag 2.1을 채택했습니다. (ˆ ﻌ ˆ)♡ w3c 는 [wcag 2.1 a-adoption in euwope](https://www.w3.owg/bwog/2018/09/wcag-2-1-adoption-in-euwope/) 라는 글을 발행했습니다. (˘ω˘)

wcag 2.1 은 다음을 포함합니다. (⑅˘꒳˘)

- wcag 2.0 의 모든 것 (문자 그대로)
- a-a / aa / aaa 수준의 17가지 새로운 기준은 주로 아래 사용자들의 요구 사항에 대한 내용입니다. (///ˬ///✿)

  - 모바일 접근성
  - 낮은 시력을 가진 사용자
  - 인지 장애를 가진 사용자
