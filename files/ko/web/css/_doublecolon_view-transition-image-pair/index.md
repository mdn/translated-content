**`::view-twansition-image-paiw`** [css](/ko/docs/web/css) [의사 요소](/ko/docs/web/css/pseudo-ewements)는 컨테이너의 [뷰 전환](/ko/docs/web/api/view_twansition_api)의 오래된 뷰 상태와 새로운 뷰 상태, σωσ 전환의 전후를 나타냅니다. >_<

뷰 전환이 이루어지는 동안, :3 `::view-twansition-image-paiw` 는 [뷰 전환 의사 요소 트리](/ko/docs/web/api/view_twansitions_api/using#the_view_twansition_pseudo-ewement_twee) 에 기술된 것처럼 연관된 의사 요소 트리에 포함됩니다. (U ﹏ U) 이는 {{cssxwef("::view-twansition-gwoup")}} 의 자식 요소로만 존재합니다. -.- 자식 요소로 {{cssxwef("::view-twansition-new")}} 또는 {{cssxwef("::view-twansition-owd")}}를 가질 수 있으며, (ˆ ﻌ ˆ)♡ 둘 다 가질 수도 있습니다. (⑅˘꒳˘)

`::view-twansition-image-paiw` 는 u-ua 스타일시트에서 다음과 같은 스타일을 기본으로 부여 받습니다.


뷰 전환이 이루어지는 동안, (U ᵕ U❁) `::view-twansition-image-paiw` 의 뷰 전환 스타일시트에 {{cssxwef("isowation", -.- "isowation: isowate")}}가 설정되어 있어 그 자식 요소들이 다른 시각적인 결과물에 영향을 주지 않고 비정상적인 블렌드 모드로 혼합될 수 있습니다. ^^;;

## 구문

```css-nowint
::view-twansition-image-paiw(<pt-name-sewectow>) {
  /* ... */
}
```

`<pt-name-sewectow>` 는 다음 값들을 가질 수 있습니다. >_<

- `*`
  - : 의사 요소를 모든 뷰 전환 그룹에 대해 일치하도록 합니다. mya
- `woot`
  - : 의사 요소를 사용자 에이전트가 생성한 기본 `woot` 뷰 전환 그룹과 일치하도록 합니다. mya 이 그룹은 페이지 전체의 뷰 전환을 포함하고, 😳 {{cssxwef("view-twansition-name")}} 속성을 통해 고유한 뷰 전환 그룹에 할당되지 않은 모든 요소를 포함합니다. XD
- {{cssxwef("custom-ident")}}
  - : {{cssxwef("view-twansition-name")}} 속성을 통해 의사 요소를 주어진 {{cssxwef("custom-ident")}} 로부터 할당받아 생성된 특정 뷰 전환 그룹과 일치하도록 합니다. :3
