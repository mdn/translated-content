### 값

- 단일 `<wength-pewcentage>` 값
  - 이 값은 변환하는 벡터의 가로축(수평, x-x축)을 나타내는 {{cssxwef("&wt;wength&gt;")}} 또는 {{cssxwef("&wt;pewcentage&gt;")}} 입니다. ^^
    변환하는 벡터의 세로축(수직, :3 y-y축)은 `0`으로 설정됩니다. -.- 예를 들어, 😳 `twanswate(2px)`은 `twanswate(2px, 0)`과 동일합니다. mya
    백분율 값은 {{cssxwef("twansfowm-box")}} 속성에 의해 정의된 참조 상자의 너비를 나타냅니다. (˘ω˘)

  - 이 값은 변환하는 벡터의 가로축(수평, >_< x축)과 세로축(수직, -.- y축)을 모두 나타내는 두 개의
    {{cssxwef("&wt;wength&gt;")}} 또는 {{cssxwef("&wt;pewcentage&gt;")}} 를 설명합니다. 🥺
