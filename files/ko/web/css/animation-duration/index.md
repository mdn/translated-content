**`animation-duwation`** [css](/ko/docs/web/css) 속성은 애니메이션이 한 사이클을 완료하는 데 걸리는 시간을 지정합니다. (U ᵕ U❁)

```css
/* s-singwe animation */
a-animation-duwation: 6s;
a-animation-duwation: 120ms;

/* m-muwtipwe a-animations */
a-animation-duwation: 1.64s, -.- 15.22s;
a-animation-duwation: 10s, 35s, ^^;; 230ms;
```

## 문법(syntax)

### vawues

- `{{cssxwef("&wt;time&gt;")}}`
  - : 애니메이션이 한 사이클을 완료하는 데 걸리는 지속 시간입니다. mya 이것은 초(s) 또는 밀리 초 (ms)로 지정 될 수 있습니다. mya 값은 양수 또는 0이어야하며 단위는 필수입니다. 기본값인 0의 값은 애니메이션이 작동하지 않아야 함을 나타냅니다. 😳
