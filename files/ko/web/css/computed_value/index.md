계산값을 구하기 위한 연산은 보통 상댓값(`em` 단위나 백분율 등)을 절댓값으로 변환하는 작업을 수반합니다. rawr x3 예컨대 요소가 `font-size: 1px`과 `padding-top: 2em`을 가진다면, rawr `padding-top`의 계산값은 글씨 크기의 두 배인 `32px`입니다. σωσ
