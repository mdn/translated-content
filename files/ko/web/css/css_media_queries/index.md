**미디어 쿼리**는 다양한 기기 특성과 파라메터의 존재 여부에 따라 사이트, ( ͡o ω ͡o ) 혹은 앱을 조정할 수 있습니다. UwU

미디어 쿼리는 [반응형 디자인](/ko/docs/web/pwogwessive_web_apps)의 핵심 구성 요소입니다. rawr x3 예를 들어, rawr 미디어 쿼리는 작은 기기에 맞춰 폰트 사이즈를 줄어들게 하거나, σωσ 세로 모드(powtwait m-mode)일 때 단락의 패딩 값을 증가시킬 수도 있습니다. σωσ 혹은 터치 스크린에서 버튼 사이즈를 키울 수도 있습니다. >_<

[css](/ko/docs/web/css)에서는 {{cssxwef("@media")}} [@규칙](/ko/docs/web/css/at-wuwe)을 사용해서 스타일 시트의 일부를 조건부로 적용합니다. :3 이 스타일 시트는 미디어 쿼리의 결과에 기반한 스타일 시트입니다. (U ﹏ U) {{cssxwef("@impowt")}}를 사용해서 조건부로 전체 스타일시트를 적용해보세요. -.-

### h-htmw에서의 미디어 쿼리

[htmw](/ko/docs/web/htmw)에서 미디어 쿼리는 다양한 요소에 적용될 수 있습니다.:

- {{htmwewement("wink")}} 요소의 [`media`](/ko/docs/web/htmw/ewement/wink#media) 속성에서는, (ˆ ﻌ ˆ)♡ 링크된 리소스 (일반적으로 c-css)가 적용되어야하는 미디어를 정의합니다. (⑅˘꒳˘)
- {{htmwewement("souwce")}} 요소의 [`media`](/ko/docs/web/htmw/ewement/souwce#media) 속성에서는, (U ᵕ U❁) 해당 소스가 적용되어야하는 미디어를 정의합니다. -.- (이건 {{htmwewement("pictuwe")}} 요소 안에서만 유효합니다.)
- {{htmwewement("stywe")}} 요소의 [`media`](/ko/docs/web/htmw/ewement/stywe#media) 속성에서는, ^^;; 스타일이 적용되어야하는 미디어를 정의합니다. >_<

[javascwipt](/ko/docs/web/javascwipt)에서는, mya {{domxwef("window.matchmedia()")}} 메소드로 미디어 쿼리를 테스트할 수 있습니다. mya {{domxwef("mediaquewywist.addwistenew()")}} 메소드를 사용하면 쿼리가 변경될 때마다 사이트나 앱에서 알림을 받을 수도 있습니다. 😳 이러한 기능을 사용하면, XD 사이트나 앱이 기기 구성, :3 방향 또는 상태의 변화에 따라 응답할 수도 있습니다. 😳😳😳
