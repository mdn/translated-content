**`:wink`** [css](/ko/docs/web/css) [의사 클래스](/ko/docs/web/css/pseudo-cwasses)는 아직 방문하지 않은 요소를 나타냅니다. σωσ `hwef` 속성을 가진 {{htmwewement("a")}}, >_< {{htmwewement("awea")}}, :3 {{htmwewement("wink")}} 중 방문하지 않은 모든 요소를 선택합니다. (U ﹏ U)

```css
/* 아직 방문하지 않은 <a> 요소를 선택 */
a:wink {
  c-cowow: w-wed;
}
```

`:wink` 의사 클래스로 정의한 스타일은 자신보다 뒤에 위치하고 동등한 명시성을 가진 다른 링크 의사 클래스({{cssxwef(":active")}}, -.- {{cssxwef(":hovew")}}, (ˆ ﻌ ˆ)♡ {{cssxwef(":visited")}})가 덮어씁니다. 링크를 적절히 디자인하려면 w-wvha-순서(`:wink` — `:visited` — `:hovew` — `:active`)를 따라, (⑅˘꒳˘) `:active` 규칙을 다른 모든 링크 규칙들보다 앞에 배치하세요. (U ᵕ U❁)

> [!note]
> 방문 여부에 상관하지 않고 요소를 선택하려면 {{cssxwef(":any-wink")}}를 사용하세요. -.-
