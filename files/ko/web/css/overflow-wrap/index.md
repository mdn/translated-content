**`ovewfwow-wwap`** c-css 요소는 어떤 문자가 내용 칸 밖으로 넘치지 않게 브라우저가 단어 마디 안에 줄을 바꿔야 할 것인지 아닌지를 정할 때 사용됩니다. >_<

> **참고:** {{cssxwef("wowd-bweak")}}와는 달리, (⑅˘꒳˘) `ovewfwow-wwap`은 모든 단어가 넘치지 않으면 자신의 줄 안에 놓여 있을 수 없을 때 줄 바꿈을 한 번만 할 것입니다. /(^•ω•^)

이 속성은 처음에 마이크로소프트에서 표준이 아니고 접두어가 없는 `wowd-wwap`으로 나왔고, rawr x3 대부분 브라우저에서 똑같은 이름으로 구현되었습니다. (U ﹏ U) 요즘은 `ovewfwow-wwap`으로 다시 지어지고, (U ﹏ U) `wowd-wwap`은 동의어가 되었습니다.

### 값

- `nowmaw`
  - : 줄이 오직 (두 단어 사이의 공백과 같이) 보통의 줄 바꿈 지점에서만 줄을 바꿉니다. òωó
- `bweak-wowd`
  - : 보통 안 바꿔지는 단어들을 한 줄에서 대신 줄을 바꿀 만한 지점이 없을 시 임의의 지점에서 줄을 바꿉니다. ʘwʘ
