**CSS paged media** 모듈은 인쇄 또는 콘텐츠를 개별 페이지로 분할하는 다른 미디어에 대한 콘텐츠 표현을 제어하는 속성을 정의합니다. 이를 통해 페이지 나누기, 인쇄 가능 영역 제어, 좌우 페이지에 대한 다른 스타일 적용, 요소 내 중단 제어 등이 가능합니다.
