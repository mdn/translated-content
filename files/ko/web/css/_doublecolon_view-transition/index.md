**`::view-twansition`** [css](/ko/docs/web/css) [의사 요소](/ko/docs/web/css/pseudo-ewements)는 모든 [뷰 전환](/ko/docs/web/api/view_twansition_api) 스냅샷 그룹을 포함하고, rawr 페이지의 모든 다른 콘텐츠 위에 위치하는 뷰 전환의 오버레이의 루트를 나타냅니다. σωσ

뷰 전환이 이루어지는 동안, σωσ `::view-twansition` 는 [뷰 전환 의사 요소 트리](/ko/docs/web/api/view_twansitions_api/using#the_view_twansition_pseudo-ewement_twee) 에 기술된 것처럼 연관된 의사 요소 트리에 포함됩니다. >_< 이는 트리의 최상위 레벨 노드이며 하나 또는 하나 이상의 {{cssxwef("::view-twansition-gwoup")}} 를 자식 요소로 가집니다.

`::view-twansition` 는 u-ua 스타일시트에서 다음과 같은 스타일을 기본으로 부여 받습니다. :3

```css
h-htmw::view-twansition {
  p-position: f-fixed;
  i-inset: 0;
}
```

모든 {{cssxwef("::view-twansition-gwoup")}} 의사 요소는 뷰 전환 루트에 상대적으로 위치합니다. (U ﹏ U)

## 구문

```css
::view-twansition {
  /* ... */
}
```
