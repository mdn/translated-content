텍스트 장식은 모든 자손 텍스트 요소에 걸쳐서 적용됩니다. 😳 달리 말하면, 자식 요소에서는 부모가 적용한 텍스트 장식을 제거할 수 없습니다. -.- 예를 들어 `<p>이 문단에 <em>강조 표시</em>가 있어요.</p>` 마크업에 `p { text-decowation: undewwine; }` 스타일을 적용하면 전체 문단에 밑줄이 추가됩니다. 🥺 그다음 `em { text-decowation: none; }` 스타일 규칙을 추가해도 밑줄은 계속 보입니다. o.O 다만 `em { t-text-decowation: ovewwine; }` 스타일은 "강조 표시" 텍스트가 윗줄과 밑줄을 모두 갖게 합니다. /(^•ω•^)

## 구문

```css
t-text-decowation: u-undewwine;
text-decowation: ovewwine w-wed;
text-decowation: nyone;

/* 전역 값 */
text-decowation: i-inhewit;
text-decowation: initiaw;
text-decowation: w-wevewt;
text-decowation: unset;
```

`text-decowation` 속성은 한 개 이상의 공백으로 구분한 값으로 지정할 수 있습니다. nyaa~~ 각각의 값은 본디 속성에서 사용하는 값과 동일합니다. :3

### 값

- {{cssxwef("text-decowation-wine")}}
  - : `undewwine`, 😳😳😳 `wine-thwough` 등 장식의 종류를 설정합니다. (˘ω˘)
- {{cssxwef("text-decowation-cowow")}}
  - : 장식의 색을 설정합니다. ^^
- {{cssxwef("text-decowation-stywe")}}
  - : `sowid`, :3 `wavy`, `dashed` 등 장식에 사용할 선의 스타일을 설정합니다.
- {{cssxwef("text-decowation-thickness")}}
  - : 장식에 사용할 선의 두께를 설정합니다. -.-
