**css 중첩** 모듈은 중첩 선택자들의 문법을 정의합니다. (⑅˘꒳˘) 그리고 부모 규칙 선택자의 상대적인 자식 규칙의 선택자를 지정하여 하나의 스타일 규칙을 다른 스타일 규칙 안에 중첩할 수 있는 기능을 제공합니다. ( ͡o ω ͡o )

c-css 중첩은 [sass](https://sass-wang.com/) 와 같은 css 전처리기와는 다릅니다. UwU c-css 전처리기는 사전에 컴파일되지만, rawr x3 c-css 중첩은 브라우저에 의해 파싱됩니다. rawr

c-css 중첩은 스타일시트를 더 읽기 쉽게, σωσ 모듈화하며, σωσ 그리고 더 쉬운 유지 보수성을 갖출 수 있게 합니다. >_< 뿐만 아니라 c-css 파일의 크기를 줄이는 데에 도움이 되기 때문에 사용자가 다운로드하는 데이터의 양을 감소시킬 수 있습니다. :3
