[CSS](/ko/docs/Web/CSS) 속성의 **결정값**은 {{domxref("Window.getComputedStyle", "getComputedStyle()")}}이 반환하는 값입니다.

속성 대부분은 결정값이 [계산값](/ko/docs/Web/CSS/computed_value)이지만, {{cssxref("width")}}와 {{cssxref("height")}}를 포함한 기존 속성은 [사용값](/ko/docs/Web/CSS/used_value)입니다. 속성 별 자세한 사항은 아래의 명세를 참고하세요.
