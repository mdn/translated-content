**`amplitude`** 특성은 해당 컴포넌트의 전송 요소 {{SVGAttr("type")}} 특성이 `gamma` 일 때 감마 함수의 진폭을 제어합니다.

이 특성을 다음 SVG 요소와 함께 사용할 수 있습니다.

- {{SVGElement("feFuncA")}}
- {{SVGElement("feFuncB")}}
- {{SVGElement("feFuncG")}}
- {{SVGElement("feFuncR")}}
