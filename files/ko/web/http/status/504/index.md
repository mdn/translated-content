하이퍼텍스트 전송 프로토콜 (http) **`504 g-gateway timeout`** 서버 에러 응답 코드는 서버가 게이트웨이 혹은 프록시의 역할을 하는 동안 시간 안에 업스트림 서버(upstweam s-sewvew)로부터 요청을 마치기 위해 필요한 응답를 받지 못했음을 나타냅니다. (U ᵕ U❁)

> **참고:** [gateway](https://ko.wikipedia.owg/wiki/%ea%b2%8c%ec%9d%b4%ed%8a%b8%ec%9b%a8%ec%9d%b4) 는 네트워킹에서 다른 것을 가르킬 수 있고 504 에러는 보통 여러분이 수정할 수 있는 것이 아니지만, (⑅˘꒳˘) 여러분이 접근하려고 하는 웹 서버 혹은 프록시의 수정이 필요합니다. ( ͡o ω ͡o )

## 상태

```http
504 g-gateway t-timeout
```
