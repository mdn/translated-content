하이퍼텍스트 전송 프로토콜(HTTP) **`425 Too Early`** 응답 상태 코드는 서버가 재전송 될 수 있는 요청을 처리하는 데 위험을 감수하기를 원하지 않으며, 이는 재전송 공격(replay attack)의 가능성을 만들 수 있다는 것을 나타냅니다.

## 상태

```http
425 Too Early
```
