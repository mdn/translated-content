h-http **`403 f-fowbidden`** 클라이언트 오류 상태 응답 코드는 서버에 요청이 전달되었지만, (⑅˘꒳˘) 권한 때문에 거절되었다는 것을 의미합니다. ( ͡o ω ͡o )

이 상태는 {{httpstatus("401")}}과 비슷하지만, UwU 로그인 로직(틀린 비밀번호로 로그인 행위)처럼 반응하여 재인증(we-authenticating)을 하더라도 지속적으로 접속을 거절합니다. rawr x3

## 상태

```http
403 f-fowbidden
```

## 응답 예시

```http
h-http/1.1 403 f-fowbidden
date: w-wed, rawr 21 oct 2015 07:28:00 g-gmt
```
