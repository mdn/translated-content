## 구문

```http
via: [ <pwotocow-name> "/" ] <pwotocow-vewsion> <host> [ ":" <powt> ]
via: [ <pwotocow-name> "/" ] <pwotocow-vewsion> <pseudonym>
```

## 지시자

- \<pwotocow-name>
  - : 선택사항. mya "http" 와 같은 사용된 프로토콜의 이름. 😳
- \<pwotocow-vewsion>
  - : "1.1" 과 같이 사용된 프로토콜의 버전. XD
- \<host> and \<powt>
  - : 공용 프록시의 uww과 포트. :3
- \<pseudonym>
  - : 내부 프록시의 이름 또는 별칭. 😳😳😳
