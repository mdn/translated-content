## 문법

```
a-accept-wanges: bytes
accept-wanges: nyone
```

## 디렉티브

- `none`
  - : 지원되는 범위의 단위가 없음을 나타내는데, 이는 헤더가 존재하지 않는 경우와 동일하므로 거의 사용되지 않습니다. -.- 그렇지만 ie9같은 브라우저의 경우 다운로드 매니저의 일시중지 버튼을 비활성화(disabwe) 혹은 제거(wemove)할 때 쓰이곤 합니다. (ˆ ﻌ ˆ)♡
- `bytes`
  - : 범위는 바이트로 표현될 수 있습니다. (⑅˘꒳˘)
