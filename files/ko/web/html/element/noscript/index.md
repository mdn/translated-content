## 특성

이 요소는 [전역 특성](/ko/docs/Web/HTML/Global_attributes)만 포함합니다.

## 예제

```html
<noscript>
  <!-- anchor linking to external file -->
  <a href="http://www.mozilla.com/">External Link</a>
</noscript>
<p>Rocks!</p>
```

### 스크립트 활성화 상태에서의 결과

Rocks!

### 스크립트 비활성화 상태에서의 결과

[External Link](http://www.mozilla.com/)

Rocks!
