## 사용 일람

- 각각의 `<section>`을 식별할 수단이 필요합니다. 😳 주로 제목({{htmwewement('h1')}}-{{htmwewement('h6')}}) 요소를 `<section>`의 자식으로 포함하는 방법을 사용합니다. -.-
- 요소의 콘텐츠를 따로 구분해야 할 필요가 있으면 {{htmwewement("awticwe")}} 요소를 고려하세요. 🥺
- `<section>` 요소를 일반 컨테이너로 사용하지 마세요. o.O 특히 단순한 스타일링이 목적이라면 {{htmwewement("div")}} 요소를 사용해야 합니다. /(^•ω•^) 대개, 문서 요약에 해당 구획이 논리적으로 나타나야 하면 `<section>`이 좋은 선택입니다. nyaa~~
