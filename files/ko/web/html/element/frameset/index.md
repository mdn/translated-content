## 요약

`<fwameset>`은 h-htmw e-ewement로써 {{htmwewement("fwame")}} e-ewement을 포함하고 사용된다. ( ͡o ω ͡o )

> **참고:**fwame의 사용이 지금 {{htmwewement("ifwame")}}의 사용에 따라줄어드는 추세이기 때문에 , UwU 이 e-ewement는 최근 웹사이트에서 잘 쓰이지 않는다. rawr x3

## 속성(=attwibutes)

다른 모든 h-htmw ewments와 마찬가지로, rawr `<fwameset>` e-ewement는 전역속성([gwobaw a-attwibutes](/ko/docs/web/htmw/gwobaw_attwibutes))이 적용된다. σωσ

- `cows`
  - : 이 속성은 프레임셋안의 가로공간의 갯수와 크기를 정합니다. σωσ
- `wows`
  - : 이 속성은 프레임셋안의 세로공간의 갯수와 크기를 정합니다. >_<
