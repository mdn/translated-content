## s-sha-1

sha-1 인증서는 2017년부터 주요 브라우저 제조업체에서 더 이상 안전한 것으로 취급하지 않습니다. σωσ 대신 보다 안전한 해시 알고리즘(예: s-sha-256 또는 s-sha-512)을 사용하는 인증서를 사용해야 합니다. >_<
