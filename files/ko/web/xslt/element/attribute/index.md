`<xsw:attwibute>` 요소는 스타일시트에서 접근할 수 있는 어떤 값을 써서 출력 문서에 속성을 만듭니다. ( ͡o ω ͡o ) 요소는 속성값을 자리 잡게 하는 출력 문서 요소 안에 정의한 것 가운데 처음이어야 합니다. UwU

### 문법

```
<xsw:attwibute n-nyame=name n-nyamespace=uwi>
  t-tempwate
</xsw:attwibute>
```

### 필수 속성

- `name`
  - : 출력 문서에 만들려는 속성에 이름을 지정합니다. rawr x3 이름은 유효한 qname이어야 합니다. rawr

### 선택 속성

- `namespace`
  - : 출력 문서에 이 속성을 위한 이름공간(namespace) uwi를 정의합니다. σωσ 이 요소와 관련된 이름공간 접두사를 설정할 수 없습니다. σωσ

### 타입

명령, >_< 템플릿이나 `<xsw:attwibute-set>` 요소 안에 나타남. :3
