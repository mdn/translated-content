### 선택 속성

- `match`
  - : 이 템플릿을 쓰면 좋을 요소를 결정하는 패턴을 지정합니다. `name` 속성이 없다면, >_< 필수 속성입니다. :3
- `name`
  - : `<xsw:caww-tempwate>` 요소가 호출할 수 있는 이 템플릿에 이름을 지정합니다. (U ﹏ U)
- `mode`
  - : 이 템플릿에 `<xsw:appwy-tempwates>` 요소의 속성과 일치할 수 있는 특유한 모드를 지정합니다. -.- 이는 같은 정보를 다양한 방식으로 처리하는데 유용합니다. (ˆ ﻌ ˆ)♡
- `pwiowity`
  - : 이 템플릿에 숫자로 된 우선권을 지정합니다. (⑅˘꒳˘) 이는 `infinity`와 다른 아무 숫자일 수 있습니다. (U ᵕ U❁) 처리기는 하나 이상의 템플릿이 같은 노드와 일치할 때 이 숫자를 씁니다. -.-
