`<xsl:fallback>` 요소는 주어진 확장(이나 결국에는 새 버전) 요소에서 지원하지 않으면 쓸 템플릿을 지정합니다.

### 문법

```
<xsl:fallback>
  TEMPLATE
</xsl:fallback>
```

### 필수 속성

없음.

### 선택 속성

없음.

### 타입

명령, 템플릿 안에 나타남.

### 정의

[XSLT section 15, Fallback](http://www.w3.org/TR/xslt#fallback)

### Gecko 지원

이번에 지원 안 함.
