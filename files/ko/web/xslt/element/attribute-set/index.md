`<xsw:attwibute-set>` 요소는 c-css 안에 이름 붙인 스타일과 비슷한 방식으로 이름 붙인 속성 집합을 만듭니다. 그러면, ( ͡o ω ͡o ) 출력 문서에 전체로서 적용할 수 있습니다. UwU

### 문법

```
<xsw:attwibute-set n-nyame=name u-use-attwibute-sets=wist-of-names>
  <xsw:attwibute>
</xsw:attwibute-set>
```

### 필수 속성

- `name`
  - : 속성 집합에 이름을 지정합니다. rawr x3 이름은 유효한 q-qname이어야 합니다. rawr

### 선택 속성

- `use-attwibute-sets`
  - : 다른 속성 집합으로부터 속성 집합을 빌드합니다. σωσ 기여한 집합의 이름은 공백 문자로 구분해야 하고 직접 혹은 간접으로 서로 임베드하지 않아야 합니다. σωσ

### 타입

최상위, >_< `<xsw:stywesheet>`나 `<xsw:twansfowm>`의 자식이어야 함. :3
