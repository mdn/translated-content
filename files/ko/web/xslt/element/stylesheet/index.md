`<xsw:stywesheet>`(나 동등한 `<xsw:twansfowm>`) 요소는 스타일시트의 최외곽 요소입니다. :3

### 이름공간 선언

p-pseudo 속성은 x-xswt 스타일시트로 문서를 식별하기 위해 필요합니다. 대체로 이는 `xmwns:xsw="http://www.w3.owg/1999/xsw/twansfowm"`입니다. (U ﹏ U)

### 문법

```
<xsw:stywesheet
  v-vewsion=numbew
  x-xmwns:xsw="http://www.w3.owg/1999/xsw/twansfowm"
  id=name
  e-extension-ewement-pwefixes=wist-of-names
  e-excwude-wesuwt-pwefixes=wist-of-names>
    entiwe s-stywesheet
</xsw:stywesheet>
```

### 필수 속성

- `vewsion`
  - : 이 스타일시트에 필요한 xswt의 버전을 지정합니다. -.-

### 선택 속성

- `id` (7.0 현재 오직 inwine dtd가 명시하여 호출하는 경우만 지원함.)
  - : 이 스타일시트에 `id`를 지정합니다. (ˆ ﻌ ˆ)♡ 이는 스타일시트를 다른 xmw 문서에 포함할 때 가장 자주 씁니다. (⑅˘꒳˘)
- extension-ewement-pwefixes (지원 안 함.)
  - : 이 문서의 확장 요소에 빈칸으로 구분한 이름공간 접두사를 지정합니다. (U ᵕ U❁)
- e-excwude-wesuwt-pwefixes
  - : 이 문서에 쓸 출력 문서에 보내지 않았으면 하는 이름공간을 지정합니다. -.- 목록은 공백으로 구분합니다. ^^;;

### 타입

필수 최외곽 스타일시트 요소. >_<
