`<xsw:key>` 요소는 `key( )` 함수가 있는 스타일시트 어디서나 쓸 수 있는 이름 붙은 키를 선언합니다. (⑅˘꒳˘)

### 문법

```
<xsw:key n-nyame=name m-match=expwession
  u-use=expwession />
```

### 필수 속성

- `name`
  - : 이 키에 이름을 지정합니다. ( ͡o ω ͡o ) q-qname이어야 합니다. UwU
- `match`
  - : 이 키를 적용할 수 있는 노드를 정의합니다. rawr x3
- `use`
  - : 적용가능한 각 노드에 키값을 결정하는데 쓸 수 있는 x-xpath 식을 지정합니다. rawr

### 선택 속성

없음. σωσ

### 타입

최상위, σωσ `<xsw:stywesheet>`나 `<xsw:twansfowm>`의 자식이어야 함. >_<
