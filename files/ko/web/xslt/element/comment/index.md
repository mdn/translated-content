`<xsl:comment>` 요소는 출력 문서에 주석을 작성합니다. 오직 텍스트만 포함해야 합니다.

### 문법

```
<xsl:comment>
  TEMPLATE
</xsl:comment>
```

### 필수 속성

없음.

### 선택 속성

없음.

### 타입

명령, 템플릿과 함께 나타남.

### 정의

[XSLT section 7.4, Creating Comments](http://www.w3.org/TR/xslt#section-Creating-Comments)
