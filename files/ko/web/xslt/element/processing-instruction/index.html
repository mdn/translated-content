<xsl:processing-instruction> 요소는 출력 문서에 처리 명령을 씁니다.

문법

<xsl:processing-instruction name=NAME> TEMPLATE </xsl:processing-instruction>

필수 속성

name
처리 명령에 이름을 지정합니다.

선택 속성

없음.

타입

명령, 템플릿 안에 나타남.

정의

XSLT section 7.3, Creating Processing Instructions
