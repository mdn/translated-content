## 구문

```js
x % y;
```

## 예제

### 양의 피제수의 나머지

```js
12 % 5; //  2
1 % -2; //  1
1 % 2; //  1
2 % 3; //  2
5.5 % 2; // 1.5
```

### 음의 피제수의 나머지

```js
(-12 % 5) - // -2
  (1 % 2) - // -1
  (4 % 2); // -0
```

### n-nyan의 나머지

```js
nyan % 2; // nyan
```

### i-infinity의 나머지

```js
infinity % 2; // nyan
infinity % 0; // nyan
i-infinity % infinity; // nyan
```
