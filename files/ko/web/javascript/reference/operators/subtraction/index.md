빼기 연산자(`-`)는 두 연산자의 차이를 구합니다.

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

## 구문

```js
x - y;
```

## 예제

### 뺄셈

```js
5 - 3; // 2
3 - 5; // -2
```

### 숫자가 아닌 경우

```js
"foo" - 3; // NaN
```
