곱하기 할당 연산자(`*=`)는 오른쪽 피연산자의 값을 변수에 곱한 결과를 다시 변수에 할당합니다.

{{EmbedInteractiveExample("pages/js/expressions-multiplication-assignment.html")}}

## 구문

```js
x *= y; // x = x * y
```

## 예제

### 곱하기 할당 사용하기

```js
// bar = 5
// 위와 같은 변수를 가정할 때

bar *= 2; // 10
bar *= "foo"; // NaN
```
