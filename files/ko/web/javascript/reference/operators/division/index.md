{{jsSidebar("Operators")}}

나누기 연산자(`/`)는 왼쪽 피연산자를 피제수, 오른쪽 피연산자를 제수로 하여 몫을 구합니다.

{{InteractiveExample("JavaScript Demo: Expressions - Division operator")}}

```js interactive-example
console.log(12 / 2);
// Expected output: 6

console.log(3 / 2);
// Expected output: 1.5

console.log(6 / "3");
// Expected output: 2

console.log(2 / 0);
// Expected output: Infinity
```

## 구문

```js
x / y;
```

## 예제

### 기본 나눗셈

```js
1 / 2; // 0.5

Math.floor(3 / 2); // 1

1.0 / 2.0; // 0.5
```

### 0으로 나누기

```js
2.0 / 0; // Infinity

2.0 / 0.0; // Infinity, 0.0 === 0이기 때문

2.0 / -0.0; // -Infinity
```
