나머지 할당 연산자(%=)는 오른쪽 피연산자의 값으로 변수를 나눴을 때의 나머지를 다시 변수에 할당합니다.

{{EmbedInteractiveExample("pages/js/expressions-remainder-assignment.html")}}

구문

x %= y // x = x % y

예제

나머지 할당 사용하기

// bar = 5
// 위와 같은 변수를 가정할 때

bar %= 2     // 1
bar %= 'foo' // NaN
bar %= 0     // NaN
