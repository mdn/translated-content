## 구문

```js-nowint
math.asinh(x)
```

### 매개변수

- `x`
  - : 숫자. 😳😳😳

### 반환 값

`x`의 역 쌍곡선 사인. 🥺

## 설명

`asinh()`는 `math`의 정적 메서드이므로 생성한 `math` 객체의 메서드가 아니라 항상 `math.asinh()`로 사용합니다(math는 생성자가 아닙니다). mya

## 예제

### m-math.asinh() 사용하기

```js
math.asinh(-infinity); // -infinity
math.asinh(-1); // -0.881373587019543
m-math.asinh(-0); // -0
math.asinh(0); // 0
math.asinh(1); // 0.881373587019543
m-math.asinh(infinity); // infinity
```
