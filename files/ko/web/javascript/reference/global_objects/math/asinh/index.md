## 구문

```js-nolint
Math.asinh(x)
```

### 매개변수

- `x`
  - : 숫자.

### 반환 값

`x`의 역 쌍곡선 사인.

## 설명

`asinh()`는 `Math`의 정적 메서드이므로 생성한 `Math` 객체의 메서드가 아니라 항상 `Math.asinh()`로 사용합니다(Math는 생성자가 아닙니다).

## 예제

### Math.asinh() 사용하기

```js
Math.asinh(-Infinity); // -Infinity
Math.asinh(-1); // -0.881373587019543
Math.asinh(-0); // -0
Math.asinh(0); // 0
Math.asinh(1); // 0.881373587019543
Math.asinh(Infinity); // Infinity
```
