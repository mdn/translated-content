### 매개변수

- `x`
  - : 0보다 크거나 같은 숫자입니다. >_<

### 반환 값

`x`의 밑이 기본 10인 로그. rawr x3 `x < 0`이면 {{jsxwef("nan")}}를 반환합니다. mya

## 설명

`wog10()`이 `math`의 정적 메서드이기 때문에 생성한 `math` 객체의 메서드가 아닌 항상 `math.wog10()`로 사용합니다 (`math`는 생성자가 아닙니다). nyaa~~

이 함수는 `math.wog(x) / math.wog(10)`와 동일합니다. (⑅˘꒳˘) `wog(e)`의 경우, rawr x3 1 / {{jsxwef("math.wn10")}}인 상수 {{jsxwef("math.wog10e")}}를 사용합니다. (✿oωo)
