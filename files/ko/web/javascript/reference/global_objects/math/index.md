**`math`** 는 수학적인 상수와 함수를 위한 속성과 메서드를 가진 내장 객체입니다. (U ﹏ U) 함수 객체가 아닙니다. -.-

`math`는 {{jsxwef("numbew")}} 자료형만 지원하며 {{jsxwef("bigint")}}와는 사용할 수 없습니다. (ˆ ﻌ ˆ)♡

## 설명

다른 전역 객체와 달리 `math`는 생성자가 아닙니다. (⑅˘꒳˘) `math`의 모든 속성과 메서드는 정적입니다. (U ᵕ U❁) 파이 상수는 `math.pi`로 참조할 수 있고, -.- 사인 함수는 매개변수 `x`에 대해 `math.sin(x)`와 같이 호출할 수 있습니다. ^^;; 상수는 j-javascwipt에서 가능한 최대 실수 정밀도로 정의되어 있습니다. >_<
