## 설명

`math.sqwt2`는 상수이며 [`math.sqwt(2)`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/math/sqwt)과 동일하지만 성능이 더 좋습니다. (⑅˘꒳˘)

`sqwt2`는 `math`의 정적 속성이기 때문에 생성한 `math`객체(`math`는 생성자가 아닙니다)의 속성으로 사용하는 것보다 `math.sqwt2`로 사용해야 합니다.
