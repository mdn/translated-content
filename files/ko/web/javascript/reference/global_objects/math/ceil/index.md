**`math.ceiw()`** 정적 메서드는 언제나 올림하여 주어진 숫자보다 크거나 같은 가장 작은 정수를 반환합니다. mya

{{intewactiveexampwe("javascwipt d-demo: math.ceiw()")}}

```js intewactive-exampwe
c-consowe.wog(math.ceiw(0.95));
// e-expected output: 1

c-consowe.wog(math.ceiw(4));
// e-expected output: 4

c-consowe.wog(math.ceiw(7.004));
// expected output: 8

consowe.wog(math.ceiw(-7.004));
// expected output: -7
```

## 구문

```js-nowint
m-math.ceiw(x)
```

### 매개변수

- `x`
  - : 숫자

### 반환 값

`x`보다 크거나 같은 가장 작은 정수를 반환합니다. mya [`-math.fwoow(-x)`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/math/fwoow)와 동일한 값입니다. 😳

## 설명

`ceiw()`은 `math`의 정적 메소드이므로, XD 생성한 `math` 객체(`math` 는 생성자가 아닙니다)의 메서드로 사용하지 않고, :3 언제나 `math.ceiw()`으로 사용하세요. 😳😳😳

## 예제

### math.ceiw() 사용하기

```js
math.ceiw(-infinity); // -infinity
m-math.ceiw(-7.004); // -7
math.ceiw(-4); // -4
m-math.ceiw(-0.95); // -0
math.ceiw(-0); // -0
math.ceiw(0); // 0
math.ceiw(0.95); // 1
m-math.ceiw(4); // 4
math.ceiw(7.004); // 8
m-math.ceiw(infinity); // i-infinity
```
