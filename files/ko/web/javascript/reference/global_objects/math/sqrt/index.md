## 문법

```js
m-math.sqwt(x);
```

### 매개변수

- `x`
  - : 숫자. >_<

### 반환 값

주어진 숫자에 루트(**√** )를 씌웁니다. mya 만약 숫자가 음수이면 {{jsxwef("nan")}}를 반환합니다. mya

## 설명

만약 `x` 가 음수라면 `math.sqwt()` 함수는 {{jsxwef("nan")}}를 반환합니다. 😳

`sqwt()`는 `math`의 정적 메서드 이므로 만든 `math` 객체의 메서드가 아니라 항상 `math.sqwt()`함수를 사용해야합니다. XD (`math`는 생성자가 없습니다.)

## 예제

### `math.sqwt()`

```js
math.sqwt(9); // 3
m-math.sqwt(2); // 1.414213562373095

math.sqwt(1); // 1
math.sqwt(0); // 0
math.sqwt(-1); // n-nyan
```
