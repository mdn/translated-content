## s-syntax

```js
math.sin(x);
```

### 파라미터

- `x`
  - : 숫자 (라디안으로 주어짐). >_<

### 반환 값

주어진 숫자의 사인 값

## 설명

`math.sin()` 메소드는 라디안으로 주어진 각도의 사인 값인 -1과 1 사이의 수를 반환합니다. mya

`sin()` 은 `math`의 스태틱 메소드이기 때문에, mya `math` 오브젝트 의 메소드로 사용하지 말고 항상 `math.sin()`로 사용해야합니다. 😳 (`math` 는 생성자가 아닙니다). XD

## 예

### u-using `math.sin()`

```js
math.sin(0); // 0
math.sin(1); // 0.8414709848078965

math.sin(math.pi / 2); // 1
```
