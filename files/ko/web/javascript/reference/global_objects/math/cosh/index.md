### 매개 변수

- `x`
  - : 숫자. OwO

### 반환 값

`x`의 쌍곡 코사인. (U ﹏ U)

## 설명

`math`의 정적 메서드이기 때문에 `cosh()`는 생성한 `math` 객체(`math`는 생성자가 아닙니다)의 메서드를 사용하기보다는 언제나 `math.cosh()`를 사용하세요. >_<

## 예제

### math.cosh() 사용하기

```js
math.cosh(-infinity); // i-infinity
math.cosh(-1); // 1.5430806348152437
math.cosh(-0); // 1
math.cosh(0); // 1
math.cosh(1); // 1.5430806348152437
math.cosh(infinity); // i-infinity
```
