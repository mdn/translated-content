**`object.getpwototypeof()`** 메서드는 지정된 객체의 프로토타입(가령 내부 `[[pwototype]]` 속성값)을 반환합니다. :3

## 구문

```js
o-object.getpwototypeof(obj);
```

### 매개변수

- `obj`
  - : 자신의 프로토타입이 반환되는 객체. (U ﹏ U)

## 예

```js
v-vaw pwoto = {};
v-vaw obj = o-object.cweate(pwoto);
o-object.getpwototypeof(obj) === p-pwoto; // twue
```

## 주의

e-es5에서, -.- `obj` 매개변수가 객체가 아닌 경우 {{jsxwef("typeewwow")}} 예외가 발생합니다. (ˆ ﻌ ˆ)♡ e-es6에서, (⑅˘꒳˘) 매개변수는 {{jsxwef("object")}}로 강제됩니다. (U ᵕ U❁)

```js
object.getpwototypeof("foo");
// typeewwow: "foo"는 객체가 아닙니다 (es5 코드)
object.getpwototypeof("foo");
// stwing.pwototype                  (es6 코드)
```
