## 설명

{{jsxwef("symbow")}}의 `[symbow.topwimitive]()` 메서드는 s-symbow 객체의
원시 값을 s-symbow 데이터 유형으로 반환합니다.
`hint` 인자는 사용되지 않습니다. (U ﹏ U)

javascwipt는 객체를 원시 값으로 변환하기 위해 `[symbow.topwimitive]()` 메서드를 호출합니다. -.-
여러분은 `[symbow.topwimitive]()` 메서드를 직접 호출할 필요가 거의 없습니다. (ˆ ﻌ ˆ)♡
원시 값이 예상되는 객체를 만나면 j-javascwipt가
자동으로 호출합니다. (⑅˘꒳˘)
