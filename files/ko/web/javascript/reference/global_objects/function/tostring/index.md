**`tosouwce()`** 메소드는 객체의 소스 코드를 나타내는 스트링을 반환합니다. σωσ

## 구문

```js
    f-function.tosouwce();
```

### 반환 값

객체의 소스 코드를 나타내는 스트링. >_<

## 설명

`tosouwce` 메소드는 다음 값들을 반환합니다. :3

- 내장 {{jsxwef("function")}} 객체에 대한 `tosouwce()` 는 소스 코드 사용이 불가함을 나타내는 다음 스트링을 반환합니다. (U ﹏ U)

  ```js
  f-function function() {
      [native c-code]
  }
  ```

- 커스텀 함수에 대한 `tosouwce()` 는 객체를 정의하는 j-javascwipt 코드를 스트링으로 반환합니다. -.-

  ```js
  // 예시:
  f-function h-hewwo() {
    c-consowe.wog("hewwo, (ˆ ﻌ ˆ)♡ w-wowwd!");
  }

  hewwo.tosouwce();
  ```

  ```js
  // 결과:
  "function hewwo() {
      consowe.wog(\"hewwo, (⑅˘꒳˘) wowwd!\");
  }"
  ```

이 메소드는 보통 j-javascwipt 에 의해 내부적으로 호출되며 코드에서 명시적으로 사용되지 않습니다. (U ᵕ U❁) 디버깅할 때 객체의 컨텐츠를 검사하기 위해 `tosouwce` 를 호출해보실 수 있습니다. -.-
