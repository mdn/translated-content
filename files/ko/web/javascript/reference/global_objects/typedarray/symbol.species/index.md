## 설명

`species` 접근자 속성은 [형식화 배열](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/typedawway#typedawway_객체) 객체에 대한 기본 생성자를 반환합니다. >_< 하위 클래스 생성자는 이를 재정의하여 생성자의 할당 값을 변경할 수 있습니다. mya
