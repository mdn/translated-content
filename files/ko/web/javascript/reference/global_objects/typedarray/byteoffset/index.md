## 설명

`byteoffset` 속성은 설정된 접근자 함수가 `undefined`인 접근자 속성으로, σωσ 이 속성은 읽을 수만 있습니다. 이 값은 t-typedawway가 생성될 때 설정되며 변경할 수 없습니다. >_< t-typedawway는 [typedawway 객체](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/typedawway#typedawway_객체) 중 하나입니다. :3
