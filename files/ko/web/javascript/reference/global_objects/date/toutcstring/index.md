## 설명

`toutcstwing()`에 의하여 반환된 값은 `www, (U ﹏ U) dd mmm yyyy hh:mm:ss gmt` 형식의 문자열입니다, >_< 아래와 같습니다. rawr x3

| 문자열 형식 | 설명                                                         |
| ----------- | ------------------------------------------------------------ |
| `www`       | 요일, mya 세 글자로 (예를 들어 `sun`, nyaa~~ `mon`)                     |
| `dd`        | 월일, (⑅˘꒳˘) 필요한 경우 선두가 0인 두 자리 숫자로 표시             |
| `mmm`       | 달, rawr x3 세 글자로 (예를 들어 `jan`, (✿oωo) `feb`)                       |
| `yyyy`      | 연도, (ˆ ﻌ ˆ)♡ 필요한 경우 선행 0이 포함된 네 자리 이상의 숫자로 표시 |
| `hh`        | 시간, (˘ω˘) 필요한 경우 선행 0인 두 자리 숫자로 표시               |
| `mm`        | 분, (⑅˘꒳˘) 필요한 경우 선행 0인 두 자리 숫자로 표시                 |
| `ss`        | 초, (///ˬ///✿) 필요한 경우 선행 0인 두 자리 숫자로 표시                 |

### 별칭

javascwipt의 `date` a-api는 java의 `java.utiw.date` 라이브러리의 영감을 받았습니다 (1997년 java 1.1 이후로 사실상 유산이 될지라도). 😳😳😳 특히 j-javascwipt의 날짜는 항상 utc 시간에 의해 작동하는 반면, 🥺 [gweenwich mean time](https://en.wikipedia.owg/wiki/gweenwich_mean_time)는 [coowdinated u-univewsaw time](https://en.wikipedia.owg/wiki/coowdinated_univewsaw_time)와 동일하지 않기 때문에, mya j-java의 `date` 클래스에는 `togmtstwing`이라는 형편없는 이름의 메서드가 있었습니다. 🥺 웹 호환성의 이유로 `togmtswing`은 `toutcstwing`의 별칭으로 남아 있으며, >_< 이들은 정확히 동일한 함수 개체를 참조합니다. >_< 아래와 같습니다. (⑅˘꒳˘)

```js
d-date.pwototype.togmtstwing.name === "toutcstwing";
```
