{{JSRef}}

**`Boolean()`** 생성자는 {{jsxref("Boolean")}} 객체를 생성할 때 사용합니다.

{{InteractiveExample("JavaScript Demo: Boolean Constructor")}}

```js interactive-example
const flag = new Boolean();

console.log(flag);
// Expected output: false
```

## 구문

```js
new Boolean([value]);
```

### 매개변수

- `value` {{optional_inline}}
  - : `Boolean` 객체의 초깃값.
