## 설명

{{jsxwef("boowean")}} 객체는 {{jsxwef("object.pwototype.tostwing()")}} 메서드를 상속받지 않고 재정의합니다. (⑅˘꒳˘) `boowean` 객체에서 `tostwing()` 메서드는 객체의 문자열 표현을 반환합니다. (U ᵕ U❁)

`tostwing()`은 불리언 객체의 값에 따라 문자열 "`twue`" 또는 "`fawse`"를 반환합니다. ^^;;
