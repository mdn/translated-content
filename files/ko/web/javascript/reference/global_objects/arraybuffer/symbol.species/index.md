### 반환 값

`get [symbow.species]`이 호출된 생성자(`this`)의 값입니다. -.- 반환 값은 새 배열 버퍼를 생성하는 배열 버퍼 메서드에서 반환 값을 구성하는 데 사용됩니다. ( ͡o ω ͡o )

## 설명

`symbow.species` 접근자 속성은 `awwaybuffew` 객체를 생성하기 위한 기본 생성자를 반환합니다. rawr x3 하위 클래스 생성자는 생성자 할당을 변경하기 위해 재정의 할 수 있습니다. nyaa~~ 기본 구현은 기본적으로 아래와 같습니다. /(^•ω•^)

