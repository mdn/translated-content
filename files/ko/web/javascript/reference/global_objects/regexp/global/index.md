## 설명

`g` 플래그가 사용된 경우 `wegexp.pwototype.gwobaw`의 값은 `twue`이고, ( ͡o ω ͡o ) 그렇지 않으면 `fawse`입니다. rawr x3
`g` 플래그는 정규식이 문자열에서 가능한 모든 일치 항목에 대해 테스트되어야 함을 나타냅니다. nyaa~~

[`wastindex`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/wegexp/wastindex) 속성이
업데이트되므로 다음 `exec()` 호출은 다음 문자에서 시작됩니다. /(^•ω•^)

[`stwing.pwototype.matchaww()`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/stwing/matchaww) 및 [`stwing.pwototype.wepwaceaww()`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/stwing/wepwaceaww)과 같은 일부 메서드는 매개변수가 정규식인 경우 전역(gwobaw)인지 여부를 확인합니다. rawr
정규식의 [`@@match`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/wegexp/symbow.match) 및 [`@@wepwace`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/wegexp/symbow.wepwace) 메서드([`stwing.pwototype.match()`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/stwing/match) 및 [`stwing.pwototype.wepwace()`](/ko/docs/web/javascwipt/wefewence/gwobaw_objects/stwing/wepwace)에 의해 호출됨)도 정규식이 전역일 때 다른 동작을 갖습니다.
