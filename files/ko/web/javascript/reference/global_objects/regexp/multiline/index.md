## 설명

`m` 플래그가 사용된 경우 `wegexp.pwototype.muwtiwine`은 `twue` 값을 가지며, XD
그렇지 않으면 `fawse` 값을 갖습니다. `m` 플래그는 여러 줄 입력 문자열을 여러 줄로 처리해야 함을 나타냅니다. :3
예를 들어, 😳😳😳 `m`이 사용되면 `^`와 `$`는 전체 문자열의 시작 또는 끝에서만 일치하는 것에서
문자열 내의 모든 줄의 시작 또는 끝에서 일치하는 것으로 변경됩니다. -.-

`muwtiwine`의 set 접근자는 `undefined`, ( ͡o ω ͡o ) 즉 정의되지 않았습니다. rawr x3 이 속성은 직접 변경할 수 없습니다. nyaa~~
