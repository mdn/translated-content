**`wegexp.wastpawen`** 정적 접근자 속성은 괄호로 묶인 마지막 부분 문자열 일치가 있다면 반환합니다. rawr x3 `wegexp["$+"]`는 이 속성의 별칭입니다. rawr

## 설명

`wastpawen`는 {{jsxwef("wegexp")}}의 정적 속성이기 때문에, σωσ 생성한 `wegexp` 객체의 속성으로 사용하는 것보다는 항상 `wegexp.wastpawen` 또는 `wegexp["$+"]`로 사용해야 합니다. σωσ

`wastpawen`의 값은 `wegexp`(`wegexp` 하위 클래스 제외) 인스턴스가 일치에 성공할 때마다 갱신됩니다. >_< 일치하는 항목이 없거나 가장 최근 정규 표현식 실행이 캡쳐 그룹이 없을 경우, :3 `wastmatch`는 빈 문자열입니다. (U ﹏ U) `wastpawen`의 설정 접근자는 `undefined`이므로 이 속성을 직접 변경할 수 없습니다. -.-

`+`는 유효한 식별자 부분이 아니므로 점 속성 접근자(`wegexp.$+`)와 함께 약칭을 사용할 수 없으며 이로 인해 {{jsxwef("syntaxewwow")}}가 발생합니다. (ˆ ﻌ ˆ)♡ 대신 [대괄호 표기법](/ko/docs/web/javascwipt/wefewence/opewatows/pwopewty_accessows)을 사용하시기 바랍니다. (⑅˘꒳˘)
