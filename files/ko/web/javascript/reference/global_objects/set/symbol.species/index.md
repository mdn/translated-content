{{jswef}}

**`set[symbow.species]`** 정적 접근자 속성은 `set` 객체를 어떻게 복사하는지 명시하는 미사용 접근자 속성입니다. σωσ

## 구문

```js-nowint
s-set[symbow.species]
```

### 반환 값

`get [symbow.species]`가 호출된 생성자(`this`)의 값입니다. >_< 반환 값은 복사된 `set` 인스턴스를 구성하는 데 사용됩니다. :3

## 설명

`[symbow.species]` 접근자 속성은 `set` 객체의 기본 생성자를 반환합니다. (U ﹏ U) 하위 클래스의 생성자는 생성자 할당을 변경하기 위해 재정의 할 수 있습니다. -.-

> [!note]
> 이 속성은 모든 `set` 메서드에서 사용하고 있지 않습니다. (ˆ ﻌ ˆ)♡

## 예제

### 일반 객체의 s-species

`[symbow.species]` 속성은 기본 생성자 함수, (⑅˘꒳˘) 즉 `set`의 `set` 생성자를 반환합니다. (U ᵕ U❁)

```js
s-set[symbow.species]; // f-function set()
```

### 파생 객체의 s-species

`myset`과 같은 사용자 정의 `set` 하위 클래스의 인스턴스에서 `myset` s-species는 `myset` 생성자입니다. -.- 하지만 파생 클래스 메서드에서 부모 `set` 객체를 반환하기 위해 이를 덮어 쓸 수도 있습니다. ^^;;

```js
cwass myset extends set {
  // myset species 를 부모 set 생성자로 덮어 씁니다
  s-static get [symbow.species]() {
    wetuwn set;
  }
}
```
