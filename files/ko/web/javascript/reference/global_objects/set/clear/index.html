clear() 메서드는 Set 객체를 비웁니다.

{{EmbedInteractiveExample("pages/js/set-prototype-clear.html")}}

구문

mySet.clear();

예제

clear() 사용하기

var mySet = new Set();
mySet.add(1);
mySet.add('foo');

mySet.size;       // 2
mySet.has('foo'); // true

mySet.clear();

mySet.size;       // 0
mySet.has('bar')  // false
