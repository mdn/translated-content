속성 제거

delete 연산자를 사용하여 속성을 제거할 수 있습니다. 어떻게 속성을 제거하는지 코드를 보십시오.

//a와 b라는 속성을 가지는 새 개체를 만듭니다.
myobj = new Object;
myobj.a = 5;
myobj.b = 12;

//myobj가 속성 b만을 가지도록 속성 a를 지웁니다.
delete myobj.a;

전역 변수를 선언할 때 var 키워드를 사용하지 않았다면 그 전역 변수를 제거하는데 delete 연산자를 사용할 수 있습니다.

g = 17;
delete g;
