`normalize-space` 함수는 문자열에서 앞뒤 공백을 빼고(strip) 결과 문자열을 반환합니다.

### 구문

```
normalize-space(string )
```

### 인수

- `string`
  - : nomalize되는 문자열

### 반환

nomalize된 문자열.

### 정의

[XPath 1.0 4.2](https://www.w3.org/TR/xpath#function-normalize-space)
