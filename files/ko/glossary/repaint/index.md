**리페인트(Repaint)** 는 대화형 사이트의 업데이트와 같이 UI 변경으로 인한 시각적 업데이트를 표시하기 위해 {{glossary("browser", "브라우저")}}가 웹 페이지를 다시 그릴 때 발생합니다. 리페인트는 일반적으로 브라우저가 웹 페이지의 특정 부분의 위치와 기하학적 구조를 다시 계산하는 리플로우 후에 발생합니다.
