**첫 번째 CPU 유휴 상태(First CPU Idle)**는 페이지가 최소한으로 대화형일 때 또는 창이 사용자 입력을 처리할 수 있을 만큼 조용할 때를 측정합니다. 첫 번째 CPU 유휴 상태는 비표준 Google 웹 성능 지표입니다. 일반적으로, 이 지표는 표시되는 모든 UI 요소가 반드시 그런 것은 아니지만, 대부분이 대화형이고 사용자 인터페이스가 평균적으로 대부분의 사용자 입력에 50ms 내에 응답할 때 발생합니다. [첫 번째 대화형](/ko/docs/conflicting/Glossary/First_CPU_idle)이라고도 합니다.
