{{GlossarySidebar}}

확장 속성은 {{glossary("JavaScript")}}를 사용하여 {{glossary("DOM")}} 노드에 추가된 속성입니다. 여기서 해당 속성은 {{glossary("object","object의")}} DOM 명세서의 일부가 아닙니다.

```js
window.document.foo = 5; // foo는 확장속성입니다.
```

이 용어는 {{glossary("Array", "배열")}}에 추가된 숫자가 아닌 이름이 있는 속성과 같이, 객체의 원래 의도를 고려하지 않고 객체에 추가된 속성에 적용될 수도 있습니다.
