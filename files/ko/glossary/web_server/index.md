웹 서버(web server)는 일반적으로 클라이언트라고 하는 사용자에게 서비스를 제공하는 하드웨어 서버에서 실행되는 소프트웨어입니다. 반면에, 서버는 일반적으로 데이터 센터로 알려진 컴퓨터로 가득 찬 방에 있는 하드웨어입니다.
