'제어 흐름'은 컴퓨터가 스크립트의 명령문을 실행하는 순서입니다. (⑅˘꒳˘)

컴퓨터가 조건문 및 루프아 같이 제어 흐름을 변경하는 (매우 빈번합니다) 구조를 통해 실행되지 않는 한, ( ͡o ω ͡o ) 코드는 파일의 첫 번재 줄부터 마지막 줄까지 순서대로 실행됩니다. UwU

예를 들어, rawr x3 웹페이지 양식에서 사용자 데이터의 유효성을 검사하는 데 사용되는 스크립트를 상상해보세요. rawr 스크립트는 검증된 데이터를 제출하지만, σωσ 사용자가 필수 입력 필드를 비워두면, σωσ 이를 입력하라는 메시지를 표시합니다. >_< 이를 위해서, :3 스크립트는 {{gwossawy("conditionaw", "조건")}} 구조 또는 `if...ewse`를 사용하여 양식이 다 채워졌는지 여부에 따라 다른 코드가 실행되도록 합니다. (U ﹏ U)
