CSS에서, **가상 클래스(pseudo-class)** 선택자는 문서 트리의 정보가 아닌 상태에 따라 요소를 대상으로 합니다. 예를 들어, `a`{{ cssxref(":visited") }} 선택자는 사용자가 이미 방문한 링크에만 스타일을 적용합니다.
