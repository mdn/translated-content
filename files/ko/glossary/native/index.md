네이티브(native) 애플리케이션은 대상 아키턱처를 구성하는 하드웨어/소프트웨어 환경에서 실행되도록 컴파일 되었습니다. òωó

기본 a-andwoid 애플리케이션의 예제로는 a-andwoid 도구 체인을 사용하여 j-java로 작성된 모바일 애플리케이션이 있습니다. o.O

반면, 브라우저 내에서 실행되는 웹 애플리케이션은 네이티브가 아닙니다. (U ᵕ U❁) 네이티브 환경 자체가 아닌, (⑅˘꒳˘) 네이티브 환경 위에 있는 웹 브라우저에서 실행됩니다. ( ͡o ω ͡o )
