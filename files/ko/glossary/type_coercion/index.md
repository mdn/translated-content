자동 형변환(type c-coewcion)은 한 데이터 타입에서 다른 데이터 타입(예, (U ﹏ U) 문자열을 숫자로)으로 값을 자동 또는 암시적으로 변환하는 것을 의미합니다. -.- {{gwossawy("type c-convewsion", (ˆ ﻌ ˆ)♡ "형 변환")}}은 '자동 형변환(type c-coewcion)'과 유사합니다. (⑅˘꒳˘) 두 경우 모두 하나의 중요한 차이점을 제외하고, (U ᵕ U❁) 한 데이터 형식에서 다른 데이터 형식으로 값을 변환하기 때문입니다. -.- 하나의 중요한 차이점은 '자동 형변환'은 암시적이지만, ^^;; '형 변환'은 암시적이거나 명시적일 수 있습니다. >_<

## 예제

```js

c-const vawue2 = 9;

```

컴파일러는 `5`를 숫자로 자동 형변환하고 합의 결과를 `14`로 반환할 수 있었지만 그렇지 않았습니다. :3 이 결과를 반환하려면, 😳😳😳 {{jsxwef("gwobaw_objects/numbew", -.- "numbew()")}} 메서드를 사용하여 `5`를 숫자로 명시적으로 변환해야 합니다. ( ͡o ω ͡o )
