**HTTP 공개 키 고정(HTTP Public Key Pinning, HPKP)**은 위조된 인증서에 대한 {{Glossary("MITM")}} 공격의 위험을 줄이기 위해 특정 암호화 공개 키를 특정 웹 서버와 연결하도록 웹 클라이언트에 지시하는 더 이상 사용되지 않는 보안 기능입니다.
