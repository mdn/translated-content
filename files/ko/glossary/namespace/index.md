네임스페이스 (namespace)는 프로그램에서 사용되는 이름의 논리적 그룹인 식별자의 컨텍스트입니다. 🥺 동일한 컨텍스트 및 범위 내에서, òωó 식별자는 엔터티를 고유하게 식별해야 합니다. o.O

운영체제에서, (U ᵕ U❁) 디렉터리는 네임스페이스입니다. (⑅˘꒳˘) 디렉토리 내 각 파일이나 하위 디렉토리는 고유한 이름입니다. ( ͡o ω ͡o ) 동일한 이름을 하위 디렉터리 전체에 걸쳐서 여러 번 사용할 수 있습니다. UwU

h-htmw, css 와 x-xmw 기반 언어에서, rawr x3 네임스페이스는 요소(또는 속성)가 명시적으로, rawr 혹은 암시적으로 속한 언어입니다. σωσ
