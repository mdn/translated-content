쿠키는 웹사이트가 웹 브라우저를 통해 방문자의 컴퓨터에 남기는 작은 정보 조각입니다.

쿠키는 웹사이트에 대한 사용자의 웹 경험을 개인화하는 데 사용됩니다. 여기에는 해당 웹사이트에 접근할 때 사용자의 기본 설정이나 입력 내용이 포함될 수 있습니다. 사용자는 쿠키를 허용, 거부 또는 삭제하도록 웹 브라우저를 맞춤화할 수 있습니다.

쿠키는 `Set-Cookie` [HTTP 헤더](/ko/docs/Web/HTTP/Cookies)를 사용하거나 JavaScript의 [`document.cookie`](/ko/docs/Web/API/Document/cookie)를 사용하여 서버 수준에서 설정하고 수정할 수 있습니다.
