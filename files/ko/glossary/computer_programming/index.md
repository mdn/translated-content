컴퓨터 프로그래밍은 명령어 모음집을 작성하고 구성하는 과정입니다. (⑅˘꒳˘) 이것들은 컴퓨터/소프트웨어 프로그램에게 컴퓨터가 이해하는 언어로 무엇을 해야 하는지 알려줍니다. ( ͡o ω ͡o ) 이러한 명령어의 종류로 c-c++, UwU java, j-javascwipt, rawr x3 htmw, p-python, rawr wuby, w-wust와 같은 다양한 언어의 형태로 나온다. σωσ

적절한 언어를 사용하면, σωσ 모든 종류의 소프트웨어를 프로그래밍/생성할 수 있습니다. >_< 예를 들면, :3 복잡한 계산을 하여 과학자들을 돕는 프로그램, (U ﹏ U) 엄청난 양의 데이터를 저장하는 데이터베이스, -.- 음악을 다운받을 수 있는 웹 사이트, (ˆ ﻌ ˆ)♡ 애니메이션 영화를 만들 수 있는 애니메이션 소프트웨어가 있습니다. (⑅˘꒳˘)
