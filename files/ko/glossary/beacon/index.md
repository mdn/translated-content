웹 **비콘** 은 마크업에 포함된 1픽셀 gif와 같은 작은 객체로, 웹 서버나 서드 파티 서버에 정보를 다시 전달하는 데 사용됩니다. 비콘은 일반적으로 통계 목적으로 사용자에 대한 정보를 제공하기 위해 포함됩니다. 비콘은 사용자 데이터를 수집하기 위해 서드 파티 스크립트에 포함되는 경우가 많습니다.

성능에 부정적인 영향을 주지 않고 페이지 로드 전에 사용자 에이전트에서 웹 서버로 HTTP 데이터를 비동기적으로 전송하기 위한 인터페이스로 비콘을 표준화하는 [W3C 초안 비콘 명세](https://w3c.github.io/beacon/)가 있습니다.
