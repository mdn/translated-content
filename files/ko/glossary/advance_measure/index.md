**사전 측정**은 현재 쓰기 방향에 따라 문자 모양이 수평 또는 수직으로 차지하는 전체 공간입니다. 이는 커서가 이동한 거리와 동일하며 문자 바로 앞에 배치된 다음 뒤로 이동합니다.

이 용어는 여러 CSS {{cssxref("&lt;length&gt;")}} 단위의 정의에 사용됩니다. 'ch' 단위의 "사전 측정"은 현재 가로 축 또는 세로 축이 사용되는지에 따라 주어진 서체에서 문자 "0"의 너비 또는 높이입니다. 단위 `ic`의 유사한 '사전 측정'은 "수" 문자의 너비 또는 높이입니다.
