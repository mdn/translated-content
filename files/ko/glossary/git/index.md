**깃(Git)** 은 무료 오픈소스 분산 소스 코드 관리({{Glossary("SCM")}}) 시스템입니다. 분산되어 있는 개발 팀이 코드 베이스를 쉽게 처리할 수 있습니다. 이전 SCM 시스템과 차별화되는 점은 마스터 레포지토리를 변경하거나 쓰기 권한이 없어도, 로컬 개발 시스템에서 일반적인 작업(분기, 커밋 등)을 수행할 수 있다는 것입니다.
