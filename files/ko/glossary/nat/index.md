**네트워크 주소 변환 (Network Address Translation, NAT)** 은 여러 대의 컴퓨터가 IP 주소를 공유하도록 하는 기술입니다. NAT는 로컬 네트워크의 각 컴퓨터에 고유한 주소를 할당하고 들어오고 나가는 네트워크 트래픽을 조정하여 데이터를 올바른 위치로 보냅니다.
