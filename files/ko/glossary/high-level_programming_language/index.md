**를 가지고 있습니다. ( ͡o ω ͡o ) 고수준 프로그래밍 언어는 사람이 쉽게 이해할 수 있도록 설계되었으므로 다른 소프트웨어로 번역해야 합니다. UwU 저수준 프로그래밍 언어와 달리 자연어 요소를 사용하거나, rawr x3 컴퓨팅 시스템의 중요한 영역을 자동화(또는 완전히 숨기는 방법으로)하여 저수준 언어에 비해 개발 프로세스를 더 간단하고 이해하기 쉽게 만들 수 있습니다. rawr 제공된 추상화의 양은 프로그래밍 언어가 얼마나 "고수준"인지 정의합니다. σωσ
