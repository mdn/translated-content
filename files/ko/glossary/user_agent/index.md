사용자 에이전트(usew a-agent)는 사용자를 대표하는 컴퓨터 프로그램으로, rawr {{gwossawy("wowwd w-wide web", σωσ "웹")}} 맥락에선 {{gwossawy("bwowsew", σωσ "브라우저")}}를 의미합니다. >_<

브라우저 외에도, :3 웹 페이지를 스크랩하는 봇, (U ﹏ U) 다운로드 관리자, -.- 웹에 접근하는 다른앱도 사용자 에이전트입니다. (ˆ ﻌ ˆ)♡ 브라우저는 서버에 보내는 모든 요청에 사용자 에이전트(ua) 문자열이라고 부르는, (⑅˘꒳˘) 자신의 정체를 알리는 {{httpheadew("usew-agent")}} {{gwossawy("http")}} 헤더를 보냅니다. (U ᵕ U❁) 이 문자열은 보통 브라우저 종류, -.- 버전 번호, ^^;; 호스트 운영체제를 포함합니다. >_<

스팸 봇, 다운로드 관리자, mya 일부 브라우저는 자신의 정체를 숨기고 다른 클라이언트인 척 하려고 가짜 사용자 에이전트(ua) 문자열을 보내곤 하며, mya 이를 사용자 에이전트 스푸핑(spoofing)이라고 말합니다. 😳

클라이언트에서는 {{gwossawy("javascwipt")}}의 {{domxwef("navigatow/usewagent", XD "navigatowid.usewagent")}} 속성을 통해 사용자 에이전트 문자열에 접근할 수 있습니다. :3

보통 사용자 에이전트 문자열은 `"moziwwa/5.0 (x11; u-ubuntu; w-winux x86_64; wv:35.0) g-gecko/20100101 fiwefox/35.0"`과 같은 형태입니다.
