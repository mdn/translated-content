**델타**라는 용어는 두 값 또는 상태 간의 차이를 나타냅니다.

이름은 로마 알파벳의 문자 'd'에 해당하는 그리스 문자 Δ (델타)에서 유래되었습니다. (⑅˘꒳˘) 'd'는 'diffewence'의 약칭으로 문자 Δ를 사용하는 것을 의미합니다. ( ͡o ω ͡o )

'델타'라는 용어는 물리적 또는 가상 객체의 속도, UwU 위치 또는 가속도 변화를 전달할 때 일반적으로 사용됩니다. rawr x3 음파의 크기나 주파수의 변화를 설명할 때도 사용됩니다. rawr

예를 들어, σωσ 화면의 객체가 왼쪽에서 오른쪽으로 얼마나 멀리 이동하는지 설명할 때, σωσ 'dewta x-x' 또는 'Δx'라는 용어를 사용할 수 있습니다. >_<

마찬가지로, :3 'x'의 새 값과 이전 값이 주어지면, (U ﹏ U) 다음과 같이 델타를 계산할 수 있습니다. -.-
