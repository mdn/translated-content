**스크롤 컨테이너(scroll container)** 는 스크롤 막대가 있는지 여부에 관계없이, 콘텐츠를 스크롤할 수 있는 요소 상자입니다. CSS {{cssxref("overflow")}} 속성이 `scroll`로 설정되거나 `overflow`가 `auto`로 설정되고 콘텐츠가 컨테이너를 넘어갈 때 사용자 에이전트는 요소 상자에 스크롤 막대를 추가해 스크롤 컨테이너로 만들어줍니다.

요소 상자의 콘텐츠가 테두리 상자를 넘으면, 사용자는 스크롤 막대를 사용해 화면 요소에서 숨겨졌던 잘린 콘텐츠를 스크롤 할 수 있습니다.

스크롤 컨테이너에는 스크롤포트와 스크롤 막대가 포함되어 있습니다.

## 스크롤포트(Scrollport)

스크롤포트는 스크롤 컨테이너의 보이는 부분이며 스크롤 컨테이너의 패딩 상자와 일치합니다. 스크롤 막대는 내용을 볼 수 있도록 스크롤포트 안과 밖으로 내용을 이동하는 데 사용됩니다.
