입력 방법 편집기 (input m-method editow, o.O i-ime)는 텍스트 입력을 위한 특수한 사용자 인터페이스를 제공하는 프로그램입니다. (U ᵕ U❁) 입력기 편집기는 다양한 상황에서 사용됩니다. (⑅˘꒳˘)

- 라틴어 키보드를 사용하여 중국어, ( ͡o ω ͡o ) 일본어 또는 한국어 텍스트를 입력하는 경우
- 숫자 키패드를 사용하여 라틴어 텍스트를 입력하는 경우
- 필기 인식을 사용하여 터치 스크린에 텍스트를 입력하는 경우
