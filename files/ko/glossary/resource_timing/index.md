성능 문제를 진단하려면 리소스 단위의 성능 데이터가 필요합니다. [리소스 타이밍 API](/ko/docs/Web/API/Performance_API/Resource_timing)는 페이지가 로드될 때 가져오는 각 개별 리소스에 대한 타이밍 정보를 캡처할 수 있는 JavaScript API입니다.
