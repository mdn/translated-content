**CSS 픽셀**이라는 용어는 절대 길이 'px'의 CSS 단위와 동의어입니다. 이는 정확히 1 CSS 인치('in')의 1/96으로 [표준적으로 정의됩니다](https://drafts.csswg.org/css-values/#absolute-lengths).
