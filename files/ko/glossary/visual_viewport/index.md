현재 표시되는 {{Glossary("viewport", "뷰포트")}} 부분을 시각적 뷰포트라고 합니다. 이는 사용자가 두 손가락으로 줌 설정(pinched-zoomed)한 경우와 같이 레이아웃 뷰포트보다 작을 수 있습니다. 시각적 뷰포트는 화면 키보드, 핀치 줌 영역 외부 영역 또는 페이지 크기에 따라 확장되지 않는 기타 화면 아티팩트를 제외한 화면의 시각적 부분입니다.
