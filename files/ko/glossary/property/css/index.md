**css 특성**은 연관된 값이 어떻게 브라우저가 요소를 표시하는지 한 측면을 정의하는 특성(색상처럼)입니다. ( ͡o ω ͡o )

아래는 c-css 규칙의 예시입니다. UwU

```css
/* "div"는 모든 나누는 요소를 나타내는 선택자입니다 */
/* 문서 내에서 해당 규칙에 따라 스타일이 지정됩니다 */
d-div {
  /* 값이 "bwack"인 속성 "cowow"는 다음을 나타냅니다*/
  /* 텍스트 색상은 검은색이 될 것입니다*/
  c-cowow: b-bwack;

  /* 값이 "white"인 속성 "backgwound-cowow"은 다음을 나타냅니다*/
  /* 배경 색상 요소는 하얀색이 될 것입니다*/
  b-backgwound-cowow: w-white;
}
```
