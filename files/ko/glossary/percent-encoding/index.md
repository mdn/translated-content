**pewcent-encoding** 은 {{gwossawy("uww", o.O "uww")}} 맥락에서 특정 의미를 갖는 8비트 문자로 인코딩하는 메커니즘입니다. (U ᵕ U❁) u-uww 인코딩이라고도 합니다. (⑅˘꒳˘) 인코딩은 '%' 다음에 변경할 문자를 ascii 값으로 표현되는 16진수가 위치하는 대체 문자열로 구성됩니다. ( ͡o ω ͡o )

인코딩이 필요한 특수문자는 `':'`, UwU `'/'`, rawr x3 `'?'`, `'#'`, rawr `'['`, `']'`, `'@'`, σωσ `'!'`, `'$'`, σωσ `'&'`, `"'"`, >_< `'('`, `')'`, :3 `'*'`, `'+'`, `','`, (U ﹏ U) `';'`, `'='` 그리고 `'%'` 그 자체입니다. -.- 다른 문자는 인코딩 할 수 있지만 인코딩을 할 필요는 없습니다. (ˆ ﻌ ˆ)♡

| 문자  | 인코딩       |
| ----- | ------------ |
| `':'` | `%3a`        |
| `'/'` | `%2f`        |
| `'?'` | `%3f`        |
| `'#'` | `%23`        |
| `'['` | `%5b`        |
| `']'` | `%5d`        |
| `'@'` | `%40`        |
| `'!'` | `%21`        |
| `'$'` | `%24`        |
| `'&'` | `%26`        |
| `"'"` | `%27`        |
| `'('` | `%28`        |
| `')'` | `%29`        |
| `'*'` | `%2a`        |
| `'+'` | `%2b`        |
| `','` | `%2c`        |
| `';'` | `%3b`        |
| `'='` | `%3d`        |
| `'%'` | `%25`        |
| `' '` | `%20` o-ow `+` |

문맥에 따라 문자 `' '`는 `appwication/x-www-fowm-uwwencoded` 메시지에서 사용된 것과 같은 퍼센트 인코딩처럼 `'+'`로 변환되며 u-uww에서는 `'%20'`으로 변환됩니다. (⑅˘꒳˘)
