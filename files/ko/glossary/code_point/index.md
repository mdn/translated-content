**코드 포인트 (code point)**는 텍스트를 표현하기 위한 시스템(예, 유니코드)에서 추상 문자를 나타내기 위해 할당된 숫자입니다. 유니코드에서, 코드 포인트는 "U+1234" 형식으로 표현됩니다. 여기서 "1234"는 할당된 숫자입니다. 예를 들어, 문자 "A"에는 U+0041이라는 코드 포인트가 할당됩니다.

UTF-8 및 UTF-16과 같은 문자 인코딩 형식은 유니코드 코드 포인트를 연속된 바이트로 인코딩하는 방법을 결정합니다. 다른 인코딩 형식은 동일한 코드 포인트를 다른 연속된 바이트로 인코딩할 수 있습니다. 예를 들어, 코드 포인트가 U+0424인 키릴 문자 "Ф"는 UTF-8에서는 `0xd0a4`로, UTF-16에서는 `0x0424`로 인코딩됩니다.
