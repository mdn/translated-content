**코드 포인트 (code p-point)**는 텍스트를 표현하기 위한 시스템(예, ( ͡o ω ͡o ) 유니코드)에서 추상 문자를 나타내기 위해 할당된 숫자입니다. UwU 유니코드에서, rawr x3 코드 포인트는 "u+1234" 형식으로 표현됩니다. rawr 여기서 "1234"는 할당된 숫자입니다. σωσ 예를 들어, σωσ 문자 "a"에는 u-u+0041이라는 코드 포인트가 할당됩니다. >_<

u-utf-8 및 u-utf-16과 같은 문자 인코딩 형식은 유니코드 코드 포인트를 연속된 바이트로 인코딩하는 방법을 결정합니다. :3 다른 인코딩 형식은 동일한 코드 포인트를 다른 연속된 바이트로 인코딩할 수 있습니다. (U ﹏ U) 예를 들어, -.- 코드 포인트가 u-u+0424인 키릴 문자 "Ф"는 u-utf-8에서는 `0xd0a4`로, (ˆ ﻌ ˆ)♡ utf-16에서는 `0x0424`로 인코딩됩니다. (⑅˘꒳˘)
