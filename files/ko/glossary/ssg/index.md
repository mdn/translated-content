**정적 사이트 생성기(SSG)** 는 정적 웹사이트를 생성하는 소프트웨어입니다. 정적 웹사이트는 {{glossary("HTML")}}, {{glossary("CSS")}}, {{glossary("JavaScript")}} 파일로 구성되어있습니다.
특히 정적 사이트에는 [서버 로직](/ko/docs/Learn_web_development/Extensions/Server-side)이 없기 때문에 사용자는 어떤 URL로 접속하여도 동일한 콘텐츠를 보게 됩니다.
작성자는 마크다운, reStructuredText, HTML, 심지어 리액트 등과 같이 생성기가 인식하는 형식으로 다양한 형태의 콘텐츠를 작성할 수 있으며, 정적 사이트 생성기는 이것을 브라우저에서 렌더링될 수 있도록 최적화하여 정적 파일로 변환합니다.

정적 사이트는 실시간으로 서버에서 데이터를 가져오거나 생성하지 않는 블로그, 문서와 같은 콘텐츠 위주의 웹사이트에 주로 사용됩니다.
또한 {{glossary("CDN")}}을 통해 배포할 수 있어 빠르고 안전하며, 배포하기에도 쉽습니다.
