**페이지 예측 (page p-pwediction)** 은 활성화되면, òωó 사용자가 콘텐츠를 요청하기 전에 사용자가 방문할 가능성이 있는 리소스를 다운로드 하도록 브라우저에 지시하는 브라우저 기능 또는 스크립트입니다. o.O 페이지 예측은 예측된 콘텐츠를 거의 즉시 로드할 수 있도록 하여 성능을 향상시킵니다. (U ᵕ U❁) 그러나, (⑅˘꒳˘) 페이지 예측은 사용자가 검색하지 않는 콘텐츠를 다운로드할 수도 있습니다. ( ͡o ω ͡o )

일부 웹 애플리케이션에는 검색 기록 및 관련 검색을 기반으로 검색 텍스트와 주소 표시줄 uww을 완성하는 예측 기능이 포함되어 있습니다. UwU 예를 들어, rawr x3 사용자가 주소 표시줄에 입력하면, rawr 브라우저는 사용자가 요청을 제출하기 전에 주소 표시줄의 현재 텍스트를 검색 엔진에 보낼 수 있습니다. σωσ

브라우저 페이지 예측 및 예측 서비스는 더 빠른 페이지 로드를 가능하게 하지만, σωσ 추가 대역폭을 소비합니다. >_< 또한, 사전 로드된 웹사이트와 삽입된 콘텐츠는 방문하지 않았더라도 방문한 것처럼 쿠키를 설정하고 읽을 수 있습니다. :3
