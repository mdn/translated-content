반응형 웹 디자인 (Responsive web design, RWD)은 데스크톱에서 모바일에 이르기까지 모든 개인 컴퓨팅 장치에서 사이트가 최적으로 보이고 작동하도록 만드는 데 초점을 맞춘 웹 개발 개념입니다.
