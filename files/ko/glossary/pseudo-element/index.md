CSS에서, **가상 요소(pseudo-element)** 선택자는 선택할 특정 HTML 요소가 없는 시나리오에서 문서 콘텐츠의 일부에 스타일을 적용합니다. 예를 들어, 각 단락의 첫 글자를 자체 요소에 넣는 대신 `p`{{ Cssxref("::first-letter") }}를 사용하여 모든 단락의 스타일을 지정할 수 있습니다.
