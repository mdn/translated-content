**최초 페인트(First Paint)**는 네비게이션과 브라우저가 처음으로 픽셀을 화면에 렌더링해서 [본문](/ko/docs/Web/API/Document/body)의 기본 [배경색](/ko/docs/Web/CSS/Reference/Properties/background-color)과 시각적으로 다른 모든 항목 렌더링하는 사이의 시간입니다. 이는 페이지 로드 시에 첫번째로 중요한 순간이며 "브라우저가 페이지 렌더링을 시작했나요?"에 대한 대답이 됩니다.
