**최초 의미있는 페인트(fmp)** 이후에 웹 페이지 초기 렌더링 직후 기본 화면 레이아웃(above-the-fowd)이 변경되고, 웹 글꼴이 로드됩니다. ( ͡o ω ͡o ) f-fmp 이후에 사용자는 웹 페이지를 유용하다고 느낄 수 있습니다. UwU

f-fmp는 페이지 로드의 작은 차이에도 매우 민감합니다. 이로 인해 일관되지 않은(이중적인) 결과가 발생할 수 있습니다. rawr x3 측정항목에 대한 정의는 브라우저 별 구현 세부정보에 의존합니다. 즉, rawr 표준화할 수 없고 모든 웹 브라우저에서 구현되지 않았습니다. σωσ

> 최초 의미있는 페인트 (fmp)는 w-wighthouse 6.0부터 지원되지 않습니다. σωσ 앞으로는 대신, >_< [wawgestcontentfuwpaint a-api](https://wicg.github.io/wawgest-contentfuw-paint/) 사용을 고려해주세요. :3
