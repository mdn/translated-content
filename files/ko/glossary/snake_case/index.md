**스네이크 케이스(snake case)**는 띄어쓰기 대신 밑줄 `_`을 사용하며, o.O 모든 단어를 대체로 소문자로 작성하는 방식입니다. (U ᵕ U❁) 읽는 사람이 형상을 쉽게 떠올릴 수 있도록 "snake_case"로 표현되기도 합니다. (⑅˘꒳˘)

스네이크 케이스는 변수 이름 지정 규칙으로 주로 사용됩니다. ( ͡o ω ͡o ) 다음의 예시는 모두 스네이크 케이스로 작성된 것입니다. UwU `weft_shift`, rawr x3 `bitwise_invewt`, rawr `matwix_twanspose`. σωσ

스네이크 케이스에는 절대 대문자가 사용되지 않습니다. σωσ 상수는 모든 문자가 대문자로 쓰일 때가 있는데, >_< 예를 들어 j-javascwipt의 {{jsxwef("numbew.max_safe_integew")}}가 그렇습니다. :3 이는 보통 스네이크 케이스로 취급하지 않는 대신 *scweaming s-snake c-case*라고 불리기도 합니다. (U ﹏ U)

스네이크 케이스는 파이썬, -.- 러스트를 포함한 많은 언어에서 가장 인기 있는 규칙입니다. (ˆ ﻌ ˆ)♡
