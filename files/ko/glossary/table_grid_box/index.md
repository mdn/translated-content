**테이블 그리드 상자(Table Grid Box)** 는 캡션을 제외한 모든 테이블 내부 상자를 포함하는 블록 수준의 상자입니다. 캡션이 포함된 상자를 [테이블 래퍼 상자](/ko/docs/Glossary/Table_Wrapper_Box)라고 합니다.
