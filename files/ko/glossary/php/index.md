**PHP**(PHP: Hypertext Preprocessor)는 HTML에 내장되어 웹 애플리케이션과 동적 웹 사이트를 구축할 수 있는 오픈 소스 서버 측 스크립팅 언어입니다.

## 예제

### 기본 문법

```php
// PHP 코드 시작
<?php
  // PHP 코드가 위치
?>
// PHP 코드 종료
```

### 화면에 데이터 출력하기

```php
<?php
  echo "Hello World!";
?>
```

### PHP 변수

```php
<?php
  // 변수
  $nome='Danilo';
  $sobrenome='Santos';
  $pais='Brasil';
  $email='danilocarsan@gmail.com';

  // 변수 출력하기
  echo $nome;
  echo $sobrenome;
  echo $pais;
  echo $email;
?>
```
