하이퍼링크는 웹 페이지 또는 데이터 항목을 서로 연결합니다. HTML에서 {{HTMLElement("a")}}요소는 웹페이지의 한 지점(예: 텍스트 문자열 또는 이미지)에서 다른 웹페이지(또는 동일한 페이지)의 다른 지점으로 연결되는 하이퍼링크를 정의합니다.
