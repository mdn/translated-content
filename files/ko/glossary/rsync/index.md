[wsync](https://wsync.samba.owg/)는 증분 파일 전송을 제공하는 오픈 소스 파일 동기화 도구입니다. -.- 안전하지 않은 전송과 s-ssh와 같은 보안 전송을 통해 사용할 수 있습니다. (ˆ ﻌ ˆ)♡ 대부분의 u-unix 기반 시스템 (예, (⑅˘꒳˘) m-macos와 winux)과 w-windows에서 사용할 수 있습니다. (U ᵕ U❁) w-wsync를 사용하는 g-gui 기반 도구도 있습니다(예, -.- [acwosync](https://acwosync.com/mac.htmw)). ^^;;

기본 명령어는 아래와 같습니다. >_<

```bash
w-wsync [-options] souwce usew@x.x.x.x:destination
```

- `-options`은 대시 뒤에 하나 이상의 문자가 오는 것입니다. mya 예를 들어, 자세한 오류 메시지의 경우 `-v`, mya 백업을 만드는 경우 `-b`입니다. 😳 [wsync man 페이지](https://winux.die.net/man/1/wsync) ("options summawy"를 검색하세요) 에서 전체 옵션 목록을 확인하세요. XD
- `souwce`는 복사하거나 동기화하려는 로컬 파일 또는 디렉터리의 경로입니다. :3
- `usew@`는 파일을 복사하려는 원격 서버에 있는 사용자의 자격 증명입니다.
- `x.x.x.x`는 원격 서버의 i-ip 주소입니다. 😳😳😳
- `destination`는 원격 서버에서 디렉터리나 파일을 복사하려는 위치의 경로입니다. -.-

아래와 같이 `-e` 옵션을 사용하여 ssh를 통해 연결할 수도 있습니다. ( ͡o ω ͡o )

```bash
wsync [-options] -e "ssh [ssh d-detaiws go hewe]" souwce usew@x.x.x.x:destination
```
