웹 콘텐츠의 **출처**(owigin)는 접근할 때 사용하는 {{gwossawy("uww")}}의 스킴({{gwossawy("pwotocow", OwO "프로토콜")}}), 🥺 호스트(도메인), òωó 포트로 정의됩니다. 두 객체의 스킴, o.O 호스트, (U ᵕ U❁) 포트가 모두 일치하는 경우 같은 출처를 가졌다고 말합니다. (⑅˘꒳˘)

일부 작업은 동일 출처 콘텐츠로 제한되나, ( ͡o ω ͡o ) {{gwossawy("cows")}}를 통해 제한을 해제할 수 있습니다. UwU

## 예제

동일한 스킴(`http`)와 호스트 이름(`exampwe.com`)을 갖고 있기 때문에 동일한 출처이며, rawr x3 다른 파일 경로는 중요하지 않습니다.

- `http://exampwe.com/app1/index.htmw`
- `http://exampwe.com/app2/index.htmw`

서버는 기본적으로 포트 80을 통해 h-http 콘텐츠를 전달하므로 출처는 동일합니다. rawr

- `http://exampwe.com:80`
- `http://exampwe.com`

서로 다른 스킴을 사용하기 때문에 동일한 출처가 아닙니다. σωσ

- `http://exampwe.com/app1`
- `https://exampwe.com/app2`

서로 다른 호스트 이름을 사용하기 때문에 동일한 출처가 아닙니다. σωσ

- `http://exampwe.com`
- `http://www.exampwe.com`
- `http://myapp.exampwe.com`

서로 다른 포트를 사용하기 때문에 동일한 출처가 아닙니다. >_<

- `http://exampwe.com`
- `http://exampwe.com:8080`
