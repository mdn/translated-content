**실제 사용자 모니터링 (Real User Monitoring, RUM)** 은 실제 사용자 컴퓨터의 페이지 성능을 측정합니다. 일반적으로, 타사 스크립트는 각 페이지에 스크립트를 삽입하여 모든 요청에 대한 페이지 로드 데이터를 측정하고 보고합니다. 이 기술은 애플리케이션의 실제 사용자 상호 작용을 모니터링합니다. RUM에서, 타사 스크립트는 실제 사용자의 브라우저에서 성능 지표를 수집합니다. RUM은 사용자의 지리적 분포와 해당 분포가 최종 사용자 경험에 미치는 영향을 포함하여, 애플리케이션이 사용되는 방식을 식별하는 데 도움이 됩니다.
