{{Glossary("HTML")}}에서는 {{Glossary("요소")}}을 생성하기 위해 **태그**가 사용됩니다.

HTML 요소의 이름은 요소의 시작 태그 시작 부분과 요소 종료 태그의 끝에 나타나는 이름입니다(요소에 종료 태그가 있는 경우). 예를 들어, `<p>` 시작 태그와 `</p>` 종료 태그의 `p`는 HTML 단락 요소의 이름입니다. 종료 태그의 요소 이름 앞에는 슬래시 문자 `</p>`가 오고 {{glossary("void element", "빈 요소")}}경우 종료 태그가 필요하지도 허용되지도 않습니다.
