Firefox 운영체제 (Firefox OS)는 Mozilla에서 개발한 중단된 오픈소스 모바일 운영체제입니다. 자세한 내용은 [Firefox 운영체제](https://en.wikipedia.org/wiki/Firefox_OS)를 참조하세요.

Firefox 운영체제는 프로젝트가 공식 이름을 가지기 전에는 Boot2Gecko라고도 불렸습니다.
