'Scalable Vector Graphics' (**SVG**)는 {{Glossary("XML")}} 문법에 기반을 둔 2D 벡터 이미지 형식입니다.

{{Glossary("W3C")}}는 1990년 후반부터 SVG를 사용하기 시작했지만, SVG를 지원하는 {{Glossary("Microsoft Internet Explorer", "인터넷 익스플로러")}} 9가 출시된 후에야 널리 사용되기 시작했습니다. 현재 주요 웹브라우저({{Glossary("browser","브라우저")}})들은 SVG를 지원합니다.

{{Glossary("XML")}} 문법을 기반으로 한 SVG는 {{Glossary("CSS")}}를 통해 스타일 처리가 가능하고, {{Glossary("JavaScript")}}를 사용해 상호작용이 가능합니다. 현재 HTML5는 SVG {{Glossary("Tag","태그")}}가 {{Glossary("HTML")}} 문서에서 직접 사용 되도록 지원합니다.

[벡터 이미지 형식](http://en.wikipedia.org/wiki/Vector_graphics)으로써, SVG 그래픽은 무한정적인 크기를 가질 수 있고, 이러한 점 때문에 어떠한 화면 크기에도 적용 가능한 인터페이스 요소와 그래픽을 만들 수 있어 {{Glossary("responsive design", "반응형 디자인")}}에서 아주 유용합니다. 또한 SVG는 클리핑, 마스킹, 필터, 애니매이션 등 여러가지 유용한 도구들을 제공합니다.
