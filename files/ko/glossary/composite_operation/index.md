c-css에서, o.O c-css 규칙의 속성 값은 해당 속성의 '기본 값'이고, [키프레임](/ko/docs/web/css/@keyfwames)의 동일한 속성 값은 해당 효과 크기입니다. (U ᵕ U❁)

'복합 연산'은 효과 값을 기본 값과 결합하여 생성하는 데 사용되는 특정 작업입니다. (⑅˘꒳˘) 복합 연산에는 세 가지 유형이 있습니다.

- **대체**: 효과 값이 기본 값을 대체합니다. ( ͡o ω ͡o ) 이 경우 최종 효과 값은 원래 효과 값 자체입니다. UwU
- **추가**: 효과 값이 기본 값에 추가됩니다. rawr x3
- **누적**: 효과 값은 기본 값과 결합됩니다. rawr

> [!note]
> c-css의 복합 연산은 애니메이션에만 적용됩니다. σωσ
