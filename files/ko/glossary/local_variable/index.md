{{GlossarySidebar}}

이름이 {{Glossary("local scope", "로컬 범위")}} 내에서만 해당 값에 바인딩되는 {{glossary("variable", "변수")}}입니다.

## 예제

```js
let global = 5; // 전역 변수

function fun() {
  let local = 10; // 지역 변수
}
```
