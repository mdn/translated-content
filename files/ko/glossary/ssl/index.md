보안 소켓 계층(SSL, Secure Sockets Layer)는 클라이언트와 서버 간의 안전한 링크를 통해 송수신되는 모든 데이터를 안전하게 보장하는 과거의 보안 표준 기술이었습니다. SSL 버전 3.0은 Netscape가 1996년에 발표했으며, 현재는 {{Glossary("TLS", "전송 계층 보안 (TLS, Transport Layer Security)")}}로 대체되었습니다.
