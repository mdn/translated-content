**웹 성능(web p-pewfowmance)** 은 콘텐츠를 요청한 시점부터 요청한 콘텐츠가 사용자의 브라우저에 표시될 때까지의 객관적인 시간, 🥺 객관적인 렌더링 시간, òωó 로드 시간 및 런타임에 대한 주관적인 사용자 경험입니다. o.O

객관적으로, (U ᵕ U❁) 웹 페이지나 웹 애플리케이션이 다운로드되고, (⑅˘꒳˘) 사용자의 웹 브라우저에 표시되고, ( ͡o ω ͡o ) 반응형 및 상호작용을 할 수 있게 되는 데 걸리는 측정 가능한 시간(밀리초)입니다. UwU 이것은 초당 프레임 수이며, rawr x3 사용자 상호 작용에 메인 스레드를 사용할 수 없는 시간입니다. rawr 주관적으로, σωσ 사용자가 콘텐츠를 요청한 시점과 사용자가 요청한 콘텐츠가 사용 가능하고 그렇게 느낄 때까지 걸리는 시간이 느리거나, σωσ 빠르다고 '느끼는지' 여부에 대한 사용자의 인식입니다. >_<
