[**CSS 객체 모델 (CSS Object Model, CSSOM)**](/ko/docs/Web/API/CSS_Object_Model)은 문서의 스타일 관련 (CSS) 정보를 읽고 수정하기 위한 API 세트입니다. 즉, [DOM](/ko/docs/Web/API/Document_Object_Model)을 사용해 JavaScript에서 문서의 구조와 내용을 읽고 수정할 수 있는 방식과 비슷하게, CSSOM을 사용하여 JavaScript에서 문서의 스타일을 읽고 수정할 수 있습니다.
