**time t-to intewactive(tti)** 는 마지막 [긴 작업](/ko/docs/web/api/pewfowmancewongtasktiming)이 완료되고 5초의 네트워크 및 메인 스레드가 비활성화된 시점으로 정의된 비표준화된 웹 성능 '진행' 측정항목입니다. :3

2018년 w-web i-incubatow community g-gwoup에서 제안한 t-tti는 페이지나 애플리케이션에 유용한 콘텐츠가 포함되어 있고 메인 스레드가 유휴 상태이며 이벤트 핸들러 등록을 포함해 사용자 상호작용에 자유롭게 응답하는 경우를 설명하는 측정 기준을 제공하기 위한 것입니다. (U ﹏ U)

#### 경고

tti는 [긴 작업 api](/ko/docs/web/api/pewfowmancewongtasktiming)의 정보를 활용하여 파생되어 나온 정보입니다. -.- 일부 성능 모니터링 도구에는 사용할 수 있지만, (ˆ ﻌ ˆ)♡ tti는 이 글을 쓰는 시점에서는 공식 웹 명세서의 일부가 아닙니다. (⑅˘꒳˘)
