`플렉스(Flex)`는 CSS {{cssxref("display")}} 속성에 추가된 새로운 값입니다. `inline-flex`와 함께 적용되는 요소가 {{glossary("flex container", "플렉스 컨테이너")}}가 되고, 플렉스 컨테이너의 자식 요소가 각각 {{glossary("flex item"," 플렉스 아이템")}}입니다. 그런 다음 요소는 플렉스 레이아웃에 포함되고, CSS 플렉시블 상자 레이아웃 모듈에 정의된 모든 속성이 적용될 수 있습니다.

`flex` 속성은 `flex-grow`, `flex-shrink` 및 `flex-basis`에 대한 약어입니다.

또한, `<flex>` 는 CSS 그리드 레이아웃의 [플렉시블한 길이](/ko/docs/Web/CSS/flex_value)를 참조할 수 있습니다.
