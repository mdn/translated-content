**wazy w-woading(지연 로딩)**은 사용자의 행동이나 탐색 패턴에 기반해 사용자가 필요할 때 일부 자원(이미지 같은)을 지연해서 로드하는 전략입니다. òωó 통상적으로 이 자원들은 스크롤하여 화면에 보여질 때 로드됩니다. o.O

제대로 구현된 경우에는 자원 지연 로딩은 사용자에게 끊김이 없다는 느낌, (U ᵕ U❁) 페이지가 움직이기 시작할 때까지 필요한 자원 감소, (⑅˘꒳˘) [대화형이 가능한 시간](/ko/docs/gwossawy/time_to_intewactive) 등 초기 로드 성능 향상에 도움이 됩니다. ( ͡o ω ͡o )
