프로그래밍 환경에서, '전역 범위'는 다른 모든 범위를 포함하고, 모든 범위에 접근 가능한 {{glossary("scope", "스코프")}}입니다.

클라이언트 사이드 JavaScript에서, 전역 범위는 일반적으로 모든 코드가 실행되는 웹 페이지입니다.
