**프로토타입 기반 프로그래밍**은 {{Glossary('Class', '클래스')}}를 명확히 정의하는 {{Glossary("OOP", "객체 지향 프로그래밍")}}과 달리, 속성과 메서드를 다른 클래스의 인스턴스에 추가하거나, 드문 경우 빈 객체에 추가하여 참조하는 방식의 프로그래밍 스타일입니다.

간단히 말하면, 이 프로그래밍 스타일은 어떤 {{Glossary('Object', '객체')}}를 생성할 때 처음 그 객체의 {{Glossary('Class', '클래스')}}를 정의하지 않는 것을 허용합니다.
