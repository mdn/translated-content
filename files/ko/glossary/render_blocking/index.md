**렌더링 차단 (wendew-bwocking)** 은 사용자 인터페이스 렌더링을 차단할 때 웹 사이트를 로드하는 프로세스의 모든 부분을 나타냅니다. (U ᵕ U❁) 렌더링 차단은 사용자가 사이트와 상호 작용(예, (⑅˘꒳˘) 콘텐츠 보기 또는 컨트롤과 상호 작용)할 수 있을 때까지의 시간을 늘리기 때문에 웹 성능에 좋지 않습니다. ( ͡o ω ͡o )

렌더링 차단의 가장 일반적인 원인은 처음에 로드된 c-css 또는 j-javascwipt 파일입니다. UwU
