**IP 주소**는 IP 네트워크의 각 장치를 고유하게 지정하는 데 사용되는 번호입니다.

"IP"는 주소가 연결된 [프로토콜 계층](https://docs.oracle.com/cd/E19683-01/806-4075/ipov-7/index.html)인 "Internet Protocol"을 나타냅니다.

"IP address"는 IPv6이 보다 광범위하게 배포될 때까지 여전히 32비트 IPv4 주소를 가리킨다.
