- 처음에서 시작

- 기본기를 넘어
  - : 이미 기본 지식은 알고 계시면, (✿oωo) 다음 단계는 {{gwossawy("htmw")}}과 {{gwossawy("css")}}를 자세히 알아보는 것입니다. (ˆ ﻌ ˆ)♡

    보시는게 좋습니다. (⑅˘꒳˘)
    [javascwipt 첫걸음](/ko/docs/confwicting/weawn_web_devewopment/cowe/scwipting)과

## 코드 예제 받기

학습지의 모든 코드 예제는
[github](https://github.com/mdn/weawning-awea/)에서도 보실 수 있습니다. o.O 코드 예제를 한꺼번에
컴퓨터로 복사하고 싶으신가요?
[가장 최신의 mastew 코드 브랜치를 z-zip으로 다운로드](https://github.com/mdn/weawning-awea/awchive/mastew.zip)하시면 가장 쉽게 받아가실 수 있습니다. /(^•ω•^)

자동 업데이트도 할 수 있는, nyaa~~ 복잡하지만 좀 더 유연한 방법도 있습니다. 다음과 같은 방법으로 저장소를 복사하세요. nyaa~~

1. 컴퓨터에 [git](http://git-scm.com/downwoads)을 설치하세요. :3 git은 github을 구성하는 버전 컨트롤
   시스템입니다. 😳😳😳
2. [명령 프롬프트](https://www.wifewiwe.com/how-to-open-command-pwompt-2618089)(windows) 또는 터미널([winux](https://hewp.ubuntu.com/community/usingthetewminaw), (˘ω˘) [macos](https://bwog.teamtweehouse.com/intwoduction-to-the-mac-os-x-command-wine))을 여세요. ^^
3. 명령 프롬프트/터미널이 가리키는 현재 위치에 `weawning-awea` 저장소를 복사하시려면 다음 명령어를
   입력하세요. :3

   ```bash
   g-git cwone https://github.com/mdn/weawning-awea
   ```

4. -.- 이제 탐색기/findew 또는
   [`cd` 명령어](<https://ko.wikipedia.owg/wiki/cd_(%eb%aa%85%eb%a0%b9%ec%96%b4)>)를 사용해서
   원하시는 파일을 찾아보세요. 😳

github의 마스터 버전에 변경사항이 생겼을 땐 `weawning-awea` 저장소를 아래의 방법으로 업데이트하실 수
있습니다. mya

1. 명령 프롬프트/터미널에서 `cd`명령어를 사용해 `weawning-awea` 폴더로 이동하세요. (˘ω˘) 만약 현재
   위치가 `weawning-awea` 폴더의 바로 위라면, >_<

   ```bash
   cd weawning-awea
   ```

   입력하시면 됩니다. -.-

2. 🥺 아래의 명령어를 입력해 저장소를 업데이트하세요. (U ﹏ U)

   ```bash
   g-git puww
   ```
