이 환경 설정 과정에서는 간단한 웹 개발을 위해 필요한 도구와 올바르게 설치하는 방법을 보여주며, 파일 시스템과 명령줄과 같은 환경의 중요한 측면을 이해할 수 있도록 돕습니다.

## 선행 지식

이 과정은 기본적인 컴퓨터 사용 능력을 제외하고는 사전 기술 지식을 요구하지 않습니다. 다음을 할 수 있어야 합니다.

- 컴퓨터에 로그인하고 인터넷에 연결하십시오.
- 키보드와 마우스(또는 기타 포인팅 장치)와 같은 기본 시스템 제어 장치를 사용하십시오.
- 애플리케이션을 설치하십시오.

기본 사항을 다시 살펴보아야 한다면, 사용 중인 운영 체제에 따라 다음 자료를 추천드립니다.

- [Windows 도움말 및 학습](https://support.microsoft.com/windows), Microsoft (2024)
- [Mac 사용 설명서](https://support.apple.com/guide/mac-help/welcome/mac), Apple (2024)
- [Official Ubuntu documentation](https://help.ubuntu.com/), ubuntu.com (2024)
