**시작하기**에 오신 것을 환영합니다! 만약 여러분이 완전 초보자라면(즉, 아직 코드 편집기를 설치해 보지 않았거나 코드를 작성해 본 적이 없다면), 이곳이 여러분을 위한 곳입니다. 이 과정에서는 필요한 소프트웨어를 설치하는 방법, 개발 환경에 익숙해지는 과정, 간단한 웹사이트 제작에 첫발을 내딛는 방법, 그리고 웹 개발을 둘러싼 필수 개념들을 이해하도록 안내합니다.

## 선행 지식

이 과정을 시작하는 데 필요한 선행 지식은 없습니다. 또한, [핵심 과정](/ko/docs/Learn_web_development/Core)으로 넘어가기 전에 이 주제들을 학습해야 하진 않지만, 먼저 이 주제들에 대해 시간을 투자하면 핵심 과정을 수월하게 진행할 수 있을 것입니다.

## 과정

- [환경 설정](/ko/docs/Learn_web_development/Getting_started/Environment_setup)
  - : 이 환경 설정 과정에서는 간단한 웹 개발을 위해 필요한 도구와 올바르게 설치하는 방법을 보여주며, 파일 시스템과 명령줄과 같은 환경의 중요한 측면을 이해할 수 있도록 돕습니다
- [Web과 함께 시작하기](/ko/docs/Learn_web_development/Getting_started/Your_first_website)
  - : 이 과정은 여러분에게 웹 개발의 실질적인 측면을 소개하는 간결한 시리즈입니다. 여러분은 간단한 웹페이지를 만들 때 필요한 도구를 설치하고 여러분의 코드를 게시할 것입니다.
- [웹 표준](/ko/docs/Learn_web_development/Getting_started/Web_standards)
  - : 이 과정은 웹이 작동하는 기본 원리를 높은 수준에서 다룹니다. 여기에는 통신에 사용되는 모델, 관련 핵심 기술, 해당 기술이 생성되는 방식, 그리고 웹 브라우저가 웹사이트를 렌더링하고 사용자에게 표시하는 방법이 포함됩니다.
- [소프트 스킬](/ko/docs/Learn_web_development/Getting_started/Soft_skills)
  - : 이 과정은 웹 개발을 배우는 동안 향상시킬 수 있는 소프트 스킬과 업계에 진입할 때 갖추면 좋은 특성에 대한 권장 사항을 제공합니다. 이러한 기량은 학습, 연구, 협업에 적합한 태도를 개발하는 데 큰 도움을 줄 뿐만 아니라 취업 가능성을 높이는 데도 기여할 것입니다.
