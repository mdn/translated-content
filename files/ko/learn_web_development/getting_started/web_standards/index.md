이 과정은 웹이 어떻게 동작하는지에 대한 기본 개념을 높은 수준에서 다룹니다. 여기에는 통신에 사용되는 모델, 핵심 기술, 이러한 기술이 어떻게 만들어지는지, 그리고 웹 브라우저가 웹사이트를 사용자에게 어떻게 렌더링하고 표시하는지 등이 포함됩니다.

일반 자료

- [Resilient Web Design](https://resilientwebdesign.com/), Jeremy Keith

## 선행 지식

이 과정은 주로 이론적인 내용을 다루며, 웹 기술에 대한 특정 실무 지식을 전제로 하지 않습니다. 그러나 주요 웹 기술이 무엇에 사용되는지에 대한 기본적인 이해가 있다면 내용을 더 잘 이해하고 더 많은 것을 얻을 수 있습니다. 아직 [Web과 함께 시작하기](/ko/docs/Learn_web_development/Getting_started/Your_first_website) 과정을 학습하지 않았다면, 먼저 해당 과정을 진행하실 것을 권장합니다.

## 자습서

- [웹의 동작 방식](/ko/docs/Learn_web_development/Getting_started/Web_standards/How_the_web_works)
  - : 이 글에서는 웹 브라우저를 사용하여 웹 페이지에 접속할 때 어떤 일이 일어나는지, 브라우저가 볼 수 있는 형태로 웹 페이지를 구성할 수 있도록 관련 코드가 컴퓨터로 전달되는 과정의 이면에 숨겨진 마법을 높은 수준에서 설명합니다.
- [웹 표준 모델](/ko/docs/Learn_web_development/Getting_started/Web_standards/The_web_standards_model)
  - : 이 글은 웹과 웹 표준에 대한 유용한 배경 지식을 제공합니다. 어떻게 웹과 웹 표준 기술이 등장하게 되었는지, 웹 표준 기술이 무엇인지, 그리고 어떻게 이들이 함께 동작하는지에 대해 설명합니다.
- [브라우저가 웹사이트를 로드하는 방법](/ko/docs/Learn_web_development/Getting_started/Web_standards/How_browsers_load_websites)
  - : 이 글은 웹사이트의 렌더링 과정에 대해 설명합니다. 브라우저가 웹사이트를 구성하는 파일과 도구를 모두 받았을 때, 사용자와 상호작용하는 완성된 경험을 만들기 위해 어떻게 결합하는지 다룹니다.
