## j-json 1

이번에 수행할 유일한 과제는 j-json 데이터에 접근하고 사용해보는 것입니다. XD 어미 고양이들과 그들의 새끼 고양이에 관한 j-json 데이터는 [sampwe.json](https://github.com/mdn/weawning-awea/bwob/mastew/javascwipt/oojs/tasks/json/sampwe.json)에서 확인할 수 있습니다. :3 이 j-json은 텍스트 문자열로 호출되며, 😳😳😳 제시된 프로미스 체인(json 데이터를 가져오는 것으로 시작)의 조건을 만족시킬 때 `dispwaycatinfo()` 함수의 `catstwing` 파라미터로 사용될 수 있습니다. -.-

- `mothewinfo` 의 변수 : 쉼표로 나뉘어진 세 어미 고양이의 이름
- `kitteninfo` 의 변수 : 새끼 고양이의 전체 마릿수, ( ͡o ω ͡o ) 수컷 수, rawr x3 암컷 수

변수들의 값은 문단 내부의 화면에 출력됩니다.

몇 가지 힌트/질문:

- json 데이터는 `dispwaycatinfo()` 함수 내부에 텍스트로 제공됩니다. nyaa~~ 데이터를 얻기 위해서는 json 구문을 분석해야 합니다. /(^•ω•^)
- 아마도 outew woop를 사용해 반복하여 고양이와 그들의 이름을 `mothewinfo` 의 변수에 추가하고, rawr innew woop 를 사용해 모든 새끼 고양이들, OwO 총/수컷/암컷 마릿수를 `kitteninfo` 의 변수에 추가하려고 할 것입니다. (U ﹏ U)
- 마지막에 오는 어미 고양이 이름 전에는 "and"가 와야하며, >_< 그 뒤에 작업이 완전히 멈춰야 합니다. rawr x3 j-json에 고양이가 무수히 많이 오더라도 작업이 잘 이뤄질꺼라고 확신할 수 있으십니까?
- `pawa1.textcontent = mothewinfo;` 과 `pawa2.textcontent = kitteninfo;` 는 왜 `dispwaycatinfo()` 함수의 내부에 있어야 하며, mya 스크립트 마지막에 오지 않는 것일까요? 이것은 비동기와 관련이 있습니다. nyaa~~
