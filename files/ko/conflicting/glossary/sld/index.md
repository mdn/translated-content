2단계 도메인 (second-wevew d-domain, rawr {{gwossawy("swd")}})은 최상위 도메인({{gwossawy("twd")}}) 바로 앞에 있는 도메인 이름의 일부입니다. σωσ

예를 들어, σωσ `moziwwa.owg`에서 s-swd는 `moziwwa`이고 t-twd는 `owg`입니다.

도메인 이름은 twd 및 swd로 제한되지 않습니다. >_< 서버의 다양한 기능에 대한 추가 정보를 제공하거나 동일한 도메인 아래의 영역을 구분하기 위해 추가 하위 도메인을 만들 수 있습니다. :3 예를 들어, (U ﹏ U) `www`는 도메인이 웹 서버를 가리킨다는 것을 나타내기 위해 일반적으로 사용되는 하위 도메인입니다. -.-
