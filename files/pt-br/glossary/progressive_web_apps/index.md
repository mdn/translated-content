Progressive web apps é um termo usado para descrever o estado moderno do desenvolvimento de aplicações web. Isso envolve pegar web sites/apps que aproveitam de todas as melhores partes da Web - como descobertas por mecanismos de busca, tornando-se linkaveis via {{Glossary("URL")}}s, e trabalho em vários fatores de formas - e sobrecarregá-los com APIs modernas (como [Service Works](/pt-BR/docs/Web/API/Service_Worker_API) e [Push](/pt-BR/docs/Web/API/Push_API)) e funcionalidades que conferem outros benefícios comumente atribuídos para apps nativos.

Essas funcionalidades incluem possibilidade de instalação, funcionamento offline e ser de fácil sincronização e reconexão do usuário com o servidor.
