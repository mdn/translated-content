O elemento circle é uma forma básica do SVG, utilizada para criar círculos baseado em um ponto central e um raio.

Contexto de uso

{{svginfo}}

Exemplo

 

<?xml version="1.0"?>
<svg viewBox="0 0 120 120" version="1.1"
  xmlns="http://www.w3.org/2000/svg">
  <circle cx="60" cy="60" r="50"/>
</svg>
