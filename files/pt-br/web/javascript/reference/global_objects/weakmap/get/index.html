O método get() retorna um elemento especificado de um objeto WeakMap.

{{EmbedInteractiveExample("pages/js/weakmap-prototype-get.html")}}

A fonte para este exemplo interativo está em um repositório no GitHub. Caso você queira contribuir para o projeto de exemplos interativos, por favor clone https://github.com/mdn/interactive-examples e nos envie uma pull request.

Sintaxe

wm.get(chave);

Parâmetros

chave
Obrigatório. A chave do elemento a ser retornado pelo objeto WeakMap.

Valor de retorno

O elemento associado à chave especificada no objeto WeakMap. Se a chave não for encontrada, retorna-se {{jsxref("undefined")}}.

Exemplos

Utilizando o método get 

var wm = new WeakMap();
wm.set(window, 'foo');

wm.get(window); // Retorna "foo".
wm.get('baz');  // Retorna undefined.
