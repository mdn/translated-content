O cabeçalho de entidade **`Content-Length`** indica o tamanho do corpo da entidade, em bytes, enviado ao destinatário.

  <tbody>

    </tr>

## Sintaxe

```
Content-Length: <tamanho>
```

## Diretivas

- \<tamanho>
  - : O tamanho em número decimal de octetos.
