## Descrição

A [pseudo-classe](/pt-BR/docs/Web/CSS/Pseudo-classes) CSS `:only-of-type` representa qualquer elemento que não possui irmãos de um determinado tipo.

## Sintaxe

```
:only-of-type { propriedades}
```
