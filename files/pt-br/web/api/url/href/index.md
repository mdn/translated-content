A propriedade **`href`** da interface {{domxref("URL")}} é uma {{jsxref("String")}} contendo toda a URL

{{AvailableInWorkers}}

## Valor

Uma {{jsxref("String")}}.

## Exemplos

```js
const url = new URL(
  "https://developer.mozilla.org/pt-BR/docs/Web/API/URL/href",
);
console.log(url.href); // Logs: 'https://developer.mozilla.org/pt-BR/docs/Web/API/URL/href'
```
