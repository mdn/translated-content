O construtor **`Document`** cria um novo objeto {{domxref("Document")}}, o qual trata de uma página carregada no navegador servindo como porta de entrada para o conteúdo da página.

## Sintaxe

```
var document = new Document()
```

### Parâmetros

Nenhum.
