O elemento `<xsl:comment>` escreve um comentário no documento de saída. Deve conter apenas texto.

## Sintaxe

```xml
<xsl:comment>
  TEMPLATE
</xsl:comment>
```

### Atributos obrigatórios

Nenhum.

### Atributos opcionais

Nenhum.

### Tipo

Instrução, aparece dentro de um template.

## Especificações

XSLT, seção 7.4.

## Compatibilidade com Gecko

Suportado.
