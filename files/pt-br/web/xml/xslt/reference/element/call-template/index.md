O elemento `<xsl:call-template>` invoca um template nomeado.

## Sintaxe

```xml
<xsl:call-template name=NAME>
  <xsl:with-param> [opcional]
</xsl:call-template>
```

### Atributos obrigatórios

- `name`
  - : Especifica o nome do template que se deseja invocar.

### Atributos opcionais

Nenhum.

### Tipo

Instrução, aparece dentro de um template.

## Especificações

XSLT, seção 6.

## Compatibilidade com Gecko

Suportado.
