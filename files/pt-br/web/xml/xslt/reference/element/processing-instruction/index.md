O elemento `<xsl:processing-instruction>` escreve uma instrução de processamento no documento de saída.

## Sintaxe

```xml
<xsl:processing-instruction name=NAME>TEMPLATE</xsl:processing-instruction>
```

### Atributos obrigatórios

- `name`
  - : Especifica o nome desta instrução de processamento.

### Atributos opcionais

Nenhum.

### Tipo

Instrução, aparece dentro de um template.

## Especificações

XSLT, seção 7.3

## Compatibilidade com Gecko

Compatível.
