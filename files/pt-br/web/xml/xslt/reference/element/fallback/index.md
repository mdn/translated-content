O elemento `<xsl:fallback>` especifica qual template usar caso um determinado elemento de extensão (ou, eventualmente, de versão mais recente) não seja suportado.

## Sintaxe

```xml
<xsl:fallback>
  TEMPLATE
</xsl:fallback>
```

### Atributos obrigatórios

Nenhum.

### Atributos opcionais

Nenhum.

### Tipo

Instrução, aparece dentro de um template.

## Especificações

XSLT, seção 15.

## Compatibilidade com Gecko

Não suportado no momento.
