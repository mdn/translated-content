O elemento `<xsl:otherwise>` é usado para definir a ação que deve ser executada quando nenhuma das condições `<xsl:when>` se aplica. É semelhante ao caso `else` ou `default` em outras linguagens de programação.

## Sintaxe

```xml
<xsl:otherwise>
  TEMPLATE
</xsl:otherwise>
```

### Atributos obrigatórios

Nenhum.

### Atributos opcionais

Nenhum.

### Tipo

Sub-instrução, deve aparecer como o último filho de um elemento `<xsl:choose>`, dentro de um template.

## Especificações

XSLT, seção 9.2

## Compatibilidade com Gecko

Compatível.
