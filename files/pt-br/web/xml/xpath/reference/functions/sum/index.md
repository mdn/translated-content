A função `sum` retorna um número que é a soma dos valores numéricos de cada nó em um conjunto de nós fornecido.

## Sintaxe

```plain
sum(node-set)
```

### Parâmetros

- `node-set`
  - : O conjunto de nós a ser avaliado. Cada nó neste conjunto de nós é avaliado como se fosse passado para a função [number()](/pt-BR/docs/Web/XML/XPath/Reference/Functions/number), e a soma dos números resultantes é retornada.

### Valor de retorno

Um número.

## Especificações

[XPath 1.0 4.3](https://www.w3.org/TR/xpath-10/#function-sum)

## Compatibilidade com Gecko

Compatível.
