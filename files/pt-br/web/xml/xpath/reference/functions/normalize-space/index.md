A função `normalize-space` remove os espaços em branco no início e no fim de uma string, substitui sequências de caracteres de espaço em branco por um único espaço e retorna a string resultante.

## Sintaxe

```plain
normalize-space( [string] )
```

### Parâmetros

- `string` (opcional)
  - : A string a ser normalizada. Se omitida, a string usada será a mesma que o nó de contexto convertido em string.

### Valor de retorno

A string normalizada.

## Especificações

[XPath 1.0 4.2](https://www.w3.org/TR/xpath-10/#function-normalize-space)

## Compatibilidade com Gecko

Compatível.
