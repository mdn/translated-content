A função `count` conta o número de nós em um conjunto de nós e retorna um inteiro.

## Sintaxe

```plain
count( node-set )
```

### Parâmetros

- `node-set`
  - : O conjunto de nós a ser contado.

### Valor de retorno

Um inteiro representando o número de nós em um conjunto de nós.

## Especificações

[XPath 1.0 4.1](https://www.w3.org/TR/xpath-10/#function-count)

## Compatibilidade com Gecko

Suportado.
