A função `starts-with` verifica se a primeira string começa com a segunda string e retorna true ou false.

## Sintaxe

```plain
starts-with(haystack, needle)
```

### Parâmetros

- `haystack`
  - : A string onde se busca.
- `needle`
  - : A string a ser procurada.

### Valor de retorno

`true` se `haystack` começa com `needle`. Caso contrário, `false`.

## Especificações

[XPath 1.0 4.2](https://www.w3.org/TR/xpath-10/#function-starts-with)
