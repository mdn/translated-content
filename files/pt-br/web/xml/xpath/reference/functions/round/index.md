A função `round` retorna um número que é o inteiro mais próximo do número fornecido.

## Sintaxe

```plain
round( decimal )
```

### Parâmetros

- `decimal`
  - : O número decimal a ser arredondado.

### Valor de retorno

O inteiro mais próximo, menor que, maior que ou igual a `decimal`.

## Descrição

- \-0.5 é arredondado para zero negativo. 0.4 é arredondado para zero positivo.
