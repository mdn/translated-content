A função `concat` concatena duas ou mais strings e retorna a string resultante.

## Sintaxe

```plain
concat( string1, string2 [,stringN]* )
```

### Parâmetros

- `stringN`
  - : Esta função aceita dois ou mais argumentos. Cada um desses argumentos é uma string.

### Valor de retorno

Uma única string que é a concatenação de todas as strings passadas à função como argumentos.

[XPath 1.0 4.2](https://www.w3.org/TR/xpath-10/#function-concat)
