A função `contains` determina se a string do primeiro argumento contém a string do segundo argumento e retorna o booleano verdadeiro ou falso.

## Sintaxe

```plain
contains(haystack, needle)
```

### Parâmetros

- `haystack`
  - : A string a ser pesquisada
- `needle`
  - : A string a ser procurada como uma substring de `haystack`

### Valor de retorno

`true` se `haystack` contiver `needle`. Caso contrário, `false`.

## Especificações

[XPath 1.0 4.2](https://www.w3.org/TR/xpath-10/#function-contains)
