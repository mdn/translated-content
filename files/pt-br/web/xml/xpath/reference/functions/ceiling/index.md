A função `ceiling` avalia um número decimal e retorna o menor inteiro maior ou igual ao número decimal.

## Sintaxe

```plain
ceiling( number )
```

### Parâmetros

- `number`
  - : O número a ser avaliado.

### Valor de retorno

O inteiro mais próximo maior ou igual a `number`.

Por exemplo:

`ceiling (5.2)` = _6_

`ceiling (-5.2)` = _-5_
