A função `last` retorna um número igual ao tamanho do contexto a partir do contexto de avaliação da expressão.

## Sintaxe

```plain
last()
```

### Valor de retorno

Um inteiro igual ao tamanho do contexto a partir do contexto de avaliação da expressão.

## Descrição

- Frequentemente é usada com a função [position()](/pt-BR/docs/Web/XML/XPath/Reference/Functions/position) para determinar se um nó específico é o último em um conjunto de nós.

## Especificações

[XPath 1.0 4.1](https://www.w3.org/TR/xpath-10/#function-last)

## Compatibilidade com Gecko

Compatível.
