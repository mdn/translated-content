A função `function-available` determina se uma determinada função está disponível e retorna o booleano verdadeiro ou falso.

## Sintaxe

```plain
function-available( name )
```

### Parâmetros

- `name`
  - : O nome da função a ser testada.

### Valor de retorno

Booleano `true` ou `false`.

## Especificações

[XSLT 1.0 15](https://www.w3.org/TR/xslt-10/#function-function-available)

## Compatibilidade com Gecko

Suportado.
