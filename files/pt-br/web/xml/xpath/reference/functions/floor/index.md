A função `floor` avalia um número decimal e retorna o maior inteiro menor ou igual ao número decimal.

## Sintaxe

```plain
floor( number )
```

### Parâmetros

- `number`
  - : O número decimal a ser avaliado.

### Valor de retorno

O inteiro mais próximo menor ou igual a `number`.

## Especificações

[XPath 1.0 4.4](https://www.w3.org/TR/xpath-10/#function-floor)
