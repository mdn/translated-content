A função `string-length` retorna um número igual ao número de caracteres em uma determinada string.

## Sintaxe

```plain
string-length( [string] )
```

### Parâmetros

- `string` (opcional)
  - : A string a ser avaliada. Se omitida, a string usada será a mesma que o nó de contexto convertido em string.

### Valor de retorno

Um inteiro igual ao número de caracteres na string.

## Especificações

[XPath 1.0 4.2](https://www.w3.org/TR/xpath-10/#function-string-length)
