Bem-vinda à **Introdução**!
Se você é uma pessoa iniciante (ou seja, ainda não instalou um editor de código
ou escreveu qualquer código), este é o lugar para você.
Os módulos de introdução te guiam pela instalação do software necessário,
familiarização com seu ambiente de desenvolvimento, sua primeira tentativa de
construir um site simples e compreensão de alguns dos conceitos essenciais do
desenvolvimento web.

## Pré-requisitos

Não há pré-requisitos de conhecimento para iniciar este curso.
Observe também que não consideramos o aprendizado desses tópicos obrigatório
antes de passar para os
[módulos básicos](/pt-BR/docs/Learn_web_development/Core), mas acreditamos que
as pessoas alunas terão mais facilidade se dedicarem algum tempo a esses tópicos
primeiro.

## Módulos

- [Configuração do ambiente](/pt-BR/docs/Learn_web_development/Getting_started/Environment_setup)
  - : No módulo _Configuração do ambiente_, mostramos quais ferramentas você
    precisa para um desenvolvimento web simples e como instalá-las corretamente,
    além de ajudar você a entender aspectos importantes do seu ambiente, como
    sistemas de arquivos e linha de comando.
- [Seu primeiro site](/pt-BR/docs/Learn_web_development/Getting_started/Your_first_website)
  - : Este módulo apresenta os aspectos práticos do desenvolvimento web.
    Você reunirá os _assets_ e escreverá o código para construir e publicar uma
    página web simples.
- [Padrões da Web](/pt-BR/docs/Learn_web_development/Getting_started/Web_standards)
  - : Este módulo aborda os fundamentos de como a web funciona em alto nível —
    incluindo o modelo usado para comunicação, as principais tecnologias
    envolvidas, como essas tecnologias são criadas e como um navegador web
    renderiza e exibe sites para uma pessoa usuária.
- [Habilidades interpessoais](/pt-BR/docs/Learn_web_development/Getting_started/Soft_skills)
  - : Este módulo fornece recomendações de habilidades interpessoais que você
    pode aprimorar enquanto aprende desenvolvimento web e que constituem boas
    características para ingressar no setor.
    Elas ajudarão imensamente no desenvolvimento das atitudes certas para
    aprender, pesquisar e colaborar, além de aumentar as chances de ser
    contratada.
