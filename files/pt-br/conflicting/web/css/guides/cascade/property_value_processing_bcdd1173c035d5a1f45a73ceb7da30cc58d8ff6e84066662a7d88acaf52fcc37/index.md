O **Valor inicial** de uma propriedade [CSS](/pt-BR/docs/Web/CSS) é o seu valor padrão, como listado em sua tabela de definição. O uso do valor inicial varia caso a propriedade seja herdada ou não.

- Para [propriedades herdadas](/pt-BR/docs/Web/CSS/Guides/Cascade/Inheritance#propriedades_herdadas), o valor inicial é usado apenas em seu elemento raíz, desde que nenhum [valor especificado](/pt-BR/docs/conflicting/Web/CSS/Guides/Cascade/Property_value_processing_8a702f25e61d14d8bd6ee6f7e88ca236512427b92b01a92c1d11f9edbb8f5fe1) seja fornecido.
- Para [propriedades não-herdadas](/pt-BR/docs/Web/CSS/Guides/Cascade/Inheritance#propriedades_nao_herdadas), o valor inicial é usado em todos os elementos, enquanto nenhum [valor especificado](/pt-BR/docs/conflicting/Web/CSS/Guides/Cascade/Property_value_processing_8a702f25e61d14d8bd6ee6f7e88ca236512427b92b01a92c1d11f9edbb8f5fe1) é fornecido

> [!NOTE]
> Você pode especificar explicitamente um valor inicial, utilizando a palavra-chave {{cssxref("initial")}}
