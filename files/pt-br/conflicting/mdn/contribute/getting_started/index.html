Somos uma comunidade aberta de desenvolvedores criando recursos para uma web ainda melhor, independente do nosso navegador, programas ou marca. Qualquer um pode contribuir e cada pessoa que participa nos faz mais forte. Juntos, podemos continuar a impulsionar a inovação na web para servir o bem maior. Tudo começa aqui, com você.

Junte-se a Comunidade

Se quiser conselhos, ajuda, ou apenas nos conhecer, por favor, junte-se a nós nos seguintes canais:

Canais no IRC – para conversar com colaboradores MDN:

  • #devmo: Discussões sobre o conteúdo da documentação na MDC
  • #mdn: Questões gerais sobre a MDN

A reunião da comunidade MDN acontece a cada duas quartas-feiras às 10:00 da manhã, horário do pacífico (18:00 UTC inverno; 17:00 UTC verão) no canal  #devmo no irc.mozilla.org.

Para entrar no IRC, siga essas instruções da wiki: https://wiki.mozilla.org/IRC#Getting_Started

Listas de Email – comunicações menos frequentes para toda a comunidade MDN:

  • mozilla.dev.mdc: MDN Documentation Center  - discussões sobre a produção de conteúdo.
  • mozilla.dev.mdn: Desenvolvimento técnico do site para a plataforma MDN.
  • mozilla.dev.mdc.es: MDN Documentation Center – Comunidade de tradução para o Espanhol
  • mozilla.dev.mdc.pt: MDN Documentation Center - Comunidade de tradução para o  Português.

Acesse o MDN

Para adicionar, editar, ou traduzir um documento, você precisará entrar no site da MDN:

https://developer.org

Uma vez no site, você terá que criar uma conta na MDN. A Mozilla Developer Network usa o Persona (também conhecido como BrowserID) para gerenciar contas. O Persona é uma opção segura e fácil que protege a privacidade do usuário.

Criando sua conta

  1. Clique no botão Entrar na parte superior da página. Uma janela abrirá com o login do Persona.
  2. Coloque o seu endereço de e-mail e clique em próximo.

O que acontece depois dependerá se você já usou este endereço de e-mail com o Persona antes.

  1. Se você ainda não tiver o e-mail no Persona, uma janela abrirá solicitando que você escolha uma senha. Digite a senha duas vezes e clique em verificar.
    1. Faça a verificação do seu e-mail, acessando sua conta. Você receberá um e-mail do BrowserID@browserid.org (verifique sua caixa de SPAM se necessário).
    2. Ao abrir o e-mail clique no link de registro.
    3. Digite a senha para este endereço de e-mail (duas vezes), e clique em Concluir. Na guia MDN ou na janela (onde originalmente você clicou para entrar), o MDN mostrará uma página de criação de conta.
    4. Digite um nome de usuário para associar à sua conta e clique em criar novo perfil. Observe que o nome de usuário não pode conter espaços.
  3. Se você já usou o Persona antes, digite sua senha existente para este endereço de e-mail e clique em verificar.

Parabéns, agora você tem uma conta na MDN e poderá criar, editar e traduzir o conteúdo da documentação.

Editando um conteúdo na MDN

Comece editando uma página

  1. Verifique se você está logado na MDN (veja a seção acima para instruções)
  2. Mude a página para o modo de edição. O conteúdo da página aparecerá dentro de um editor.

Salve o seu trabalho

O editor do MDN oferece várias maneiras para você verificar e salvar seu trabalho.

  1. Save changes: Salva as alterações e sai do modo de edição, voltando à página para modo de leitura. Este é o mesmo que o botão Salvar e Sair na barra de ferramentas do editor.
  2. Save and keep editing: Salva as alterações, mantendo-se no modo de edição. Este é o mesmo que o botão de Salvar e Continuar Editando na barra de ferramentas do editor.
  3. Preview changes: Aplica as alterações em outra aba do navegador. Esta aba executa todos os modelos que são referenciados na página, para que você possa ver como os modelos aparecerão para os leitores.
  4. Discard changes: Joga fora suas alterações desde a última gravação, e retorna a página para modo de leitura.

Modifique a informação da página

Enquanto uma página está em modo de edição, você pode modificar alguns metadados da mesma. Clique no botão "i" para abrir os campos de informações das páginas.

  • Title: O título que aparece na barra de título do navegador e na parte superior do conteúdo da página.
  • Slug: A última parte da URL para a página. Normalmente, isso é baseado no título da página (substituições apropriadas para espaços e caracteres especiais), mas você pode modifica-lo.
  • TOC: Se esta caixa for marcada, uma tabela de conteúdo é gerada automaticamente para a página, com base nas posições dentro da página (até H4), em uma caixa de perto do topo da página.

Quaisquer alterações a estes campos são salvos quando você salva a página como um todo.

Formatação do Texto

O editor oferece várias ferramentas, que poderão ajuda-lo a formatar o texto. Algumas delas são comuns em muitos editores e não precisam de explicação.

  • Source: Muda o editor de edição visual para edição de código HTML.
  • H1, H2, H3, H4, H5, H6: Esses botões correspondem exatamente aos elementos de titulo em HTML. Deve haver apenas um elemento <h1> em uma página.
  • PRE: Envolve o texto selecionado em um elemento <pre>. Ao clicar neste botão, você ativa o menu Syntax Highlighter que aparecerá ao lado dele, o que permite que você selecione o tipo de código, de modo que o Syntax Highlighter possa ser aplicado adequadamente.
  • Styles: Selecionando um item deste menu, envolve o texto selecionado em um <div> (para block styles) ou um <span> (para inline styles) com o identificador de classe correspondente. Isso permite que o texto seja modificado de maneira apropriada no style sheets da MDN.
  • <>: Este botão envolve a seleção em um bloco <code>, que é útil para indicar nomes de APIs, variáveis ​​e arquivos.

Traduzindo uma página

  1. No menu Languages​​, clique em Add Translation.
  2. A página do idioma selecionado aparecerá.
  3. Clique no idioma que deseja traduzir. Deverá abrir o Translating Article, com o texto no idioma original apresentado no lado esquerdo.
  4. Traduza o título e o conteúdo da página para a língua selecionada. Para isso, clique no botão "i" para aparecer as informações que você pode editar.
  5. Clique em salvar mudanças quando tiver terminado.

Nota: Os elementos da interface do usuário "Translating Article" são inicialmente mostrados em Inglês.

Editando uma página traduzida

Em uma página traduzida, clique no botão Editar (que pode estar na sua língua nativa). A "Visão do Artigo Traduzido" se abrirá.
