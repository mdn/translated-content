概述

onmouseup属性用来获取或设置当前元素的mouseup事件的事件处理函数。

语法

element.onMouseUp = event handling code

备注

当用户在当前元素上放开鼠标某个按键时会触发mouseup事件。

规范

不属于任何公开的规范。
