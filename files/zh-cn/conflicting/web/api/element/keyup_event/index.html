Summary

onkeyup属性用来获取或设置当前元素的keyup事件的事件处理函数。

语法

element.onkeyup = event handling code

例子

 <input type="text" onKeyUp="keyWasPressed(event)">
 <script>function keyWasPressed(evt){ alert(evt.keyCode) }</script>

备注

在当前元素上释放键盘按键时会触发keyup 事件。

规范

不属于任何公开的规范。
