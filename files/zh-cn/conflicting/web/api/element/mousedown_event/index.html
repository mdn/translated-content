概述

onmousedown属性用来获取或设置当前元素的mousedown事件的事件处理函数。

语法

element.onmousedown= event handling code

备注

在当前元素上鼠标按下瞬间会触发 mousedown 事件。

规范

不属于任何公开的规范。
