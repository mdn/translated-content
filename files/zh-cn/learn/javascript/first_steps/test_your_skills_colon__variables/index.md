## 变量 3

现在是最后一个任务——现有代码中存在 2 处错误，请找出并修正，使输出窗口得到正确结果，即：“我是某某，20 年后我？岁。”

```html hidden

  <section class="preview"></section>

  <textarea class="playable playable-js" style="height: 300px;">

let myAge = '18';
```
