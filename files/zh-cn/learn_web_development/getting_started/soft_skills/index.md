本模块提供了在学习 Web 开发过程中可努力提升的软技能建议，这些技能也是进入行业时应具备的良好素质。它们将极大助益于培养正确的学习态度、研究态度与协作态度，并提高获得聘用的机会。

## 前置知识

本模块主要为理论性内容，不要求你事先掌握任何特定的 Web 技术实践知识。不过，我们仍建议你在学习本模块之前，按顺序完成前面的各模块。

## 教程

- [研究与学习](/zh-CN/docs/Learn_web_development/Getting_started/Soft_skills/Research_and_learning)
  - : 作为一名 Web 开发者，你需要掌握的知识非常广泛，而且需要持续不断地学习。但你该如何最有效地发现自身最需掌握的内容呢？本文提供了一些关于研究与学习的实用技巧。
- [协作与团队合作](/zh-CN/docs/Learn_web_development/Getting_started/Soft_skills/Collaboration_and_teamwork)
  - : 作为 Web 行业的专业人员，你将在项目的多个阶段（例如：头脑风暴、撰写提案、编写代码等）与其他人员共同工作。本文探讨了如何与他人协同合作。
- [工作流与流程](/zh-CN/docs/Learn_web_development/Getting_started/Soft_skills/Workflows_and_processes)
  - : 初学者往往专注于技术中具体的个别方面，却忽略了整个项目更宏观的背景。本文提供了一些高层次的上下文信息。
- [求职](/zh-CN/docs/Learn_web_development/Getting_started/Soft_skills/Finding_a_job)
  - : 技术岗位面试可能要求极高，有些职位还有相当具体的要求。而首先找到适合申请的职位本身便颇具挑战性！本文提供了一些技巧与资源，以协助你完成从寻找潜在职位、提交申请，到成功通过面试的全过程。
