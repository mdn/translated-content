欢迎来到**入门模块**！如果你是个完全的初学者（例如你还没有安装代码编辑器或开始编写代码），那就来这里吧。入门模块将带领你安装需要的软件，熟悉开发环境，尝试构建一个简单的网站，了解 Web 开发中的一些基本概念。

## 前提

开始这门课程不需要前置知识。另请注意，我们不认为在继续学习[核心模块](/zh-CN/docs/Learn_web_development/Core)之前学习下面这些内容是必需的，但如果先在这里上花一些时间，后面的课程将会更容易学习。

## 模块

- [环境搭建](/zh-CN/docs/Learn_web_development/Getting_started/Environment_setup)
  - : 在*环境搭建*模块中，我们将向你展示进行简单 Web 开发所需的工具、正确安装它们的方法，并帮助你了解有关环境的要点，如文件系统和命令行。
- [你的第一个网站](/zh-CN/docs/Learn_web_development/Getting_started/Your_first_website)
  - : 本模块将向你介绍 Web 开发的实际操作。你将收集资源并编写代码，构建并发布一个简单的网页。
- [Web 标准](/zh-CN/docs/Learn_web_development/Getting_started/Web_standards)
  - : 本模块涵盖了 Web 的基本工作原理——包括用于通信的模型、涉及的核心技术、这些技术是如何创建的，以及 Web 浏览器是如何将网站渲染并显示给用户的。
- [软技能](/zh-CN/docs/Learn_web_development/Getting_started/Soft_skills)
  - : 本模块提供了一些你可以在学习 Web 开发时着力提升的软技能建议，这也是进入行业所需的良好特质。它们将极大地帮助培养正确的学习、研究、协作态度，让你更容易找到工作。
