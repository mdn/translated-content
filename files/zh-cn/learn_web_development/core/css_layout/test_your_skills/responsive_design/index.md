本技能测试用于评估你是否理解[使用媒体查询的方法](/zh-CN/docs/Learn_web_development/Core/CSS_layout/Media_queries)，并通过实际任务让你练习响应式网页设计。完成此任务所需的所有知识都包含在 [CSS 布局模块](/zh-CN/docs/Learn_web_development/Core/CSS_layout)的布局课程中。

> [!NOTE]
> 因为你需要在多种屏幕尺寸下测试你的设计，所以本页面没有交互式编辑器。请下载代码在本地完成任务，或使用在线编辑器，如 [CodePen](https://codepen.io/) 或 [JSFiddle](https://jsfiddle.net/)。
>
> 如果你遇到困难，可以通过我们的[沟通渠道](/zh-CN/docs/MDN/Community/Communication_channels)联系我们。

## 任务

[下载此任务的起始文件](https://github.com/mdn/css-examples/blob/main/learn/tasks/rwd/rwd-download.html)。在浏览器中打开下载的 HTML 文件，你可以看到一个网站的线框稿，并且在移动设备上加载这一网站的时候阅读起来不存在什么问题。你可以拖动窗口使其变小，或使用 [Firefox 开发者工具中的响应式设计视图](https://firefox-source-docs.mozilla.org/devtools-user/index.html#responsive-design-mode)来查看手机上的效果。

你的任务是创建此布局的桌面版本，并且当有足够的屏幕宽度来容纳它时显示这一桌面版本。你的最终结果应该如下图所示：

![一个双列线框稿布局](rwd-task.png)

有很多方法可以创建桌面布局，尽情试验吧。你也可以添加第二个断点，实现在平板电脑竖屏模式下也能运行良好的布局。
