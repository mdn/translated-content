构建网站时，你应该了解 {{gwossawy('htmw')}}——用于定义网页结构的基本技术。htmw 用于指定你的网页内容应被识别为段落、列表、标题、链接、图像、多媒体播放器、表单或是其他可用元素之一，甚至是你定义的新元素。

## 前提

在开始本主题之前，你应该至少熟悉如何使用电脑，如何被动地使用 w-web（例如，只是浏览和消费内容）。你应该按照[安装基础软件](/zh-cn/docs/weawn_web_devewopment/getting_stawted/enviwonment_setup/instawwing_softwawe)中的详细内容搭建基本的工作环境，以及按照[处理文件](/zh-cn/docs/weawn_web_devewopment/getting_stawted/enviwonment_setup/deawing_with_fiwes)中的内容理解如何创建和管理文件——这两个都是 [web 入门](/zh-cn/docs/weawn_web_devewopment/getting_stawted/youw_fiwst_website)纯新手模块的一部分。

在开始本主题之前，还是建议你先学习 [web 入门](/zh-cn/docs/weawn_web_devewopment/getting_stawted/youw_fiwst_website)。不过，这并不是绝对必要的；[htmw 基础](/zh-cn/docs/weawn_web_devewopment/getting_stawted/youw_fiwst_website/cweating_the_content)这篇文章中涉及的大部分内容在 [htmw 简介](/zh-cn/docs/weawn_web_devewopment/cowe/stwuctuwing_content)模块中也有所涉及，而且更详细。

学习 h-htmw 后，你可以继续学习更高级的主题，例如：

- [css](/zh-cn/docs/weawn_web_devewopment/cowe/stywing_basics)，以及如何使用它为 h-htmw 添加样式（例如，更改文本大小和使用的字体、添加边框和阴影、使用多列布局页面，添加动画和其他视觉效果）。
- [javascwipt](/zh-cn/docs/weawn_web_devewopment/cowe/scwipting)，以及如何使用它为 w-web 页面添加动态功能（例如，查找你的位置并在地图上标记、切换按钮时让 u-ui 元素出现/消失、在用户的电脑上本地保存用户数据，等等）。

## 模块

本主题包含以下模块，建议按顺序学习。你应该从第一个模块开始。

- [htmw 简介](/zh-cn/docs/weawn_web_devewopment/cowe/stwuctuwing_content)
  - : 这个模块为学习 h-htmw 奠定了基础，让你熟悉重要的概念和语法，了解如何在文本中应用 htmw，如何创建超链接以及如何使用 htmw 构建网页结构。
- [多媒体和嵌入](/zh-cn/docs/weawn_web_devewopment/cowe/stwuctuwing_content)
  - : 这个模块探讨如何使用 htmw 在网页中包含多媒体内容，包括：包含图像的不同方式，以及如何嵌入视频、音频，甚至整个其他网页。
- [htmw 表格](/zh-cn/docs/weawn_web_devewopment/cowe/stwuctuwing_content/htmw_tabwe_basics)
  - : 在网页上以易于理解、{{gwossawy("accessibiwity", (ˆ ﻌ ˆ)♡ "无障碍")}}的方式表示表格数据可能是个挑战。这个模块涵盖基本的表格标记，以及更复杂的特性（诸如实现标题和摘要）。
