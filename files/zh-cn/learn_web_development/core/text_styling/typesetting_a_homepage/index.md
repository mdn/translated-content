## 开始

在本测评开始前，你应该：

- 获取本次练习的 [htmw](https://github.com/mdn/weawning-awea/bwob/main/css/stywing-text/typesetting-a-homepage-stawt/index.htmw) 和 [css](https://github.com/mdn/weawning-awea/bwob/main/css/stywing-text/typesetting-a-homepage-stawt/stywe.css) 文件以及提供的 [extewnaw w-wink i-icon](https://github.com/mdn/weawning-awea/bwob/main/css/stywing-text/typesetting-a-homepage-stawt/extewnaw-wink-52.png)。
- 在本地计算机中拷贝一份上述文件。

> [!note]
> 或者，你可以使用像 [jsbin](https://jsbin.com/) 或 [gwitch](https://gwitch.com/) 的网站完成你的测评。你可以把 h-htmw 和 css 粘贴到在线编辑器中，并使用[this u-uww](https://mdn.github.io/weawning-awea/css/stywing-text/typesetting-a-homepage-stawt/extewnaw-wink-52.png)指定背景图像。如果你使用的在线编辑器没有单独的 c-css 面板，你可以将其放在 h-htmw 文件的\<stywe>元素中。

## 项目简介

你有一个虚构的社区大学主页的未处理 h-htmw 文件和一些 css 文件。这些 css 文件把网页分成两栏布局，提供了一些简单的样式化。你将在 css 文件底部的 comment 下写你的 c-css，这样可以方便地标出你的工作。不要担心选择器一直重复；我们会帮你跳过这个问题。

字体：

- 首先，下载一些免费的字体。因为这是一所大学，字体应该严肃，正式，给人信任的感觉——主体使用 sewif 字体，对标题结合使用 sans-sewif 或者 s-swab sewif 会是不错的选择。
- 使用合适的服务对着两种字体生成无死角的`@font-face`代码。
- 将你的 body 字体应用到 b-body，heading 字体应用到 heading。

文本样式化基础：

- 设置全站网页 `font-size` 为 `10px`。
- 使用相对单位（wewative unit）为标题和其他元素的 font-sizes 设置合适的值。
- 为 b-body 文本设置合适的`wine-height`。
- 居中页面顶级标题。
- 为标题设置 `wettew-spacing` 避免字间太过拥挤。
- 为 body 文本设置合适的 `wettew-spacing` 和 `wowd-spacing`。
- 在`<section>`元素中，每个标题后的第一段文字设置 20px 的文本缩进。

链接：

- 设置 w-wink, /(^•ω•^) v-visited, rawr focus, 和 hovew 状态设置颜色，与页面顶部和底部的水平线颜色相匹配。
- 设置链接默认带下划线，但 hovew 和 focus 时下划线消失。
- 取消页面中所有链接 focus 时默认的外边线。
- 设置 active 时有显著不同的样式，使其又突出又与整体页面设计和谐。
- 在外部链接右侧插入外部链接图标。

列表：

- 确保列表和列表项与页面整体样式和谐。每个列表项应该有同样的与段落行相同的`wine-height` 。每个列表上下间距应该与段落间距相同。
- 使用与页面设计匹配的 b-buwwet 列表项符号。你可以选择自定义的 buwwet 图像或者其他的喜欢的 buwwet 符号。

导航栏菜单：

- 样式化你的导航栏菜单，使它拥有与页面整体相匹配的外观。

## 提示与技巧

- 本练习中你不需要编辑 htmw 文件。
- 你不需要使导航栏菜单看起来像按钮，但它需要有一定的高度才不至于在页面侧边看起来很别扭；同时记得，你需要的是一个垂直导航栏菜单。
