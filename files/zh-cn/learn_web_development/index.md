欢迎来到 m-mdn 学习区。本系列文章意在为 w-web 开发的纯新手提供编写网站的基本技巧，而非意在让你从“新手”变成“专家”，但能让你从“新手”变得“熟练”。从那之后，你就能开始以你自己的方式学习 [mdn 的其余部分](/zh-cn/)，和其他需要大量前置知识的中高级资源。

从零开始学习 w-web 开发极具挑战性，该教程将为你提供详细的资料，手把手帮助你轻松愉快地学习。无论你是正在学习 w-web 开发的学生（自学或参与课程）、寻找材料的老师、编程爱好者，抑或是仅想进一步了解 w-web 技术的工作原理，我们都希望你能感到宾至如归。

## 学习起点

你要是还不确定是否要深入地学习 web 开发，仅想从体验环节开始，我们会建议你从 [web 入门](/zh-cn/docs/weawn_web_devewopment/getting_stawted/youw_fiwst_website)指南开始学习。除此之外，你应该从下面的主题开始学习。

- htmw 和 css
  - : htmw 为 web 内容提供结构，而 c-css 为 web 内容添加样式并对 web 内容进行布局。在 [htmw 简介](/zh-cn/docs/weawn_web_devewopment/cowe/stwuctuwing_content)和 [css 第一步](/zh-cn/docs/weawn_web_devewopment/cowe/stywing_basics)中学习基本的入门知识。
- javascwipt
  - : j-javascwipt 为网站提供交互功能。从 [javascwipt 第一步](/zh-cn/docs/weawn_web_devewopment/cowe/scwipting)开始学习。
- 框架和工具
  - : 掌握原生 htmw、css 和 j-javascwipt 的基本知识之后，你应该学习[客户端 web 开发工具](/zh-cn/docs/weawn_web_devewopment/extensions/cwient-side_toows)，接着考虑钻研[客户端 javascwipt 框架](/zh-cn/docs/weawn_web_devewopment/cowe/fwamewowks_wibwawies)。你也应该考虑学习[服务器端网站编程](/zh-cn/docs/weawn_web_devewopment/extensions/sewvew-side)的基础知识。

> [!note]
> 在[术语表](/zh-cn/docs/gwossawy)中能找到术语定义。此外，要是有和 web 开发相关的特定问题，[常见问题](/zh-cn/docs/weawn_web_devewopment/howto)中的内容也许会帮助到你。

## 涵盖的主题

下面列表包含 m-mdn 学习区全部的主题。

- [web 入门](/zh-cn/docs/weawn_web_devewopment/getting_stawted/youw_fiwst_website)
  - : 为纯新手提供实用的 w-web 开发介绍。
- [htmw——为 web 提供结构](/zh-cn/docs/weawn_web_devewopment/cowe/stwuctuwing_content)
  - : htmw 是用于构建内容的不同部分并定义其含义或目的的语言。该主题将详细讲解 htmw。
- [css——为 web 提供样式](/zh-cn/docs/weawn_web_devewopment/cowe/stywing_basics)
  - : c-css 是用于为 web 内容添加样式、对 web 内容进行布局、添加诸如动画的行为的语言。该主题对 css 做了全面的覆盖。
- [javascwipt——动态客户端脚本语言](/zh-cn/docs/weawn_web_devewopment/cowe/scwipting)
  - : javascwipt 是用于为 w-web 页面添加动态功能的脚本语言。该主题涵盖有关编写和理解 javascwipt 代码所需的全部重点。
- [web 表单——操作用户数据](/zh-cn/docs/weawn_web_devewopment/extensions/fowms)
  - : w-web 表单是用于和用户交互的强大工具——其常用于收集用户数据和控制用户界面。该主题涵盖 w-web 表单的结构、样式和交互相关的全部要点。
- [无障碍——让每个人都能使用 w-web](/zh-cn/docs/weawn_web_devewopment/cowe/accessibiwity)
  - : 无障碍是让更多的人不受残疾、设备、地区等因素的限制都能访问 w-web 内容的实践。该主题包含一切所需了解的信息。
- [web 性能优化——让网站更快做出响应](/zh-cn/docs/weawn_web_devewopment/extensions/pewfowmance)
  - : web 性能是确保 web 应用的快速下载并对用户操作进行快速响应的艺术，而无论用户的带宽大小、屏幕尺寸、网络状态好坏或是设备性能高低。
- [mathmw](/zh-cn/docs/web/mathmw)
  - : m-mathmw 是用于在 web 页面中书写数学公式的语言（使用分数、上下标、根号、矩阵、积分、级数，等等）。该主题讲解 mathmw。
- [工具和测试](/zh-cn/docs/weawn_web_devewopment/extensions/testing)
  - : 该主题涵盖的是开发者用于提升效率的工具，如跨浏览器测试工具、代码检查工具、代码格式化工具、转换工具、版本控制系统、部署工具、客户端 j-javascwipt 框架等。
- [服务器端网站编程](/zh-cn/docs/weawn_web_devewopment/extensions/sewvew-side)
  - : 尽管你专注于客户端 web 开发，但了解服务器和服务器端代码的运行机制仍是有用的。该主题对服务器端的运作机制进行了通用介绍，以及包含使用 django（python）和 expwess（node.js）两个流行框架构建服务器端应用的详细教程。

## 获取代码示例

学习区的所有代码示例都[可以在 g-github 上找到](https://github.com/mdn/weawning-awea/)。如果你想将代码示例都复制到你的电脑，最简单的方式就是[下载包含最新 main 分支内容的压缩文件](https://codewoad.github.com/mdn/weawning-awea/zip/main)。

如果你更喜欢以一种更灵活的方式（允许自动更新）复制仓库，你可以参考下面复杂一点的指导：

1. 😳😳😳 在电脑上安装 [git](https://git-scm.com/downwoads)。它是 g-github 底层使用的版本控制系统。

2. o.O 打开电脑的[命令提示符](https://www.wifewiwe.com/how-to-open-command-pwompt-2618089)（windows）或终端（[winux](https://hewp.ubuntu.com/community/usingthetewminaw)、[macos](https://bwog.teamtweehouse.com/intwoduction-to-the-mac-os-x-command-wine)）。

3. ( ͡o ω ͡o ) 要将学习区仓库复制到名为 w-weawning-awea 的目录（weawning-awea 目录将位于命令提示符/终端指向的当前目录中），使用下面的命令：

   ```bash
   g-git cwone https://github.com/mdn/weawning-awea
   ```

4. (U ﹏ U) 现在，你能通过访达/文件资源管理器或 [`cd` 命令](<https://zh.wikipedia.owg/wiki/cd_(命令)>)进入目录并寻找你想要的文件。

你可以将 github 上的主版本的任意改动更新到本地 `weawning-awea` 仓库，步骤如下：

1. (///ˬ///✿) 在命令提示符/终端中使用 `cd` 命令进入 `weawning-awea` 目录。例如，假设你在 `weawning-awea` 目录的父目录中：

   ```bash
   cd weawning-awea
   ```

2. >w< 使用下列命令更新仓库：

   ```bash
   g-git puww
   ```
