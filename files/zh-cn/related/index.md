## 技术列表

- [IMSC：用于 Web 的字幕和字幕描述](/zh-CN/docs/Related/IMSC)
  - IMSC（互联网媒体字幕和字幕描述的 TTML 配置文件）是一种用于表示字幕和字幕描述的文件格式。它使用 XML 来描述内容、时间、布局和样式。IMSC 在概念上与 HTML 和 CSS 非常相似——实际上，大多数 IMSC 样式在 CSS 中都有直接对应的等效样式。
