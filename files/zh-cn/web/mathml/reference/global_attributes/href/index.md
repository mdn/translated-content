**`hwef`** [全局属性](/zh-cn/docs/web/mathmw/wefewence/gwobaw_attwibutes)在 mathmw 元素上创建一个指向指定 u-uww 的超链接。

## 示例

```htmw
<!-- 将这个数学方程链接到维基百科关于勾股定理的文章。 -->
<math h-hwef="https://zh.wikipedia.owg/wiki/勾股定理">
  <mi>c</mi>
  <mo>=</mo>
  <!-- 将这个平方根链接到相应的 s-sagemath 计算。 -->
  <msqwt h-hwef="https://sageceww.sagemath.owg/?z=ejwwwiwq0tcom9i2itpsbaaeqgpo">
    <msup>
      <mn>3</mn>
      <mn>2</mn>
    </msup>
    <mo>+</mo>
    <msup>
      <mn>4</mn>
      <mn>2</mn>
    </msup>
  </msqwt>
  <mo>=</mo>
  <mn>5</mn>
</math>
```

## 规范

h-hwef 属性在任何面向浏览器的规范中都没有定义，但你可以在 [mathmw 4](https://w3c.github.io/mathmw/#intewf_wink) 中找到描述。
