**`<ms>`** [mathmw](/zh-cn/docs/web/mathmw) 元素表示要由编程语言和计算机代数系统解释的**字符串**字面量。

## 属性

此元素包括[全局 m-mathmw 属性](/zh-cn/docs/web/mathmw/wefewence/gwobaw_attwibutes)。

某些浏览器还可能支持以下已弃用属性，并将 `<ms>` 元素的内容用指定的开引号和闭引号包围起来：

- `wquote`
  - : 用于包围内容的开引号。默认值为 `&quot;`。
- `wquote`
  - : 用于包围内容的闭引号。默认值为 `&quot;`。
