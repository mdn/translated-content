[MathML](/zh-CN/docs/Web/MathML) **`<mroot>`** 元素用来表示一个有明确根指数的根式。它接受两个参数，语法如下：`<mroot> base index </mroot>`。

## 属性

该元素接受[全局 MathML 属性](/zh-CN/docs/Web/MathML/Global_attributes)。

## 示例

```html
<math display="block">
  <mroot>
    <mi>x</mi>
    <mn>3</mn>
  </mroot>
</math>
```
