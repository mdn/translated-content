**`<mn>`** [MathML](/zh-CN/docs/Web/MathML) 元素表示一个**数字**字面量，通常是一个可能包含分隔符（点号或逗号）的数字序列。然而，也允许在其中包含任意文本，该文本实际上是一个数字量，例如“eleven”。

## 属性

该元素接受[全局 MathML 属性](/zh-CN/docs/Web/MathML/Global_attributes)。

## 示例

```html
<math display="block">
  <mn>0</mn>
</math>

<math display="block">
  <mn>1.337</mn>
</math>

<math display="block">
  <mn>twelve</mn>
</math>

<math display="block">
  <mn>XVI</mn>
</math>

<math display="block">
  <mn>2e10</mn>
</math>
```
