**HTML Ruby 文本 (`<rt>`) 元素**包含字符的发音，字符在 ruby 注解中出现，它用于描述东亚字符的发音。这个元素始终在 {{ HTMLElement("ruby") }} 元素中使用。

## 属性

该元素仅仅包含 [全局属性](/zh-CN/docs/HTML/Global_attributes)。

## 示例

```html
<ruby> 汉 <rt>Hàn</rt> 字 <rt>Zì</rt> </ruby>
```
