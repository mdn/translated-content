## 属性

本元素支持 [全局属性](/zh-CN/docs/Web/HTML/Global_attributes)。

## 示例

### 示例 1: 字

```html
<ruby>
  汉 <rp>(</rp><rt>han</rt><rp>)</rp> 字 <rp>(</rp><rt>zi</rt><rp>)</rp>
</ruby>
```

### 示例 2: 词

```html
<ruby> 明 日 <rp>(</rp><rt>ming ri</rt><rp>)</rp> </ruby>
```
