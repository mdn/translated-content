**`<noscript>`** [HTML](/zh-CN/docs/Web/HTML) 元素定义了在页面上的脚本类型不支持或浏览器当前关闭脚本时插入的 HTML 部分。

## 属性

这个元素只包含[全局属性](/zh-CN/docs/Web/HTML/Global_attributes)。

## 示例

```html
<noscript>
  <!-- 外部文件的锚链接 -->
  <a href="https://www.mozilla.org/">外部链接</a>
</noscript>
<p>摇滚！</p>
```

### 启用脚本后的结果

摇滚！

### 禁用脚本后的结果

[外部链接](https://www.mozilla.org/)

摇滚！
