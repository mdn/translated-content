**`<noembed>`** 元素是个废除的和不标准的方式，用于向不支持 {{HTMLElement("embed")}} ，或者不支持作者希望的 [嵌入式内容](/zh-CN/docs/Web/Guide/HTML/Content_categories#Embedded_content) 的浏览器提供替代（或者“后备”）内容。这个元素在 HTML 4.01 起废除，以支持 {{HTMLElement("object")}}。后备内容应该插在 {{HTMLElement("object")}} 的开始和结束标签之间。

> [!NOTE]
> 虽然这个元素仍旧在许多浏览器中有效，它已经废除了，并且不应使用。使用 {{HTMLElement("object")}} 来代替。
