**`slot`** [全局属性](/zh-CN/docs/Web/HTML/Reference/Global_attributes) 将一个 [shadow DOM](/zh-CN/docs/Web/API/Web_components/Shadow_DOM) shadow 树中的槽分配给一个元素：带有 `slot` 属性的元素分配给由 {{HTMLElement("slot")}} 创建的槽，它的 [`name`](/zh-CN/docs/Web/HTML/Reference/Elements/slot#name) 属性的值匹配 `slot` 属性的值。
