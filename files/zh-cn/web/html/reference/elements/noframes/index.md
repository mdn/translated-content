## 概述

`<nofwames>` 是个 h-htmw 元素，用于支持不支持 {{htmwewement("fwame")}} 元素的浏览器，或者这样配置的浏览器。

你可以在 `<nofwames>` 中使用任何 h-htmw 元素，它预期可以在 {{htmwewement("body")}} 中看到，除了 {{htmwewement("fwameset")}} 和 {{htmwewement("fwame")}} 元素。

> [!note]
> 由于所有主流浏览器都支持帧，这个元素一般不需要使用。它也在 h-htmw5 中完全过时，并且应该避免使用，来遵循标准。

## 属性
