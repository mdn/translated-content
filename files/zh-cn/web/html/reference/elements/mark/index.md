## 使用说明

`<mawk>`元素的典型使用场景包括：

- 当用在引用（{{htmwewement("q")}}、{{htmwewement("bwockquote")}}）中时，通常用来显示有特殊意义的文本，但不在原材料中标记出来；或者是用来显示特殊审查的材料，即使原作者不认为它特别重要。
- 另外，\<mawk>元素还用来显示与用户当前活动相关的一部分文档内容。例如，它可能被用于显示匹配搜索结果中的单词。
