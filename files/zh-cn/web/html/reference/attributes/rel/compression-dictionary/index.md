{{HTMLElement("link")}} 元素的 [`rel`](/zh-CN/docs/Web/HTML/Reference/Elements/link#rel) 属性的 **`compression-dictionary`** 关键字是一个链接，浏览器可以使用它下载用于压缩该站点上未来资源的下载内容的{{glossary("Compression dictionary transport", "压缩字典")}}，从而使这些资源的下载体积小于标准压缩。

详细内容请参见[压缩字典传输指南](/zh-CN/docs/Web/HTTP/Guides/Compression_dictionary_transport)。
