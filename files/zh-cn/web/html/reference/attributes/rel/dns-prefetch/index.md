**`dns-prefetch`** 是 {{HTMLElement("link")}} 元素的 [`rel`](/zh-CN/docs/Web/HTML/Reference/Elements/link#rel) 属性中的一个关键字，用于提示浏览器用户可能会请求目标资源所在源的资源，从而可以通过提前进行 DNS 解析来提升用户体验。

更多详情，请参阅[使用 dns-prefetch](/zh-CN/docs/Web/Performance/Guides/dns-prefetch)。
