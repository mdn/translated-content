{{HTMLElement("a")}}、{{HTMLElement("area")}} 和 {{HTMLElement("form")}} 元素的 [`rel`](/zh-CN/docs/Web/HTML/Reference/Attributes/rel) 属性的 **`noreferrer`** 关键字指示浏览器在导航到目标资源时省略 {{HTTPHeader("Referer")}} 标头，并且不以其他方式泄露引荐来源信息；此外，它还会表现得像同时指定了 [`noopener`](/zh-CN/docs/Web/HTML/Reference/Attributes/rel/noopener) 关键字一样。
