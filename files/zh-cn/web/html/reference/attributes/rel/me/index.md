{{HTMLElement("link")}} 和 {{HTMLElement("a")}} 元素的 [`rel`](/zh-CN/docs/Web/HTML/Reference/Elements/link#rel) 属性的 **`me`** 关键字表示当前资源由链接方代表。`me` 值是在 [XHTML Friends Network (XFN) 规范](https://gmpg.org/xfn/) 中引入的。

```html
<link rel="me" href="example.com" />
```

`rel="me"` 属性在 [RelMeAuth](https://microformats.org/wiki/RelMeAuth) 和 [Web 登录](https://microformats.org/wiki/web-sign-in)规范中被用作一种方式，使个人能够使用其域名或特定 URL 向 Web 服务标识自己。

## 浏览器兼容性

浏览器兼容性与此无关，因为 `rel="me"` 用于 Web 服务而非浏览器。
