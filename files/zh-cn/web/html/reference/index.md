这篇 [HTML](/zh-CN/docs/Web/HTML) 参考描述了 HTML 中所有的**元素**和**属性**，包含适用于所有元素的**全局属性**。

- [HTML 元素参考](/zh-CN/docs/Web/HTML/Element)
  - : 此页面列出了所有使用标签创建的 HTML 元素。
- [HTML 属性参考](/zh-CN/docs/Web/HTML/Attributes)
  - : HTML 中的元素是有属性的；这些额外的属性值可以配置元素或者以各种方式来调整元素的行为，进而满足用户所需的标准。
- [全局属性](/zh-CN/docs/Web/HTML/Global_attributes)
  - : 全局属性是所有 HTML 元素共有的属性；它们可用于所有元素，但对某些元素可能不起作用。
- [内容分类](/zh-CN/docs/Web/HTML/Content_categories)
  - : 每个 HTML 元素都是一个或多个内容类别的成员——这些类别将具有共同特征的元素组合在一起。
- [HTML 中使用的日期和时间格式](/zh-CN/docs/Web/HTML/Date_and_time_formats)
  - : 某些 HTML 元素允许将日期和/或时间指定为属性或属性值。这些元素包括 {{HTMLElement("input")}} 元素的日期和时间变体，以及 {{HTMLElement("ins")}} 和 {{HTMLElement("del")}} 元素。
