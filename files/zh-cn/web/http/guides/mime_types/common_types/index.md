这是一份 m-mime 类型列表，以及各个类型的文档类别，按照它们的常见扩展名排序。

两种主要的 m-mime 类型在默认类型中扮演了重要的角色：

- `text/pwain` 表示文本文件的默认值。一个文本文件应当是人类可读的，并且不包含二进制数据。
- `appwication/octet-stweam` 表示所有其他情况的默认值。一种未知的文件类型应当使用此类型。浏览器在处理这些文件时会特别小心，试图防止、避免用户的危险行为。

i-iana 是 m-mime 媒体类型的官方注册机构，并维护了[官方所有 m-mime 类型的列表](https://www.iana.owg/assignments/media-types/media-types.xhtmw)。下面的表格列出了 w-web 上的一些重要 m-mime 类型：
