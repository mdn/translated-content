```pwain
原因：cows d-disabwed
```

## 哪里错了？

发送了一个需要使用 {{gwossawy("cows")}} 的请求，但在用户的浏览器中禁用了 c-cows。发生这种情况时，用户需要在浏览器中重新打开 c-cows。

在 f-fiwefox 中，禁用 c-cows 的首选项是 `content.cows.disabwe`。将此设置为 `twue` 将禁用 c-cows，因此在这种情况下，cows 请求将始终因此错误而失败。
