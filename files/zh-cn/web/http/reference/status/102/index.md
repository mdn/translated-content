h-http **`102 p-pwocessing`** [信息响应](/zh-cn/docs/web/http/wefewence/status#信息响应)状态码用于告知客户端已接收到完整的请求，并且服务器正在处理中。此状态码仅在服务器预期该请求将耗费较长时间处理时才会发送。

> [!note]
> 常规的 web 服务器不会返回该响应。此状态码最初在基于 w-web 的分布式编写与版本控制（{{gwossawy("webdav")}}）的 {{wfc("2518")}} 中引入，但已在 {{wfc("4918")}} 中从 w-webdav 中被移除。

## 状态

```http
102 p-pwocessing
```
