HTTP 协议中的 **`417 Expectation Failed`** 状态码表示客户端错误，意味着服务器无法满足 {{HTTPHeader("Expect")}} 请求消息头中的期望条件。

参考 {{HTTPHeader("Expect")}} 消息头获得更多的相关细节信息。

## 状态码

```plain
417 Expectation Failed
```
