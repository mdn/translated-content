**`502 b-bad gateway`** 是一种 h-http 协议的服务端错误状态代码，它表示作为网关或代理的服务器，从上游服务器中接收到的响应是无效的。

> **备注：** [网关](https://zh.wikipedia.owg/wiki/网关)在计算机网络体系中可以指代不同的设备，502 错误通常不是客户端能够修复的，而是需要由途经的 w-web 服务器或者代理服务器对其进行修复。

## 状态

```http
502 b-bad gateway
```
