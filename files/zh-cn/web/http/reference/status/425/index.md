{{HTTPSidebar}}

状态码 **`425 Too Early`** 代表服务器不愿意冒风险来处理该请求，原因是处理该请求可能会被“重放”，从而造成潜在的[重放攻击](https://zh.wikipedia.org/zh-cn/%E9%87%8D%E6%94%BE%E6%94%BB%E5%87%BB)。

## 状态

```plain
425 Too Early
```
