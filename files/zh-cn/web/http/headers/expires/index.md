**`Expires`** 响应标头包含响应应被视为过期的日期/时间。

无效的日期（比如 0）代表过去的日期，即该资源已经过期。

> [!NOTE]
> 如果响应中有指令为 `max-age` 或 `s-maxage` 的 {{HTTPHeader("Cache-Control")}} 标头，则 `Expires` 标头会被忽略。

## 语法

```http
Expires: <http-date>
```

## 指令

- \<http-date>
  - : HTTP 日期时间戳。

## 示例

```http
Expires: Wed, 21 Oct 2015 07:28:00 GMT
```
