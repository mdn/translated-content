**`Age`** 标头包含对象在代理缓存中停留的时间，以秒为单位。

`Age` 标头通常接近于 0。如果显示为 `Age: 0`，则表示该内容可能是从源服务器上获取的；否则，它通常是通过代理服务器当前日期与 HTTP 响应中包含的 {{HTTPHeader("Date")}} 通用标头之间的差值来计算得出的。

## 语法

```http
Age: <delta-seconds>
```

## 指令

- \<delta-seconds>
  - : 一个非负整数，表示对象在代理缓存中停留的时间，以秒为单位。

## 示例

```http
Age: 24
```
