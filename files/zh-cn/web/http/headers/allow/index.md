**`Allow`** 标头列出了资源支持的方法集。

当服务器响应带有 {{HTTPStatus("405")}} `Method Not Allowed` 状态码时必须发送此标头，以表示可以使用哪些请求方法。`Allow` 空标头表示该资源不允许使用任何请求方法，例如，这种情况可能针对某个特定资源会临时出现。

## 语法

```http
Allow: <http-methods>
```

## 指令

- \<http-methods>
  - : 以逗号分隔的允许 [HTTP 请求方法](/zh-CN/docs/Web/HTTP/Methods)列表。

## 示例

```http
Allow: GET, POST, HEAD
```
