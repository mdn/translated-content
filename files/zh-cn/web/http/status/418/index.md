HTTP **`418 I'm a teapot`** 客户端错误响应状态码表示服务器拒绝冲泡咖啡，因为它一直都是茶壶。暂时没有咖啡的组合式咖啡/茶壶应该返回 503。这个错误是对 1998 年和 2014 年愚人节玩笑中定义的超文本咖啡壶控制协议的参考。

一些网站使用这个响应来处理它们不想处理的请求，比如自动查询。

## 状态

```http
418 I'm a teapot
```
