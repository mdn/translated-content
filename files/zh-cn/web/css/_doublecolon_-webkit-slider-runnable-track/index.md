**`::-webkit-slider-runnable-track`** CSS [伪元素](/zh-CN/docs/Web/CSS/Pseudo-elements)表示 {{HTMLElement("input/range", '&lt;input type="range"&gt;')}} 中的“轨道”（即，滑块可在其中滑动的凹槽）。

## 语法

```css
::-webkit-slider-runnable-track {
  /* ... */
}
```

## 规范

不属于任何规范。
