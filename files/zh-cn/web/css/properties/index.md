[CSS](/zh-CN/docs/Web/CSS) **_属性_** 是在 [CSS 声明](/zh-CN/docs/Web/CSS/CSS_syntax/Syntax#css_声明)中使用的参数，用于设置选中元素的某些样式特征。例如，{{CSSxRef("opacity")}} 属性用于设置选中元素的不透明度，从而控制该元素后方的内容是否可见：

```css
/* 将 <img> 元素的透明度设置为 0.8 */
img {
  opacity: 0.8;
}
```

每个属性都包含一个名称（例如 `opacity`）、一个值（例如 `0.8`），以及在文档渲染中的特定行为。CSS 还定义了简写属性，允许你在一条声明中同时设置多个相关属性。例如，{{CSSxRef("margin")}} 属性是 {{CSSxRef("margin-top")}}、{{CSSxRef("margin-right")}}、{{CSSxRef("margin-bottom")}} 和 {{CSSxRef("margin-left")}} 的简写形式，用于同时设置元素四个方向的外边距：

```css
/* 为 <img> 元素设置 1rem 的边距 */
img {
  margin: 1rem;
}
```

## 语法

```css
selector {
  property: value;
}
```
