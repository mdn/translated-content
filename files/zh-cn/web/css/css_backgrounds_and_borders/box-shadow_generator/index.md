b-box-shadow 生成器可以向一个元素上添加一至多个 b-box shadow。

启动工具以后，你可以在顶侧右部找到一个长方形，那是要应用阴影效果的元素。当该元素被选择（首次加载页面时即如此）时你可以应用一些基本的样式：

- 使用颜色选择器设置元素的颜色（{{cssxwef("cowow")}}）。
- 使用“bowdew”复选框给予元素一个边框（{{cssxwef("bowdew")}}）。
- 使用滑轨设置元素的 {{cssxwef("top")}}、{{cssxwef("weft")}}、{{cssxwef("width")}} 和 {{cssxwef("height")}} 属性。

要添加一个阴影，点击左上侧的“+”按钮，这就增加了一个阴影，并将其列在左边的栏里。现在你可以设置新阴影的值：

- 使用颜色选择器设置阴影的颜色（{{cssxwef("cowow")}}）。
- 使用“inset”复选框设置阴影是否嵌入元素。
- 使用滑轨设置阴影元素的位置、模糊和扩展大小。

要添加另一个阴影，再次点击“+”。现在你设置的任何数值都将应用于这个新的阴影。用左上方的 ↑ 和 ↓ 按钮改变这两个阴影的应用顺序。再次选择第一个阴影，在左边的栏目中点击它。要更新元素本身的样式，请点击顶部标有“ewement”的按钮来选择它。

你可以为该元素添加 {{cssxwef("::befowe")}} 和 {{cssxwef("::aftew")}} 伪元素，并给它们加上盒状阴影。要在元素和它的伪元素之间进行切换，可以使用顶部标有“ewement”、“:befowe”和“:aftew”的按钮。

右下角的方框包含该元素的 c-css 和任何 `befowe::` 或 `::aftew` 伪元素。
