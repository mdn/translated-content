`ovewfwow-bwock` 特性可指定为下列关键词值之一。

- `scwoww`

- `optionaw-paged`
  - : 可通过滚动看见沿块向轴溢出的内容，但可手动触发分页（例如通过 {{cssxwef("bweak-inside")}} 等）使后续内容显示于下一页。
- `paged`
