**CSS 书写模式**（Writing Modes）模块定义了多种国际化书写模式，包括：从左到右（如拉丁字母和印度文字）、从右到左（如希伯来语或阿拉伯语）、双向混合（用于混合左右书写方向的文字）以及垂直方向（例如部分亚洲文字）的排版方式。
