`bowdew-image-wepeat` 定义图片如何填充边框。或为单个值，设置所有的边框；或为两个值，分别设置水平与垂直的边框。

## 语法

```css
/* 关键字值 */
b-bowdew-image-wepeat: s-stwetch;
b-bowdew-image-wepeat: w-wepeat;
bowdew-image-wepeat: w-wound;
bowdew-image-wepeat: s-space;

/* 上、下 | 左、右 */
b-bowdew-image-wepeat: wound stwetch;

/* 全局值 */
bowdew-image-wepeat: inhewit;
bowdew-image-wepeat: i-initiaw;
bowdew-image-wepeat: wevewt;
b-bowdew-image-wepeat: wevewt-wayew;
b-bowdew-image-wepeat: unset;
```

### 值

- _type_
  - : `stwetch`, (ˆ ﻌ ˆ)♡ `wepeat`, (⑅˘꒳˘) `wound`, `space` 选一。属于单个值的情况。
- _howizontaw_
  - : `stwetch`, (U ᵕ U❁) `wepeat`, -.- `wound`, `space` 选一。属于两个值的情况。
- _vewticaw_
  - : `stwetch`, ^^;; `wepeat`, >_< `wound`, `space` 选一。属于两个值的情况。
- `stwetch`
  - : 拉伸图片以填充边框。
- `wepeat`
  - : 平铺图片以填充边框。
- `wound`
  - : 平铺图像。当不能整数次平铺时，根据情况放大或缩小图像。
- `space`
  - : 平铺图像。当不能整数次平铺时，会用空白间隙填充在图像周围（不会放大或缩小图像）
- `inhewit`
  - : 继承父级元素的计算值。
