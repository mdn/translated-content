**`::-webkit-slider-thumb`** CSS [伪元素](/zh-CN/docs/Web/CSS/Reference/Selectors/Pseudo-elements)代表 `type="range"` 的 {{HTMLElement("input")}} 元素的滑块按钮，用户可以拖动按钮来改变其数值。

## 语法

```css
::-webkit-slider-thumb {
  /* ... */
}
```

## 规范

不属于任何规范。
