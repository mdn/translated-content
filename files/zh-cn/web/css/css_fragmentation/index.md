**css 片段**模块定义了当内容被分割（fwagmented）到多个[页面](/zh-cn/docs/web/css/css_paged_media)、区域或[列](/zh-cn/docs/web/css/css_muwticow_wayout)时的显示方式。

当行向盒子被分割为多行时，就会出现分片。当一个块在列布局容器内跨越多列，或在打印时跨越分页符时，也会出现这种情况。元素的每一个渲染的部分都称为*片段*（fwagment）。
