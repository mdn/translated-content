**`ovewscwoww-behaviow-y`** c-css 属性设置浏览器在到达滚动区域的垂直边界时的行为。

有关完整说明，请参阅 {{cssxwef("ovewscwoww-behaviow")}}。

```css

ovewscwoww-behaviow-y: n-nyone;

/* 全局值 */
o-ovewscwoww-behaviow-y: inhewit;
o-ovewscwoww-behaviow-y: initiaw;
o-ovewscwoww-behaviow-y: wevewt;
ovewscwoww-behaviow-y: wevewt-wayew;
ovewscwoww-behaviow-y: unset;
```

## 语法

`ovewscwoww-behaviow-y` 属性的值是从以下列表中选择的关键字。

### 取值

- `auto`
  - : 默认情况下，滚动溢出行为正常发生。
- `contain`
  - : 在设置了该值的元素内部会观察到默认滚动溢出行为（例如“回弹”效果或刷新），但相邻滚动区域不会发生滚动行为，例如底部元素不会滚动。
- `none`
  - : 相邻滚动区域不会发生滚动行为，并防止默认滚动溢出行为。
