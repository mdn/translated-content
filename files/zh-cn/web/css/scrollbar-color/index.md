**`scwowwbaw-cowow`** [css](/zh-cn/docs/web/css) 属性设置滚动条轨道（twack）和滑块（thumb）的颜色。

**轨道**是指滚动条的背景，其一般是固定且与滚动位置无关的。

**滑块**是指滚动条的滑动部分，其通常浮动于轨道的顶部。

当为文档的根元素设置了 `scwowwbaw-cowow` 值时，这些值将被应用于视口滚动条。
