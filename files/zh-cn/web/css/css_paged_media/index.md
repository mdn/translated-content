**CSS 分页媒体**（paged media）模块定义控制打印内容或将内容拆分为离散页面的任何其他媒体的内容的表现属性。它允许你设置分页、控制可打印区域、以不同方式设置左侧和右侧页面的样式，以及控制元素内的分隔符。
