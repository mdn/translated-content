## 简介

**`-webkit-tap-highlight-color`** 是一个没有标准化的属性，能够设置点击链接的时候出现的高亮颜色。显示给用户的高光是他们成功点击的标识，以及暗示了他们点击的元素。

{{cssinfo}}

## Syntax

```css
-webkit-tap-highlight-color: red;
```

### 值

A {{Cssxref("color value")}}.
