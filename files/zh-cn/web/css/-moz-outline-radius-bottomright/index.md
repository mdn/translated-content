{{Non-standard_header}}{{CSSRef}}

在 Mozilla 应用中，**`-moz-outline-radius-bottomright`** [CSS](/zh-CN/docs/Web/CSS) 属性用于设置元素的左下角{{cssxref("outline")}} 圆角。

{{cssinfo}}

## 示例

### HTML

```html
<p>看这段话的右下角。</p>
```

### CSS

```css
p {
  margin: 5px;
  border: solid cyan;
  outline: dotted red;
  -moz-outline-radius-bottomright: 2em;
}
```
