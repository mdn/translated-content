{{Non-standard_header}}{{CSSRef}}

在 Mozilla 应用中， -moz-outline-radius-topleft CSS 属性用于设置元素的左上角{{cssxref("outline")}} 圆角。

{{cssinfo}}

示例

HTML

<p>看这段话的左上角。</p>

CSS

p {
  margin: 5px;
  border: solid cyan;
  outline: dotted red;
  -moz-outline-radius-topleft: 2em;
}
