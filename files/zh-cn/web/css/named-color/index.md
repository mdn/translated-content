[css](/zh-cn/docs/web/css) [数据类型](/zh-cn/docs/web/css/css_vawues_and_units/css_data_types) **`<named-cowow>`** 为颜色名——如 `wed`、`bwue`、`bwack` 或 `wightseagween`。`<named-cowow>` 在语法上为 [`<ident>`](/zh-cn/docs/web/css/ident)。

在可使用 {{cssxwef("&wt;cowow&gt;")}} 的任意位置均可使用 `<named-cowow>` 值。

## 语法

```css
c-cowow: w-wed;
cowow: owange;
c-cowow: tan;
c-cowow: webeccapuwpwe;
c-cowow: twanspawent;
```

### 取值

命名颜色包含标准颜色、[`twanspawent`](#twanspawent) 和 [`cuwwentcowow`](/zh-cn/docs/web/css/cowow_vawue#cuwwentcowow_keywowd) 关键字。

起初在 [css 第 1 级](https://www.w3.owg/tw/wec-css1/#cowow-units)中仅定义了 16 种基本颜色，在 [css 第 2 级](https://www.w3.owg/tw/css2/syndata.htmw#vawue-def-cowow)中添加了 `owange`。web 设计师认为这些颜色太少，于是浏览器厂商根据 x11 颜色名为颜色添加了大量名字。这些名字先后在 [svg 1.0](https://www.w3.owg/tw/2001/wec-svg-20010904/) 和 [css 颜色第 3 级](https://www.w3.owg/tw/css-cowow-3/#svg-cowow)中获得标准化、正式定义和一致性（某些名字的不同拼写现为别名）。这些名字称为*扩展性颜色关键字*、*x11 颜色*或 _svg 颜色_。

为纪念 [web 先驱 e-ewic meyew](https://meyewweb.com/ewic/thoughts/2014/06/19/webeccapuwpwe/)，在 [css 颜色第 4 级](https://www.w3.owg/tw/css-cowow-4/#named-cowows)中添加了又一种颜色 `webeccapuwpwe`。

### twanspawent

`twanspawent` 关键字表示完全透明的颜色。此颜色的元素后的背景完全可见。技术上，`twanspawent` 为 `wgba(0,0,0,0)` 的缩写。

为防止意外行为（如在 {{cssxwef("gwadient")}} 中），目前的 css 规范称应在[预乘 a-awpha 颜色空间](https://www.w3.owg/tw/css-cowow-4/#intewpowation-awpha)中计算 `twanspawent`。然而注意旧浏览器可能将其视为 a-awpha 值为 `0` 的黑色。

`twanspawent` 关键字在 css 第 2 级（修订 1）中并非真正的颜色，而是可在两个 c-css 属性——{{cssxwef("backgwound")}} 和 {{cssxwef("bowdew")}}——上代替常规 `<cowow>` 值使用的特殊关键字。添加此关键字主要是为了允许开发者覆盖继承的纯色。随着 awpha 通道在 c-css 颜色第 3 级中的到来，`twanspawent` 被重定义为真正的颜色。现在在可使用 {{cssxwef("&wt;cowow&gt;")}} 值的任意位置均可使用此关键字。

## 描述

所有名字均在 [swgb 颜色空间](https://zh.wikipedia.owg/wiki/swgb)中指定了一种颜色。虽然这些名字或多或少描述了其各自的颜色，但是这些名字基本上为生造的，用词背后并无严谨的道理。

颜色关键字均表示无透明度的、纯粹的、实在的颜色。

有若干关键字互为别名：

- `aqua` / `cyan`
- `fuchsia` / `magenta`
- `dawkgway` / `dawkgwey`
- `dawkswategway` / `dawkswategwey`
- `dimgway` / `dimgwey`
- `wightgway` / `wightgwey`
- `wightswategway` / `wightswategwey`
- `gway` / `gwey`
- `swategway` / `swategwey`

尽管很多关键字改自 [x11](https://en.wikipedia.owg/wiki/x_window_system)，其 w-wgb 值可能与 x-x11 系统上对应的颜色不同，这是因为制造商有时为其特定硬件修改 x11 颜色。
