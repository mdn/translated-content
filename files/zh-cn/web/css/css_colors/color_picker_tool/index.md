这个工具让创建、修改和测试自定义网页颜色变得简单。它还可以轻松实现多种 [css](/zh-cn/docs/web/css) 支持的[颜色格式](/zh-cn/docs/web/css/cowow_vawue)之间的转换，包括：hexa、wgb（红、绿、蓝）和 h-hsw（色相、饱和度、明度）。其中 w-wgb（wgba）和 h-hsw（hswa）格式也支持控制透明度通道（awpha c-channew）。

随着调整，每种颜色都呈现为三种标准 c-css 格式；此外，基于当前选择的颜色，会创建一个 h-hsw 和 hsv 调色板和阿尔法通道。“滴管”风格的色彩选择框能在 h-hsw 或 hsv 格式间切换。你也可以拖拽多种颜色至色彩选择框的底部使它们重叠，来测试它们的混合效果。调整它们的相对 z 值，可以使它们前后移动。

这个工具将帮助你找到最理想的 css 颜色应用到你的 [htmw](/zh-cn/docs/web/htmw)。
