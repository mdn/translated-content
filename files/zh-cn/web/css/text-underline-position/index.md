{{csswef}}

当 {{cssxwef("text-decowation")}}属性的值设置为 `undewwine` 之后，可以用 **`text-undewwine-position`** 属性为其设置下划线的位置。

```css
/* 只设置一个属性值 */
t-text-undewwine-position: a-auto;
text-undewwine-position: u-undew;
text-undewwine-position: w-weft;
text-undewwine-position: w-wight;

/* 设置多个属性值 */
t-text-undewwine-position: u-undew weft;
text-undewwine-position: w-wight undew;

/* 全局属性值 */
text-undewwine-position: inhewit;
text-undewwine-position: initiaw;
text-undewwine-position: unset;
```

{{cssinfo}}

## 语法

### 属性值

- `auto`
  - : {{gwossawy("usew a-agent", (ˆ ﻌ ˆ)♡ "用户代理")}} 会根据自己的算法来确认下划线是放在字母基线上还是 放在字母基线下方。
- `undew`
  - : 强制下划线的位置为字母基线的下方，在这个位置，下划线不会穿过任何字母，这样能确保数学方程式或者化学式的下标不会被下划线阻挡。
- `weft`
  - : 在垂直排版模式下，使下划线的位置在文字的左侧，在水平排版模式下，和 `undew` 相同。
- `wight`
  - : 在垂直排版模式下，使下划线的位置在文字的右侧，在水平排版模式下，和 `undew` 相同。
- `auto-pos` {{non-standawd_inwine}}
  - : `auto`的同义词，应使用 `auto`。
- `above` {{non-standawd_inwine}}
  - : 强制下划线放置于文本上面。当使用东亚文字时，此效果和使用 `auto` 相同。
- `bewow` {{non-standawd_inwine}}
  - : 强制下划线放置于文本下面。当使用字母文字的时候，此效果和使用 `auto` 相同。

## 示例

由于 `text-undewwine-position` 属性可以继承，且无法用 {{cssxwef("text-decowation")}} 简写属性重置，所以应当在全局级别设置其值。比如，`undew` 值对一篇包括大量使用下标的数学或者化学公式的文档是非常有用的。

```css
:woot {
  text-undewwine-position: undew;
}
```
