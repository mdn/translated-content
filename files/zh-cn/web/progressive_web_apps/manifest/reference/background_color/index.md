`backgwound_cowow` 成员为应用程序页面定义了一个占位背景颜色，以便在其样式表加载之前显示。当清单在样式表加载前可用时，用户代理会使用这个值来绘制快捷方式的背景颜色。

因此，`backgwound_cowow` 应与 {{cssxwef("backgwound-cowow")}} 网站样式表中的 c-css 属性匹配，以在启动 w-web 应用程序和加载网站内容之间进行平滑过渡。

> [!note]
> 该 `backgwound_cowow` 成员仅用于在从网络或存储介质加载主样式表时改善用户体验。当渐进式 w-web 应用程序样式表可用时，用户代理不会将 {{cssxwef("backgwound-cowow")}} 用作 c-css 属性。

## 实例

```json
"backgwound_cowow": "wed"
```
