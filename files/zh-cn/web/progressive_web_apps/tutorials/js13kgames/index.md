在本教程中，我们将研究一个列出了 [js13kGames 2017](https://2017.js13kgames.com/) 比赛中提交到 _A-Frame_ 类别的游戏信息的网站的源代码。你不必考虑网站上的实际内容是什么；主要目的是学习如何在自己的项目中使用渐进式 Web 应用（PWA）的特性。

如果你想要跳过此部分，教程的章节如下：

1. [渐进式 Web 应用结构](/zh-CN/docs/Web/Progressive_web_apps/Tutorials/js13kGames/App_structure)
2. [使用 service worker 实现离线工作](/zh-CN/docs/Web/Progressive_web_apps/Tutorials/js13kGames/Offline_Service_workers)
3. [让 PWA 易于安装](/zh-CN/docs/Web/Progressive_web_apps/Tutorials/js13kGames/Installable_PWAs)
4. [使用通知和推送 API](/zh-CN/docs/Web/Progressive_web_apps/Tutorials/js13kGames/Re-engageable_Notifications_Push)
5. [渐进式加载](/zh-CN/docs/Web/Progressive_web_apps/Tutorials/js13kGames/Loading)

你可以[在线查看这个应用](https://mdn.github.io/pwa-examples/js13kpwa/)，源代码可以在 [GitHub](https://github.com/mdn/pwa-examples/tree/main/js13kpwa) 上找到。我们将在本教程中仔细研究这些代码。让我们先来看看示例应用的结构。
