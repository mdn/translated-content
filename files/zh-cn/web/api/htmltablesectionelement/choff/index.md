{{APIRef("HTML DOM")}}{{deprecated_header}}

{{domxref("HTMLTableSectionElement")}} 接口的 **`chOff`** 属性什么都不做，它反映 section 元素的 `charoff` 属性。

> [!NOTE]
> 此属性旨在支持将表格单元格内容对齐到特定字符（通常是小数点），但浏览器从未实现过。
>
> 要实现这种对齐，请注意 {{cssxref("text-align")}} CSS 属性对字符串值的支持。

## 值

一个整数。
