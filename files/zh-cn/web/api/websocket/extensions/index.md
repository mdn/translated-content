{{APIRef("Web Sockets API")}}

**`WebSocket.extensions`** 是只读属性，返回服务器已选择的扩展值。目前，链接可以协定的扩展值只有空字符串或者一个扩展列表。

## 语法

```plain
var extensions = aWebSocket.extensions;
```

## 返回值

一个 {{jsxref("String")}}.
