{{domxref("CSSTransformValue")}} 接口的只读属性 **`is2D`** 返回变换是 2D 还是 3D。

当且仅当任何单个函数为 `Is2D` 返回 false 时返回 false，否则 `CSSTransformValue` 的这个属性将返回 true。

## 值

布尔值。当变换是 2D 变换时为 true，否则为 false。
