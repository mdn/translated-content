**`CSSMathProduct()`** 构造函数创建一个新的会将传入的参数相乘的 {{domxref("CSSMathProduct")}} 对象。

## 语法

```js-nolint
new CSSMathProduct(args)
```

### 参数

- `args`
  - : 用于 {{domxref('CSSMathProduct')}} 对象的值（可以是双精度整数或 {{domxref('CSSNumericValue')}}）列表。
