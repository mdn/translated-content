[CSS 类型对象模型 API](/zh-CN/docs/Web/API/CSS_Object_Model) 的 **`CSSMathProduct`** 接口表示在 {{domxref('CSSNumericValue')}} 值上调用 {{domxref('CSSNumericValue.add','add()')}}、{{domxref('CSSNumericValue.sub','sub()')}}、或 {{domxref('CSSNumericValue.toSum','toSum()')}} 得到的结果。它继承了其父接口 {{domxref('CSSNumericValue')}} 的属性和方法。

{{InheritanceDiagram}}

## 构造函数

- {{domxref("CSSMathProduct.CSSMathProduct", "CSSMathProduct()")}} {{Experimental_Inline}}
  - : 创建一个新的 `CSSMathProduct` 对象。

## 实例属性

- {{domxref('CSSMathProduct.values')}}
  - : 返回包含一个或多个 {{domxref('CSSNumericValue')}} 对象的 {{domxref('CSSNumericArray')}} 对象。
