{{domxref("Credential")}} 接口的只读属性 **`id`** 返回一个包含凭据标识符的字符串。这可能是 GUID、用户名、电子邮件地址或其他值，具体取决于凭证的类型。

## 值

包含凭据标识符的字符串。
