{{domxref("NDEFReader")}} 接口的 **`NDEFReader()`** 构造函数返回一个新的 `NDEFReader` 对象，用于从读取器磁感应场内的兼容 NFC 设备（例如 NDEF 标签）读取 NDEF 消息。

## 语法

```js-nolint
new NDEFReader()
```

### 参数

无。

### 返回值

一个新的 {{DOMxRef("NDEFReader")}}。
