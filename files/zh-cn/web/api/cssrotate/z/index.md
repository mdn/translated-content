{{APIRef("CSS Typed OM")}}

{{domxref("CSSRotate")}} 接口的 **`z`** 属性表示平移向量的 z 分量。正值会使元素向观察者移动，负值则使元素远离观察者。

## 值

{{domxref("CSSNumericValue")}}。可以设置为数字（该数字会被包装在 `unit: "number"` 的 {{domxref("CSSUnitValue")}} 中）。
