{{APIRef("CSS Typed Object Model API")}}

[CSS 类型对象模型 API](/zh-CN/docs/Web/API/CSS_Object_Model) 的 **`CSSRotate`** 接口表示 CSS 中 {{CSSXRef('transform')}} 属性的旋转值。它继承父接口 {{domxref('CSSTransformValue')}} 的属性和方法。

{{InheritanceDiagram}}

## 构造函数

- {{domxref("CSSRotate.CSSRotate", "CSSRotate()")}}
  - : 创建一个新的 `CSSRotate` 对象。

## 实例属性

- {{domxref('CSSRotate.x','x')}}
  - : 返回或设置 x 轴的值。
- {{domxref('CSSRotate.y','y')}}
  - : 返回或设置 y 轴的值。
- {{domxref('CSSRotate.z','z')}}
  - : 返回或设置 z 轴的值。
- {{domxref('CSSRotate.angle','angle')}}
  - : 返回或设置角度值。
