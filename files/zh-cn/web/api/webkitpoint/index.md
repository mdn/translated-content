**`Point`** 是一个表示二维空间中点的接口。它属于非标准接口，兼容性不佳，不应使用。

> [!NOTE]
> 尽管与此已废弃的接口无直接关联，但你可能想找 {{domxref("DOMPoint")}}。

## 实例属性

- `x` {{Deprecated_Inline}} {{Non-standard_Inline}}
  - : 一个浮点数值，用于指定该点相对于 X（水平）轴的位置。
- `y` {{Deprecated_Inline}} {{Non-standard_Inline}}
  - : 一个浮点数值，用于指定该点相对于 Y（垂直）轴的位置。

## 规范

[2009 年 3 月 20 日 CSS 2D Transforms Module 第 3 版的废弃工作草案](https://www.w3.org/TR/2009/WD-css3-2d-transforms-20090320/)中定义了该类，但该类并未在任何现行规范中存在。
