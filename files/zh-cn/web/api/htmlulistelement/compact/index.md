{{domxref("HTMLUListElement")}} 接口的 **`compact`** 属性表明列表项之间的间距应减小。`compact` 属性的具体处理方式因浏览器而异。建议改用 CSS {{cssxref("line-height")}}，而非使用该属性。

该属性反映了 {{HTMLElement("ul")}} 元素的 [`compact`](/zh-CN/docs/Web/HTML/Reference/Elements/ul#compact) 属性。

## 值

布尔值。
