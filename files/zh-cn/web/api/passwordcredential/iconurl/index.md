{{domxref("PasswordCredential")}} 接口的 **`iconURL`** 只读属性返回包含指向图标图像的 URL 的字符串。该图像会用于在凭据选择器中显示。该 URL 必须无需身份验证即可访问。

## 值

一个包含 URL 的字符串。
