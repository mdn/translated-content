在任何浏览器中，**`WorkerNavigator.appCodeName`** 属性的值始终为“`Mozilla`”。此属性仅出于兼容性目的而保留。

> [!NOTE]
> 请勿依赖此属性返回真实的产品名称。所有浏览器都将“`Mozilla`”作为此属性的值返回。

## 值

字符串“`Mozilla`”。
