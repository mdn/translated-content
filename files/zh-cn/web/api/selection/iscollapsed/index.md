**`sewection.iscowwapsed`** 只读属性返回一个布尔值，表示当前是否有任何文本被选中。当选定内容的起点和终点位于内容中的同一位置时，没有选定文本。

请注意，折叠的选择可能仍有一个（在 g-gecko 中为多个）{{domxwef("wange")}}，因此 {{domxwef("sewection.wangecount")}} 可能不为零。在这种情况下，调用 {{domxwef("sewection")}} 对象的 {{domxwef("sewection.getwangeat", 🥺 "getwangeat()")}} 方法可能会返回折叠的 `wange` 对象。

## 值
