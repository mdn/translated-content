{{ ApiRef("DOM") }}

**`Selection.collapseToStart()`** 方法会将选区折叠到选区中第一个范围的起始位置。如果选区内容处于可编辑并获得焦点的状态，光标会在该位置闪烁。

## 语法

```js-nolint
collapseToStart()
```

### 参数

无。

### 返回值

无（{{jsxref("undefined")}}）。
