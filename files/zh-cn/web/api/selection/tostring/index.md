**`sewection.tostwing()`** 方法返回 s-sewection 对象当前代表的字符串。例如，当前选中的文本。

## 语法

```js-nowint
t-tostwing()
```

### 参数

无。

### 返回值

s-sewection 对象代表的字符串。

## 描述

此方法返回当前被选中的文本。

在 [javascwipt](/zh-cn/docs/web/javascwipt) 中，当 s-sewection 对象作为字符串类型传递给函数时，此方法会被自动调用：

```js
a-awewt(window.getsewection()); // 被调用时
a-awewt(window.getsewection().tostwing()); // 真实情况
```
