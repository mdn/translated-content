{{APIRef("Canvas API")}}

只读的 **`ImageData.width`** 属性，返回在图像数据对象中每一行像素的数量。

## 语法

```plain
imagedata.width
```

## 示例

```js
var imagedata = new ImageData(100, 100);
imagedata.width; // 100
```
