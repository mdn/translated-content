{{APIRef("CSS Typed Object Model API")}}

[CSS 类型对象模型 API](/zh-CN/docs/Web/API/CSS_Object_Model) 的 **`CSSTranslate`** 接口表示 CSS 中 {{CSSXRef('transform')}} 属性的单个 [translate()](/zh-CN/docs/Web/CSS/Reference/Values/transform-function/translate) 值。它从 {{domxref('CSSTransformValue')}} 父接口继承属性和方法。

{{InheritanceDiagram}}

## 构造函数

- {{domxref("CSSTranslate.CSSTranslate", "CSSTranslate()")}}
  - : 创建一个新的 `CSSTranslate` 对象。

## 实例属性

- {{domxref('CSSTranslate.x','x')}}
  - : 返回或设置 x 轴值。
- {{domxref('CSSTranslate.y','y')}}
  - : 返回或设置 y 轴值。
- {{domxref('CSSTranslate.z','z')}}
  - : 返回或设置 z 轴值。
