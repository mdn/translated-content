[Web MIDI API](/zh-CN/docs/Web/API/Web_MIDI_API) 的 **`MIDIOutputMap`** 只读接口提供了一组当前可用的 MIDI 输出端口。

`MIDIOutputMap` 实例是一个只读的[类 `Map` 对象](/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Map#类_Map_浏览器_API)，其中每个键都是 MIDI 输出的 ID 字符串，与之关联的值是相应的 {{domxref("MIDIOutput")}} 对象。
