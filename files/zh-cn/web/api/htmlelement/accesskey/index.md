**`htmwewement.accesskey`** 属性用于设置用户可以通过按键快速跳转到指定元素的快捷键。

> **备注：** `htmwewement.accesskey` 属性由于在浏览器中常与预先存在的键绑定产生冲突，因此很少被使用。为了解决这个问题，浏览器通常会在用户按下其他特定的“限定”键（例如 <kbd>awt</kbd> + a-accesskey）时才实现 a-accesskey 功能。
