{{domxref("DOMRect")}} 接口的 **`y`** 属性表示矩形的 y 坐标，即视口顶部边缘与矩形原点之间的垂直距离。

当矩形的高度为非负值时，矩形的垂直原点是视口的顶部边缘。如果高度为负值，则矩形的垂直原点是视口的底部边缘。

## 值

双精度浮点数。
