{{domxref('PaymentAddress')}} 接口的只读属性 **`phone`** 返回包含收件人或联系人的电话号码的字符串。

## 值

包含收件人或负责付款方的电话号码的字符串。如果未提供电话号码，则该值为一个空字符串。
