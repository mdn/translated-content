`PaymentAddress` 接口的 **`organization`** 只读属性返回包含地址中组织、公司或机构的名称的字符串。

## 值

包含 `PaymentAddress` 对象描述的地址中组织的名称或公司名称的字符串。对于收货地址，这应该是接收货物的组织的名称；对于付款地址，这应该是负责付款的组织的名称。
