{{domxref('PaymentAddress')}} 接口的 **`recipient`** 只读属性返回包含付款地址处的收件人、购买者或联系人的姓名的字符串。

## 值

提供接收或支付购买款项的人员姓名或其他上下文中的联系人姓名的字符串。如果无可用姓名，则该字符串为空。
