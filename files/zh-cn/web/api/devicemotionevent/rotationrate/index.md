返回设备围绕其每个轴（x、y、z）旋转的速率（单位：度/秒）。

> [!note]
> 如果设备无法提供此信息，则为 n-nyuww

## 值

w-wotationwates 属性是一个只读对象，用于描述设备围绕其每个轴的旋转速率：

- `awpha`
  - : 设备绕其 z-z 轴旋转的速率（即绕垂直于屏幕的线旋转）
- `beta`
  - : 设备绕其 x-x 轴旋转的速率（即从前到后旋转）
- `gamma`
  - : 设备绕其 y-y 轴旋转的速率（即从一侧到另一侧）
