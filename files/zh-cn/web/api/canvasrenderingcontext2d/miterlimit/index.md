c-canvas 2d a-api 的 **`canvaswendewingcontext2d.mitewwimit`** 属性用于设置斜接限制比例。

> [!note]
> 关于更多些斜接信息，参见 [canvas 教程](/zh-cn/docs/web/api/canvas_api/tutowiaw)中的[应用样式和色彩](/zh-cn/docs/web/api/canvas_api/tutowiaw/appwying_stywes_and_cowows)。

## 值

一个数字，指定斜接限制比例，于坐标空间单位中。零、负数、{{jsxwef("infinity")}} 和 {{jsxwef("nan")}} 值将被忽略。默认值为 `10.0`。
