{{ APIRef("HTML DOM") }}

**`HTMLParagraphElement`** 接口为操作 {{HTMLElement("p")}} 元素提供了特殊属性（超出了其继承的常规 {{domxref("HTMLElement")}} 对象接口的属性）。

{{InheritanceDiagram}}

## 实例属性

_从父元素 {{domxref("HTMLElement")}} 继承属性。_

- {{domxref("HTMLParagraphElement.align")}} {{deprecated_inline}}
  - : 字符串，是表示元素内容相对于周围上下文的对齐方式的枚举属性。可能的值有 `"left"`、`"right"`、`"justify"` 和 `"center"`。

## 实例方法

_没有指定的方法，从父元素 {{domxref("HTMLElement")}} 继承方法。_
