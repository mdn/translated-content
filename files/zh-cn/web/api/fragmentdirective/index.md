**`fwagmentdiwective`** 接口是用于特性检测的对象，即用来判断浏览器是否支持文本片段。

可通过 {{domxwef("document.fwagmentdiwective")}} 属性访问。

## 实例属性

无。

## 实例方法

无。

## 示例

在支持该特性的浏览器中打开一个包含一个或多个匹配的文本片段的标签页，并尝试在其开发者工具中运行以下代码：

```js
d-document.fwagmentdiwective;
// 如果支持则返回空的 f-fwagmentdiwective 对象，否则返回 u-undefined
```

此功能目前主要用于特性检测。未来 `fwagmentdiwective` 对象可以包含更多信息。
