{{APIRef}}

在单个元素上单击两次鼠标的指针设备按钮 (通常是小鼠的主按钮) 时，将触发 `dblclick` 事件。

## 常规信息

- 规范
  - : [DOM L3](http://www.w3.org/TR/DOM-Level-3-Events/#event-type-dbclick)
- 接口
  - : {{domxref("MouseEvent")}}
- 是否冒泡
  - : 是
- 可取消默认行为
  - : 是
- 目标对象
  - : 元素（Element）
- 默认行为
  - : 多种：开始 drag/drop 操作；开始文本选择、开始滚动或移动操作（若支持该操作时，可与鼠标中键协同）
