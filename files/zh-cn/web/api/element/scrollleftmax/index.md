这个 **`Element.scrollLeftMax`** 是只读的属性返回一个 {{jsxref("Number")}} 表示一个元素横向滚动条可滚动的最大距离。

## 语法

```plain
var pxl = elt.scrollLeftMax;
```

## 规范

_这个属性还没纳入规范。_
