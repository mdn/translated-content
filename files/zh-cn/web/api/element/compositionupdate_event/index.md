| Property                        | Type                              | Description                                                         |
| ------------------------------- | --------------------------------- | ------------------------------------------------------------------- |
| `target` {{ReadOnlyInline}}     | {{domxref("EventTarget")}}        | 焦点所在的，处理文字输入的元素。                                    |
| `type` {{ReadOnlyInline}}       | {{domxref("DOMString")}}          | The type of event.                                                  |
| `bubbles` {{ReadOnlyInline}}    | `boolean`                         | Does the event normally bubble?                                     |
