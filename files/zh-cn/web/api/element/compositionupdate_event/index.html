compositionupdate 事件触发于字符被输入到一段文字的时候（这些可见字符的输入可能需要一连串的键盘操作、语音识别或者点击输入法的备选词）

Bubbles Yes
Cancelable No

locale {{ReadOnlyInline}} {{domxref("DOMString")}} (string) 输入事件的语言代号，或者空字符串。只读。
