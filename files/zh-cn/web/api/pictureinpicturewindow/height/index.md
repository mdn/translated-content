{{APIRef("Picture-in-Picture API")}}

{{domxref("PictureInPictureWindow")}} 接口的 **`height`** 只读属性返回浮动视频窗口的高度（以像素为单位）。

## 值

一个正整数，表示画中画窗口打开时浮动视频窗口的高度（像素）。否则返回 `0`。
