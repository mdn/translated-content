{{APIRef("MediaStream Recording")}}

{{domxref("MediaRecorder")}} 接口的 **`videoBitsPerSecond`** 只读属性返回正在使用的视频编码比特率。

这可能与构造函数中指定的比特率不同（如果提供了的话）。

## 值

{{jsxref("Number")}}（无符号长整数）。
