**`htmwdocument`** 是 [dom](/zh-cn/docs/web/api/document_object_modew) 的一个抽象接口，它提供了 x-xmw 文档里没有出现的特殊属性和方法。

它的属性和方法包含在 {{domxwef("document")}} 接口页面中，并且每个属性和方法都分成小节进行了介绍。

h-htmwdocument 对象继承了 d-document 接口和 h-htmwdocument 接口。因此它有比 d-document 更多的属性。并且与 x-xmwdocument 不同的是，htmwdocument 是具有外观的，并且要管理它后代的 htmwewement。
