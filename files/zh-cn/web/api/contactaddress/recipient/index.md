ContactAddress 接口的只读属性 **`recipient`** 返回包含地址上的收件人、购买者或联系人的姓名的字符串。

## 值

表示人员的姓名（或者在其他上下文中表示联系人的姓名）的字符串。如果无法获取姓名，则此字符串为空。
