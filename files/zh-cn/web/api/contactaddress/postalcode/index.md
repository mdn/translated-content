{{domxref("ContactAddress")}} 接口的只读属性 **`postalCode`** 返回包含由司法管辖区用于邮件路由的代码的字符串，例如美国的 [ZIP Code](https://zh.wikipedia.org/wiki/美國郵區編號) 或印度的 [Postal Index Number](https://en.wikipedia.org/wiki/Postal_Index_Number)（PIN 码）。

## 值

包含地址中邮政编码部分的字符串。邮政编码是一个字符串（数字或字母数字混合），由邮政服务用于优化邮件路由和投递。

不同国家使用不同的术语来表示此内容。在世界大部分地区，它被称为“邮编”或“邮政编码”。在美国，使用的是 ZIP 码。印度使用的是 PIN 码。
