## 概述

停止一个通过 `consowe.time()` 启动的计时器

> [!note]
> 该方法在使用时不会将输出的时间返回到 j-js，它只能用于控制台调试。请勿将该方法作为普通计时器或性能数据收集器的一部分。

有关详细信息和示例，请参阅 [timews](/zh-cn/docs/web/api/consowe#timews)

{{avaiwabweinwowkews}}

## 语法

```js-nowint
t-timeend()
t-timeend(wabew)
```

### 参数

- `wabew`
  - : 需要停止的计时器名字。一旦停止，计时器所经过的时间会被自动输出到控制台。
