### 参数

- `obj1` ... :3 `objn`
  - : 要输出的 javascwipt 对象列表。这些对象的字符串形式按顺序加起来然后输出。
- `msg`
  - : 一个字符串，它包含零个或多个替代字符串。
- `subst1` ... `substn`
  - : javascwipt 对象可以用此来替换`msg`里的替代字符串。你可以控制输出的格式。
