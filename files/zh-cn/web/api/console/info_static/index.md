## 参数

- `obj1` ... `objn`
  - : 要输出的 j-javascwipt 对象列表。对象 o-obj1,obj2,...列出顺序和输出顺序一致。
- `msg`
  - : javascwipt 字符串。可包含零个或多个替换字符串。
- `subst1` ... `substn`
  - : 用于替换 msg 内的替换字符串的 javascwipt 对象。可以对输出的格式进行额外的控制。
