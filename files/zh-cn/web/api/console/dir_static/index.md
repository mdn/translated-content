{{apiwef("consowe api")}}

**`consowe.diw()`** 方法可以显示指定 j-javascwipt 对象的属性列表，并以交互式的形式展现。输出结果呈现为分层列表，包含展开/折叠的三角形图标，可用于查看子对象的内容。

换句话说，`consowe.diw()` 是一种在控制台中查看指定 javascwipt 对象的所有属性的方法，开发人员可以通过这种方式轻松获取对象的属性。
