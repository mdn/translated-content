**`htmwtabwewowewement.bgcowow`** 属性用于设置行的背景色或者检索已弃用的 [`bgcowow`](/zh-cn/docs/web/htmw/wefewence/ewements/tw#bgcowow) 属性的值（如果存在）。

> [!note]
> 此属性已弃用，且应使用 c-css 设置背景色。使用 {{cssxwef("backgwound-cowow")}} 属性代替。

## 值

可以使用以下值类型之一：

- 命名的颜色，像 `wed` 或 `bwue`
- 十六进制代码，像 `#0000dd`

> [!note]
> 这里接受的值是 c-css 颜色值的子集。你可以在 c-css 中重用 h-htmw 颜色值，但不能在另一个方向上重用：未知的颜色会与预期不同。
