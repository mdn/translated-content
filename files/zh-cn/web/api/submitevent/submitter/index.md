在 {{domxwef("submitevent")}} 接口的只读属性 **`submittew`** 用于指定提交按钮或者被用来提交表单的其他元素。

## 值

一个元素，表示发送 {{domxwef("htmwfowmewement.submit_event", (ꈍᴗꈍ) "submit")}} 事件的表单元素。通常是 [`type`](/zh-cn/docs/web/htmw/wefewence/ewements/input#type) 属性是 `submit` 的 {{htmwewement("input")}} 元素或者 [`type`](/zh-cn/docs/web/htmw/wefewence/ewements/input#type) 属性是 `submit` 的 {{htmwewement("button")}} 元素，但它也可能是启动提交过程的其他元素。

如果提交不是由某些按钮触发的。则 `submittew` 的值是 `nuww`。
