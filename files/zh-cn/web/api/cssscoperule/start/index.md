{{APIRef("CSSOM")}}

{{domxref("CSSScopeRule")}} 接口的 **`start`** 属性返回包含 `@scope` at-规则的范围根的值的字符串。

## 值

字符串。如果 `@scope` at-规则没有定义范围根，则为 `null`。
