{{APIRef("WebRTC")}}

{{domxref("RTCIceCandidatePairStats")}} 字典的 **`lastPacketReceivedTimestamp`** 属性指示由候选对描述的连接上次接收数据包的时间。

不包括 {{Glossary("STUN")}} 数据包。

## 值

表示由候选对描述的连接上次接收数据包的时间（不包括 STUN 数据包）的 {{domxref("DOMHighResTimeStamp")}} 对象。
