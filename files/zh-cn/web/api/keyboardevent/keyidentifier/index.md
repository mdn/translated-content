{{APIRef("UI Events")}}{{non-standard_header}}{{deprecated_header}}

已弃用的 **`KeyboardEvent.keyIdentifier`** 只读属性返回一个“按键标识符”字符串，可用于确定按下了哪个键。与其对应的未被弃用的替代属性是 {{domxref("KeyboardEvent.key")}}。

## 规范

*不属于任何当前规范。*该属性曾是 [DOM 第 3 版事件规范的一个旧草案](https://www.w3.org/TR/2009/WD-DOM-Level-3-Events-20090908/#events-Events-KeyboardEvent-keyIdentifier)的一部分，但在后续的草案中，为了支持 {{domxref("KeyboardEvent.key")}}，它已被移除。
