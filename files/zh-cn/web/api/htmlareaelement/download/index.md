{{APIRef("HTML DOM")}}

{{domxref("HTMLAreaElement")}} 接口的 **`download`** 属性是用于表明所链接的资源应被下载而非在浏览器中显示的字符串。该值表示下载时建议使用的文件名。如果该名称不是底层操作系统所支持的有效文件名，浏览器将对其进行相应调整。

该属性反映了 {{HTMLElement("area")}} 元素的 `download` 属性。

## 值

字符串。
