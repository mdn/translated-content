{{APIRef("History API")}}

{{DOMxRef("History")}} 接口的 **`length`** 只读属性返回一个表示会话历史中的条目数量（包括当前加载的页面）的整数。

例如，对于在新标签页中加载的页面，此属性返回 `1`。

## 值

数字。
