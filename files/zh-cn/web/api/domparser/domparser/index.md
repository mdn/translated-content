**`DOMParser()`** 构造函数新建一个 [`DOMParser`](/zh-CN/docs/Web/API/DOMParser) 对象实例。

## 语法

```js
var parser = new DOMParser();
```

### 参数

无。

### 返回值

一个新建的[`DOMParser`](/zh-CN/docs/Web/API/DOMParser) 对象实例，可以通过这个对象的 `parseFromString()` 方法将字符串解析为 DOM 对象。
