{{APIRef("DOM")}}

**`DOMParser()`** 构造函数用于创建一个新的 [`DOMParser`](/zh-CN/docs/Web/API/DOMParser) 对象。该对象可用于通过 `parseFromString()` 方法解析文档的文本内容。

## 语法

```js-nolint
new DOMParser()
```

### 参数

无。

### 返回值

一个新的 [`DOMParser`](/zh-CN/docs/Web/API/DOMParser) 对象。该对象可用于通过 `parseFromString()` 方法解析文档的文本内容。
