{{APIRef("Web Animations")}}

[Web Animations API](/zh-CN/docs/Web/API/Web_Animations_API) 的 **`Animation.pending`** 只读属性指示动画当前是否正在等待异步操作，例如启动播放或暂停正在运行的动画。

## 值

如果动画正在等待，则为 **`true`**；否则为 **`false`**。
