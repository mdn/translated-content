**`originalTarget`** 只读属性是事件重定向之前的原始目标。与 {{domxref("Event.explicitOriginalTarget")}} 不同，原始目标也可能是原生的匿名内容。

## 规范

_这是一个 Mozilla 特有的属性，不属于任何规范。它不会成为标准。_
