{{APIRef("Web Animations")}}

**`AnimationEvent.pseudoElement`** 只读属性是一个以 `'::'` 开头，包含动画运行在其上的[伪元素](/zh-CN/docs/Web/CSS/Pseudo-elements)的名称的字符串。如果动画不是在伪元素上运行而是直接在元素上运行，则该属性值为空字符串 `''`。

## 值

一个以 `'::'` 开头的字符串，包含正在运行着对应动画的[伪元素](/zh-CN/docs/Web/CSS/Pseudo-elements)的名称。
