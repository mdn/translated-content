{{APIRef("HTML DOM")}}

这个 **`HTMLTableElement.createCaption()`** 方法返回这个 table 元素的 caption(HTMLTableCaptionElement). 如果这个 table 没有 caption，这个方法创建并返回 caption.

## 语法

```js-nolint
createCaption()
```

## 示例

```js
mycap = mytable.createCaption();
```
