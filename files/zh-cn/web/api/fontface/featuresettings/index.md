{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

{{domxref("FontFace")}} 接口的 **`featureSettings`** 属性用于检索或设置那些无法通过字体的变体属性获得的、不常使用的字体特性。

该属性等效于 {{cssxref("font-feature-settings")}} 描述符。

## 值

包含描述符的字符串。
