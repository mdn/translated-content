{{domxref("FontFace")}} 接口的 **`variationSettings`** 属性用于获取或设置底层 OpenType 或 TrueType 字体变体。

该属性等效于 {{cssxref("@font-face/font-variation-settings", "font-variation-settings")}} 描述符。

## 值

包含描述符的字符串。
