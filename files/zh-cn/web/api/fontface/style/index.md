{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

{{domxref("FontFace")}} 接口的 **`style`** 属性用于获取或设置字体的样式。

该属性等效于 {{cssxref("@font-face/font-style", "font-style")}} 描述符。

## 值

一个字符串，内容为样式表 `@font-face` 规则中所定义的描述符。
