{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

{{domxref("FontFace")}} 接口的 **`stretch`** 属性用于获取或设置字体的拉伸方式。

该属性等效于 {{cssxref("@font-face/font-stretch", "font-stretch")}} 描述符。

## 值

包含样式表 `@font-face` 规则中所定义的描述符的字符串。
