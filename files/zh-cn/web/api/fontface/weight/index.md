{{domxref("FontFace")}} 接口的 **`weight`** 属性用于获取或设置字体的字重。

该属性等效于 {{cssxref("@font-face/font-weight", "font-weight")}} 描述符。

## 值

包含样式表 `@font-face` 规则中所定义的描述符的字符串。
