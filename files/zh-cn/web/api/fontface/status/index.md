{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

{{domxref("FontFace")}} 接口的 **`status`** 只读属性返回指示字体的状态的枚举值，可以是 `"unloaded"`、`"loading"`、`"loaded"` 或 `"error"` 之一。

## 值

`"unloaded"`、`"loading"`、`"loaded"` 或 `"error"` 之一。
