{{APIRef("HTML DOM")}}

**`HTMLAnchorElement.download`** 属性是一个字符串，表示链接的资源旨在被下载而非在浏览器中显示。如果该属性有值，则指定了在本地文件系统中标记资源时使用的默认文件名。如果名称在底层操作系统中不是有效的文件名，浏览器将会对其进行调整。

> [!NOTE]
> 这个值可能不会被用于下载。这个值不能用来确定下载是否会发生。

## 值

一个字符串。
