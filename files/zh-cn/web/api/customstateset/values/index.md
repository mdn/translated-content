{{APIRef("Web Components")}}

{{domxref("CustomStateSet")}} 接口的 **`values()`** 方法返回一个包含给定 `CustomStateSet` 对象中的每个元素的值（按插入顺序排列）的新迭代器对象。

## 语法

```js-nolint
values()
```

### 参数

无。

## 返回值

包含给定 `CustomStateSet` 中每个元素的值（按插入顺序排列）的新迭代器对象。
