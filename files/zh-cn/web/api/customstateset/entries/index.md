{{APIRef("Web Components")}}

{{domxref("CustomStateSet")}} 接口的 **`entries`** 方法返回一个包含对应于 `CustomStateSet` 中每个元素（按插入顺序排列）的数组 `[value, value]` 的新[迭代器](/zh-CN/docs/Web/JavaScript/Reference/Iteration_protocols)对象。

## 语法

```js-nolint
entries()
```

### 参数

无。

## 返回值

包含对应于 `CustomStateSet` 中每个元素（按插入顺序排列）的 `[value, value]` 数组的新迭代器对象。
