{{APIRef("Web Components")}}

{{domxref("CustomStateSet")}} 接口的 **`keys()`** 方法是 {{domxref("CustomStateSet.values")}} 方法的别名。

## 语法

```js-nolint
keys()
```

### 参数

无。

## 返回值

新的迭代器对象。它包含给定 `CustomStateSet` 中每个元素的值，并按插入顺序排列。
