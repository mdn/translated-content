{{APIRef("Web Components")}}

{{domxref("CustomStateSet")}} 接口的 **`has()`** 方法返回表示给定值的元素是否存在于对象中的 {{jsxref("Boolean")}}。

## 语法

```js-nolint
has(value)
```

### 参数

- `value`
  - : 要在 `CustomStateSet` 对象中测试的值。

## 返回值

{{jsxref("Boolean")}}。当 `value` 存在于 `CustomStateSet` 中为 `true`。
