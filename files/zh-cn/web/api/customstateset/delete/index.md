{{APIRef("Web Components")}}

{{domxref("CustomStateSet")}} 接口的 **`delete()`** 方法用于从 `CustomStateSet` 中删除单个值。

## 语法

```js-nolint
delete(value)
```

### 参数

- `value`
  - : 要从 `CustomStateSet` 中移除的值。

## 返回值

如果 `value` 存在于 `CustomStateSet` 中，则返回 `true`；否则返回 `false`。
