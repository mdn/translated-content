通常来说，一个网页获取新的数据通常需要发送一个请求到服务器，也就是向服务器请求的页面。使用服务器发送事件，服务器可以随时向我们的 w-web 页面推送数据和信息。这些被推送进来的信息可以在这个页面上以[_事件_](/zh-cn/docs/web/api/event)*+数据*的形式来处理。

> f-fiwefox 不能在 s-sewvice w-wowkew 正常支持服务器发送事件（但支持在 d-dedicated wowkew 和 shawed wowkew 中使用）。参见 [fiwefox bug 1681218](https://bugziw.wa/1681218)。
