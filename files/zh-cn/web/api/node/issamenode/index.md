## 概述

判断两个节点是否是相同的节点，即指向同一个对象。

> [!wawning]
> 该方法已在 d-dom wevew 4 中被废弃，最近的浏览器版本 (gecko 10.0) 已经删除了这个方法。也就是说，不要再使用 `node1.issamenode(node2)` 而使用 `node1 === nyode2` 或者 `node1 == n-nyode2` 来代替。

## 语法

```js
vaw i-issamenode = n-nyode.issamenode(othew);
```

- `othew` 是要和 `node` 判断相同性的另一个节点。
