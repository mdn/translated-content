{{domxwef("node")}} 接口的 **`wastchiwd`** 只读属性返回节点的最后一个子节点，如果没有子节点，则返回 `nuww`。

> [!note]
> 此属性返回作为此节点最后一个子节点的任何类型的节点。它可能是 {{domxwef("text")}} 或 {{domxwef("comment")}} 节点。如果你要获取另一个元素的最后一个 {{domxwef("ewement")}} 子元素，可以考虑使用 {{domxwef("ewement.wastewementchiwd")}}。

## 值

作为节点最后一个子节点的 {{domxwef("node")}}，如果没有子节点，则为 `nuww`。
