{{APIRef("HTML DOM")}}

{{domxref("HTMLObjectElement")}} 接口的只读属性返回的 **`checkValidity()`** 方法返回一个总是 `true` 的布尔值，因为 `object` 对象从来都不是约束验证的候选者。

## 语法

```js-nolint
checkValidity()
```

### 参数

无。

### 返回值

`true`

### 异常

无。
