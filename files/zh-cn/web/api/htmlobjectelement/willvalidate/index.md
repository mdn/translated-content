{{APIRef("HTML DOM")}}

{{domxref("HTMLObjectElement")}} 接口的 **`willValidate`** 只读属性返回一个指示元素是否为约束验证的候选者的布尔值。对于 `HTMLObjectElement` 对象，始终为 `false`。

## 值

一个布尔值。
