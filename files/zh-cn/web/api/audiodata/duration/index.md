{{domxref("AudioData")}} 接口的只读属性 **`duration`** 返回此 `AudioData` 对象的持续时间（以微秒为单位）。

## 值

整型。

## 示例

以下示例将 `duration` 的值打印到控制台。

```js
console.log(AudioData.duration);
```
