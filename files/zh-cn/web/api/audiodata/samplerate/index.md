{{domxref("AudioData")}} 接口的只读属性 **`sampleRate`** 返回采样率（以 Hz 为单位）。

## 值

一个十进制数。

## 示例

以下示例将 `sampleRate` 的值打印到控制台。

```js
console.log(audioData.sampleRate);
```
