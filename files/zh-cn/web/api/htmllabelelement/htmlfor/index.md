{{APIRef("HTML DOM")}}

**`HTMLLabelElement.htmlFor`** 属性反映 [`for`](/zh-CN/docs/Web/HTML/Element/label#for) 内容属性的值。意味着此脚本可访问属性用于设置和读取内容属性 `for`（该属性是标签关联的控件元素的 ID）的值。

## 值

包含与控件相关元素 ID 的字符串。

> [!NOTE]
> 如果属性有值，{{domxref("HTMLLabelElement.control")}} 属性必须引用相同的控件。
