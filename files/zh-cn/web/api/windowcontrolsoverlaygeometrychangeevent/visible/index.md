{{domxref("WindowControlsOverlayGeometryChangeEvent")}} 接口的只读属性 **`visible`** 是一个布尔标志，用于指示在桌面安装的渐进式 Web 应用中，窗口控件叠加层是否可见。

如果用户选择退出[窗口控件叠加](/zh-CN/docs/Web/API/Window_Controls_Overlay_API)功能，则会显示默认标题栏，且窗口控件按钮不会以叠加层形式出现，此时 `visible` 标志为 false。
