{{domxref("MediaKeyMessageEvent")}} 的 **`initData`** 只读属性在该事件中包含初始化数据时返回这些数据。

## 值

包含初始化数据的 {{jsxref("ArrayBuffer")}}。如果没有这些数据则返回 `null`。
