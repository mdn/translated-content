{{APIRef("DOM")}}

{{domxref("AbstractRange")}} 接口的 **`endContainer`** 只读属性返回范围结束位置所在的{{domxref("Node", "节点", , 1)}}。

## 值

包含范围最后一个字符的 {{domxref("Node")}}。

## 示例

```js
let endNode = range.endContainer;
```
