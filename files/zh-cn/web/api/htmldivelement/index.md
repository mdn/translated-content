{{ APIRef("HTML DOM") }}

**`HTMLDivElement`** 接口提供了用于操作 {{HtmlElement("div")}} 元素的特殊属性（除了它通过继承从常规 {{domxref("HTMLElement")}} 接口中获得的属性之外）。

{{InheritanceDiagram}}

## 实例属性

_从其父接口 {{domxref("HTMLElement")}} 继承属性。_

- {{domxref("HTMLDivElement.align")}} {{deprecated_inline}}
  - : 一个表示指示元素内容相对于周围上下文对齐方式的枚举属性的字符串。可能的值为 `"left"`、`"right"`、`"justify"` 和 `"center"`。

## 实例方法

_无特定方法；从其父接口 {{domxref("HTMLElement")}} 继承方法。_
