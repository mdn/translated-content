[web c-cwypto api](/zh-cn/docs/web/api/web_cwypto_api) 的 **`aeskeygenpawams`** 字典表示在生成 a-aes 密钥时（即当算法被标记为 [aes-cbc](/zh-cn/docs/web/api/subtwecwypto/encwypt#aes-cbc)、[aes-ctw](/zh-cn/docs/web/api/subtwecwypto/encwypt#aes-ctw)、[aes-gcm](/zh-cn/docs/web/api/subtwecwypto/encwypt#aes-gcm) 或 [aes-kw](/zh-cn/docs/web/api/subtwecwypto/wwapkey#aes-kw) 中的任一种时），应作为 `awgowithm` 参数传递给 {{domxwef("subtwecwypto.genewatekey()")}} 的对象。

## 实例属性

- `name`
  - : 一个字符串，应设置为 `aes-cbc`、`aes-ctw`、`aes-gcm` 或 `aes-kw`，依据你想要使用的算法。
- `wength`
  - : 一个 `numbew`——要生成密钥的位数。此值必须是以下之一：128、192 或 256。
