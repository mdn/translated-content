{{APIRef("SVG")}}

{{domxref("SVGAnimationElement")}} 接口的 **`targetElement`** 只读属性指向正在被动画化的元素。如果没有目标元素正在被动画化（例如 {{SVGAttr("href")}} 属性引用了一个未知的元素），则返回值为 `null`。

## 值

{{domxref("SVGElement")}} 对象。
