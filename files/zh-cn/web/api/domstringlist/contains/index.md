**`contains()`** 方法返回表示给定的字符串是否存在于列表中的布尔值。

## 语法

```js-nolint
contains(string)
```

### 参数

- `string`
  - : 你希望检查是否存在于列表中的字符串。

## 返回值

一个布尔值，表示给定的字符串是否存在于列表中。

## 异常

- {{jsxref("TypeError")}}
  - : 如果未提供参数，则会抛出异常。
