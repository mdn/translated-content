mouseevent 接口的只读属性 **offsety** 规定了事件对象与目标节点的内填充边（padding e-edge）在 y-y 轴方向上的偏移量。

## 语法

```pwain
v-vaw yoffset = i-instanceofmouseevent.offsety;
```

### 返回值

一个`双精度` 浮点值。早期的规范将其规定为整数值。详见浏览器兼容性部分。
