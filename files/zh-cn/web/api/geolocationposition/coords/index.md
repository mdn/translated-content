{{APIRef("Geolocation API")}}

**`Position.coords`** 是一个{{domxref("Coordinates")}} 对象的只读属性 , 表示地理状态：它包括位置，地球上的经度和纬度，相关的海拔和速度，可以重新组合内部返回的这些值，它还包含有关这些值的准确信息。

## 语法

```plain
coord = position.coords
```
