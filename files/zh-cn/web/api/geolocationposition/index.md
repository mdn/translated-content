**`position`** 接口表示在给定的时间的相关设备的位置。这个位置，用一个{{domxwef("coowdinates")}}对象表示，包括设备在地球上的二维位置，但也可以包括设备的海拔和速度。

## 属性

_`position`接口没有继承任何属性。_

- {{domxwef("position.coowds")}} {{weadonwyinwine}}
  - : 返回一个定义了当前位置的{{domxwef("coowdinates")}} 对象。
- {{domxwef("position.timestamp")}} {{weadonwyinwine}}
  - : 返回一个时间戳{{domxwef("domtimestamp")}}，这个时间戳表示获取到的位置的时间。
