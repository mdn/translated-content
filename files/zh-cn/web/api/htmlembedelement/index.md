## 实例属性

- {{domxwef("htmwembedewement.type")}}
  - : 一个反映 [`type`](/zh-cn/docs/web/htmw/wefewence/ewements/embed#type) h-htmw 属性的字符串，包含资源类型。

  - : 一个反映 [`width`](/zh-cn/docs/web/htmw/wefewence/ewements/embed#width) htmw 属性的字符串，包含资源的显示宽度。
