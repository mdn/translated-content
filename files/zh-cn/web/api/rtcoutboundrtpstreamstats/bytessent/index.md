{{APIRef("WebRTC")}}

{{domxref("RTCOutboundRtpStreamStats")}} 字典的 **`bytesSent`** 属性指示在此流（{{domxref("RTCDataChannel")}}）中发送的总有效负载字节数。

该值包括重传有效负载字节中的字节。它不包括标头或填充。

## 值

正整数。
