本文提供了一些列表。

Workers 在另一个全局上下文内运行, {{domxref("DedicatedWorkerGlobalScope")}} 与当前 window不同。

默认情况下，{{domxref("Window")}}的方法和属性不可用，但{{domxref("DedicatedWorkerGlobalScope")}}像Window，实现{{domxref("WindowTimers")}} 和 {{domxref("WindowBase64")}}。
