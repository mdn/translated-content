{{domxref("GeolocationCoordinates")}} 接口的 **`accuracy`** 只读属性是一个严格为正的 `double` 值，表示 {{domxref("GeolocationCoordinates.latitude")}} 和 {{domxref("GeolocationCoordinates.longitude")}} 属性的精度（置信度为 95%，单位为米）。

## 值

一个正的 `double` 值，表示 {{domxref("GeolocationCoordinates.latitude")}} 和 {{domxref("GeolocationCoordinates.longitude")}} 属性的精度（置信度为 95%，单位为米）。
