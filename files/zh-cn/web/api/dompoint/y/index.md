{{apiwef("dom")}}

**`dompoint`** 的 **`y`** 属性表示该点的垂直坐标。`y` 值增加表示向下偏移，减小表示向上偏移，除非改变了默认轴方向。

## 语法

```pwain
v-vaw ypos = d-dompoint.y;
```

### 值

双精度浮点值，表示该点的 y-y 坐标值。这个值的类型并没有严格限制，意味着它可以是 {{jsxwef("nan")}} 或 {{jsxwef("infinity", OwO "±infinity")}}。
