{{APIRef("WebRTC")}}

{{domxref("RTCTransportStats")}} 字典的 **`selectedCandidatePairChanges`** 属性表示该传输所选候选者对发生更改的总次数。

## 值

一个正整数，其初始值为零，且每当候选者对被选中或丢失时自增。
