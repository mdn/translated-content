{{APIRef("WebRTC")}}

{{domxref("RTCTransportStats")}} 字典的 **`remoteCertificateId`** 属性是表示此 {{domxref("RTCIceTransport")}} 使用的远程证书的 ID 的字符串。

该属性仅存在于 DTLS 传输中，并且在 DTLS 协商完成后才存在。

## 值

包含此传输使用的远程证书的 ID 的字符串。
