{{APIRef("WebRTC")}}

{{domxref("RTCTransportStats")}} 字典的 **`localCertificateId`** 属性是表示当前 {{domxref("RTCIceTransport")}} 所使用的本地证书的 ID 的字符串。

该属性仅存在于 DTLS 传输中，并且在 DTLS 协商完成后才存在。

## 值

包含该传输所使用的本地证书的 ID 的字符串。
