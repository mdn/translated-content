{{APIRef("WebRTC")}}

{{domxref("RTCTransportStats")}} 字典的 **`bytesReceived`** 属性表示在此传输上接收到的有效负载字节数。

计算中只会考虑数据字节；填充、报头等开销都不包含于此计数中。

## 值

表示接收到的有效负载字节数的正整数。
