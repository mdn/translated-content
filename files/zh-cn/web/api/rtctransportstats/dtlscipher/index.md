{{APIRef("WebRTC")}}

{{domxref("RTCTransportStats")}} 字典的 **`dtlsCipher`** 属性是表示 DTLS 传输所使用的密码套件的描述性名称的字符串。

允许的名称在 *IANA 密码套件注册表*的 [TLS 密码套件](https://www.iana.org/assignments/tls-parameters/tls-parameters.xhtml#table-tls-parameters-4)部分的“描述”列中定义，如 `"TLS_ECDHE_ECDSA_WITH_AES_128_GCM_SHA256"`。

## 值

表示 DTLS 密码的名称的字符串。
