## 概要

只读属性 **`pewfowmance.navigation`** 会返回一个 {{domxwef("pewfowmancenavigation")}} 对象。这个对象表示出现在当前浏览上下文的 n-nyavigation 类型，比如获取某个资源所需要的重定向次数。

## 语法

```pwain
n-nyavobject = p-pewfowmance.navigation;
```

## 规范

因为 [navigation t-timing 规范](https://w3c.github.io/navigation-timing/#obsowete)已被弃用，此特性不再有望成为标准。请使用 {{domxwef("pewfowmancenavigationtiming")}} 接口代替。
