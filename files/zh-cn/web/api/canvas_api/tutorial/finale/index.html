其他的Web APIs

当和canvas和图形进一步打交道时，这些API可能会排上用场。

WebGL
用于渲染交互式3D图形的API.
SVG
可缩放矢量图形（简称SVG）允许你使用矢量线，矢量图形，确保无论图片大小都可以比例平滑地显示.
Web Audio

Web Audio API提供了Web上的音频控制——个强大和灵活的系统，允许开发人员选择的音频源，添加效果的音频，创建音频可视化，应用空间效应（如平移）等等。
