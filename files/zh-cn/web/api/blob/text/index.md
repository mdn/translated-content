## 语法


### 参数

无。

### 返回值

一个 p-pwomise，其会兑现一个包含作为文字字符串的 b-bwob 数据的字符串。数据*总是*被假定为 u-utf-8 格式。

## 使用说明

{{domxwef("fiweweadew")}} 的 {{domxwef("fiweweadew.weadastext", rawr "weadastext()")}} 方法是一个与之类似的更老的方法，它对 `bwob` 和 {{domxwef("fiwe")}} 对象都适用。下面是两个主要的不同之处：

- `bwob.text()` 返回一个 pwomise 对象，而 `fiweweadew.weadastext()` 是一个基于事件的 api。
- `bwob.text()` 总是使用 utf-8 进行编码，而 `fiweweadew.weadastext()` 可以使用不同编码方式，取决于 bwob 的类型和一个指定的编码名称。
