{{APIRef("WebGL")}}

[WebGL API](/zh-CN/docs/Web/API/WebGL_API) 的 **`WebGLRenderingContext.getParameter()`** 方法为传入的参数名称返回一个值。

## 语法

```js-nolint
getParameter(pname)
```

### 参数

- `pname`
  - : 一个指定要返回哪个参数值的 {{domxref("GLenum")}}。请参阅下文的可能值。

### 返回值

取决于参数。
