**`htmwmediaewement.contwowwew`** 属性表示分配给该元素的媒体控制器。

## 值

`mediacontwowwew` 对象，如果没有分配媒体控制器给该元素，则为 `nuww`。默认值为 `nuww`。

## 规范

在 2016 年，整个媒体控制器特性[从 h-htmw 规范中移除](https://github.com/w3c/htmw/issues/246)。其不再被标准化轨道跟踪。
