{{APIRef("HTML DOM")}}

 HTMLMediaElement.audioTracks 获得可用音频轨道的数量

语法

audio|video.audioTracks

返回值

AudioTrackList    对象 表示音频/视频的可用音频轨道

用例

以 audio 标签为例

<audio id='audio'></audio>

调用

myVid=document.getElementById("audio");
alert(myVid.audioTracks.length);
