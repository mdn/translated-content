{{APIRef("HTML DOM")}}

只读属性 `HTMLMediaElement.paused` 指示视频是否正在暂停。

## 返回值

布尔值。`true` 表示正在暂停，`false` 表示没有暂停。

## 示例

```js
const obj = document.createElement("video");
console.log(obj.paused); // true
```
