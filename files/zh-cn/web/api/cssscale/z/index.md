{{APIRef("CSS Typed OM")}}

{{domxref("CSSScale")}} 接口的 **`z`** 属性表示平移向量的 z 分量。正值会使元素向观察者移动，负值则使元素远离观察者。

如果此值存在，则变换为 3D 变换，且 `is2D` 属性将设置为 false。

## 值

{{domxref("CSSNumericValue")}}。可以设置为数值（该数值会被包装在具有 `unit: "number"` 的 {{domxref("CSSUnitValue")}} 中）。
