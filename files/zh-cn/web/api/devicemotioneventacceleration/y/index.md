**`y`** 是 {{domxref("DeviceMotionEventAcceleration")}} 接口的只读属性，表示在 [`DeviceMotionEventAcceleration`](/zh-CN/docs/Web/API/DeviceMotionEventAcceleration) 对象中沿 Y 轴发生的加速度值。

## 值

`double` 类型的值。它表示沿 Y 轴的加速度。更多详细信息，请参见[详述方向与运动数据](/zh-CN/docs/Web/API/Device_orientation_events/Orientation_and_motion_data_explained)。
