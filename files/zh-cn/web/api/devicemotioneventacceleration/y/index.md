{{domxref("DeviceMotionEventAcceleration")}} 接口的只读属性 **`y`** 表示 [`DeviceMotionEventAcceleration`](/zh-CN/docs/Web/API/DeviceMotionEventAcceleration) 对象中沿 Y 轴产生的加速度值。

## 值

`double` 类型的值，表示沿 Y 轴的加速度。更多详细信息，请参见[方向与运动数据解释](/zh-CN/docs/Web/API/Device_orientation_events/Orientation_and_motion_data_explained)。
