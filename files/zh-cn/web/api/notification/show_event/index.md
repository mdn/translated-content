## Summary

{{domxref("Notification")}} 接口的 onshow 属性指定一个事件侦听器来接收 {{event("show")}}事件。

当一个 {{domxref("Notification")}} 显示时，会发生这些事件。

## Syntax

```plain
Notification.onshow = function() { ... };
```
