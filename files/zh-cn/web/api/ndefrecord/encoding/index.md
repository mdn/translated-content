{{DOMxRef("NDEFRecord")}} 接口的 `encoding` 属性返回文本负载的编码方式。如果不是文本负载，则返回 `null`。

## 值

字符串，可以是以下值之一：`"utf-8"`、`"utf-16"`、`"utf-16le"` 或 `"utf-16be"`。
