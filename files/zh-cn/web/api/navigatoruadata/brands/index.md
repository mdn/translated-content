{{domxref("NavigatorUAData")}} 接口的 **`brands`** 只读属性返回一个包含品牌信息的数组。

## 值

一个包含每个品牌如下信息的数组：

- `brand`
  - : 一个包含品牌的字符串。例如，`"Google Chrome"`。
- `version`
  - : 一个包含版本的字符串。例如，`"91"`。

## 示例

以下示例将 `brands` 的值打印到控制台。

```js
console.log(navigator.userAgentData.brands);
```
