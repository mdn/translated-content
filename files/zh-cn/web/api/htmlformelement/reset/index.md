### 概述

`reset` 方法可以重置一个表单内的所有表单控件的值到初始状态。

### 语法

```plain
HTMLFormElement.reset()
```

### 例子

```plain
document.forms["myform"].reset();
```

### 备注

运行该方法和点击表单的重置按钮是一样的效果。
