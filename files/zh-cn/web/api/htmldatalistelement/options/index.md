{{APIRef("HTML DOM")}}

{{domxref("HTMLDataListElement")}} 接口的 **`options`** 只读属性返回一个包含在 {{htmlelement("datalist")}} 元素内的所有 {{domxref("HTMLOptionElement")}} 元素的 {{domxref("HTMLCollection")}}。这些后代 {{htmlelement("option")}} 元素为与 `<datalist>` 关联的 {{htmlelement("input")}} 控件提供预定义选项。

## 值

包含 {{domxref("HTMLOptionElement")}} 元素的 {{domxref("HTMLCollection")}}。
