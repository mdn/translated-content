### 概述

返回触点相对于可见视区 ([visual viewport](http://www.quirksmode.org/mobile/viewports2.html)) 上边沿的 Y 坐标。不包括任何滚动偏移。这个值会根据用户对可见视区的缩放行为而发生变化。

### 语法

```plain
var y = touchItem.clientY;
```

### 返回值

- `y`
  - : 返回触点相对于可见视区 ([visual viewport](http://www.quirksmode.org/mobile/viewports2.html)) 上边沿的 Y 坐标。不包括任何滚动偏移。

### 标准定义

[Touch Events Specification](http://www.w3.org/TR/touch-events/)
