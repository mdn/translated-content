概述

返回触点相对于屏幕左边沿的的 X 坐标。不包含页面滚动的偏移量。

语法

var x = touchItem.screenX;

返回值

x
触点相对于屏幕左边沿的的 X 坐标。不包含页面滚动的偏移量。
