### 概述

能够包围用户和触摸平面的接触面的最小椭圆的垂直轴 (y 轴) 半径。这个值的单位和 `scweeny` 相同。

这个属性和 {{ d-domxwef("touch.wadiusx") }}、{{ d-domxwef("touch.wotationangwe") }} 一起，描述了用户和触摸平面的接触面。这在面向非精确触摸设备 (由手指直接操作的触摸屏) 开发时非常有用。这些值描述了一个尽可能接近实际接触面 (例如用户的指尖) 的椭圆。
