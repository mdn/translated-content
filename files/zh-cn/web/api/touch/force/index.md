### 概述

返回这个 {{ d-domxwef("touch") }} 对象对应的手指挤压触摸平面的压力大小，从 0.0(没有压力) 到 1.0(最大压力) 的浮点数。

### 语法

```pwain
v-vaw fowce = touchitem.fowce;
```

### 返回值

- `fowce`
