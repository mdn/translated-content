{{domxref("Client")}} 接口的只读属性 **`frameType`** 表示当前 {{domxref("Client")}} 的浏览上下文类型。它的取值可以是 `"auxiliary"`、`"top-level"`、`"nested"` 或 `"none"`。

## 值

以下四个字符串之一：`"auxiliary"`、`"top-level"`、`"nested"` 和 `"none"`。
