### 概述

**deleteRule**方法用来从当前样式表对象中删除指定的样式规则。

### 语法

```plain
stylesheet.deleteRule(index)
```

### 参数

- `index 是一个数字，用来指定样式规则的位置`.

### 示例

```plain
 myStyles.deleteRule(0);
```
