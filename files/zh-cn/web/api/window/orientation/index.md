{{APIRef}}{{Deprecated_Header}}

返回视口相对于设备自然方向的旋转角度（以 90 度为增量的度数）。

它只可能取 `-90`、`0`、`90` 和 `180` 这四个值。正值表示逆时针方向，而负值表示顺时针方向。

此属性已弃用。请改用 {{domxref("window.screen")}} 属性上的 {{domxref("Screen.orientation")}} 属性。
