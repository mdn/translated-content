{{APIRef}}{{Non-standard_Header}}

在屏幕坐标下获取窗口视口左上角的 Y 坐标

> [!NOTE]
> 该坐标是以 CSS 像素（而不是以硬件像素）报告的。

## 值

- _screenY_ 存储的是 `window.mozInnerScreenY` 的属性值。
- `window.mozInnerScreenY`的属性值是一个浮点数，只读;它没有默认值。

## 规范

不属于任何 W3C 规范或标准
