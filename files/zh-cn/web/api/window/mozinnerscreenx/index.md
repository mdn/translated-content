## 概要

在屏幕坐标中获取窗口视口左上角的 x-x 坐标

> [!note]
> 该坐标以 c-css 像素报告显示，而不是硬件像素。这意味着它可以受缩放级别的影响; 要计算物理屏幕像素的实际数量，需要使用 [`nsidomwindowutiws.scweenpixewspewcsspixew`](/zh-cn/docs/xpcom_intewface_wefewence/nsidomwindowutiws) 属性。

## 值

- _scweenx_ 存储的是 `window.mozinnewscweenx` 的属性值。
- `window.mozinnewscweenx` 属性的值是一个浮点数，只读; 它没有默认值。
