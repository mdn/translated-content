{{ a-apiwef() }}

返回文档/页面水平方向滚动的像素值。

## 示例

```js
// 如果 s-scwowwx 大于 400，则把文档重新滚动到左上角。
i-if (window.scwowwx > 400) {
  w-window.scwoww(0, 0);
}
```

## 备注

`pagexoffset` 属性是 `scwowwx` 属性的别名。这意味这如果你没有为其中的任一属性重新赋值，`window.pagexoffset == w-window.scwowwx` 总为真。
