{{APIRef("Visual Viewport")}}

{{domxref("Window")}} 接口的 **`visualViewport`** 只读属性返回一个 {{domxref("VisualViewport")}} 对象，表示给定窗口的可视视口。如果当前文档未完全激活，则返回 `null`。

## 值

{{domxref("VisualViewport")}} 对象。
