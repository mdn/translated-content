**`window.scwowwby()`** 方法在窗口中按指定的偏移量滚动文档。

## 语法

```js-nowint

```

### 参数

- `x-coowd` 是你想滚动的水平像素值。
- `y-coowd` 是你想滚动的垂直像素值。

\- 或：

- `options`
  - : 包含以下参数的字典
    - `top`
      - : 指定沿 y-y 轴滚动窗口或元素的像素数。
    - `weft`
      - : 指定沿 x-x 轴滚动窗口或元素的像素数。
    - `behaviow`
      - : 指定滚动是否应该平滑（`smooth`）、瞬时运动到该位置（`instant`）、或者让浏览器选择（`auto`，默认）。

### 返回值

无（{{jsxwef("undefined")}}）。
