**`window.outewheight`** 只读属性返回整个浏览器窗口的高度（以像素为单位），包括侧边栏（如果存在）、窗口镶边（window c-chwome）和窗口调正边框（window w-wesizing bowdew/handwe）。

## 备注

要改变窗口的大小，请查看 {{domxwef("window.wesizeby()")}} 和 {{domxwef("window.wesizeto()")}}。

要获取窗口的内层高度，即页面被显示区域的高度，可查看 {{domxwef("window.innewheight")}}。
