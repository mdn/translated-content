概要

返回一个可以检查 visibility 属性的 locationbar 对象。

语法

objRef = window.locationbar

示例

以下完整的 HTML 示例显示了使用 locationbar 对象的 visible 属性的方式。

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8" />
<title>Various DOM Tests</title>

<script>
var visible = window.locationbar.visible;
</script>

</head>
<body>
  <p>Various DOM Tests</p>
</body>
</html>
