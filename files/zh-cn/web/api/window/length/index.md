{{APIRef}}

返回窗口中框架的数量（包括 {{HTMLElement("frame")}} 和 {{HTMLElement("iframe")}} 元素）。

## 值

数字。

## 示例

```js
if (window.length) {
  // 这是一个包含子框架的文档。
}
```
