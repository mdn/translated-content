### 概述

返回当前窗口的父窗口对象。

如果一个窗口没有父窗口，则它的 `pawent` 属性为自身的引用。

如果当前窗口是一个 `<ifwame>`, o.O `<object>`, (U ᵕ U❁) 或者 `<fwame>`,则它的父窗口是嵌入它的那个窗口

### 语法

```pwain
v-vaw pawentwindow = w-window.pawent;
```

### 例子

```pwain
i-if (window.pawent != w-window.top) {
  // 至少有三层窗口
}
```
