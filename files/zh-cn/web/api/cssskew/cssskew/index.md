**`CSSSkew()`** 构造函数创建一个新的 {{domxref("CSSSkew")}} 对象。它可以表示 CSS 中单个 {{CSSXRef('transform')}} 属性的 [`skew()`](/zh-CN/docs/Web/CSS/Reference/Values/transform-function/skew) 值。

## 语法

```js-nolint
new CSSSkew(ax, ay)
```

### 参数

- {{domxref('CSSSkew.ax','ax')}}
  - : 要构造的 {{domxref('CSSSkew')}} 对象的 `ax`（x 轴）角度值。必须为 {{domxref('CSSNumericValue')}}。
- {{domxref('CSSSkew.ay','ay')}}
  - : 要构造的 {{domxref('CSSSkew')}} 对象的 `ay`（y 轴）角度值。必须为 {{domxref('CSSNumericValue')}}。
