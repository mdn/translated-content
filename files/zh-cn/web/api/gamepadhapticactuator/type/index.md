{{domxref("GamepadHapticActuator")}} 接口的 **`type`** 只读属性返回一个表示触觉硬件类型的枚举值。

## 值

一个 [`GamepadHapticActuatorType`](https://w3c.github.io/gamepad/extensions.html#gamepadhapticactuatortype-enum) 类型的枚举，当前可用的类型是：

- `vibration` —— 振动硬件，产生隆隆声效果。
