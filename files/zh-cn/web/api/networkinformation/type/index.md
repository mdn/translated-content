{{domxref("NetworkInformation")}} 接口的 **`type`** 只读属性返回设备用于与网络通信的连接类型。

## 值

为以下值之一的枚举值：

- `"bluetooth"`
- `"cellular"`
- `"ethernet"`
- `"none"`
- `"wifi"`
- `"wimax"`
- `"other"`
- `"unknown"`
