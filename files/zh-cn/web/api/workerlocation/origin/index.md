{{ApiRef("WorkerLocation")}}{{AvailableInWorkers("worker")}}

{{domxref("WorkerLocation")}} 对象的 **`origin`** 属性返回该 Worker 所处位置的 {{domxref("URL.origin", "origin")}}。

## 取值

字符串。

## 示例

```js
// 在此页面上，返回 origin
const result = self.location.origin; // 返回“https://developer.mozilla.org:443”
```
