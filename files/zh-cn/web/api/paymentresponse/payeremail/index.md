{{securecontext_header}}{{APIRef("Payment Request API")}}

{{domxref("PaymentResponse")}} 接口的 `payerEmail` 只读属性返回用户提供的电子邮件地址。仅当在传递给 {{domxref('PaymentRequest.PaymentRequest','PaymentRequest')}} 构造函数的 `options` 对象中将 `requestPayerEmail` 选项设置为 `true` 时，该选项才会存在。

## 值

字符串。
