{{domxref("PaymentResponse")}} 接口的 `payerPhone` 只读属性返回用户提供的电话号码。仅当传递给 {{domxref('PaymentRequest.PaymentRequest','PaymentRequest')}} 构造函数的 `options` 对象中的 `requestPayerPhone` 选项设置为 `true` 时，该选项才会存在。

## 值

字符串。
