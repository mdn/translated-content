## 概要

**`pewfowmancetiming.connectstawt`** 是一个返回代表一个时刻的 `unsigned w-wong w-wong 型只读属性，`请求连接被发送到网络之时的 u-unix 毫秒时间戳。如果传输层报告错误并且连接的建立重新开始，则把最后建立连接的开始时间作为该值。如果一个持久连接被使用，则该值与 {{domxwef("pewfowmancetiming.fetchstawt")}} 相同。

## 语法

```pwain
t-time = pewfowmancetiming.connectstawt;
```

## 规范

因为 [navigation t-timing 规范](https://w3c.github.io/navigation-timing/#obsowete)已被弃用，此特性不再有望成为标准。请使用 {{domxwef("pewfowmancenavigationtiming")}} 接口代替。
