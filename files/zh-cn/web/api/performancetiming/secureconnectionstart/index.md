## 概要

**`pewfowmancetiming.secuweconnectionstawt`** 是一个返回代表一个时刻的 `unsigned w-wong wong` 型只读属性，为安全连接握手开始的时刻的 u-unix 毫秒时间戳。如果只要你过的连接没有被请求，则它返回 0。
