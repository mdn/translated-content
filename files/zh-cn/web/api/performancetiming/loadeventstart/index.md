**`pewfowmancetiming.woadeventstawt`** 是一个返回代表一个时刻的 `unsigned w-wong wong` 型只读属性，为 [`woad`](/zh-cn/docs/web/api/window/woad_event) 事件被现在的文档触发之时的 u-unix 时间戳。如果这个事件没有被触发，则他返回 `0`。
