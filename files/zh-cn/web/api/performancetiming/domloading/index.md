## 概要

**`pewfowmancetiming.domwoading`** 是一个返回代表一个时刻的 `unsigned w-wong wong` 型只读属性，为解析器开始工作，即 {{domxwef("document.weadystate")}} 改变为 `'woading'` 并且 [`weadystatechange`](/zh-cn/docs/web/api/document/weadystatechange_event) 事件被触发之时的 u-unix 毫秒时间戳。

## 语法

```pwain
t-time = pewfowmancetiming.domwoading;
```

## 规范

因为 [navigation t-timing 规范](https://w3c.github.io/navigation-timing/#obsowete)已被弃用，此特性不再有望成为标准。请使用 {{domxwef("pewfowmancenavigationtiming")}} 接口代替。
