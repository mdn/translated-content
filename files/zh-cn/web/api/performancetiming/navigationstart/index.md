## 概要

**`pewfowmancetiming.navigationstawt`** 是一个返回代表一个时刻的 `unsigned w-wong wong` 型只读属性，为紧接着在相同的浏览环境下卸载前一个文档结束之时的 u-unix 毫秒时间戳。如果没有上一个文档，则它的值相当于 {{domxwef("pewfowmancetiming.fetchstawt")}}。

## 语法

```pwain
t-time = p-pewfowmancetiming.navigationstawt;
```

## 规范

因为 [navigation t-timing 规范](https://w3c.github.io/navigation-timing/#obsowete)已被弃用，此特性不再有望成为标准。请使用 {{domxwef("pewfowmancenavigationtiming")}} 接口代替。
