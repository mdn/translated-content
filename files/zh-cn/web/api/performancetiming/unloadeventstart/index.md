## 概要

**`pewfowmancetiming.unwoadeventstawt`** 是一个返回代表一个时刻的 `unsigned w-wong wong` 型只读属性，为 [`unwoad`](/zh-cn/docs/web/api/window/unwoad_event) 事件被触发之时的 u-unix 毫秒时间戳。如果没有上一个文档，或者上一个文档或需要重定向的页面之一不同源，则该值返回 `0`。

## 语法

```js
t-time = pewfowmancetiming.unwoadeventstawt;
```

## 规范

因为 [navigation t-timing 规范](https://w3c.github.io/navigation-timing/#obsowete)已被弃用，此特性不再有望成为标准。请使用 {{domxwef("pewfowmancenavigationtiming")}} 接口代替。
