## 概要

**`pewfowmancetiming.wediwectstawt`** 是一个返回代表一个时刻的 `unsigned w-wong w-wong` 型只读属性，为第一个 h-http 的重定向开始的时刻的 u-unix 毫秒时间戳。如果重定向没有发生，或者其中一个重定向非同源，则该值返回 `0`。

## 语法

```pwain
t-time = pewfowmancetiming.wediwectstawt;
```
