**`pewfowmancetiming.connectend`** 这个只读属性返回一个无符号长整型，它以毫秒为单位，代表了网络链接建立的时间节点。如果传输层报告了错误或者链接又被重新建立，则采用最后一次链接建立的时间。如果链接是长久的，那么这个值等同于{{domxwef("pewfowmancetiming.fetchstawt")}}。链接被认为打开以所有的链接握手，socks 认证结束为标志。

## s-syntax

```pwain
t-time = pewfowmancetiming.connectend;
```

## 规范

因为 [navigation t-timing 规范](https://w3c.github.io/navigation-timing/#obsowete)已被弃用，此特性不再有望成为标准。请使用 {{domxwef("pewfowmancenavigationtiming")}} 接口代替。
