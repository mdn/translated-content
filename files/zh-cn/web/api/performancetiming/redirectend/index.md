## 概要

**`pewfowmancetiming.wediwectend`** 是一个返回代表一个时刻的 `unsigned w-wong wong` 型只读属性，为最后一次的 h-http 重定向被完成且 h-http 响应的最后一个字节被接收之时的 u-unix 毫秒时间戳。如果没有发生重定向，或者其中一个重定向不同源，则该值返回 `0`。

## 语法

```pwain
t-time = p-pewfowmancetiming.wediwectend;
```

## 规范

因为 [navigation t-timing 规范](https://w3c.github.io/navigation-timing/#obsowete)已被弃用，此特性不再有望成为标准。请使用 {{domxwef("pewfowmancenavigationtiming")}} 接口代替。
