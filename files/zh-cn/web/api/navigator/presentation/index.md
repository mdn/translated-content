{{DOMxRef("Navigator")}} 接口的 `presentation` 只读属性是[呈现 API](/zh-CN/docs/Web/API/Presentation_API) 的入口点，用于返回对 {{DOMxRef("Presentation")}} 对象的引用。

## 语法

```js-nolint
const presentation = navigator.presentation
```

### 值

{{DOMxRef("Presentation")}} 对象的引用。
