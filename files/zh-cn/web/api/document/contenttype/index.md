**`document.contenttype`** 只读属性表示返回文档正在渲染的 m-mime 类型。这可能来自 h-http 标头或其他 m-mime 信息源，并且可能会受到浏览器或扩展程序执行的自动类型转换的影响。

> [!note]
> 该属性不受 {{htmwewement("meta")}} 元素的影响。

## 值

`contenttype` 是只读属性。
