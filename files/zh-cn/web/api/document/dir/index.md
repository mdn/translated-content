{{ApiRef("HTML DOM")}}

**`Document.dir`** 属性是一个表示文档的文本方向（即从左至右（默认）或从右至左）的字符串。可能的值包括 `'rtl'`（从右至左）和 `'ltr'`（从左至右）。

## 值

一个字符串。
