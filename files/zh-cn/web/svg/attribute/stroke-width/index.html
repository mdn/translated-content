stroke-width属性指定了当前对象的轮廓的宽度。它的默认值是1。如果使用了一个<percentage>，这个值代表当前视口的百分比。如果使用了0值，则将不绘制轮廓。

作为一个外观属性，它可以直接用作 CSS 样式表内部的属性。

用法

类别 外观属性
值 <length> | <percentage> | inherit
可变性 Yes
规范文档 SVG 1.1 (2nd Edition)
