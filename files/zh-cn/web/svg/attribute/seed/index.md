`seed`属性代表了 {{SVGElement("feTurbulence")}} 元素生成的伪随机数的开始数字。

如果没有指定这个属性，它等效于指定了一个 `0` 值。

## 使用说明

| 类别   | _None_                                           |
| ------ | ------------------------------------------------ |
| 值     | [\<number>](/zh-CN/docs/SVG/Content_type#Number) |
| 可变性 | Yes                                              |

## 元素
