{{SVGRef}}

该属性指定了填色的不透明度或当前对象的内容物的不透明度。

## 使用说明

| 类别     | 外观属性                                                                            |
| -------- | ----------------------------------------------------------------------------------- |
| 值       | [\<opacity-value>](/zh-CN/SVG/Content_type#Opacity_value) \| inherit                |
| 初始值   | 1                                                                                   |
| 可变性   | Yes                                                                                 |
| 规范文档 | [SVG 1.1 (2nd Edition)](http://www.w3.org/TR/SVG/painting.html#FillOpacityProperty) |
