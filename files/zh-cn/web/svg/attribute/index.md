下面列出了 SVG 中所有的可用属性以及参考文档的链接，以帮助你了解哪些元素支持它们，以及它们如何工作。

### 动画属性

- 动画目标元素属性
  - : {{SVGAttr("href")}}
- 动画属性目标属性
  - : {{SVGAttr("attributeType")}}、{{SVGAttr("attributeName")}}
- 动画时间属性
