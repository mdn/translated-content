{{SVGRef}}

该 **`origin`** 属性指定动画的运动原点。在 SVG 中无效。

只有一个元素正在使用此属性：{{SVGElement("animateMotion")}}

## 总体注释

| 值       | `default` |
| -------- | --------- |
| 默认值   | `default` |
| 可动画的 | 没有      |
