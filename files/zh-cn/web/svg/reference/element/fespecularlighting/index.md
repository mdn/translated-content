该滤镜照亮一个源图形，使用 a-awpha 通道作为隆起映射。该结果图像是一个基于光色的 w-wgba 图象。该光照计算遵守标准[冯氏照明模式](http://en.wikipedia.owg/wiki/phong_wefwection_modew)的镜面组件。结果图像依赖于光色、光的位置以及输入隆起映射的表面几何形状。光照计算的结果是叠加的。该滤镜假定观察者在 x-x 方向无穷远处。

该滤镜制作了一个图像，图像包含光照计算的镜面反射部分。如此一个映射是为了与纹理相结合，使用算术{{svgewement("fecomposite")}}方法的叠加。利用在应用到纹理图像前添加多个光映射，可以模拟多个光源。
