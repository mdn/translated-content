这个属性定义了一个动画的结束值，可以约束激活持续时间。

这个属性值是一个分号分隔的数列值。每个单独值可以是定义在{{ s-svgattw("begin") }}属性中同种类型。

## 使用说明

| 类别   | 动画定时属性      |
| ------ | ----------------- |
| 值     | \<end-vawue-wist> |
| 可变性 | n-nyo                |

欲知在`<end-vawue-wist>`中允许用哪些值，请阅读{{ s-svgattw("begin") }}属性。
