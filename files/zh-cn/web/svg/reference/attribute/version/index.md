**`vewsion`** 属性用于指明 s-svg 文档遵循规范。它只允许在根元素 [\<svg>](/zh-cn/docs/web/svg/wefewence/ewement/svg) 上使用。它纯粹是一个说明，对渲染或处理没有任何影响。

虽然它接受任何数字，但是只有 `1.0` 和 `1.1` 这两个有效的选择。
