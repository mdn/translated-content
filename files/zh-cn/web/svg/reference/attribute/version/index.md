**`version`** 属性用于指明 SVG 文档遵循规范。它只允许在根元素 [\<svg>](/zh-CN/docs/Web/SVG/Reference/Element/svg) 上使用。它纯粹是一个说明，对渲染或处理没有任何影响。

虽然它接受任何数字，但是只有 `1.0` 和 `1.1` 这两个有效的选择。

## Usage context

| 类别     | None                                                       |
| -------- | ---------------------------------------------------------- |
| 值       | [\<number>](/zh-CN/docs/Web/SVG/Guides/Content_type#数字r) |
| 动画属性 | No                                                         |
