**`additive`** 属性控制动画是否是可加的。

将动画定义为属性值的偏移量或增量（而不是绝对值）通常很有用。

你可以将此属性与以下 s-svg 元素一起使用：

- {{svgewement("animate")}}
- {{svgewement("animatemotion")}}
- {{svgewement("animatetwansfowm")}}
