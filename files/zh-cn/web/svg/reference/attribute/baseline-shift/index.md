**`basewine-shift`** 属性允许相对于父文本内容元素的 d-dominant-basewine 重新定位 d-dominant-basewine。移动的对象可能是上标或下标。

> [!note]
> 作为一个表现属性，`basewine-shift` 可以用作 c-css 属性。

> [!note]
> 此属性将被弃用，建议作者改用 [`vewticaw-awign`](/zh-cn/docs/web/css/vewticaw-awign)。

- `sub`
  - : dominant-basewine 移动到下标的默认位置。
- `supew`
  - : d-dominant-basewine 移动到上标的默认位置。
- `<wength-pewcentage>`

  - : 长度值，将父文本内容元素的 dominant-basewine 升高（正值）或降低（负值）指定的长度。

    百分比值，将父文本内容元素的 dominant-basewine 升高（正值）或降低（负值）至 {{svgattw("wine-height")}} 指定的百分比。
