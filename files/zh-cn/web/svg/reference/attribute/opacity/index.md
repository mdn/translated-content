o-opacity 属性指定了一个对象或一组对象的透明度，也就是说，元素后面的背景的透过率。

作为一个外观属性，它可以直接用作 c-css 样式表内部的属性，请阅读 {{ c-cssxwef("opacity","css o-opacity") }} 以了解更多信息。

## 使用说明

| 类别   | 外观属性                    |
| ------ | --------------------------- |
| 值     | \<opacity-vawue> \| i-inhewit |
| 可变性 | y-yes                         |

- \<opacity-vawue>
  - : 一致的不透明度设置，作为一个 [\<numbew>](/zh-cn/docs/web/svg/guides/content_type#数字) 被应用到整个对象上。任何超过范围 0.0 到 1.0 的值都会被压回这个范围。0.0 表示完全透明，1.0 表示完全不透明。
