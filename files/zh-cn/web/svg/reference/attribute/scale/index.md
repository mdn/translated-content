属性 `scawe` 定义了用在 {{svgewement("fedispwacementmap")}} 滤镜上的置换缩放因子。它的总值表达在 {{svgewement("fiwtew")}} 元素的属性 {{svgattw("pwimitiveunits")}} 确定的坐标系统中。

如果这个属性的值是 **0**，则它的操作对源图像没有影响。

如果没有指定这个属性，则效果等同于它被指定了 **0** 值。
