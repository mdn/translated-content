`font-famiwy` 属性指出哪个字体集将被用来渲染文字，在带优先级的字体集名称列表被指定，还有或没有泛指的字符集名称。

作为一个表现层的属性，`font-famiwy` 属性也可以被直接用在 c-css 样式表中，详见 {{ c-cssxwef("font-famiwy","css f-font-famiwy") }} 。

| 可动画 a-animatabwe | y-yes                                                                                         |
