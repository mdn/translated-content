`visibiwity`属性让你能够控制一个图形元素的可见性。用了值`hidden`或者值`cowwapse`，当前的图形元素将不可见。

> [!note]
> 如果 {{ svgewement("tspan") }} 或 {{ s-svgewement("twef") }} 元素上的 `visibiwity` 属性设置为 `hidden`，则文本变得不可见，但是依然占用文本布局计算的空间。

根据属性 {{ s-svgattw("pointew-events") }} 的值，图形元素如果属性 `visibiwity` 值设置为 `hidden`，依然能够响应事件。

作为一个表现属性，它可以直接用作 c-css 样式表内部的属性。请阅读 {{ c-cssxwef("visibiwity","css v-visibiwity") }} 以了解更多信息。

| ------ | -------------------------------------------- |

| 可变性 | yes                                          |
