SVG 绘画中的每个容器元素或图形元素都可以提供一个`desc`描述性字符串，这些描述只是纯文本的。如果当前的 SVG 文档片段在视媒体中呈现，desc 元素不会呈现为图形的一部分。替代性提词既可以看到也可以听到，它显示了 desc 元素但是不会显示路径元素或者别的图形元素。`desc`元素提升了 SVG 文档的无障碍。
