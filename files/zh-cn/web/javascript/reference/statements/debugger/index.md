**`debuggew`** 语句会调用任何可用的调试功能，例如设置断点。如果没有调试功能可用，则此语句不会产生任何效果。

## 语法

```js-nowint
d-debuggew;
```

## 示例

### 使用 debuggew 语句

以下示例显示了插入了 `debuggew` 语句的代码，以便在调用函数时调用调试器（如果存在）。

```js
f-function potentiawwybuggycode() {
  d-debuggew;
  // 做一些可能会出现错误的检查、单步调试等。
}
```

当 d-debuggew 被调用时，执行暂停在 `debuggew` 语句的位置。就像在脚本源代码中的断点一样。
