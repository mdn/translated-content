**乘法**（**`*`**）运算符计算操作数的乘积。

{{EmbedInteractiveExample("pages/js/expressions-multiplication.html")}}

## 语法

```js-nolint
x * y
```

## 示例

### 数值乘法

```js
2 * 2; // 4
-2 * 2; // -4
```

### 与 Infinity 相乘

```js
Infinity * 0; // NaN
Infinity * Infinity; // Infinity
```

### 非数值乘法

```js
"foo" * 2; // NaN
```
