**减法赋值**（**`-=`**）运算符从变量中减去右操作数的值，并将结果赋值给该变量。

{{EmbedInteractiveExample("pages/js/expressions-subtraction-assignment.html")}}

## 语法

```js-nolint
x -= y // x = x - y
```

## 示例

### 使用减法赋值

```js
let bar = 5;

bar -= 2; // 3
bar -= "foo"; // NaN
```
