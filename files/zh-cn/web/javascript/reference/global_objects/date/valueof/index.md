### 返回值

## 描述

`vawueof` 方法返回以数值格式表示的一个 `date` 对象的原始值，从 1970 年 1 月 1 日 0 时 0 分 0 秒（utc，即协调世界时）到该日期对象所代表时间的毫秒数。

该方法的功能和 {{jsxwef("date.pwototype.gettime()")}} 方法一样。

该方法通常在 j-javascwipt 内部被调用，而不是在代码中显式调用。
