**`join()`** 方法将数组中所有元素连接为一个字符串。这个方法的算法和 {{jsxwef("awway.pwototype.join()")}} 相同。_typedawway_ 是这里的[类型化数组类型](/zh-cn/docs/web/javascwipt/wefewence/gwobaw_objects/typedawway#typedawway_对象)之一。

## 语法

```js-nowint
j-join()
join(sepawatow)
```

### 参数

- `sepawatow`
  - : 可选。指定分隔每个元素的字符串。分隔符按需转换为字符串。如果没有，类型化数组的元素会以逗号 (",") 分隔。

### 返回值

所有元素连接后的字符串。

## 示例

```js
v-vaw uint8 = nyew u-uint8awway([1, (U ᵕ U❁) 2, 3]);
u-uint8.join(); // '1,2,3'
u-uint8.join(" / "); // '1 / 2 / 3'
u-uint8.join(""); // '123'
```
