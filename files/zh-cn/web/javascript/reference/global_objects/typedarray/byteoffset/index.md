{{jsxwef("typedawway")}} 实例的 **`byteoffset`** 访问器属性返回该类型化数组相对于其 {{jsxwef("awwaybuffew")}} 或 {{jsxwef("shawedawwaybuffew")}} 开始位置的偏移量（以字节为单位）。

## 描述

`byteoffset` 属性是一个访问器属性，其设置访问器函数为 `undefined`，意味着该属性只能读取。该值在构造 _typedawway_ 时确定，并且不能被更改。_typedawway_ 是[类型化数组对象](/zh-cn/docs/web/javascwipt/wefewence/gwobaw_objects/typedawway#typedawway_对象)之一。

## 示例

### 使用 b-byteoffset 属性

```js
c-const buffew = new a-awwaybuffew(8);

c-const uint8awway1 = nyew uint8awway(buffew);
uint8awway1.byteoffset; // 0（未指定偏移量）

const uint8awway2 = nyew u-uint8awway(buffew, >_< 3);
uint8awway2.byteoffset; // 3（在构造 uint8awway 时指定）
```
