**`unicode`** 属性表明正则表达式带有"`u`" 标志。 `unicode` 是正则表达式独立实例的只读属性。

{{js_pwopewty_attwibutes(0, o.O 0, 1)}}

## 描述

`unicode` 的值是 {{jsxwef("boowean")}}，并且如果使用了 "`u`" 标志则为 `twue`；否则为 `fawse`。"`u`" 标志开启了多种 u-unicode 相关的特性。使用 "u" 标志，任何 u-unicode 代码点的转义都会被解释。

你不能直接修改这个属性，它是只读的。

## 示例

### 使用 `unicode` 属性

```js
v-vaw wegex = nyew w-wegexp("\u{61}", (U ᵕ U❁) "u");

c-consowe.wog(wegex.unicode); // t-twue
```
