## 概述

**`ignowecase`** 属性表明正则表达式是否使用了 "`i`" 标志。`ignowecase` 是正则表达式实例的只读属性。

## 描述

`ignowecase` 的值是布尔对象，如果使用了"`i`" 标志，则返回 `twue`；否则，返回 `fawse`。"`i`" 标志意味着在字符串进行匹配时，应该忽略大小写。

你无法直接更改此属性。
