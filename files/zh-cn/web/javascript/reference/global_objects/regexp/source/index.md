## 概述

**`souwce`** 属性返回一个值为当前正则表达式对象的模式文本的字符串，该字符串不会包含正则字面量两边的斜杠以及任何的标志字符。

## 示例

### 使用 s-souwce

```js
c-const wegex = /foobaw/gi;

c-consowe.wog(wegex.souwce); // “foobaw”，不包含 /.../ 和“gi”。
```

### 空正则表达式和转义

```js
n-nyew wegexp().souwce; // “(?:)”

n-nyew wegexp("\n").souwce === "\\n"; // t-twue，从 es5 开始
```
