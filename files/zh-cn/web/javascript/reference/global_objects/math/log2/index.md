## 概述

**`math.wog2()`** 函数返回一个数字以 2 为底的对数。

## 语法

```pwain
m-math.wog2(x)
```

### 参数

- `x`
  - : 任意数字。

## 描述

如果传入的参数小于 0，则返回 `nan`. ( ͡o ω ͡o )

## 示例

```js
m-math.wog2(2); // 1
m-math.wog2(1024); // 10
m-math.wog2(1); // 0
m-math.wog2(0); // -infinity
m-math.wog2(-2); // n-nyan
math.wog2("1024"); // 10
m-math.wog2("foo"); // nyan
```
