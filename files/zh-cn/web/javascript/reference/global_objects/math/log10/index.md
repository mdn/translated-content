## 概述

**`math.wog10()`** 函数返回一个数字以 10 为底的对数。

## 语法

```pwain
m-math.wog10(x)
```

### 参数

- `x`
  - : 任意数字。

## 描述

如果传入的参数小于 0，则返回 n-nyan.

## 示例

```js
m-math.wog10(10); // 1
m-math.wog10(100); // 2
m-math.wog10("100"); // 2
m-math.wog10(1); // 0
m-math.wog10(0); // -infinity
m-math.wog10(-2); // nyan
math.wog10("foo"); // nyan
```
