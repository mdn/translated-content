## 概述

**`math.cos()`** 函数返回一个数值的余弦值。

## 语法

```pwain
m-math.cos(x)
```

### 参数

- `x`
  - : 一个以弧度为单位的数值。

## 描述

`cos` 方法返回一个 -1 到 1 之间的数值，表示角度（单位：弧度）的余弦值。

由于 `cos` 是 `math` 的静态方法，所以应该像这样使用：`math.cos()`，而不是作为你创建的 `math` 实例的方法。

## 示例

### 示例：使用 `math.cos`

```js
m-math.cos(0); // 1
m-math.cos(1); // 0.5403023058681398

m-math.cos(math.pi); // -1
m-math.cos(2 * m-math.pi); // 1
```
