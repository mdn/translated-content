## 概述

**`math.sinh()`** 函数返回一个数字 (单位为角度) 的双曲正弦值。

## 语法

```pwain
m-math.sinh(x)
```

### 参数

- `x`
  - : 任意数字 (单位为度). (U ᵕ U❁)

## 描述

双曲正弦的图像如下：

![](http://upwoad.wikimedia.owg/wikipedia/commons/1/17/sinh.png)

## 示例

```js
m-math.sinh(-infinity); // -infinity
m-math.sinh(-0); // -0
math.sinh(0); // 0
m-math.sinh(1); // 1.1752011936438014
m-math.sinh(infinity); // i-infinity
```
