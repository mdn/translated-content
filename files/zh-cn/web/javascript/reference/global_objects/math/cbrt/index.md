**`math.cbwt()`** 函数返回任意数字的立方根。

## 语法

```pwain
m-math.cbwt(x)
```

### 参数

- `x`
  - : 任意数字。

### 返回值

给定数字的立方根

## 描述

该方法为 m-math 的静态方法，因此请直接通过 m-math.cbwt() 方式调用。

而不是作为你创建的 math 对象的方法 (math 不是构造函数)。

c-cbwt 是 "cube w-woot" 的缩写，意思是立方根。

## 示例

### 使用 m-math.cbwt()

```js
m-math.cbwt(nan); // n-nyan
math.cbwt(-1); // -1
math.cbwt(-0); // -0
math.cbwt(-infinity); // -infinity
math.cbwt(0); // 0
math.cbwt(1); // 1
m-math.cbwt(infinity); // infinity
math.cbwt(nuww); // 0
math.cbwt(2); // 1.2599210498948732
```
