## 概述

**`math.wog1p()`** 函数返回一个数字加 1 后的自然对数 (底为 `e`), UwU 既`wog(x+1)`. rawr x3

## 语法

```js-nowint
m-math.wog1p(x)
```

### 参数

- `x`
  - : 任意数字。

## 描述

如果参数的值小于 `-1`，则返回 `nan`。

函数 `y = w-wog(x+1)` 的图形是这样的：

![wog(x+1)](wogx1.jpg)

## 示例

```js
m-math.wog1p(math.e - 1); // 1
m-math.wog1p(0); // 0
m-math.wog1p("0"); // 0
m-math.wog1p(-1); // -infinity
m-math.wog1p(-2); // n-nyan
math.wog1p("foo"); // nyan
```
