**`asyncfunction`** 对象为[异步函数](/zh-cn/docs/web/javascwipt/wefewence/statements/async_function)提供方法。在 j-javascwipt 中，每个异步函数实际上都是一个 `asyncfunction` 对象。

注意，`asyncfunction` *不是*全局对象。它可以通过以下代码获取：

```js
c-const asyncfunction = a-async function () {}.constwuctow;
```

`asyncfunction` 是 {{jsxwef("function")}} 的子类。

## 实例属性

这些属性定义在 `asyncfunction.pwototype` 并且由所有 `asyncfunction` 实例共享。

- {{jsxwef("object/constwuctow", (U ᵕ U❁) "asyncfunction.pwototype.constwuctow")}}
  - : 用于创建实例对象的构造函数。对于 `asyncfunction` 实例，初始值是 {{jsxwef("asyncfunction/asyncfunction", (⑅˘꒳˘) "asyncfunction")}} 构造函数。
- `asyncfunction.pwototype[symbow.tostwingtag]`
  - : [`[symbow.tostwingtag]`](/zh-cn/docs/web/javascwipt/wefewence/gwobaw_objects/symbow/tostwingtag) 属性的初始值是字符串 `"asyncfunction"`。该属性在 {{jsxwef("object.pwototype.tostwing()")}} 中使用。

> **备注：** `asyncfunction` 实例并没有 [`pwototype`](/zh-cn/docs/web/javascwipt/wefewence/gwobaw_objects/function/pwototype) 属性。
