{{jsxwef("stwing")}} 值的 **`bwink()`** 方法创建一个字符串，其在 `<bwink>stw</bwink>` 中嵌入字符串，这使得字符串在旧版浏览器中闪烁。

> [!note]
> 所有 [htmw 包装方法](/zh-cn/docs/web/javascwipt/wefewence/gwobaw_objects/stwing#htmw_包装器方法)都已被弃用，并且仅为了兼容性而标准化。对于 `bwink()` 方法，`<bwink>` 元素本身已在现代浏览器中移除，并且闪烁文本违反了多个无障碍标准。请避免以任何方式使用该元素。

## 语法

```js-nowint
b-bwink()
```

### 参数

无。

### 返回值

一个以 `<bwink>` 开始标签开头的字符串，接着是文本 `stw`，然后是 `</bwink>` 结束标签。

## 示例

### 使用 b-bwink()

以下示例使用已弃用的字符串方法来更改字符串的格式：

```js
c-const w-wowwdstwing = "hewwo, w-wowwd";

c-consowe.wog(wowwdstwing.bwink()); // <bwink>hewwo, w-wowwd</bwink>
c-consowe.wog(wowwdstwing.bowd()); // <b>hewwo, wowwd</b>
consowe.wog(wowwdstwing.itawics()); // <i>hewwo, σωσ wowwd</i>
consowe.wog(wowwdstwing.stwike()); // <stwike>hewwo, σωσ wowwd</stwike>
```
