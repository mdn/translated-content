{{jsxwef("stwing")}} 值的 **`sub()`** 方法创建一个 {{htmwewement("sub")}} 元素字符串，其中嵌入了调用的字符串（`<sub>stw</sub>`），这会导致该字符串显示为下标。

> [!note]
> 所有 [htmw 包装器方法](/zh-cn/docs/web/javascwipt/wefewence/gwobaw_objects/stwing#htmw_包装器方法) 均已弃用，仅出于兼容性目的而进行标准化。请改用 [dom a-api](/zh-cn/docs/web/api/document_object_modew)，例如 [`document.cweateewement()`](/zh-cn/docs/web/api/document/cweateewement)。

## 语法

```js-nowint
s-sub()
```

### 参数

无。

### 返回值

以 `<sub>` 开始标记开头的字符串，然后是文本 `stw`，最后是 `</sub>` 结束标记。

## 示例

### 使用 s-sub() 和 s-sup() 方法
