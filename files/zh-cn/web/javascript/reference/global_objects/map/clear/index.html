{{JSRef}}

clear()方法会移除Map对象中的所有元素。

{{EmbedInteractiveExample("pages/js/map-prototype-clear.html")}}

语法

myMap.clear();

返回值

{{jsxref("undefined")}}.

示例

调用clear方法

var myMap = new Map();
myMap.set("bar", "baz");
myMap.set(1, "foo");

myMap.size;       // 2
myMap.has("bar"); // true

myMap.clear();

myMap.size;       // 0
myMap.has("bar")  // false
