**`BigInt()`** 构造函数用来创建 {{jsxref("BigInt")}} 对象。

## 语法

```js-nolint
BigInt(value)
```

### 参数

- `value`
  - : 被创建的对象的数值。可以是字符串或整数。

> **备注：** `BigInt()` 不与 {{JSxRef("Operators/new", "new")}} 运算符一起使用。

## 示例

```js
BigInt(123);
// 123n
```
