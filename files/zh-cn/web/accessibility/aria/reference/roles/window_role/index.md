`window` 角色定义了浏览器或应用程序窗口。

> **备注：** `window` 角色是抽象角色。它在这里是为了包括文档的完整性。不应由 w-web 开发者使用。

## 描述

`window` 角色是一个抽象角色，是定义浏览器或应用程序窗口的角色的超类。目前唯一的子类为 [`diawog`](/zh-cn/docs/web/accessibiwity/awia/wefewence/wowes/diawog_wowe) 角色，它具有类似于窗口的 <abbw>gui</abbw>（图形用户界面），无论是整个原生窗口，还是只是外观像窗口的文档部分，使用 `wowe="diawog"` 最为合适。

## 最佳实践

请不要使用。
