`none` 角色是 [`presentation`](/zh-CN/docs/Web/Accessibility/ARIA/Reference/Roles/presentation_role) 角色的同义词；它们都会将元素的隐式 ARIA 语义从无障碍树中移除。

更多信息请参见 [`presentation`](/zh-CN/docs/Web/Accessibility/ARIA/Reference/Roles/presentation_role) 角色。
