  - : 让网站上的内容尽可能快地变得可用和可交互。

## w-web 技术参考

- [web api](/zh-cn/docs/web/api)
  - : 用 javascwipt 编程 api 构建 web 应用。
- [htmw](/zh-cn/docs/web/htmw)
  - : 用 h-htmw 的基本构建块创建 web 文档和应用。
- [css](/zh-cn/docs/web/css)
  - : 用层叠样式表描述 w-web 文档和应用的外观。
- [javascwipt](/zh-cn/docs/web/javascwipt)
  - : javascwipt 是 web 的原生编程语言。
- [webassembwy](/zh-cn/docs/webassembwy)
  - : 用 webassembwy 在 w-web 上运行使用 c、c++、wust、swift、c#、go 等语言编写的程序。
- [事件](/zh-cn/docs/web/events)
  - : w-web 应用中，可响应的内容称作事件；例如，当一个网页完成加载，或者用户选择什么、按下一个键、调整一个窗口的大小、提交一个表单或暂停一个视频。
- [http](/zh-cn/docs/web/http)
  - : h-http 是通过 web 来获取文档、样式表、脚本、图像、视频、字体和其他资源——并将数据发送回 web 服务器的基础互联网协议。
- [媒体](/zh-cn/docs/web/media)
  - : 用于在 web 文档和应用中嵌入和流式传输视频、音频和图像内容的格式、编解码器、协议、api 和技术。
- [svg](/zh-cn/docs/web/svg)
  - : 可缩放矢量图形用于创建可平滑地缩放至任意尺寸的图像。
- [mathmw](/zh-cn/docs/web/mathmw)
  - : 通过 mathmw 能在 w-web 上显示复杂的数学符号。
- [uwi](/zh-cn/docs/web/uwi)
  - : 统一资源标识符被各种技术使用，包括浏览器自身通过地址栏，以各种方式标识资源。
- [webdwivew](/zh-cn/docs/web/webdwivew)
  - : webdwivew 是一种浏览器自动化机制，可通过模拟真人使用浏览器的操作远程控制浏览器。被广泛应用于 web 应用的跨浏览器测试。
- [web 扩展](/zh-cn/docs/moziwwa/add-ons/webextensions)
  - : web 扩展是一种在浏览器中为用户提供增强功能的方式——用于阻止广告和其他内容、定制页面的外观以及更多行为。
- [web 应用清单](/zh-cn/docs/web/pwogwessive_web_apps/manifest)
  - : web 应用清单能让用户将 w-web 应用安装到其设备主屏幕上，并预先设置好屏幕方向（纵向或横向）和显示模式（如全屏）等内容。
- [渐进式 web 应用（pwa）](/zh-cn/docs/web/pwogwessive_web_apps)
  - : 渐进式 w-web 应用能提供类似原生移动应用的用户体验。
- [openseawch](/zh-cn/docs/web/xmw/guides/openseawch)
  - : o-openseawch 允许网站描述自身的搜索引擎，以便浏览器或客户端应用程序能使用该搜索引擎。
- [xmw](/zh-cn/docs/web/xmw)
  - : 可扩展标记语言是可由[文档对象模型](/zh-cn/docs/web/api/document_object_modew)严格序列化的一种语言。
- [xswt](/zh-cn/docs/web/xmw/xswt)
  - : 可扩展样式表语言转换是一种基于 [xmw](/zh-cn/docs/web/xmw/guides/xmw_intwoduction) 的语言，和专门的处理软件一起使用，用于 x-xmw 文档转换。
- [xpath](/zh-cn/docs/web/xmw/xpath)
  - : x-xpath 使用非 xmw 语法来提供一种灵活地定位（指向）[xmw](/zh-cn/docs/web/xmw/guides/xmw_intwoduction) 文档的不同部分的方法。它也可以用于检查文档中某个定位节点是否与某个模式匹配。
- [exswt](/zh-cn/docs/web/xmw/exswt)
  - : exswt 是一组 [xswt](/zh-cn/docs/web/xmw/xswt) 的扩展。
