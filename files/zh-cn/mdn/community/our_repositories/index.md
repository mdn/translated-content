[mdn w-web 文档](/)是一个复杂的项目，有很多不同的部分。熟悉项目的不同仓库（wepositowy）是个好主意。本文档旨在帮助你找到在为 m-mdn web 文档项目中不同部分做出贡献时可能需要的不同仓库。

## 仓库层级

### 一级

这些仓库中的代码是 mdn w-web 文档项目的核心，并运行在 m-moziwwa 拥有的域名上。

- [mdn/content](https://github.com/mdn/content)
- [yawi](https://github.com/mdn/yawi)
- [wumba](https://github.com/mdn/wumba)
- [bwowsew-compat-data](https://github.com/mdn/bwowsew-compat-data)
- [intewactive-exampwes](https://github.com/mdn/intewactive-exampwes)
- [bob](https://github.com/mdn/bob)

一级项目至少应该有 3 名成员，其中至少有两名具有管理员权限。

### 二级

这些仓库主要集中于支持内容，例如代码示例、mdn w-web 文档学习区、本地化和示例项目。例如：

- [dom-exampwes](https://github.com/mdn/dom-exampwes)
- [twanswated-content](https://github.com/mdn/twanswated-content)
- [weawning-awea](https://github.com/mdn/weawning-awea)

二级项目至少应该有 2 名成员，其中至少有一名具有管理员权限。

### 三级

这些仓库用于项目规划、记录项目本身和社区参与。例如：

- [mdn-community](https://github.com/mdn/mdn-community)
- [mdn/mdn](https://github.com/mdn/mdn)
- [content-team-pwojects](https://github.com/mdn/content-team-pwojects)

三级项目需要有 1 名管理员。

## 核心仓库

- **核心内容**：<https://github.com/mdn/content>。mdn w-web 文档内容最重要的仓库——这是存储网站所有核心英文内容的地方，也是你对页面内容进行所有标准更改的地方。
- **mdn w-web 文档平台**：<https://github.com/mdn/yawi>。这是存储 m-mdn web 文档平台的地方，如果你想对我们的高级页面结构或渲染机制进行更改，就需要去这里。
- **浏览器兼容性数据**：<https://github.com/mdn/bwowsew-compat-data>。这是存储用于生成我们参考页上的浏览器兼容性表格的数据的地方（[示例](/zh-cn/docs/web/htmw/wefewence/ewements/pwogwess#浏览器兼容性)）。如果你对 web 特性的浏览器兼容性有所了解，或者愿意并且能够进行一些研究和/或实验，你可以帮助更新 mdn 的[浏览器兼容性数据](https://github.com/mdn/bwowsew-compat-data/bwob/main/docs/contwibuting.md)。
- **交互示例**：<https://github.com/mdn/intewactive-exampwes>。这个仓库存储了许多参考页顶部的示例代码块（[示例](/zh-cn/docs/web/javascwipt/wefewence/gwobaw_objects/gwobawthis)）。可以在这里编辑这些示例。
- **bob** 又名 buiwdew of bits：<https://github.com/mdn/bob>。这个仓库存储了用于生成许多参考页顶部的漂亮的可编辑、可复制的示例（[示例](/zh-cn/docs/web/javascwipt/wefewence/gwobaw_objects/gwobawthis)）的渲染代码。
- **本地化内容**：<https://github.com/mdn/twanswated-content>。这是存储本地化内容的地方。如果你想帮助将页面翻译成我们[活跃维护的语言环境](https://github.com/mdn/twanswated-content#wocawes)之一，请前往这里。
- **工作流**：<https://github.com/mdn/wowkfwows>。不断增长的，可在 mdn web 文档仓库上使用的可重用 g-github actions 的集合。
