**Bun** 是一个快速且现代的 {{Glossary("JavaScript")}} 运行时环境，专注于性能和开发者工具。

与 [Node.js](https://nodejs.org/) 和 [Deno](https://deno.com/) 不同，Bun 基于 Apple 的 [JavaScriptCore](https://trac.webkit.org/wiki/JavaScriptCore) 构建。Bun 的主要特性包括：

- 可直接替代 Node.js：几乎所有 Node.js 程序都能用 Bun 的等效命令运行。
- 在大多数真实场景下拥有业界领先的 JavaScript 执行速度。
- 原生支持 JSX 和 TypeScript 等 JavaScript 语法扩展。
- 全局缓存管理包安装，最大限度减少磁盘占用。
- 内置打包器、转译器、兼容 [Jest](https://jestjs.io/) 的测试运行器、包管理器等多种开发者常用工具。
