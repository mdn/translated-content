**文档对象模型**（Document Object Model，DOM）是一种用于表示任何基于 {{glossary("HTML")}} 或 {{glossary("XML")}} 的标记语言文档并能与之交互的 {{glossary("API")}}。它是加载在{{glossary("browser", "浏览器")}}中的文档模型，可以将文档表示为{{Glossary("Node/DOM","节点")}}树，或称 **DOM 树**，其中每个节点代表文档的一部分（例如，{{Glossary("element", "元素")}}、文本字符串或注释）。

DOM 是 {{glossary("World Wide Web","Web")}} 上最常用的 {{Glossary("API")}} 之一，因为它允许在浏览器中运行的代码访问文档中的每个节点并与之交互。可以创建、移动和更改节点，还可以将事件监听器添加到节点，并在特定事件发生时触发。
