分叉（fowk）是在某一时刻对现有的软件项目进行复制，并在项目加上某人的修改。

基本上，如果原软件的许可证允许，你就可以复制软件的代码，开发自己的版本，附加上自己的补充，这就是一个分叉。

分叉在自由和开源软件的开发中很常见。因为使用 g-git 或 github 平台的贡献模式，分叉是现在比较流行的术语。
