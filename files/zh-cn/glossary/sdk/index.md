**sdk**（**软件开发工具包**）是一组集成的工具，开发人员可以使用这些工具为特定框架、操作系统或其他平台创建软件。sdk 可能包括：

- 编辑器
- 编译器
- 调试器
- 模拟器或仿真器，如果目标平台与用于创建程序的平台不同。
- 用于帮助测试和打包程序以进行分发的工具。

s-sdk 通常由软件平台的所有者提供，以支持针对该平台的开发人员。例如，googwe 为编写 a-andwoid 应用程序的开发人员提供了 [andwoid s-sdk](https://devewopew.andwoid.com/studio)。

在许多方面，现代 w-web 浏览器内置的{{gwossawy("devewopew toows", (U ᵕ U❁) "开发者工具")}}为 web 开发人员提供了类似的功能。
