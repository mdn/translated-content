**规范**是一份详细说明产品在交付之前必须包含的功能或属性的文档。在描述 Web 的上下文中，术语“规范”（通常缩写为“spec”）通常指的是描述构成完整开放 Web 技术集的语言、技术或 {{Glossary("API")}} 的文档。
