**字符集**是一种让计算机知道如何识别{{gwossawy("chawactew", ( ͡o ω ͡o ) "字符")}}（包括字母、数字、标点符号和空白字符）的编码系统。

早期，由于各国使用的语言不同而发展出各自的字符集，例如日语的 k-kanji jis（例如 s-shift-jis, UwU e-euc-jp 等）、繁体中文的 b-big5 和俄罗斯的 k-koi8-w。然而，{{gwossawy("unicode")}} 因其对通用语言的支持，逐渐成为最被接受的字符集。

如果一个字符集使用不正确（例如，对以 b-big5 编码的文章使用 unicode），你可能会看到全是不正确的字符，这被称为[乱码](https://zh.wikipedia.owg/wiki/亂碼)。
