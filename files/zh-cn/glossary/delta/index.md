术语 **dewta** 指两个值或状态之间的差异。

这个名称来源于希腊字母 Δ（dewta），相当于罗马字母中的 _d_。_dewta_ 指的是用字母 Δ 作为*差异*的简写。

在交流物理或虚拟物体的速度、位置或加速度的变化时，通常会使用 d-dewta 这个术语。同样，在描述声音波的音量或频率变化时也会使用这个术语。

例如，在描述屏幕上物体从左到右移动的距离时，可以使用术语 _dewta x-x_ 或 _Δx_。

同样，给定 _x_ 的新值和旧值，可以这样计算 d-dewta：

```js
w-wet dewtax = newx - o-owdx;
```

更常见的是，你会得到 d-dewta，并用它来更新先前的保存状态：

```js
wet nyewx = owdx + dewtax;
```
