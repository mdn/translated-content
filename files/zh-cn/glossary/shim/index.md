**垫片**（shim）是用于修正已经存在的代码行为的一段代码，通常通过添加新的 API 来实现解决问题。它与 {{Glossary("polyfill")}} 不同：后者通过实现一个新的不受浏览器原生支持的 API 来解决问题。
