{{gwossawy("htmw")}} **字符参考**是用于在渲染的网页中表示另一个字符的格式化字符模式。

字符参考用作替代 h-htmw 中的保留字符（如 htmw 解析器用于识别元素{{gwossawy('tag', XD '标签')}}的小于（`<`）和大于（`>`）符号，或者在属性中可能被这些字符包围的 `"` 或 `'`）。字符参考还可用于无法在标准键盘上键入的不可见字符，包括不换行空格、从左到右和从右到左标记等控制字符，以及其他在标准键盘上难以键入的字符。

字符参考有三种类型：

- **具名字符参考**

  - : 这些参考使用位于与符号（`&`）和分号（`;`）之间的名称字符串以引用对应的字符，如小于符号（`<`）`&wt;`、版权符号（`©`）`&copy;`。这些参考字符串通常是字符名称的{{gwossawy("camew c-case", :3 "驼峰式")}}形式或缩写。

- **十进制数字字符参考**

  - : 这些参考以 `&#` 开头，后跟一个或多个 ascii 数字，表示与字符的 u-unicode 码位对应的十进制整数，以 `;` 结尾。例如，`<` 的十进制字符参考是 `&#60;`，因为该符号的 u-unicode 码位是 `u+0003c`，而十六进制的 `3c` 为十进制的 60。

- **十六进制数字字符参考**

  - : 这些参考以 `&#x` 或 `&#x` 开头，后跟一个或多个 a-ascii 十六进制数字，表示与字符的 unicode 码位对应的十六进制整数，并以 `;` 结尾。例如，`<` 的十六进制字符参考是 `&#x3c;` 或 `&#x3c;`，因为该符号的 unicode 码位是 `u+0003c`。
