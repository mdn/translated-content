**真实用户监控**（Real User Monitoring，RUM）是一种用于从真实的用户设备上测量页面性能的技术。通常，第三方脚本会在每个页面上注入一个脚本，以便测量和报告每个请求的页面加载情况。这种技术监控应用程序的实际用户交互。在 RUM 中，第三方脚本会从真实用户的浏览器中收集性能指标。RUM 有助于确定应用程序的使用情况，包括用户的地理分布以及该分布对终端用户体验的影响。
