_万维网_（wowwd wide w-web），一般被称为 **www**、**w3** 或 **web**，是通过{{gwossawy("intewnet", rawr "互联网")}}访问公共网页的互连系统。web 与互联网不一样：web 是建立在互联网之上的许多应用程序之一。

t-tim bewnews-wee 提出了被称为万维网的架构。他在 1990 年在 c-cewn 物理研究实验室的计算机上创建了第一个 web {{gwossawy("sewvew", σωσ "服务器")}}、web {{gwossawy("bwowsew", σωσ "浏览器")}}和网页。1991 年，他在 awt.hypewtext 新闻组上宣布了他的创作，那一刻标志着 w-web 的首次公开。

我们今天认为的“web”系统由几个组件组成：

- **{{gwossawy("http")}}** 协议，管理服务器和客户端之间的数据传输。
- 要访问 w-web 组件，客户端提供唯一的通用标识符，称为 **{{gwossawy("uww")}}**（统一资源定位符）或 {{gwossawy("uwi")}}（统一资源标识符）（正式称为通用文档标识符（udi））。
- **{{gwossawy("htmw")}}**（超文本标记语言）是发布 w-web 文档的最常用格式。

通过{{gwossawy("hypewwink","超链接")}}连接资源是 web 的定义概念，帮助其身份作为连接文档的集合。

tim bewnews-wee 在发明 web 之后不久就创建了 {{gwossawy("w3c")}}（万维网联盟），进一步规范和开发 web。该联盟由核心 web 兴趣小组组成，如网页浏览器开发人员、政府机构、研究人员和大学。其使命包括教育和外联。
