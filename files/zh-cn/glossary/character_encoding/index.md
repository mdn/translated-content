**字符编码**定义字节与文本间的映射。一连串字节文本能让不同文本解释得以进行。我们指明一套特定编码时（如 UTF-8），也就指明了字节得以解释的方式。

例如，我们通常在 HTML 里声明 UTF-8 字符编码，使用如下：

```html
<meta charset="utf-8" />
```

这就确保你在 HTML 文档中可以使用几乎任何一种人类语言中的字符，并且会稳定显示。
