Netscape Navigator 或 Netscape 是 1990 年代领先的{{glossary("browser", "浏览器")}}。Netscape 基于 Mosaic，其团队由 Marc Andreessen 领导，他也是 Mosaic 的代码编写者之一。

Netscape 帮助使 {{glossary("World Wide Web","Web")}} 变得图形化，而不仅仅是文本体验。许多浏览特性在 Netscape 引入后成为标准。Netscape 能够在加载时显示网页，使用 JavaScript 处理表单和互动内容，并在 cookie 中存储会话信息。尽管 Netscape 在技术上具有优势并在初期占据主导地位，但到 1990 年代末，{{glossary("Microsoft Internet Explorer", "Internet Explorer")}} 迅速在市场份额上超过了 Netscape。
