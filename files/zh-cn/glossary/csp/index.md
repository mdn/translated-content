[**内容安全策略**](/zh-CN/docs/Web/HTTP/CSP)（CSP）用于检测和减轻用于 Web 站点的特定类型的攻击，例如{{Glossary("Cross-site_scripting", "跨站脚本攻击")}}和数据注入。

该安全策略的实现基于一个称作 {{HTTPHeader("Content-Security-Policy")}} 的 HTTP 标头 。
