JavaScript 的**严格模式**是使用受限制的 JavaScript 的一种方式，从而隐式地退出“{{Glossary("Sloppy_mode", "正常模式")}}”。严格模式不仅仅是一个子集：这种模式*有意地*与普通代码具有不同语义。

通过在任何其他语句前面添加 `"use strict";` 语句，即可对整个脚本启用严格模式。
