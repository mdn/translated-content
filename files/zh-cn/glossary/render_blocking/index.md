**阻塞渲染**指的是加载网站时阻止用户界面渲染的任何部分。阻塞渲染对网站性能不利，因为它增加了用户与网站交互的时间，例如查看内容或与控件交互的时间。

最常见的阻塞渲染原因是初始加载的 CSS 或 JavaScript 文件。
