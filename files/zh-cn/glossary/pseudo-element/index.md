在 CSS 中，**伪元素**选择器用于在没有特定 HTML 元素可供选择的情况下，为文档内容的部分应用样式。例如，你可以使用 `p`{{ Cssxref("::first-letter") }} 来为每个段落的首字母应用样式，而不是将它们放在自己的元素中。
