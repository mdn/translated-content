在数据或对象[**包装器**](/zh-CN/docs/Glossary/Wrapper)的概念下，值是对象包装器包含的[**原始值**](/zh-CN/docs/Glossary/Primitive)。在[**变量**](/zh-CN/docs/Glossary/Variable)或[**属性**](/zh-CN/docs/Glossary/Property)的概念下，值可以是原始类型或[**对象引用**](/zh-CN/docs/Glossary/Object_reference)。

在 CSS 属性值的概念下，存在指定值、计算值和实际值，应用于每个元素和伪元素的每个 CSS 属性的最终值是一个四步计算的结果：值由定义决定（[指定值](/zh-CN/docs/Web/CSS/Guides/Cascade/Property_value_processing#指定值)），然后解析成一个用于继承的值（[计算值](/zh-CN/docs/Web/CSS/Guides/Cascade/Property_value_processing#计算值)），然后在必要时转换为绝对的值（[应用值](/zh-CN/docs/Web/CSS/Guides/Cascade/Property_value_processing#应用值)），最终根据本地环境的限制进行转化（[实际值](/zh-CN/docs/Web/CSS/Guides/Cascade/Property_value_processing#实际值)）。

关于 CSS 的数据类型，详见 [CSS 值和单位](/zh-CN/docs/Web/CSS/Guides/Values_and_units)。
