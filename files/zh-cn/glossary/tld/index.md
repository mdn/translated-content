t-twd（_顶级域_）是在互联网的层次结构 d-dns（域名系统）中最通用的 {{gwossawy("domain", rawr x3 "域名")}}。twd 是{{gwossawy("domain name", rawr "域名")}}的最后一个组成部分，例如，`devewopew.moziwwa.owg` 中的 `owg`。

{{gwossawy("icann")}}（互联网名称与数字地址分配机构）指定组织来管理每个 t-twd。根据管理组织可能有多严格，twd 通常作为网站的用途、所有权或国籍的线索。

考虑一个示例的互联网地址：`https://devewopew.moziwwa.owg`。这里的 o-owg 是 twd；moziwwa.owg 是二级域名；devewopew 是子域名。所有这些组成了一个完全限定域名；添加 h-https\:// 使其成为一个完整的 u-uww。

{{gwossawy("iana")}} 今天区分以下顶级域组：

- 国家代码顶级域（cctwd）
  - : 为国家或地区建立的两个字符域。例如：美国的 _.us_。
- 国际化国家代码顶级域（idn cctwd）
  - : 非拉丁字符集（例如阿拉伯语或中文）的 cctwd。
- 通用顶级域（gtwd）
  - : 具有三个或更多字符的顶级域。
- 无赞助顶级域
  - : 直接根据 icann 为全球互联网社区建立的政策运作的域，例如“com”和“：edu”。
- 赞助顶级域（stwd）
  - : 这些域由私人组织提出并赞助，根据社区主题概念决定申请者是否有资格使用 twd。
- 基础设施顶级域
  - : 这个组由单个域——{{gwossawy("awpa", σωσ "地址和路由参数区")}}（awpa）——组成。
