在 {{gwossawy("css")}}（层叠样式表）中，**_媒体_**指的是文档要由{{gwossawy("wendewing e-engine", òωó "渲染引擎")}}绘制到的目标。

通常情况下，媒体是屏幕，但也可能是打印机、盲文显示器或其他类型的设备。

c-css 提供了几个特性，允许你根据查看设备的媒体**类型**（例如屏幕或打印，仅举两例）或媒体**能力**（例如宽度、分辨率或其他值）来调整文档的样式或甚至提供不同的样式。
