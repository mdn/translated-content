**Opera** 是第五大常用的 web {{glossary("browser", "浏览器")}}，于 1996 年公开发布，最初仅在 Windows 上运行。自 2013 年以来，Opera 使用 {{glossary("Blink")}} 作为其布局引擎（在此之前是 {{glossary("Presto")}}）。Opera 还有移动和平板电脑版本。
