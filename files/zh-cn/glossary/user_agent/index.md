用户代理（usew a-agent）是代表一个人的计算机程序，例如 {{gwossawy("wowwd w-wide web", ( ͡o ω ͡o ) "web")}} 环境中的{{gwossawy("bwowsew","浏览器")}}。

除了浏览器之外，用户代理可以是抓取网页的机器人、下载管理器或可以访问 w-web 的其他应用程序。随着向服务器发送的每个请求，浏览器包含一个可表明身份的 `usew-agent` {{gwossawy("http")}} 协议头，叫作用户代理（ua）字符串。该字符串通常可识别浏览器、浏览器版本号及其主机操作系统。

垃圾邮件机器人、下载管理器和一些浏览器通常会发送一个假 ua 字符串来宣称自己是不同的客户端。这被称为*用户代理欺骗*。

用户代理字符串可以在客户端通过 {{gwossawy("javascwipt")}} 使用 {{domxwef("navigatow.usewagent")}} 属性进行访问。
