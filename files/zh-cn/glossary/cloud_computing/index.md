**云计算**指通过互联网（云）按需提供各种计算服务，如存储、数据库、网络、分析和智能等，这些服务也被称为**云服务**。

云服务是由第三方提供商（如 Google（Google Cloud Platform）、Amazon AWS（Amazon Web Services）、Microsoft（Azure）等）通过互联网托管的基础设施、平台或软件。用户可以通过“按需付费”模式访问云服务，只为实际使用的部分付费，无需在本地计算机上进行复杂的软件安装。这种模式带来了更快的创新速度、灵活的可扩展性和显著的成本节约。

云计算主要分为三种类型：基础设施即服务（IaaS）、平台即服务（PaaS）和软件即服务（SaaS）。每种类型为用户提供不同程度的控制、灵活性和管理。

- 基础设施即服务（IaaS）
  - : IaaS 通过互联网提供虚拟化的计算资源，包括服务器、存储和网络，允许用户自行管理基础设施。这为用户带来了高度的灵活性和资源控制能力。使用 IaaS 的公司可以快速扩展服务器基础设施，而无需自有物理服务器。典型例子有 Amazon Web Services（AWS EC2）、Microsoft Azure 虚拟机、Google Compute Engine（GCE）。
- 平台即服务（PaaS）
  - : PaaS 提供一个平台，使客户能够开发、运行和管理应用程序，而无需关心底层的服务器、存储和网络等基础设施。PaaS 让开发者专注于编写代码和应用逻辑，无需管理服务器或操作系统。典型例子有 Google App Engine、Microsoft Azure App Service、Heroku、AWS Elastic Beanstalk。
- 软件即服务（SaaS）
  - : SaaS 通过互联网交付软件应用，用户可通过网页浏览器访问。服务提供商负责所有底层基础设施、平台和数据的管理。企业使用 SaaS 无需本地安装或维护软件，可直接通过浏览器访问如电子邮件、CRM、协作工具等软件。典型例子有 Google Workspace、Microsoft 365、Slack、GitHub、ChatGPT。
