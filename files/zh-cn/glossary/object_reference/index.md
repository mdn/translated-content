对象引用是指向[_对象_](/zh-CN/docs/Glossary/Object)的链接。对象引用可以像链接的对象一样使用。

当将同一个对象赋值给多个[_属性_](/zh-CN/docs/Glossary/Property/JavaScript)时，对象引用的概念就会变得清晰。每个赋值的属性持有指向同一个对象的对象引用，而不是对象的副本，因此当对象发生变化时，所有引用该对象的属性都会反映该变化。
