**Gecko** 是 Mozilla 项目开发的布局引擎，在许多应用程序和设备中使用，包括 {{glossary("Mozilla Firefox","Firefox")}} 和 {{glossary("Firefox OS")}}。

Web {{glossary("browser","浏览器")}}需要一个被称为布局引擎的软件来解释 {{glossary("HTML")}}、{{glossary("CSS")}}、{{glossary("JavaScript")}} 和嵌入式内容（如：图片），并将所有内容绘制到屏幕上。除此之外，Gecko 确保相关的 {{glossary("API")}} 在 Gecko 支持的每一个操作系统上都能很好的工作，并且适当的 API 只暴露给相关的支持目标。这意味着 Gecko 包括网络栈、图形栈、布局引擎、Javascript 虚拟机和移植层等。

由于所有的 Firefox OS 应用都是 Web 应用程序，所以 Firefox OS 会使用 Gecko 作为应用的运行时。
