**Trident**（或 _MSHTML_）是一个用于驱动 {{Glossary("Microsoft Internet Explorer", "Internet Explorer")}} 的专有{{Glossary("Engine/Rendering", "渲染引擎")}}。一个名为 _EdgeHTML_ 的{{Glossary("fork", "分叉")}}取代了 Trident，成为 Internet Explorer 的继任者——{{Glossary("Microsoft Edge", "Edge")}}（后来被称为“Microsoft Edge Legacy”）。

Trident 也在 Microsoft Edge 中“IE 模式”打开时使用。这是一个允许 Edge 使用 Trident 渲染网页的特性，以为旧版网站提供兼容性。Trident 会继续接受安全更新直至 2029 年，以支持 IE 模式的使用。
