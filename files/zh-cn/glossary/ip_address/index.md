**IP 地址**是一个数字，用于对 IP 网络上的每台设备进行唯一的定位。

_IP_ 代表*互联网协议*（Internet Protocol），是与地址相关的[协议层](https://docs.oracle.com/cd/E19683-01/806-4075/ipov-7/index.html)。

在更广泛地使用 IPv6 之前，“IP 地址”通常仍指 32 位 IPv4 地址。
