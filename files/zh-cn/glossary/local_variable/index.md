变量名只在{{Glossary("local scope", "局部作用域")}}中与其{{glossary("value", "值")}}相绑定的{{glossary("variable", "变量")}}是局部变量。

## 示例

```js
let global = 5; // 全局变量

function fun() {
  let local = 10; // 局部变量
}
```
