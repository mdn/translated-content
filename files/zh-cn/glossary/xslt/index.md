_可扩展样式表转换语言_（**XSLT**）是一种声明式语言，用于将 {{Glossary("XML")}} 文档转换为其他 XML 文档、{{Glossary("HTML")}}、{{Glossary("PDF")}}、纯文本等。

XSLT 有它自己的可接受 XML 输入或任何可转换为 XQuery 和 XPath 数据模型的格式的处理器。XSLT 处理器根据 XML 文档和 XSLT 样式表生成新的文档，在此过程中不会对原始文件进行任何更改。
