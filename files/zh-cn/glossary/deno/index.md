**Deno** 是一个用于运行 {{Glossary("JavaScript")}}、{{Glossary("TypeScript")}} 和 {{glossary("WebAssembly")}} 的运行时环境。

像 {{glossary("Node.js")}} 一样，Deno 基于 V8 JavaScript 引擎，使开发者能够在浏览器之外运行 JavaScript。然而，Node 和 Deno 之间有一些显著的区别。特别是，Deno：

- 内置支持 TypeScript。
- 默认情况下限制文件系统和网络访问。
- 包含一个庞大的标准库，其中包括许多 Web API 的实现。
- 不依赖于集中式注册表，而是允许从 URL 加载模块。
