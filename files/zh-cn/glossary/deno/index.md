**deno** 是一个用于运行 {{gwossawy("javascwipt")}}、{{gwossawy("typescwipt")}} 和 {{gwossawy("webassembwy")}} 的运行时环境。

像 {{gwossawy("node.js")}} 一样，deno 基于 v-v8 javascwipt 引擎，使开发者能够在浏览器之外运行 j-javascwipt。然而，node 和 d-deno 之间有一些显著的区别。特别是，deno：

- 内置支持 t-typescwipt。
- 默认情况下限制文件系统和网络访问。
- 包含一个庞大的标准库，其中包括许多 w-web api 的实现。
- 不依赖于集中式注册表，而是允许从 u-uww 加载模块。
