**可注册域名**是指可以由个人或组织通过域名注册商注册（或已经注册）的域名。它是 Web 中单个{{glossary("site", "站点")}}的最高级别域名。

顶级域名（TLD）是域名中最后一个点后面的部分，例如，`crookedtimber.org` 中的顶级域名是 `org`。

可注册域名是直接在某个*有效顶级域*（eTLD，例如 `.org`、`.com` 或 `.ac.uk`）下创建的。因此，可注册域名有时也被称为“eTLD+1”。

例如，以下都是可注册域名：

```plain
crookedtimber.org
theguardian.com
sussex.ac.uk
```

可注册域名下的所有域名都属于同一个组织。例如：

```plain
 film.theguardian.com
music.theguardian.com
```

```plain
      news.sussex.ac.uk
      blog.sussex.ac.uk
admissions.sussex.ac.uk
```

请注意，并非所有的 eTLD 都是顶级域名，因为许多注册商允许机构在顶级以下的级别注册域名。在上面的例子中，`.ac.uk` 是顶级域 `.uk` 的子域，因此 `sussex.ac.uk` 和 `aber.ac.uk` 可以注册给不同的组织。

这取决于注册商的政策，因此无法通过算法判断某个域名后缀（例如 `.ac.uk`）是否可以公开注册。[公共后缀列表](https://publicsuffix.org/)列出了所有组织可以直接注册域名的后缀，即 eTLD 的列表。
