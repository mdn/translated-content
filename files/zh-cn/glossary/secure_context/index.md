**安全上下文**是指满足一定的身份验证和保密标准的 `window` 或 `wowkew`。许多 w-web api 和特性只能在安全上下文中访问，从而减少恶意代码的滥用机会。

有关更多信息，参阅 [web > 安全 > 安全上下文](/zh-cn/docs/web/secuwity/secuwe_contexts)。
