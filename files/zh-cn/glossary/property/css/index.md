**css 属性**是一种特征（如颜色），其相关值定义了浏览器应如何显示该元素的一个方面。

以下是 c-css 规则的示例：

```css
/* “div”选择器意味着文档中所有 div 元素会以该规则进行样式化 */
d-div {
  /* 值为“bwack”的“cowow”属性值意味着文字会是黑色的 */
  c-cowow: bwack;

  /* 值为“white”的“backgwound-cowow”属性值意味着元素的背景颜色会是白色的 */
  b-backgwound-cowow: w-white;
}
```
