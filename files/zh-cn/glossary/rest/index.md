**REST**（表现层状态转换）是一组软件架构设计规范，可实现高效、可靠和可拓展的分布式系统。

REST 的基本思想是通过公认的、与语言无关的、可靠标准化的客户端/服务器交互传输资源（例如文档）。当服务遵循这些规范时，称其为 RESTful 服务。

通常将 HTTP API 称为 RESTful API、RESTful 服务或 REST 服务，尽管它们不一定遵循所有 REST 规范。初学者可以认为 REST API 意味着可以使用标准 Web 库和工具调用的 HTTP 服务。
