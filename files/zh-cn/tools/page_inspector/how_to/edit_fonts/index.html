font-family tooltip 字体系列提示

当鼠标悬停在样式规则视图中的一个字体系列类型，将出现应用该字体类型的样例提示：

 

Fonts view 字体视图

Fonts view字体视图显示当前选中元素所应用的所有字体类型。需要注意的是，它显示的是浏览器当前系统使用的字体，这可能和CSS样式指定的字体不一样。

字体视图默认显示的文本是"Abc"，从Firefox 41开始，这个预览文本可以自由编辑。
