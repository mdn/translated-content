为了能给用户提供在不同场景下都具有价值的功能，WebExtensions 提供了许多的用户界面选项。下面是这些选项的摘要，在本章中将会对每个选项做出更详细的说明。

> [!NOTE]
> 想要使你的扩展能提供良好的用户体验吗？请查看文章[用户体验的最佳实践](/zh-CN/docs/Mozilla/Add-ons/WebExtensions/User_experience_best_practices)。
