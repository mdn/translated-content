存储在会话存储中的数据最大数量（以字节为单位）。你可以使用 {{WebExtAPIRef("storage.StorageArea.getBytesInUse()", "storage.session.getBytesInUse()")}} 来获取当前已存储在绘画存储中的数据的大小。

其取值为 `10485760`。
