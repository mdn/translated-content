{{AddonSidebar}}

扩展可以添加的动态和会话作用域规则的组合最大数。

在 Chrome 中，此限制适用于动态和会话作用域规则的组合。

在 Firefox 中，每个规则集都有自己的配额。

其值为 `5000`。
