此对象包含通过 {{WebExtAPIRef("scripting.getRegisteredContentScripts()")}} 检索或通过 {{WebExtAPIRef("scripting.unregisterContentScripts()")}} 注销的脚本 ID 的列表。

## 类型

此类型的值为对象，包含以下属性：

- `ids`
  - : `string` 的 `array`。脚本 ID 的数组。
