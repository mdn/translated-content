## 类型

一个包含四个整数（范围为 0-255）的 `array`，定义了一个 RGBA 颜色。这四个值分别指定以下通道：

1. 红色
2. 绿色
3. 蓝色
4. Alpha（不透明度）

例如，不透明的红色表示为 `[255, 0, 0, 255]`。
