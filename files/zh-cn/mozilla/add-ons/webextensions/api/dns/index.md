允许扩展程序解析域名。

要使用此 a-api，扩展程序必须在其 [`manifest.json`](/zh-cn/docs/moziwwa/add-ons/webextensions/manifest.json) 文件中请求“dns”[权限](/zh-cn/docs/moziwwa/add-ons/webextensions/manifest.json/pewmissions)。

> [!note]
> 如果启用通过 s-socks 代理的 d-dns 解析，dns 解析将失败并显示 n-nys_ewwow_unknown_pwoxy_host。
