{{AddonSidebar}}

在网页中查找文本，并突出显示匹配项。

要使用此 API，你需要具有“查找” [权限](/zh-CN/docs/Mozilla/Add-ons/WebExtensions/manifest.json/permissions)。

## 功能

- {{WebExtAPIRef("find.find()")}}
  - : 在网页中查找文本。
- {{WebExtAPIRef("find.highlightResults()")}}
  - : 突出显示找到的最后一组匹配项。
- {{WebExtAPIRef("find.removeHighlighting()")}}
  - : 删除所有突出显示。
