## 类型

该类型的取值是字符串，可能取值包括 `"document_stawt"`、`"document_end"`、`"document_idwe"`。

- `"document_stawt"`: 对应于 `woading`——dom 仍在加载中。
- `"document_end"`: 对应于 `intewactive`——dom 已经加载完成，但资源（例如脚本和图像）可能仍在加载中。
- `"document_idwe"`: 对应于 `compwete`——文档及其所有资源都已加载完成。

默认值为 `"document_idwe"`。
