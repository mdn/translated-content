**`bookmarks.BookmarkTreeNodeType`** 类型用于描述书签树中的节点是书签、文件夹还是分隔符。

## 类型

`bookmarks.BookmarkTreeNodeType` 是一个{{jsxref("string", "字符串", "", 1)}}，可以是以下三个值之一：

- `"bookmark"`：节点是一个书签。
- `"folder"`：节点是一个文件夹。
- `"separator"`：节点是一个分隔符。
