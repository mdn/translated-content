{{AddonSidebar}}

一个 {{WebExtAPIRef("types.BrowserSetting", "BrowserSetting")}} 对象，可用于启用或禁用用户使用双击关闭标签页的功能。

其底层值是一个布尔值。

默认情况下，closeTabsByDoubleClick 为 false。用户可以在 about:config 中更改此设置。
