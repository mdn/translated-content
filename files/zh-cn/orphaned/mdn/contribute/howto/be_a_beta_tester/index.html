随着MDN Kuma平台的开发人员不断地对站点进行更改，我们为那些选择成为测试版测试人员提供对这些新特性的早期访问。与任何“beta”测试一样，在某些情况下，一些功能可能无法正常工作。

参与beta测试

  1. 登录MDN，在顶部导航栏点击你的用户名。

    随后跳转到你的资料页。
  2. 点击编辑按钮。

    随后在编辑模式下打开资料页。
  3. 勾选复选框成为 Beta 测试者。
  4. 点击资料页底部的发布按钮

退出Beta测试

  1. 登录MDN，在顶部导航栏点击你的用户名。随后会跳转到你的资料页。
  2. 点击编辑按钮。随后在编辑模式下打开资料页。
  3. 取消 Beta 测试者的复选框
  4. 点击发布按钮.

对beta测试给予反馈

你有两种方式可以对 beta 测试进行反馈:

  • 在MDN讨论组里分享质量反馈. 在 Beta feedback thread 发布一个帖子.  或者,
  • 按照下面的步骤提交 bug:
  1. 如果你还没有账号，创建一个 Bugzilla 账号.
  2. 打开 bug report in Bugzilla for MDN.
  3. 在“摘要”字段中包含 “beta” 一词，帮助 MDN开发人员过滤和区分传入的 bug。.
  4. 尽你所能填写 bug 报告. 越详细越好.
  5. 点击提交按钮.
