Sumário

Macro

Faz addref ao argumento se ele for não-null. Não faz nada a um apontador null. Usar apenas caso o apontador possa ser NULL; Use NS_ADDREF caso não possa.

NS_IF_ADDREF(foo) equivale a

if (foo)
  foo->AddRef();

Sintaxe

NS_IF_ADDREF(foo);
