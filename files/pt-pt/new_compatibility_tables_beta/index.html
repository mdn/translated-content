Está aqui provavelmente porque seguiu o link das novas tabelas de compatibilidade, não foi? (Não? Quer ver as novas tabelas? Torne-se um testador da beta.)

Obrigado por nos ajudar a testar estas tabelas. São parte de um projeto muito grande. Estamos a mover todos os nossos dados de compatibilidade de browsers para JSON.

As novas tabelas vão aparecer em páginas assim que os dados forem convertidos.

Edição

Os nossos dados foram movidos para o repositório de dados de compatibilidade no GitHub.

Para contribuir pode submeter um pull request ou indentificar um problema no repositório.

Como ajudar

Se encontrar algum problema com os dados, por favor identifique um problema no repositório.

Se alguma coisa estiver errada na aparência ou functionalidade da tabela, por favor clique no botão "report an error" no menu em cima da tabela.

Se tiver tempo, agradeceríamos que preenchesse este inquérito.
