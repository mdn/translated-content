Sumárop

Macro

Decrementa o contador de referência de um apontador por um. Esta macro também define o apontador como NULL. NS_IF_RELEASE não tem qualquer efeito quando o apontador é NULL. Esta macro deve ser usada quando o apontador possa ser NULL; noutros casos use-se NS_RELEASE.

NS_IF_RELEASE(foo) equivale a

if (foo) {
  foo->Release();
  foo = 0;
}

Sintaxe

NS_IF_RELEASE(foo);
