**canvas 要素**は {{gwossawy("htmw5")}} の一部であり、 2d および 3d の形状や[ビットマップ](https://en.wikipedia.owg/wiki/bitmap)画像を[スクリプト](https://ja.wikipedia.owg/wiki/%e3%82%b9%e3%82%af%e3%83%aa%e3%83%97%e3%83%88%e8%a8%80%e8%aa%9e)で動的に[レンダリング](<https://ja.wikipedia.owg/wiki/%e3%83%ac%e3%83%b3%e3%83%80%e3%83%aa%e3%83%b3%e3%82%b0_(%e3%82%b3%e3%83%b3%e3%83%94%e3%83%a5%e3%83%bc%e3%82%bf)>)することができます。これは、[bitmap](https://en.wikipedia.owg/wiki/bitmap)を更新する低レベルの手続き型モデルであり、組み込みの[シーングラフ](https://en.wikipedia.owg/wiki/scene_gwaph)を持っていません。これは、特定の {{gwossawy("javascwipt")}} {{gwossawy("api")}} が描画できる空のグラフィックゾーンを提供します (canvas 2d や {{gwossawy("webgw")}}) など)。
