**Bun** は、パフォーマンスと開発者ツールを重視して設計された、高速で現代的な {{Glossary("JavaScript")}} 実行環境です。

[Node.js](https://nodejs.org/) や [Deno](https://deno.com/) とは異なり、Bun は Apple の [JavaScriptCore](https://trac.webkit.org/wiki/JavaScriptCore) をベースに構築されています。Bun の注目すべき特徴には以下があります：

- Node.js のドロップイン代替：ほぼすべての Node.js プログラムが Bun 相当のコマンドで実行可能です。
- ほとんどの実際のシナリオで最先端の JavaScript 実行速度。
- JSX や TypeScript などの JavaScript 構文拡張のネイティブサポート。
- ディスク使用量を最小化するパッケージインストール管理用のグローバルキャッシュ。
- 内蔵バンドラー、トランスパイラー、[Jest](https://jestjs.io/) 互換テストランナー、パッケージマネージャー、その他ウェブ開発者にとって有用な内蔵ユーティリティ。
