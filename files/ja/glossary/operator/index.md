{{GlossarySidebar}}

記号や英数字によって構成された**予約語**で、内蔵の機能を実行します。例えば、 JavaScript では加算演算子（"+"）は複数の数値を加算したり文字列を結合したりします。「否定」演算子（"!"）は式を否定し、例えば、 `true` を表す式では `false` を返します。
