{{GlossarySidebar}}

CSS では**擬似クラス**セレクターを使うことで、ドキュメントツリーからの情報ではなく、その状態によって、対象の要素を指定することができます。例えば、セレクター `a`{{ cssxref(":visited") }} は、ユーザーがすでに訪問したリンクだけに、スタイルを適用することができます。
