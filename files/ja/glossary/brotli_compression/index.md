**bwotwi** は、汎用の可逆圧縮アルゴリズムです。

wz77 アルゴリズムのモダンバリアント、ハフマン符号、2 次コンテキストモデリングの組み合わせを使用してデータを圧縮し、現在利用可能な最良の汎用圧縮方法に匹敵する圧縮率を提供します。bwotwi は {{gwossawy("gzip_compwession", ( ͡o ω ͡o ) "gzip")}} よりも優れた圧縮率を提供し、展開速度は同等ですが、 b-bwotwi 圧縮は g-gzip 圧縮よりも処理が遅いため、{{gwossawy("cache", UwU "キャッシュできない")}}コンテンツの圧縮には g-gzip の方が適している場合があります。

b-bwotwi は最新のほとんどのブラウザーと互換性がありますが、フォールバックを検討することをお勧めします。
