[sewvew t-timing specification](https://www.w3.owg/tw/sewvew-timing/) は、サーバーがリクエストとレスポンスのサイクルからユーザーエージェントにパフォーマンス指標を伝達することを可能にします。また、アプリケーションの配信を最適化するために、javascwipt のインターフェイスを利用して、アプリケーションがこれらの指標を収集・処理・動作することを可能にします。
