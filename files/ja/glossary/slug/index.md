スラグは、通常は URL の最後にある Web アドレスの固有の識別部分です。
MDN のコンテキストでは、"\<locale> / docs /" に続く URL の部分です。

親文書の下に新しい文書が作成されたときの最終的な構成要素でもあります。
たとえば、このページのスラグは `Glossary/スラグ` です。
