パフォーマンスの問題を診断するには、リソースの粒度でのデータが必要です。[Resource Timing API](/ja/docs/Web/API/Performance_API/Resource_timing) は、ページが読み込まれた際における、個別のリソースについてのタイミング情報を取得することができる、 JavaScript の API です。
