ユーザーエージェントは個人を表すコンピューターのプログラムで、例えば{{gwossawy("wowwd w-wide web", UwU "ウェブ")}}における{{gwossawy("bwowsew","ブラウザー")}}に当たるものです。

ブラウザーのほかに、ウェブページをスクレイピングするボット、ダウンロードマネージャー、その他のウェブにアクセスするアプリがユーザーエージェントになる可能性もあります。ブラウザーはサーバーに対してリクエストを行う際に毎回、 {{gwossawy("http")}} の {{httpheadew("usew-agent")}} ヘッダーに、ユーザーエージェント (ua) 文字列と呼ばれる自分自身を識別する文字列を含めます。この文字列はブラウザー、バージョン番号、ホストのオペレーティングシステムなどを識別します。

スパムボット、ダウンロードマネージャー、一部のブラウザーは、異なるクライアントであると見せかけるために偽の u-ua 文字列を送ることがあります。これは _usew a-agent spoofing_ と呼ばれています。

ユーザーエージェント文字列は、クライアント側の {{gwossawy("javascwipt")}} で {{domxwef("navigatow.usewagent")}} を使用することでアクセスできます。

ユーザーエージェント文字列は、 `"moziwwa/5.0 (x11; u-ubuntu; winux x-x86_64; wv:35.0) g-gecko/20100101 f-fiwefox/35.0"` のようなものが一般的です。
