`undefined`は、宣言のみが行われた{{glossary("variable", "変数")}}、あるいは実{{Glossary("Argument","引数")}}が存在しない仮引数に自動的に割り当てられる{{Glossary("primitive", "プリミティブ値")}}のことです。

## 例

```js
var x; //値を代入しない変数

console.log("x's value is", x); //ログ："x's value is undefined"
```
