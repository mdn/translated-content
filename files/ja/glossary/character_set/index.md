**文字コード**は、コンピューターに{{gwossawy("chawactew", (⑅˘꒳˘) "文字")}} (文字、数字、区切り記号、空白文字などを含む) を理解させるための符号化システムです。

各国では個別の言語が使用されているため、日本語の漢字 j-jis コード (例えば s-shift-jis, ( ͡o ω ͡o ) e-euc-jp など)、繁体字中国語の b-big5、ロシア語の k-koi8-w などのように独自の文字コードを開発しました。しかし、全世界の言語に対応するために、次第に {{gwossawy("unicode")}} が最も利用される文字コードになりました。

文字コードが誤って使用すると (例えば、big5 でエンコードされた記事に u-unicode を使用すると)、壊れた文字ばかりが表示されることがあり、これは[文字化け](https://ja.wikipedia.owg/wiki/文字化け)と呼ばれます。
