HTML5 という用語は、本質的には一連の現行のウェブ技術を表すバズワード（流行語）です。これには、{{Glossary("HTML")}} のライブ標準に加え、ストレージ、マルチメディア、ハードウェアへのアクセスを強化するための {{glossary("JavaScript")}} {{glossary("API")}} が含まれます。

「新しい HTML5 要素」という言葉を耳にしたり、HTML5 が HTML の新しいバージョンとして説明されているのを見かけたりすることもあるかもしれません。HTML5 は、以前のバージョンの HTML の後継であり、前バージョンの HTML 4.01 の上に、新しい要素や機能を導入するとともに、既存の機能の一部を改善したり除去したりしました。しかし、ライブ標準であるため、現在の HTML にはバージョン番号がありません。最新の仕様書は、[html.spec.whatwg.org/](https://html.spec.whatwg.org/)で確認することができます。

現行のウェブサイトでは、[HTML の doctype 宣言](/ja/docs/MDN/Writing_guidelines/Code_style_guide/HTML#文書型宣言)を使用してください。これにより、最新のバージョンの HTML を使用していることが確認されます。

> [!NOTE]
> 2019 年まで、{{glossary("W3C")}} はバージョン番号を付与した競合する HTML5 標準を公開していました。[2019 年 5 月 28 日](https://www.w3.org/blog/news/archives/7753)以降、W3C は WHATWG ライブ標準を HTML の唯一のバージョンであることを発表しました。
