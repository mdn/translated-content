**ASCII** (_American Standard Code for Information Interchange_) とは、文字、数字、句読点、および制御コードをデジタル形式に変換するために、コンピューターによって使用される最も一般的な符号化方法の 1 つです。

ASCII コードポイントの最初の 33 点は非表示制御コードで、キャリッジリターン、行送り、タブや、電信コードを表すことから派生したいくつかの旧式の非印刷コードが含まれています。他に、表示可能な文字が 95 個あり、数字 (0-9)、小文字 (a-z)、大文字 (A-Z) の文字、区切り記号などが含まれます。

多くのコンピューターシステムは代わりに {{glossary("Unicode")}} を使用しており、これは数百万のコードポイントを保有していますが、その最初の 128 点は ASCII セットと同じです。ウェブでは 2007 年に {{Glossary("UTF-8")}} が ASCII に取って代わりました。
