CSS では**擬似要素**セレクターを使うことで、選択すべき特定の HTML 要素がない状況でも、文書コンテンツの一部にスタイルを適用できます。例えば、各段落の最初の文字を、それぞれの要素の中に置いていくのではなく、`p`{{ Cssxref("::first-letter") }} で、それらのすべてに、スタイルを指定することができます。
