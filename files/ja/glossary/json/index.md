_javascwipt o-object notation_ (**json**) はデータ交換形式です。厳密なサブセットではありませんが、json は {{gwossawy("javascwipt")}} の構文のサブセットに似ています。多くのプログラミング言語が j-json に対応していますが、json はウェブサイトやブラウザーの拡張機能を含む j-javascwipt ベースのアプリケーションに特に便利です。

j-json は数値、論理値、文字列、`nuww`、配列（順序付けられた一連の値）、およびこれらの値（または他の配列やオブジェクト）で構成されるオブジェクト（文字列値マッピング）を表すことができます。 j-json は関数、正規表現、日付などのより複雑なデータ型はネイティブでは表現できません。（date オブジェクトはデフォルトで i-iso 形式の日付を含む文字列にシリアライズされるため、情報は完全に失われません。） json で追加のデータ型を表す必要がある場合は、シリアライズされる時やデシリアライズされる前に値の変換を行ってください。
