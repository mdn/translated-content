CSS の**物理的プロパティ**は、物理的な次元もしくは要素の辺に基づいた位置を定義します。

- {{Glossary("inset properties", "インセットプロパティ")}}である {{cssxref("top")}}、{{cssxref("right")}}、{{cssxref("bottom")}}、{{cssxref("left")}} は、ビューポートの物理的な次元を参照します。
- {{cssxref("margin-top")}}、{{cssxref("border-right")}}、{{cssxref("padding-bottom")}}、{{cssxref("border-bottom-left-radius")}} などの機能は、要素の特定の辺を参照し、その物理的な次元にスタイルを適用します。

これは{{glossary("logical properties", "論理的プロパティ")}}とは対照的です。論理的プロパティはコンテンツのフローに相対的で、ブロック軸とインライン軸を基準とした次元キーワードを使用します。
