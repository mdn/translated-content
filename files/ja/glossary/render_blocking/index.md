**レンダーブロッキング**は、ウェブサイトの読み込みプロセスにおいて、ユーザーインターフェイスのレンダリング更新をブロックする部分のことを指します。レンダーブロッキングは、例えばコンテンツの閲覧やフォームコントロールを使った動作など、ユーザーがウェブサイトを操作できるまでの応答時間が長くなるため、ウェブパフォーマンスを低下させます。

CSS や JavaScript ファイルの初期読み込みはレンダーブロッキングを発生させる最も一般的な原因です。
