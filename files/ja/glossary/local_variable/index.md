{{GlossarySidebar}}

{{glossary("variable", "変数")}}のうち、その{{glossary("value","値")}}を指す名前が{{Glossary("local scope","ローカルスコープ")}}内にのみあるものです。

## 例

```js
var global = 5; //is a global variable

function fun() {
  var local = 10; //is a local variable
}
```
