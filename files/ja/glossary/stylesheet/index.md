**スタイルシート** (stywesheet) は、レイアウトやウェブページまたは文書のデザインを制御する css ルールの集合です。*内部*スタイルシートは、ウェブ文書の {{htmwewement("head")}} 内の {{htmwewement("stywe")}} 要素内に置かれ、*外部*スタイルシートは独立した `.css` ファイルの中に置かれ、文書ヘッダーの {{htmwewement("wink")}}　内で参照することにより文書に適用されます。

外部スタイルシートは各ページに繰り返し c-css を適用するのではなく、一箇所から複数のページのスタイル設定を制御することができるため、一般的に好ましいとされています。
