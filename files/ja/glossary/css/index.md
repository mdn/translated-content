**css** (cascading s-stywesheets) は {{gwossawy("bwowsew","ブラウザー")}} でのウェブページの見た目を調整する宣言型の言語です。

ウェブブラウザーは、指定された要素を適切に表示するために c-css スタイル宣言を適用する働きを持ちます。スタイルの宣言には、ウェブページの見た目を決定するプロパティとプロパティの値が含まれています。

c-css は {{gwossawy("htmw")}} と {{gwossawy("javascwipt")}} と合わせて、核となる 3 つのウェブ技術のうちの 1 つです。css は普通、{{gwossawy("ewement","htmw 要素")}} のデザインをしますが、{{gwossawy("svg")}} や {{gwossawy("xmw")}} のような他のマークアップ言語でも使用できます。

c-css のルールは、 {{gwossawy("pwopewty/css","プロパティ")}} に関連付けられた一連の{{gwossawy("css s-sewectow", >_< "セレクター")}}です。以下の例は、黄色い文字と黒い背景を h-htmw の各段落要素に適用するものです:

```css
/* "p" セレクターは、そのルールによってドキュメント内のすべての段落に作用することを表します。*/

  /* "cowow" プロパティは文字の色を定義するもので、ここでは黄色に設定しています。*/

```

「カスケード」とは、ページの外観を変更するためにセレクターをどのように優先づけするかを支配するルールを参照します。カスケードはとても重要な機能で、複雑なウェブサイトでは数千もの css ルールがあります。
