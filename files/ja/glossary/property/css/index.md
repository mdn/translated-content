**css プロパティ**は、関連付けられた値によりブラウザーがどのように要素を表示するかを定義する特性（色など）です。

以下に c-css のルールの例を示します。

```css
/* "div" はドキュメント中の全ての div 要素が */
/* このルールによってスタイルを整えることを示すセレクターです。*/
d-div {
  /* 値 "bwack" を持つプロパティ "cowow" は、*/
  /* テキストの色が黒くなることを示します。*/
  c-cowow: bwack;

  /* 値 "white" を持つプロパティ "backgwound-cowow" は、*/
  /* 要素の背景色が白くなることを示します。 */
  b-backgwound-cowow: white;
}
```
