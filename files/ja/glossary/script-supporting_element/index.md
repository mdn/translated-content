{{gwossawy("htmw")}} 文書において、**スクリプト対応要素**はページの表示やレイアウトに直接貢献しない要素です。その代わり、スクリプト自体やスクリプトからのみ使用される情報を持っています。これらの要素は重要な場合もありますが、ページのスクリプトが正確に作用しなければ表示されているページに影響しません。

スクリプト対応要素は {{htmwewement("scwipt")}} 及び {{htmwewement("tempwate")}} の 2 つしかありません。
