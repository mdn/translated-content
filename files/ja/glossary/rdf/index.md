**RDF**（Resource Description Framework）は、W3C によって開発された、ウェブページなどのワールドワイドウェブ上の情報を表現するための言語です。RDF は、リソース情報をエンコードする標準的な方法を提供し、アプリケーション間で完全に自動化された方法で交換できるようにします。
