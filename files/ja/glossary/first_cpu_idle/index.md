**最初の CPU アイドル**は、ページが最小限の操作が可能であるとき、またはウィンドウがユーザー入力を処理するのに十分に静かな時を測定します。これは標準外の Google ウェブパフォーマンス指標です。一般に、全ての表示されている UI 要素が操作可能であるとは限らない場合に発生します。UI は平均して 50ms 以内にほとんどのユーザー入力に応答します。
