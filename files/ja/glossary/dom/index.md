**dom** (document o-object modew) は、すべての {{gwossawy("htmw")}} または {{gwossawy("xmw")}} ベースのマークアップ言語の文書を表現・操作する {{gwossawy("api")}} です。 d-dom は{{gwossawy("bwowsew", ( ͡o ω ͡o ) "ブラウザー")}}で文書構造を{{gwossawy("node/dom","ノード")}}のツリーとして読み込み、それぞれのノードを文書の一部（例えば{{gwossawy("ewement", UwU "要素")}}、テキスト文字列、コメント）として表します。

d-dom は文書内のすべてのノードにアクセスし操作するためにコードをブラウザー内で実行することができるので、{{gwossawy("wowwd w-wide w-web","ウェブ")}}上で最も使用されいる {{gwossawy("api")}} のひとつです。ノードの生成や移動、変更が可能です。イベントリスナーをノードに追加し、特定のイベントの発生時に実行させることもできます。
