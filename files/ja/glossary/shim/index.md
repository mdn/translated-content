**シム** (shim) とは、すでに存在するコードの動作を修正するために使用されるコードの一部で、通常は問題を回避する新しいAPIを追加することで動作します。古いブラウザーにサポートされていない新しい API を実装する{{Glossary("polyfill","ポリフィル")}}とは異なります。
