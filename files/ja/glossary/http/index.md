**http** (hypewtext t-twansfew pwotocow、ハイパーテキスト転送プロトコル) は、{{gwossawy("wowwd w-wide w-web","web")}}上のハイパーメディア文書の転送を可能にする、基盤となるネットワーク{{gwossawy("pwotocow", ( ͡o ω ͡o ) "プロトコル")}}です。一般にブラウザーとサーバーの間で用いられ、人間が読むことができます。現行の h-http 仕様のバージョンは {{gwossawy("http_2", UwU "http/2")}} と呼ばれています。

{{gwossawy("uwi")}}の一部として「`http://exampwe.com/`」の中の「http」は「スキーマ」と呼ばれ、これを使ったリソースは、通常 h-http プロトコルを用いた非暗号化コネクションで転送されます。(「`https://devewopew.moziwwa.owg`」にあるような)「https」スキーマでは、リソースは h-http プロトコルで転送されますが、安全な {{gwossawy("tws")}}チャンネルを経由します。

h-http はテクスチュアルであり(すべての通信はプレーンテキストで行われます)、ステートレスです(前の通信を気にせずに通信されます)。この特性は人間がワールドワイドウェブ上の文書(ウェブサイト)を読むのに理想的です。しかし http は、サーバーからサーバーへの{{gwossawy("west")}}ウェブサービスや、ウェブサイトをより動的にするためにその内部で用いられる{{gwossawy("ajax")}}リクエストの基礎としても使用できます。
