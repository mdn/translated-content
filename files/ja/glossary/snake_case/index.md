**スネークケース**とは、スペースを使わずにフレーズを書く方法で、スペースをアンダースコア `_` で置き換え、単語は通常すべて小文字で表記します。読者にスネークケースの姿を思い出させるために、よく "snake_case" と表記されます。

スネークケースは変数の名前付き規約としてよく使用されています。 `left_shift`, `bitwise_invert`, `matrix_transpose` などの名前はスネークケースです。

スネークケースは決して大文字を含まないことに注意してください。 JavaScript の {{jsxref("Number.MAX_SAFE_INTEGER")}} のように、定数がすべて大文字で書かれていることがあります。これは通常スネークケースとはみなされません。代わりに、スクリーミングスネークケースと名付けられることがあります。

スネークケースは、 Python や Rust など、様々な言語で最も一般的な規約です。
