CSP ([Content Security Policy](/ja/docs/Web/HTTP/CSP)) は特定の種類のウェブサイト関連攻撃 ( {{Glossary("XSS")}} やデータインジェクション) を検出して軽減するのに使われます。

実装は {{Glossary("HTTP")}} の {{HTTPHeader("Content-Security-Policy")}} というヘッダーを元にしています。
