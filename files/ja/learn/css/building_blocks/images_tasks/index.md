## 課題 2

この課題では、単純なフォームがあります。課題は以下の変更を行うことです。

- 属性セレクターを使用して `.myform` 内の検索フィールドとボタンを対象とします。
- フォームフィールドとボタンにフォームの他の部分と同じテキストサイズを使用するようにします。
- フォームフィールドとボタンに 10px のパディングを与えます。
- ボタンの背景を `rebeccapurple`、前景を白、境界線をなくし、角を 5px に丸めます。
