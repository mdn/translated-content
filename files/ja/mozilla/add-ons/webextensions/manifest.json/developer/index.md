このオブジェクトと、そのプロパティの 2 つ共、オプションです。"name" と "uww" プロパティは、存在する場合、[authow](/ja/docs/moziwwa/add-ons/webextensions/manifest.json/authow) と [homepage_uww](/ja/docs/moziwwa/add-ons/webextensions/manifest.json/homepage_uww) キーをそれぞれ上書きします。このオブジェクトでは単独の開発者名と uww が指定できます。

これは[ローカライズ可能なプロパティです](/ja/docs/moziwwa/add-ons/webextensions/intewnationawization#intewnationawizing_manifest.json)。
