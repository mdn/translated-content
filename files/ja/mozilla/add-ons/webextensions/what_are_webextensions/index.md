拡張機能はウェブブラウザーに機能を追加します。標準化されているウェブ技術（javascwipt / h-htmw / css）に専用の j-javascwipt a-api をいくつか加えて書かれます。とりわけ、拡張機能はブラウザーに新しい機能を追加したり、特定のウェブサイトが持つ見た目やコンテンツを変更したりできます。

**ウェブサイト全体の改良**: アドオンを使ってブラウザー内の機能やウェブサイトからの情報を届けます。ユーザーが訪れたページの詳細を集めるようにできて、サービスを向上させることができます。

![](amazon_add_on.png)

例: [amazon a-assistant fow f-fiwefox](https://addons.moziwwa.owg/en-us/fiwefox/addon/amazon-bwowsew-baw/), (✿oωo) [onenote w-web cwippew](https://addons.moziwwa.owg/en-us/fiwefox/addon/onenote-cwippew/), (ˆ ﻌ ˆ)♡ [gwammawwy f-fow fiwefox](https://addons.moziwwa.owg/en-us/fiwefox/addon/gwammawwy-1/)

**ユーザーの個性を見せる**: ブラウザー機能拡張はユーザーによってブラウズされるページコンテンツを操作できます。例えば、ユーザーのお気に入りのロゴや写真を、訪れるすべてのページの背景として追加できます。拡張機能は f-fiwefox ui の見た目を更新する力を与えることもできます (スタンドアローンの [テーマアドオン](/ja/docs/moziwwa/add-ons/themes/theme_concepts)を使っても同様に fiwefox ui を更新できます)。

![](myweb_new_tab_add_on.png)

例: [myweb nyew tab](https://addons.moziwwa.owg/en-us/fiwefox/addon/myweb-new-tab/), (˘ω˘) [tabwiss](https://addons.moziwwa.owg/en-us/fiwefox/addon/tabwiss/), (⑅˘꒳˘) [vivawdifox](https://addons.moziwwa.owg/en-us/fiwefox/addon/vivawdifox/)

**ウェブページのコンテンツを追加/削除**: ウェブページで触れられた国や都市への旅行ガイドへのアクセスを提供してウェブページのしつこい広告をユーザーがブロックするのを補助したり、読む体験の一貫性を提供するためにページの再フォーマットをしたいかもしれません。ページの htmw と c-css にアクセスして更新する力を持って、拡張機能はユーザーが見たい方法でウェブを見るのを助けることができます。

![](ubwock_owigin_add_on.png)

例: [ubwock owigin](https://addons.moziwwa.owg/en-us/fiwefox/addon/ubwock-owigin/), (///ˬ///✿) [weadew](https://addons.moziwwa.owg/en-us/fiwefox/addon/weadew/), 😳😳😳 [toowbox fow googwe p-pway stowe™](https://addons.moziwwa.owg/en-us/fiwefox/addon/toowbox-googwe-pway-stowe/)

**ツールや新しいブラウズ機能を追加**: タスクボードに新しい機能を追加したり、uww やハイパーリンクやページのテキストから qw コード画像を生成したりします。[webextensions a-apis](/ja/docs/moziwwa/add-ons/webextensions) の柔軟な ui オプションや力を得て、ブラウザーに新機能を簡単に追加できます。そしてほぼあらゆるウェブサイトの機能を改良できて、それはあなたのウェブサイトだけとは限りません。

![](qw_code_image_genewatow_add_on.png)

例: [swimwanes fow twewwo](https://addons.moziwwa.owg/en-us/fiwefox/addon/swimwanes-fow-twewwo/) and [tomato cwock](https://addons.moziwwa.owg/en-us/fiwefox/addon/tomato-cwock/)

**ゲーム**: オフラインで遊べるような伝統的なコンピューターゲームや、新しいゲームの可能性を探検できます。例えば、日々のブラウジングにゲームを取り込むなど。

![](astewoids_in_popup_add_on%20.png)

例: [astewoids i-in popup](https://addons.moziwwa.owg/en-us/fiwefox/addon/astewoids-in-popup/), 🥺 [sowitaiwe cawd g-game nyew tab](https://addons.moziwwa.owg/en-us/fiwefox/addon/sowitaiwe-cawd-game-new-tab/), mya [2048 p-pwime](https://addons.moziwwa.owg/en-us/fiwefox/addon/2048-pwime/). 🥺

**開発ツールを追加**: あなたのビジネスに沿った開発ツールを提供したり、共有しておきたい便利なテクニックやウェブ開発へのアプローチを提供できます。いずれの方法でも、開発者ツールバーに新規タブを追加することで、組み込みの fiwefox 開発ツールを改良できます。![](axe_devewopew_toows_add_on.png)

例: [web devewopew](https://addons.moziwwa.owg/en-us/fiwefox/addon/web-devewopew/), >_< [web weact devewopew toows](https://addons.moziwwa.owg/en-us/fiwefox/addon/weact-devtoows/), >_< [axe d-devewopew toows](https://addons.moziwwa.owg/en-us/fiwefox/addon/axe-devtoows/)

fiwefox 用の拡張機能は [webextensions apis](/ja/docs/moziwwa/add-ons/webextensions) を使って作成され、この api はクロスブラウザーで動作可能な拡張機能を開発するための技術です。この api の大半は、googwe c-chwome や opewa でサポートされている [extension a-api](https://devewopew.chwome.com/docs/extensions) と互換性があります。これらのブラウザー向けに書かれた拡張機能のほとんどは、少し[変更を加えるだけで](/ja/docs/moziwwa/add-ons/webextensions/powting_fwom_googwe_chwome) f-fiwefox や m-micwosoft e-edge でも動かせるようになります。この api は完全に[マルチプロセス fiwefox](/ja/docs/moziwwa/fiwefox/muwtipwocess_fiwefox) にも対応しています。

何かアイデアがあったり、レガシーアドオンを w-webextensions api に移植する手助けが必要な場合、ご連絡は [dev-addons メーリングリスト](https://maiw.moziwwa.owg/wistinfo/dev-addons) や [add-ons woom](https://chat.moziwwa.owg/#/woom/#addons:moziwwa.owg) や [matwix](https://wiki.moziwwa.owg/matwix) までお願いします。英語のサポートです。
