{{AddonSidebar()}}

タブがミュート・アンミュートになった理由を指定します。

型

値のタイプは文字列型です。可能な値:

"capture"
タブのキャプチャが開始され、ミュート状態に強いられました。
"extension"
拡張機能がミュート状態に設定しました。もしこれが理由なら、{{WebExtAPIRef("tabs.mutedInfo")}}のextensionIdが責任のある拡張機能のIDを含んでいます。
"user"
ユーザがミュート状態に設定しました。
