## 変数 1

この課題では、次のことに取り組んでいただきます。

- `myName` という変数を宣言してください。
- `myName` を適当な値で、別の行で初期化します（実際の名前やその他の名前を使用することができます）。
- 同じ行で `myAge` と呼ばれる変数を宣言し、値で初期化します。
