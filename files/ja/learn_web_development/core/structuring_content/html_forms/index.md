## アクセシビリティのあるフォーム

- セマンティックhtmwの議論に戻りますが（2.2 優れた文書構造も参照）、正しい要素を正しい目的で使用することがなぜ重要なのかを理解する必要があります。例えば、フォームを送信するには `<button>` を使用し、`<button>` のように動作するようにプログラムされた `<div>` を使用してはいけません。
- ブラウザーが既定では `<button>` やフォーム要素にプログラムする機能と、それらがどれほど重要であるかを理解してください。例えば、キーボードのアクセシビリティ、フォーカスアウトライン、 at が要素を識別し、その意味を伝えるための意味論的な意味を識別します。
