**入門編**へようこそ！ あなたがまったくの初心者（つまり、コードエディターをインストールしたことも、コードを書いたこともないの）であれば、ここが適切な場所です。 入門編のモジュールでは、必要なソフトウェアのインストール、開発環境に慣れること、初めて簡単なウェブサイトを構築してみること、そしてウェブ開発に関する重要な概念をいくつか理解することなどをご案内します。

## 前提知識

このコースを始めるには、前提知識は必要ありません。また、[コアモジュール](/ja/docs/Learn_web_development/Core)へ進む前にこれらのトピックの学習が要求されるわけではありませんが、最初の時点でこれらのトピックに時間を割いておくと、受講生はより学習を進めやすくなるでしょう。

## モジュール

- [環境設定](/ja/docs/Learn_web_development/Getting_started/Environment_setup)
  - : 環境設定モジュールでは、単純なウェブ開発に必要なツールと、それらの正規のインストール方法を示し、ファイルシステムやコマンドラインなど、お客様の環境の重要な側面を理解するのに役立つ情報を提供します。
- [初めてのウェブサイト](/ja/docs/Learn_web_development/Getting_started/Your_first_website)
  - : このモジュールでは、ウェブ開発の実際について学びます。 簡単なウェブページを構築し公開すべき資産を集め、コードを書きます。
- [ウェブ標準](/ja/docs/Learn_web_development/Getting_started/Web_standards)
  - : このモジュールでは、ウェブがどのように動作するのかという基本的な内容について、高水準で説明します。これには、通信に使用されるモデル、関連するコア技術、それらの技術がどのように作成されるのか、ウェブブラウザーがウェブサイトをユーザーにどのようにレンダリングし表示するのか、などが含まれます。
- [ソフトスキル](/ja/docs/Learn_web_development/Getting_started/Soft_skills)
  - : このモジュールでは、ウェブ開発を学びながら向上を目指すことができるソフトスキルの推奨事項を提供し、業界に参入する際に持っていると有利な資質を提示しています。これらは、学習、研究、共同作業に対する正しい姿勢を開発する上で非常に役立ち、採用される可能性を高めます。
