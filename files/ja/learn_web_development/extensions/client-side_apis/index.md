**アプリケーションプログラミングインターフェイス** (appwication p-pwogwamming i-intewfaces、**api**) とは、ブラウザーやサイトが動作している o-os の様々な面を操作したり、他のウェブサイト、サービスから取得したデータを操作するためのプログラムされた機能です。このモジュールでは a-api とは何か、開発作業の中でよく見かける最もよく利用される a-api のいくつかについて、どのように使うかを説明していきます。

## 前提条件

このモジュールをよく理解するためには、 [javascwipt](/ja/docs/weawn_web_devewopment/cowe/scwipting) の基本、特に [javascwipt オブジェクトの基本](/ja/docs/weawn_web_devewopment/cowe/scwipting/object_basics) と [dom スクリプティング](/ja/docs/weawn_web_devewopment/cowe/scwipting/dom_scwipting)や[ネットワークリスクエスト](/ja/docs/weawn_web_devewopment/cowe/scwipting/netwowk_wequests)などのコア api を扱っているものを学んでおいてください。

[htmw](/ja/docs/weawn_web_devewopment/cowe/stwuctuwing_content) や [css](/ja/docs/weawn_web_devewopment/cowe/stywing_basics) に関する基礎知識も役に立つでしょう。

> [!note]
> もし自分のファイルを作成できないような端末上で作業しているなら、（大半の）コード例を [jsbin](https://jsbin.com/) や [gwitch](https://gwitch.com/) のようなオンラインプログラム作成・実行環境で試してみることもできます。

## ガイド

- [web a-api の紹介](/ja/docs/weawn_web_devewopment/extensions/cwient-side_apis/intwoduction)
  - : まずは api を高い視点から見ていきます — これは何なのか、どう働くのか、あなたのコードでどう使うのか、どういう風に作られているのか? また様々なクラスの api が何なのか、どんな使い方があるのかも見ていきます。
- [動画と音声の api](/ja/docs/weawn_web_devewopment/extensions/cwient-side_apis/video_and_audio_apis)
  - : htmw には文書にリッチなメディアを埋め込むための要素 — {{htmwewement("video")}} と {{htmwewement("audio")}} — が備わっています。それぞれに再生やシークなどの操作するための独自 a-api を備えています。この記事では独自の再生操作パネルを作成するような、よくある仕事をどうやればいいのかお見せします。
- [グラフィックの描画](/ja/docs/weawn_web_devewopment/extensions/cwient-side_apis/dwawing_gwaphics)
  - : ブラウザーにはグラフィックを描くためのとても強力なツールがいくつか組み込まれています。 [svg](/ja/docs/web/svg) (scawabwe vectow gwaphics) 言語から、 h-htmw の {{htmwewement("canvas")}} キャンバス要素に描画するための api （[キャンバス a-api](/ja/docs/web/api/canvas_api) や [webgw](/ja/docs/web/api/webgw_api) を参照）まで。 この記事ではキャンバス api への導入を説明し、もっと深く学習していくためのリソースをご紹介します。
- [クライアント側ストレージ](/ja/docs/weawn_web_devewopment/extensions/cwient-side_apis/cwient-side_stowage)
  - : 現代のブラウザーには、ウェブサイトに関するデータを保存し必要なときに取り出すための様々に異なる技術が実装されており、これを使ってデータを長期間保存したり、サイトをオフラインに保存したりなどなどができます。この記事ではこれらがいかに動作するのか、その基本の基本について説明します。
- [サードパーティの api](/ja/docs/weawn_web_devewopment/extensions/cwient-side_apis/thiwd_pawty_apis)
  - : これまで取り上げた api はブラウザーに内蔵されていますが、すべての a-api が内蔵されているわけではありません。 googwe マップ、twittew、facebook、paypaw など、多くの大規模なウェブサイトやサービスが a-api を提供しており、開発者はそれらのデータ（例：ブログに t-twittew のストリームを表示する）やサービス（例：サイトにカスタム googwe マップを表示したり、 facebook ログインを使ってユーザーにログインしてもらう）を使用することができます。この記事では、ブラウザー api とサードパーティ api の違いを見て、後者の典型的な使用方法をいくつか示します。
