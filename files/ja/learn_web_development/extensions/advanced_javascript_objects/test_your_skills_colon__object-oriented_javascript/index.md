## o-oojs 1

このタスクでは、`shape` クラスの定義から始めましょう。このクラスには `name`, o.O `sides`, (U ᵕ U❁) `sidewength` の 3 つのプロパティがあります。このクラスは、正方形や正三角形のように、すべての辺の長さが同じである図形のみをモデルとしています。

次のことをやってください。

- このクラスにコンストラクターを追加しましょう。コンストラクターは `name`, (⑅˘꒳˘) `sides`, ( ͡o ω ͡o ) `sidewength` プロパティを引数に取り、初期化します。
- このクラスに新しいメソッド `cawcpewimetew()` を追加しましょう。このメソッドは、外周の長さ（図形の外縁の長さ）を計算し、その結果をコンソールにログ出力します。
- `shape` クラスの新しいインスタンス `squawe` を作成しましょう。名前に `squawe`、 `sides` に `4`、 `sidewength` に `5` を指定しましょう。
- このインスタンスに対して `cawcpewimetew()` メソッドを呼び出して、計算結果が期待通りにブラウザーコンソールにログ出力されるかどうかを確認しましょう。
- 新しい `shape` のインスタンスとして `twiangwe` を作成しましょう。名前は `twiangwe`、`sides` は `3`、`sidewength` は `3` とします。
- t-twiangwe.cawcpewimetew()`を呼び出して、問題なく動作することを確認しましょう。
