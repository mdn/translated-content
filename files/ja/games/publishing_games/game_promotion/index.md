ゲームを開発して公開するだけでは十分ではありません。人々が楽しんでプレイできる興味深いものが利用可能であることを世界に知らせる必要があります。ゲームを宣伝する方法はたくさんあります。そのほとんどは無料なので、予算ゼロで生活に苦労しているインディー開発者であっても、素晴らしい新しいゲームについて人々に知らせるためにできることはたくさんあります。ゲームを宣伝することは、後で[収益化](/ja/docs/Games/Publishing_games/Game_monetization)する際にも大いに役立つため、正しく行うことが重要です。

## コンペティション

コンペティションに参加することは、ゲーム開発スキルをレベルアップさせ、友達になって学ぶことができる新しい開発者に会うだけでなく、コミュニティに参加することができます。コンペティションで良いゲームを作成し、その過程でいくつかの賞を獲得すると、ゲームは主催者や他の参加者によって自動的に宣伝されます。あなたは金持ちで有名になるでしょう、と彼らは言います。

多くの素晴らしいゲームは、コンペティションに提出された迅速で雑なデモとして始まります。アイデアと実行の両方が十分に良ければ、成功します。さらに、コンペティションは通常、ゲームが特定のテーマに従うことを要求するため、アイデアに行き詰まっている場合は、テーマを中心に創造的になることができます。

## ウェブサイトとブログ

ゲームに関するすべての情報を含む独自のウェブサイトを作成する必要があります。そうすれば、人々はあなたが取り組んできたものを見ることができます。含めることができる情報が多ければ多いほど良いです。スクリーンショット、説明、トレーラー、プレスキット、要件、利用可能なプラットフォーム、サポートの詳細などを含める必要があります。ユーザーがゲームをオンラインで直接プレイできるようにすると、ボーナスポイントが得られます。少なくともデモ形式で。また、人々がゲームをより簡単に見つけられるように、SEO に取り組む必要があります。

また、ゲーム開発活動に関連するすべてについてブログを書く必要があります。開発プロセス、遭遇する厄介なバグ、面白い話、学んだ教訓、ゲーム開発者であることの浮き沈みについて書いてください。ゲームに関する情報を継続的に公開することは、他の人を教育し、コミュニティでの評判を高め、SEO をさらに改善するのに役立ちます。さらなるオプションは、すべての進捗状況をまとめた[月次レポート](https://end3r.com/blog?s=monthly+report)を公開することです。月を通じて達成したことと、まだ残っていることを確認するのに役立ち、ゲームがまもなくリリースされることを人々に思い出させ続けます。話題を作ることは常に良いことです。

ウェブサイトをゼロから作成することもできますが、プロセスを簡単にするのに役立つツールもあります。[ManaKeep](https://manakeep.com/) は、インディーゲーム開発者向けに作られたウェブサイトビルダーで、ウェブサイトを作成するための素晴らしい出発点を提供します。[Presskit()](https://dopresskit.com/) は、メディアと共有するプレスページを作成するのに役立つプレスキットビルダーです。

## ソーシャルメディア

ソーシャルメディアでの存在感は非常に重要です。`#gamedev` ハッシュタグは、友達を見つけ、コミュニティと関わり、困っている他の開発者を助けることができます。正直さが鍵であり、本物である必要があります。誰も退屈なプレスリリースや押し付けがましい広告が好きではありません。時が来たら、コミュニティはあなたの輝く新しいゲームについての言葉を広めるのを手伝ってくれます！

YouTube や Twitch でストリーミングするゲーマーに注目し、[HTML5GameDevs.com](https://www.html5gamedevs.com/) などのフォーラムで活動してください。ニュースを共有し、質問に答えて、人々があなたがしていることを評価し、あなたが信頼できることを知るようにしてください。ゲームについてみんなに伝えることに関しては、あまり押し付けがましくならないように注意してください。あなたは歩く広告ではありません。

聴衆と話し、ヒントを共有し、割引を提供し、コンペティションで賞品を配り、または天気やバグのあるブラウザーについて不平を言うことで、聴衆を増やしてください。寛大で、自分らしく、他の人のためにそこにいてください。そうすれば、尊敬されます。

## ゲームポータル

ゲームポータルの使用は、主に[収益化](/ja/docs/Games/Publishing_games/Game_monetization)に関係していますが、ゲームを購入できるように[ライセンスを販売](/ja/docs/Games/Publishing_games/Game_monetization#ライセンス)する予定がなく、代わりに[広告を実装](/ja/docs/Games/Publishing_games/Game_monetization#広告)またはアプリ内購入を行う予定がある場合、無料ポータル全体でゲームを宣伝することは効果的です。

ゲームを公開のために送信できるさまざまなゲームポータルがいくつかあります。一部のポータルには、ユーザーを承認し、進捗状況を保存し、アプリ内購入を処理できる独自の API があります。ブラウザーのデモバージョン内からゲームのフルバージョンを販売することもできます。これは、高い競争を考慮すると素晴らしい動きになります。一部の開発者は、フルブラウザーバージョンを作成することさえできます。ほとんどのポータルは、収益分配取引を提供するか、非独占ライセンスを購入します。

無料ポータルはトラフィックを提供しますが、最高のものだけが広告やアプリ内購入から収益を生み出すのに十分人気があります。一方で、予算が限られていて時間が限られている場合、より広い聴衆にゲームを見えるようにするための完璧なツールです。

## プレス

ゲームについて[プレス](https://indiegamesplus.com/)に連絡しようとすることができますが、彼らは毎日このようなリクエストをたくさん受け取ることに注意してください。そのため、すぐに返信がない場合は謙虚で忍耐強く、彼らと話すときは礼儀正しくしてください。特定のジャンルのゲームやプラットフォームを扱っているかどうかを最初に確認してください。そうすれば、そもそも関連性のないものを送信することはありません。あなたのアプローチに正直で、ゲームが良ければ、成功の可能性が高くなります。

プレスに連絡するエチケットについてもっと学びたい場合は、Pixel Prospector の素晴らしいガイドである [How To Contact Press](https://app.box.com/s/p0ft5zdolpi0ydkrykab) を確認してください。

## チュートリアル

他の開発者と知識を共有することは良いことです。結局のところ、オンライン記事から何かを学んだ可能性があるので、その知識を次につなげるための時間を取ってください。達成したことや克服した問題について話したり書いたりすることは、人々が興味を持つものです。また、自分のゲームを例として使用できます。特にチュートリアルで、[すでに実装したことを行う方法を示している](/ja/docs/Games/Techniques/Controls_Gamepad_API)場合は特にそうです。そうすれば、みんなが利益を得ます。人々は新しいスキルを学び、ゲームが宣伝され、運が良ければ、十分に良ければチュートリアルを書くことで収入を受けることさえできます。

[Tuts+ Game Development](https://code.tutsplus.com/c/game-development) のようなポータルは、あなたが彼らのために書くことを喜んでいます。彼らはコンテンツに対して支払いますが、すべてのトピックのアイデアが受け入れられるわけではありません。チュートリアルを書くときは、読者に価値のあるものを提供することに焦点を当てることを忘れないでください。彼らは何かを学びたいのです。専門知識を提供し、ゲームをケーススタディとして使用してください。1 つの側面に焦点を当て、詳細に説明してください。また、人々が質問がある場合は、コメントでのディスカッションをフォローアップすることを忘れないでください。

連絡したポータルがあなたのコンテンツに興味がない場合、まだ経験がないため、チュートリアルを書いて、[自分のブログ](#ウェブサイトとブログ)に公開してみてください。それは、自分で執筆スキルを訓練する最も簡単な方法です。

## イベント

上記のすべてのオプションを実行した場合でも、ゲームを宣伝する新しい創造的な方法を見つけることができます。イベントは別の良い例です。ローカルとグローバルの両方のイベントに参加することで、ファンと直接会うことができ、開発コミュニティの他のメンバーとも会うことができます。彼らがあなたに会うために時間を費やしたという事実を大切にしてください。

### カンファレンス

克服した技術的な困難や特定の API の実装方法について話すことができる多くのカンファレンスがあります。繰り返しますが、そのためにゲームを例として使用してください。知識の部分に焦点を当て、マーケティングをトーンダウンすることが重要です。開発者はこの問題に敏感であり、何かを売ろうとするだけであれば、怒った群衆になる可能性があります。

### フェア

その他のイベント関連のオプションは、フェア（またはエキスポ）です。このようなイベントでは、他の開発者の間にブースを取得し、通り過ぎるすべての参加者にゲームを宣伝できます。そうする場合は、ユニークでオリジナルであるように努めてください。そうすれば、群衆から簡単に目立ちます。正しい方法で行えば、誰もがあなたとあなたのゲームについて話します。ブースを持つことで、ファンと直接やり取りする可能性が得られます。宣伝の部分に加えて、通常の人々でゲームの新しいビルドをテストし、発見したバグ（または組み込んだフィードバック）を修正することもできます。ゲームをプレイするときに人々が何を思いつくか、そしてあなたが何時間も磨いている間に見逃した明白な問題を想像することはできません。

## プロモーションコード

ゲームを販売している場合は、ゲームに無料でアクセスできるプロモーションコードを配布する機能を作成してください（または少なくともデモまたは時間制限バージョン）。そして、それらをあらゆる場所に送信してください。プレス、YouTuber、コンペティションの賞品などに。ゲームが特定の人々に届けば、何千人ものプレイヤーに無料の広告を得ることができます。運が良ければ、他の何よりもゲームへの興味を高めることができます。

## コミュニティの育成

コミュニティの成長を支援し、同時に自分自身とゲームを宣伝できます。[週刊ニュースレター](https://gamedevjsweekly.com/)を送信し、[オンラインコンペティション](https://js13kgames.com/)や[ローカルミートアップ](https://gamedevjs.com/)を開催することで、あなたが行っていることに情熱を持っていること、そして彼らがあなたに頼ることができることを他の人に示します。そうすれば、助けが必要なときに、彼らがあなたのためにそこにいます。

## まとめ

ゲームを宣伝するあらゆる方法は良いです。選択できるオプションがたくさんあり、そのほとんどは無料なので、熱意と利用可能な時間だけの問題です。時々、ゲームを実際に開発するよりも、ゲームを宣伝するのに多くの時間を費やす必要があります。世界で最高のゲームを持っていても、誰もそれが存在することを知らなければ意味がないことを忘れないでください。

さて、[収益化](/ja/docs/Games/Publishing_games/Game_monetization)の部分に取り掛かり、生計を立てるために何かを稼ぎましょう。
