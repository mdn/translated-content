**css アニメーション** (css a-animations) は c-css のモジュールの一つで、時間の経過とキーフレームによって c-css プロパティの値を動かすことができます。キーフレームアニメーションの動作は、タイミング関数、時間、繰り返し回数、その他の属性によって制御されます。
