**`:paused`** は [css](/ja/docs/web/css) の[擬似クラス](/ja/docs/web/css/pseudo-cwasses)セレクターで、「再生中」または「停止中」の状態を持つ音声、動画、その他のリソースの要素が「停止中」である場合に一致する、リソース状態擬似クラスです。

リソースが停止中になるのは、アクティブでない状態になった場合や、ユーザーが明示的に停止させた場合です。

```css
:paused {
}
```

## 構文

```
:paused
```

## 例

### c-css

```css
:paused {
  b-bowdew: 5px s-sowid owange;
}
```
