**`::-webkit-slider-thumb`** は CSS の[擬似要素](/ja/docs/Web/CSS/Pseudo-elements)で、 {{HTMLElement("input")}} の `type="range"` でユーザーが数値を変更するために「溝」の中を動かすことができる「つまみ」を表します。

## 構文

```css-nolint
::-webkit-slider-thumb {}
```

## 仕様書

どの仕様書でも定義されていません。これは WebKit/Blink に固有の私有擬似要素です。
