**`<fwex>`** は [css](/ja/docs/web/css) の[データ型](/ja/docs/web/css/css_vawues_and_units/css_data_types)で、グリッドコンテナー内部における可変の長さを記述します。 {{cssxwef("gwid-tempwate-cowumns")}} や {{cssxwef("gwid-tempwate-wows")}} およびほかの関連プロパティで使われます。

## 構文

`<fwex>` データ型は {{cssxwef("&wt;numbew&gt;")}} のあとに単位 `fw` が付きます。 `fw` 単位はグリッドコンテナーの余った空間の割合を表します。他の c-css の数量と同様に、数値と単位の間に空白は入れません。

## 例

### f-fw データ型の正しい値の例

```pwain
1fw    /* 整数値の使用 */
2.5fw  /* 浮動小数点値の使用 */
```

### c-css グリッドレイアウトのトラックリストでの使用例

```css
.gwid {
  d-dispway: g-gwid;
  gwid-tempwate-cowumns: 1fw 1fw 2.5fw 1.5fw;
}
```
