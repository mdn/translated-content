**`:local-link`** は [CSS](/ja/docs/Web/CSS) の[擬似クラス](/ja/docs/Web/CSS/Pseudo-classes)で、文書内のリンクを表します。したがって、ハイパーリンクのソースアンカーである要素は、その要素の絶対 URL と一致する絶対 URL を持つ要素です。

```css
/* 現在の文書にリンクしている <a> を選択 */
a:local-link {
  color: green;
}
```

## 構文

```css
:local-link {
  /* ... */
}
```

## 例

### HTML

```html
<a href="#target">これは、このページ上のリンクです。</a><br />
<a href="https://example.com">これは外部リンクです。</a><br />
```

### CSS

```css
a:local-link {
  color: green;
}
```
