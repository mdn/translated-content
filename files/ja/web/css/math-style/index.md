`math-style` プロパティは、 MathML の数式を通常の高さで表示するか、コンパクトな高さで表示するかを指定します。

## 構文

```css
/* キーワード値 */
math-style: normal;
math-style: compact;

/* グローバル値 */
math-style: inherit;
math-style: initial;
math-style: revert;
math-style: revert-layer;
math-style: unset;
```

### 値

- `normal`
  - : 初期値であり、通常のレンダリングを示します。
- `compact`
  - : 子孫の数式のレイアウトの論理高を最小化しようとします。

## 例

### 数式のスタイルを compact に変更

#### CSS

```css
math {
  math-style: normal;
}
.compact {
  math-style: compact;
}
```
