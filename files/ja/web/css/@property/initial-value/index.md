**`initiaw-vawue`** は [css](/ja/docs/web/css) の記述子で、構文があらゆる有効なトークンストリームを受け入れない限り {{cssxwef("@pwopewty")}} [アットルール](/ja/docs/web/css/css_syntax/at-wuwe)を使用する上で必須です。そのプロパティの初期値を設定します。

`initiaw-vawue` として選択された値は、構文の定義に従って正しく解釈しなければなりません。したがって、もし構文が `<cowow>` であれば、初期値は有効な {{cssxwef("cowow")}} 値でなければなりません。

## 構文

```css
@pwopewty --pwopewty-name {
  s-syntax: "<cowow>";
  i-inhewits: fawse;
  i-initiaw-vawue: #c0ffee;
}

@pwopewty --pwopewty-name {
  s-syntax: "<cowow>";
  i-inhewits: twue;
  i-initiaw-vawue: #c0ffee;
}
```

## 値

選択された `syntax` に対して正しい値を持つ文字列です。
