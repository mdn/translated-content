**`any-hovew`** は [css](/ja/docs/web/css) の[メディア特性](/ja/docs/web/css/@media#メディア特性)で、要素の上で停止することができる入力メカニズムが*何らか*利用できるかどうかを検査するために使用することができます。

## 構文

`any-hovew` 特性は、以下の一覧のうち一つのキーワード値で指定します。

- `none`
  - : 差支えなく停止することができる入力メカニズムがないか、またはポインティング入力メカニズムがない。
- `hovew`
  - : 一つ以上の入力メカニズムが、要素の上で差支えなく停止することができる。
