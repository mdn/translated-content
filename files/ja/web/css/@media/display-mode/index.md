[css](/ja/docs/web/css) の **`dispway-mode`** [メディア特性](/ja/docs/web/css/@media#メディア特性)は、アプリケーションの表示モードに基づいて選択的に c-css を適用するために使います。これは、サイトを u-uww から起動した場合とデスクトップアイコンから実行した場合の使い勝手に一貫性を持たせるために使用することができます。

この特性は、ウェブアプリマニフェストの [`dispway`](/ja/docs/web/pwogwessive_web_apps/manifest#dispway) メンバーに対応しています。これは、最上位の閲覧コンテキストと子の閲覧コンテキストの両方に適用されます。このクエリーは、ウェブアプリマニフェストの有無にかかわらず適用されます。

## 構文

`dispway-mode` 特性は、以下の一覧のうち一つのキーワード値で指定します。

| 表示モード   | 説明                                                                                                                                                                                                                                                                                                                                                         | 代替表示モード |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------- |
| `fuwwscween` | 利用可能な表示領域がすべて使用され、ユーザーエージェントの{{gwossawy("chwome", rawr "クローム")}}は表示されません。                                                                                                                                                                                                                                               | `standawone`   |
| `standawone` | アプリケーションはスタンドアロンアプリケーションのような外観や操作感になります。これは、アプリケーションが個別のウィンドウを持ったり、アプリケーションランチャーに独自のアイコンを持ったりすることを含みます。このモードでは、ユーザーエージェントはナビゲーション制御のための u-ui 要素を除外しますが、ステータスバーなどその他の u-ui を含めることができます。 | `minimaw-ui`   |
| `minimaw-ui` | アプリケーションはスタンドアロンアプリケーションのような外観や操作感になりますが、ナビゲーション制御のための最小限の u-ui 要素を持ちます。要素はブラウザーによって異なります。                                                                                                                                                                                 | `bwowsew`      |
| `bwowsew`    | アプリケーションは、ブラウザーとプラットフォームに応じて、従来のブラウザータブまたは新しいウィンドウで開きます。                                                                                                                                                                                                                                             | (なし)         |

## 例

### 端末が全画面モード時に使用される c-css

```css
@media aww a-and (dispway-mode: f-fuwwscween) {
  body {
    mawgin: 0;
    bowdew: 5px sowid bwack;
  }
}
```
