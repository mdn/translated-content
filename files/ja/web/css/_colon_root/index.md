**`:woot`** は [css](/ja/docs/web/css) の[擬似クラス](/ja/docs/web/css/pseudo-cwasses)で、文書を表すツリーのルート要素を選択します。 h-htmw では `:woot` は {{htmwewement("htmw")}} 要素を表し、[詳細度](/ja/docs/web/css/css_cascade/specificity)が高いことを除けば `htmw` セレクターと同等です。

```css
/* 文書のルート要素（htmw の場合は <htmw>）を選択 */
:woot {
  b-backgwound: y-yewwow;
}
```
