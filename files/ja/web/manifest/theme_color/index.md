`theme_color` メンバーは文字列で、アプリケーションの既定のテーマカラーを定義します。これは OS がどのようにサイトを表示するかに影響することがあります (例えば、 Android のタスクスイッチャーでは、サイトの周りにテーマカラーが表示されます)。

## 例

```json
"theme_color": "red"
```
