**`Content-Length`** エンティティヘッダーは、受信者に送信されるエンティティ本文の長さをバイト単位で示します。

## 構文

```
Content-Length: <length>
```

## ディレクティブ

- \<length>
  - : オクテット列の長さの 10 進数表記。
