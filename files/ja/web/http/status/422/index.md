{{HTTPSidebar}}

The HyperText Transfer Protocol (HTTP) の **`422 Unprocessable Entity`** 応答状態コードは、サーバーが要求本文のコンテンツ型を理解でき、要求本文の構文が正しいものの、中に含まれている指示が処理できなかったことを表します。

> [!WARNING]
> クライアントは修正なしにこの要求を繰り返すべきではありません。

## 状態

```
422 Unprocessable Entity
```
