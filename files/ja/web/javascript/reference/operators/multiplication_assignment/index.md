乗算代入演算子 (`*=`) は、変数に右のオペランドの値を乗算し、結果を変数に代入します。

{{EmbedInteractiveExample("pages/js/expressions-multiplication-assignment.html")}}

## 構文

```js
x *= y; // x = x * y
```

## 例

### 乗算代入の使用

```js
// 次の変数を想定
//  bar = 5

bar *= 2; // 10
bar *= "foo"; // NaN
```
