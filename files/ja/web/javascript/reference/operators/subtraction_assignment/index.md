減算代入演算子 (`-=`) は、変数から右辺のオペランドの値を減算し、結果を変数に代入します。

{{EmbedInteractiveExample("pages/js/expressions-subtraction-assignment.html")}}

## 構文

```js
x -= y; // x = x - y
```

## 例

### 減算代入の使用

```js
// 次の変数を想定
//  bar = 5

bar -= 2; // 3
bar -= "foo"; // NaN
```
