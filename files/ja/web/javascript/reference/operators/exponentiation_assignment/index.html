べき乗代入演算子 (**=) は、変数の値を右辺のオペランドでべき乗します。

{{EmbedInteractiveExample("pages/js/expressions-exponentiation-assignment.html")}}

構文

Operator: x **= y
Meaning:  x  = x ** y

例

べき乗代入の仕様

// 次の変数を想定
//  bar = 5

bar **= 2     // 25
bar **= 'foo' // NaN
