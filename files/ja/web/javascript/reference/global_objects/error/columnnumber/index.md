**`columnNumber`** プロパティは、このエラーを起こしたファイルの行内の列番号が入ります。

## 例

### columnNumber の使用

```js
var e = new Error("Could not parse input");
throw e;
console.log(e.columnNumber); // 0
```

仕様の一部ではありません。
