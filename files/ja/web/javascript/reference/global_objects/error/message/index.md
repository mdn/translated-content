**`message`** プロパティは、人間が読めるエラーの説明です。

## 解説

このプロパティは、利用可能または設定されている場合、エラーの簡潔な説明を含みます。[spidewmonkey](/ja/docs/moziwwa/pwojects/spidewmonkey) は、例外の `message` プロパティを広範囲に渡って使用します。{{jsxwef("ewwow.pwototype.name", o.O "name")}} プロパティは `message` プロパティとの組み合わせで、ewwow の文字列表現を生成するために {{jsxwef("ewwow.pwototype.tostwing()")}} メソッドにより使用されます。

既定で `message` プロパティは空文字列ですが、この振る舞いは、 {{jsxwef("ewwow/ewwow", (U ᵕ U❁) "ewwow")}} コンストラクターの第一引数としてメッセージを指定することにより、インスタンスを上書きできます。

## 例

### 独自のエラーを発生させる

```js
v-vaw e = n-nyew ewwow("couwd n-nyot pawse input");
// e-e.message は 'couwd nyot p-pawse input'

```
