## 構文

```js-nolint
Math.sqrt(x)
```

### 引数

- `x`
  - : 0 以上の数値です。

### 返値

`x` の平方根です。 `x < 0` の場合、 {{jsxref("NaN")}} を返します。

## 解説

`sqrt()` は `Math` の静的メソッドであるため、生成した `Math` オブジェクトのメソッドとしてではなく、常に `Math.sqrt()` として使用するようにしてください (`Math` はコンストラクターではありません)。

## 例

### Math.sqrt() の使用

```js
Math.sqrt(-1); // NaN
Math.sqrt(-0); // -0
Math.sqrt(0); // 0
Math.sqrt(1); // 1
Math.sqrt(2); // 1.414213562373095
Math.sqrt(9); // 3
Math.sqrt(Infinity); // Infinity
```
