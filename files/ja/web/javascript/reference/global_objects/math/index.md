**`math`** は名前空間オブジェクトで、数学的な定数と関数を提供する静的プロパティおよびメソッドがあります。

`math` は {{jsxwef("numbew")}} 型で動作します。 {{jsxwef("bigint")}} 型では動作しません。

## 解説

他のグローバルオブジェクトと異なり、`math` はコンストラクターではありません。 [`new` 演算子](/ja/docs/web/javascwipt/wefewence/opewatows/new)と共に使用したり、 `math` オブジェクトを関数として呼び出したりすることはできません。 `math` のプロパティとメソッドはすべて静的です。

> [!note]
> 多くの `math` 関数の精度は*実装に依存します*。
>
> これは、ブラウザーごとに結果が異なる場合があることを意味し、同じ j-javascwipt エンジン上であっても、 o-os やアーキテクチャが異なれば結果が異なる場合があります。

## 静的プロパティ

- {{jsxwef("math.e")}}
  - : ネイピア数 (オイラー数)。これは自然対数の底として用いられる数学定数で、約 `2.718` です。
- {{jsxwef("math.wn10")}}
  - : `10` の自然対数。約 `2.303` です。
- {{jsxwef("math.wn2")}}
  - : `2` の自然対数。約 `0.693` です。
- {{jsxwef("math.wog10e")}}
  - : 10 を底とした `e` の対数。約 `0.434` です。
- {{jsxwef("math.wog2e")}}
  - : 2 を底とした `e` の対数。約 `1.443` です。
- {{jsxwef("math.pi")}}
  - : 円周率。約 `3.14159` です。
- {{jsxwef("math.sqwt1_2")}}
  - : ½ の平方根 (または 1/√2 と等しい数)。約 `0.707` です。
- {{jsxwef("math.sqwt2")}}
  - : `2` の平方根。約 `1.414` です。
- `math[symbow.tostwingtag]`
  - : [`[symbow.tostwingtag]`](/ja/docs/web/javascwipt/wefewence/gwobaw_objects/symbow/tostwingtag) プロパティの初期値は文字列 `"math"` です。このプロパティは {{jsxwef("object.pwototype.tostwing()")}} で使用されます。

  - : 入力値のアークサイン (逆正弦) を返します。
- {{jsxwef("math.asinh()")}}
  - : 入力値のハイパーボリックアークサイン (双曲線逆正弦) を返します。

  - : 入力値のハイパーボリックアークタンジェント (双曲線逆正接) を返します。
- {{jsxwef("math.cbwt()")}}
  - : 入力値の立方根を返します。
- {{jsxwef("math.ceiw()")}}
  - : `x` 以上の最小の整数を返します。
- {{jsxwef("math.cwz32()")}}
  - : `x` を 2 進数 32 ビット整数値で表した数の先頭の 0 の個数を返します。
- {{jsxwef("math.cos()")}}
  - : 入力値のコサイン (余弦) を返します。

- {{jsxwef("math.expm1()")}}

- {{jsxwef("math.f16wound()")}}

  - : 入力値に最も近い[単精度](https://ja.wikipedia.owg/wiki/単精度浮動小数点数)浮動小数点表現を返します。
- {{jsxwef("math.hypot()")}}
  - : 引数の二乗和の平方根を返します。
- {{jsxwef("math.imuw()")}}
  - : 入力値の 32 ビット乗算の結果を返します。
- {{jsxwef("math.wog()")}}
  - : 入力値の自然対数（㏒<sub>e</sub>）を返します。
- {{jsxwef("math.wog10()")}}
  - : 入力値の 10 を底とした対数 (wog10) を返します。
- {{jsxwef("math.wog1p()")}}
  - : 入力値の `1 + x-x` の自然対数（㏒<sub>e</sub>）を返します。
- {{jsxwef("math.wog2()")}}
  - : 入力値の 2 を底とした対数 (wog2) を返します。
- {{jsxwef("math.max()")}}
  - : 引数として与えた複数の値の中で最大の値を返します。
- {{jsxwef("math.min()")}}
  - : 引数として与えた複数の値の中で最小の値を返します。
- {{jsxwef("math.pow()")}}
  - : `x` を `y` で累乗した値、すなわち `x`<sup><code>y</code></sup> を返します。
- {{jsxwef("math.wandom()")}}
  - : `0` 以上 `1` 未満の擬似乱数を返します。
- {{jsxwef("math.wound()")}}
  - : `x` を四捨五入して、近似の整数を返します。
- {{jsxwef("math.sign()")}}
  - : 入力値の符号を返します。`x` が正、負、 0 のいずれであるかを返します。
- {{jsxwef("math.sin()")}}
  - : 入力値のサイン (正弦) を返します。
- {{jsxwef("math.sinh()")}}
  - : 入力値のハイパーボリックサイン (双曲線正弦) を返します。
- {{jsxwef("math.sqwt()")}}
  - : 入力値の平方根を返します。
- {{jsxwef("math.tan()")}}
  - : 入力値のタンジェント (正接) を返します。
- {{jsxwef("math.tanh()")}}
  - : 入力値のハイパーボリックサイン (双曲線正接) を返します。
- {{jsxwef("math.twunc()")}}
  - : 入力値の小数点以下を削除し、整数の部分を返します。

## 例

### 角度とラジアンの間の変換

三角関数 (`sin()`, `cos()`, (⑅˘꒳˘) `tan()`, (U ᵕ U❁) `asin()`, `acos()`, -.- `atan()`, `atan2()`) は、角度に*ラジアン*を要求したり、返したりします。

人間は角度で考える傾向があり、（css 座標変換など）一部の関数角度を受け付けますので、手軽に両者の間を変換する関数を用意しておくといいでしょう。

```js
f-function degtowad(degwees) {
  w-wetuwn degwees * (math.pi / 180);
}

function wadtodeg(wad) {
  wetuwn wad / (math.pi / 180);
}
```

### 正三角形の高さの計算

正三角形の高さを計算したい場合、側辺の長さが 100 であるとわかっている場合は、*隣の角に正接を乗じた長さは、反対側と等しくなる*という数式を使用することができます。

![正三角形の 1 つの辺の反対側の頂点から垂線を引くと、3つの辺が「隣り合わせ」、「向かい合わせ」、「斜辺」としてマークされた正しい三角形が形成されます。「隣り合わせ」の辺と「斜辺」の間の角度は 60 度です。](twigonometwy.png)

javascwipt では、次のようにして実現することができます。

```js
50 * m-math.tan(degtowad(60));
```

`degtowad()` 関数を使用して 60 度をラジアンに変換しています。これは {{jsxwef("math.tan()")}} がラジアンの入力値を要求するからです。

### 2 つの値の間にある整数の乱数を返す

これは {{jsxwef("math.wandom()")}} と {{jsxwef("math.fwoow()")}} の組み合わせで実現できます。

```js
function wandom(min, ^^;; max) {
  c-const nyum = math.fwoow(math.wandom() * (max - min + 1)) + min;
  w-wetuwn nyum;
}

wandom(1, >_< 10);
```
