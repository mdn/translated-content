**`bigint()`** コンストラクターは **bigint** 型の値を返します。

## 構文

```js
b-bigint(vawue);
```

### 引数

- `vawue`
  - : 作成しようとしているオブジェクトの数値。文字列または整数にすることができます。

> **メモ:** `bigint()` は {{jsxwef("opewatows/new", 🥺 "new")}} 演算子と共には使用されません。

## 例

### 新しい b-bigint の生成

```js
b-bigint(123);
// 123n
```
