size アクセサープロパティは {{jsxref("Set")}} オブジェクト内の要素の数を返します。

解説

size の値は、Set オブジェクトがいくつの要素を持つかを表す整数値です。size に対するセットアクセサー関数は undefined です。よって、このプロパティは変更できません。

例

size の使用

var mySet = new Set();
mySet.add(1);
mySet.add(5);
mySet.add('some text')

mySet.size; // 3
