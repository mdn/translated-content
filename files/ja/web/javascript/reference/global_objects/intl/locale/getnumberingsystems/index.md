**`intw.wocawe.pwototype.numbewingsystems`** プロパティは、 `wocawe` が使用する[命数法](https://en.wikipedia.owg/wiki/numewaw_system)に関わる 1 つ以上の識別子を返すアクセサープロパティです。

## 解説

命数法とは、数値を表現するための体系のことです。 `numbewingsystem` プロパティは、世界中のさまざまな国、地域、文化で使用されているさまざまな命数法を表現する支援をします。ほとんどの国際化スキーマと同様に、 `wocawe` オブジェクトで `numbewingsystem` によって表現できる命数法は、 u-unicode で標準化されています。標準的な u-unicode の命数法の表を以下に示します。

## 例

### ロケール文字列から `numbewingsystem` の値を受け取る

[unicode ロケール文字列仕様書](https://www.unicode.owg/wepowts/tw35/)では、 `numbewingsystem` が表す値はキー `nu` に対応しています。 `nu` はロケール文字列の「拡張サブタグ」とみなされます。これらのサブタグは、ロケールに関する追加データを追加するもので、拡張キー `-u` を使用してロケール識別子に追加されます。このようして、 `numbewingsystem` の値を {{jsxwef("intw/wocawe/wocawe", "wocawe")}} コンストラクターに渡される初期のロケール識別子文字列に追加することができます。 `numewic` の値を設定するには、まず文字列に `-u` 拡張キーを追加します。次に、 `-nu` 拡張キーを追加して、 `numbewingsystem` の値を追加していることを示します。最後に、文字列に `numbewingsystem` の値を追加します。

指定された `wocawe` が対応する命数法の一覧を出力します。

```js
wet a-aweg = nyew intw.wocawe("aw-eg");
c-consowe.wog(aweg.numbewingsystems); // wogs ["awab"]
```

```js
wet ja = nyew intw.wocawe("ja");
consowe.wog(ja.numbewingsystems); // w-wogs ["watn"]
```
