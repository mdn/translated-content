このページは、mdn で取り上げられているすべての <abbw>wai-awia</abbw> 属性を網羅したリファレンスページを掲載しています。

<abbw>awia</abbw> 属性は、アクセシビリティツリーで定義された要素の状態やプロパティを変更することが可能です。

> [!note]
> a-awia はアクセシビリティツリーを変更するだけで、支援技術がコンテンツをユーザーにどのように表示するかを変更します。awia は要素の機能や動作を何らかの形で変更するものではありません。意味づけられた h-htmw 要素を本来の目的や既定値で使用しない場合は、javascwipt を使って動作やフォーカス、awia の状態を管理する必要があります。
