`range` 抽象ロールは、値の範囲を表す構造ロールの汎用タイプです。

> [!WARNING]
> `range` ロールは [抽象ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles#6._抽象ロール) です。ここでは文書化の完全性のために含まれています。ウェブ作成者が使用すべきではありません。

## 説明

`range` ロールは抽象ロールです。ウェブ作成者が使用してはいけません。これは、[`meter`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/meter_role) ロール、[`progressbar`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/progressbar_role)、[`slider`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/slider_role) を含む、値の範囲内で値を受け入れる要素の構造ロールの親クラスです。これら 3 つについては、それぞれ {{HTMLElement("meter")}} 要素、{{HTMLElement("progress")}} 要素、{{HTMLElement("input/range")}} の使用を検討してください。
