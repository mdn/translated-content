**`section` ロール**は、抽象ロールであり、レンダリング可能な構造的格納コンポーネントの親クラスロールです。

> [!WARNING]
> `section` ロールは [抽象ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles#6._abstract_roles) です。ドキュメントの完全性のためにここに含まれています。ウェブ開発者は使用すべきではありません。

## 説明

構造的な `section` ロールは、すべての section サブクラスロールを分類するための抽象ロールです。このロールは使用してはいけません。 [`alert`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/alert_role)、[`note`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/note_role)、[`tooltip`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/tooltip_role) などの一部のサブクラスは有用であり、セマンティック HTML 要素がコンポーネントの目的に完全に適合しない場合にセマンティクスを追加するために使用できます。他の [`code`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/structural_roles)、[`figure`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/figure_role)、[`subscript`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/structural_roles) などは、 HTML 要素の同等物があるため必要ありません。この場合、それぞれ {{HTMLElement('code')}}、{{HTMLElement('figure')}}、{{HTMLElement('sub')}} です。
