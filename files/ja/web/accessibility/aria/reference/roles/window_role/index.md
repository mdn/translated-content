`window` ロールは、ブラウザーまたはアプリのウィンドウを定義します。

> [!WARNING]
> `window` ロールは抽象ロールです。ドキュメントの完全性のためにここに含まれています。ウェブ開発者は使用すべきではありません。

## 説明

`window` ロールは、抽象ロールであり、ブラウザーまたはアプリのウィンドウを定義するロールの親クラスです。サブクラスロール（現在は [`dialog`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/dialog_role) ロールのみ）は、完全なネイティブウィンドウであるか、ウィンドウのように見えるようにスタイル設定されたドキュメントのセクションであるかに関係なく、`role="dialog"` が適切なウィンドウライクな <abbr>GUI</abbr>（グラフィカルユーザーインターフェース）を持ちます。
