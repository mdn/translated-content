ARIA 文書構造ロールは、コンテンツのセクションに構造的説明を提供するために使用されます。

構造的な ARIA ロールは、もともとブラウザーでまだ完全に対応されていない HTML5 要素を支援技術に知らせるブリッジとして作成されました。`presentation`、`toolbar`、`tooltip` などの一部のロールは、同等のネイティブ HTML 要素が存在しない場合に、ドキュメント構造に関する情報を支援技術に提供します。以下の表にリストされているものを含む他のロールは、同じ意味を持つセマンティック HTML 要素があるため、必要ありません。多くの場合、これらと同等の HTML 要素は常にサポートされています。

> [!WARNING]
> これらの構造ロールには、すべてセマンティック HTML の同等物があります。ここではドキュメントの完全性のために含まれています。できれば、ウェブ開発者は使用すべきではありません。代わりに HTML セマンティック要素を選択してください。

一部の構造ロール、例えば [`suggestion`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/suggestion_role) などは、HTML の同等物を持たないため、別のドキュメントがあります。HTML の同等物を持つ一部の構造ロール、例えば [`heading`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/heading_role) などは、ARIA 属性の要件があります。これらは個別のロールドキュメントへのリンクとともに以下にリストされています。

以下のロールのほとんどは必要ではありませんでしたが、完全性のために ARIA に追加されました。ここでも完全性のために含めています。

## HTML 同等物を持つ構造ロール

| **`associationlistitemkey`** <br/> `associationlist` に含まれている必要があります。                                                                                                                                                                                                                                              | {{HTMLElement('dt')}}                                                                                                                                                                                                                                            |
| **`associationlistitemvalue`** <br/> 常に `associationlistitemkey` に続く兄弟です。                                                                                                                                                                                                                                              | {{HTMLElement('dd')}}                                                                                                                                                                                                                                            |
| **`blockquote`** <br/> 別のソースから引用されたコンテンツのセクションです。                                                                                                                                                                                                                                                      | {{HTMLElement('blockquote')}}                                                                                                                                                                                                                                    |
| **`caption`** <br> `figure`、`table`、`grid`、`treegrid` に名前を付け、説明することもある表示コンテンツです。<br/> これらの 4 つのロールでのみ見つかります。caption の `id` は一般的に `figure`、`grid`、`table`、`treegrid` の `aria-labelledby` 属性によって参照されます。<br/> 禁止属性： `aria-label` と `aria-labelledby`。 | {{HTMLElement('caption')}} <br/> {{HTMLElement('figcaption')}}                                                                                                                                                                                                   |
| **`code`** <br/> コンピューターコードの断片を表すセクションです。 <br/> 禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                                                                          | {{HTMLElement('code')}}                                                                                                                                                                                                                                          |
| **`deletion`** <br/>削除済みまたは削除が提案されているとマークされたコンテンツです。<br/> 禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                                                        | {{HTMLElement('del')}}                                                                                                                                                                                                                                           |
| **`emphasis`** <br/> コンテンツを強調または重視するために使用されますが、重要性を示唆するものではありません。<br/>禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                                | {{HTMLElement('em')}}                                                                                                                                                                                                                                            |
| [`figure`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/figure_role) <br/>グラフィカルドキュメント、画像、コードスニペット、またはサンプルテキストのコンテナーです。                                                                                                                                                          | {{HTMLElement('figure')}}                                                                                                                                                                                                                                        |
| [`heading`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/heading_role) <br/> ページのセクションの見出しです。<br/>ネストレベルを示すために `aria-level` 属性が必要です。<br/>詳細については [`heading` ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/heading_role) を参照してください。                             | {{HTMLElement("Heading_Elements", "h1")}}、{{HTMLElement("Heading_Elements", "h2")}}、{{HTMLElement("Heading_Elements", "h3")}}、{{HTMLElement("Heading_Elements", "h4")}}、{{HTMLElement("Heading_Elements", "h5")}}、{{HTMLElement("Heading_Elements", "h6")}} |
| **`image`** <br/>画像を形成する要素のコレクションのコンテナーです。`img` ロールの同義語です。                                                                                                                                                                                                                                    | {{HTMLElement('img')}}                                                                                                                                                                                                                                           |
| [`img`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/img_role) <br/>画像を形成する要素のコレクションのコンテナーです。 <br/> アクセシブル名が必要です。 <br/>詳細については [`img` ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/img_role) を参照してください。                                                     | {{HTMLElement('img')}}                                                                                                                                                                                                                                           |
| **`insertion`** <br/>追加済みまたは追加が提案されているコンテンツとしてマークされたコンテンツです。<br/> 禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                                         | {{HTMLElement('ins')}}                                                                                                                                                                                                                                           |
| [`list`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/list_role)<br/>`listitem` 要素を含むセクションです。<br/> 詳細については [`list` ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/list_role) を参照してください                                                                                                  | {{HTMLElement('ol')}}<br/>{{HTMLElement('ul')}}                                                                                                                                                                                                                  |
| [`listitem`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/listitem_role)<br/>`list` または `directory` 内の単一のアイテムです。<br/>`list` に含まれている必要があります（`<li>` のように）。<br/>詳細については [`listitem` ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/listitem_role) を参照してください。       | {{HTMLElement('li')}}                                                                                                                                                                                                                                            |
| [`mark`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/mark_role)<br/>参照または記述の目的でマークまたはハイライトされます。<br/>詳細については [`mark` ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/mark_role) を参照してください。                                                                                | {{HTMLElement('mark')}}                                                                                                                                                                                                                                          |
| [`meter`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/meter_role) <br/>既知の範囲内のスカラー量、または分数値です。<br/> アクセシブル名が必要です。`aria-valuenow` が必要です。<br/>詳細については [`meter` ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/meter_role) を参照してください。                         | {{HTMLElement('meter')}}                                                                                                                                                                                                                                         |
| **`paragraph`** <br/>コンテンツの段落です。<br/> 禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                                                                                                 | {{HTMLElement('p')}}                                                                                                                                                                                                                                             |
| **`strong`** <br/>重要で、深刻な、または緊急のコンテンツです。<br/> 禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                                                                              | {{HTMLElement('strong')}}                                                                                                                                                                                                                                        |
| **`subscript`** <br/>1 つ以上の下付き文字です。<br/>ロールが存在しないとコンテンツの意味が変わる場合のみ使用してください。<br/> 禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                  | {{HTMLElement('sub')}}                                                                                                                                                                                                                                           |
| **`superscript`** <br/>1 つ以上の上付き文字です。<br/>ロールが存在しないとコンテンツの意味が変わる場合のみ使用してください。<br/> 禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                | {{HTMLElement('sup')}}                                                                                                                                                                                                                                           |
| [`term`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/term_role)<br/>対応する定義を持つオプションの単語または句です。<br/> 禁止属性： `aria-label` と `aria-labelledby` 。<br/>詳細については [`term` ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/term_role) を参照してください。                                 | {{HTMLElement('dfn')}}                                                                                                                                                                                                                                           |
| **`time`** <br> 特定の時点をリストする有効な日付または時刻文字列フォーマットです。<br/> 禁止属性： `aria-label` と `aria-labelledby` 。                                                                                                                                                                                          | {{HTMLElement('time')}}                                                                                                                                                                                                                                          |

> [!NOTE]
> `aria-label` および `aria-labelledby` 属性は、`code`、`caption`、`deletion`、`emphasis`、`generic`、`insertion`、`mark`、`paragraph`、`presentation`、`none`、`strong`、`subscript`、`superscript`、`suggestion`、`term`、`time` では禁止されており、対話型コンテンツでのみ使用すべきです。
