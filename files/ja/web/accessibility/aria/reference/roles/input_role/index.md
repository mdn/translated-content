`input` 抽象ロールは、ユーザー入力を可能にするウィジェットの汎用タイプです。

> [!WARNING]
> `input` ロールは [抽象ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles#6._抽象ロール) です。ここでは文書化の完全性のために含まれています。ウェブ作成者が使用すべきではありません。

## 説明

`input` ロールは抽象ロールです。ウェブ作成者が使用してはいけません。これは、[`checkbox`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/checkbox_role)、[`radio`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/radio_role)、[`textbox`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/textbox_role) を含む、ユーザー入力を提供する入力ウィジェットの親クラスです。これら 3 つすべてについて、それぞれタイプが [`checkbox`](/ja/docs/Web/HTML/Reference/Elements/input/checkbox)、[`radio`](/ja/docs/Web/HTML/Reference/Elements/input/radio)、[`text`](/ja/docs/Web/HTML/Reference/Elements/input/text) の {{HTMLElement("input")}} 要素の使用を検討してください。
