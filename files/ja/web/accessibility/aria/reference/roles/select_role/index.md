**`select` ロール** は抽象ロールで、ユーザーが選択肢の集合から選択を行うことができるフォームウィジェットの親クラスロールです。

> [!WARNING]
> `select` ロールは [抽象ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles#抽象ロール) です。文書の完全性のためにここに含めています。ウェブ作成者が使用するべきではありません。

## 説明

構造的な `select` ロールは、抽象ロールで、ユーザーが選択肢の集合から選択を行うことができる4つのフォームウィジェット（ [`listbox`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/listbox_role)、[`menu`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/menu_role)、[`radiogroup`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/radiogroup_role)、[`tree`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/tree_role) ）の親クラスロールです。
