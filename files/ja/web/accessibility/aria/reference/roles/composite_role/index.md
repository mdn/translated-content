`composite` [抽象ロール](/ja/docs/Web/Accessibility/ARIA/Reference/Roles#6._抽象ロール) は、ナビゲート可能な子孫または所有される子を含む可能性があるウィジェットを示します。

> [!WARNING]
> `composite` ロールは抽象ロールです。ここでは文書化の完全性のために含まれています。ウェブ作成者が使用すべきではありません。

## 説明

`composite` はオントロジーのために使用される抽象ロールです。コンテンツでこのロールを使用しないでください。代わりに、[`grid`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/grid_role)、[`select`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/select_role)、[`spinbutton`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/spinbutton_role)、[`tablist`](/ja/docs/Web/Accessibility/ARIA/Reference/Roles/tablist_role) の composite サブクラスを使用してください。
