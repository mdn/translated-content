`none` ロールは [`presentation`](/ja/docs/Web/Accessibility/ARIA/Roles/presentation_role) ロールの別名です。どちらも、要素の暗黙の ARIA の意味づけを除去し、アクセシビリティツリーに公開されないようにします。

詳しくは [`presentation`](/ja/docs/Web/Accessibility/ARIA/Roles/presentation_role) ロールをさh勝してください。
