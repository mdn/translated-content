WebDriver は、外部プログラムがブラウザーをリモートで監視・制御することができるブラウザー自動化インターフェイスです。これは、プラットフォームや言語に依存しない通信プロトコルであり、コマンドベースの自動化と、リアルタイムのイベント駆動型通信の両方を対応しています。

WebDriver は、[ブラウザー横断テスト](/ja/docs/Learn_web_development/Extensions/Testing/Introduction)のためのツールを提供し、チームが多くのブラウザーやプラットフォームにわたって一貫したユーザー体験を届けることができるよう支援します。これは、ウェブ開発者が、別個のコントローラーからブラウザーを自動化するテストを作成することができることを意図しています。

WebDriver は、HTTP を使用する [WebDriver classic](/ja/docs/Web/WebDriver/Reference/Classic) と、WebSocket を使用し、双方向のイベント駆動型通信をすることができる [WebDriver BiDi](/ja/docs/Web/WebDriver/Reference/BiDi) の 2 つのバリエーションを利用できます。
