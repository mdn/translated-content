WebDriver classic は、ブラウザーの自動化のための HTTP ベースのコマンドを提供する、オリジナルの WebDriver プロトコルです。
このリファレンスドキュメントでは、WebDriver classic で利用できるコマンドとタイムアウトについて解説しています。

- [コマンド](/ja/docs/Web/WebDriver/Reference/Classic/Commands)
  - : HTTP ベースの操作により、ブラウザーの動作を監視・制御し、ブラウザーのセッションやウィンドウを管理し、DOM 要素を操作します。
- [タイムアウト](/ja/docs/Web/WebDriver/Reference/Classic/Timeouts)
  - : スクリプトの挿入、ドキュメントのナビゲーション、要素の取得に関する動作を制御するタイムアウトの定義です。
