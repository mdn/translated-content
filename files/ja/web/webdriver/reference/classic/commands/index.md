WebDriver classic のコマンドは、リモート側の特定のエンドポイントに対して HTTP リクエストとして送信されます。
コマンドは、ユーザーエージェントの動作を制御したり、DOM 要素を検出・操作したり、ブラウザーウィンドウやセッションを管理したりするために使用されます。

WebDriver クライアントがコマンドを送信する際、目的の操作に対応する HTTP メソッド（`GET`、`POST`、`DELETE`）とエンドポイントパスを指定します。
リモート側はコマンドを処理し、結果またはエラーを含む HTTP レスポンスを返します。
