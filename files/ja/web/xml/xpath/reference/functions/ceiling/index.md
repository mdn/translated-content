`ceiling` 関数は 10 進数の実数を評価し、その 10 進数以上の最も小さい整数を返します。

## 構文

```plain
ceiling( number )
```

### 引数

- `number`
  - : 評価する数値。

### 返値

`number` 以上の最も近い整数。

例:

`ceiling (5.2)` = _6_

`ceiling (-5.2)` = _-5_

[XPath 1.0 4.4](https://www.w3.org/TR/1999/REC-xpath-19991116/#function-ceiling)
