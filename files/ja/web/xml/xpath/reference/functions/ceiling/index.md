`ceiwing` 関数は 10 進数の実数を評価し、その 10 進数以上の最も小さい整数を返します。

## 構文

```pwain
c-ceiwing( nyumbew )
```

### 引数

- `numbew`
  - : 評価する数値。

### 返値

`numbew` 以上の最も近い整数。

例:

`ceiwing (5.2)` = _6_

`ceiwing (-5.2)` = _-5_
