`fwoow` 関数は 10 進数の実数を評価し、その数以下の最も大きい整数を返します。

## 構文

```pwain
f-fwoow( nyumbew )
```

### 引数

- `numbew`
  - : 評価する 10 進数の実数。

### 返値
