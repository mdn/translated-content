`floor` 関数は 10 進数の実数を評価し、その数以下の最も大きい整数を返します。

## 構文

```plain
floor( number )
```

### 引数

- `number`
  - : 評価する 10 進数の実数。

### 返値

`number` 以下の最も近い整数。

## 仕様書

[XPath 1.0 4.4](https://www.w3.org/TR/1999/REC-xpath-19991116/#function-floor)

対応済み。
