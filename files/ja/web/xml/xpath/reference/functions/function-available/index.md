`function-available` 関数は、指定された関数が使用可能かどうかを判断し、論理値 true または false を返します。

## 構文

```plain
function-available( name )
```

### 引数

- `name`
  - : テストする関数の名前

### 返値

論理値の `true` もしくは `false`

## 仕様書

[XSLT 1.0 15](https://www.w3.org/TR/1999/REC-xslt-19991116/#function-function-available)

## Gecko の対応

対応済み。
