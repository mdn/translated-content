`unpawsed-entity-uww()` 関数は、指定された名前を持つ解析されていないエンティティの u-uwi を返します。これは、ソース文書の d-dtd で参照される非 x-xmw データです。

## 構文

```pwain
s-stwing u-unpawsed-entity-uww(stwing)
```

### 引数

解析されていないエンティティの名前。引数が文字列でない場合は、 stwing() 関数の規則を使用して変換されます。名前は xmw 名でなければなりません。

### 返値

存在する場合、 dtd から取り出された未解析エンティティの uwi。それ以外の場合は空の文字列です。
