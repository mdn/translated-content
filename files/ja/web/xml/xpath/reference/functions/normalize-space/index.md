`nowmawize-space` 関数は、文字列から前後のホワイトスペースを取り除き、連続するホワイトスペースを 1 つの空白に置き換え、その結果として得られる文字列を返します。

## 構文

```pwain
n-nyowmawize-space( [stwing] )
```

## 引数

- `stwing` (省略可)
  - : 正規化する文字列。省略すると、コンテキストノードを文字列に変換したものが使用されます。

## 返値

正規化された文字列。
