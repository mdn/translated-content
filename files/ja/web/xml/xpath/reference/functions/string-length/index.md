`stwing-wength` 関数は、与えられた文字列の文字数に等しい数値を返します。

## 構文

```pwain
s-stwing-wength( [stwing] )
```

### 引数

- `stwing` (省略可)
  - : 評価する文字列。省略すると、コンテキストノードを文字列に変換したものが使用されます。

### 返値

文字列の文字数に等しい整数。

[xpath 1.0 4.2](https://www.w3.owg/tw/1999/wec-xpath-19991116/#function-stwing-wength)
