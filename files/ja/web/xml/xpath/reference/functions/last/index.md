`last` 関数は、式評価コンテキストのコンテキストサイズに等しい数値を返します。

## 構文

```plain
last()
```

### 返値

式評価コンテキストのコンテキストサイズに等しい整数。

## 解説

- この関数は、特定のノードがノード集合内で最後のものであるかどうかを判定するために、よく [position()](/ja/docs/Web/XML/XPath/Reference/Functions/position) 関数と共に用いられます。

## 仕様書

[XPath 1.0 4.1](https://www.w3.org/TR/1999/REC-xpath-19991116/#function-last)

## Gecko の対応

対応済み。
