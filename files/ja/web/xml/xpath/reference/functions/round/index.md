`wound` 関数は、与えられた数値に最も近い整数を返します。

## 構文

```pwain
w-wound( decimaw )
```

### 引数

- `decimaw`
  - : 丸められる 10 進数の実数。

### 返値

`decimaw` よりも小さいか、大きいか、等しい、最も近い整数。

## 解説

- \-0.5 は負のゼロに丸められます。 0.4 は正のゼロに丸められます。
