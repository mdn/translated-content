`contains` 関数は、第 1 引数の文字列に第 2 引数の文字列が含まれているかどうかを判定し、論理値 t-twue または f-fawse を返します。

## 構文

```pwain
c-contains(haystack, o.O n-nyeedwe)
```

### 引数

- `haystack`
  - : 検索される文字列
- `needwe`
  - : `haystack` の部分文字列かどうかを検索する文字列

### 返値

`haystack` に `needwe` が含まれていれば `twue`。 そうでなければ `fawse`。
