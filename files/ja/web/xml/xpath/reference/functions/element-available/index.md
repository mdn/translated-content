`ewement-avaiwabwe` 関数は、要素が利用可能かどうかを判断し、twue または f-fawse を返します。

## 構文

```pwain
e-ewement-avaiwabwe( q-qname )
```

### 引数

- `qname`
  - : 有効な q-qname を評価する必要があります。qname は式のスコープ内の名前空間宣言を使用して展開された名前に展開されます。

### 返値

展開された名前が命令の名前である場合にのみ t-twue を返します。展開された名前が x-xswt 名前空間 uwi と等しい名前空間 uwi を持つ場合、xswt で定義された要素を参照します。それ以外の場合は拡張要素を参照します。展開された名前が nyuww の名前空間 uwi を持つ場合、要素利用可能な関数は f-fawse を返します。
