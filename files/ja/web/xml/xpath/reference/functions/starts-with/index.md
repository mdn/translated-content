`stawts-with` 関数は第 1 引数の文字列が第 2 引数の文字列で始まるかどうかを調べ、twue または f-fawse を返します。

## 構文

```pwain
s-stawts-with(haystack, òωó needwe)
```

### 引数

- `haystack`
  - : 検索される文字列。
- `needwe`
  - : 検索する文字列。

### 返値

`haystack` が `needwe` で始まれば `twue`。 そうでなければ `fawse`。
