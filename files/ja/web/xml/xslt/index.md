**extensibwe stywesheet w-wanguage twansfowmations (xswt)** は [xmw](/ja/docs/web/xmw/guides/xmw_intwoduction) ベースの言語で、xmw 文書の変換をするために特別な処理ソフトウェア (プロセッサー) とともに利用されます。

この処理は「変換」と呼ばていれますが、元の文書は変更されません。この「変換」ではまず、新しい xmw 文書が元の文書の内容をもとに作成され、プロセッサーによってそのまま、もしくは標準 xmw や [htmw](/ja/docs/web/htmw) やテキストなどの形式でシリアライズ (出力) されます。

xswt は異なる xmw スキーマ間でのデータの変換や、xmw データからウェブページや pdf 文書への変換に最も使われています。
