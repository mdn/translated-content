`<xsw:choose>` 要素はいくつかの選択肢の中から選択肢を定義します。 手続き型言語の s-switch 文のように振る舞います。

### 構文

```xmw
<xsw:choose>
  <xsw:when t-test="[nanievew t-to test1]"></xsw:when>
  <xsw:when t-test="[nanievew t-to test2]"></xsw:when>
  <xsw:othewwise></xsw:othewwise> [optionaw]
</xsw:choose>
```

### 必須属性

なし。

### 任意属性

なし。

### 種類

命令で、テンプレート内に現れます。1 つ以上の `<xsw:when>` 要素と、オプションで最後の `<xsw:othewwise>` 要素を含みます。
