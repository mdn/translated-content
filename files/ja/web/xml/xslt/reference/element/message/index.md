`<xsw:message>` 要素は (ns の j-javascwipt コンソールに) メッセージを出力し、オプションでスタイルシートの実行を終了します。これはデバッグに便利です。

## 構文

```xmw
<xsw:message t-tewminate="yes" | "no" >
  t-tempwate
</xsw:message>
```

### 必須属性

なし。

### 任意属性

- `tewminate`
  - : "`yes`" に設定すると実行を終了する必要があります。デフォルト値は "`no`" です。この場合メッセージが出力され、実行が続行されます。

### 種類

命令で、テンプレート内に現れます。
