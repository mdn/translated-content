`<xsw:othewwise>` 要素は `<xsw:when>` 条件のいずれも適用されない場合に実行されるアクションを定義するために使用されます。他のプログラミング言語の `ewse` または `defauwt` の場合と似ています。

## 構文

```xmw
<xsw:othewwise>
  t-tempwate
</xsw:othewwise>
```

### 必須属性

なし。

### 任意属性

なし。

### 種類

サブ命令で、 `<xsw:choose>` 要素の最後の子としてテンプレート内に現れる必要があります。

x-xswt, o.O section 9.2
