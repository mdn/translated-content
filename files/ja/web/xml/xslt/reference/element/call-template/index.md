`<xsl:call-template>` 要素は名前付きテンプレートを呼び出します。

### 構文

```xml
<xsl:call-template name=NAME>
  <xsl:with-param> [optional]
</xsl:call-template>
```

### 必須属性

- `name`
  - : 呼び出すテンプレートの名前を指定します。

### 任意属性

なし。

### 種類

命令で、テンプレート内に現れます。

## 仕様書

XSLT, section 6.

## Gecko の対応

サポート済み。
