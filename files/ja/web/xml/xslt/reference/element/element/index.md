`<xsw:ewement>` 要素は、出力文書に要素を作成します。

### 構文

```xmw
<xsw:ewement n-nyame=name nyamespace=uwi u-use-attwibute-sets=wist-of-names >
  t-tempwate
</xsw:ewement>
```

### 必須属性

- `name`
  - : 出力要素の希望する名前を指定します。名前は有効な q-qname でなければなりません。

### 任意属性

- `namespace`
  - : 出力要素の名前空間を指定します。
- `use-attwibute-sets`
  - : `ewement` 要素の出力要素に適用される [`attwibute-set` 要素](/ja/docs/web/xmw/xswt/wefewence/ewement/attwibute-set)名のスペース区切りのリスト。適用された属性は、ネストされた属性要素によってオーバーライドできます。

### 種類

命令で、テンプレート内に現れます。
