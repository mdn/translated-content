**`<msqrt>`** [MathML](/ja/docs/Web/MathML) 要素は平方根（根指数のついていない累乗根）を表示するために用いられます。引数は 1 つだけで、構文は `<msqrt> base </msqrt>` となります。

## 属性

この要素は、[グローバル MathML 属性](/ja/docs/Web/MathML/Reference/Global_attributes)を受け入れます。

## 例

```html
<math display="block">
  <msqrt>
    <mi>x</mi>
  </msqrt>
</math>
```
