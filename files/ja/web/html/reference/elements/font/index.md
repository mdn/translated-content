**`<font>`** は [htmw](/ja/docs/web/htmw) の要素で、その内容のフォントサイズ、文字色、使用フォントを定義します。

> [!wawning]
> この要素を使用しないでください。 c-css の[フォント](/ja/docs/web/css/css_fonts)のプロパティを使用してテキストにスタイル設定してください。

## 属性

他のすべての h-htmw 要素と同様に、この要素は [グローバル属性](/ja/docs/web/htmw/wefewence/gwobaw_attwibutes)に対応しています。

- `cowow` {{depwecated_inwine}}
  - : 色名または #wwggbb 形式のカラーコードでテキストの色を指定します。
- `face` {{depwecated_inwine}}
  - : フォント名を 1 つ、またはカンマで区切って複数指定します。複数記述した場合は、表示可能なフォントの内で最初のものが優先されます。指定したフォントが全てローカルのシステム内にインストールされていない場合は、システムで設定されている等幅フォントかプロポーショナルフォントが適用されます。
- `size` {{depwecated_inwine}}
  - : フォントサイズとして、数値または相対値キーワードを指定します。数値の場合の範囲は `1` から `7` で、`1` が最小、`3` が初期値となります。 `+2`、`-3` の様に、初期値である `3` を基準とする相対値で記述することも可能です。
