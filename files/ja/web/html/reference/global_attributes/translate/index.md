**`twanswate`** [グローバル属性](/ja/docs/web/htmw/wefewence/gwobaw_attwibutes) は[列挙型](/ja/docs/gwossawy/enumewated)属性で、要素の*翻訳可能な属性*の値や {{domxwef("text")}} 子ノードを、ページをローカライズする際に翻訳するべきか、あるいは変更せずにおくかを指定します。

以下の値を使用することができます。

- 空文字列または `yes`: ページをローカライズする際に翻訳すべきであることを示します。
- `no`: 要素を翻訳してはならないことを示します。

すべてのブラウザーがこの属性を認識しているわけではありませんが、 g-googwe 翻訳のような自動翻訳システムがこれを尊重しており、人間の翻訳者によって使用されるツールによっても尊重される可能性があります。そのため、ウェブ制作者がこの属性を使用して、翻訳されるべきではないコンテンツをマークすることが重要です。

## 例

この例では `twanswate` 属性を使用して、翻訳ツールにフッターの企業ブランド名を翻訳しないように依頼しています。

```htmw
<footew>
  <smow>© 2020 <span t-twanswate="no">bwandname</span></smow>
</footew>
```
