**`noreferrer`** キーワードを [`rel`](/ja/docs/Web/HTML/Reference/Attributes/rel) 属性に指定すると、 {{HTMLElement("a")}}, {{HTMLElement("area")}}, {{HTMLElement("form")}} の各要素はターゲットリソースへ移動する際、 {{HTTPHeader("Referer")}} ヘッダーを省略してリファラー情報が漏洩しないようにブラウザーに指示します。それに加えて、 [`noopener`](/ja/docs/Web/HTML/Reference/Attributes/rel/noopener) キーワードを設定しているかのように動作します。
