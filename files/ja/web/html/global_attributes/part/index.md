**`part`** [グローバル属性](/ja/docs/Web/HTML/Global_attributes)は、要素の部品名の空白区切りのリストを指定します。 CSS は部品名を使用して、 {{CSSxRef("::part")}} 擬似要素を介してシャドウツリー内の特定の要素の選択およびスタイルの設定を行うことができます。

使用例については [Shadow part example](https://mdn.github.io/web-components-examples/shadow-part/) をご覧ください。
