**`x`** は {{domxref("DOMRect")}} インターフェイスのプロパティで、矩形の x 座標を表します。これは、ビューポートの左端と矩形の原点との水平距離です。

矩形の幅が負でない場合、矩形の水平方向の原点はビューポートの左端になります。幅が負の場合、矩形の水平方向の原点はビューポートの右端になります。

## 値

倍精度浮動小数点値。
