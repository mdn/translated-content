**`y`** は {{domxref("DOMRect")}} インターフェイスのプロパティで、矩形の y 座標を表します。これは、ビューポートの上端と矩形の原点との垂直距離です。

矩形の幅が負でない場合、矩形の垂直方向の原点はビューポートの上端になります。幅が負の場合、矩形の垂直方向の原点はビューポートの下端になります。

## 値

倍精度浮動小数点値。
