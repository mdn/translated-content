{{APIRef("HTML DOM")}}

**`checkValidity()`** は {{domxref("HTMLObjectElement")}} インターフェイスのメソッドで、常に真である論理値を返します。なぜなら、object のオブジェクトは制約検証の候補になることがないからです。

## 構文

```js-nolint
checkValidity()
```

### 引数

なし。

### 返値

`true`

### 例外

なし。
