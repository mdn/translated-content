{{APIRef("Encoding API")}}

**`TextEncoder()`** コンストラクターは、新しく生成された {{DOMxRef("TextEncoder")}} オブジェクトを返します。これは UTF-8 エンコーディングのバイトストリームを生成します。

## 構文

```js-nolint
new TextEncoder()
```

### 引数

なし。
