[Web MIDI API](/ja/docs/Web/API/Web_MIDI_API) の読み取り専用インターフェイス **`MIDIOutputMap`** は、現在利用可能な MIDI 出力ポートの集合を提供します。

`MIDIOutputMap` のインスタンスは、キーを MIDI 出力の ID 文字列、対応する値を対応する {{domxref("MIDIOutput")}} オブジェクトとする読み取り専用の [`Map` 風オブジェクト](/ja/docs/Web/JavaScript/Reference/Global_Objects/Map#map_風のブラウザー_api)です。
