**`NDEFReader()`** は {{domxref("NDEFReader")}} インターフェイスのコンストラクターで、新しい `NDEFReader` オブジェクトを返します。これはリーダーの磁気誘導範囲にある互換性のある NFC 機器、例えば NDEF タグからの NDEF メッセージを読み取るのに使用されます。
field.

## 構文

```js-nolint
new NDEFReader()
```

### 引数

なし。

### 返値

新しい {{DOMxRef("NDEFReader")}} オブジェクト。
