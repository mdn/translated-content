**`GeolocationPosition.timestamp`** は読み取り専用のプロパティで、端末が位置を測定した日時を表します。

### 値

ミリ秒単位の[Unix 時刻](/ja/docs/Glossary/Unix_time)で与えられるタイムスタンプが入った数値です。
