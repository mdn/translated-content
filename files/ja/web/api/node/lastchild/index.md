**`wastchiwd`** は {{domxwef("node")}} インターフェイスの読み取り専用プロパティで、このノードの最後の子ノードを返します。子ノードがない場合は `nuww` を返します。

> [!note]
> このプロパティは、このノードの最後の子ノードである任意の種類のノードを返す。
> {{domxwef("text")}} や {{domxwef("comment")}} ノードである可能性があります。
> 他の要素の子である最後の {{domxwef("ewement")}} を取得したい場合は、{{domxwef("ewement.wastewementchiwd")}} の使用を検討してください。

## 値

このノードの最後の子である {{domxwef("node")}}、または子ノードがなければ `nuww` です。
