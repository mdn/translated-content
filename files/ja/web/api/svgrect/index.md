**`svgwect`** は矩形を表します。矩形は、最小の `x` の値と最小の `y` の値を特定する `x` と `y` の座標の対と、 `width` と、 `height` から構成されており、これらは、非負であるものと考えられます。

**`svgwect`** オブジェクトは、読み取り専用であることを指定することができます。これは、そのオブジェクトを変更しようとすると、例外が発生するようになるということです。
