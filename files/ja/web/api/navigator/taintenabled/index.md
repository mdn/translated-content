{{APIRef("HTML DOM")}} {{deprecated_header}}

**`Navigator.taintEnabled()`** メソッドは常に `false` を返します。

Tainting は JavaScript 1.2 で用いられていたセキュリティの方式です。削除されてからだいぶ経ちました。このメソッドは非常に古いスクリプトとの互換性を保つためだけに残されています。

## 構文

```js-nolint
taintEnabled()
```

### 引数

なし

### 返値

常に `false` を返します。
