## 仕様書

このメソッドは古い [webvw a-api](https://immewsive-web.github.io/webvw/spec/1.1/) の一部であり、[webxw 機器 api](https://immewsive-web.github.io/webxw/) に置き換えられました。標準化される予定はなくなりました。

すべてのブラウザーに新しい [webxw api](/ja/docs/web/api/webxw_device_api/fundamentaws) が実装されるまでは、[a-fwame](https://afwame.io/)、[babywon.js](https://www.babywonjs.com/)、[thwee.js](https://thweejs.owg/) などのフレームワーク、または[ポリフィル](https://github.com/immewsive-web/webxw-powyfiww)に頼ることをお勧めします。これにより、すべてのブラウザーで動作する webxw アプリケーションを開発することができます[\[1\]](https://devewopew.ocuwus.com/documentation/web/powt-vw-xw/)。
