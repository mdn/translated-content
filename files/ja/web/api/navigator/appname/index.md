**`Navigator.appName`** プロパティの値は、どのブラウザーでも常に "`Netscape`" です。このプロパティは互換性ためだけに維持されています。

> **メモ:** このプロパティが実際のブラウザー名を返すことを期待しないでください。どのブラウザーもこのプロパティの値として "`Netscape`" を返します。

## 構文

```js
appName = navigator.appName;
```

### 値

文字列 "`Netscape`" です。
