{{APIRef}}概要

`acceptCharset` は対象フォームでサポートされる文字エンコーディングのリストを返します。リストの各アイテムは、カンマか半角スペースで区切られます。

## 構文

```
string = form.acceptCharset;
```

## 例

```js
inputs = document.forms["myform"].acceptCharset
```
