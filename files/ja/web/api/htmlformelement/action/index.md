{{APIRef("HTML DOM")}}

**`HTMLFormElement.action`** プロパティは、この {{HTMLElement("form")}} 要素のアクションを表します。

フォームのアクションは、フォームが送信されたときにサーバー上で実行されるプログラムです。このプロパティは、取得または設定することができます。

## 構文

文字列です。

## 例

```js
form.action = "/cgi-bin/publish";
```
