{{APIRef("SVG")}}

**`SVGLinearGradientElement`** インターフェイスは {{SVGElement("linearGradient")}} 要素に対応します。

{{InheritanceDiagram}}

## インスタンスプロパティ

_このインターフェイスには、親である {{domxref("SVGGradientElement")}} から継承したプロパティもあります。_

- {{domxref("SVGLinearGradientElement.x1")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、この {{SVGElement("linearGradient")}} 要素の {{SVGAttr("x1")}} 属性に対応します。
- {{domxref("SVGLinearGradientElement.y1")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、この {{SVGElement("linearGradient")}} 要素の {{SVGAttr("y1")}} 属性に対応します。
- {{domxref("SVGLinearGradientElement.x2")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、この {{SVGElement("linearGradient")}} 要素の {{SVGAttr("x2")}} 属性に対応します。
- {{domxref("SVGLinearGradientElement.y2")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、この {{SVGElement("linearGradient")}} 要素の {{SVGAttr("y2")}} 属性に対応します。

## インスタンスメソッド

_このインターフェイスは固有のメソッドを実装していませんが、親インターフェイスである {{domxref("SVGGradientElement")}} からメソッドを継承しています。_
