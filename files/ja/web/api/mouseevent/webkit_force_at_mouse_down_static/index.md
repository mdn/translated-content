{{APIRef("UI Events")}}{{Non-standard_header}}

**`MouseEvent.WEBKIT_FORCE_AT_MOUSE_DOWN`** は非公式な WebKit 固有の静的な数値プロパティで、通常クリックの最小圧力の値です。

`WEBKIT_FORCE_AT_MOUSE_DOWN` は `MouseEvent` の静的プロパティなので、常に `MouseEvent.WEBKIT_FORCE_AT_MOUSE_DOWN` として使用してください。 `MouseEvent` インターフェイスのプロパティとしては使用できません。

## 仕様書

_どの仕様書にもありません。_ Apple は [Mac Developer Library で説明](https://developer.apple.com/library/archive/documentation/AppleApplications/Conceptual/SafariJSProgTopics/RespondingtoForceTouchEventsfromJavaScript.html)しています。
