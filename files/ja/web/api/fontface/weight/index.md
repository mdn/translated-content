{{APIRef("CSS Font Loading API")}}

**`weight`** は {{domxref("FontFace")}} インターフェイスのプロパティで、フォントの太さを取得または設定します。

このプロパティは {{cssxref("@font-face/font-weight", "font-weight")}} 記述子と等価です。

## 値

スタイルシートの `@font-face` ルールで定義するような記述子の入った文字列です。
