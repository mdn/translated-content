{{ APIRef("HTML DOM") }}

**`HTMLTableCaptionElement`** インターフェイスは（通常の {{domxref("HTMLElement")}} インターフェイス経由で継承によって利用できるものに加えて）表のキャプション要素を操作する特別なプロパティを提供します。

{{InheritanceDiagram}}

## プロパティ

_親である {{domxref("HTMLElement")}} から継承したプロパティがあります。_

- {{domxref("HTMLTableCaptionElement.align")}} {{deprecated_inline}}
  - : 文字列で、表に対するキャプションの配置を示す列挙型の属性を表します。

## メソッド

_固有のメソッドはありません。親である {{domxref("HTMLElement")}} から継承したメソッドがあります。_
