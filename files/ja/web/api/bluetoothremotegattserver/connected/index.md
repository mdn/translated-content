読み取り専用プロパティ **`BluetoothRemoteGATTServer.connected`** は、このスクリプト実行環境が `this.device` に接続されている間 `true` を返す論理値です。ユーザーエージェントが物理的に接続されていても `false` になる可能性があります。

## 値

`boolean` です。
