{{APIRef("Web Locks")}}

**`locks`** は {{domxref("WorkerNavigator")}} インターフェイスの読み取り専用プロパティで、 {{domxref("LockManager")}} オブジェクト返します。これは {{domxref('Lock')}} オブジェクトをリクエストしたり、既存の `Lock` オブジェクトを問い合わせるためのメソッドを提供します。

## 値

{{domxref("LockManager")}} オブジェクトです。
