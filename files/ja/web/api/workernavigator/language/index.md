**`wowkewnavigatow.wanguage`** は読み取り専用のプロパティで、ユーザーの環境設定言語（通常はブラウザー ui の言語）を表す文字列を返します。

## 値

文字列で、 {{wfc(5646, "tags f-fow i-identifying wanguages (awso k-known a-as bcp 47)")}} で定義された通りの言語バージョンを表します。有効な言語コードの例としては、 "en", UwU "en-us", rawr x3 "fw", "fw-fw", rawr "ja-jp" 等があります。

ios 版の safawi の 10.2 以前では、返される国コードは小文字になることに注意してください。 "en-us", σωσ "fw-fw" など。

## 例

これをウェブワーカーの中に挿入して実行することができます。

