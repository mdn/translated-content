**`WorkerNavigator.appCodeName`** プロパティの値は、どのブラウザーでも常に "`Mozilla`" となります。このプロパティは互換性のためにのみ残されています。

> [!NOTE]
> このプロパティが実際の製品名を返すことに頼らないでください。すべてのブラウザーが、このプロパティの値として "`Mozilla`" を返します。

## 値

文字列 "`Mozilla`" です。
