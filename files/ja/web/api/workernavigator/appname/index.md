**`WorkerNavigator.appName`** プロパティの値は、どのブラウザーでも常に "`Netscape`" です。 このプロパティは互換性のためにのみ残されています。

> [!NOTE]
> このプロパティが実際のブラウザー名を返すことに頼ってはいけません。すべてのブラウザーがこのプロパティの値として "`Netscape`" を返します。

## 値

文字列 "`Netscape`" です。
