**`ProgressEvent.lengthComputable`** は読み取り専用のプロパティであり、論理値のフラグで、 {{domxref("ProgressEvent")}} が関係するリソースに計算可能な長さがあるかどうかを示します。そうでない場合、{{domxref("ProgressEvent")}}プロパティは重要な値を持ちません。

## 値

論理値です。
