ブラウザーにおいて、**`document.defaultView`** はドキュメントに関連付けられている {{domxref("Window", "window")}} オブジェクトを返します。もし存在しない場合は `null` を返します。

このプロパティは読み取り専用です。

## 値

{{domxref("Window", "window")}} オブジェクトです。
