{{APIRef("Screen Orientation")}}

{{domxref("ScreenOrientation")}} インターフェイスの **`unlock()`** プロパティは、含んでいる文書の向きのデフォルトの向きへの固定を解除します。

## 構文

```js-nolint
unlock()
```

### 引数

なし

### 返値

なし ({{jsxref("undefined")}})
