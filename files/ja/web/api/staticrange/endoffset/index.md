{{APIRef("DOM")}}

**`endOffset`** は {{domxref("StaticRange")}} インターフェイスの読み取り専用プロパティで、範囲の終了位置の最後のノードの中のオフセットを返します。

## 値

範囲の最後の文字が位置する {{domxref("StaticRange.endContainer", "endContainer")}} が示す {{domxref("Node")}} 内の文字数を示す整数値です。
