{{APIRef("DOM")}}

**`endContainer`** は {{domxref("AbstractRange")}} インターフェイスの読み取り専用プロパティで、範囲の末尾がある {{domxref("Node")}} を返します。

## 値

範囲の最後の文字を含む {{domxref("Node")}} です。

## 例

```js
let endNode = range.endContainer;
```
