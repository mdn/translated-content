{{APIRef("DOM")}}

**`startContainer`** は {{domxref("AbstractRange")}} インターフェイスの読み取り専用プロパティで、範囲の先頭の {{domxref("Node")}} を返します。

## 値

範囲の開始位置が内部にある {{domxref("Node")}} です。

## 例

```js
let startNode = range.startContainer;
```
