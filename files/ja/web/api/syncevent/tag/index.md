{{domxref("SyncEvent")}} インターフェイスの **`SyncEvent.tag`** 読み取り専用プロパティは、この `SyncEvent` の開発者定義の識別子を返します。これは、 {{domxref("SyncEvent.SyncEvent","SyncEvent()")}} コンストラクタの `tag` パラメータで渡される値です。

## 構文

```
var tag = SyncEvent.tag
```

### 値

この `SyncEvent` の開発者定義の識別子。
