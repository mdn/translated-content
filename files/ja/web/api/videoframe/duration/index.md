{{APIRef("Web Codecs API")}}

**`duration`** は {{domxref("VideoFrame")}} インターフェイスのプロパティで、動画の長さをマイクロ秒単位で表す整数を返します。

## 値

整数です。

## 例

次の例は `duration` をコンソールに出力します。

```js
console.log(VideoFrame.duration);
```
