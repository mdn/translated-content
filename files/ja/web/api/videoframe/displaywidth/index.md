{{APIRef("Web Codecs API")}}

**`displayWidth`** は {{domxref("VideoFrame")}} インターフェイスのプロパティで、アスペクト比の調整後の `VideoFrame` の幅を返します。

## 値

整数です。

## 例

次の例は `displayWidth` をコンソールに出力されます。

```js
console.log(VideoFrame.displayWidth);
```
