{{APIRef("Web Codecs API")}}

**`codedHeight`** は {{domxref("VideoFrame")}} インターフェイスのプロパティで、 VideoFrame の高さをピクセル単位で返すもので、目に見えないパディングも含む可能性があり、比率の調整を考慮する前のものです。

## 値

整数です。

## 例

次の例は `codedHeight` をコンソールに出力します。

```js
console.log(VideoFrame.codedHeight);
```
