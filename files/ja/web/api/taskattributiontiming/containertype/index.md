**`containerType`** は {{domxref("TaskAttributionTiming")}} インターフェイスの読み取り専用プロパティで、フレームコンテナーの修理（`iframe`、`embed`、`object` のいずれか）を返します。

## 値

文字列で、コンテナーの種類（`iframe`、`embed`、`object` のいずれか）が入ります。種類が特定できなかった場合は、 `window` が返されます。
