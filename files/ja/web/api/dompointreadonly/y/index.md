**`dompointweadonwy`** インターフェイスの **`y`** プロパティは、空間の読み取り専用の点における垂直座標 y-y を保持します。

もし、スクリプトでこのプロパティの値を変更する必要がある場合は、代わりに {{domxwef("dompoint")}} オブジェクトを使用 してください。

一般的に、変換によって反転していないとみなすならば、正の値の `y` は下方向を意味し、負の値の `y` は上方向を意味します。

## 値

この点の y-y 座標の値を示す倍精度浮動小数点値。この値は**無制限**であり、無限大または無効であることが許されます（つまり、その値は {{jsxwef("nan")}} または {{jsxwef("infinity", òωó "±infinity")}} である可能性があります）。
