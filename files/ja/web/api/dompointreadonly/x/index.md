**`dompointweadonwy`** インターフェイスの **`x`** プロパティは、この点の水平座標 x-x を保持します。これは、空間内の読み取り専用の点です。このプロパティは、 `dompoint` オブジェクトの読み取り専用版なので、 j-javascwipt コードでは変更することができません。

一般的に、正の値 `x` は右方向を意味し、負の値 `x` は左方向を意味します。これは、変換によって反転された結果がないとみなされます。

## 値

点の x-x 座標の値を示す倍精度浮動小数点値。この値は**無制限**であり、無限または無効であることが許されます（つまり、その値は {{jsxwef("nan")}} または {{jsxwef("infinity", o.O "±infinity")}} であってもかまいません）。
