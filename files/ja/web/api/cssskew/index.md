{{APIRef("CSS Typed OM")}}{{SeeCompatTable}}

**`CSSSkew`** は {{domxref('CSS_Object_Model#css_typed_object_model','CSS 型付きオブジェクトモデル','',' ')}}のインターフェイスで、 {{domxref('CSSTransformValue')}} インターフェイスの一部です。これは CSS における個々の {{CSSXRef('transform')}} プロパティの [`skew()`](/ja/docs/Web/CSS/transform-function/skew) の値を表します。

{{InheritanceDiagram}}

## コンストラクター

- {{domxref("CSSSkew.CSSSkew", "CSSSkew()")}} {{Experimental_Inline}}
  - : 新しい `CSSSkew` オブジェクトを作成します。

## インスタンスプロパティ

- {{domxref('CSSSkew.ax','ax')}} {{Experimental_Inline}}
  - : X 軸の値を返したり設定したりします。
- {{domxref('CSSSkew.ay','ay')}} {{Experimental_Inline}}
  - : Y 軸の値を返したり設定したりします。
