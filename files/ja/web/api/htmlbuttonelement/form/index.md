{{APIRef("HTML DOM")}}

**`form`** は {{domxref("HTMLButtonElement")}} インターフェイスの読み取り専用プロパティで、この {{HTMLElement("button")}} を所有する {{domxref("HTMLFormElement")}} オブジェクトを返し、このボタンがどのフォームにも所有されていない場合は `null` を返します。

## 値

{{domxref("HTMLFormElement")}} または `null` です。
