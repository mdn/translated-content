**`TransitionEvent.elapsedTime`** は読み取り専用プロパティで、このイベントが発行された時にアニメーションが実行されていた秒数を
`float` 型で返します。この値は {{cssxref("transition-delay")}} プロパティの影響を受けません。

## 値

数値です。
