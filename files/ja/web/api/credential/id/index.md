{{APIRef("Credential Management API")}}

**`id`** は {{domxref("Credential")}} インターフェイスのプロパティで、この資格情報の識別子の入った文字列を返します。これは GUID、ユーザー名、メールアドレスなどになる可能性があります。

## 値

資格情報の識別子が入った文字列です。
