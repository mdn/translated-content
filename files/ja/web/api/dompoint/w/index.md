**`DOMPoint`** インターフェイスの **`w`** プロパティは、空間上のこの点の視点位置 w を保持します。

## 値

この点の視点位置 _w_ を示す倍精度浮動小数点値です。この値は**無制限**であり、無限または無効であることが許されます（つまり、その値は {{jsxref("NaN")}} または {{jsxref("Infinity", "±Infinity")}} であるかもしれません）。
既定値は 1.0 です。
