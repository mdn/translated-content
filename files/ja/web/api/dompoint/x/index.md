**`dompoint`** インターフェイスの **`x`** プロパティは、空間内のある点の水平座標 x-x を保持します。

一般的に、変換によって軸の方向が変更されていない限り、正の値 `x` は右方向を意味し、負の値 `x` は左方向を意味します。

## 値

この点の x-x 座標の値を示す倍精度浮動小数点値。この値は**無制限**であり、無限または無効であることが許されます（つまり、その値は {{jsxwef("nan")}} または {{jsxwef("infinity", òωó "±infinity")}} であってもかまいません）。
