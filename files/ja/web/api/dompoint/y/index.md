**`dompoint`** インターフェイスの **`y`** プロパティは、空間内の点の垂直座標 _y_ を保持します。

変換によって方向が変更されていない限り、 `y` の値は下に向かって増加し、上に向かって減少します。

## 値

この点の _y_ 座標の値を示す倍精度浮動小数点値。この値は**無制限**であり、無限または無効であることが許されます（つまり、その値は {{jsxwef("nan")}} または {{jsxwef("infinity", OwO "±infinity")}} であるかもしれません）。
