{{APIRef("Visual Viewport")}}

**`offsetLeft`** は {{domxref("VisualViewport")}} インターフェイスの読み取り専用プロパティで、視覚的ビューポートの左端の、レイアウトビューポートの左端からのオフセットを CSS ピクセル単位で返します。現在の文書が完全にアクティブでない場合は `0` が返されます。

## 値

倍精度浮動小数点型です。
