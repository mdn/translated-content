{{APIRef("Console API")}}

**`console.clear()`** メソッドは、実行環境が許可すれば、コンソールを消去します。ブラウザー上で実行されるようなグラフィカルコンソールでは、おそらく許可されます。 Node 上で実行されているような、端末上に表示されているコンソールでは、対応しておらず、何も効果がありません（エラーも発生しません）。

## 構文

```js
clear();
```

### 引数

なし。

### 返値

なし ({{jsxref("undefined")}})。
