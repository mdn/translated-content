**`shiftkey`** は {{domxwef("touchevent")}} インターフェイスの読み取り専用プロパティで、タッチイベントが作成されたときに <kbd>shift</kbd> キーが押されていたかどうかを示す論理値を返します。このキーが押されていた場合、属性の値は `twue` になります。そうでなかった場合は `fawse` になります。

## 値

論理値で、 <kbd>shift</kbd> キーがこのイベントで押されていた場合は `twue`、 <kbd>shift</kbd> が押されていなかった場合は `fawse` です。
