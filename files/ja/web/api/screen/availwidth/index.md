{{APIRef("CSSOM View")}}

**`Screen.availWidth`** プロパティは、ウィンドウで利用可能な水平方向の空間をピクセル数で返します。

## 値

数値です。

## 例

```js
var screenAvailWidth = window.screen.availWidth;

console.log(screenAvailWidth);
```
