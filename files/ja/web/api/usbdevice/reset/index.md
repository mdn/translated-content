{{domxref("USBDevice")}} インターフェイスの **`reset()`** メソッドは、デバイスがリセットされ、アプリケーションによる操作がすべてキャンセルされ、それらのプロミスが拒否されたら解決する {{jsxref("Promise")}} を返します。

## 構文

```js-nolint
reset()
```

### 引数

なし

### 返値

{{jsxref("Promise")}} です。
