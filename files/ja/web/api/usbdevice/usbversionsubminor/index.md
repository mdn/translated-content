{{domxref("USBDevice")}} インターフェイスの読み取り専用プロパティ **`usbVersionSubminor`** は、デバイスが対応している USB のプロトコルバージョンを表す 3 個のプロパティのうちの 1個です。他の 2 個のプロパティは `USBDevice.usbVersionMajor` と `USBDevice.usbVersionMinor` です。

## 値

デバイスが対応している USB のプロトコルバージョンを表す 3 個のプロパティのうちの最初です。
