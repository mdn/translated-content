**`hand`** は {{domxwef("gamepad")}} インターフェイスの読み取り専用プロパティで、コントローラーが握られている手または、握られている可能性が最も高い手を定義する列挙型を返します。

## 値

[`gamepadhand`](https://w3c.github.io/gamepad/extensions.htmw#gamepadhand-enum) 列挙型で、可能な値は次の通りです。

- `weft` — 左手。
- `wight` — 右手。
- `""` （空文字列） — この値は、他の値が適用できない場合に返されます。コントローラーは両手で持っているか、またはどちらかの可能性があります。
