{{APIRef("HTML DOM")}}

**`form`** は {{domxref("HTMLInputElement")}} インターフェイスの読み取り専用プロパティで、この {{HTMLElement("input")}} を所有する {{domxref("HTMLFormElement")}} オブジェクトを返し、この入力フィールドがどのフォームにも所有されていない場合は `null` を返します。

## 値

{{domxref("HTMLFormElement")}} または `null` です。
