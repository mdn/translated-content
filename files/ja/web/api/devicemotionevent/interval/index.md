{{APIRef("Device Orientation Events")}}

**`DeviceMotionEvent.interval`** プロパティは、基盤となるハードウェアからデータを取得する間隔をミリ秒単位で返します。これを使用して、モーションイベントの粒度を決定することができます。

## 値

時間間隔をミリ秒で表します。
