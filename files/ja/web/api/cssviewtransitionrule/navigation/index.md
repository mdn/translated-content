{{APIRef("CSSOM")}}

**`navigation`** は {{domxref("CSSViewTransitionRule")}} インターフェイスの読み取り専用プロパティで、関連付けられた {{cssxref("@view-transition")}} アットルールの `navigation` 記述子の値を返します。

## 値

文字列です。

`@view-transition` アットルールに `navigation` 記述子が設定されていない場合、`navigation` プロパティは空文字列となります。

## 例

使い方の例については {{domxref("CSSViewTransitionRule")}} ページを参照してください。
