{{APIRef("CSSOM")}}

**`types`** は {{domxref("CSSViewTransitionRule")}} インターフェイスの読み取り専用プロパティで、関連付けられた {{cssxref("@view-transition")}} アットルールの `types` 記述子の値の入った配列を返します。

## 値

文字列の配列です。

`@view-transition` アットルールに `types` 記述子が設定されていなかった場合、`types` プロパティは空の配列となります。

## 例

使い方の例については {{domxref("CSSViewTransitionRule")}} ページを参照してください。
