{{APIRef("Picture-in-Picture API")}}

{{domxref("PictureInPictureWindow")}} の読み取り専用プロパティである **`height`** は、浮動動画ウィンドウの高さをピクセル単位で返します。

## 構文

```js-nolint
pictureInPictureWindow.height
```

### 値

整数で、浮動動画ウィンドウの高さをピクセル単位で示します。このプロパティは読み取り専用で、既定値はありません。
