**`SVGLineElement`** インターフェイスは {{ SVGElement("line") }} 要素のプロパティへのアクセス手段や、操作を行うメソッドを提供します。

{{InheritanceDiagram}}

## インスタンスプロパティ

_このインターフェイスには、親である {{domxref("SVGGeometryElement")}} から継承したプロパティもあります。_

- {{domxref("SVGLineElement.x1")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、この {{SVGElement("line")}} 要素の {{SVGAttr("x1")}} 属性に対応します。
- {{domxref("SVGLineElement.y1")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、この {{SVGElement("line")}} 要素の {{SVGAttr("y1")}} 属性に対応します。
- {{domxref("SVGLineElement.x2")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、この {{SVGElement("line")}} 要素の {{SVGAttr("x2")}} 属性に対応します。
- {{domxref("SVGLineElement.y2")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、この {{SVGElement("line")}} 要素の {{SVGAttr("y2")}} 属性に対応します。
