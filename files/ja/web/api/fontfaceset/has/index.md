{{APIRef("CSS Font Loading API")}}

**`has()`** は {{domxref("FontFaceSet")}} インターフェイスのメソッドで、指定された値を持つ要素が存在するかどうかを検査して {{jsxref("Boolean")}} を返します。

## 構文

```js-nolint
has(value)
```

### 引数

- `value`
  - : `FontFaceSet` オブジェクトでテストする値。

### 返値

論理値で、 `value` が `FontFaceSet` に存在する場合に `true` となります。
