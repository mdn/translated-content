{{APIRef("CSS Font Loading API")}}

**`entries()`** は {{domxref("FontFaceSet")}} インターフェイスのメソッドで、 `FontFaceSet` のそれぞれの要素に対して `[value,value]` の配列を含む新しい {{jsxref("Iterator")}} オブジェクトを返します。

## 構文

```js-nolint
entries()
```

### 引数

なし。

### 返値

`CustomStateSet` のそれぞれの要素に対して `[value, value]` の配列を挿入順に保持する、新しいイテレーターオブジェクトです。
