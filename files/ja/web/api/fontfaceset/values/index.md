{{APIRef("CSS Font Loading API")}}

**`values()`** は {{domxref("FontFaceSet")}} インターフェイスのメソッドで、 `FontFaceSet` オブジェクトの各要素の値を挿入順に返す新しいイテレーターオブジェクトを返します。

## 構文

```js-nolint
values()
```

### 引数

なし。

### 返値

指定された `FontFaceSet` の各要素の値を挿入順に返す、新しいイテレーターオブジェクトです。
