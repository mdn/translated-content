{{APIRef("CSS Font Loading API")}}

**`keys()`** は {{domxref("FontFaceSet")}} インターフェイスのメソッドで、 {{domxref("FontFaceSet.values")}} の別名です。

## 構文

```js-nolint
keys()
```

### 引数

なし。

### 返値

指定された `FontFaceSet` の各要素の値を挿入順に保持する、新しいイテレーターオブジェクトです。
