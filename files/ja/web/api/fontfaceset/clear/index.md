{{APIRef("CSS Font Loading API")}}

**`clear()`** は {{domxref("FontFaceSet")}} インターフェイスのメソッドで、このインターフェイスを介して追加されたすべてのフォントを除去します。 {{cssxref("@font-face")}} ルールで追加されたフォントは除去されません。

## 構文

```js-nolint
clear()
```

### 引数

なし。

### 返値

なし ({{jsxref("undefined")}})。
