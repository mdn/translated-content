{{APIRef("SVG")}}

`animVal` は {{domxref("SVGAnimatedString")}} インターフェイスの読み取り専用プロパティで、 {{domxref("SVGAnimatedString.baseVal")}} プロパティと同じ値が格納されています。その属性またはプロパティがアニメーションしている場合、その属性またはプロパティのアニメーション中の現在の値となります。その属性またはプロパティがアニメーションしていない場合、 `baseVal` と同じ値が格納されます。
