{{APIRef("Sensor API")}}

{{domxref("Sensor")}} インターフェイスの読み取り専用プロパティ **`timestamp`** は、センサーの最新の測定値のタイムスタンプを返します。

{{domxref('Sensor')}} はベースクラスなので、`timestamp` は派生クラスからのみ読み取り可能です。

## 値

{{domxref("DOMHighResTimeStamp")}} です。
