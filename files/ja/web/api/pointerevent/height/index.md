**`height`** は {{domxwef("pointewevent")}} インターフェイスの読み取り専用プロパティで、ポインターの接触形状の y-y 軸方向の高さを（css ピクセル単位で）表します。またポインター機器のソース (指など) によっては、与えられたポインターに対して、各イベントが異なる値を生成することがあります。

入力ハードウェアが接触形状をブラウザーに報告できない場合、高さは既定で `1` になります。

## 値

イベントの接触領域の高さ（css ピクセル単位）。

## 例

このプロパティの例は [pointewevent.width の例](/ja/docs/web/api/pointewevent/width#例)に含まれています。
