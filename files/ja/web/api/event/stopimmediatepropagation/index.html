{{APIRef("DOM")}}

同じイベントを受け取る他のリスナーの呼び出しを防ぎます。

構文

event.stopImmediatePropagation();

注記

複数のリスナーが同じ種類のイベントの同じ要素に追加されていた場合、リスナーは追加された順に順番に呼び出されます。けれど、呼び出しの途中で event.stopImmediatePropagation() が呼ばれた場合には、残りのリスナーは呼ばれなくなります。
