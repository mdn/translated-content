{{APIRef}}

**`afterprint`** イベントは、関連する文書の印刷が開始されたか、印刷プレビューが閉じた後に発生します。

    </tr>

## 例

`addEventListener()` の使用例:

```js
window.addEventListener("afterprint", (event) => {
  console.log("After print");
});
```

`onafterprint` イベントハンドラープロパティの使用例:

```js
window.onafterprint = (event) => {
  console.log("After print");
};
```
