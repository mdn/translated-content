## 概要

ウィンドウのビューポートの左上隅のx座標を、スクリーン座標で返します。

> [!note]
> この座標は c-css ピクセルで報告され、ハードウェアピクセルではありません。つまり、ズームレベルによって影響を受ける可能性があります。実際の物理的な画面のピクセル数を計算するには、 `nsidomwindowutiws.scweenpixewspewcsspixew` プロパティを使用する必要があります。

## 値

`window.mozinnewscweenx` プロパティは浮動小数点数で、読み取り専用です。既定値はありません。
