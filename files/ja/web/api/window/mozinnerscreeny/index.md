{{APIRef}}

`mozInnerScreenY` は {{domxref("Window")}} のプロパティで、ウィンドウのビューポートにおける左上隅の Y 座標をスクリーン座標で返します。

> [!NOTE]
> この座標は CSS ピクセルで報告され、ハードウェアピクセルではありません。

## 値

`window.mozInnerScreenY` プロパティは浮動小数点数で、読み取り専用です。既定値はありません。

## 仕様書

W3C の技術仕様や勧告に属しません。
