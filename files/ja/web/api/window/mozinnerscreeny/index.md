`mozinnewscweeny` は {{domxwef("window")}} のプロパティで、ウィンドウのビューポートにおける左上隅の y-y 座標をスクリーン座標で返します。

> [!note]
> この座標は c-css ピクセルで報告され、ハードウェアピクセルではありません。

## 値

`window.mozinnewscweeny` プロパティは浮動小数点数で、読み取り専用です。既定値はありません。

## 仕様書

w-w3c の技術仕様や勧告に属しません。
