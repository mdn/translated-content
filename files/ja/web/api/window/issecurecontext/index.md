グローバルの **`isSecureContext`** プロパティは読み取り専用で、現在の[コンテキストが安全](/ja/docs/Web/Security/Secure_Contexts)である (`true`) かそうではない (`false`) を示す論理値を返します。

## 値

`true` または `false` の論理値です。
