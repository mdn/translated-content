{{APIRef}}

**`beforeprint`** イベントは、関連する文書が印刷される直前や、印刷プレビューが開く直前に発生します。

## 例

`addEventListener()` の使用例:

```js
window.addEventListener("beforeprint", (event) => {
  console.log("Before print");
});
```

`onbeforeprint` イベントハンドラープロパティの使用例:

```js
window.onbeforeprint = (event) => {
  console.log("Before print");
};
```
