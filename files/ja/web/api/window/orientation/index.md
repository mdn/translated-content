ビューポートの向きを、端末の自然な向きに対する角度（90 度単位）で返します。

可能な値は `-90`、`0`、`90`、`180` のみです。正の値は反時計回り、負の値は時計回りを表します。

これは非推奨のプロパティです。代わりに、{{domxref("window.screen")}} プロパティで利用できる {{domxref("Screen.orientation")}} プロパティを使用してください。
