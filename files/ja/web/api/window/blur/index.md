{{APIRef}}

ウィンドウからフォーカスを外します。

## 構文

```js
blur();
```

### 引数

なし。

### 返値

なし ({{jsxref("undefined")}})。

## 例

```js
window.blur();
```

## メモ

window\.blur() メソッドは、プログラム的にユーザーが現在のウィンドウからフォーカスを移した場合に等価です。
