**`unlock()`** は {{domxref("Keyboard")}} インターフェイスのメソッドで、 {{domxref('Keyboard.lock()')}} メソッドによってキャプチャされているすべてのキーのロックを解放し、同期的に返ります。

## 構文

```js-nolint
unlock()
```

### 引数

なし。

### 返値

なし ({{jsxref("undefined")}})。
