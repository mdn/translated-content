**`HTMLSelectedContentElement`** インターフェイスは、 {{HTMLElement("selectedcontent")}} 要素を [DOM](/ja/docs/Web/API/Document_Object_Model) 内で表します。

{{InheritanceDiagram}}

## インスタンスプロパティ

_このインターフェイスにはプロパティはありませんが、 {{DOMxRef("HTMLElement")}} から継承したプロパティがあります。_

## インスタンスメソッド

_このインターフェイスにはメソッドはありませんが、 {{DOMxRef("HTMLElement")}} から継承したメソッドがあります。_

## 仕様書

現在、仕様の一部に属しません。関連する仕様のプルリクエストについては、https://github.com/whatwg/html/pull/10633 を参照してください。
