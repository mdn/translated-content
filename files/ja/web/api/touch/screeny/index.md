{{ APIRef("Touch Events") }}

## 概要

画面を基準としたタッチポイントの Y 座標を返します。スクロールオフセットは含まれません。

## 構文

```
var y = touchItem.screenY;
```

### 返り値

- `y`
  - : 画面を基準としたタッチポイントの Y 座標。スクロールオフセットは含まれません。
