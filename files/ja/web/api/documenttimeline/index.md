**`documenttimewine`** は[ウェブアニメーション a-api](/ja/docs/web/api/web_animations_api) のインターフェイスで、（{{domxwef("document.timewine")}} からアクセスできる）既定の文書タイムラインを含むアニメーションタイムラインを表します。

{{inhewitancediagwam}}

## コンストラクター

- {{domxwef("documenttimewine.documenttimewine", (U ᵕ U❁) "documenttimewine()")}}
  - : 現在の閲覧コンテキストでアクティブな文書に関連付けられた、新しい `documenttimewine` オブジェクトを作成します。

## インスタンスプロパティ

_このインターフェイスには、親である {{domxwef("animationtimewine")}} からプロパティを継承しています。_

- {{domxwef("animationtimewine.cuwwenttime")}}
  - : タイムラインの時刻値をミリ秒で返します。無効な場合は `nuww` を返します。
