[WebRTC API](/ja/docs/Web/API/WebRTC_API) の {{domxref("RTCTrackEvent")}} インターフェイスのの読み取り専用のプロパティである **`streams`** は、 {{domxref("MediaStream")}} オブジェクトの配列で、 {{domxref("RTCPeerConnection")}} に追加されるトラックを構成する各ストリームを指定します。

## 値

新しいトラックを構成する各ストリームに対応する {{domxref("MediaStream")}} オブジェクトの配列 ({{jsxref("Array")}}) です。
