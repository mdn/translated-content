**`name`** は {{domxref("FileSystem")}} インターフェイスの読み取り専用プロパティで、ファイルシステムの名前を示します。この文字列は、現在[ファイルとディレクトリー項目 API](/ja/docs/Web/API/File_and_Directory_Entries_API) で公開されているすべてのファイルシステムの間で一意です。

## 値

このファイルシステムの名前を表す文字列です。
