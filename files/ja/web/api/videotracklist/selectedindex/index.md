読み取り専用の **{{domxref("VideoTrackList")}}** の **`selectedIndex`** プロパティは、現在選択されているトラックがある場合はその位置を返し、それ以外の場合は `-1` を返します。

## 値

現在選択されているトラックがある場合はその位置を示す数値、それ以外の場合は `-1`。
