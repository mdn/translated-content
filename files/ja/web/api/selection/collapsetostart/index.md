**`Selection.collapseToStart()`** メソッドは、選択範囲をその範囲内の最初の範囲の先頭まで折りたたみます。選択範囲の内容にフォーカスが当たっており、編集可能な状態の場合、その位置でカーソルが点滅します。

## 構文

```js-nolint
collapseToStart()
```

### 引数

なし。

### 返値

なし ({{jsxref("undefined")}})。
