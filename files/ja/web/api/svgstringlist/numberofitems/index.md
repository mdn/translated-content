**`numberOfItems`** は {{domxref("SVGStringList")}} インターフェイスのプロパティで、リスト内のアイテム数を返します。 {{domxref("SVGStringList.length", "length")}} はこの別名です。

## 値

リストのアイテム数を表す非負の整数です。
