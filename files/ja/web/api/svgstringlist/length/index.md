{{APIRef("SVG")}}

**`length`** は {{domxref("SVGStringList")}} インターフェイスのプロパティで、リスト内のアイテム数を返します。これは、 {{domxref("SVGStringList.numberOfItems", "numberOfItems")}} の別名であり、 SVG のリストをより[配列風](/ja/docs/Web/JavaScript/Reference/Global_Objects/Array#配列風オブジェクト)にするためのものです。

## 値

リストのアイテム数を表す非負の整数です。
