**`DeviceMotionEventRotationRate`** は {{domxref("Device Orientation Events", "端末方向イベント", "", "nocode")}}のインターフェイスで、端末が回転する 3 つの軸すべての速度に関する情報を提供します。

## インスタンスプロパティ

- {{ domxref("DeviceMotionEventRotationRate.alpha") }} {{ReadOnlyInline}}
  - : Z 軸上の回転量を秒あたりの角度で表します。
- {{ domxref("DeviceMotionEventRotationRate.beta") }} {{ReadOnlyInline}}
  - : X 軸上の回転量を秒あたりの角度で表します。
- {{ domxref("DeviceMotionEventRotationRate.gamma") }} {{ReadOnlyInline}}
  - : Y 軸上の回転量を秒あたりの角度で表します。
