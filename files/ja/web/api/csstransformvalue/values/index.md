**`CSSTransformValue.values()`** は、 CSSTransformValue オブジェクトの各位置に対応する値を格納した新しい _配列イテレーター_ オブジェクトを返します。

## 構文

```js-nolint
values()
```

### 引数

なし。

### 返値

新しい {{jsxref("Array")}} です。
