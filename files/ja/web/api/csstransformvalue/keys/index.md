**`CSSTransformValue.keys()`** メソッドは、配列の各インデックスに対応するキーを格納した新しい _配列イテレーター_ オブジェクトを返します。

## 構文

```js-nolint
keys()
```

### 引数

なし。

### 返値

新しい {{jsxref("Array")}} です。
