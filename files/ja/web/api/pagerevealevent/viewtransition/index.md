**`viewTransition`** は {{domxref("PageRevealEvent")}} インターフェイスの読み取り専用プロパティで、文書間のナビゲーションでアクティブなビュー遷移を表す {{domxref("ViewTransition")}} オブジェクトが含まれています。

## 値

{{domxref("ViewTransition")}} オブジェクト、またはイベントが発行される際にアクティブなビュー遷移がない場合は `null`。
