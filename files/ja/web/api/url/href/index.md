{{ApiRef("URL API")}}

**`href`** は {{domxref("URL")}} インターフェイスのプロパティで、URL 全体の入った文字列です。

{{AvailableInWorkers}}

## 値

文字列です。

## 例

```js
const url = new URL("https://developer.mozilla.org/ja/docs/Web/API/URL/href");
console.log(url.href); // 'https://developer.mozilla.org/ja/docs/Web/API/URL/href' とログ出力
```
