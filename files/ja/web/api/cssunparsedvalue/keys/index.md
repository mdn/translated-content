**`CSSUnparsedValue.keys()`** メソッドは、配列内のそれぞれの添字のキーを持つ新しい配列イテレーターオブジェクトを返します。

## 構文

```js-nolint
keys()
```

### 引数

なし。

### 返値

新しい {{jsxref("Array")}}。
