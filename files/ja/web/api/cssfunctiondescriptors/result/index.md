**`result`** は {{domxref("CSSFunctionDescriptors")}} インターフェイスのプロパティで、`result` 記述子が関係する一連の宣言に存在すれば、それを表す文字列を返します。

## 値

文字列です。

関連付けられた一連の宣言に `result` 記述子が存在しない場合は、空の配列が返されます。
