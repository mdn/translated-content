{{APIRef("MediaStream Recording")}}

**videoBitsPerSecond`** は {{domxref("MediaRecorder")}} インターフェイスの読み取り専用プロパティは、使用されている動画エンコーディングのビットレートを返します。

これは、コンストラクターで指定されたビットレートが提供された場合、そのビットレートとは異なる形になる可能性があります。

## 値

数値 (unsigned long) です。
