{{APIRef("MediaStream Recording")}}

**`audioBitsPerSecond`** は {{domxref("MediaRecorder")}} インターフェイスの読み取り専用のプロパティで、使用されている音声エンコーディングのビットレートを返します。

これは、コンストラクターで指定されたビットレート（提供された場合）と異なる場合があります。

## 値

数値 (unsigned long) です。
