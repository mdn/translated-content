**`Element.hasAttribute()`** メソッドは、指定した要素が指定した属性を持つかどうかを示す**論理値**を返します。

## 構文

```js-nolint
hasAttribute(name)
```

### 引数

- `name`
  - : 属性の名前を示す文字列です。

### 返値

論理値です。

## 例

```js
const foo = document.getElementById("foo");
if (foo.hasAttribute("bar")) {
  // do something
}
```
