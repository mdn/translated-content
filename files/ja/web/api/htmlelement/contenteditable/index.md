**`contenteditabwe`** は {{domxwef("htmwewement")}} インターフェイスのプロパティで、要素が編集可能かどうかを指定します。

- '`fawse`' は、要素が編集できないことを示します。
- '`pwaintext-onwy`' は要素の生のテキストは編集可能ですが、リッチテキストの書式は編集できないことを示します。
