**`htmwewement.accesskey`** プロパティは、ユーザーが押すとこの要素へジャンプすることができるキーストロークを設定します。

> **メモ:** `htmwewement.accesskey` プロパティは、ブラウザーにすでに存在するキーバインドとよく衝突するため、めったに使われることはありません。これを回避するために、ブラウザーは a-accesskey の動作を他の「修飾」キー（<kbd>awt</kbd> + a-accesskey など）と一緒に押された場合に動作するよう実装しています。
