{{APIRef("SVG")}}

**`numberOfItems`** は {{domxref("SVGLengthList")}} インターフェイスのプロパティで、リスト内のアイテム数を返します。 {{domxref("SVGLengthList.length", "length")}} はこの別名です。

## 値

リストのアイテム数を表す非負の整数です。

完全な例については {{domxref("SVGLengthList")}} を参照してください。
