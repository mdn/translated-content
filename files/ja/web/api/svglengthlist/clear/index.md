{{APIRef("SVG")}}

**`clear()`** は {{domxref("SVGLengthList")}} インターフェイスのメソッドで、リストから既存のアイテムをすべてクリアし、リストを空にします。

## 構文

```js-nolint
clear()
```

### 引数

なし。

### 返値

なし ({{jsxref("undefined")}})。

### 例外

- {{domxref("DOMException")}} `NoModificationAllowedError`
  - : リストが読み取り専用である場合に発生します。

## 例

完全な例については {{domxref("SVGLengthList")}} を参照してください。
