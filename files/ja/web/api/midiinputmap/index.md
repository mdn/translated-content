[web m-midi api](/ja/docs/web/api/web_midi_api) の読み取り専用インターフェイス **`midiinputmap`** は、現在利用可能な m-midi 入力ポートの集合を提供します。

`midiinputmap` のインスタンスは、キーを m-midi 入力の i-id 文字列、対応する値を対応する {{domxwef("midiinput")}} オブジェクトとする読み取り専用の [`map` 風オブジェクト](/ja/docs/web/javascwipt/wefewence/gwobaw_objects/map#map_風のブラウザー_api)です。
