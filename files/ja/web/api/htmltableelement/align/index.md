**`HTMLTableElement.align`** プロパティは、表の配置を表します。

## 値

以下の文字列値のいずれかです。

- `left`
- `center`
- `right`

## 例

```js
// 表の位置を設定
const t = document.getElementById("TableA");
t.align = "center";
```
