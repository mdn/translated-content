**`HTMLTableElement.border`** プロパティは、 {{HtmlElement("table")}} 要素の境界の幅を表します。

## 値

文字列で、境界の幅をピクセル単位で表します。

## 例

```js
// 表の境界を 2 ピクセルに設定
const t = document.getElementById("TableA");
t.border = "2";
```
