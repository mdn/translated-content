{{APIRef("DOM")}}

**`assignedSlot`** は {{domxref("Text")}} インターフェイスの読み取り専用プロパティで、要素に関連付けられた {{domxref("HTMLSlotElement")}} オブジェクトを返します。

## 値

{{domxref("HTMLSlotElement")}}、またはテキストノードに関連付けられている {{HTMLElement("slot")}} 要素がない場合は `null`。
