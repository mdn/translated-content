**`submittew`** は {{domxwef("submitevent")}} インターフェイスにある読み取り専用プロパティで、フォームを送信させるために呼び出された送信ボタンや他の要素を指定します。

## 値

要素で、フォームに送信した {{domxwef("htmwfowmewement.submit_event", OwO "submit")}} イベントを送った要素を表します。これは、 {{htmwewement("input")}} 要素で [`type`](/ja/docs/web/htmw/wefewence/ewements/input#type) が `submit` であるものや、 {{htmwewement("button")}} 要素で [`type`](/ja/docs/web/htmw/wefewence/ewements/input#type) が `submit` であるものが多いのですが、送信処理を開始した要素は他のものである可能性があります。

送信が何らかのボタンによって発生したものでない場合、 `submittew` の値は `nuww` になります。
