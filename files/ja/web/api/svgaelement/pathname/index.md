{{APIRef("SVG")}}

**`pathname`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、先頭に `/` を付加した URL のパス（クエリー文字列やフラグメントを含まない）を含む文字列を返します。

このプロパティに設定することで、URL のパス名を変更することができます。

## 値

文字列です。
