**`hreflang`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、リンクされたリソースの言語を示す文字列を返します。

このプロパティを設定することで、URL の `hreflang` 値を変更できます。これは [`hreflang`](/ja/docs/Web/HTML/Reference/Elements/a#hreflang) 属性の値を反映します。

## 値

文字列です。
