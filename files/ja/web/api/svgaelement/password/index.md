{{APIRef("SVG")}}

**`password`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、ドメイン名の前に指定されたパスワードを含む文字列を返します。

このプロパティを設定することで、URL のパスワードを変更することができます。

## 値

文字列です。
