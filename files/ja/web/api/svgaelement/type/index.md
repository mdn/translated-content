**`type`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、 リンクされているリソースの MIME タイプを示す文字列を返します。

このプロパティを設定することで、URL の `type` 値を変更できます。これは [`type`](/ja/docs/Web/HTML/Reference/Elements/a#type) 属性の値を反映します。

## 値

文字列です。
