{{APIRef("SVG")}}

**`search`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、URL のクエリー文字列（存在する場合）を表す文字列を返します。先頭の疑問符 (`?`) を含みます。

このプロパティを設定することで、URL のクエリー成分を変更することができます。

## 値

文字列です。
