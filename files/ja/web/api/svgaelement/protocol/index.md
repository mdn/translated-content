{{APIRef("SVG")}}

**`protocol`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、参照された URL のプロトコル部分（末尾のコロン (`:`) を含む）を表す文字列を返します。

このプロパティを設定することで、URL のプロトコルを変更可能です。

## 値

文字列です。
