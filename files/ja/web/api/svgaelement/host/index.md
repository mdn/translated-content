{{APIRef("SVG")}}

**`host`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、参照された URL のホスト名とポート番号（デフォルトポートでない場合）を表す文字列を返します。

このプロパティを設定することで、URL のホストとポート番号が変更することができます。

## 値

文字列です。
