{{APIRef("SVG")}}

**`hostname`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、参照先の URL のホスト名を表す文字列を返します。

このプロパティを設定することで、URL のホスト名を変更することができます。

## 値

文字列です。
