{{APIRef("SVG")}}

**`port`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、参照された URL のポート番号成分（存在する場合）を表す文字列を返します。

このプロパティを設定することで、URL のポート番号を変更可能です。

## 値

文字列です。
