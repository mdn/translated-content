{{APIRef("SVG")}}

**`hash`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、 参照された URL 内のフラグメント識別子（先頭のハッシュ記号 (`#`) がある場合はそれを含む）を表す文字列を返します。

このプロパティを設定すると、URLのフラグメントを変更することが可能です（先頭の `#` は無視されます）。

## 値

文字列です。
