{{APIRef("SVG")}}

**`download`** は {{domxref("SVGAElement")}} インターフェイスのプロパティで、リンクされた URL をダウンロードとして扱うべきであることを示す文字列を返します。

このプロパティを設定することで、URL の `download` 値を変更できます。これは [`download`](/ja/docs/Web/HTML/Reference/Elements/a#download) 属性の値を反映します。

> [!NOTE]
> この値はダウンロードに使用されないことがあります。この値はダウンロードが行われるかどうかを判断するために使用できません。

## 値

文字列です。
