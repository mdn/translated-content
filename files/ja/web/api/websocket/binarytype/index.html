{{APIRef("Web Sockets API")}}

WebSocket.binaryType は、コネクションによって送信されているバイナリデータの型を返すプロパティです。

構文

var binaryType = aWebSocket.binaryType;

値

{{DOMXref("DOMString")}}:

"blob"
{{domxref("Blob")}} オブジェクトが使われている場合
"arraybuffer"
{{jsxref("ArrayBuffer")}} オブジェクトが使われている場合
