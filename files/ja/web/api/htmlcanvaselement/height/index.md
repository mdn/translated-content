{{APIRef("Canvas API")}}

**`HTMLCanvasElement.height`** プロパティは CSS ピクセル数として解釈される正の整数であり、この {{HTMLElement("canvas")}} 要素の [`height`](/ja/docs/Web/HTML/Element/canvas#height) 属性を反映します。この属性が指定されなかった場合、または負の数など、無効な値をが設定された場合、既定値の `150` が使用されます。

これはキャンバスのサイズを制御する 2 つのプロパティのうちの 1 つで、もう一方は {{domxref("HTMLCanvasElement.width")}} です。

## 値

数値です。

## 例

この {{HTMLElement("canvas")}} 要素があったとします。

```html
<canvas id="canvas" width="300" height="300"></canvas>
```

次のコードでキャンバスの高さを取得することができます。

```js
const canvas = document.getElementById("canvas");
console.log(canvas.height); // 300
```
