{{APIRef("Geometry Interfaces")}}

**`top`** は **`DOMRectReadOnly`** の読み取り専用インターフェイスで、 `DOMRect` の上端の座標値を返します。（`y` と同じ、または `height` が負の数の場合は `y + height` と同じです。）

## 値

倍精度浮動小数点値です。
