**`numberOfItems`** は {{domxref("SVGNumberList")}} インターフェイスのプロパティで、リスト内のアイテム数を返します。 {{domxref("SVGNumberList.length", "length")}} はこの別名です。

## 値

リストのアイテム数を表す非負の整数です。
