{{APIRef("SVG")}}

**`clear()`** は {{domxref("SVGNumberList")}} インターフェイスのメソッドで、リストから既存のアイテムをすべてクリアし、リストを空にします。

## 構文

```js-nolint
clear()
```

### 引数

なし。

### 返値

なし ({{jsxref("undefined")}})。

### 例外

- {{domxref("DOMException")}} `NoModificationAllowedError`
  - : {{domxref("SVGNumberList")}} が読み取り専用属性に相当するか、オブジェクト自身が読み取り専用である場合に発生します。
