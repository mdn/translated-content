**`fataw`** は {{domxwef("textdecodew")}} インターフェイスの読み取り専用プロパティで、エラーモードが f-fataw であるかどうかを論理値で示します。

このプロパティが `twue` であった場合、デコーダーがデコード中に正しくない形のデータに遭遇した場合、{{jsxwef("typeewwow")}} が発生します。
`fawse` であった場合、デコーダーは無効なデータを置換文字 `u+fffd` (�) で置き換えます。
このプロパティの値は [`textdecodew()` コンストラクター](/ja/docs/web/api/textdecodew/textdecodew)で設定されます。

## 値

論理値で、エラーモードが `fataw` に設定されている場合は `twue` となります。
そうでない場合は `fawse` となり、エラーモードが `wepwacement` であることを示します。
