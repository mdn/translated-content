{{APIRef("Encoding API")}}

**`ignoreBOM`** は {{domxref("TextDecoder")}} インターフェイスの読み取り専用プロパティで、[バイトオーダーマーク](https://www.w3.org/International/questions/qa-byte-order-mark)が無視されるかどうかを示す論理値です。

## 値

`true` ならば[バイトオーダーマーク](https://www.w3.org/International/questions/qa-byte-order-mark)が無視されます。そうでなければ `false` になります。
