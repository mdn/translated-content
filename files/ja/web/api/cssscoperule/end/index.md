**`end`** は {{domxref("CSSScopeRule")}} インターフェイスのプロパティで、 `@scope` アットルールのスコープリミットの値が入った文字列を返します。

## 値

文字列です。 `@scope` アットルールにスコープリミットが定義されていない場合は `null` です。
