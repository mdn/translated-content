**`SVGGeometryElement.getTotalLength()`** メソッドは、ユーザーエージェントが計算したパスの全長をユーザー単位で返します。

## 構文

```js-nolint
getTotalLength()
```

### 引数

なし。

### 返値

浮動小数点数で、パス全体の長さをユーザー単位で示します。
