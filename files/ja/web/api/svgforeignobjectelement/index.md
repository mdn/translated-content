{{APIRef("SVG")}}

**`SVGForeignObjectElement`** インターフェイスは、{{SVGElement("foreignObject")}} 要素のプロパティや、操作するためのメソッドへのアクセス手段を提供します。

{{InheritanceDiagram}}

## インスタンスプロパティ

_このインターフェイスには、親である {{domxref("SVGGraphicsElement")}} から継承したプロパティもあります。_

- {{domxref("SVGForeignObjectElement.x")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、指定された {{SVGElement("foreignObject")}} 要素の {{SVGAttr("x")}} 属性に対応します。
- {{domxref("SVGForeignObjectElement.y")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、指定された {{SVGElement("foreignObject")}} 要素の {{SVGAttr("y")}} 属性に対応します。
- {{domxref("SVGForeignObjectElement.width")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、指定された {{SVGElement("foreignObject")}} 要素の {{SVGAttr("width")}} 属性に対応します。
- {{domxref("SVGForeignObjectElement.height")}} {{ReadOnlyInline}}
  - : {{domxref("SVGAnimatedLength")}} で、指定された {{SVGElement("foreignObject")}} 要素の {{SVGAttr("height")}} 属性に対応します。

## インスタンスメソッド

_このインターフェイスは固有のメソッドを実装していませんが、親である {{domxref("SVGGraphicsElement")}} から継承したメソッドがあります。_
