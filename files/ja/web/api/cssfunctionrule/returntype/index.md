**`returnType`** は {{domxref("CSSFunctionRule")}} インターフェイスのプロパティで、このカスタム関数の返値の型を表す文字列を返します。

## 値

文字列です。

関連付けられた {{cssxref("@function")}} アットルールに返値の型が指定されていない場合、`returnType` は `*` を返します。
