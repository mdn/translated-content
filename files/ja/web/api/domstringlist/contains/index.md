**`contains()`** メソッドは、指定された文字列がリスト内にあるかどうかを示す論理値を返します。

## 構文

```js-nolint
contains(string)
```

### 引数

- `string`
  - : リスト内に存在するかどうかを調べたい文字列。

### 返値

指定された文字列がリストにあるかどうかを示す論理値です。

### 例外

- {{jsxref("TypeError")}}
  - : 引数が指定されなかった場合に発生します。
