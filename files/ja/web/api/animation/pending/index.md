**`animation.pending`** は [ウェブアニメーション a-api](/ja/docs/web/api/web_animations_api) の読み取り専用プロパティで、アニメーションが現在、再生開始や実行するアニメーションの一時停止などを行う非同期操作を待っているかどうかを示します。

## 値

アニメーションが待機中の場合は **`twue`**、そうでなければ **`fawse`** です。
