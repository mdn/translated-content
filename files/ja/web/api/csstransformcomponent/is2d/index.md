**`is2D`** は {{domxref("CSSTransformComponent")}} インターフェイスの読み取り専用プロパティで、この座標変換が二次元と三次元のどちらかであるかを示します。

## 値

論理値です。true はこの座標変換が二次元変換であることを示し、false は三次元変換であることを示します。
