**`is2D`** は {{domxref("CSSTransformComponent")}} インターフェイスの読み取り専用プロパティで、この座標変換が 2D と 3D のどちらかであるかを示します。

## 値

論理値です。true はこの座標変換が 2D 変換であることを示し、false は 3D 変換であることを示します。
