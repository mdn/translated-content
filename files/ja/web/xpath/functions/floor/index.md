`floor` 関数は小数を評価し、その小数以下の最も大きい整数を返します。

### 構文

```
floor( number )
```

### 引数

- `number`
  - : 評価する小数。

### 返値

`number` 以下の最も近い整数。

### 定義

[XPath 1.0 4.4](https://www.w3.org/TR/xpath#function-floor)

### Gecko での対応

対応済み。
