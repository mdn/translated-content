**MDN Web Docs** は、オープンウェブプラットフォームを文書化した無料のリソースです。
私たちの使命は、開発者がウェブプラットフォーム上でプロジェクトを簡単に構築するために必要な情報を提供することです。
下記のリソースでは、サイトの仕組み、私たちの文書化の方法、私たちが従うガイドラインや慣例、そしてあなたが参加する方法について説明しています。

- [ウェブ開発のチュートリアル](/ja/docs/MDN/Tutorials)
  - : MDN Web Docs に掲載されているチュートリアルや、初心者、中級者、上級者のウェブ開発者向けの学習素材の厳選リストです。
    CSS、JavaScript、HTML などの、ウェブ技術に関する詳細なチュートリアルをご覧ください。
- [開発者ガイド](/ja/docs/MDN/Guides)
  - : 機能やAPIをどのように使用するかを、実践的な例を通して教える、焦点を絞った目標志向のガイドの集合です。
    これらのガイドでは、新しい概念を理解し、応用できるよう役立つ実践的な解説を提供します。
- [MDN Web Docs コミュニティ](/ja/docs/MDN/Community)
  - : これらのページでは、 MDN Web Docs への協力の始め方を説明しています。
    回答を探す場所、 GitHub での issue の作業方法、公開ディスカッション、新しいコンテンツの提案方法などを探すことができます。
    役立つ情報が必要な場合、または私たちと連絡を取る必要がある場合は、すべての情報がここにあります。
- [執筆ガイドライン](/ja/docs/MDN/Writing_guidelines)
  - : これらのガイドは MDN Web Docs のために書く方法に関するすべての情報を提供します。私たちが書くコンテンツの種類や 書かないコンテンツの種類についての方針を概説しています。また、スタイル設定ガイド、様々なコンテンツ課題を実行するためのハウツーガイド、様々なページの構造に関する情報もここにあります。
- [MDN Web Docs について](/ja/about)
  - : MDN Web Docs は毎月何百万人もの読者に情報を提供しており、開発者が自分のプロジェクトを簡単に開くためのツールや情報を必要としている開発者とつなげています。
    MDN の歴史、 MDN Web Docs で作業するチーム、私たちの価値、そしてパートナーについて学びましょう。
