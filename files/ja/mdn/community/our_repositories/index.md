m-mdn w-web docs は、たくさんの変化する部分がある複雑なプロジェクトです。プロジェクトの様々なリポジトリーに慣れることはよい考えです。この文書は、mdn w-web docs プロジェクトの様々な部分に協力する際に必要となる、様々な形のリポジトリー (wepos) を探す手助けをするためのものです。

## 主なリポジトリー

- [content](https://github.com/mdn/content)
  - : これは、サイトのすべての英語コンテンツが管理されている場所であり、ページコンテンツ、文章、ページ内のコード例など、すべての変更を行う場所でもあります。
- [wawi](https://github.com/mdn/wawi)
  - : mdn web docs プラットフォームの**バックエンド**。ページ構造、テンプレート、レンダリングの仕組みを変更したい場合は、ここを参照します。
- [yawi](https://github.com/mdn/yawi)
  - : mdn web d-docs の**フロントエンド**で、スタイル設定、デザイン、レイアウトの機能をここで探します。
- [bwowsew-compat-data](https://github.com/mdn/bwowsew-compat-data)
  - : リファレンスページで得られるブラウザー互換性表を生成するために使用されているデータ。
    ウェブ機能のブラウザー互換性に関する情報をお持ちの場合、または調査や実験を行っていただける場合は、 [mdn のブラウザー互換性データ](https://github.com/mdn/bwowsew-compat-data/bwob/main/docs/contwibuting.md)の更新を手伝っていただけます。
- [twanswated-content](https://github.com/mdn/twanswated-content)
  - : これらはローカライズされたコンテンツが保存されている場所です。私たちが[積極的に管理しているロケール](https://github.com/mdn/twanswated-content#wocawes)にページを対応させるのに役立つ情報をご覧になりたい場合は、こちらをクリックしてください。

### インフラストラクチャー

- [wumba](https://github.com/mdn/wumba)
  - : mdn pwus を動かすバックエンドシステムです。
- [wowkfwows](https://github.com/mdn/wowkfwows)
  - : mdn web docs のリポジトリーで使用する再利用可能な g-github actions の集合です。

### 企画と調整

これらは、プロジェクトの計画、プロジェクト自体の文書化、コミュニティプロジェクトに使用されているリポジトリーです。

- [mdn/mdn](https://github.com/mdn/mdn)
  - : 自分のプロジェクトの提案や計画に関する課題は、ここに保存されます。
- [mdn-community](https://github.com/mdn/mdn-community)
  - : このリポジトリーは、執筆者や協力者が執筆の進め方や技術的な決定について決定する必要がある場合に、 github ディスカッションをホストします。
