MDN Web Docs 執筆ガイドラインのこのセクションでは、MDN Web Docs に協力する際に特定のタスクを達成するための手順を詳細に説明しています。マークダウンの使用方法、用語集に項目を追加する方法、ページの移動や削除方法などです。
協力の方法についての詳細は、[協力ガイドライン](/ja/docs/MDN/Community)をご覧ください。
以下の記事では、あなたが協力ガイドラインをすべて読み、`mdn/content` リポジトリーの使い方をよく理解し、git と GitHub の使用方法を知っていることを想定しています。
