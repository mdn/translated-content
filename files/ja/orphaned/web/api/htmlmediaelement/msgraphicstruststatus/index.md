{{APIRef("HTML DOM")}}

{{Non-standard_header()}}

読み取り専用のプロパティである **msGraphicsTrustStatus`** は、保護された動画の再生に関する情報を格納したオブジェクトを返します。

この独自プロパティは、Internet Explorer と Microsoft Edge に固有のものです。

## 値

{{domxref("MSGraphicsTrust")}} オブジェクトです。
