注意: 私たちは開発者とテスター以外には Firefox 2 ベータ 1 マイルストーンリリースのダウンロードをすすめていません。テスト目的のみを意図しています。

Firefox 2 ベータ 1 はダウンロード可能になりました。これは Firefox 2 に予定された多くの新機能とプラットフォームの変更によってもたらされたコア機能のテストに焦点を当てた開発者向けの 4 番目のマイルストーンです。mozilla.dev.planning や irc.mozilla.org の #bonecho と同じようにでも Firefox 2 の進行中の計画を追いかけることができます。

このマイルストーンのフィードバックが必要な新機能と変更には以下が含まれます。

  • 組み込みフィッシング保護機能
  • Google と Yahoo! と Answers.com の検索ボックスでは検索履歴とともに検索サジェストが表示されます。
  • タブブラウジングの動作の変更
  • あやまって閉じてしまったタブを再び開く機能
  • ウェブフィードのプレビューと登録のよりよいサポート
  • テキストボックスのインラインスペルチェック
  • 検索エンジンの削除と並べ替えのための検索プラグインマネージャ
  • ブックマークの新しい機能
  • クラッシュした際のセッションの自動的復元
  • 統合され改善された拡張とテーマのアドオンマネージャ
  • Nullsoft Scriptable Install Systemを元にした Windows 用の新しいインストーラ
  • JavaScript 1.7 のサポート
  • クライアントサイドのセッション及び永続ストレージのサポート
  • 強化されたセキュリティの提供と拡張の地域化をより簡単にする拡張システムの更新
  • svg:textPath 要素を利用した SVG テキストのサポート

Firefox 2 Beta 1 プロジェクトページはテスタにフィードバックの提供とバグの報告を楽にします。幾つかの言語で Windows と Mac OSX と Linux の Firefox 2 ベータ 1のビルドをダウンロードできます。テスタは既知の問題を知るためにリリースノートを読むべきです。開発者は Mozilla Developer Center の開発者のための Firefox 2 に目を通すべきです。

注意: ダウンロードサイトに直接リンクを張らないでください。かわりに私達はこの Firefox 2 ベータ 1 アナウンスへのリンクを強く勧めます。なぜなら全ての人にこのマイルストーンが何であり、彼らに何が求められ、誰がこの開発段階のテストに参加するためにダウンロードすべきかを知るべきだからです。
