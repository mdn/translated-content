Le basi dei Web Service
Una breve introduzione ai web service.

Un Web service è un insieme di protocolli e standard utilizzati per lo scambio dei dati tra applicazioni e sistemi. Le applicazioni scritte in vari linguaggi di programmazione e che funzionano su differenti piattaforme possono utilizzare i web service per comunicare attraverso le reti, come Internet, in un modo simile alla comunicazione tra diversi processi che sono attivi su uno stesso computer. Nei web service XML, tutti i dati sono formattati tramite i tag.
