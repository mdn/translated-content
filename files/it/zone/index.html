Zone MDN fornisce l'accesso punto per punto ad informazioni sui settori o prodotti o argomenti specifici. Ecco un elenco di tutte le zone disponibili.

Sviluppo Web ed applicazioni

App Center
Impara come creare applicazioni Web aperte: vivi l'esperienza di eseguire, su più dispositivi, gli stessi standard Web e tecnologie aperte che già conosci.
Area di apprendimento
Questa sezione contiene tutti gli articoli per supportare i neofiti dello sviluppo sul Web ad iniziare in questo eccitante nuovo mondo.
Tool di Sviluppo
Impara come usare i tool di sviluppo di Firefox per il debug, il test e l'ottimizzazione delle tue applicazioni e siti Web.
Firefox Marketplace
Un marketplace aperto, online e non proprietario, per applicazioni Web sviluppate utilizzando HTML, CSS e JavaScript. Carica le applicazioni nel Marketplace di Firefox o utilizza il codice per costruire il tuo marketplace.
Sviluppo giochi
Impara come sviluppare giochi per il Web, come trasporre giochi esistenti verso tecnologie Web, e come trasformare i tuoi giochi in applicazioni Web.

Prodotti e progetti

Emscripten
Un compilatore da LLVN a JavaScript; questo compilatore permette di compilare, per esempio,  da codice C++ a codice JavaScript, il quale può essere eseguito in qualsiasi Web browser.
L20n
Un framework di localizzazione JavaScript per scatenare il potere della tua lingua naturale con del semplice codice.
Il progetto MDN
Il Mozilla Developer Network (questo sito) si affida alla sua comunità di lettori e collaboratori per crescere e migliorare. Qui puoi imparare a collaborare, contribuire, e costruire il codice dietro MDN!
Persona
Un nuovo sistema semplice, sensibile alla privacy, per l'accesso unificato, sviluppato da Mozilla che permette agli utenti di loggarsi nel tuo sito Web utilizzando il proprio indirizzo email, liberandoti dalla gestione delle password.

Tecnologie Mozilla

Add-ons
Imparare a sviluppare ed installare estensioni, temi, e plug-in per il software basato su Mozilla, incluso Firefox, il popolare browser Web.
Firefox
Impara tutto su Firefox, da come compilarlo e verificarlo a come costruire componenti aggiuntivi specifici.
Firefox OS
Un nuovo sistema operativo mobile, sviluppato da Mozilla che permette di installare ed eseguire applicazioni Web create utilizzando HTML, CSS e JavaScript.
