HyperText Markup Language‏ (HTML) הינה שפה המשמשת ליצירת דפי אינטרנט ומסמכים אחרים הניתנים לצפייה בדפדפן אינטרנט.
HTML הוא תקן בינלאומי המוגדר על ידי ה־World Wide Web Consortium. הגרסה הנוכחית של התקן היא גרסה 5 שהושקה בחודש אוקטובר 2014.

ניתן להעשיר את דפי ה-HTML באמצעות גיליונות סגנון מדורגים (CSS) איתם ניתן לעצב את דפי האינטרנט והמסמכים שנבנו.
בנוסף, ניתן להטמיע תסריטים בדף באמצעות שפת ג'אווה סקריפט (JavasScript) הנחשבת לשפת תכנות דינמית מונחית־עצמים.

 

היכרות ראשונית

מבוא לשפת HTML
היכרות ראשונה עם שפת HTML למשתמשים מתחילים.
רשימת תגיות של שפת HTML
מאגר תגיות בהן ניתן לבצע שימוש.
הטמעת מולטימדיה
כיצד לשלב תמונות, סירטונים וקובצי קול בדף האינטרנט.
טבלאות
בניה והצגת נתונים באמצעות טבלאות בדף אינטרנט.
טפסים
טפסים הם נדבך חשוב בדף האינטרנט ומשמשים ליצירת קשר ראשוני עם הלקוחות באמצעות טפסים ליצירת קשר, קניה ומכירה של מוצרים וליצירת משתמשים חדשים.
פתרון בעיות נפוצות
מאגר שאלות ותשובות המספקות מענה לבעיות נפוצות בכתיבת מסמכים בשפת HTML.
